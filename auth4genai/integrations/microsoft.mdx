The Microsoft Integration enables your GenAI application to authenticate users with their Microsoft accounts and access Microsoft 365 services through Token Vault.

## Overview

The Microsoft Integration provides enterprise-grade integration with Microsoft's ecosystem, including Office 365, Azure AD, and Microsoft Graph APIs. This integration is essential for B2B applications where users need to access their work data and collaborate through Microsoft services.

## Configuration

<Steps>
  <Step title="Register App in Azure AD">
    1. Go to [Azure Portal](https://portal.azure.com)
    2. Navigate to **Azure Active Directory** > **App registrations**
    3. Click **New registration** and configure your app
    4. Note the **Application (client) ID** and create a **Client secret**
  </Step>
  <Step title="Configure API Permissions">
    1. In your app registration, go to **API permissions**
    2. Add permissions for Microsoft Graph APIs you need
    3. Grant admin consent if required for your organization
  </Step>
  <Step title="Set up Auth0 Integration">
    1. In Auth0 Dashboard, go to **Authentication** > **Enterprise**
    2. Select **Microsoft Azure AD** and configure with your credentials
    3. Set the required scopes for your application
    4. Enable **Token Vault** in the Advanced settings
  </Step>
  <Step title="Enable for your Application">
    Ensure the Microsoft Integration is enabled for your Auth0 application.
  </Step>
</Steps>

## Required Scopes

| Service | Scope | Description |
|---------|-------|-------------|
| Profile | `openid profile email` | Basic user profile information |
| Mail | `Mail.Read Mail.Send` | Read and send emails |
| Calendar | `Calendars.ReadWrite` | Manage calendar events |
| Files | `Files.ReadWrite` | Access OneDrive files |
| Teams | `Chat.ReadWrite` | Read and send Teams messages |
| Directory | `User.Read Directory.Read.All` | Read user and directory information |

## Example Use Cases

### Email Automation
```javascript
// Read user's recent emails
const emails = await fetch('https://graph.microsoft.com/v1.0/me/messages', {
  headers: {
    'Authorization': `Bearer ${userToken}`
  }
});
```

### Calendar Management
```javascript
// Create a calendar event
const event = await fetch('https://graph.microsoft.com/v1.0/me/events', {
  method: 'POST',
  headers: {
    'Authorization': `Bearer ${userToken}`,
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    subject: 'AI Generated Meeting',
    start: { dateTime: '2024-01-15T10:00:00', timeZone: 'UTC' },
    end: { dateTime: '2024-01-15T11:00:00', timeZone: 'UTC' }
  })
});
```
