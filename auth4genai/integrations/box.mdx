The Box Integration enables your GenAI application to authenticate users with their Box accounts and access enterprise file storage and collaboration features.

## Overview

Box is an enterprise-focused cloud storage and collaboration platform. This integration allows AI agents to securely access, manage, and collaborate on files stored in Box, making it ideal for business applications that need document management capabilities.

## Configuration

<Steps>
  <Step title="Create Box App">
    1. Go to [Box Developer Console](https://app.box.com/developers/console)
    2. Create a new Custom App with OAuth 2.0
    3. Configure redirect URIs and application scopes
    4. Note your **Client ID** and **Client Secret**
  </Step>
  <Step title="Set up Auth0 Integration">
    1. In Auth0 Dashboard, go to **Authentication** > **Social**
    2. Select **Box** and configure with your credentials
    3. Enable **Token Vault** in Advanced settings
  </Step>
</Steps>

## Required Scopes

| Scope | Description |
|-------|-------------|
| `root_readonly` | Read all files and folders |
| `root_readwrite` | Read and write all files and folders |
| `manage_users` | Manage users (admin only) |
| `manage_groups` | Manage groups (admin only) |
