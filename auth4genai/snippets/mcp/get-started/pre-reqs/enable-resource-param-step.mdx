To use the `resource` parameter in your access tokens, you need to enable the Resource Parameter Compatibility Profile.

The quickest way to enable it is through the [Auth0 Dashboard](https://manage.auth0.com/dashboard/):

1. Navigate to **Settings** on the left sidebar.
2. Select [**Advanced**](https://manage.auth0.com/dashboard/#/tenant/advanced) in the top right corner.
3. Scroll down to the **Settings** section and enable the **Resource Parameter Compatibility Profile** toggle.
