<Tabs>
  <Tab title="Auth0 FGA">
    <Steps>
      <Step title="Create an Auth0 FGA account">
        Sign up for a free account at [fga.dev](https://fga.dev).
      </Step>

      <Step title="Create a store">
        In the [Auth0 FGA dashboard](https://dashboard.fga.dev/), create a new store for your MCP server.
      </Step>

      <Step title="Generate API credentials">
        1. Navigate to **Settings** in your store.
        2. In the **Authorized Clients** section, click **+ Create Client**.
        3. Give your client a name and select the following permissions:
           - **Read/Write model, changes, and assertions** - To query authorization decisions
           - **Write and delete tuples** - To import tuples (user permissions)
           - **Read and query** - To import the authorization model
        4. Click **Create**.
        5. Save the credentials shown in the confirmation modal.
      </Step>

      <Step title="Configure environment variables">
        Set the following environment variables in your terminal. These will be used by the FGA CLI to connect to your store.

        ```shell wrap lines
        export FGA_API_URL='https://api.us1.fga.dev'
        export FGA_STORE_ID='<your-store-id>'
        export FGA_API_TOKEN_ISSUER='auth.fga.dev'
        export FGA_API_AUDIENCE='https://api.us1.fga.dev/'
        export FGA_CLIENT_ID='<your-client-id>'
        export FGA_CLIENT_SECRET='<your-client-secret>'
        ```

        Replace the placeholders with the credentials from the previous step. Note that `FGA_API_URL` may vary based on your region (e.g., `api.eu1.fga.dev` for Europe).
      </Step>
    </Steps>
  </Tab>

  <Tab title="OpenFGA (Local)">
    <Steps>
      <Step title="Install OpenFGA">
        Install OpenFGA using Homebrew or Docker:

        <Tabs>
          <Tab title="Homebrew">
            ```shell
            brew install openfga
            ```
          </Tab>

          <Tab title="Docker">
            Docker installation will be used in the next step to run the server.
          </Tab>
        </Tabs>

        For other installation methods, see the [OpenFGA releases page](https://github.com/openfga/openfga/releases).
      </Step>

      <Step title="Start OpenFGA">
        Start the OpenFGA server:

        <Tabs>
          <Tab title="Homebrew">
            ```shell
            openfga run
            ```
          </Tab>

          <Tab title="Docker">
            ```shell
            docker run -p 8080:8080 -p 8081:8081 -p 3000:3000 openfga/openfga run
            ```
          </Tab>
        </Tabs>

        The server will start on port 8080 by default. Leave this running in a separate terminal window.
      </Step>

      <Step title="Configure environment variables">
        Set the following environment variables in your terminal (in a different terminal from where OpenFGA is running):

        ```shell
        export FGA_API_URL='http://localhost:8080'
        ```

        <Note>
        The `FGA_STORE_ID` will be set after importing the model in the next section.
        </Note>
      </Step>
    </Steps>
  </Tab>
</Tabs>
