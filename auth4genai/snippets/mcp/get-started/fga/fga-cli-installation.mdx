This guide uses the [OpenFGA CLI](https://openfga.dev/docs/getting-started/install-sdk#installing-the-cli) to manage authorization models and permissions.

1. Install the FGA CLI:

<Tabs>
  <Tab title="macOS">
    ```shell
    brew install openfga/tap/fga
    ```
  </Tab>

  <Tab title="Linux">
    Download the latest release from the [OpenFGA CLI releases page](https://github.com/openfga/cli/releases) and add it to your PATH.

    ```shell
    # Example for Linux (replace version with latest)
    wget https://github.com/openfga/cli/releases/download/v0.6.3/fga_0.6.3_linux_amd64.tar.gz
    tar -xzf fga_0.6.3_linux_amd64.tar.gz
    sudo mv fga /usr/local/bin/
    ```
  </Tab>

  <Tab title="Windows">
    Download the latest release from the [OpenFGA CLI releases page](https://github.com/openfga/cli/releases) and add it to your PATH.
  </Tab>
</Tabs>

2. Verify the installation:
  ```shell
  fga version
  ```
