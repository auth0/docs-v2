---
description: Explique comment réinitialiser les mots de passe des comptes Auth0.
title: Réinitialiser les mots de passe des comptes
---

Si vous devez changer votre mot de passe ou si vous avez oublié celui de votre compte Auth0, dans la plupart des cas, vous pouvez en définir un nouveau à partir du <Tooltip tip="Auth0 Dashboard : principal produit Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Les réinitialisations de mot de passe entraînent l’expiration des sessions Auth0.

1. À l’écran de connexion du compte Auth0, saisissez l’adresse courriel associée à votre compte, puis cliquez sur **Continuer**, puis choisissez **Mot de passe oublié ?** à l’écran suivant.
2. Confirmez votre adresse courriel et cliquez sur **Continuer** pour envoyer la demande. Vous recevrez un courriel contenant des instructions supplémentaires pour réinitialiser votre mot de passe.
3. Si votre demande a bien été reçue, le message suivant s’affichera : « Veuillez vérifier l’adresse courriel [your email] pour obtenir les instructions de réinitialisation de votre mot de passe. »

Il existe certaines circonstances particulières concernant la réinitialisation du mot de passe :

* Si vous avez activé la <Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=MFA">MFA</Tooltip> et que vous devez réinitialiser la MFA de votre compte d’administrateur, communiquez avec le [soutien Auth0](https://support.auth0.com).
* Si vous utilisez un compte social ou d’entreprise pour vous connecter à Auth0, réinitialisez votre mot de passe auprès du <Tooltip tip="Fournisseur d’identité (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=identity+provider">fournisseur d’identité</Tooltip> approprié.
* Si vous êtes un administrateur qui tente de réinitialiser le mot de passe d’un utilisateur, il existe deux méthodes :

  * Utilisez le flux interactif de réinitialisation du mot de passe avec Universal Login ou l’Authentication API, dans le cadre duquel l’utilisateur reçoit un courriel contenant un lien qui ouvre la page Auth0 de réinitialisation du mot de passe pour saisir un nouveau mot de passe.
  * Définissez directement le nouveau mot de passe à l’aide de la Management API ou de l’Auth0 Dashboard.
