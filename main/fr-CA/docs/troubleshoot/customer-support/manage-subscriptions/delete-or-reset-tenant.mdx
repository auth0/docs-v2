Vous pouvez supprimer votre locataire Auth0 ou le rétablir à la configuration par défaut dans le <Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Vous pouvez supprimer les données de votre compte Auth0 avec l’aide d’Auth0 Support.

<div id="delete-tenant">
  ## Supprimer le locataire
</div>

<Warning>
  La suppression d’un locataire est une action irréversible et destructive qui ne peut pas être annulée. Les locataires supprimés ne peuvent pas être restaurés, le nom du locataire ne peut pas être réutilisé lors de la création d’un nouveau locataire, et les métriques associées au locataire sont perdues.
</Warning>

Avant de supprimer un locataire, tenez compte des options suivantes :

* [Changer le propriétaire du locataire](/fr-CA/docs/get-started/manage-dashboard-access).
* [Réinitialiser la configuration du locataire](#reset-tenant-configuration).

<div id="remove-custom-domain">
  ### Retirer le domaine personnalisé
</div>

Si vous avez configuré un <Tooltip tip="Domaine personnalisé : domaine tiers avec un nom spécialisé ou personnalisé." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=custom+domain">domaine personnalisé</Tooltip> pour un locataire que vous souhaitez supprimer, vous devez d’abord retirer le domaine personnalisé. Sinon, vous ne pourrez pas configurer ce domaine personnalisé pour un autre locataire.

<div id="determine-deletion-process">
  ### Déterminer le processus de suppression
</div>

Le processus de suppression d’un locataire dépend du [forfait d’abonnement](https://www.auth0.com/pricing) auquel il est associé et du [type de locataire](/fr-CA/docs/get-started/auth0-overview/create-tenants/child-tenants) (principal ou enfant). Vous pourrez peut-être supprimer le locataire dans l’[Auth0 Dashboard](#delete-tenant-in-auth0-dashboard), ou vous devrez peut-être [contacter Auth0 Support ou Auth0 Community](#contact-auth0-support-or-community).

|                                                     | forfait Free                 | Forfait en libre-service | Forfait Enterprise      |
| --------------------------------------------------- | ---------------------------- | ------------------------ | ----------------------- |
| **Locataire principal**                             | Auth0 Dashboard              | Auth0 Dashboard          | Contacter Auth0 Support |
| **Locataire principal avec des locataires enfants** | Contacter Auth0 Community    | Contacter Auth0 Support  | Contacter Auth0 Support |
| **Locataire enfant**                                | Auth0 Dashboard              | Auth0 Dashboard          | Auth0 Dashboard         |

<div id="downgrade-tenant">
  ### Rétrograder un locataire
</div>

Avant de pouvoir supprimer un locataire dans l’Auth0 Dashboard, vous devez d’abord le faire passer au forfait Free. Pour en savoir plus, consultez [Rétrograder ou annuler des abonnements Auth0](/fr-CA/docs/troubleshoot/customer-support/manage-subscriptions/downgrade-or-cancel-subscriptions).

<div id="delete-tenant-in-dashboard">
  ### Supprimer le locataire dans l’Auth0 Dashboard
</div>

Vous pouvez supprimer un locataire dans l’Auth0 Dashboard si vous avez le [niveau d’accès approprié](/fr-CA/docs/get-started/manage-dashboard-access/feature-access-by-role).

1. Accédez à [Auth0 Dashboard &gt; Paramètres](https://manage.auth0.com/#/tenant/advanced), puis passez à la vue **Avancé**.
2. Repérez la section **Zone de danger** au bas de la page, puis sélectionnez **Supprimer**.
3. Vérifiez l’invite de confirmation et confirmez que vous voulez supprimer définitivement votre locataire.
4. Sélectionnez **Oui, supprimer le locataire**.

<div id="contact-auth0-support-or-community">
  ### Contacter Auth0 Support ou la communauté
</div>

Si votre locataire est associé à un forfait Trial, Essential, Professional ou Enterprise et que vous ne pouvez pas supprimer votre locataire dans l’Auth0 Dashboard, [ouvrez un ticket auprès d’Auth0 Support](/fr-CA/docs/troubleshoot/customer-support/open-and-manage-support-tickets) pour obtenir de l’aide.

Si votre locataire est associé au forfait Free et que vous ne pouvez pas le supprimer dans l’Auth0 Dashboard, [publiez un message dans Auth0 Community](https://community.auth0.com/) pour obtenir de l’aide.

<div id="reset-tenant-configuration">
  ## Réinitialiser la configuration du locataire
</div>

Vous pouvez réinitialiser votre locataire à sa configuration par défaut à l’aide de [Auth0 Deploy CLI](https://github.com/auth0/auth0-deploy-cli).

Deploy CLI est un outil qui vous aide à gérer la configuration de votre locataire Auth0. Il s’intègre à vos flux de travail de développement en tant qu’interface de ligne de commande autonome ou comme module Node.js. Pour en savoir plus sur l’outil Deploy CLI et pour commencer, consultez [l’outil Deploy CLI](/fr-CA/docs/deploy-monitor/deploy-cli-tool).

<div id="delete-auth0-account-data">
  ## Supprimer les données de votre compte Auth0
</div>

Si vous souhaitez supprimer les données de votre compte Auth0 (y compris l’adresse de courriel, les renseignements de facturation, etc.), suivez la procédure pour supprimer votre locataire, puis [publiez un message dans Auth0 Community](https://community.auth0.com/) pour demander de l’aide.

Pour en savoir plus sur les responsabilités d’Auth0 à titre de responsable du traitement des données, consultez la [Politique de confidentialité d’Okta](https://www.okta.com/privacy-policy/).
