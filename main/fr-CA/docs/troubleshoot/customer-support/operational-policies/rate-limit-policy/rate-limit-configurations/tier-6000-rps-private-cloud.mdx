title: Performance de cloud privé 6000 RPS (60x) et 6000 RPS (60x) Burst
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total de l’Authentication API et de la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une forte utilisation combinée des Authentication API et Management API, ou certains flux d’authentification comme le ROPG vers l’Authentication API peuvent entraîner une baisse des performances à des valeurs de RPS inférieures.
</Callout>

Consultez ci-dessous les limites de débit pour les types d’abonnement Performance de cloud privé 6000 RPS (60x) et 6000 RPS (60x) Burst.

Nous recommandons donc de déployer un locataire par environnement de cloud privé afin de réduire les risques.
