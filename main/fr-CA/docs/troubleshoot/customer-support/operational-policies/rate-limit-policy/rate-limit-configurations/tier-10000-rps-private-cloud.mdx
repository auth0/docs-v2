title: Performance du nuage privé 10 000 RPS (100x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Le débit soutenu total pour l’Authentication API et la MyAccount API dans un environnement est généralement pris en charge jusqu’à 500 RPS; toutefois, certaines charges de travail particulières, une utilisation combinée intensive de l’Authentication API et de la Management API, ou certains flux d’authentification comme le ROPG avec l’Authentication API peuvent entraîner une baisse des performances à un nombre de RPS inférieur.
</Callout>

Consultez ci-dessous les limites de débit associées au type d’abonnement Performance du nuage privé 10 000 RPS (100x).

Nous recommandons donc de déployer un locataire par environnement de nuage privé afin de réduire les risques.
