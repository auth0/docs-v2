title: Mettre à jour les informations de facturation
---

Vous pouvez utiliser le <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> pour mettre à jour vos informations de facturation. Cette fonctionnalité est offerte uniquement aux administrateurs du locataire pour les abonnements par locataire. Pour les abonnements par compte, vous devez être propriétaire de l’équipe pour y accéder.

<div id="payment-and-billing">
  ## Paiement et facturation
</div>

Pour mettre à jour vos renseignements de facturation :

1. Accédez à [Auth0 Dashboard &gt; Paramètres](https://manage.auth0.com/#/dashboard/settings).

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Si vous utilisez une équipe, vous trouverez ce paramètre sous **Équipes &gt; Paramètres &gt; Abonnements**.
   </Callout>

2. Sous **Paramètres du locataire**, sélectionnez **Paiement et facturation**.

3. Cliquez sur **Ajouter les détails de paiement** pour saisir vos renseignements de paiement.

4. Cliquez sur **Mettre à jour** pour enregistrer vos modifications.
