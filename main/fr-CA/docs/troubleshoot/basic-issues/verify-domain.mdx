title: Vérifier le domaine
---

* Assurez-vous que le domaine est le même que celui utilisé lors de l’authentification.
* Si vous utilisez un [Domaine personnalisé](/fr-CA/docs/customize/custom-domains) d’Auth0, il est important d’utiliser le même domaine que celui utilisé dans l’application pour déclencher l’authentification. Consultez [Résoudre les problèmes liés aux Domaines personnalisés](/fr-CA/docs/troubleshoot/integration-extensibility-issues/troubleshoot-custom-domains) pour plus de détails.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Consultez [la page Monitoring](/fr-CA/docs/deploy-monitor/monitor) pour en savoir plus sur chacun des événements du journal pouvant vous aider à résoudre les problèmes.
</Callout>
