<div id="user-issues">
  ## Problèmes rencontrés par les utilisateurs
</div>

<div id="if-you-do-not-have-your-mobile-device-or-your-mobile-device-is-turned-off">
  ### Si vous n’avez pas votre appareil mobile avec vous ou si votre appareil mobile est éteint
</div>

Si vous avez perdu votre appareil, vous pouvez terminer l’authentification à l’aide du code de récupération fourni lors de votre inscription initiale.

1. Entrez votre courriel et votre mot de passe pour vous connecter, puis cliquez sur le lien **Utiliser le code de récupération**.
2. Entrez votre code de récupération.

Si vous n’avez plus votre code de récupération, vous ne pourrez pas vous connecter. Communiquez avec votre administrateur système pour obtenir de l’aide afin d’accéder à votre compte.

<div id="if-you-forget-your-password">
  ### Si vous oubliez votre mot de passe
</div>

Si vous avez oublié votre mot de passe, cliquez sur le lien **Vous avez oublié votre mot de passe ?** situé sous les champs de courriel et de mot de passe. Saisissez ensuite votre adresse de courriel pour recevoir un courriel contenant un lien qui vous permettra de réinitialiser votre mot de passe.

<div id="if-your-transaction-expires">
  ### Si votre transaction expire
</div>

Lorsque vous vous connectez au moyen de <Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification d’un utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=MFA">MFA</Tooltip>, il ne peut pas s’écouler plus de cinq minutes entre la saisie du premier et du deuxième facteur. Vous pouvez voir combien de temps s’est écoulé depuis que vous vous êtes connecté avec le premier facteur en vérifiant l’horodatage des messages reçus.

Si plus de cinq minutes se sont écoulées, vous devrez vous reconnecter et obtenir un nouveau code ou une nouvelle notification.

Si vous demandez des SMS, assurez-vous de ne pas dépasser les limites de débit.

<div id="if-you-need-to-remove-or-delete-mfa-from-a-user-in-your-tenant">
  ### Si vous devez supprimer ou réinitialiser le MFA pour un utilisateur dans votre locataire
</div>

Si vous devez supprimer ou réinitialiser le MFA pour un utilisateur, vous devez [réinitialiser le MFA](/fr-CA/docs/secure/multi-factor-authentication/reset-user-mfa).

<div id="sms-related-issues">
  ## Problèmes liés aux SMS
</div>

<div id="if-you-did-not-receive-an-sms-message">
  ### Si vous n’avez pas reçu de SMS
</div>

Si vous n’avez pas reçu votre code à six chiffres par SMS, vérifiez que le numéro de téléphone que vous avez fourni est exact. Si c’est le cas, assurez-vous d’avoir un signal cellulaire.

Si vous ne recevez toujours pas les messages, vérifiez auprès de votre fournisseur de services mobiles qu’ils ne sont pas bloqués.

<div id="sms-message-rate-limits">
  ### Limites de débit des messages SMS
</div>

Si vous tentez d’envoyer plus de dix messages SMS à votre appareil dans un délai d’une heure, un message d’erreur lié à une limite de débit s’affichera.

Lorsque vous dépassez votre limite de messagerie, vous devez attendre au moins une heure après avoir demandé votre premier message avant d’en demander un autre. Une tentative supplémentaire vous sera accordée après chaque heure additionnelle écoulée.

<div id="rejected-codes">
  ## Codes refusés
</div>

Si le code à 6 chiffres affiché dans l’application Guardian ou Google Authenticator est refusé à la connexion (souvent avec le message `Incorrect Code`), vérifiez d’abord que vous avez sélectionné la bonne entrée dans la liste de votre application d’authentification. Si vous avez confirmé que c’est bien la bonne entrée, assurez-vous que les paramètres d’horloge de votre appareil mobile sont corrects. Les mots de passe à usage unique sont générés à partir du temps universel coordonné (UTC); l’heure de votre appareil doit donc être exacte pour que votre code fonctionne.

Pour vérifier les paramètres de votre horloge :

* **Appareils Android** - Accédez à **Settings** &gt; **Date &amp; Time**. Assurez-vous que la case à côté de **Automatic** est cochée. Pour la désactiver, accédez à **Settings &gt; Date &amp; Time**. Touchez la case à côté de **Automatic** pour la décocher.
* **Appareils iOS** - Accédez à **Settings** &gt; **General** &gt; **Date &amp; Time**. Activez **Set Automatically**. Si ce paramètre est déjà activé, vous pouvez le désactiver un instant, puis le réactiver.
