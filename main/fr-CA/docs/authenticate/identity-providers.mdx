---
title: Fournisseurs d’identité
description: Introduction aux sources d’utilisateurs pour les applications, y compris les fournisseurs d’identité, les bases de données et les méthodes d’authentification Passwordless.
---

Une connexion représente le lien entre Auth0 et une source d’utilisateurs, qui peut inclure des fournisseurs d’identité externes (comme Google ou LinkedIn), des bases de données ou des méthodes d’authentification Passwordless. Auth0 s’interpose entre votre application et ses sources d’utilisateurs, ce qui ajoute un niveau d’abstraction et isole votre application de toute modification ainsi que des particularités d’implémentation propres à chaque source.

Par défaut, Auth0 synchronise automatiquement les données de profil de l’utilisateur à chaque ouverture de session, ce qui garantit que les modifications apportées à la source de la connexion sont automatiquement répercutées dans Auth0. Vous pouvez désactiver cette synchronisation afin de pouvoir mettre à jour les attributs du profil depuis votre application.

Vous pouvez configurer un nombre illimité de connexions pour vos applications.
