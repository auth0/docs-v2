---

La connexion sociale est une méthode d’authentification qui permet aux utilisateurs de se connecter à une application à l’aide d’identifiants existants d’un <Tooltip tip="Fournisseur d’identité (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=identity+provider">fournisseur d’identité</Tooltip> social, comme Google ou Facebook. Comme les utilisateurs ont souvent leurs identifiants sociaux enregistrés dans leur navigateur ou sur leur appareil, la connexion sociale offre une expérience utilisateur fluide qui exige très peu d’interaction manuelle avec une application.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  L’authentification des utilisateurs est distincte de la connexion et de l’autorisation d’applications auprès d’un fournisseur externe, ce qui permet à celles-ci d’accéder à des API externes au nom de l’utilisateur. Pour en savoir plus, consultez [Authentification des utilisateurs ou comptes connectés](/fr-CA/docs/secure/tokens/token-vault/connected-accounts-for-token-vault#user-authentication-vs-connected-accounts).
</Callout>

Auth0 prend en charge la connexion sociale pour les applications Web et natives. Vous pouvez consulter la liste des fournisseurs d’identité sociaux pris en charge sur le [Auth0 Marketplace](https://marketplace.auth0.com/categories/social-login) ou dans le <Tooltip tip="Auth0 Dashboard : produit principal d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> :

* La [catégorie Social Login de l’Auth0 Marketplace](https://marketplace.auth0.com/categories/social-login) fournit des informations pour chaque connexion sociale prise en charge. Vous pouvez utiliser le bouton **Add Integration** pour lancer le processus de configuration dans votre locataire Auth0 et l’onglet **Installation** pour consulter la documentation de mise en œuvre.

* Dans [Auth0 Dashboard &gt; Authentication &gt; Social](https://manage.auth0.com/#/connections/social), sélectionnez **Create Connection** pour afficher une liste des fournisseurs d’identité sociaux pris en charge. Le **Setup Guide** contient les étapes pour configurer la connexion ainsi que des liens vers la documentation pertinente du fournisseur d’identité social.

Pour mener à bien l’implémentation, assurez-vous d’[enregistrer des clés de développeur](/fr-CA/docs/authenticate/identity-providers/social-identity-providers/devkeys) pour le fournisseur sélectionné.

Si un fournisseur particulier n’est pas disponible, vous pouvez aussi [configurer des connexions sociales personnalisées](/fr-CA/docs/authenticate/identity-providers/social-identity-providers/oauth2).
