description: Utilisez Auth0 Dashboard pour trouver l’ID de connexion d’une connexion Auth0 afin de référencer et de configurer des services Auth0, par exemple dans Actions et la gestion des utilisateurs.
title: Trouver l’ID ou le nom de la connexion
---

Certaines tâches de gestion des utilisateurs nécessitent l’ID d’une connexion, soit un identifiant unique pour une source d’utilisateurs. Par exemple, si vous importez ou exportez des utilisateurs, ou si vous vous connectez à des <Tooltip tip="fournisseur d’identité (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=identity+providers">fournisseurs d’identité</Tooltip>, vous devez fournir un ID de connexion.

1. Accédez à [Auth0 Dashboard](https://manage.auth0.com/#/), sélectionnez **Authentication**, puis le type de connexion dont vous avez besoin de l’ID.
2. Sélectionnez le nom de la connexion dont vous avez besoin de l’ID.
3. Dans la vue **Settings**, repérez le champ **Name**; il contient l’ID de la connexion.
