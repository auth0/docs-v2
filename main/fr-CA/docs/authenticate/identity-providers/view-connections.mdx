---
description: Décrit comment afficher les connexions activées dans Auth0 Dashboard.
title: Voir les connexions
---

Vous pouvez afficher toutes les connexions disponibles et configurées à l’aide de <Tooltip tip="Auth0 Dashboard : la console principale d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Les connexions configurées peuvent être utilisées pour se connecter à vos applications.

1. Accédez à [Auth0 Dashboard &gt; Authentication](https://manage.auth0.com/#/connections).
2. Sélectionnez un type de connexion :

* [Base de données](https://manage.auth0.com/#/connections/database)
* [Social](https://manage.auth0.com/#/connections/social)
* [Entreprise](https://manage.auth0.com/#/connections/enterprise)
* [Passwordless](https://manage.auth0.com/#/connections/passwordless)
