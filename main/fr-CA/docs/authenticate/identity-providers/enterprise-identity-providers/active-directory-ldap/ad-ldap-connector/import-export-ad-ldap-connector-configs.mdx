---
description: Décrit comment importer ou exporter une configuration du connecteur AD/LDAP à partir de la Connector Admin Console.
title: Importer et exporter des configurations du connecteur AD/LDAP
---

Vous pouvez exporter la configuration du connecteur AD/LDAP ou importer une configuration précédemment exportée. Cela est utile pour les déploiements comportant plus d’un nœud du connecteur AD/LDAP afin d’assurer une haute disponibilité. Vous pouvez configurer et tester la configuration sur un nœud, puis exporter la configuration fonctionnelle et l’importer sur tous les nœuds suivants.

1. Dans un navigateur, accédez à `http://localhost:8357` pour lancer la **Connector Admin Console** sur le serveur du connecteur AD/LDAP.
2. Cliquez sur l’onglet **Import / Export**.
3. Exportez un fichier .zip contenant le fichier `config.json`, le dossier `certs` et le fichier `lib\\profileMapper.js` du dossier `%Program Files(x86)%\Auth0\AD LDAP Connector\`.
4. Pour importer une configuration, téléversez un fichier .zip de sauvegarde dans l’onglet **Import / Export**.
