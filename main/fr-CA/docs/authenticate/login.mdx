Auth0 offre deux façons d’implémenter l’authentification de connexion pour vos applications :

* **<Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur le serveur d’autorisation d’Auth0, afin de vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Universal+Login">Universal Login</Tooltip>** où les utilisateurs se connectent à votre application au moyen d’une page hébergée par Auth0.

* **Embedded Login** où les utilisateurs se connectent à votre application au moyen d’une page que vous hébergez.

Dans la grande majorité des cas, nous recommandons Universal Login, mais vous pouvez aussi utiliser Client-Initiated Backchannel Authentication pour authentifier les utilisateurs au moyen d’un flux d’authentification découplé.
