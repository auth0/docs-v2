description: Découvrez comment déprovisionner les utilisateurs de votre intégration SAML.
title: Déprovisionner des utilisateurs dans les intégrations SAML
---

Si vous devez supprimer l’accès à l’application pour des utilisateurs, vous devrez au minimum les déprovisionner auprès du <Tooltip tip="Fournisseur d’identité (IdP) : service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=identity+provider">fournisseur d’identité</Tooltip>. Selon le fournisseur d’identité, les étapes requises pour déprovisionner un compte utilisateur varient. Communiquez avec votre fournisseur pour obtenir des instructions supplémentaires.

Une fois le compte d’un utilisateur supprimé ou désactivé auprès du fournisseur d’identité, l’utilisateur ne pourra plus se connecter.

Vous pouvez également supprimer les comptes utilisateur Auth0 des utilisateurs déprovisionnés si Auth0 est le fournisseur de services ou si votre application s’intègre à Auth0. Qu’Auth0 soit le fournisseur d’identité ou le fournisseur de services, vous pouvez supprimer des utilisateurs à l’aide du [Auth0 Dashboard](https://manage.auth0.com/#/users) ou de la <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>.
