description: Découvrez comment activer le module complémentaire SAML2 Web App pour l’utiliser avec Auth0 comme fournisseur d’identité SAML pour l’authentification unique (SSO).
title: Activer le module complémentaire SAML2 Web App
---

Vous pouvez configurer un module complémentaire SAML2 Web App à utiliser avec une application.

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications).
2. Repérez l’application avec laquelle vous souhaitez utiliser SAML2 Web App, puis cliquez sur le nom de l’application pour accéder à ses paramètres de configuration.
3. Accédez à l’onglet **Addons**.
4. Dans la section **SAML2 Web App**, cliquez sur le curseur pour activer le module complémentaire.

   <Frame>![Tableau de bord Applications Applications onglet Addons onglet Settings SAML2 Web App](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>
5. Accédez à l’onglet **Settings** et saisissez votre **Application Callback URL**, qui reçoit la réponse SAML, ainsi que les **Settings**. L’onglet **Settings** contient un exemple d’objet que vous pouvez utiliser pour déterminer les paramètres à activer.
6. Cliquez sur **Debug** pour vérifier si les paramètres fournis sont valides.
7. Cliquez sur **Save** lorsque vous avez terminé.

<div id="provide-application-details">
  ## Fournir les détails de l’application
</div>

Bien que les étapes précises pour fournir les détails de votre application à votre SAML2 Web App puissent varier, vous pouvez obtenir les renseignements de configuration nécessaires dans l’onglet **Usage** après avoir configuré le module complémentaire.

<Frame>![Tableau de bord Applications Applications onglet Addons onglet Usage de la SAML2 Web App](/docs/images/cdy7uua7fh8z/3T90BDpyTXFUWDp1JkncBU/86bdba6bed8130b34b42b25c9bf2d232/dashboard-applications-applications-addons-saml2-web-app-usage.png)</Frame>

  ## Limitation
</div>

Le module complémentaire SAML2 Web App ne prend pas en charge les requêtes <Tooltip tip="Security Assertion Markup Language (SAML) : protocole standardisé permettant à deux parties d’échanger des renseignements d’authentification sans mot de passe." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=SAML">SAML</Tooltip> passives (p. ex. lorsque `isPassive=true`).
