sidebarTitle: Méthodes d’authentification
description: Décrit les méthodes d’authentification prises en charge par les connexions Passwordless d’Auth0, notamment le courriel, le lien magique et le SMS.
---

Avec les connexions <Tooltip tip="Passwordless : forme d’authentification qui ne repose pas sur un mot de passe comme premier facteur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Passwordless">Passwordless</Tooltip>, les utilisateurs peuvent se connecter sans mot de passe. Ils peuvent plutôt utiliser diverses autres méthodes d’authentification. Les connexions Passwordless d’Auth0 prennent en charge les méthodes d’authentification suivantes :

| Facteur                                                                                 | Description                                                                                                                                                                                                                                                                             |
| --------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Courriel](/fr-CA/docs/authenticate/passwordless/authentication-methods/email-otp)            | L’utilisateur est invité à saisir son adresse courriel. Auth0 lui envoie ensuite un code à usage unique, que l’utilisateur saisit dans votre application.                                                                                                                               |
| [Lien magique](/fr-CA/docs/authenticate/passwordless/authentication-methods/email-magic-link) | L’utilisateur est invité à saisir son adresse courriel. Auth0 lui envoie ensuite un courriel contenant un lien. L’utilisateur clique sur le lien et est automatiquement connecté à votre application.                                                                                   |
| [SMS](/fr-CA/docs/authenticate/passwordless/authentication-methods/sms-otp)                   | L’utilisateur est invité à saisir son numéro de téléphone. Auth0 lui envoie ensuite un code à usage unique. Par défaut, Auth0 utilise Twilio pour envoyer le code, mais si vous avez une passerelle SMS personnalisée, vous pouvez modifier votre connexion pour l’utiliser à la place. |
