<Card title="Authentification de base de données sur les plateformes natives">
  L’authentification par nom d’utilisateur/courriel et mot de passe à partir d’applications natives est désactivée par défaut pour les nouveaux locataires depuis le 8 juin 2017. Il est recommandé d’utiliser Universal Login et d’effectuer plutôt une authentification Web. Si vous souhaitez quand même continuer, vous devrez d’abord activer le type d’octroi par mot de passe dans votre tableau de bord. Consultez [Types d’octroi d’application](/fr-CA/docs/get-started/applications/application-grant-types) pour en savoir plus.
</Card>

L’API d’authentification fournit des méthodes pour authentifier des utilisateurs et les inscrire dans une connexion de base de données.

<div id="signing-up-with-a-database-connection">
  ## S’inscrire avec une connexion de base de données
</div>

Pour s’inscrire, vous devez appeler la méthode `createUser` en fournissant l’`email`, le `password` et le nom de la `connection` de l’utilisateur afin de lancer le processus d’inscription. Vous pouvez aussi préciser des métadonnées utilisateur supplémentaires à stocker.

```swift lines
Auth0
    .authentication()
    .createUser(
        email: "support@auth0.com",
        password: "secret-password",
        connection: "Username-Password-Authentication",
        userMetadata: ["first_name": "First",
                       "last_name": "Last"])
    .start { result in
        switch result {
        case .success(let user):
            print("User: \(user)")
        case .failure(let error):
            print("Failed with \(error)")
        }
    }
```

<div id="login-with-a-database-connection">
  ## Se connecter avec une connexion de base de données
</div>

Pour vous connecter avec une connexion de base de données, appelez `login` en fournissant le nom d’utilisateur ou le courriel de l’utilisateur, son mot de passe et le nom de la connexion (par exemple `Username-Password-Authentication`) avec laquelle vous voulez vous authentifier. La réponse est un objet Credentials.

```swift lines
Auth0
    .authentication()
    .login(
        usernameOrEmail: "support@auth0.com",
        password: "secret-password",
        realm: "Username-Password-Authentication",
        scope: "openid")
     .start { result in
         switch result {
         case .success(let credentials):
            print("Credentials: \(credentials)")
         case .failure(let error):
            print("Failed with \(error)")
         }
     }
```
