ResendCodeOptions
description Options pour la méthode `resendCode` sur l’écran Login de vérification du courriel.
Cette opération est utilisée lorsque l’utilisateur demande l’envoi d’un nouveau code de vérification à son courriel,
par exemple, s’il n’a pas reçu le code initial ou si celui-ci a expiré.

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
