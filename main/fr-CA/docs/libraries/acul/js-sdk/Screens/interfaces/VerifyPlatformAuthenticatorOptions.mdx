VerifyPlatformAuthenticatorOptions

```ts Example
export interface VerifyPlatformAuthenticatorOptions extends CustomOptions {
  /**
   * Facultatif. Si la valeur est true et que `screen.showRememberDevice` est true, cela indique que l'utilisateur
   * a choisi de mémoriser cet appareil, généralement pendant 30 jours, afin de contourner la MFA lors des connexions ultérieures.
   * @type {boolean}
   */
  rememberDevice?: boolean;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="rememberDevice?" type="boolean">
  Facultatif. Si la valeur est `true`, tente de mémoriser le navigateur pour les futures demandes de MFA.
  Correspond au champ de formulaire `rememberBrowser`. S’applique uniquement si `screen.showRememberDevice` est `true`.
</ParamField>
