TryAnotherMethodOptions

```ts Example
export interface TryAnotherMethodOptions extends CustomOptions {
  /**
   * La méthode d'envoi initialement présentée ou sélectionnée sur l'écran actuel.
   * Elle est requise par le point de terminaison de l'API pour traiter correctement l'action 'pick-authenticator'.
   * - `sms`: Indique que l'écran actuel était destiné au défi par SMS.
   * - `voice`: Indique que l'écran actuel était destiné au défi vocal.
   * @type {'sms' | 'voice'}
   */
  type: 'sms' | 'voice';
}
```

<ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;">
  La méthode d’envoi initialement affichée ou sélectionnée sur l’écran en cours.
  Ce champ est requis par le point de terminaison de l’API pour traiter correctement l’action `pick-authenticator`.

  * `sms` : indique que l’écran en cours correspondait à un défi SMS.
  * `voice` : indique que l’écran en cours correspondait à un défi vocal.
</ParamField>
