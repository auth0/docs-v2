---
title: "ShowErrorOptions"
---

Options permettant de signaler une erreur du navigateur liée à WebAuthn.

```ts Example
export interface ShowErrorOptions {
  /**
   * Les détails de l'erreur WebAuthn.
   */
  error: WebAuthnErrorDetails;
  /**
   * Toutes les options personnalisées supplémentaires.
   */
  [key: string]: string | number | boolean | undefined | WebAuthnErrorDetails;
}
```
