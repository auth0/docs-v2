```ts Example
export interface UserMembers {
  id: string | null;
  email: string | null;
  username: string | null;
  phoneNumber: string | null;
  picture: string | null;
  enrolledFactors: Array<string> | null;
  enrolledEmails: Array<EnrolledEmail> | null;
  enrolledPhoneNumbers: Array<EnrolledPhoneNumber> | null;
  enrolledDevices: Array<EnrolledDevice> | null;
  organizations: Organizations[] | null; // ACTIVATION REQUISE 
  userMetadata: { [key: string]: string } | null; // ACTIVATION REQUISE
  appMetadata: { [key: string]: string } | null; // ACTIVATION REQUISE
}
```

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * ACTIVATION EXPLICITE : vous devez configurer explicitement ces propriétés pour qu’elles soient accessibles dans l’écran. Pour en savoir plus, consultez [Configurer ACUL](/fr-CA/docs/customize/login-pages/advanced-customizations/configure).

  * Lorsque le serveur Auth0 n’a pas connaissance de l’utilisateur actuel, les propriétés `user` ne sont pas accessibles dans les écrans suivants :

  | Écrans                            | Écrans                              | Écrans                                     | Écrans                                              |
  | --------------------------------- | ----------------------------------- | ------------------------------------------ | --------------------------------------------------- |
  | Écrans de connexion               | Écrans MFA OTP                      | Écrans d’acceptation d’invitation          | Écrans InterstitialCaptcha                          |
  | Écrans d’inscription              | Écrans MFA par téléphone            | Écrans Device Code                         | Écrans de réinitialisation du mot de passe          |
  | Écrans de clés d’accès            | Écrans MFA Push                     | Écrans de vérification du courriel         | Écrans de codes de pays MFA                         |
  | Écrans de vérification            | Écrans de codes de récupération MFA | Écrans de déconnexion annulée              | Écrans MFA de détection des capacités du navigateur |
  | Écrans d’identifiant de courriel  | Écrans MFA WebAuthn                 | Écrans de déconnexion terminée             | Écrans d’inscription MFA                            |
  | Écrans d’identifiant de téléphone | Écrans d’acceptation d’invitation   | Écrans de protection contre la force brute |                                                     |
</Callout>
