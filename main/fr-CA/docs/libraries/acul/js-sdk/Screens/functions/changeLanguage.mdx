> **changeLanguage**(`options`: [`LanguageChangeOptions`](/fr-CA/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)): `Promise`&lt;`void`&gt;

Soumet la nouvelle préférence de langue au serveur à l’aide de l’action `change-language`, puis réaffiche l’interface dans la langue sélectionnée.

<div id="parameters">
  ## Paramètres
</div>

<ParamField body="options" type="LanguageChangeOptions" required>
  La configuration du changement de langue. Consultez [`LanguageChangeOptions`](/fr-CA/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).

  <Expandable title="options">
    <ParamField body="language" type="string" required>
      Un code de langue pour la langue cible (par ex. `'en'`, `'fr'`, `'es'`). Il doit correspondre à l’une des langues activées dans votre locataire Auth0.
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      La portée de persistance de la préférence de langue. Actuellement, seule la valeur `"session"` est prise en charge.
    </ParamField>
  </Expandable>
</ParamField>

<div id="returns">
  ## Renvoie
</div>

`Promise`&lt;`void`&gt;

La promesse est résolue lorsque la préférence de langue a été soumise et que l’écran est réaffiché dans la langue sélectionnée.
