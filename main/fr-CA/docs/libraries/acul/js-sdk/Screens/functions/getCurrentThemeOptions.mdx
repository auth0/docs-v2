> **getCurrentThemeOptions**(): [`FlattenedTheme`](/fr-CA/docs/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

**`Fonctionnalité commune`**

Récupère les options du thème actuel avec une configuration aplatie issues du contexte d’image de marque

<div id="returns">
  ## Retourne
</div>

[`FlattenedTheme`](/fr-CA/docs/libraries/acul/js-sdk/Screens/interfaces/FlattenedTheme) | `null`

Objet FlattenedTheme contenant les couleurs, les polices, les bordures, pageBackground et la configuration des widgets, ou `null` si aucune image de marque n’est disponible
