Le SDK Auth0 ACUL React vous permet de créer des expériences Universal Login personnalisées avec un minimum d’effort. Que vous travailliez sur la connexion, l’inscription, Passwordless ou l’inscription de clés d’accès, ce SDK est conçu pour prendre en charge les flux d’authentification modernes en toute simplicité.

<div id="why-choose-the-auth0-acul-react-sdk">
  ### Pourquoi choisir le SDK Auth0 ACUL React ?
</div>

* **Intégration rapide :** Intégrez des écrans prêts à l’emploi comme `login-id` ou `signup-password`.
* **Fonctionnalités modernes :** Prenez en charge Passwordless, les passkeys et les connexions sociales.
* **Modules flexibles :** Importez uniquement ce dont vous avez besoin.
* **Gestion unifiée de l’état :** Utilisez un objet de transaction unique pour gérer tous les flux d’authentification.

<div id="architecture">
  ### Architecture
</div>

<div id="how-it-works">
  #### Comment cela fonctionne-t-il ?
</div>

Le SDK est conçu dans une optique de modularité et d’évolutivité. Voici comment ses composants s’articulent :

* **Écrans :** Chaque flux d’authentification, comme `login-id` et `signup-password`, constitue un module autonome.
* **Gestionnaires de soumission :** Gèrent les soumissions de formulaires et les transitions.
* **Formulaires dynamiques :** Adaptent les champs et les validations selon la configuration de l’écran.
* **Propriétés utilitaires :** Permettent d’accéder de façon cohérente aux données communes, comme l’image de marque ou la transaction.
