<div id="methods">
  ## Méthodes
</div>

* [useAuth0Themes](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes): Hook React permettant d’obtenir les options de thème actuelles avec une configuration aplatie issue du contexte d’image de marque.
* [useChangeLanguage](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage): Hook React qui renvoie une fonction permettant de changer la langue d’affichage sur l’écran ACUL actuel.
* [useCurrentScreen](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen): Hook React qui récupère le contexte et l’état de l’écran actuel.
* [useErrors](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors): Hook React permettant de lire et de gérer les erreurs dans ACUL (Advanced Customization of Universal Login). Inclut toutes les erreurs de validation et les erreurs côté serveur. Il regroupe les erreurs en trois catégories : - `server` — erreurs renvoyées par Auth0 ou votre propre backend. - `client` — erreurs provenant de la validation côté client (par ex., saisie de formulaire invalide). - `developer` — erreurs causées par une intégration incorrecte ou une mauvaise utilisation du SDK.
* [useLoginIdentifiers](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useLoginIdentifiers): Renvoie une liste des types d’identifiant actifs (comme `'email'`, `'phone'` ou `'username'`) actuellement utilisés dans le flux d’authentification ou le processus de connexion.
* [useMfaPolling](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling): Hook React permettant de gérer l’attente liée au push MFA (par ex., attendre l’approbation d’une notification poussée) sur un écran Auth0 Advanced Customization of Universal Login (ACUL).
* [useResend](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useResend): Ce hook React gère les actions de « renvoi » (par ex., renvoyer un code de vérification) sur les écrans ACUL.
* [useSignupIdentifiers](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useSignupIdentifiers): Renvoie une liste des identifiants activés (courriel, téléphone ou nom d’utilisateur), chacun avec son statut `required`, selon les identifiants d’inscription de l’écran actuel.
* [usePasswordValidation](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation): Ce hook React valide un mot de passe par rapport à la politique actuelle des mots de passe d’Auth0 et renvoie un résultat structuré indiquant si le mot de passe respecte chaque règle.
* [useUsernameValidation](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useUsernameValidation): Hook React permettant de valider un nom d’utilisateur par rapport à la politique actuelle des noms d’utilisateur d’Auth0.
