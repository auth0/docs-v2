---

L’écran `phone-identifier-enrollment` s’affiche lorsqu’un utilisateur doit inscrire un numéro de téléphone comme identifiant. Il permet à l’utilisateur de choisir sa méthode de vérification préférée (SMS ou appel vocal), puis de poursuivre le processus d’inscription ou de revenir à l’étape précédente.

<div id="import">
  ## Import
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**import partiel** et l’**import racine** pour chaque écran.

* Un import partiel vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* Un import racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous voulez qu’un build unifié prenne en charge tous les écrans possibles.

```jsx Import Example
// import racine
import { usePhoneIdentifierEnrollment } from '@auth0/auth0-acul-react';

// import partiel
import {
  usePhoneIdentifierEnrollment,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continuePhoneEnrollment,
  returnToPrevious,
} from '@auth0/auth0-acul-react/phone-identifier-enrollment';

function PhoneIdentifierEnrollmentScreen() {
  const { continuePhoneEnrollment } = usePhoneIdentifierEnrollment();
  return (
    <button onClick={() => continuePhoneEnrollment({ type: 'text' })}>Send Code via SMS</button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte d’Auth0 sur l’écran `phone-identifier-enrollment`. Importez-les depuis `@auth0/auth0-acul-react/phone-identifier-enrollment`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les paramètres d’image de marque, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `phone-identifier-enrollment`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/phone-identifier-enrollment';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit les paramètres liés à l’application, comme `id`, `name` et `logoUrl`, pour l’écran `phone-identifier-enrollment`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/phone-identifier-enrollment';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des informations sur l’Organisation de l’utilisateur si l’inscription du téléphone est limitée à une Organisation. Retourne `null` lorsqu’aucun contexte d’Organisation n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/phone-identifier-enrollment';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur l’invite actuelle dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/phone-identifier-enrollment';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnPhoneIdentifierEnrollment">ScreenMembersOnPhoneIdentifierEnrollment</a></span>}>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>

<ParamField body="useUser" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>

</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continuePhoneEnrollment" type="Promise<void>">
  Cette méthode envoie la méthode de vérification par téléphone privilégiée par l’utilisateur et poursuit le flux d’inscription.

  ```jsx Example
  import { usePhoneIdentifierEnrollment } from '@auth0/auth0-acul-react/phone-identifier-enrollment';

  function EnrollButton() {
    const { continuePhoneEnrollment } = usePhoneIdentifierEnrollment();
    return (
      <button onClick={() => continuePhoneEnrollment({ type: 'text' })}>
        Envoyer le code par SMS
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [PhoneEnrollmentOptions](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PhoneEnrollmentOptions).
    </ParamField>

    <ParamField body="type" type="&#x22;text&#x22; | &#x22;voice&#x22;" required>
      La méthode d’envoi du code de vérification. Utilisez &quot;text&quot; pour les SMS ou &quot;voice&quot; pour un appel téléphonique.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="returnToPrevious" type="Promise<void>">
  Cette méthode renvoie l’utilisateur à l’étape précédente du flux d’authentification.

  ```jsx Example
  import { usePhoneIdentifierEnrollment } from '@auth0/auth0-acul-react/phone-identifier-enrollment';

  function BackButton() {
    const { returnToPrevious } = usePhoneIdentifierEnrollment();
    return (
      <button onClick={() => returnToPrevious()}>
        Retour
      </button>
    );
  }
  ```
</ParamField>
