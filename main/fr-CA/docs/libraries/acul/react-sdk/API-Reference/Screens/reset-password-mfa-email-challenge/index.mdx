---

L’écran `reset-password-mfa-email-challenge` s’affiche pendant le processus de réinitialisation du mot de passe lorsqu’une MFA par courriel est requise. Il invite l’utilisateur à saisir un code de vérification envoyé à son adresse courriel pour poursuivre la réinitialisation.

<div id="import">
  ## Import
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge **l’import partiel** et **l’import racine** pour chaque écran.

* L’import partiel vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’import racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez qu’une build unifiée prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useResetPasswordMfaEmailChallenge } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useResetPasswordMfaEmailChallenge,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useResend,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
  resendCode,
  tryAnotherMethod,
} from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';

function ResetPasswordMfaEmailChallengeScreen() {
  const { continueMethod } = useResetPasswordMfaEmailChallenge();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks associés à l’écran qui fournissent un accès en lecture seule aux données de contexte d’Auth0 sur l’écran `reset-password-mfa-email-challenge`. Importez-les depuis `@auth0/auth0-acul-react/reset-password-mfa-email-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les configurations d’image de marque, comme le logo, les couleurs et les paramètres du thème affichés sur l’écran `reset-password-mfa-email-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit les configurations liées à l’application, comme `id`, `name` et `logoUrl`, pour l’écran `reset-password-mfa-email-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des informations sur l’Organisation de l’utilisateur si la réinitialisation du mot de passe est limitée à une Organisation. Il renvoie `null` lorsqu’aucun contexte d’Organisation n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur l’invite actuelle dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnResetPasswordMfaEmailChallenge">ScreenMembersOnResetPasswordMfaEmailChallenge</a></span>}>
  Ce hook contient des détails propres à l’écran `reset-password-mfa-email-challenge`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au locataire, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `reset-password-mfa-email-challenge`, comme les connexions actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook fournit les données non fiables transmises à l’écran, comme un courriel ou un nom d’utilisateur prérempli à partir de paramètres d’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des renseignements sur l’utilisateur actif, notamment `username`, `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useResetPasswordMfaEmailChallenge" type={<a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordMfaEmailChallengeMembers">ResetPasswordMfaEmailChallengeMembers</a>}>
  Ce hook renvoie toutes les méthodes et tout le contexte disponibles pour l’écran `reset-password-mfa-email-challenge`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode envoie le code de vérification du courriel pour poursuivre l’étape MFA du flux de réinitialisation du mot de passe.

  ```jsx Example
  import { useResetPasswordMfaEmailChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';

  function VerifyCodeForm() {
    const { continueMethod } = useResetPasswordMfaEmailChallenge();
    return (
      <button onClick={() => continueMethod({ code: '123456' })}>
        Verify Code
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ContinueOptions](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code de vérification saisi par l’utilisateur.
    </ParamField>

    <ParamField body="rememberDevice" type="boolean">
      Indique s’il faut mémoriser l’appareil pour les prochaines authentifications.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Cette méthode renvoie à l’utilisateur le code de vérification du courriel.

  ```jsx Example
  import { useResetPasswordMfaEmailChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';

  function ResendButton() {
    const { resendCode } = useResetPasswordMfaEmailChallenge();
    return (
      <button onClick={() => resendCode()}>
        Resend Code
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode redirige l’utilisateur pour qu’il sélectionne une autre méthode MFA.

  ```jsx Example
  import { useResetPasswordMfaEmailChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-email-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useResetPasswordMfaEmailChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>

<ParamField body={<a href="/docs/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
