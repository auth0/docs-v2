L’écran `email-otp-challenge` s’affiche lorsqu’un utilisateur doit vérifier son identité en saisissant un code à usage unique envoyé à son adresse courriel. Il permet à l’utilisateur de saisir le code ou d’en demander un nouveau.

<div id="import">
  ## Importation
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge **l’importation partielle** et **l’importation racine** pour chaque écran.

* L’importation partielle vous permet d’inclure uniquement le code nécessaire à votre cas d’utilisation.
* L’importation racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez qu’un build unifié prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useEmailOTPChallenge } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useEmailOTPChallenge,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useResend,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  submitCode,
  resendCode,
} from '@auth0/auth0-acul-react/email-otp-challenge';

function EmailOtpChallengeScreen() {
  const { submitCode } = useEmailOTPChallenge();
  return (
    <button onClick={() => submitCode({ code: '123456' })}>Submit</button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `email-otp-challenge`. Importez-les depuis `@auth0/auth0-acul-react/email-otp-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les paramètres d’image de marque, comme le logo, les couleurs et les réglages de thème affichés sur l’écran `email-otp-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/email-otp-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit les configurations liées à l’application, comme `id`, `name` et `logoUrl`, pour l’écran `email-otp-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/email-otp-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’organisation de l’utilisateur si le défi OTP s’applique à une organisation. Renvoie `null` lorsqu’aucun contexte d’organisation n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/email-otp-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur l’invite en cours dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/email-otp-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnEmailOTPChallenge">ScreenMembersOnEmailOTPChallenge</a></span>}>
  Ce hook contient des détails propres à l’écran `email-otp-challenge`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/email-otp-challenge';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données sur le locataire, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/email-otp-challenge';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `email-otp-challenge`, comme les connexions actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/email-otp-challenge';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook traite les données non fiables transmises à l’écran, comme les paramètres préremplis provenant de la chaîne de requête de l’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/email-otp-challenge';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit les détails sur l’utilisateur actif, notamment `username`, `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/email-otp-challenge';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useEmailOTPChallenge" type={<a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/EmailOTPChallengeMembers">EmailOTPChallengeMembers</a>}>
  Ce hook renvoie toutes les méthodes et le contexte disponibles pour l’écran `email-otp-challenge`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="resendCode" type="Promise<void>">
  Cette méthode renvoie le code à usage unique à l’adresse courriel de l’utilisateur.

  ```jsx Example
  import { useEmailOTPChallenge } from '@auth0/auth0-acul-react/email-otp-challenge';

  function ResendButton() {
    const { resendCode } = useEmailOTPChallenge();
    return (
      <button onClick={() => resendCode()}>
        Resend Code
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="submitCode" type="Promise<void>">
  Cette méthode transmet le code à usage unique saisi par l’utilisateur.

  ```jsx Example
  import { useEmailOTPChallenge } from '@auth0/auth0-acul-react/email-otp-challenge';

  function SubmitButton() {
    const { submitCode } = useEmailOTPChallenge();
    return (
      <button onClick={() => submitCode({ code: '123456' })}>
        Continue
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [OtpCodeOptions](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OtpCodeOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code à usage unique saisi par l’utilisateur.
    </ParamField>
  </Expandable>
</ParamField>

<div id="commonutility-hooks">
  ## Hooks communs/utilitaires
</div>

<ParamField body={<a href="/docs/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes">useAuth0Themes</a>} type="Hooks">
  Ce hook permet d’obtenir les options du thème actuel avec une configuration aplatie à partir du contexte d’image de marque.

<ParamField body={<a href="/docs/libraries/acul/react-sdk/API-Reference/Hooks/useCurrentScreen">useCurrentScreen</a>} type="Hooks">
  Ce hook permet d’obtenir le contexte et l’état de l’écran actuel.
</ParamField>

<ParamField body={<a href="/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors">useErrors</a>} type="Hooks">
  Ce hook lit et gère les erreurs de serveur, d’application et de développeur à l’écran.
</ParamField>

<ParamField body={<a href="/docs/libraries/acul/react-sdk/API-Reference/Hooks/useResend">useResend</a>} type="Hooks">
  Ce hook gère l’état de réenvoi et la logique de temporisation pour le code OTP par courriel.
</ParamField>
