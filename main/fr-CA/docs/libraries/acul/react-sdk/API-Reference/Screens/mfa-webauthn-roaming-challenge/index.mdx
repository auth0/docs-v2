L’écran `mfa-webauthn-roaming-challenge` demande à l’utilisateur de répondre à un défi d’AMF à l’aide d’une clé de sécurité itinérante, comme une YubiKey ou un autre authentificateur matériel FIDO2.

<div id="import">
  ## Importation
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge **l’importation partielle** et **l’importation depuis la racine** pour chaque écran.

* Avec l’importation partielle, vous pouvez inclure uniquement le code nécessaire à votre cas d’usage.
* Avec l’importation depuis la racine, vous pouvez charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez qu’une compilation unifiée prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useMfaWebAuthnRoamingChallenge,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  reportWebAuthnError,
  tryAnotherMethod,
  verify,
} from "@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge";

function MfaWebAuthnRoamingChallengeScreen() {
  const { verify } = useMfaWebAuthnRoamingChallenge();
  return (
    <button onClick={() => verify({ rememberDevice: true })}>
      Verify with Security Key
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Des hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 sur l’écran `mfa-webauthn-roaming-challenge`. Importez-les depuis `@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les paramètres de personnalisation de la marque, comme le logo, les couleurs et les paramètres de thème affichés sur l’écran `mfa-webauthn-roaming-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit des paramètres liés au client, comme `id`, `name` et `logoUrl`, pour l’écran `mfa-webauthn-roaming-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’organisation de l’utilisateur si le flux MFA est limité à une organisation. Retourne `null` lorsqu’aucun contexte d’organisation n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>Aucun contexte d’organisation</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur l’invite actuelle dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnMfaWebAuthnRoamingChallenge">ScreenMembersOnMfaWebAuthnRoamingChallenge</a></span>}>
  Ce hook contient des détails propres à l’écran `mfa-webauthn-roaming-challenge`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au locataire, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `mfa-webauthn-roaming-challenge`, comme l’état actuel du flux MFA.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>


<ParamField body="useUser" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>


</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="reportWebAuthnError" type="Promise<void>">
  Cette méthode signale une erreur WebAuthn survenue lors du test avec une clé de sécurité itinérante.

  ```jsx Example
  import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';

  function HandleWebAuthnError({ error }) {
    const { reportWebAuthnError } = useMfaWebAuthnRoamingChallenge();
    return (
      <button onClick={() => reportWebAuthnError({ error })}>
        Signaler l’erreur
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [MfaWebAuthnRoamingChallengeReportErrorOptions](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaWebAuthnRoamingChallengeReportErrorOptions).
    </ParamField>

    <ParamField body="error" type={<span><a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/WebAuthnErrorDetails">WebAuthnErrorDetails</a></span>} required>
      L’objet d’erreur de l’API WebAuthn (`navigator.credentials.get()`) à signaler.

      <ParamField body="name" type="string" required>
        Le nom de l’erreur (p. ex., `'NotAllowedError'`).
      </ParamField>

      <ParamField body="message" type="string" required>
        Le message d’erreur décrivant le problème.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode redirige vers l’écran de sélection de la méthode MFA pour permettre à l’utilisateur de choisir un autre facteur d’authentification.

  ```jsx Example
  import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useMfaWebAuthnRoamingChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Essayer une autre méthode
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="verify" type="Promise<void>">
  Cette méthode lance la vérification de la clé de sécurité itinérante afin de terminer le défi d’AMF.

  ```jsx Example
  import { useMfaWebAuthnRoamingChallenge } from '@auth0/auth0-acul-react/mfa-webauthn-roaming-challenge';

  function VerifyButton() {
    const { verify } = useMfaWebAuthnRoamingChallenge();
    return (
      <button onClick={() => verify({ rememberDevice: true })}>
        Vérifier avec la clé de sécurité
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [VerifySecurityKeyOptions](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/VerifySecurityKeyOptions).
    </ParamField>

    <ParamField body="rememberDevice" type="boolean">
      Facultatif. Si la valeur est true, le navigateur sera mémorisé pour les futurs défis d’AMF.
      Correspond au champ de formulaire `rememberBrowser`.
    </ParamField>
  </Expandable>
</ParamField>
