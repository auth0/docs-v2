<div id="import">
  ## Import
</div>

Chaque écran a son propre ensemble de hooks et de méthodes. Le SDK prend en charge les **imports partiels** et les **imports à la racine** pour chaque écran.

* Un import partiel vous permet d’inclure uniquement le code nécessaire à votre cas d’usage.
* Un import à la racine vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez qu’une build unifiée prenne en charge tous les écrans possibles.

```jsx Import Example
// importation racine
import { useLogout } from '@auth0/auth0-acul-react';

// importation partielle
import {
  useLogout,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  confirmLogout,
} from '@auth0/auth0-acul-react/logout';

function LogoutScreen() {
  const { confirmLogout } = useLogout();
  return (
    <button onClick={() => confirmLogout({ action: 'accept' })}>
      Sign Out
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks associés à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 dans l’écran `logout`. Importez-les depuis `@auth0/auth0-acul-react/logout`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit la configuration d’image de marque, comme le logo, les couleurs et les paramètres de thème affichés dans l’écran `logout`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/logout';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit la configuration liée à l’application, comme `id`, `name` et `logoUrl`, pour l’écran `logout`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/logout';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’organisation de l’utilisateur si la déconnexion est limitée à une organisation. Il retourne `null` lorsqu’aucun contexte d’organisation n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/logout';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur l’invite actuelle dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/logout';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  Ce hook contient des renseignements propres à l’écran `logout`, y compris sa configuration et son contexte.

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/logout';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  Ce hook contient des données liées au locataire, comme `id` et les métadonnées associées.

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/logout';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  Ce hook fournit des données propres à la transaction pour l’écran `logout`, comme les connexions actives et l’état actuel du flux.

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/logout';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  Ce hook gère les données non fiables transmises à l’écran, comme les paramètres provenant de la chaîne de requête de l’URL.

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/logout';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  Ce hook fournit des renseignements sur l’utilisateur actif, notamment `username`, `email` et les méthodes d’authentification disponibles.

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/logout';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useLogout" type={<a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/LogoutMembers">LogoutMembers</a>}>
  Ce hook renvoie toutes les méthodes et tout le contexte disponibles pour l’écran `logout`.
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="confirmLogout" type="Promise<void>">
  Cette méthode confirme ou annule la déconnexion de la session en cours.

  ```jsx Example
  import { useLogout } from '@auth0/auth0-acul-react/logout';

  function LogoutScreen() {
    const { confirmLogout } = useLogout();
    return (
      <>
        <button onClick={() => confirmLogout({ action: 'accept' })}>
          Sign Out
        </button>
        <button onClick={() => confirmLogout({ action: 'deny' })}>
          Stay Signed In
        </button>
      </>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ConfirmLogoutOptions](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ConfirmLogoutOptions).
    </ParamField>

    <ParamField body="action" type="&#x22;accept&#x22; | &#x22;deny&#x22;" required>
      Indique s’il faut confirmer (`"accept"`) ou annuler (`"deny"`) la déconnexion.
    </ParamField>
  </Expandable>
</ParamField>
