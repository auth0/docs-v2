L’écran `reset-password-mfa-voice-challenge` s’affiche pendant le flux de réinitialisation du mot de passe lorsqu’une authentification multifacteur par appel vocal est requise. Il invite l’utilisateur à saisir le code de vérification envoyé à son téléphone pour poursuivre le processus de réinitialisation.

<div id="import">
  ## Import
</div>

Chaque écran possède son propre ensemble de hooks et de méthodes. Le SDK prend en charge l’**import partiel** et l’**import racine** pour chaque écran.

* L’**import partiel** vous permet d’inclure uniquement le code nécessaire à votre cas d’usage.
* L’**import racine** vous permet de charger tous les écrans à partir d’un seul bundle, ce qui est utile si vous souhaitez utiliser une version unifiée pour prendre en charge tous les écrans possibles.

```jsx Import Example
// import racine
import { useResetPasswordMfaVoiceChallenge } from '@auth0/auth0-acul-react';

// import partiel
import {
  useResetPasswordMfaVoiceChallenge,
  // Hooks de contexte
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Hooks communs
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  useResend,
  // Hooks utilitaires
  useChangeLanguage,
  // Méthodes
  continueMethod,
  resendCode,
  switchToSms,
  tryAnotherMethod,
} from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';

function ResetPasswordMfaVoiceChallengeScreen() {
  const { continueMethod } = useResetPasswordMfaVoiceChallenge();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

<div id="context-hooks">
  ## Hooks de contexte
</div>

Hooks propres à l’écran qui fournissent un accès en lecture seule aux données de contexte Auth0 dans l’écran `reset-password-mfa-voice-challenge`. Importez-les depuis `@auth0/auth0-acul-react/reset-password-mfa-voice-challenge`.

<ParamField body="useBranding" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  Ce hook fournit les paramètres d’image de marque, comme le logo, les couleurs et les paramètres de thème affichés dans l’écran `reset-password-mfa-voice-challenge`.

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  Ce hook fournit les paramètres liés à l’application, comme `id`, `name` et `logoUrl`, pour l’écran `reset-password-mfa-voice-challenge`.

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  Ce hook fournit des renseignements sur l’Organisation de l’utilisateur si la réinitialisation du mot de passe est associée à une Organisation. Renvoie `null` lorsqu’aucun contexte d’Organisation n’est présent.

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No Organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  Ce hook contient des données sur l’invite active dans le flux d’authentification.

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<div id="methods">
  ## Méthodes
</div>

<ParamField body="continueMethod" type="Promise<void>">
  Cette méthode envoie le code de vérification vocal pour poursuivre l’étape MFA du flux de réinitialisation du mot de passe.

  ```jsx Example
  import { useResetPasswordMfaVoiceChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';

  function VerifyCodeForm() {
    const { continueMethod } = useResetPasswordMfaVoiceChallenge();
    return (
      <button onClick={() => continueMethod({ code: '123456' })}>
        Verify Code
      </button>
    );
  }
  ```

  **Paramètres de la méthode**

  <Expandable title="Paramètres">
    <ParamField body="options">
      [ResetPasswordMfaVoiceChallengeContinueOptions](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordMfaVoiceChallengeContinueOptions).
    </ParamField>

    <ParamField body="code" type="string" required>
      Le code de vérification vocal saisi par l’utilisateur.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="resendCode" type="Promise<void>">
  Cette méthode renvoie le code de vérification vocal au numéro de téléphone de l’utilisateur.

  ```jsx Example
  import { useResetPasswordMfaVoiceChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';

  function ResendCodeButton() {
    const { resendCode } = useResetPasswordMfaVoiceChallenge();
    return (
      <button onClick={() => resendCode()}>
        Resend Code
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="switchToSms" type="Promise<void>">
  Cette méthode fait passer la méthode d’envoi MFA de l’appel vocal au SMS.

  ```jsx Example
  import { useResetPasswordMfaVoiceChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';

  function SwitchToSmsButton() {
    const { switchToSms } = useResetPasswordMfaVoiceChallenge();
    return (
      <button onClick={() => switchToSms()}>
        Switch to SMS
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  Cette méthode dirige l’utilisateur vers l’écran de sélection d’une autre méthode MFA.

  ```jsx Example
  import { useResetPasswordMfaVoiceChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-voice-challenge';

  function TryAnotherMethodButton() {
    const { tryAnotherMethod } = useResetPasswordMfaVoiceChallenge();
    return (
      <button onClick={() => tryAnotherMethod()}>
        Try Another Method
      </button>
    );
  }
  ```
</ParamField>
