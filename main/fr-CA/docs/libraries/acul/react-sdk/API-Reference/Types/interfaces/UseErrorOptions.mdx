---
title: "UseErrorOptions"
---

Objet d’options du hook [`useError`](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useError). Contrôle le comportement d’affichage des erreurs dans les écrans de Universal Login d’Auth0.

```ts Example
export interface UseErrorOptions {
  includeDevErrors?: boolean;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="includeDevErrors" type="boolean">
  Si la valeur est `true`, inclut dans la sortie d’erreur des détails propres au développement. Utile pour le débogage en cours de développement; doit être désactivé en production.
</ParamField>
