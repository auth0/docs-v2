---
title: "CurrentScreenOptions"
---

Objet d’options décrivant le contexte de l’écran actuel dans un flux de Universal Login d’Auth0. Donne accès aux métadonnées de l’application, de l’organisation, du locataire, de la transaction et de l’écran disponibles dans l’environnement d’exécution.

```ts Example
export interface CurrentScreenOptions {
  client: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  organization: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  prompt: {
    name: string;
  } | null;
  screen: {
    name: string;
  } | null;
  tenant: {
    enabledLocales: string[];
  } | null;
  transaction: {
    errors: TransactionError[] | null;
    state: string;
    locale: string;
  } | null;
  untrustedData: {
    authorizationParams: Record<string, string> | null;
  } | null;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="client" type="{ id: string; metadata: Record<string, string> | null } | null">
  Détails de l’application pour la demande d’autorisation en cours, y compris l’identifiant de l’application et toutes les métadonnées associées.
</ParamField>

<ParamField body="organization" type="{ id: string; metadata: Record<string, string> | null } | null">
  Contexte de l’organisation pour la demande en cours, y compris l’identifiant de l’organisation et les métadonnées.
</ParamField>

<ParamField body="prompt" type="{ name: string } | null">
  Configuration de l’invite actuelle, qui indique quelle étape de l’invite est affichée.
</ParamField>

<ParamField body="screen" type="{ name: string } | null">
  Configuration de l’écran actuel, qui indique quel écran de l’invite est actif.
</ParamField>

<ParamField body="tenant" type="{ enabledLocales: string[] } | null">
  Paramètres au niveau du locataire, y compris la liste des langues activées pour ce déploiement d’Universal Login.
</ParamField>

<ParamField body="transaction" type="{ errors: TransactionError[] | null; state: string; locale: string } | null">
  État de la transaction pour le flux d’authentification en cours, y compris les erreurs de validation, la valeur state et la langue active.
</ParamField>

<ParamField body="untrustedData" type="{ authorizationParams: Record<string, string> | null } | null">
  Données non fiables transmises au flux, comme les paramètres d’autorisation, qui ne doivent pas servir à prendre des décisions de sécurité.
</ParamField>
