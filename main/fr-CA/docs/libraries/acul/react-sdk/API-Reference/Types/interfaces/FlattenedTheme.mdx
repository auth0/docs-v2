---
title: "FlattenedTheme"
---

Objet de thème structuré renvoyé par [`useAuth0Themes`](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useAuth0Themes). Fournit un accès à plat, au niveau des jetons, à l’ensemble des styles visuels appliqués aux composants d’Universal Login d’Auth0.

```ts Example
export interface FlattenedTheme {
  colors: Record<string, string>;
  fonts: Record<string, string | number | boolean | object>;
  borders: Record<string, string | number | boolean>;
  pageBackground: Record<string, string>;
  widget: Record<string, string | number>;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="colors" type="Record<string, string>" required>
  Jetons de couleur appliqués à l’ensemble de l’interface de connexion — inclut les couleurs du texte, de l’arrière-plan, des bordures et des états interactifs.
</ParamField>

<ParamField body="fonts" type="Record<string, string | number | boolean | object>" required>
  Jetons typographiques qui définissent la famille de polices, la taille, le poids, la hauteur de ligne et les styles de texte connexes.
</ParamField>

<ParamField body="borders" type="Record<string, string | number | boolean>" required>
  Jetons de bordure qui définissent le rayon, la largeur et le style des éléments de l’interface utilisateur.
</ParamField>

<ParamField body="pageBackground" type="Record<string, string>" required>
  Jetons de style d’arrière-plan appliqués au conteneur de page qui entoure le widget de connexion.
</ParamField>

<ParamField body="widget" type="Record<string, string | number>" required>
  Jetons de style propres à la carte du widget de connexion, comme la marge intérieure, l’ombre et les dimensions.
</ParamField>
