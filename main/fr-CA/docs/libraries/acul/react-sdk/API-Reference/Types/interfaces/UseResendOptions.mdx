title: "UseResendOptions"
---

Paramètres facultatifs pour [useResend](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

```ts Example
export interface UseResendOptions {
  timeoutSeconds?: number;
  onTimeout?: () => void;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="timeoutSeconds" type="number" default="10">
  Durée du compte à rebours (en secondes) avant qu’un nouveau renvoi soit autorisé.
  La valeur par défaut est `10`.
</ParamField>

<ParamField body="onTimeout" type="() => void">
  Fonction appelée lorsque le compte à rebours prend fin et que l’action de renvoi est de nouveau disponible.
</ParamField>
