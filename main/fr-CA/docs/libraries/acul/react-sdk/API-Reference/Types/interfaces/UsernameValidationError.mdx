---
title: "UsernameValidationError"
---

Représente une erreur de validation de nom d’utilisateur renvoyée lors de la validation.

Chaque erreur comprend un `code` lisible par machine et un `message` lisible par l’humain.

```ts Example
export interface UsernameValidationError {
  code: string;
  message: string;
  field: string;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="code" type="string" required>
  Un identifiant unique de la règle de validation ayant échoué.

  Exemple : `"username-too-short"`, `"username-invalid-characters"`
</ParamField>

<ParamField body="message" type="string" required>
  Une description de l’erreur lisible par l’humain.

  Exemple : `"Username must be at least 3 characters long."`
</ParamField>

<ParamField body="field" type="string" required>
  Le champ associé à l’erreur de validation, généralement « username », « email » ou « phone ».
</ParamField>
