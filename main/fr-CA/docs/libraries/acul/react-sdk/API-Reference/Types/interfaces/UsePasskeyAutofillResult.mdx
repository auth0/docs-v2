Type de retour de [`usePasskeyAutofill`](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasskeyAutofill).

```typescript
interface UsePasskeyAutofillResult {
  inputRef: RefObject<HTMLInputElement>;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="inputRef" type="RefObject<HTMLInputElement>" required>
  Une référence React à associer au champ de saisie du nom d’utilisateur. Lorsqu’elle est associée, le SDK configure automatiquement l’attribut `autocomplete` du champ à `"username webauthn"`, ce qui active l’interface utilisateur conditionnelle du navigateur pour le remplissage automatique par clé d’accès. L’association de cette référence est facultative si `autocomplete="username webauthn"` est déjà déclaré dans votre balisage.
</ParamField>
