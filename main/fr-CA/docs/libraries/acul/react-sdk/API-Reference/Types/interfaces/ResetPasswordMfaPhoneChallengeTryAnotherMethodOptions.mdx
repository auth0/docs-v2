TryAnotherMethodOptions

```ts Example
export interface TryAnotherMethodOptions extends CustomOptions {
  /**
   * La méthode de livraison initialement présentée ou sélectionnée sur l'écran actuel.
   * Ce paramètre est requis par le point de terminaison de l'API pour traiter correctement l'action 'pick-authenticator'.
   * - `sms`: Indique que l'écran actuel correspondait à un défi par SMS.
   * - `voice`: Indique que l'écran actuel correspondait à un défi vocal.
   * @type {'sms' | 'voice'}
   */
  type: 'sms' | 'voice';
}
```
