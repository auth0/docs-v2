---
title: "LanguageChangeOptions"
---

Options acceptées par la fonction retournée par [`useChangeLanguage`](/fr-CA/docs/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage).

```typescript
interface LanguageChangeOptions {
  language: string;
  persist?: "session";
  [key: string]: string | number | boolean | undefined;
}
```

<div id="properties">
  ## Propriétés
</div>

<ParamField body="language" type="string" required>
  Le code de langue à utiliser. Il doit correspondre à l’une des langues activées dans votre locataire Auth0. Exemple : `"en"`, `"fr"`, `"es"`.
</ParamField>

<ParamField body="persist" type="&#x22;session&#x22;">
  La portée de persistance de la préférence de langue. Seule la valeur `"session"` est actuellement prise en charge. La valeur par défaut est `"session"`.
</ParamField>

<div id="remarks">
  ## Remarques
</div>

* Des champs personnalisés additionnels peuvent être transmis avec `language` et `persist`. Les champs personnalisés doivent avoir le préfixe `"ulp-"` et accepter des valeurs de type `string`, `number`, `boolean` ou `undefined`.
