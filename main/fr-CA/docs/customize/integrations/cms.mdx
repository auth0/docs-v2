Auth0 fournit des plugins/extensions pour système de gestion de contenu afin d’intégrer votre installation CMS à votre compte Auth0. Ces plugins permettent l’<Tooltip tip="Single Sign-On (SSO) : service qui, après qu’un utilisateur se connecte à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Single+Sign-on">authentification unique</Tooltip> (SSO) pour les entreprises, la connexion sociale et la connexion par nom d’utilisateur/mot de passe dans l’ensemble de vos instances et plateformes.

Les fonctionnalités de connexion sont mises en œuvre au moyen d’un nouveau widget de connexion (propulsé par Auth0) qui permet :

* le SSO avec des annuaires d’entreprise (LDAP, AD, Google Workspace, Office365 et le fournisseur <Tooltip tip="Security Assertion Markup Language (SAML) : protocole normalisé permettant à deux parties d’échanger des renseignements d’authentification sans mot de passe." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=SAML">SAML</Tooltip>)
* le partage du nom d’utilisateur et du mot de passe entre plusieurs comptes WordPress pour le SSO
* le SSO avec plus de 30 [fournisseurs sociaux](/fr-CA/docs/authenticate/identity-providers)
* le tableau de bord de gestion des utilisateurs
* l’authentification à deux facteurs facultative
* le SSO entre WordPress et d’autres applications
* les rapports et les analyses
