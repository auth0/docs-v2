title: Exporter des données utilisateur vers Constant Contact
---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 dans un fichier CSV, puis à les importer dans le tableau de bord de Constant Contact.

<div id="create-a-user-data-file">
  ## Créer un fichier de données d’utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez l’extension **User Import / Export**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez **Export Format** sur le format de fichier requis. Constant Contact accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **champ utilisateur** et un **nom de colonne** pour chaque attribut d’utilisateur à inclure dans l’exportation. Par exemple :

| Champ utilisateur | Nom de colonne   |
| ----------------- | ---------------- |
| `email`           | Adresse courriel |
| `created_at`      | Créé le          |
| `given_name`      | Prénom           |
| `family_name`     | Nom de famille   |

Une fois les champs utilisateur ajoutés, cliquez sur le bouton **Export Users** pour lancer l’exportation. Lorsque l’exportation est terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données d’utilisateurs
</div>

Pour importer votre fichier CSV dans Constant Contact, suivez ces étapes :

1. [Connectez-vous à Constant Contact](https://login.constantcontact.com) et accédez à **Contacts**.
2. À partir de la page **Contacts**, cliquez sur **Add Contacts &gt; Upload from file**.
3. Sur la page **Upload from file**, téléversez votre fichier CSV, puis cliquez sur **Continue**.
4. Vérifiez les colonnes et faites-les correspondre aux champs de Constant Contact. Une fois terminé, cliquez sur **Continue**.
5. Sur la page **Select lists**, choisissez la ou les listes auxquelles ajouter vos contacts.
6. Cliquez sur **Upload**.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Constant Contact.
