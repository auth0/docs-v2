title: Exporter les données des utilisateurs vers Oracle Eloqua
---

Dans cet article, vous apprendrez à exporter les données des utilisateurs d’Auth0 vers un fichier CSV, puis à les importer dans Eloqua à l’aide de l’assistant de téléversement des contacts.

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez l’extension **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez **Export Format** sur le format de fichier requis. Eloqua accepte l’importation de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut d’utilisateur à inclure dans l’exportation. Par exemple :

| Champ utilisateur | Nom de colonne   |
| ----------------- | ---------------- |
| `email`           | Adresse courriel |
| `created_at`      | Date de création |
| `given_name`      | Prénom           |
| `family_name`     | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Avant de commencer, vous pouvez obtenir plus de renseignements en consultant le [Oracle Eloqua Help Center: Uploading Contacts](https://docs.oracle.com/en/cloud/saas/marketing/eloqua-user/Help/Contacts/Tasks/UploadingContacts.htm).

Pour importer votre fichier CSV dans Eloqua, suivez les étapes suivantes :

1. Connectez-vous à Eloqua et accédez à **Audience &gt; Contacts &gt; Upload** pour ouvrir l&#39;**Assistant de téléversement de contacts**.
2. Configurez l&#39;importation à l&#39;aide des champs fournis dans l&#39;onglet **Pick Data Source**.
3. Cliquez sur le bouton **Cloud** et sélectionnez le fichier à téléverser.
4. Après avoir vérifié le contenu du fichier dans l&#39;onglet **Review**, cliquez sur **Next Step**.
5. Dans l&#39;onglet **Map Fields**, indiquez les paramètres de mappage des champs pour faire correspondre les données de votre fichier CSV aux champs de contact. Cliquez sur **Next Step** pour continuer.
6. Terminez la dernière étape de l&#39;assistant, puis cliquez sur **Finish**.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Eloqua.
