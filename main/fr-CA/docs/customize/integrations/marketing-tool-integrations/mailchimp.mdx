title: Exporter les données des utilisateurs vers MailChimp
---

Dans cet article, vous apprendrez à exporter les données des utilisateurs depuis Auth0 vers un fichier CSV, puis à les importer dans le [tableau de bord de MailChimp](https://login.mailchimp.com/).

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du tableau de bord, puis ouvrez l’extension **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez **Export Format** sur le format de fichier requis. MailChimp accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

Pour en savoir plus, consultez [la base de connaissances MailChimp : Consignes de format pour votre fichier d’importation](https://kb.mailchimp.com/lists/growth/format-guidelines-for-your-import-file).

En haut de la page, dans la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut d’utilisateur à inclure dans l’exportation. Pour MailChimp, un champ de courriel portant le nom de colonne `Email Address` est requis; assurez-vous donc de l’inclure. Par exemple :

| Champ utilisateur | Nom de colonne |
| ----------------- | -------------- |
| `email`           | Email Address  |
| `created_at`      | Créé le        |
| `given_name`      | Prénom         |
| `family_name`     | Nom de famille |

Après avoir ajouté les champs d’utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données d’utilisateurs
</div>

Avant de commencer, vous pouvez obtenir de plus amples renseignements en consultant [base de connaissances MailChimp: Import Subscribers to a List](https://kb.mailchimp.com/lists/growth/import-subscribers-to-a-list).

Connectez-vous à votre compte MailChimp et accédez à la page **Lists**. Sélectionnez une liste pour y importer vos utilisateurs d’Auth0 ([ou créez une nouvelle liste](https://kb.mailchimp.com/lists/growth/create-a-new-list)).

1. Sur la page de votre liste MailChimp, cliquez sur **Import Contacts** dans le menu **Add Contacts**.
2. Sélectionnez l’option `CSV or tab-delimited text file` pour importer des contacts.
3. Téléversez le fichier CSV de données d’utilisateurs que vous avez exporté d’Auth0 dans la section précédente. MailChimp interprétera vos données d’utilisateurs sur la page suivante.
4. Vérifiez que les noms de colonnes et les types de champs sont corrects.
5. Cliquez sur le bouton **Next**.
6. Passez en revue vos sélections et définissez la catégorie d’importation.
7. Cliquez sur le bouton **Import** pour démarrer l’importation des utilisateurs.

Et voilà ! Vous avez importé avec succès vos utilisateurs d’Auth0 dans MailChimp.
