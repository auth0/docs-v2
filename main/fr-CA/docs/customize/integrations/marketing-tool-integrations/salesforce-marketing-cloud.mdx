title: Exporter des données utilisateur vers Salesforce Marketing Cloud
---

Dans cet article, vous apprendrez à exporter les données utilisateur d’Auth0 dans un fichier CSV, puis à les importer dans Salesforce Marketing Cloud à l’aide d’Email Studio.

Pour en savoir plus, consultez l’article de Salesforce [Get Started Checklist for Email Studio](https://help.salesforce.com/s/articleView?id=mktg.mc_es_get_started_checklist_for_email_studio.htm\&type=5).

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez l’**User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, réglez **Export Format** sur le format de fichier requis. Salesforce Marketing Cloud accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut de la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données d’utilisateurs
</div>

Avant de commencer, assurez-vous que votre compte Salesforce dispose des autorisations nécessaires pour importer des enregistrements. Prenez également un moment pour consulter les étapes décrites dans l’[article Salesforce sur l’importation d’abonnés](https://help.salesforce.com/s/articleView?id=mktg.mc_es_import_subscribers_from_file.htm\&type=5), ainsi que [Quelles autorisations sont requises pour importer des enregistrements](https://help.salesforce.com/s/articleView?id=xcloud.faq_import_general_permissions.htm\&type=5).

Pour importer votre fichier CSV dans Salesforce Marketing Cloud, suivez ces étapes :

1. Connectez-vous à Salesforce Marketing Cloud et ouvrez **Email Studio**.
2. Accédez à **Subscribers &gt; Lists**.
3. Choisissez la liste dans laquelle effectuer l’importation, puis sélectionnez l’action **Import**.
4. Après l’introduction de l’assistant d’importation, sélectionnez votre fichier CSV comme **Upload Source** et choisissez CSV comme **Data Format**. Renseignez les autres paramètres, puis cliquez sur **Next**.
5. Dans la boîte de dialogue **Map Attributes**, faites correspondre les champs de données de votre fichier CSV aux champs appropriés. Une fois le mappage terminé, cliquez sur **Next**.
6. Après avoir vérifié vos correspondances dans la boîte de dialogue **Confirmed Mappings**, cliquez sur **Begin** pour lancer l’importation.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Salesforce Marketing Cloud.
