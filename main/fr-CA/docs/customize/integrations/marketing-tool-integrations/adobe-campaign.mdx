title: Exporter les données des utilisateurs vers Adobe Campaign
---

Dans cet article, vous apprendrez à exporter les données des utilisateurs depuis Auth0 vers un fichier CSV, puis à les importer dans Adobe Campaign à l’aide de l’[assistant d’importation d’Adobe Campaign](https://docs.adobe.com/content/help/en/campaign-classic/using/getting-started/importing-and-exporting-data/importing-data.html).

<div id="create-a-user-data-file">
  ## Créer un fichier de données utilisateur
</div>

Commencez par accéder à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard, puis ouvrez **User Import / Export Extension**. Sur la page de l’extension, sélectionnez **Export** dans le menu.

Ensuite, définissez **Export Format** au format de fichier requis. Adobe Campaign accepte l’importation de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut, dans la section **Fields**, indiquez un **User Field** et un **Column Name** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

| User Field    | Column Name      |
| ------------- | ---------------- |
| `email`       | Adresse courriel |
| `created_at`  | Date de création |
| `given_name`  | Prénom           |
| `family_name` | Nom de famille   |

Après avoir ajouté les champs utilisateur, cliquez sur **Export Users** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV pour l’utiliser dans la section suivante.

<div id="import-a-user-data-file">
  ## Importer un fichier de données utilisateur
</div>

Avant de commencer, vous trouverez plus d’informations dans [Adobe Campaign Documentation: Importing Data](https://docs.adobe.com/content/help/en/campaign-classic/using/getting-started/importing-and-exporting-data/importing-data.html).

Ouvrez une session dans le tableau de bord du client Adobe Campaign, puis accédez à **Profiles and Targets &gt; Jobs**. Créez une nouvelle tâche d’importation en cliquant sur le bouton **Create**, puis en sélectionnant **New Import**.

Une nouvelle fenêtre **Import Wizard** devrait s’ouvrir. À l’étape **Template Selection**, vous pouvez définir les paramètres d’importation.

Une fois les paramètres d’importation configurés, cliquez sur le bouton **Next** pour continuer.

À l’étape **File to Import**, téléversez le fichier CSV de données utilisateur que vous avez exporté depuis Auth0 dans la section précédente. Cliquez sur le bouton **Next** pour passer à **Field Mapping**.

Ensuite, faites correspondre le schéma du fichier d’exportation à celui de votre base de données Adobe Campaign. Vérifiez que les noms et les types de champs sont corrects, puis cliquez sur le bouton **Next**.

Terminez les autres étapes de configuration en définissant votre mode de réconciliation des données et en sélectionnant un dossier, une liste ou un service pour les utilisateurs importés.

Enfin, lancez l’importation en cliquant sur le bouton **Start** dans la fenêtre **Data Import Execution**.

Et voilà ! Vous avez importé avec succès vos utilisateurs Auth0 dans Adobe Campaign.
