L’intégration native d’Auth0 avec Vercel vous permet de connecter votre plateforme d’identité à votre projet Vercel avec un minimum de configuration. Configurez et personnalisez votre interface de connexion sans avoir à repartir de zéro. Vous pourrez ensuite étendre et faire évoluer vos flux d’authentification et d’autorisation grâce à des fonctionnalités avancées, notamment [Organisations B2B](/fr-CA/docs/manage-users/organizations), [Fine-Grained Authorization (FGA)](https://docs.fga.dev/), [Auth0 for AI Agents](https://auth0.com/ai/docs), et plus encore.

L’intégration native d’Auth0 avec Vercel offre ce qui suit :

* La création automatique d’une application Auth0 connectée à votre projet Vercel.
* Le préchargement de vos identifiants Auth0 dans le Vercel Dashboard pour une configuration simple et rapide.
* Une prise en charge prête à l’emploi pour vos applications Next.js à l’aide des [SDK Auth0](/fr-CA/docs/libraries).

L’intégration native d’Auth0 avec Vercel crée un nouvel environnement de locataire dans Auth0 pour votre projet Vercel. Si vous souhaitez utiliser un compte Auth0 existant, intégrez Vercel manuellement lors de la création d’une application dans l’Auth0 Dashboard.

<div id="prerequisites">
  ## Prérequis
</div>

Avant de commencer, assurez-vous d’avoir :

* Un compte Vercel avec un projet actif. Au besoin, [créez un compte Vercel](https://vercel.com/signup)
* Une application Next.js qui utilise la dernière version du [Auth0 Next.js SDK](/fr-CA/docs/quickstart/webapp/nextjs)

Après avoir installé l’intégration, [activez l’intégration dans une iframe](/fr-CA/docs/customize/login-pages/manage-core-configuration#iframe-embedding) pour permettre à Universal Login ou à Classic Login de se charger dans une iframe, comme l’exige Vercel.

<div id="install-integration">
  ## Installer l’intégration
</div>

Installez l’intégration native d’Auth0 pour Vercel à partir du Vercel Marketplace :

1. Accédez à **Integrations &gt; Browse Marketplace** et recherchez Auth0. Sélectionnez l’**intégration Auth0** sous **Native Integrations**, puis sélectionnez **Install**.

2. Auth0 renseigne automatiquement l’URL localhost et l’URL de callback de votre application. Sélectionnez votre **Installation Plan**, puis sélectionnez **Continue**.

3. Saisissez un nom pour votre application. Sélectionnez **Create**. Une fois le nouveau locataire Auth0 et l’application créés avec succès, sélectionnez **Continue**.

4. Pour connecter un projet Vercel existant, sélectionnez **Connect Project**, puis votre projet et vos environnements. Saisissez un préfixe personnalisé à ajouter à vos variables d’environnement, au besoin. Sélectionnez **Connect**. Une fois le projet Vercel connecté, vous devriez être redirigé vers la section [Getting Started](#getting-started) de l’intégration.

<div id="getting-started">
  ## Bien démarrer
</div>

La section Bien démarrer dans Vercel Dashboard affiche un guide de démarrage rapide pour mettre votre application en service rapidement et de façon sécuritaire. Vous pouvez aussi gérer les paramètres de votre application dans le menu de navigation de gauche.

<div id="quickstart">
  ### Guide de démarrage rapide
</div>

Pour commencer à utiliser l’intégration native Vercel d’Auth0, suivez les instructions dans le Vercel Dashboard pour créer et déployer votre application.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Vous ne pouvez configurer les variables d’environnement Auth0 que dans un environnement de production.
</Callout>

<div id="manage-application-settings">
  ## Gérer les paramètres de l’application
</div>

Vous pouvez gérer les paramètres de votre application dans le [Vercel Dashboard](#vercel-dashboard) et l’[Auth0 Dashboard](#auth0-dashboard).

<div id="vercel-dashboard">
  ### Vercel Dashboard
</div>

Pour gérer les paramètres de votre application dans le Vercel Dashboard, accédez à **Settings** dans le menu de navigation de gauche.

Vous pouvez mettre à jour les paramètres d’application suivants :

* **Modifier le nom** : Modifiez le nom de l’application.
* **Mettre à jour la configuration** :
  * **Paramètres** : Mettez à jour l’URL localhost et le point de terminaison de callback de votre application.
  * **Plan actuel à l’échelle de l’installation** : Accédez aux paramètres de l’intégration pour mettre à jour le plan à l’échelle de l’installation de votre intégration. Pour en savoir plus, consultez [Settings](#settings).
* **Sécuriser cette ressource**
  * **Rotation des secrets** : Effectuez la rotation des secrets de votre application. Pour en savoir plus, consultez [Rotating environment variables](https://vercel.com/docs/projects/environment-variables/managing-environment-variables#rotating-environment-variables).
  * **Environnements autorisés** : Sélectionnez si l’intégration est disponible dans **Tous les environnements** ou **L’environnement de production uniquement**.
* **Supprimer l’intégration** : Lorsque vous supprimez l’intégration, vous passez automatiquement au forfait Free de Vercel. Pour en savoir plus, consultez [Delete integration](#delete-integration).

<div id="auth0-dashboard">
  ### Auth0 Dashboard
</div>

Gérez des paramètres de l’application supplémentaires dans l’[Auth0 Dashboard](https://manage.auth0.com/).

Lorsque vous installez l’intégration native, Auth0 crée une nouvelle application dans un locataire dédié, préremplie avec vos URI Vercel, les URL de callback autorisées et les URL de déconnexion.

Vous pouvez modifier cette application, par exemple en personnalisant votre expérience Universal Login, comme vous le feriez pour n’importe quelle autre application dans votre Auth0 Dashboard.

<div id="manage-integration">
  ## Gérer l’intégration
</div>

Pour gérer l’intégration depuis votre Vercel Dashboard, sélectionnez l’onglet **Integration** de votre projet, faites défiler la page jusqu’à l’intégration Auth0, puis sélectionnez **Manage**.

<div id="invoices">
  ### Factures
</div>

Sur la page **Factures**, consultez la liste des factures liées à votre intégration Auth0. Pour en savoir plus, consultez [Billing &amp; Invoices](https://vercel.com/docs/pricing/understanding-my-invoice).

<div id="settings">
  ### Paramètres
</div>

Sur la page **Paramètres**, ajoutez des modes de paiement, modifiez le plan à l’échelle de l’installation et supprimez l’intégration.

<div id="payment-methods">
  #### Modes de paiement
</div>

Pour ajouter un mode de paiement, sélectionnez **Ajouter**. Pour en savoir plus sur les modes de paiement, consultez [la tarification de Vercel](https://vercel.com/docs/accounts/plans).

<div id="current-installation-level-plan">
  #### Plan actuel à l’échelle de l’installation
</div>

Le plan à l’échelle de l’installation s’applique à tous les produits de l’installation.

Pour modifier le plan à l’échelle de l’installation :

1. Sous **Settings**, faites défiler la page jusqu’à **Current Installation Level Plan**, puis sélectionnez **Change Plan**.
2. Sélectionnez le plan souhaité à l’échelle de l’installation.
3. Sélectionnez **Continue**.
4. Vérifiez votre sélection, puis sélectionnez **Update**.
5. Une fois l’installation mise à jour, sélectionnez **Done**.

<div id="delete-integration">
  #### Supprimer l’intégration
</div>

Lorsque vous supprimez l’intégration, votre forfait Vercel passe automatiquement au forfait Free.

Pour supprimer l’intégration :

1. Sous **Settings**, faites défiler la page jusqu’à **Delete &lt;YOUR&#95;PRODUCT&gt;** et sélectionnez le bouton **Remove &lt;YOUR&#95;PRODUCT&gt;**.
2. Saisissez le slug de l’intégration et votre équipe, puis sélectionnez **Je reconnais que cela supprimera mon compte connecté à Auth0**.
3. Sélectionnez **Remove &lt;YOUR&#95;PRODUCT&gt;**.
