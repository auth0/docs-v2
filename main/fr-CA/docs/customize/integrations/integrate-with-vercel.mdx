<div id="overview">
  ## Aperçu
</div>

Connectez une instance Auth0 nouvelle ou existante à votre projet Vercel pour renforcer la sécurité, l’évolutivité et la personnalisation de votre application à l’aide de [intégration Auth0](https://vercel.com/integrations/auth0). Cette intégration vous permet de mettre en place une approche simplifiée de l’identité client directement dans votre application.

Utilisez la nouvelle intégration pour créer et personnaliser le flux d’identité de votre projet Vercel. Pour mettre votre application en service rapidement et de façon sécuritaire, cette intégration :

* Configure un environnement de production dans Vercel qui s’intègre à votre locataire Auth0 en quelques clics.
* Crée automatiquement une Application dans votre locataire Auth0 qui se connecte à votre projet Vercel.
* Précharge vos identifiants Auth0 dans Vercel, ce qui vous permet d’éviter certaines étapes manuelles du processus.

<div id="prerequisites">
  ### Prérequis
</div>

Avant de commencer l’intégration d’Auth0 à votre projet Vercel, vous devez configurer quelques éléments :

* Un compte Vercel avec un projet que vous souhaitez intégrer. Si vous n’avez pas encore de compte Vercel, consultez [Créez votre compte Vercel](https://vercel.com/signup) pour commencer.
* Une application Next.js basée sur la dernière version du [SDK Auth0 pour Next.js](/fr-CA/docs/quickstart/webapp/nextjs#install-the-auth0-next-js-sdk).

<div id="add-a-vercel-integration">
  ## Ajouter une intégration Vercel
</div>

Pour ajouter l’intégration Auth0 à un projet Vercel existant :

1. Accédez à la [page de l’intégration Auth0](https://vercel.com/integrations/auth0).
2. Lorsque la fenêtre d’installation s’affiche, sélectionnez l’équipe Vercel et le projet dans lequel vous souhaitez installer l’intégration Auth0.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Nous offrons actuellement l’intégration pour un seul projet à la fois. Si vous sélectionnez **Tous les projets**, l’écran de sélection du projet sera affiché plus tard dans le processus.
   </Callout>
3. Sélectionnez **Installer** pour poursuivre l’installation dans votre portail d’intégration Auth0.
4. Vérifiez que vous êtes connecté au bon compte Auth0 et que vous effectuez l’intégration dans le bon tableau de bord du locataire.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Connectez-vous à Auth0 **ou** vérifiez que vous êtes connecté au bon compte Auth0 et que vous effectuez l’intégration dans le bon tableau de bord du locataire.

     * Si aucun locataire n’est sélectionné, vous serez redirigé vers la création d’un locataire.
     * Si vous créez un nouveau locataire, vous suivrez notre processus de configuration initiale standard.
     * Si vous débutez avec Auth0, c’est ici que vous créerez un compte Auth0.
   </Callout>
5. Sélectionnez **Accepter**.
6. Ensuite, sélectionnez le projet Vercel que vous souhaitez intégrer à Auth0, puis sélectionnez **Continuer**.
7. Confirmez la sélection de l’application, puis sélectionnez **Continuer**.
8. Vous avez maintenant ajouté l’intégration Auth0 à votre projet Vercel. Pour voir les intégrations installées, accédez à l’onglet **Intégrations** sur la page d’aperçu de votre projet. Auth0 apparaît maintenant comme une intégration installée pour votre projet.

<div id="manage-a-vercel-integration">
  ## Gérer une intégration Vercel
</div>

Après avoir ajouté avec succès l’intégration Auth0 à votre projet Vercel, vous pouvez la gérer à partir du tableau de bord de votre application dans Vercel.

<div id="manage-from-vercel">
  ### Gérer depuis Vercel
</div>

Pour gérer l’application Auth0 depuis votre tableau de bord Vercel, sélectionnez l’intégration Auth0 dans l’onglet **Integration** de votre projet.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Les variables d’environnement peuvent être configurées uniquement dans le domaine de production.
</Callout>

<div id="manage-access">
  #### Gérer l’accès
</div>

Pour gérer l’accès de l’intégration à certains projets dans votre application Vercel, sélectionnez **Gérer l’accès.** Cette option vous permet d’appliquer une restriction d’accès au niveau du projet pour l’intégration Auth0.

<div id="remove-integration">
  #### Supprimer l’intégration
</div>

Pour désinstaller l’intégration Auth0, sélectionnez **Supprimer l’intégration**; l’application et tous les paramètres associés seront supprimés de votre <Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+dashboard">Auth0 Dashboard</Tooltip>. Notez que cette opération est irréversible et que, si vous la supprimez, vous devrez réinstaller l’application.

<div id="manage-from-the-auth0-dashboard">
  ### Gérer à partir de l’Auth0 Dashboard
</div>

Après avoir ajouté l’intégration Auth0 à votre projet Vercel, l’[Auth0 Dashboard](https://manage.auth0.com/) crée une nouvelle application pour gérer et personnaliser votre parcours CIAM.

1. Accédez à l’[Auth0 Dashboard](https://manage.auth0.com/).
2. Sélectionnez **Applications &gt; Applications &gt; [Votre application Vercel]**
3. Confirmez l’URI de l’application Vercel sous **Settings &gt; Application URIs.**

   <Frame>![](/docs/images/cdy7uua7fh8z/62UwXZPwqrjYly0rkXawYb/31a270f2e8b1413bac2fd879edd1dd6e/border_Screenshot_2024-05-30_at_10.54.28_AM.png)</Frame>
4. Vous pouvez maintenant personnaliser et modifier cette application comme n’importe quelle autre application dans l’Auth0 Dashboard.
