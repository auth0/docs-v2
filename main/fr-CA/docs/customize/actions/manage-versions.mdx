title: Gérer les versions
---

Grâce à l’historique des versions d’Actions, vous pouvez comparer les versions en brouillon et les versions déployées d’Actions, et rétablir des versions antérieures au besoin.

<div id="deploy-actions">
  ## Déployer des Actions
</div>

Avant qu’une Action puisse être associée à un flux, elle doit être déployée.

1. Accédez à [Auth0 Dashboard &gt; Actions &gt; Custom Actions](https://manage.auth0.com/#/actions/library).
2. Choisissez votre Action.
3. Sélectionnez **Deploy**.

L’Action sélectionnée sera déployée et ajoutée à la liste des actions disponibles pour les flux. De plus, une nouvelle version brouillon de l’Action déployée sera créée pour vous permettre d’y apporter d’autres modifications. Comme les secrets et les dépendances sont associés aux versions, tous les secrets et dépendances enregistrés seront disponibles dans cette nouvelle version brouillon.

<div id="revert-actions">
  ## Rétablir une version antérieure d’Actions
</div>

Au besoin, vous pouvez rétablir la dernière version valide connue d’une Action.

1. Accédez à [Auth0 Dashboard &gt; Actions &gt; Custom Actions](https://manage.auth0.com/#/actions/library).
2. Choisissez votre Action.
3. Sélectionnez **Version History**.
4. Sélectionnez la version à laquelle vous voulez rétablir, puis cliquez sur **Revert**.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Dans l’historique des versions, « Draft » désigne une version qui n’a jamais été déployée.
   </Callout>

Une nouvelle copie de la version sélectionnée sera créée et déployée. De plus, une nouvelle version brouillon de l’Action déployée sera créée pour modification ultérieure. Comme les secrets et les dépendances sont associés aux versions, tous les secrets ou dépendances enregistrés seront disponibles dans la nouvelle version brouillon.
