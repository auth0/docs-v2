Chaque courriel envoyé dans le cadre des flux de travail Auth0 a un modèle de courriel correspondant qui détermine l’adresse d’expéditeur, l’objet, le corps du message, et plus encore.

Après avoir [configuré un fournisseur de téléphone externe](../phone-messages/configure-phone-messaging-providers), vous pouvez modifier les modèles de téléphone pour personnaliser vos messages téléphoniques.

Les modèles de messages texte prennent en charge [Liquid](https://liquidjs.com/index.html), un langage de modèles open source qui étend les fonctionnalités de HTML. Liquid vous permet de :

* Remplir dynamiquement des variables avec des données, par exemple en remplaçant `{{ user.name }}` par le nom de l’utilisateur
* Implémenter une logique, comme des instructions `if`/`else`
* Modifier des données à l’aide de filtres pour manipuler des chaînes et des tableaux, d’opérateurs mathématiques, et plus encore

Vous pouvez activer, désactiver et personnaliser les modèles de téléphone sur la [page des modèles de téléphone dans l’Auth0 Dashboard](https://manage.auth0.com/#/phone).

<div id="available-phone-templates">
  ## Modèles de téléphone disponibles
</div>

Voici un résumé des modèles de téléphone disponibles :

| Nom du modèle de téléphone                 | Description                                                                                         |
| ------------------------------------------ | --------------------------------------------------------------------------------------------------- |
| Compte bloqué                              | Informe le client que son compte a été bloqué en raison d’une activité suspecte.                    |
| Changement de mot de passe                 | Envoie au client son code de changement de mot de passe.                                            |
| Mot de passe à usage unique - Inscription  | Envoie au client le code de vérification pour l’inscription au mot de passe à usage unique (OTP).   |
| Mot de passe à usage unique - Vérification | Envoie au client le code de vérification pour la vérification du mot de passe à usage unique (OTP). |
