<Warning>
  Pour personnaliser les modèles de téléphone, vous devez [configurer un fournisseur de services téléphoniques externe](../configure-phone-messaging-providers).
</Warning>

<div id="customize-phone-templates">
  ## Personnaliser les modèles de téléphone
</div>

Pour personnaliser un modèle de téléphone :

1. Accédez à [**Dashboard &gt; Image de marque &gt; Modèles de téléphone**](https://manage.auth0.com/#/phone).

2. Dans la liste déroulante **Modèle**, sélectionnez le modèle de téléphone que vous voulez mettre à jour.

3. Activez le curseur **Statut** pour activer le modèle s’il ne l’est pas déjà.

4. Mettez à jour le numéro de téléphone **De** ou le **Modèle de message texte**.

5. Sélectionnez **Enregistrer** pour enregistrer vos modifications, **Tester** pour les tester ou **Réinitialiser** pour les annuler.

<div id="from-phone-number">
  ### Numéro de téléphone d’envoi
</div>

<Warning>
  Si vous utilisez les [Twilio Messaging Services](https://www.twilio.com/docs/messaging/services) comme source SMS, vous ne pouvez pas définir de numéro de téléphone **From** personnalisé. Twilio choisit plutôt un numéro dans le pool d’expéditeurs de votre Messaging Service, selon votre configuration Twilio.
</Warning>

Le champ **From** définit le numéro de téléphone à partir duquel les messages texte sont envoyés. Par défaut, le numéro de téléphone **From** est celui que vous avez défini lors de la configuration de votre fournisseur de téléphone.

Pour définir un numéro de téléphone **From** personnalisé pour un modèle de message téléphonique, saisissez le numéro de téléphone dans ce champ.

<div id="text-message-template">
  ### Modèle de message texte
</div>

Le champ **Modèle de message texte** définit le contenu du message texte. Chaque modèle est fourni avec un corps de message par défaut, que vous pouvez modifier ou remplacer entièrement par le vôtre.

Ce champ prend en charge la syntaxe Liquid pour générer du contenu dynamique. Pour en savoir plus, consultez [la syntaxe Liquid prise en charge pour les modèles de téléphone](./supported-liquid-syntax).

<div id="test-updated-templates">
  ## Tester les modèles mis à jour
</div>

Pour tester un modèle de téléphone, cliquez sur **Essayer**. Auth0 envoie le message téléphonique à une application par défaut portant le nom de votre locataire. Pour tester des modèles pour différentes applications, créez un utilisateur de test afin de passer par les flux de messages téléphoniques.

Vous pouvez déclencher manuellement des courriels de vérification pour des applications et des utilisateurs précis à l’aide du point de terminaison [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) de la Management API.
