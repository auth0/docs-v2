title: Configurer l’expérience téléphonique unifiée d’Auth0
---

<Card title="Avant de commencer">
  Si vous utilisez l’[authentification SMS Passwordless](/fr-CA/docs/authenticate/passwordless/authentication-methods/use-sms-gateway-passwordless), les [notifications vocales et SMS pour MFA](/fr-CA/docs/secure/multi-factor-authentication/multi-factor-authentication-factors/configure-sms-voice-notifications-mfa), ou Phone as ID, vous devez configurer un fournisseur téléphonique avec l’expérience téléphonique unifiée.

  Si vous avez besoin de fournisseurs différents pour MFA, Passwordless et/ou Phone as ID, vous devez configurer un [fournisseur téléphonique personnalisé](/fr-CA/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider) à l’aide d’Auth0 Actions.
</Card>

L’expérience téléphonique unifiée centralise toutes les configurations des fournisseurs téléphoniques en un seul endroit dans votre locataire. Cette centralisation évite d’avoir à configurer un fournisseur téléphonique plusieurs fois pour différents facteurs d’authentification par téléphone.

Si vous voulez essayer les fonctionnalités de messagerie téléphonique, vous devez configurer un fournisseur téléphonique au niveau du locataire.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 n’indique pas à l’utilisateur final si un message téléphonique a été envoyé avec succès par le fournisseur configuré au niveau du locataire. Pour résoudre les problèmes, consultez les [journaux du locataire](/fr-CA/docs/deploy-monitor/logs).
</Callout>

<div id="configure-mfa">
  ## Configurer MFA
</div>

Vous pouvez utiliser l’expérience téléphonique unifiée avec les méthodes d’envoi et les facteurs SMS, Voice et SMS + Voice. Pour en savoir plus sur l’<Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=MFA">MFA</Tooltip> avec Auth0, consultez [Authentification multifacteur](/fr-CA/docs/secure/multi-factor-authentication).

1. Dans l’Auth0 Dashboard, accédez à [**Branding &gt; Phone Provider**](https://manage.auth0.com/#/phone/templates/phone/provider).
2. Activez le bouton bascule Phone Message Provider.

   <Frame>![Auth0 Dashboard > Branding > Phone Provider > Activer Phone Message Provider](/docs/images/cdy7uua7fh8z/1k93jvdxDq2FZLiRdvr5aR/d917431cd0265943c5118174a8673118/2025-02-24_16-43-14.png)</Frame>
3. Sélectionnez votre fournisseur de messagerie téléphonique.

   1. Pour utiliser Twilio comme fournisseur téléphonique, suivez [Configurer Twilio comme fournisseur de messagerie téléphonique](/fr-CA/docs/customize/phone-messages/configure-phone-messaging-providers/configure-twilio-as-a-phone-messaging-provider).
   2. Pour utiliser un fournisseur personnalisé, suivez [Configurer un fournisseur téléphonique personnalisé](/fr-CA/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider) pour utiliser Actions.
4. Sélectionnez votre ou vos méthodes d’envoi.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Les facteurs MFA souhaités doivent être un sous-ensemble des méthodes d’envoi que vous configurez au niveau du locataire.
   </Callout>
5. **Enregistrez** vos modifications.
6. Une fois l’opération terminée, accédez à [Security &gt; Multi-factor Auth](https://manage.auth0.com/#/security/mfa).
7. Sélectionnez **Phone Message**.
8. Choisissez les facteurs d’authentification.
9. **Enregistrez** vos modifications.

<div id="configure-passwordless">
  ## Configurer Passwordless
</div>

Vous pouvez utiliser l’expérience téléphonique unifiée avec une connexion SMS <Tooltip tip="Passwordless : forme d’authentification qui ne repose pas sur un mot de passe comme premier facteur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Passwordless">Passwordless</Tooltip>. Pour en savoir plus sur l’authentification Passwordless avec Auth0, consultez [Connexions Passwordless](/fr-CA/docs/authenticate/passwordless).

1. Dans l’Auth0 Dashboard, accédez à [**Branding &gt; Phone Provider**](https://manage.auth0.com/#/phone/templates/phone/provider).
2. Activez le bouton bascule **Phone Message Provider**.

   <Frame>![Auth0 Dashboard > Branding > Phone Provider > Activer Phone Message Provider](/docs/images/cdy7uua7fh8z/1k93jvdxDq2FZLiRdvr5aR/d917431cd0265943c5118174a8673118/2025-02-24_16-43-14.png)</Frame>
3. Sélectionnez votre fournisseur de messagerie téléphonique.

   1. Pour utiliser Twilio comme fournisseur téléphonique, suivez [Configurer Twilio comme fournisseur de messagerie téléphonique](/fr-CA/docs/customize/phone-messages/configure-phone-messaging-providers/configure-twilio-as-a-phone-messaging-provider).
   2. Pour utiliser un fournisseur personnalisé, suivez [Configurer un fournisseur téléphonique personnalisé](/fr-CA/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider) afin d’utiliser les Auth0 Actions.
4. Sélectionnez votre méthode d’envoi.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     Si vous ne sélectionnez pas **Text** comme méthode d’envoi configurée, votre intégration avec Passwordless ne fonctionnera pas.
   </Callout>
5. **Enregistrez** vos modifications.
6. Une fois cette étape terminée, accédez à [**Authentication &gt; Passwordless &gt; SMS**](https://manage.auth0.com/#/connections/passwordless).
7. **Enregistrez** vos modifications.
8. Si vous utilisez Twilio :

   1. Accédez à [**Applications &gt; Applications**](https://manage.auth0.com/#/applications).
   2. Sélectionnez la ou les applications que vous souhaitez utiliser pour votre connexion SMS Twilio avec Passwordless.
   3. Sous **Connections**, activez l’option SMS (Twilio).
   4. Retournez à [**Authentication &gt; Passwordless &gt; SMS**](https://manage.auth0.com/#/connections/passwordless).
   5. Sous l’onglet **Applications**, activez toutes les applications que vous souhaitez utiliser pour Passwordless par SMS.
   6. **Enregistrez** vos modifications.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Si votre cas d’utilisation requiert l’ancienne expérience [MFA](/fr-CA/docs/secure/multi-factor-authentication/multi-factor-authentication-factors/configure-sms-voice-notifications-mfa) et [Passwordless](/fr-CA/docs/authenticate/passwordless/authentication-methods/use-sms-gateway-passwordless), communiquez avec [l’assistance Auth0](/fr-CA/docs/troubleshoot/customer-support).
</Callout>

<div id="test-your-phone-provider">
  ### Tester votre fournisseur téléphonique
</div>

Testez le fournisseur configuré au niveau du locataire avant de l’utiliser dans un environnement de production :

1. Accédez à [**Branding &gt; Phone Provider**](https://manage.auth0.com/#/phone/templates/phone/provider).
2. Sélectionnez **Send Test Message** pour tester le fournisseur configuré au niveau du locataire avant de l’utiliser dans un environnement de production.

<div id="configure-phone-templates">
  ## Configurer les modèles de téléphone
</div>

Le fournisseur téléphonique au niveau du locataire utilise la [syntaxe Liquid](https://github.com/Shopify/liquid/wiki/Liquid-for-Designers) pour insérer des variables contextuelles dans le contenu de vos messages téléphoniques. Pour configurer vos modèles de téléphone, suivez les instructions dans [Personnaliser les modèles de téléphone](/fr-CA/docs/customize/phone-messages/phone-templates/customize).

<div id="disable-phone-provider">
  ## Désactiver le fournisseur téléphonique
</div>

Si vous configurez l’expérience téléphonique unifiée, puis la désactivez, les intégrations existantes avec votre fournisseur téléphonique pourraient cesser de fonctionner, y compris votre facteur MFA par téléphone.
