---

<div id="prerequisites">
  ## Prérequis
</div>

Vous devez avoir un compte [Twilio](https://twilio.com/) avec une option d’envoi valide par SMS et/ou par téléphone.

<div id="configure-twilio">
  ## Configurer Twilio
</div>

1. Connectez-vous à votre compte [Twilio](https://www.twilio.com/login).
2. Accédez à votre [Twilio Console](https://console.twilio.com/). Vous aurez besoin de votre SID Twilio et de votre Twilio Auth Token pour configurer Twilio dans votre Auth0 Dashboard.
3. Accédez à Auth0 [Dashboard &gt; image de marque &gt; fournisseur de téléphone](https://manage.auth0.com/#/branding/phone/provider). Sélectionnez **Twilio** comme **fournisseur de téléphone**.
4. Sélectionnez une **méthode de livraison**. Vous pouvez choisir **Texte**, **Voix** ou les deux comme méthodes de livraison pour vos clients.
5. Dans les **paramètres du fournisseur Twilio**, sélectionnez une **source SMS** :

   1. Si vous sélectionnez **Use From**, saisissez un numéro de téléphone par défaut à partir duquel vos clients recevront les messages.
   2. Si vous sélectionnez [Messaging Service](https://www.twilio.com/docs/messaging/services), saisissez le SID de votre service de messagerie Twilio. Vous trouverez cette valeur dans votre Twilio Console.
   3. Saisissez votre SID Twilio et votre Twilio Auth Token à partir de la Twilio Console. Cliquez sur **Enregistrer**.

<div id="test-phone">
  ## Tester le téléphone
</div>

Cliquez sur le bouton **Essayer** pour envoyer un message de test. Si vous avez correctement configuré Twilio comme fournisseur de téléphonie externe, Auth0 enverra un message de confirmation selon le mode d’envoi sélectionné. Si vous ne recevez aucun message après quelques minutes, consultez vos [journaux Auth0](https://manage.auth0.com/#/logs) pour vérifier s’il y a eu des échecs.

La page Twilio [Messaging Activity](https://console.twilio.com/us1/develop/sms/overview) affiche des renseignements sur la livraison de tous les SMS envoyés à vos utilisateurs.
