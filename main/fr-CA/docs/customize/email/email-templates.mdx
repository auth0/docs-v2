---
title: Modèles de courriel
description: Activez et personnalisez les courriels envoyés par les flux de travail Auth0 à l’aide de modèles de courriel.
---

Chaque courriel envoyé dans le cadre des flux de travail Auth0 possède un modèle de courriel correspondant, qui détermine l’adresse de l’expéditeur, l’objet, le corps du message et plus encore.

Après avoir [configuré un fournisseur de courriel SMTP externe](/fr-CA/docs/customize/email/smtp-email-providers), vous pouvez modifier les modèles de courriel pour personnaliser vos courriels.

Les champs de contenu des modèles de courriel prennent en charge [Liquid](https://liquidjs.com/index.html), un langage de modèles open source qui étend les fonctionnalités du HTML. Liquid vous permet de :

* Renseigner dynamiquement des variables avec des données, par exemple en remplaçant `{{ user.name }}` par le nom de l’utilisateur
* Implémenter une logique, par exemple avec des instructions `if`/`else`
* Modifier des données à l’aide de filtres pour manipuler des chaînes et des tableaux, utiliser des opérateurs mathématiques, et plus encore

Vous pouvez activer, désactiver et personnaliser les modèles de courriel dans la [page des modèles de courriel du tableau de bord](https://manage.auth0.com/#/templates).

<div id="email-template-limits">
  ## Limites des modèles de courriel
</div>

* La personnalisation des modèles de courriel n’est pas offerte lorsque vous utilisez le fournisseur de courriel intégré. Pour utiliser des modèles de courriel, [configurez un fournisseur SMTP externe pour l’envoi de courriels](/fr-CA/docs/customize/email/smtp-email-providers).

* Chaque type de courriel possède exactement un modèle. Vous pouvez utiliser Liquid pour définir des conditions et d’autre logique pour des cas d’utilisation comme les courriels multilingues.

* Les modèles de courriel ne prennent pas en charge les courriels en texte brut. Tous les courriels doivent être au format HTML.

* Dans le modèle de courriel d’inscription à la MFA, la variable `connection.name` n’est pas disponible.

* Dans le modèle de courriel Passwordless OTP, seule la propriété `email` de l’objet `user` est disponible.

* Contrairement aux autres modèles de courriel, le modèle de courriel Passwordless OTP se configure dans la [page des paramètres de courriel des connexions Passwordless](https://manage.auth0.com/#/connections/passwordless), plutôt que dans la page Modèles de courriel.

* Pour des raisons de sécurité, le champ **From Address** dans les modèles de courriel ne peut pas contenir `@auth0.com`. Si vous entrez une adresse courriel `@auth0.com` dans From Addresses, votre locataire n’enverra que les courriels par défaut et ignorera toute personnalisation. Les utilisateurs de Private Cloud peuvent configurer une liste de refus de domaines semblable.
