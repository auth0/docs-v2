---
title: Configuration d’un fournisseur de courriel personnalisé avec Actions
description: Rédigez une Action pour acheminer des messages vers votre fournisseur de courriel et implémenter une logique personnalisée pour votre cas d’utilisation.
---

Pour personnaliser davantage la configuration de votre fournisseur de courriel, vous pouvez configurer votre locataire avec une <Tooltip tip="Action : une fonction Node.js qui s’exécute à des points définis durant l’exécution d’Auth0." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Action">Action</Tooltip>.

Au lieu d’utiliser notre fournisseur de courriel ou nos intégrations de serveur SMTP, vous rédigez le code de votre Action pour acheminer les messages vers le fournisseur de courriel ou le serveur SMTP de votre choix.

Cela vous donne plus de contrôle sur le processus d’envoi des courriels et vous permet d’implémenter une logique personnalisée pour votre cas d’utilisation, par exemple pour réessayer après un échec, changer les destinataires ou créer une logique propre à une organisation.
