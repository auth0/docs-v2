Flows vous permet de créer visuellement une logique côté serveur qui peut être exécutée dans vos Forms pour traiter les informations recueillies, intégrer n’importe quel service au moyen d’une API ou déclencher des automatisations.

<Frame>![Dashboard > Forms > Flows](/docs/images/cdy7uua7fh8z/6u6Huh5CbH3GEQJv3AH7cx/e49ea613b2c46a43c83c89ffecddbafe/Flows.png)</Frame>

Vous pouvez utiliser les actions de Flows avec votre formulaire pour :

* Créer des validations personnalisées côté serveur pour les étapes de votre formulaire.
* Intégrer votre propre API en effectuant des requêtes HTTP personnalisées.
* Intégrer des services tiers à l’aide de nos intégrations prêtes à l’emploi.
* Exécuter une logique métier différente selon les saisies ou les propriétés de l’utilisateur.
* Valider les données à l’aide de <Tooltip tip="JSON Web Token (JWT) : format standard d’ID Token (et souvent de jeton d’accès) utilisé pour représenter de façon sécurisée des revendications entre deux parties." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=JSON+Web+Tokens">JSON Web Tokens</Tooltip>.
