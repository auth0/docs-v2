<div id="get-access-to-auth0-management-api">
  ## Accéder à l’API de gestion Auth0
</div>

[Créez une application Machine-to-Machine (M2M)](/fr-CA/docs/get-started/auth0-overview/create-applications/machine-to-machine-apps) autorisée à accéder à l’API de gestion Auth0.

<Warning>
  N’oubliez pas d’autoriser votre application M2M à accéder à l’API de gestion et de sélectionner les portées appropriées conformément au principe du moindre privilège :

  * Pour récupérer un utilisateur : `read:users`
  * Pour créer un utilisateur : `create:users`
  * Pour mettre à jour un utilisateur : `update:users`
</Warning>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Aucune autorisation pour l’API de gestion Auth0 n’est requise pour envoyer des notifications par l’entremise des fournisseurs configurés dans votre tenant Auth0.
</Callout>

<div id="configure-your-vault-connection">
  ## Configurez votre connexion Vault
</div>

Pour configurer une connexion à Auth0 Vault pour vos actions de flux Auth0, définissez les paramètres requis en fonction de l’application M2M créée précédemment.

<Frame>![Dashboard > Actions > Forms > Vault > Auth0](/docs/images/customize/forms/vaults/auth0-vault-connection.png)</Frame>

<div id="input-settings">
  #### Paramètres d&#39;entrée
</div>

| Paramètre             | Description                                                                         |
| --------------------- | ----------------------------------------------------------------------------------- |
| Domaine du tenant     | Domaine du tenant de votre application M2M.                                         |
| ID client             | ID client de votre application M2M autorisée à utiliser l’API de gestion Auth0.     |
| Secret client         | Secret client de votre application M2M autorisée à utiliser l’API de gestion Auth0. |
| Audience (facultatif) | Audience de l’API de gestion Auth0.                                                 |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Par défaut, l’Audience sera déterminée à partir du domaine du tenant.
</Callout>
