<div id="get-your-sendgrid-keys">
  ## Obtenez vos clés SendGrid
</div>

Vous les trouverez dans les paramètres du [tableau de bord SendGrid](https://app.sendgrid.com/settings/api_keys), dans la section des clés API.

<div id="configure-your-sendgrid-vault-connection">
  ## Configurez votre Connexion SendGrid Vault
</div>

Pour configurer une Connexion SendGrid Vault pour vos Actions de flux SendGrid, vous devez définir les paramètres requis.

<Frame>![Tableau de bord > Actions > Formulaires > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre | Description                                             |
| --------- | ------------------------------------------------------- |
| Clé API   | Clé API utilisée pour l’autorisation de l’API SendGrid. |
