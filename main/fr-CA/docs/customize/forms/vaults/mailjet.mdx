<div id="get-your-mailjet-keys">
  ## Récupérez vos clés MailJet
</div>

Vous les trouverez dans le tableau de bord MailJet, sur la [page Account](https://app.mailjet.com/account/api_keys), sous API keys.

<div id="configure-your-mailjet-vault-connection">
  ## Configurez votre Connexion Vault MailJet
</div>

Pour configurer une Connexion Vault MailJet pour vos Actions du flux MailJet, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre   | Description                                                   |
| ----------- | ------------------------------------------------------------- |
| Clé API     | Clé API utilisée pour l’authentification à l’API MailJet.     |
| Clé secrète | Clé secrète utilisée pour l’authentification à l’API MailJet. |
