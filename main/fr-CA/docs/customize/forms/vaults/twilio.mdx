<div id="get-your-twilio-credentials">
  ## Obtenez vos identifiants Twilio
</div>

Vous les trouverez dans le [tableau de bord de la console Twilio](https://console.twilio.com/).

<div id="configure-your-twilio-vault-connection">
  ## Configurez votre connexion Twilio Vault
</div>

Pour configurer une connexion Twilio Vault pour vos Actions Twilio Flow, vous devez définir les paramètres requis.

<Frame>![Tableau de bord > Actions > Formulaires > Vault > Twilio](/docs/images/customize/forms/vaults/twilio-vault-connection.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                | Description                                                             |
| ------------------------ | ----------------------------------------------------------------------- |
| SID du compte            | SID du compte utilisé pour autoriser l’accès à l’API Twilio.            |
| Jeton d’authentification | Jeton d’authentification utilisé pour autoriser l’accès à l’API Twilio. |
