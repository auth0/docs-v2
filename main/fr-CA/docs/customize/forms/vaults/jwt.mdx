  ## Configurez votre connexion JWT Vault
</div>

Pour configurer une connexion JWT Vault afin de permettre à vos Actions JWT dans Flows de signer, de décoder et de vérifier des JWT, vous devez définir les paramètres requis.

<Frame>![Dashboard > Actions > Forms > Vault > JWT](/docs/images/customize/forms/vaults/jwt-vault-connection.png)</Frame>

<div id="input-settings">
  ## Paramètres d’entrée
</div>

| Paramètre | Description                                                |
| --------- | ---------------------------------------------------------- |
| Secret    | Secret utilisé par l’algorithme HS256 pour encoder le JWT. |
