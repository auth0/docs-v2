<div id="http-request">
  ## HTTP Request
</div>

L’action HTTP Request vous permet d’appeler une API ou un service externe et de traiter la réponse.

<Frame>![](/docs/images/cdy7uua7fh8z/6eIOv07XenyaazzKixigei/2e9efd632137829a8ddb03aacd3197df/http-request-action.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre          | Description                                                                                                                                                            |
| ------------------ | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Méthode            | La méthode HTTP. Les valeurs possibles sont : <ul> <li>`GET`</li> <li>`POST`</li> <li>`PUT`</li> <li>`PATCH`</li> <li>`DELETE`</li> </ul>                              |
| Type de contenu    | L’en-tête `Content-Type`. Les valeurs possibles sont : <ul> <li>`application/json`</li> <li>`application/x-www-form-urlenconded`</li> <li>`application/xml`</li> </ul> |
| Corps              | Le corps de la requête.                                                                                                                                                |
| En-têtes           | Les en-têtes de la requête.                                                                                                                                            |
| Paramètres d’URL   | Les paramètres d’URL. Ces paramètres seront automatiquement encodés.                                                                                                   |
| Activer Basic Auth | Active l’authentification à l’aide d’un nom d’utilisateur et d’un mot de passe.                                                                                        |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type    | Description                                                      |
| --------- | ------- | ---------------------------------------------------------------- |
| `headers` | Object  | Les en-têtes de la réponse.                                      |
| `body`    | Object  | Le corps de la réponse.                                          |
| `status`  | Number  | Le code d’état de la réponse.                                    |
| `success` | Boolean | Renvoie `true` ou `false` en fonction du `status` de la réponse. |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
  "headers": {
    ...
  },
  "status": 200,
  "body": {
    ...
  },
  "success": true
}
```
