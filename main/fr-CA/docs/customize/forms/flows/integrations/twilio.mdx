Cette liste d’actions Twilio vous permet d’envoyer des SMS et de passer des appels en utilisant Twilio comme fournisseur.

<div id="configure-your-twilio-vault-connection">
  ## Configurez votre connexion Vault Twilio
</div>

Découvrez comment configurer une connexion Vault pour vos Actions de flux Twilio dans [Connexion Vault Twilio](/fr-CA/docs/customize/forms/vaults/twilio).

<div id="send-sms">
  ## Envoyer un SMS
</div>

Envoie un SMS.

<Frame>![](/docs/images/cdy7uua7fh8z/4lPqmcYGCJ2oRHM3GSta6w/10b0141d163e4be601c2786895486440/sendsmstwilio.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Parameter             | Description                                                                                    |
| --------------------- | ---------------------------------------------------------------------------------------------- |
| From (obligatoire)    | Le nom de l’expéditeur du SMS.                                                                 |
| To (obligatoire)      | Le numéro de téléphone du destinataire au [format E.164](https://en.wikipedia.org/wiki/E.164). |
| Message (obligatoire) | Le contenu du message SMS.                                                                     |

<div id="output-object">
  ### Objet de sortie
</div>

Vous trouverez tous les codes de statut attendus dans [la documentation de Twilio](https://www.twilio.com/docs/messaging/api/message-resource#message-status-values).

<div id="make-call">
  ## Passer un appel
</div>

Passe un appel.

<Frame>![](/docs/images/cdy7uua7fh8z/4JaAbJIVuVUJY40lhQuVUN/1a1bd6143c8033c12e18fa49252bd36b/makecalltwilio.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                   | Description                                                                                                                                                       |
| --------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| From (obligatoire)          | Votre numéro de téléphone Twilio. Le code de pays doit être inclus. Exemple : +18005550175, où 1 est le code de pays des États-Unis.                              |
| To (obligatoire)            | Le numéro de téléphone de destination au [format E.164](https://en.wikipedia.org/wiki/E.164).                                                                     |
| Payload TwiML (obligatoire) | Les instructions TwiML pour votre message vocal.<br />Pour en savoir plus, consultez [la documentation TwiML de Twilio](https://www.twilio.com/docs/voice/twiml). |

<div id="output-object">
  ### Objet de sortie
</div>

Vous trouverez tous les codes de statut attendus dans la [documentation Twilio](https://www.twilio.com/docs/voice/twiml).
