Cette liste d’actions de vérification des données vous permet de vérifier des adresses de courriel, ainsi que de générer et de valider des OTP.

<div id="generate-one-time-password">
  ## Générer un mot de passe à usage unique
</div>

Génère un code numérique valable pendant 5 minutes et pouvant être envoyé par courriel, par SMS ou par toute application de messagerie.

<Frame>![](/docs/images/cdy7uua7fh8z/1h69FWzZwk9beKKjB4Cq38/16aa515e61cc6e9ebdfd0849201e3141/generate-one-time-password.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                         | Description                                                                                                           |
| --------------------------------- | --------------------------------------------------------------------------------------------------------------------- |
| Valeur de référence (obligatoire) | Vous pouvez associer le code généré à un numéro de téléphone, à une adresse de courriel ou à n’importe quelle valeur. |
| Longueur du code (obligatoire)    | Le nombre de chiffres du code généré.                                                                                 |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type   | Description    |
| --------- | ------ | -------------- |
| `code`    | String | Le code généré |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
  "code": "███",
}
```

<div id="verify-one-time-password">
  ## Vérifier le mot de passe à usage unique
</div>

Vérifie si le mot de passe à usage unique fourni est valide.

<Frame>![](/docs/images/cdy7uua7fh8z/3bkT8QDN4dcl0YHTrSOIGj/9698a07f55353a11ca98164377cd4217/verify-otp.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                         | Description                                                                                         |
| --------------------------------- | --------------------------------------------------------------------------------------------------- |
| Valeur de référence (obligatoire) | Utilisez la même valeur de référence que celle configurée dans l’action Générer un mot de passe à usage unique. |
| Longueur du code (obligatoire)    | Le nombre de chiffres du code généré.                                                               |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type    | Description                                                                                                                                                                                                                                                                    |
| --------- | ------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `valid`   | Boolean | Indique si le code fourni est valide en renvoyant la valeur `true` ou `false`.                                                                                                                                                                                                 |
| `result`  | String  | Renvoie différents codes de résultat selon le code fourni : <ul> <li> `MISSING:` Le code fourni n’existe pas. </li> <li> `EXPIRED:` Le code fourni a expiré. </li> <li> `VALID:` Le code fourni est valide. </li> <li> `UNEQUAL:` Le code fourni n’est pas valide. </li> </ul> |

<div id="output-object-examples">
  ### Exemples d’objets de sortie
</div>

```json lines
{
  "valid": false,
  "result": "UNEQUAL"
}
```

```json lines
{
  "valid": true
}
```

<div id="verify-email-address">
  ## Vérifier l’adresse de courriel
</div>

Applique un certain nombre de règles de vérification sélectionnées à l’adresse de courriel fournie.

<Frame>![](/docs/images/cdy7uua7fh8z/6qkqny1hhObFalHwdEG342/1a2e18195b29102c3f68744f42277917/verify-email.png)</Frame>

<div id="input-settings">
  ### Paramètres d’entrée
</div>

| Paramètre                                        | Description                                                                                                                                                     |
| ------------------------------------------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Courriel (obligatoire)                           | adresse de courriel à vérifier.                                                                                                                                 |
| Exiger un enregistrement MX                      | Exiger des enregistrements MX. Lorsqu’un domaine n’a pas d’enregistrements MX, les serveurs de courriel ne savent pas où envoyer les courriels pour ce domaine. |
| Bloquer les fournisseurs de courriel gratuits    | Bloquer les fournisseurs de courriel gratuits comme @gmail.com, @hotmail.com, etc.                                                                              |
| Bloquer les fournisseurs de courriel temporaires | Bloquer les adresses de courriel temporaires comme @mailnator.com, @maildrop.cc, etc.                                                                           |
| Bloquer les alias de comptes de courriel         | Bloquer les courriels avec des alias comme `jane.doe+alias@gmail.com`                                                                                           |
| Domaines autorisés                               | Autoriser uniquement les courriels provenant des domaines approuvés dans votre liste.                                                                           |
| Domaines bloqués                                 | Bloquer des domaines de courriel précis.                                                                                                                        |

<div id="output-object">
  ### Objet de sortie
</div>

| Propriété | Type    | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| --------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `valid`   | Boolean | Renvoie `true` ou `false` selon que le courriel respecte ou non les règles de vérification.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            |
| `cause`   | String  | Si la propriété `valid` est `<br/>false<br/>` , renvoie une référence à la première règle à laquelle le courriel ne satisfait pas : <ul> <li> `MISSING_MX_RECORD:` Le domaine du courriel ne possède pas d’enregistrement MX. </li> <li> `FREE_EMAIL:` Le domaine du courriel correspond à un fournisseur de courriel gratuit.</li> <li> `DISPOSABLE_EMAIL:` Le courriel est une adresse de courriel jetable.</li> <li> `BLOCKLISTED:` Le domaine du courriel est inclus dans le paramètre des domaines de la liste de blocage.</li> <li> `NOT_ALLOWED:` Le domaine du courriel n’est pas inclus dans le paramètre des domaines de la liste d’autorisation.</li> </ul> |

<div id="output-object-example">
  ### Exemple d’objet de sortie
</div>

```json lines
{
  "valid": false,
  "cause": "FREE_EMAIL"
}
```

```json lines
{
  "valid": true,
  "cause": null
}
```
