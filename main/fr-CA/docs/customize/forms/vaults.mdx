Le Vault vous permet de stocker de façon sécurisée des secrets ou des paramètres courants (par exemple : des clés d’API, des <Tooltip tip="Jeton d’accès : information d’identification d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=access+tokens">jetons d’accès</Tooltip>, des <Tooltip tip="Jeton d’accès : information d’identification d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisée pour accéder à une API." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=refresh+tokens">jetons d’actualisation</Tooltip>) sous forme de connexions Vault afin d’authentifier les requêtes vers des services externes ou des intégrations.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * Certaines connexions exigent que l’utilisateur fournisse des informations d’identification (par exemple, avec Auth0 et une application machine à machine (M2M), vous générez un jeton d’accès avec une expiration et un jeton d’actualisation). La connexion stocke les deux jetons et génère un nouveau jeton d’accès avant son expiration.
  * Certains services externes exigent des scopes autorisés pour générer des identifiants d’API. Si vous modifiez les scopes, mettez à jour la connexion à ces services pour éviter les problèmes de connexion.
  * Lorsqu’une connexion est ajoutée, elle est prête à être utilisée dans vos flux. Toute mise à jour d’une connexion existante sera propagée automatiquement.
</Callout>

<div id="create-and-modify-vault-connections">
  ## Créer et modifier des connexions Vault
</div>

Créez et modifiez des connexions Vault à partir de la page du Vault ou de l’éditeur de flux lorsqu’une action nécessite des informations d’identification.

<div id="create-vault-connections-from-the-vault-page">
  ### Créer des connexions Vault depuis la page Vault
</div>

Ajoutez une connexion Vault depuis la page Vault en suivant ces étapes :

1. Ouvrez l’éditeur de formulaires en sélectionnant **Auth0 Dashboard &gt; Actions &gt; Forms.**
2. Dans le coin supérieur gauche, sélectionnez le menu du locataire, puis **Vault** pour ouvrir la page Vault.

   <Frame>![Dashboard > Forms > Vault](/docs/images/cdy7uua7fh8z/1sPNPKO6nuEDCuiYGRWSur/d85dc616170eaf2d7d5f321750863ccf/vault-connection.png)</Frame>
3. Sélectionnez **Add Vault Connection**.
4. Sélectionnez l’action souhaitée, puis **Continue**.
5. Saisissez un **nom** de connexion Vault, puis sélectionnez **Continue**.
6. Entrez les informations d’identification.
   Si vous n’avez pas les informations d’identification requises, vous pouvez sélectionner **Create an empty connection** et les ajouter plus tard. Les connexions vides affichent une icône orange indiquant que des informations d’identification sont manquantes.
7. Sélectionnez **Add Vault Connection**.

<div id="create-vault-connections-from-the-flow-editor">
  ### Créer des connexions Vault à partir de l’éditeur de flux
</div>

Ajoutez une connexion Vault à partir d’une action de flux, par exemple Twilio, en suivant ces étapes :

1. Ouvrez l’éditeur de flux en sélectionnant **Auth0 Dashboard &gt; Actions &gt; Forms &gt; Flows**
2. Sélectionnez un flux et ajoutez une action **Envoyer un SMS avec Twilio**.
3. Dans le paramètre **Connexion Vault** de l’action, sélectionnez l’**icône +**.

   <Frame>![Auth0 Dashboard > Forms > Vault ](/docs/images/cdy7uua7fh8z/1AsaznhRKxm5LZxPlcbfUw/10e5a8af700aab909f5b03fcd266da7e/twilio-vault-connection.png)</Frame>
4. Ajoutez un **nom de connexion** Vault et sélectionnez **Continuer**.
5. Entrez les identifiants.
   Si vous n’avez pas les identifiants requis, vous pouvez sélectionner **Créer une connexion vide** et ajouter les identifiants plus tard. Les connexions vides ont une icône orange qui indique que des identifiants sont manquants.

   <Frame>![Auth0 Dashboard > Forms > Vault > paramètres de la connexion Twilio](/docs/images/cdy7uua7fh8z/yobXanutPRCO3FWek8cOk/bc29e39fb8593cff35e2738234bfe6fe/twilio-vault-settings.png)</Frame>
6. Sélectionnez **Ajouter une connexion Vault.**
