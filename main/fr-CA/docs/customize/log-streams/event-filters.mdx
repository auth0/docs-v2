description: Répertorie les filtres disponibles pour les flux de journaux et les journaux inclus dans chacun d’eux.
title: Filtres des flux de journaux
---

Vous pouvez filtrer vos flux de journaux afin que seuls certains journaux et certaines catégories de journaux soient transmis. Pour configurer le filtre d’un flux de journaux :

1. Accédez à [l’Auth0 Dashboard &gt; Monitoring &gt; Streams](https://manage.auth0.com/#/log-streams).
2. Sélectionnez un flux de journaux, puis accédez à l’onglet **Settings**.
3. Activez ou désactivez les filtres dans le menu **Filter By Event Category**.
4. Cliquez sur **Save**.
