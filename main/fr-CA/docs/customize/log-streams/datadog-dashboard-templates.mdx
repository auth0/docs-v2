title: Utiliser des modèles de tableau de bord Auth0 avec Datadog
---

Auth0 fournit des modèles prédéfinis que vous pouvez utiliser pour visualiser les données de votre locataire Auth0 dans Datadog. Affichez des visualisations pour surveiller l’état du trafic de connexion d’un locataire. Les modèles vous permettent d’utiliser les agrégations recommandées par Auth0 ou de vous en servir comme point de départ pour créer vos propres visualisations personnalisées.

| Modèle                      | Objectif                                                                                                                                   | Fichier téléchargeable                                                                                        |
| --------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------------------- |
| Tableau de bord de sécurité | Surveillez le trafic de connexion du locataire, y compris les métriques de connexion, de création de compte et de détection des anomalies. | [Auth0SecDashboard.json](https://cdn.auth0.com/website/docs/logs/streams/datadog/Auth0SecurityDashboard.json) |

Vous pouvez aussi organiser votre visualisation à l’aide des fonctionnalités suivantes :

* Utilisez le sélecteur de période de Datadog pour choisir l’intervalle de temps de votre visualisation.
* Utilisez des filtres pour affiner l’analyse des données à l’aide de paramètres configurables comme l’adresse IP, le pays, l’utilisateur et l’application cliente.
* Utilisez **Edit Widgets** pour réorganiser les tuiles et les visualisations, ainsi que pour créer des copies dans lesquelles vous pouvez examiner les requêtes et y ajouter des paramètres au besoin.

<div id="prerequisites">
  ## Prérequis
</div>

* [Configurer la transmission des journaux vers Datadog](https://marketplace.auth0.com/integrations/datadog-log-streaming).
* Téléchargez depuis Auth0 le fichier JSON du modèle de tableau de bord souhaité.

<div id="upload-dashboard-templates-in-datadog">
  ## Importer des modèles de tableau de bord dans Datadog
</div>

1. Connectez-vous à [Datadog](https://www.datadoghq.com/).
2. Accédez à **Dashboards** dans la barre de navigation de gauche.
3. Sélectionnez **Screenboard** comme type de tableau de bord.
4. Sur le canevas vide, dans le coin supérieur droit, sélectionnez l’icône d’engrenage, puis cliquez sur **Import Dashboard JSON**.
5. Importez le fichier JSON du tableau de bord que vous souhaitez utiliser (par exemple, `Auth0SecDashboard.json`).

<div id="security-dashboard-visualizations">
  ## Visualisations du tableau de bord de sécurité
</div>

Une fois que vous avez importé le tableau de bord de sécurité à partir d’Auth0, vous pouvez utiliser les visualisations suivantes pour évaluer l’état de votre locataire ou comme point de départ pour créer vos propres visualisations et alertes dans Datadog.

| Visualisation                                             | Objectif                                                                                                                                                                                                                             |
| --------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Connexions réussies                                       | Affiche le nombre d’événements de connexion réussis pour le locataire.                                                                                                                                                               |
| Échecs de connexion                                       | Affiche le nombre d’événements de connexion ayant échoué pour le locataire. Des pics d’échecs de connexion peuvent indiquer un trafic anormal.                                                                                       |
| Échecs de connexion par géolocalisation                   | Affiche le nombre d’événements de connexion ayant échoué pour le locataire, par géolocalisation. Des pics de trafic inattendus dans certaines zones géographiques peuvent indiquer un trafic anormal.                                |
| Nombre d’IP uniques envoyant du trafic                    | Affiche le nombre d’adresses IP qui envoient du trafic vers le locataire. Ce nombre suit souvent des tendances prévisibles, et de fortes variations peuvent indiquer un trafic anormal.                                              |
| Top 5 des IP avec le plus d’échecs                        | Affiche les 5 adresses IP ayant enregistré le plus grand nombre d’erreurs de connexion.                                                                                                                                              |
| Événements de création de compte dans les principaux pays | Affiche le nombre de comptes créés dans les principaux pays. Des variations dans les tendances prévisibles et les zones géographiques peuvent indiquer un trafic anormal.                                                            |
| Événements de création de compte par plateforme           | Affiche le nombre de comptes créés par plateforme, comme Windows, OSX, Android ou iOS.                                                                                                                                               |
| IP ayant créé le plus de comptes                          | Affiche les adresses IP à partir desquelles le plus grand nombre de comptes ont été créés.                                                                                                                                           |
| Événements de détection d’anomalies                       | Affiche le nombre d’événements de détection d’anomalies pour le locataire, y compris le nombre d’adresses IP bloquées en raison de la protection contre les attaques par force brute et de la détection des mots de passe compromis. |
| Flux de journaux                                          | Affiche les événements bruts du journal du locataire.                                                                                                                                                                                |

<div id="customize-dashboard">
  ## Personnaliser le tableau de bord
</div>

Vous pouvez personnaliser votre tableau de bord de sécurité Auth0 dans Datadog pour y ajouter des widgets de données personnalisés.

1. Accédez à votre tableau de bord de sécurité Auth0, puis sélectionnez l’élément **Dashboards** dans la barre de navigation latérale.
2. Sélectionnez dans la liste le nom du tableau de bord que vous avez créé.
3. Sélectionnez une visualisation et faites-la glisser dans la section principale de la page.
4. Configurez les préférences de votre graphique de données.
