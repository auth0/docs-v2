<Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur le serveur d’autorisation d’Auth0, pour vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Universal+Login">Universal Login</Tooltip> se compose d’un ensemble de pages qui facilitent plusieurs actions liées au compte, comme la connexion, l’inscription à des facteurs d’<Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur le serveur d’autorisation d’Auth0, pour vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=multi-factor+authentication">authentification multifacteur</Tooltip> ou la réinitialisation de mots de passe. Auth0 fournit le texte de ces pages dans plusieurs langues.

Dans certains cas, il se peut que vous souhaitiez modifier le texte de ces pages afin qu’il corresponde mieux au ton de votre application ou à des besoins particuliers.

Vous pouvez personnaliser le texte affiché sur les pages [Universal Login](/fr-CA/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) dans toutes les langues prises en charge, soit au moyen de l’<Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, soit au moyen de la <Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>. Les définitions, valeurs et variables suivantes s’appliquent aux deux méthodes.

<div id="configure-prompts-and-screens">
  ## Configurer les invites et les écrans
</div>

Vous pouvez personnaliser n’importe quel élément parmi les [invites et écrans](#prompt-values) disponibles à l’aide d’Auth0 Dashboard ou de la [Management API](https://auth0.com/docs/api/management/v2).

<Tabs>
  <Tab title="Utiliser le tableau de bord">
    Vous pouvez utiliser le Dashboard pour personnaliser le texte des invites du nouveau Universal Login.

    1. Dans le Dashboard, accédez à [**Image de marque &gt; Universal Login &gt; Options avancées**](https://manage.auth0.com/#/custom_text), puis sélectionnez **Texte personnalisé**.
    2. Sélectionnez l’**invite** et l’**écran** que vous souhaitez modifier, puis apportez les modifications souhaitées. Un aperçu en temps réel est affiché à droite.
    3. Lorsque vous êtes prêt à faire apparaître vos modifications dans votre environnement de production, sélectionnez **Enregistrer les modifications.**
  </Tab>

  <Tab title="Utiliser Management API">
    Vous pouvez utiliser la Management API pour personnaliser les invites textuelles du nouveau Universal Login.

    1. Obtenez un [jeton d’accès pour la Management API](/fr-CA/docs/secure/tokens/access-tokens/management-api-access-tokens/get-management-api-access-tokens-for-production) avec les scopes `read:prompts` et `update:prompts`.
    2. Appelez le point de terminaison Management API [Obtenir le texte personnalisé d’une invite](https://auth0.com/docs/api/management/v2#!/Prompts/get_custom_text_by_language) pour obtenir les données de configuration existantes de l’invite et de la langue spécifiées.
    3. Modifiez les données de configuration au besoin, en vous référant aux [paramètres pris en charge](#supported-parameters) et aux [valeurs d’invite](#prompt-values).
    4. Appelez le point de terminaison de la [Management API pour définir le texte personnalisé d’une invite précise](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) avec les données de configuration mises à jour.

       <Warning>
         Le point de terminaison de la Management API [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) remplace l’ensemble des données de configuration existantes pour l’invite indiquée par celles fournies dans le paramètre `body`.

         Si vous voulez configurer plusieurs valeurs `screen` pour une invite donnée, vous devez toutes les inclure dans un seul appel à la Management API.
       </Warning>

    #### Exemple

    Vous pouvez définir le composant de texte `description` de l&#39;écran `login` de l&#39;invite `login` à `Log in to ACME's website` en appelant le point de terminaison [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) de la Management API comme suit :

    <AuthCodeGroup>
      ```bash cURL
      curl --request PUT \
        --url 'https://{yourDomain}/api/v2/prompts/login/custom-text/en' \
        --header 'authorization: Bearer {yourMgmtApiAccessToken}' \
        --header 'content-type: application/json' \
        --data '{ "login": { "description": "Login to ACME'''s Website" } }'
      ```

      ```csharp C#
      var client = new RestClient("https://{yourDomain}/api/v2/prompts/login/custom-text/en");
      var request = new RestRequest(Method.PUT);
      request.AddHeader("content-type", "application/json");
      request.AddHeader("authorization", "Bearer {yourMgmtApiAccessToken}");
      request.AddParameter("application/json", "{ "login": { "description": "Login to ACME's Website" } }", ParameterType.RequestBody);
      IRestResponse response = client.Execute(request);
      ```

      ```go Go
      package main

      import (
      	"fmt"
      	"strings"
      	"net/http"
      	"io/ioutil"
      )

      func main() {

      	url := "https://{yourDomain}/api/v2/prompts/login/custom-text/en"

      	payload := strings.NewReader("{ "login": { "description": "Login to ACME's Website" } }")

      	req, _ := http.NewRequest("PUT", url, payload)

      	req.Header.Add("content-type", "application/json")
      	req.Header.Add("authorization", "Bearer {yourMgmtApiAccessToken}")

      	res, _ := http.DefaultClient.Do(req)

      	defer res.Body.Close()
      	body, _ := ioutil.ReadAll(res.Body)

      	fmt.Println(res)
      	fmt.Println(string(body))

      }
      ```

      ```java Java

        .header("content-type", "application/json")

      ```

<div id="supported-parameters">
  ### Paramètres pris en charge
</div>

| Paramètre  | Définition                                                                                                                                                                                                                                        |
| ---------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `prompt`   | L’étape du flux de connexion. Pour obtenir la liste des valeurs offertes, consultez [Valeurs d’invite](#prompt-values).                                                                                                                           |
| `language` | La langue à laquelle la personnalisation s’applique. Pour obtenir la liste des langues offertes, consultez [Internationalisation de Universal Login](/fr-CA/docs/customize/internationalization-and-localization/universal-login-internationalization). |
| `screen`   | L’écran précis au sein d’une invite.                                                                                                                                                                                                              |
| `key`      | L’identifiant d’un composant texte.                                                                                                                                                                                                               |
| `text`     | Le message affiché à l’utilisateur.                                                                                                                                                                                                               |

<div id="prompt-values">
  ### Valeurs de `prompt`
</div>

Le terme `prompt` désigne une étape précise du flux de connexion. Chaque `prompt` comporte au moins un `screen`, et chaque `screen` comprend un ensemble de composants texte que vous pouvez personnaliser.

Les valeurs de `prompt` disponibles sont :

<div id="variables">
  ## Variables
</div>

Certains écrans contiennent des variables dans le texte qui sont remplacées au moment de l’exécution en fonction du contexte. Comme les variables disponibles diffèrent d’un écran à l’autre, il n’est pas garanti qu’une variable donnée fonctionne ailleurs que sur son écran d’origine.

| Variable                      | Description                                                                                                         |
| ----------------------------- | ------------------------------------------------------------------------------------------------------------------- |
| $&#123;clientName&#125;       | Nom de l’application Auth0                                                                                          |
| $&#123;connectionName&#125;   | Nom de la connexion (p. ex. Google)                                                                                 |
| $&#123;tenantName&#125;       | Nom du locataire Auth0                                                                                              |
| $&#123;organizationName&#125; | Nom de l’organisation de l’utilisateur si celui-ci s’authentifie dans le contexte d’une Organisation                |
| $&#123;companyName&#125;      | Nom du locataire Auth0, ou nom de l’organisation si l’utilisateur s’authentifie dans le contexte d’une Organisation |
| $&#123;userName&#125;         | Nom de l’utilisateur connecté                                                                                       |
| $&#123;email&#125;            | Courriel de l’utilisateur connecté                                                                                  |
| $&#123;appName&#125;          | Nom de l’application Guardian Push personnalisée                                                                    |
