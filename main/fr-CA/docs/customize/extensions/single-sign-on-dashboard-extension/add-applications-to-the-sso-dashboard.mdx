title: Ajouter des applications au tableau de bord Single Sign-On
---

Utilisez l’extension de tableau de bord Single Sign-On (<Tooltip tip="Single Sign-On (SSO) : service qui, lorsqu’un utilisateur se connecte à une application, le connecte automatiquement aux autres applications." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=SSO">SSO</Tooltip>) pour gérer la connexion SSO de vos utilisateurs dans plusieurs applications d’entreprise.

<div id="prerequisites">
  ## Prérequis
</div>

* [Créer l&#39;application SSO Dashboard](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application).
* [Installer l&#39;extension SSO Dashboard](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).

<div id="add-applications">
  ## Ajouter des applications
</div>

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez votre nouvelle extension SSO Dashboard.
   Si vous êtes administrateur, vous pouvez aussi ouvrir une session dans le SSO Dashboard à l’aide de `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
2. Dans le coin supérieur droit, sélectionnez **Settings** dans le menu déroulant sous le nom de votre locataire.
3. Cliquez sur **Create App** pour ajouter une nouvelle application.

   Le formulaire **New Application** s’affiche.
4. Remplissez les champs suivants pour la nouvelle application :

   | Champ           | Description                                                                                                                                                                                                                                                                                                                                 |
   | --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Type**        | Une liste déroulante où vous sélectionnez SAML, OpenID-Connect ou WS-Federation selon le type d’application.                                                                                                                                                                                                                                |
   | **Application** | Une liste déroulante où vous sélectionnez l’application à ajouter.                                                                                                                                                                                                                                                                          |
   | **Name**        | Le nom est rempli automatiquement en fonction de l’application sélectionnée. Vous pouvez le modifier ou utiliser la valeur par défaut.                                                                                                                                                                                                      |
   | **Logo**        | L’URL du logo que vous souhaitez utiliser comme icône pour l’application.                                                                                                                                                                                                                                                                   |
   | **Callback**    | Une des **Allowed Callback URLs** dans les [Application Settings](https://manage.auth0.com/#/applications) de l’application.                                                                                                                                                                                                                |
   | **Connection**  | (*Facultatif*) Le type de connexion. Vous pouvez ajouter ou modifier les types de connexion disponibles dans la [section Connections du tableau de bord de gestion Auth0](https://manage.auth0.com/#/connections/database). Si aucune connexion n’est définie et que l’utilisateur n’est pas connecté, il verra la page de connexion Auth0. |
   | **Custom URL**  | (*Facultatif*) Case à cocher pour utiliser une URL personnalisée au lieu de l’URL Auth0. Si vous cochez la case, un champ s’affiche dans lequel vous pouvez saisir votre URL personnalisée.                                                                                                                                                 |
   | **Enabled**     | Case à cocher pour rendre cette application visible (publiée) pour vos utilisateurs.                                                                                                                                                                                                                                                        |
5. Sélectionnez **Create**. Votre nouvelle application apparaîtra ensuite sur la page **Applications** du SSO Dashboard avec toutes les autres applications créées.
6. Cliquez sur une application pour tester la connexion.
