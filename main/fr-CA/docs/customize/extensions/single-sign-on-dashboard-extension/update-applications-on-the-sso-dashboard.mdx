title: Mettre à jour des applications dans le SSO Dashboard
---

Utilisez l’extension SSO Dashboard pour gérer la connexion <Tooltip tip="Single Sign-On (SSO) : service permettant, après la connexion d’un utilisateur à une application, de le connecter automatiquement à d’autres applications." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=SSO">SSO</Tooltip> pour vos utilisateurs dans plusieurs applications d’entreprise.

Pour en savoir plus, consultez [Single Sign-On Dashboard Extension](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension) ou [consultez cette extension sur GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).

<div id="prerequisites">
  ## Prérequis
</div>

* [Créez l’application du SSO Dashboard](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application) dans Auth0.
* [Installez l’extension de SSO Dashboard](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).
* [Ajoutez des applications au SSO Dashboard](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard).

<div id="update-application">
  ## Mettre à jour l’application
</div>

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) et sélectionnez la vue **Extensions installées**.
2. Sélectionnez **SSO Dashboard**.
   Si vous êtes administrateur, vous pouvez aussi vous connecter au SSO Dashboard à l’adresse `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
3. Sélectionnez **Paramètres** dans le menu déroulant sous le nom de votre locataire.

* Pour indiquer si les utilisateurs peuvent voir l’application lorsqu’elle est activée, sélectionnez **Publier** ou **Dépublier**.
* Pour mettre à jour les paramètres d’une application, sélectionnez son icône d’engrenage.
* Pour supprimer une application, sélectionnez son **X**. Une fenêtre de confirmation s’affichera pour confirmer la suppression.
