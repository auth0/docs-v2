L’extension Auth0 AD/LDAP Connector Health Monitor expose un point de terminaison API de votre choix pour vous permettre de surveiller vos connecteurs AD/LDAP.

<div id="configure-the-extension">
  ## Configurer l’extension
</div>

Pour installer et configurer cette extension :

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez **Auth0 AD/LDAP Connector Health Monitor**. La fenêtre **Install Extension** s’ouvre.
2. Configurez les paramètres suivants, puis sélectionnez **Install** :

* **Auth0&#95;Domain**: Le domaine de votre application Auth0. Vous le trouverez dans les [paramètres de votre application Auth0](https://manage.auth0.com/#/applications/\{yourClientId}/settings).
* **Auth0&#95;Global&#95;Client&#95;ID**: L’<Tooltip tip="ID client : valeur d’identification attribuée par Auth0 à votre ressource enregistrée." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Client+ID">ID client</Tooltip> global de votre application Auth0. Vous le trouverez dans la section **Global Application Information** de vos [paramètres avancés du locataire](https://manage.auth0.com/#/tenant/advanced).
* **Auth0&#95;Global&#95;Client&#95;Secret**: Le <Tooltip tip="Secret client : secret utilisé par une application pour s’authentifier auprès du serveur d’autorisation; il ne doit être connu que de l’application et du serveur d’autorisation, et doit être suffisamment aléatoire pour ne pas pouvoir être deviné." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Client+Secret">Secret client</Tooltip> global de votre application Auth0. Vous le trouverez dans la section **Global Application Information** de vos [paramètres avancés du locataire](https://manage.auth0.com/#/tenant/advanced).

<div id="use-the-extension">
  ## Utiliser l’extension
</div>

Pour obtenir l’URL à utiliser avec votre service de surveillance :

1. Accédez à [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions), puis cliquez sur **Installed Extensions**.

2. Sélectionnez **Auth0 AD/LDAP Connector Health Monitor**. Une nouvelle fenêtre s’ouvre avec un extrait JSON semblable à celui-ci :

   ```json lines
   {
     "message": "Use this url: 'https://sandbox.it.auth0.com/api/run/auth0user/auth0-ldap-conector-health-monitor?connection={MY-LDAP-CONNECTOR}' for monitoring your AD/LDAP connector."
   }
   ```

3. Saisissez l’ID de votre connexion LDAP. Pour obtenir cet ID, accédez à  [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), cliquez sur la connexion, puis copiez la valeur du champ **Identifier**.

Une fois cette étape terminée, vous pouvez utiliser l’URL fournie avec votre service de surveillance pour surveiller vos connecteurs AD/LDAP.
