Le tableau de bord <Tooltip tip="Authentification unique (SSO) : service qui, après la connexion d’un utilisateur à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Single+Sign-on">d’authentification unique</Tooltip> (SSO) est une application Web conçue pour résoudre un problème bien connu. Les organisations de toutes tailles utilisent diverses applications pour gérer différentes fonctions d’affaires, comme la comptabilité, les RH, le développement, le soutien, etc. Il peut être fastidieux de se souvenir des noms d’utilisateur, des mots de passe et des URL de connexion de toutes vos applications. Avec cette extension, vous pouvez simplifier l’expérience d’authentification en activant la connexion SSO pour vos utilisateurs sur plusieurs applications d’entreprise. Elle vous permet de créer une liste de tous les services infonuagiques auxquels un utilisateur peut accéder au moyen du SSO.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Un utilisateur ne doit pas pouvoir accéder, à partir du tableau de bord ou autrement, à un fournisseur de services sans disposer des autorisations appropriées (groupes/rôles). Idéalement, ces utilisateurs ne devraient voir dans le tableau de bord aucun fournisseur de services auquel l’accès ne leur a pas été accordé.
</Callout>

Le tableau de bord SSO prend en charge deux types d’utilisateurs :

* **Utilisateurs** qui peuvent se connecter au tableau de bord pour sélectionner une application à laquelle se connecter avec le SSO.
* **Administrateurs** qui peuvent configurer les applications visibles par les utilisateurs.

Pour configurer cette extension, procédez comme suit :

1. [Créer une application SSO Dashboard](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)
2. [Installer l’extension SSO Dashboard](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)
3. [Ajouter des applications au tableau de bord SSO](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)
4. [Mettre à jour les applications du tableau de bord SSO](/fr-CA/docs/customize/extensions/single-sign-on-dashboard-extension/update-applications-on-the-sso-dashboard)

Vous pouvez aussi [consulter cette extension SSO Dashboard sur GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).
