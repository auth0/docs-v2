---
description: Découvrez comment modifier les rôles des utilisateurs dans Auth0 Dashboard pour votre locataire Auth0.
title: Modifier les membres du locataire
---

Si vous êtes administrateur du locataire, vous pouvez modifier le ou les rôles attribués à un utilisateur. Les changements prennent effet une fois que les utilisateurs ont actualisé leur Auth0 Dashboard.

Vous pouvez attribuer plus d’un rôle à un utilisateur; les autorisations seront alors combinées. Vous devez d’abord avoir ajouté le membre du locataire.

1. Accédez à [Dashboard &gt; Settings &gt; Tenant Members](https://manage.auth0.com/#/tenant/admins).
2. Cliquez sur **...** à droite du nom ou du courriel d’un utilisateur, puis cliquez sur **Edit Roles**.
   Si vous ne voyez pas cette option, cela signifie que l’utilisateur n’a pas encore accepté son invitation.
3. Sélectionnez des rôles supplémentaires ou supprimez des rôles.
4. Cliquez sur **Save**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Les autorisations de l’utilisateur seront visibles lorsqu’il actualisera son Auth0 Dashboard. Si l’utilisateur ne l’actualise pas, il pourrait voir des erreurs indiquant qu’il n’a pas accès à une section précise. Si l’une de ces erreurs se produit, l’utilisateur devra se déconnecter, puis se reconnecter.
</Callout>

**Réinitialisation du mot de passe** : Pour en savoir plus, consultez [Reset Auth0 Account Password](/fr-CA/docs/troubleshoot/customer-support/reset-account-passwords) si vous ou l’un des utilisateurs de votre Auth0 Dashboard éprouvez des problèmes de connexion.

**Adresse de courriel invalide** : Pour en savoir plus, consultez [Update Dashboard User Email Addresses](/fr-CA/docs/get-started/manage-dashboard-access/update-dashboard-user-email) pour afficher les adresses courriel des membres de votre locataire et les mettre à jour au besoin.
