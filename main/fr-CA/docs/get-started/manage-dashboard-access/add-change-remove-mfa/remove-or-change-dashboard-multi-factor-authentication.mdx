title: Supprimer ou modifier l’authentification multifacteur de l’Auth0 Dashboard
---

Les utilisateurs de l’Auth0 Dashboard peuvent supprimer ou modifier les facteurs d’<Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=multi-factor+authentication">authentification multifacteur</Tooltip> (MFA) qui ne sont plus disponibles ou dont ils ne veulent plus. Pour en savoir plus sur le MFA pour les utilisateurs de l’Auth0 Dashboard, consultez [Authentification multifacteur pour les utilisateurs de l’Auth0 Dashboard](/fr-CA/docs/get-started/manage-dashboard-access/add-change-remove-mfa).

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 recommande les facteurs WebAuthn comme méthodes d’authentification les plus sûres et les plus faciles à utiliser. Pour en savoir plus, consultez [Authentification FIDO avec WebAuthn](/fr-CA/docs/secure/multi-factor-authentication/fido-authentication-with-webauthn).
</Callout>

<Warning>
  Les administrateurs doivent activer au moins un facteur pour utiliser le MFA. Auth0 recommande fortement de configurer plusieurs facteurs afin que vous puissiez quand même accéder à votre compte si vous perdez votre appareil principal.

  Une configuration idéale consiste à utiliser trois facteurs :

  1. WebAuthn, Guardian ou OTP comme méthode principale.
  2. Un ou plusieurs numéros de téléphone pour les SMS comme solution de secours (disponible uniquement pour les locataires associés à un forfait d’abonnement).
  3. Un code de récupération.

  Si vous ne pouvez pas fournir votre jeton MFA et que vous ne disposez pas de méthodes de secours appropriées, votre compte pourrait être irrécupérable.
</Warning>

<div id="remove-or-change-an-mfa-factor-from-the-dashboard">
  ## Supprimer ou changer un facteur MFA dans l’Auth0 Dashboard
</div>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Vous ne pouvez pas ajouter un facteur de biométrie de l’appareil (comme la Touch Bar d’un MacBook, Windows Hello, Touch ID ou Face ID sur iOS, ou la reconnaissance faciale ou par empreinte digitale sur Android) depuis l’Auth0 Dashboard. Pour savoir comment ajouter un facteur d’authentification par biométrie de l’appareil, consultez [Ajouter l’authentification multifacteur pour les utilisateurs du Auth0 Dashboard.](/fr-CA/docs/get-started/manage-dashboard-access/add-change-remove-mfa/add-mfa)
</Callout>

Les utilisateurs du Auth0 Dashboard qui peuvent ouvrir une session avec leurs facteurs MFA actuels peuvent suivre ces étapes :

1. Dans le coin supérieur droit de l’Auth0 Dashboard, cliquez sur votre nom d’utilisateur, puis sur [Votre profil](https://manage.auth0.com/#/profile).

   <Frame>![Dashboard - Profil - Authentification multifacteur](/docs/images/cdy7uua7fh8z/2NyLNvP92RruwZ46dLPm5x/8ece3df2b550af8dd78368ca36d8a661/MFA.png)</Frame>
2. Repérez le nouveau facteur d’authentification que vous voulez utiliser, puis cliquez sur **+ ADD** sur cette ligne. Suivez les instructions à l’écran pour terminer l’inscription.
3. Toujours dans [Votre profil](https://manage.auth0.com/#/profile), repérez le facteur d’authentification que vous ne voulez plus utiliser, puis cliquez sur **REMOVE**.
4. Cliquez sur **Yes** pour confirmer la suppression.
5. Auth0 vous demande de vous authentifier à l’aide de vos facteurs actuels (anciens). Une fois l’authentification réussie, Auth0 supprime le facteur.

<div id="remove-or-change-a-lost-mfa-factor">
  ## Supprimer ou modifier un facteur MFA perdu
</div>

Les utilisateurs du Auth0 Dashboard qui **ne peuvent pas** se connecter avec leurs facteurs MFA actuels peuvent suivre les étapes suivantes :

1. Essayez de vous connecter au Auth0 Dashboard. Auth0 vous invite à vous authentifier avec vos facteurs actuels.
2. Quand Auth0 vous demande l’appareil ou les identifiants que vous avez perdus, cliquez sur **Essayer une autre méthode**.
3. Dans la zone Other Methods, cliquez sur une autre méthode pour vous authentifier.
4. Connectez-vous pour accéder au Auth0 Dashboard.
5. Dans le coin supérieur droit du Auth0 Dashboard, cliquez sur votre nom d’utilisateur, puis sur [Votre profil](https://manage.auth0.com/#/profile).

   <Frame>![Dashboard - Profil - Authentification multifacteur](/docs/images/cdy7uua7fh8z/2NyLNvP92RruwZ46dLPm5x/8ece3df2b550af8dd78368ca36d8a661/MFA.png)</Frame>
6. Repérez la méthode d’authentification que vous ne pouvez plus utiliser, puis cliquez sur **REMOVE**.
7. Cliquez sur **Yes** pour confirmer la suppression.
8. Auth0 vous invite de nouveau à vous authentifier à l’aide de vos facteurs actuels. Répétez les étapes 2 à 4 pour vérifier votre identité. Auth0 supprime le facteur perdu.

<div id="get-help-from-auth0-support">
  ## Obtenir de l’aide auprès de l’assistance Auth0
</div>

Si vous êtes bloqué et que vous n’avez accès à aucun de vos facteurs MFA activés, rien ne garantit que vous pourrez récupérer l’accès à votre compte. Un autre administrateur doit soumettre un ticket d’assistance Auth0 en votre nom. Dans certains cas, Auth0 peut vérifier la demande et procéder à une réinitialisation de la MFA. Cependant, il se peut que nous ne soyons pas en mesure de vérifier que le compte vous appartient. C’est pourquoi il est si important d’activer plusieurs facteurs différents.

L’assistance Auth0 ne réinitialise pas les comptes des **utilisateurs finaux**. Vous êtes responsable des comptes qui accèdent à vos applications et à vos API. Pour en savoir plus sur les comptes d’utilisateurs finaux, consultez [Manage Users](/fr-CA/docs/manage-users).
