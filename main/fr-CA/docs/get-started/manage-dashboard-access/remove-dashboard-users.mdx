---
description: "Découvrez comment supprimer l’accès des membres du locataire à l’Auth0 Dashboard de votre locataire Auth0."
title: "Supprimer des membres du locataire"
---

Si vous êtes administrateur d’un locataire, vous pouvez révoquer l’accès à l’Auth0 Dashboard.

1. Accédez à [Auth0 Dashboard &gt; Paramètres &gt; Membres du locataire](https://manage.auth0.com/#/tenant/admins).
2. Cliquez sur **...** à droite du nom ou du courriel d’un utilisateur, puis cliquez sur **Supprimer**.
3. Cliquez sur **Enregistrer**.
