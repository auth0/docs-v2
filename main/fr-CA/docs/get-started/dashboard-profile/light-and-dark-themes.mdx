---
description: Comment basculer entre les thèmes clair et sombre pour votre Auth0 Dashboard
title: Thèmes clair et sombre
---

Les clients d’Auth0 peuvent utiliser le mode clair ou le mode sombre dans l’Auth0 Dashboard de leur locataire. Vous pouvez également configurer votre Auth0 Dashboard pour qu’il utilise les mêmes paramètres que votre système local.

Il existe deux façons de modifier le thème de votre Auth0 Dashboard :

1. Sélectionnez le menu déroulant de profil en haut à droite de votre Auth0 Dashboard.
2. Utilisez le menu pour passer d’un thème à l’autre.

<Frame>![Où trouver l’option pour basculer entre le mode sombre et le mode clair](/docs/images/cdy7uua7fh8z/1BImxFvHXoULQVTkqCsQL0/a37f6cfdfce935fd20731aa68f8067de/2024-09-10_09-08-27.png)</Frame>

Vous pouvez aussi cliquer sur **Votre profil** dans le même menu déroulant et faire défiler jusqu’en bas, où vous pourrez choisir parmi les trois thèmes.

<Frame>![Où modifier votre thème dans la section Profil](/docs/images/cdy7uua7fh8z/231W9kPYTeEABOsV3rOZxF/1c451d4942d86432a0e701f497c32f24/Light_mode_profile_section.png)</Frame>
