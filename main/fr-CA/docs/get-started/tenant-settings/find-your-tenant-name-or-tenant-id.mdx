title: Trouver le nom ou l’ID de votre locataire
---

Chaque locataire Auth0 a un nom. Vous utilisez souvent ce nom dans votre code pour identifier le locataire. Lorsque vous voyez la variable `{yourTenantId}`, `{yourTenantName}`, `{yourTenant}` ou une variable semblable dans la documentation Auth0, remplacez-la par le nom du locataire.

1. Accédez à l’[Auth0 Dashboard](https://manage.auth0.com/#).
2. Dans le coin supérieur gauche, repérez le nom du locataire à côté du logo Auth0.
