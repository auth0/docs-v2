L’[Auth0 Dashboard](https://manage.auth0.com/#) est l’endroit où vous gérez tous les aspects de votre abonnement et de votre configuration Auth0.

<Frame>![Page Activity de l’Auth0 Dashboard](/docs/images/cdy7uua7fh8z/6k0cRLNEPko6vY6Kw7nIQD/4590d1b9d3e79f8d9c9970d733b08d02/2025-01-28_14-34-41.png)</Frame>

Il comporte plusieurs sections, accessibles à partir du menu latéral de gauche.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Pour connaître les bonnes pratiques d’utilisation du Teams Dashboard, [consultez Bonnes pratiques générales d’utilisation et d’exploitation](/fr-CA/docs/troubleshoot/general-usage-and-operations-best-practices).
</Callout>

<div id="configure-implementation">
  ## Configurer l’implémentation
</div>

Le tableau suivant présente un bref aperçu des différentes sections de l’Auth0 Dashboard et de ce que vous pouvez faire dans chacune d’elles.

| Section             | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| ------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Applications**    | Gérez vos applications, vos API et vos intégrations d’authentification unique (SSO).<br />**Applications** : pour chacune de vos applications pour lesquelles vous souhaitez authentifier des utilisateurs avec Auth0, enregistrez une application.<br />**APIs** : pour chacune de vos API que vous souhaitez sécuriser avec Auth0, enregistrez une API. Créez de nouvelles API et gérez celles qui existent déjà.<br />**SSO Integrations** : affichez et activez des services externes pour le SSO. Créez de nouvelles intégrations SSO et configurez, passez en revue et gérez leurs paramètres.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Authentication**  | Gérez les fournisseurs d’identité par lesquels vous autorisez les utilisateurs à s’authentifier à vos applications.<br />**Database** : stockez et gérez de façon sécurisée les identifiants et mots de passe soit dans le stockage de données Auth0, soit dans votre propre base de données. Connectez-vous à des bases de données existantes à l’aide de scripts JavaScript basés sur des modèles qui s’exécutent sur le serveur d’Auth0 lors de chaque authentification. Migrez graduellement vers Auth0 une base de données existante d’anciens identifiants à mesure que les utilisateurs s’authentifient (aucune réinitialisation du mot de passe requise).<br />**Social** : configurez les fournisseurs d’identité sociaux (comme Facebook, X et GitHub) par lesquels vos utilisateurs peuvent se connecter.<br />**Enterprise** : configurez les fournisseurs d’identité d’entreprise (comme Active Directory, SAML et Office 365) par lesquels vos utilisateurs peuvent se connecter à l’aide de leurs identifiants d’entreprise.<br />**Passwordless** : permettez à vos utilisateurs de s’inscrire et de se connecter à l’aide de codes à usage unique (envoyés par courriel ou SMS) ou de liens en un clic, plutôt qu’avec des mots de passe. |
| **Organizations**   | Gérez les organisations avec lesquelles vous faites affaire et personnalisez l’expérience de leurs utilisateurs lorsqu’ils accèdent à vos applications.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **User Management** | Gérez les identités et les autorisations de vos utilisateurs.<br />**Users** : affichez et créez des profils utilisateur, effectuez des réinitialisations du mot de passe, bloquez et supprimez des utilisateurs, et plus encore.<br />**Roles** : créez et gérez des rôles pour vos applications. Les rôles contiennent des ensembles d’autorisations et peuvent être attribués à des utilisateurs.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Branding**        | **Universal Login** : créez et personnalisez une page de connexion vers laquelle vous pouvez diriger les utilisateurs pour qu’ils s’authentifient.<br />**Custom Domains** : créez un domaine personnalisé afin d’offrir une expérience cohérente à vos utilisateurs.<br />**Email Templates** : utilisez des modèles pour créer des flux de travail par courriel de bienvenue, de réinitialisation du mot de passe et de vérification de compte.<br />**Email Provider** : indiquez et configurez les renseignements de votre fournisseur de courriel personnalisé.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Security**        | Configurez des couches de sécurité supplémentaires en activant des mécanismes de protection contre différents types d’attaques et les anomalies d’accès des utilisateurs.<br />**Attack Protection** : gérez les paramètres liés aux bots, à la limitation par IP, aux attaques par force brute et aux mots de passe compromis.<br />**Multi-factor Auth** : exigez des facteurs supplémentaires pendant le processus de connexion afin d’empêcher les accès non autorisés.<br />**Monitoring** : surveillez les événements de renseignement sur les menaces à l’aide de l’une de nos intégrations de visualisation de données et d’alerte.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                |
| **Actions**         | Configurez des flux comme la connexion, le machine-à-machine, l’inscription d’utilisateurs et les réinitialisations du mot de passe. Créez et gérez des actions personnalisées utilisées dans les flux.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **Auth Pipeline**   | **Rules** : configurez des extraits JavaScript personnalisés qui s’exécutent dans Auth0 dans le cadre de chaque transaction d’authentification utilisateur. Vous pouvez appeler des API externes, filtrer les utilisateurs autorisés à se connecter à votre application, utiliser une AllowList, configurer un accès géolocalisé, etc.<br />**Hooks** : personnalisez le comportement d’Auth0 lorsque vous utilisez des connexions de base de données en configurant du code Node.js qui s’exécute sur des points d’extensibilité (comparables à des webhooks fournis avec un serveur).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    |
| **Monitoring**      | **Logs** : 	affichez les données de journalisation des actions effectuées dans l’Auth0 Dashboard par les administrateurs ainsi que les connexions des utilisateurs.<br />**Streams** : créez et gérez la diffusion en continu des événements de journal vers des services externes d’analyse de données.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   |
| **Marketplace**     | Explorez les intégrations qui aident votre entreprise à tirer davantage parti d’Auth0.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                     |
| **Extensions**      | Étendez la plateforme Auth0 avec des options officielles et tierces.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |
| **Settings**        | Configurez vos locataires, gérez vos options d’abonnement et de paiement Auth0, et contrôlez les administrateurs de vos locataires ainsi que les autres rôles utilisateur. Gérez d’autres paramètres des locataires liés à vos domaines personnalisés, à vos clés de signature et à d’autres paramètres avancés.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
| **Get Support**     | Accédez à notre [Support Center](https://support.auth0.com). Si votre forfait ne donne pas accès aux services d’assistance, consultez la [Auth0 Community](https://community.auth0.com/).                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                  |

<div id="manage-account-settings">
  ## Gérer les paramètres du compte
</div>

En haut à gauche, vous pouvez voir le nom et l’icône de votre locataire, ainsi qu’une petite flèche. Cette flèche affiche un menu déroulant que vous pouvez utiliser pour configurer différents aspects de votre compte :

* [**Paramètres**](/fr-CA/docs/get-started/tenant-settings) : Configurez plusieurs aspects de votre locataire.
* [**Inviter un membre**](/fr-CA/docs/get-started/manage-dashboard-access) : Ajoutez un utilisateur supplémentaire en tant qu’Administrateur ou avec un autre rôle à la configuration de votre locataire.
* [**Utilisation du quota**](https://support.auth0.com/reports/quota) : Consultez les renseignements sur l’utilisation du quota de votre abonnement et de vos locataires.
* [**Créer un locataire**](/fr-CA/docs/get-started/auth0-overview/create-tenants) : Utilisez cette option pour créer un nouveau locataire.
* **Changer de locataire** : Si vous avez [plusieurs locataires](/fr-CA/docs/get-started/auth0-overview/create-tenants/create-multiple-tenants), utilisez cette option pour passer de l’un à l’autre. Si vous créez une application pour un locataire, elle n’apparaîtra pas dans la liste d’un autre locataire.

En haut à droite, vous voyez plusieurs options sur lesquelles vous pouvez cliquer :

* **Recherche** : Recherchez des applications, des éléments de la Marketplace, des utilisateurs et d’autres contenus liés à votre locataire.
* **Discutez de vos besoins :** Communiquez avec un expert Auth0 pour vous aider à atteindre vos objectifs avec Auth0.
* **Docs :** Un lien vers le site de documentation que vous consultez en ce moment.
* **Icône de notifications :** Vous informe des nouvelles communications d’Auth0 ou des changements apportés aux paramètres de votre locataire.
* **Profil :** Un lien vers le profil de votre locataire actuel, ainsi que des options pour [modifier votre thème](/fr-CA/docs/get-started/dashboard-profile/light-and-dark-themes) et vous déconnecter.
