Accédez aux informations sur l’équipe et modifiez-les depuis votre locataire. Si aucun locataire n’est encore associé à votre compte, consultez [Créer des locataires.](/fr-CA/docs/get-started/auth0-overview/create-tenants)

<div id="access-the-teams-dashboard">
  ## Accéder au tableau de bord Teams
</div>

Si vous êtes propriétaire d’équipe, vous pouvez utiliser le tableau de bord Teams pour :

* Choisir qui peut créer de nouveaux locataires : les propriétaires d’équipe ou tous les administrateurs de locataire.
* Voir tous les locataires créés dans le cadre de votre abonnement Auth0 ou de votre entente personnalisée.
* Voir les propriétaires d’équipe et les inviter.

Pour accéder au tableau de bord Teams à partir de n’importe lequel de vos locataires :

1. Sélectionnez le nom de votre locataire dans le menu déroulant des locataires de l’[Auth0 Dashboard](https://manage.auth0.com/dashboard).
2. Repérez le nom de votre équipe au bas du menu et sélectionnez **Accéder à l’équipe**.

<Frame>![Accéder au tableau de bord d’équipe Auth0](/docs/images/cdy7uua7fh8z/3207T6ELEBc56dLZT84hjT/13779947b38dd52ab4694b3869271961/Tenant_Management_-_Teams.png)</Frame>

Pour accéder au tableau de bord Teams à l’aide d’une URL :

1. Accédez à `https://accounts.auth0.com/`
2. Si votre compte utilisateur est membre d’une seule équipe, saisissez vos identifiants pour accéder à votre tableau de bord Teams. Si votre compte utilisateur est membre d’une ou de plusieurs équipes, sélectionnez l’équipe dans la liste des équipes auxquelles votre compte utilisateur est associé pour vous connecter.

Si l’option d’accès à Teams n’est pas visible, essayez ce qui suit :

| Cause possible                                   | Action                                                          |
| ------------------------------------------------ | --------------------------------------------------------------- |
| Le locataire actuel ne fait pas partie de Teams. | Essayez de passer à un autre locataire.                         |
| Vous n’êtes pas propriétaire d’équipe.           | Demandez à un propriétaire d’équipe de vous inviter à l’équipe. |

<div id="view-tenants">
  ## Afficher les locataires
</div>

Dans le Teams Dashboard, vous pouvez afficher tous les locataires inclus dans votre entente personnalisée ou votre abonnement Auth0, dans un environnement de cloud public ou privé. Pour chaque locataire, vous pouvez consulter :

* Nom du locataire
* Région
* Date de création
* Environnement (développement, préproduction ou production)
* Administrateurs du locataire

1. Sélectionnez **Locataires** dans la partie gauche du Teams Dashboard.

   Pour les environnements de cloud privé, sélectionnez le menu déroulant **Overview** pour afficher l’option de menu **Locataires**.
2. Si vous êtes administrateur d’un locataire, sélectionnez le nom du locataire pour y accéder. Si vous n’êtes pas administrateur de ce locataire, vous ne pourrez pas le sélectionner.
3. Pour consulter les membres du locataire, sélectionnez **View Members**.

<div id="view-private-cloud-environments">
  ## Afficher les environnements de nuage privé
</div>

Vous pouvez consulter la liste des environnements de nuage privé à partir du tableau de bord Teams. Vous pouvez y voir les renseignements suivants

* Nom de l’environnement
* Type de fournisseur infonuagique (AWS ou Azure)
* Région de déploiement
* Région de basculement
* Version actuelle de l’environnement.

1. Sélectionnez le menu déroulant **Overview** pour afficher l’option de menu **Private Cloud**.

<div id="link-tenants">
  ## Lier des locataires
</div>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Cette fonctionnalité est actuellement offerte pour les types d’abonnement Team suivants :

  * B2C - Essentials
  * B2B - Essentials
  * B2C - Professional
  * B2B - Professional
</Callout>

La liaison de locataires vous permet de partager avec le locataire lié les fonctionnalités et les limites de quota comprises dans votre abonnement Team. Une fois la liaison établie, les abonnements de ce locataire seront rattachés à votre équipe et vous n’aurez plus à gérer individuellement l’abonnement de ce locataire.

Vous pouvez lier des locataires à votre équipe si les conditions suivantes sont remplies :

* Le locataire ne fait pas déjà partie d’une équipe.
* Vous êtes administrateur du locataire que vous souhaitez lier.
* Vous avez le rôle de propriétaire de l’équipe.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  La liaison d’un locataire annule l’abonnement associé à ce locataire.
</Callout>

Si ces conditions sont remplies, le locataire s’affichera comme option de liaison à votre équipe lorsque vous sélectionnerez **Link Existing Tenant** dans la page Locataires.

<div id="create-new-tenant">
  ## Créer un nouveau locataire
</div>

<Warning>
  Cette fonctionnalité est actuellement offerte pour les types d’abonnement Team suivants :

  * B2C - Essentials
  * B2B - Essentials
  * B2C - Professional
  * B2B - Professional
  * Enterprise Private Cloud

  En utilisant cette fonctionnalité, vous acceptez les conditions applicables de l’essai gratuit figurant dans le [Master Subscription Agreement](https://www.okta.com/legal/) d’Okta.
</Warning>

Les propriétaires d’équipe peuvent créer un locataire depuis le tableau de bord Teams, ce qui lie automatiquement le locataire créé à l’équipe et associe les détails de l’abonnement Team (fonctionnalités offertes et limites de quota) au nouveau locataire.

Vous pouvez le faire en cliquant sur le bouton **Créer un nouveau locataire** dans la page Locataires, ce qui ouvrira le formulaire Créer un locataire.

<div id="manage-tenants">
  ## Gérer les locataires
</div>

Vous pouvez contrôler qui peut créer de nouveaux locataires dans le cadre de votre entente personnalisée ou de votre abonnement Auth0 sous **Paramètres de l’équipe**. Lors de la création d’un nouveau locataire, les administrateurs peuvent :

* Associer le locataire à l’entente personnalisée. Le nouveau locataire relève alors de votre entente personnalisée ou de votre abonnement. Il bénéficie des mêmes fonctionnalités, est pris en compte dans les limites et est visible par les propriétaires de l’équipe.
  ou
* Créer un locataire sous leur compte personnel. Le nouveau locataire ne fait pas partie de l’entente personnalisée ou de l’abonnement de votre équipe. L’administrateur qui crée le locataire peut y accéder avec les mêmes identifiants de connexion.

Pour permettre aux administrateurs d’associer un locataire à une entente personnalisée, le propriétaire de l’équipe doit :

1. Sélectionner **Paramètres** dans la partie gauche du tableau de bord Teams.
2. Sélectionner l’option **Autoriser les administrateurs de locataire à créer des locataires sous le compte Teams**. Si cette option n’est pas cochée, les administrateurs de locataire peuvent créer seulement des comptes personnels.

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     L’option **Création de locataire** ne s’applique pas aux propriétaires de l’équipe. Les propriétaires sont autorisés à créer de nouveaux locataires dans l’instance Teams dans le cadre de l’entente personnalisée ou de l’abonnement.
   </Callout>
3. Sélectionner **Enregistrer**.

   <Warning>
     Si un administrateur de locataire sans autorisation tente de créer des locataires, il sera invité à communiquer avec le propriétaire de l’équipe.
   </Warning>
