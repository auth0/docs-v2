title: À propos du Quarterly Snapshot
---

Le rapport Quarterly Snapshot présente les points saillants de l’utilisation et de la valeur à l’échelle du compte, y compris des renseignements agrégés sur votre consommation par rapport au quota, votre environnement d’identité et la détection des menaces.

Cette fonctionnalité est activée automatiquement lors de la mise en service d’Auth0 Teams. Elle est offerte exclusivement à certains clients Enterprise pris en charge par leur équipe de compte.

<div id="things-to-know">
  ## Points à savoir
</div>

* Le Quarterly Snapshot est accessible uniquement par l’entremise d’[Auth0 Teams](/fr-CA/docs/get-started/auth0-teams). Auth0 Teams doit être activé pour le compte avant le début du trimestre civil.
* Le Quarterly Snapshot est une fonctionnalité accessible à un groupe restreint de clients d’entreprise gérés par le DTAM.
* Les Team Owners et les lecteurs de rapports peuvent consulter et télécharger le Quarterly Snapshot dans la section **Reports** du tableau de bord Auth0 Teams.
* Les Team Owners peuvent gérer la liste des lecteurs de rapports dans le tableau de bord Auth0 Teams.
* Les objets de métriques dans les Quarterly Snapshots peuvent varier selon les droits du compte, les fonctionnalités activées et l’utilisation.
* Toutes les métriques sont agrégées au niveau du compte. Pour en savoir plus, consultez le tableau **Metrics** ci-dessous.

<div id="metrics">
  ## Métriques
</div>

Voici les métriques que vous pourriez voir dans votre Quarterly Snapshot.

| Métrique                                      | Ce qu’elle mesure                                                                                                                                                                                                                                                                                                                                                             | Commentaires                                                                                                                                                       |
| --------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| Utilisateurs actifs mensuels                  | Mesure le nombre mensuel d’utilisateurs actifs uniques sur le compte par rapport à votre quota.                                                                                                                                                                                                                                                                               | Pour obtenir des renseignements détaillés au niveau d’un locataire, veuillez consulter le rapport [Support Center Quota](https://support.auth0.com/reports/quota). |
| Jetons machine à machine                      | Affiche le nombre de jetons d’accès émis par Auth0 pour l’[octroi des identifiants de l’application](/fr-CA/docs/get-started/authentication-and-authorization-flow/client-credentials-flow). Les jetons émis pour Auth0 Management API ou d’autres API intégrées d’Auth0 ne sont pas comptabilisés.                                                                                 | Pour obtenir des renseignements détaillés au niveau d’un locataire, veuillez consulter le rapport [Support Center Quota](https://support.auth0.com/reports/quota). |
| Connexions d’entreprise actives               | Cette métrique affiche le nombre de connexions d’entreprise utilisées par un compte.                                                                                                                                                                                                                                                                                          | Pour plus de détails sur votre quota, communiquez avec votre chargé de compte.                                                                                     |
| Inscriptions de nouveaux utilisateurs         | Indique combien de nouveaux utilisateurs (inscriptions d’utilisateurs) ont été ajoutés à votre ou vos locataires au cours des mois de ce trimestre.                                                                                                                                                                                                                           | Consultez la [page Activity](/fr-CA/docs/get-started/auth0-overview/dashboard/activity) pour en savoir plus sur les nouveaux utilisateurs.                               |
| Utilisateurs inactifs                         | Cette métrique représente la moyenne mensuelle des utilisateurs qui ne se sont pas authentifiés au cours du dernier mois.                                                                                                                                                                                                                                                     |                                                                                                                                                                    |
| Mot de passe compromis                        | Nombre de tentatives bloquées en raison de justificatifs compromis par la fonctionnalité [détection des mots de passe compromis](/fr-CA/docs/secure/attack-protection/breached-password-detection).                                                                                                                                                                                 |                                                                                                                                                                    |
| Protection contre la force brute              | Requêtes bloquées par la fonctionnalité [protection contre la force brute](/fr-CA/docs/secure/attack-protection/brute-force-protection), si elle est activée. Cette métrique résume toutes les requêtes bloquées spécifiquement sur les locataires de production.                                                                                                                   |                                                                                                                                                                    |
| Limitation du débit des adresses IP suspectes | Requêtes bloquées par la fonctionnalité [Limitation du débit des adresses IP suspectes](/fr-CA/docs/secure/attack-protection/suspicious-ip-throttling), si elle est activée. Cette métrique totalise toutes les requêtes bloquées spécifiquement sur les locataires de production.                                                                                                  |                                                                                                                                                                    |
| Détection des bots                            | Requêtes qui n’ont pas réussi le défi CAPTCHA. Cette métrique totalise toutes les requêtes de bots bloquées par la fonctionnalité [détection des bots](/fr-CA/docs/secure/attack-protection/bot-detection) après l’échec du défi CAPTCHA.                                                                                                                                           |                                                                                                                                                                    |
| MFA adaptative                                | Requêtes pour lesquelles une demande de vérification [MFA](/fr-CA/docs/secure/multi-factor-authentication/adaptive-mfa) a été déclenchée en raison d’un faible score de confiance. Cette métrique totalise toutes les tentatives de connexion pour lesquelles une vérification d’authentification multifacteur a été déclenchée et dont le score de confiance calculé était faible. |                                                                                                                                                                    |
