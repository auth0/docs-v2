Le serveur Auth0 pour le Model Context Protocol (MCP) connecte les agents d’IA à votre tenant Auth0, ce qui leur permet d’effectuer dans Auth0 des opérations complexes en plusieurs étapes, comme créer des applications, gérer des utilisateurs ou déployer des Actions. Il s’appuie sur le [Model Context Protocol (MCP)](https://modelcontextprotocol.io/introduction), un protocole normalisé qui permet aux agents d’IA de communiquer avec des outils externes et des API afin d’exécuter des tâches en votre nom.

Conçu pour les développeurs qui utilisent des outils propulsés par l’IA comme Cursor, Windsurf ou Claude Desktop, le serveur Auth0 MCP transforme la façon dont ils gèrent leur infrastructure d’identité, en rendant l’utilisation des services Auth0 plus rapide et plus simple.

<div id="authentication-and-credential-security">
  ## Authentification et sécurité des identifiants
</div>

Auth0 MCP Server accorde la priorité à une sécurité renforcée en gérant l’authentification, en validant les requêtes et en assurant une communication sécurisée avec l’API <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip> d’Auth0. Cela comprend :

* [Flux d’autorisation d’appareil OAuth 2.0](/fr-CA/docs/get-started/authentication-and-authorization-flow/device-authorization-flow) : authentifie de façon sécurisée les requêtes auprès d’Auth0.
* [Stockage sécurisé des identifiants](/fr-CA/docs/get-started/applications/rotate-credentials) : les identifiants sont stockés de façon sécurisée dans le trousseau de votre système et ne sont jamais exposés en texte brut.
* [Permissions minimales de l’API](/fr-CA/docs/secure/tokens/access-tokens/management-api-access-tokens) : seules les permissions essentielles sont demandées, dans le strict respect du principe du moindre privilège.

<div id="supported-tools-and-integrations">
  ## Outils et intégrations pris en charge
</div>

Auth0 MCP Server s’intègre actuellement aux [clients MCP](https://modelcontextprotocol.io/clients) suivants, notamment :

* [Claude Desktop](https://claude.ai/download)
* [Cursor](https://www.cursor.com/downloads)
* [Windsurf](https://www.windsurf.com/editor)

D’autres intégrations seront ajoutées à mesure que de nouveaux clients importants feront leur apparition. Pour consulter la liste complète des outils pris en charge et le détail de leurs fonctionnalités, veuillez vous reporter à notre [référence des outils d’Auth0 MCP Server](/fr-CA/docs/get-started/auth0-mcp-server/auth0-mcp-tools-reference).
