description: Découvrez comment activer la prise en charge des liens universels pour votre application Auth0 dans Apple Xcode à l’aide de l’Auth0 Dashboard.
title: Activer la prise en charge des liens universels dans Apple Xcode
---

Les liens universels établissent une relation vérifiée entre les domaines et les applications; les paramètres de votre application Auth0 et ceux de votre application iOS doivent donc correspondre. Pour cela, vous devez fournir à Auth0 l’Apple Team ID et le Bundle Identifier. Vous pouvez activer la prise en charge des liens universels pour votre application enregistrée dans Auth0 à l’aide de l’<Tooltip tip="Auth0 Dashboard : la console principale d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

<div id="obtain-apple-team-id-and-bundle-identifier">
  ## Obtenir le Apple Team ID et le Bundle Identifier
</div>

1. Pour trouver votre **Apple Team ID**, accédez à la [page récapitulative de votre compte développeur Apple](https://developer.apple.com/membercenter/index.action#accountSummary).
2. Pour trouver le **Bundle Identifier** de votre application iOS, accédez à la page des [réglages du projet dans Xcode](https://developer.apple.com/library/content/documentation/IDEs/Conceptual/AppDistributionGuide/ConfiguringYourApp/ConfiguringYourApp.html).

<div id="provide-apple-team-id-and-bundle-identifier-to-auth0">
  ## Fournissez le Apple Team ID et le Bundle Identifier à Auth0
</div>

1. Accédez à [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/clients) et sélectionnez le nom de l’application à consulter.

   <Frame>![Liste des applications dans le Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Faites défiler jusqu’au bas de la page **Paramètres**, puis sélectionnez **Afficher les paramètres avancés**.
3. Sélectionnez **Paramètres de l’appareil**, saisissez les valeurs **Team ID** et **Bundle Identifier** de votre application iOS, puis sélectionnez **Enregistrer les modifications**.

   <Frame>![Dashboard Applications Onglet paramètres de l’application Paramètres avancés Onglet Paramètres de l’appareil](/docs/images/cdy7uua7fh8z/YKKPtWtswHn2FZGFSVWNw/086b32e41b3fd446ac8aa678be33648d/Device_Settings_-_EN.png)</Frame>

<div id="test-link">
  ## Tester le lien
</div>

Vérifiez si le fichier `apple-app-site-association` des liens universels est accessible pour votre application en ouvrant l’URL suivante dans votre navigateur : `{yourDomain}/apple-app-site-association`

Si le lien fonctionne, vous verrez le JSON suivant (mis en forme pour en faciliter la lecture) :

```json lines
{
  "applinks": {
    "apps": [],
    "details": [{
      "appID": "86WQXF56BC.com.auth0.Passwordless-Email",
      "paths": ["/ios/com.auth0.Passwordless-Email/*"]
    }]
  }
}
```
