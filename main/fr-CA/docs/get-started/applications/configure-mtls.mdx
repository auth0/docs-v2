<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Pour utiliser les fonctionnalités Highly Regulated Identity, vous devez disposer d’un plan Enterprise avec l’option Highly Regulated Identity. Consultez [Auth0 Pricing](https://auth0.com/pricing/) pour en savoir plus.
</Callout>

Utilisez l’[Auth0 Management API](https://auth0.com/docs/api/management/v2) ou l’[Auth0 Dashboard](https://manage.auth0.com/) pour configurer l’authentification mTLS. Pour en savoir plus sur le fonctionnement de l’authentification mTLS chez Auth0, consultez [S’authentifier avec mTLS](/fr-CA/docs/get-started/authentication-and-authorization-flow/authenticate-with-mtls).
