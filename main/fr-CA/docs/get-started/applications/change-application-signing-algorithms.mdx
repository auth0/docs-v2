---
description: Décrit comment modifier l’algorithme de signature d’une application dans Auth0 Dashboard.
title: Modifier les algorithmes de signature d’une application
---

Vous pouvez modifier l’<Tooltip tip="Algorithme de signature : algorithme utilisé pour signer numériquement des jetons afin de garantir qu’ils n’ont pas été altérés." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=signing+algorithm">algorithme de signature</Tooltip> de votre application dans Auth0 Dashboard.

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) et sélectionnez le nom de l’application à consulter.

   <Frame>![Liste des applications du Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Faites défiler jusqu’au bas de l’onglet **Settings** et sélectionnez **Show Advanced Settings**.
3. Accédez à l’onglet **OAuth**.

   <Frame>![Paramètres de l’application dans le Dashboard, paramètres avancés, onglet OAuth](/docs/images/cdy7uua7fh8z/5bbq6VrI9tUMht0PwMimre/08574482efef944b16ecfbb445be50ab/Oauth_Tab_-_English.png)</Frame>
4. Repérez **JsonWebToken Signature Algorithm** et sélectionnez l’algorithme de signature approprié pour l’application.
5. Cliquez sur **Save Changes**.
