---
description: "Recommandations pour améliorer votre application Auth0 avant sa mise en production"
title: 'Vérifications de préparation à la production : correctifs non critiques'
---

Les vérifications suivantes permettent de déterminer si vous avez suivi toutes les recommandations (facultatives) pour réussir votre mise en production.

Pour en savoir plus sur les résultats de vos vérifications, consultez [Exécuter les vérifications de production](/fr-CA/docs/deploy-monitor/pre-deployment-checks/how-to-run-production-checks).

| Vérification                                                                                                                            | Description                                                                                                                                                                                                                                                                                                                                                                                                                  |
| --------------------------------------------------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Configurer Log Streaming                                                                                                                | Assurez-vous de transmettre les entrées de journal du locataire vers un emplacement de stockage ou un fournisseur externe. Nous vous recommandons de [configurer un service de diffusion de journaux](/fr-CA/docs/customize/log-streams) pour conserver les journaux plus de 30 jours et faciliter l&#39;analyse et les alertes.                                                                                                   |
| [Domaine personnalisé](/fr-CA/docs/customize/custom-domains) doit être configuré                                                              | Utilisez les [domaines personnalisés avec Universal Login](https://manage.auth0.com/#/tenant/custom_domains) pour offrir l&#39;expérience la plus fluide et la plus sécurisée à vos utilisateurs finaux.                                                                                                                                                                                                                     |
| [Page d&#39;erreur personnalisée](/fr-CA/docs/customize/login-pages/custom-error-pages) doit être configurée                                  | [Configurez une page d&#39;erreur personnalisée](https://manage.auth0.com/#/account) avec les renseignements propres à votre application et l&#39;image de marque de votre entreprise.                                                                                                                                                                                                                                       |
| Activer Universal Login                                                                                                                 | Assurez-vous d&#39;utiliser la version la plus récente de [Universal Login](/fr-CA/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login). Nous vous recommandons de faire passer votre page de connexion de l&#39;expérience Classic à Universal Login afin de profiter d&#39;une meilleure accessibilité, de meilleures performances, de la personnalisation sans code et de nouvelles fonctionnalités. |
| [Modèles de courriel](/fr-CA/docs/customize/email/manage-email-flow) doivent être configurés                                                  | [Configurez des modèles de courriel personnalisés](https://manage.auth0.com/#/emails) avec les renseignements propres à votre application et l&#39;image de marque de votre entreprise.                                                                                                                                                                                                                                      |
| [Authentification multifacteur](/fr-CA/docs/secure/multi-factor-authentication) doit être configurée                                          | Nous vous recommandons d&#39;envisager l&#39;authentification multifacteur comme pratique sécuritaire pour authentifier vos utilisateurs.                                                                                                                                                                                                                                                                                    |
| [MFA pour les administrateurs du locataire](/fr-CA/docs/get-started/manage-dashboard-access) est activée                                      | [Activez l&#39;authentification multifacteur](https://manage.auth0.com/#/account/admins) pour les administrateurs du locataire.                                                                                                                                                                                                                                                                                              |
| [URL de redirection de déconnexion](/fr-CA/docs/authenticate/login/logout#set-the-allowed-logout-urls-at-the-account-level) doit être définie | Vérifiez les [URL de redirection de déconnexion autorisées](https://manage.auth0.com/#/account/advanced) pour votre application.                                                                                                                                                                                                                                                                                             |
| Les applications devraient utiliser RS256 au lieu de HS256 pour l&#39;algorithme de signature JSON Web Token (JWT)                      | Définissez l&#39;[algorithme de signature](/fr-CA/docs/get-started/apis#signing-algorithms) du JSON Web Token (JWT) sur RS256 plutôt que HS256 pour chaque application.                                                                                                                                                                                                                                                            |
| Définir l&#39;[URI de connexion de l&#39;application](/fr-CA/docs/get-started/applications/application-settings#application-uris)             | Indiquez une [URI de connexion de l&#39;application](/fr-CA/docs/get-started/applications/application-settings#application-uris) pour toutes les applications interactives.                                                                                                                                                                                                                                                        |
| Définir l&#39;[URL de déconnexion autorisée du locataire](/fr-CA/docs/get-started/applications/application-settings#application-uris)         | Créez au moins une route de déconnexion accessible publiquement et [mettez à jour les paramètres du locataire](/fr-CA/docs/get-started/applications/application-settings#application-uris) pour qu&#39;ils pointent vers celle-ci.                                                                                                                                                                                                 |
| Définir l&#39;[URI de connexion du locataire](/fr-CA/docs/get-started/tenant-settings#login-and-logout)                                       | Publiez une route de connexion par défaut et [mettez à jour les paramètres de votre locataire](/fr-CA/docs/get-started/tenant-settings#login-and-logout) pour qu&#39;ils pointent vers celle-ci.                                                                                                                                                                                                                                   |
