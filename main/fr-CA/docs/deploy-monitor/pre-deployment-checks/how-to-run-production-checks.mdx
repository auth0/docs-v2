description: Découvrez comment exécuter les vérifications de production d’Auth0 pour vous assurer que vos applications sont prêtes pour la mise en production.
title: Exécuter les vérifications d’état de préparation à la production
---

Passez en revue les résultats des vérifications de production de votre locataire avant de mettre en production votre environnement.

1. Accédez à l’Auth0 Dashboard de gestion et connectez-vous à votre compte.
2. Dans la liste déroulante, sélectionnez le locataire pour lequel vous souhaitez exécuter les vérifications de production
3. Dans cette même liste déroulante, cliquez sur **Exécuter la vérification d’état de préparation** pour ouvrir l’interface

Facultatif :

1. Sélectionnez une ou plusieurs [Applications](/fr-CA/docs/get-started/applications) associées à ce locataire pour lesquelles vous souhaitez exécuter des vérifications en cliquant sur **Filtres**.
2. Cliquez sur **Appliquer**.

Une fois le test terminé, votre écran s’actualisera automatiquement pour afficher les résultats des vérifications.

<div id="production-check-results">
  ## Résultats des vérifications en production
</div>

Il existe trois catégories de corrections pour les résultats des vérifications :

* [Correctifs critiques](/fr-CA/docs/deploy-monitor/pre-deployment-checks/production-check-required-fixes)
* [Correctifs non critiques](/fr-CA/docs/deploy-monitor/pre-deployment-checks/production-check-recommended-fixes)
* [Bonnes pratiques](/fr-CA/docs/deploy-monitor/pre-deployment-checks/production-checks-best-practices)

Voici les résultats possibles de votre vérification :

* Réussite
* Échec
* Bonnes pratiques

Pour chaque ensemble de résultats, Auth0 indique combien de vérifications votre application a réussies, ainsi que combien de vérifications doivent être examinées.

Si votre application a **échoué** à une ou plusieurs vérifications, vous verrez :

* Le nom de la vérification
* Des renseignements sur ce que la vérification cherche à détecter
* Un lien **Configurer** vers la section appropriée, où vous pouvez apporter les corrections requises pour que votre application réussisse la vérification
* Un hyperlien vers la documentation correspondante

Toutes les vérifications que votre application a **réussies** sont regroupées dans l’objet passed de l’ensemble de résultats. Vous pouvez consulter le nom de la vérification et les renseignements à son sujet, examiner la documentation associée et utiliser l’hyperlien pour accéder à la section de configuration correspondante où vous pouvez apporter des modifications, au besoin.
