description: Décrit les options de déploiement d’Auth0 en cloud public et en cloud privé.
title: Options de déploiement
---

Auth0 offre des options de déploiement en cloud public et en cloud privé, ainsi que des outils pour vous aider dans votre déploiement d’entreprise.

La plateforme d’identité Auth0 peut être déployée de la façon suivante :

| Option                                                                                        | Description                                                                                                                                                                                 |
| --------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Cloud public**                                                                              | Un service infonuagique multilocataire hébergé dans le cloud d’Auth0, avec des fonctionnalités standard d’authentification, de personnalisation, de gestion des utilisateurs et de sécurité |
| [**Cloud privé sur AWS**](/fr-CA/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | Un service infonuagique dédié et géré exécuté sur AWS, offrant l’isolation, de meilleures performances, des instances de développement, des modules complémentaires Geo-HA, et plus encore  |
| [**Cloud privé sur Azure**](/fr-CA/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Un service infonuagique dédié exécuté sur Azure, offrant l’isolation, de meilleures performances, des instances de développement, des modules complémentaires Geo-HA, et plus encore        |

Les offres de **cloud privé** sont des services gérés que vous pouvez utiliser si vous :

* ne pouvez pas utiliser un service de cloud public multilocataire dans votre organisation
* avez besoin d’un niveau garanti de requêtes par seconde (RPS)

Si vous avez des exigences particulières en matière de soutien ou souhaitez obtenir plus d’information sur les services professionnels que nous offrons, veuillez [communiquer avec l’équipe des ventes d’Auth0](https://auth0.com/contact-us).
