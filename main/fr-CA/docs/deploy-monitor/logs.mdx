Auth0 fournit des journaux que vous pouvez analyser selon vos besoins d’affaires. Vous pouvez :

* Voir les actions effectuées par les administrateurs du locataire.
* Voir les opérations effectuées au moyen de la <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>.
* Voir les authentifications effectuées par vos utilisateurs.
* Voir les erreurs intermittentes qui peuvent être difficiles à repérer lors des tests d’assurance qualité.
* Recueillir des données médico-légales à des fins de sécurité et pour repérer les déclencheurs de détection d’anomalies.
* Identifier des tendances d’utilisation.
* Vérifier les erreurs d’authentification et les erreurs d’appel à la Management API.
* Configurer une analyse automatique des journaux pour détecter les erreurs de limite de débit afin de pouvoir traiter de façon proactive l’activité qui atteint les limites de débit avant qu’elle n’ait une incidence sur vos utilisateurs.
* Corréler les événements de la Management API avec les requêtes qui les ont déclenchés en faisant correspondre les valeurs d’en-tête `X-Correlation-ID`.

<div id="log-streaming-and-exporting">
  ## Journal Streaming et exportation
</div>

Utilisez Journal Streaming pour exporter les journaux vers les outils de traitement et d’analyse de votre choix. Nous offrons des intégrations avec plusieurs services d’analyse, ainsi qu’un moyen de créer un flux de journaux personnalisé à l’aide de webhooks. Pour en savoir plus, consultez [Journal Streaming](/fr-CA/docs/customize/log-streams).
