Les forfaits Private Cloud sont des services gérés que vous pouvez utiliser si vous :

* ne pouvez pas utiliser un service Public Cloud mutualisé au sein de votre organisation
* avez des exigences de déploiement géographique pour lesquelles l’option Public Cloud n’est pas viable
* avez besoin d’un niveau garanti de requêtes par seconde (RPS) ou d’une capacité supérieure à celle offerte par Public Cloud

Le cloud privé Auth0 peut être déployé des façons suivantes :

| Option                                                                                         | Description                                                                                                                                                                                    |
| ---------------------------------------------------------------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [**Private Cloud on AWS**](/fr-CA/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | Un service infonuagique dédié et géré fonctionnant sur AWS, offrant l’isolation, de meilleures performances, des instances de développement, des modules complémentaires Geo-HA et plus encore |
| [**Private Cloud on Azure**](/fr-CA/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Un service infonuagique dédié fonctionnant sur Azure, offrant l’isolation, de meilleures performances, des instances de développement, des modules complémentaires et plus encore              |

Nous offrons aussi des [fonctionnalités complémentaires](/fr-CA/docs/deploy-monitor/deploy-private-cloud/private-cloud-add-on-features) qui peuvent bonifier votre déploiement Auth0 Private Cloud.

Si vous avez des exigences particulières en matière de soutien ou souhaitez obtenir plus d’information sur les services professionnels que nous offrons, veuillez [communiquer avec l’équipe des ventes d’Auth0](https://auth0.com/contact-us).
