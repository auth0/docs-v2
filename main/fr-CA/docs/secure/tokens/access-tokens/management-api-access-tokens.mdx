title: Jetons d’accès de la Management API
---

Pour appeler l’[Auth0 Management API](https://auth0.com/docs/api/management/v2), vous devez fournir un <Tooltip tip="Jeton d’accès : justificatif d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisé pour accéder à une API." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=access+token">jeton d’accès</Tooltip> qui contient l’<Tooltip tip="Jeton d’accès : justificatif d’autorisation, sous la forme d’une chaîne opaque ou d’un JWT, utilisé pour accéder à une API." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=audience">audience</Tooltip> appropriée et les scopes appropriés.

<div id="audience">
  ## Audience
</div>

La valeur du paramètre audience pour la <Tooltip tip="Management API : produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip> est `https://{yourDomain}/api/v2/`.

<div id="scopes">
  ## Scopes
</div>

Chaque endpoint de la Management API requiert un ensemble précis de scopes. Consultez le [Management API Explorer](https://auth0.com/docs/api/management/v2/) pour en savoir plus.

<div id="lifetime">
  ## Durée de vie
</div>

La durée de vie par défaut du jeton de la Management API est de 86 400 secondes (24 heures).

Vous pouvez [configurer la durée de vie du jeton](/fr-CA/docs/secure/tokens/access-tokens/update-access-token-lifetime) pour chaque API séparément.

<div id="security">
  ## Sécurité
</div>

* **jeton d’accès** : Une fois émis, un jeton d’accès ne peut pas être révoqué. Auth0 recommande de limiter la durée de validité de vos jetons d’accès à la plus courte valeur possible compatible avec votre cas d’utilisation.
* **<Tooltip tip="Jeton d’actualisation : jeton utilisé pour obtenir un nouveau Jeton d’accès sans obliger les utilisateurs à se reconnecter." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Refresh+token">Jeton d’actualisation</Tooltip>** : Vous pouvez [révoquer un jeton d’actualisation](/fr-CA/docs/secure/tokens/refresh-tokens/revoke-refresh-tokens) à l’aide de l’Auth0 Dashboard, de l’Authentication API ou de la Management API.
* **<Tooltip tip="Secret client : secret utilisé par une application pour s’authentifier auprès du serveur d’autorisation; il ne doit être connu que de l’application et du serveur d’autorisation, et doit être suffisamment aléatoire pour être impossible à deviner." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Client+secret">Secret client</Tooltip>** : Vous pouvez [effectuer la rotation du secret client d’une application](/fr-CA/docs/get-started/applications/rotate-client-secret) à l’aide de l’Auth0 Dashboard ou de la Management API.

<div id="quota">
  ## Quota
</div>

Les quotas de jetons sont établis selon votre palier d’abonnement. Vous pouvez [consulter votre quota actuel](https://support.auth0.com/reports/quota) dans Auth0 Support Center. Si vous avez des questions sur la tarification ou les quotas, veuillez [contacter notre équipe des ventes](https://auth0.com/contact-us).

* Les jetons émis pour des [audiences internes d’Auth0](/fr-CA/docs/secure/tokens/access-tokens/get-access-tokens#control-access-token-audience) ne sont pas comptabilisés dans votre quota.
* Les jetons émis pour des audiences personnalisées sont comptabilisés dans votre quota et sont assujettis aux [limites de débit d’Auth0 Management API](/fr-CA/docs/troubleshoot/customer-support/operational-policies/rate-limit-policy/management-api-endpoint-rate-limits).
