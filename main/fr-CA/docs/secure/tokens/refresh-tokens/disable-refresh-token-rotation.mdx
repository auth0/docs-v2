Vous pouvez désactiver la rotation des <Tooltip tip="Jeton d’actualisation : jeton utilisé pour obtenir un nouveau jeton d’accès sans obliger les utilisateurs à se reconnecter." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=refresh+token">jetons d’actualisation</Tooltip> pour chaque application au moyen de l’Auth0 Dashboard ou de la <Tooltip tip="Jeton d’actualisation : jeton utilisé pour obtenir un nouveau jeton d’accès sans obliger les utilisateurs à se reconnecter." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>.

<div id="disable-with-the-dashboard">
  ## Désactiver dans l’Auth0 Dashboard
</div>

1. Dans l’[Auth0 Dashboard](https://manage.auth0.com/#), accédez à **Applications &gt; Applications** et sélectionnez l’application que vous souhaitez configurer.
2. Dans l’onglet **Settings**, repérez la section **Rotation des jetons d’actualisation** et désactivez le bouton bascule **Autoriser la rotation des jetons d’actualisation**.
3. Sélectionnez **Enregistrer les modifications** au bas de l’écran.

<div id="disable-with-the-management-api">
  ## Désactiver avec la Management API
</div>

1. Désactivez la rotation des jetons d’actualisation pour chaque application au moyen de la Management API :

   {/* codeblockOld.header.login.logInButton codeblockOld.header.login.configureSnippet */}

   <AuthCodeBlock children={codeExample} language="javascript" />

2. Configurez les paramètres du jeton d’actualisation non rotatif comme suit :

   ```http lines
   PATCH /api/v2/clients/{client_id}
   {
      "refresh_token": {
        "rotation_type": "non-rotating",
        "expiration_type": "non-expiring"
      }
   }
   ```
