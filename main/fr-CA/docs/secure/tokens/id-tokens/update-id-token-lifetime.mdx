title: Mettre à jour la durée de vie de l’ID Token
---

Vous pouvez modifier la durée de vie du <Tooltip tip="ID Token : Jeton destiné à l’application cliente elle-même, plutôt qu’à l’accès à une ressource." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=ID+token">jeton d’identité</Tooltip> dans l’<Tooltip tip="ID Token : Jeton destiné à l’application cliente elle-même, plutôt qu’à l’accès à une ressource." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Par défaut, un jeton d’identité est valide pendant 36000 secondes (10 heures). Si vous avez des préoccupations liées à la sécurité, vous pouvez réduire la durée avant l’expiration du jeton, en gardant à l’esprit que l’un de ses objectifs est d’améliorer l’expérience utilisateur en mettant en cache les renseignements sur l’utilisateur.

1. Accédez à [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) et cliquez sur le nom de l’application que vous voulez afficher.
2. Faites défiler jusqu’à **ID Token** et repérez le champ **ID Token Expiration**. Saisissez la durée de vie appropriée du jeton d’identité (en secondes) pour l’application.
3. Cliquez sur **Save Changes**.
