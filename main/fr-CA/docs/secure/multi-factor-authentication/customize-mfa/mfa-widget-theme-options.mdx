---
description: Décrit les options de thème du widget MFA pour personnaliser les propriétés de thème des pages MFA.
title: Options de thème du widget MFA
---

Lorsque vous utilisez votre propre code HTML pour les pages <Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=MFA">MFA</Tooltip> avec Classic Login, vous dépendez du widget MFA d’Auth0, qui présente les limites suivantes :

* Il ne prend pas en charge la MFA par courriel.
* Si les utilisateurs ont inscrit plus d’un facteur, ils ne peuvent pas choisir lequel utiliser ; le widget MFA les invite à se connecter avec le facteur le plus sécuritaire.
* Il n’utilise pas les fonctionnalités d’[internationalisation](/fr-CA/docs/customize/internationalization-and-localization/universal-login-internationalization) de <Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur le serveur d’autorisation d’Auth0, pour vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Universal+Login">Universal Login</Tooltip>.

Les options de thème du widget MFA sont définies sous la propriété `theme`.

| Option         | Description                                                                                                                                                                         |
| -------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `icon`         | L’URL de l’image utilisée dans l’en-tête. La valeur par défaut est le logo Auth0. La hauteur recommandée est de 58 pixels.                                                          |
| `primaryColor` | Définit la couleur principale du widget MFA. Utile lorsque vous utilisez une `icon` personnalisée pour assurer une bonne harmonie des couleurs. La valeur par défaut est `#ea5323`. |

<div id="examples">
  ## Exemples
</div>

```js lines
return new Auth0MFAWidget({

...

  theme: {
    icon: 'https://example.com/assets/logo.png',
    primaryColor: 'blue'
  },
  
...  
  
})
```

Consultez un [exemple de widget Guardian de base](https://github.com/auth0/auth0-guardian.js/blob/master/example/basic_widget.html) utilisant [auth0-guardian.js](https://github.com/auth0/auth0-guardian.js). Consultez le [README](https://github.com/auth0/auth0-guardian.js/tree/master/example) pour en savoir plus.
