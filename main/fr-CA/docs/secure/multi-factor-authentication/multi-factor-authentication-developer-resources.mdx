---

Grâce aux SDK Auth0, vous pouvez personnaliser l’expérience d’<Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification d’un utilisateur qui utilise un facteur supplémentaire au nom d’utilisateur et au mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=multi-factor+authentication">authentification multifacteur</Tooltip> (MFA) de vos utilisateurs, et même créer des applications qui s’appuient sur nos capacités multifacteur.

<div id="mfa-api">
  ## API MFA
</div>

Les points de terminaison de l’[API MFA](/fr-CA/docs/secure/multi-factor-authentication/multi-factor-authentication-developer-resources/mfa-api) vous permettent d’exiger la MFA lorsque les utilisateurs interagissent avec les [points de terminaison de jeton](https://auth0.com/docs/api/authentication#get-token), ainsi que d’inscrire et de gérer des facteurs MFA.

<div id="create-custom-mfa-pages">
  ## Créer des pages MFA personnalisées
</div>

Utilisez les bibliothèques clientes suivantes pour personnaliser l’apparence de la page MFA afin qu’elle corresponde à l’image de votre organisation.

* [Bibliothèque cliente MFA Auth0](https://github.com/auth0/auth0-guardian.js)
* [Créer un MFA Widget personnalisé](https://github.com/auth0/auth0-guardian.js/tree/master/example)

<div id="create-custom-enrollments">
  ## Créer des inscriptions personnalisées
</div>

Vous pouvez [personnaliser le processus d’inscription](/fr-CA/docs/secure/multi-factor-authentication/multi-factor-authentication-developer-resources/create-custom-enrollment-tickets) de vos utilisateurs en créant des tickets d’inscription personnalisés.

<div id="build-custom-mobile-applications">
  ## Créer des applications mobiles personnalisées
</div>

Créez des applications personnalisées semblables à Guardian ou ajoutez des fonctionnalités d’authentification multifacteur à vos applications à l’aide de nos SDK.

* [Guardian pour Android](/fr-CA/docs/secure/multi-factor-authentication/auth0-guardian/guardian-for-android-sdk)
* [Guardian pour iOS](/fr-CA/docs/secure/multi-factor-authentication/auth0-guardian/guardian-for-ios-sdk)
