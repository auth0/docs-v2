title: API MFA d’Auth0
---

Auth0 fournit un flux intégré d’inscription et d’authentification à l’<Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=multi-factor+authentication">authentification multifacteur</Tooltip> (MFA) à l’aide de [Universal Login](/fr-CA/docs/authenticate/login/auth0-universal-login). Utilisez l’[API MFA](https://auth0.com/docs/api/authentication#multi-factor-authentication) dans les scénarios suivants si vous souhaitez :

* [Authentifier des utilisateurs avec le Resource Owner Password Grant](/fr-CA/docs/secure/multi-factor-authentication/authenticate-using-ropg-flow-with-mfa).
* Créer une interface permettant aux [utilisateurs de gérer leurs propres facteurs d’authentification](/fr-CA/docs/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

Pour utiliser l’API MFA, vous devez activer le type d’autorisation MFA pour votre application. Pour activer l’autorisation MFA dans le <Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 permettant de configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, accédez à [Dashboard &gt; Applications &gt; Advanced Settings &gt; Grant Types](https://manage.auth0.com/#/applications) et sélectionnez **MFA**. Consultez [Enable Multi-Factor Authentication](/fr-CA/docs/secure/multi-factor-authentication/enable-mfa) pour en savoir plus.

<div id="limitations">
  ## Limitations
</div>

L’API MFA est conçue pour fonctionner avec les facteurs SMS, vocaux, les notifications push par l’entremise de l’application Auth0 Guardian, le courriel et les OTP. Elle ne prend actuellement pas en charge l’inscription avec Duo ou WebAuthn.
