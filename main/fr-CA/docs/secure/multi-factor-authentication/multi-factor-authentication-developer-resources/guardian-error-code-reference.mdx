---
description: Répertorie les codes d’erreur et les descriptions de Guardian.
title: Référence des codes d’erreur Guardian
---

Utilisez les codes d’erreur pour afficher des messages informatifs et distinguer les erreurs récupérables des erreurs irrécupérables.

| Code d’erreur                         | Description                                                                                                                                                  |
| ------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `invalid_token`                       | Jeton de requête ou de transaction invalide                                                                                                                  |
| `insufficient_scope`                  | Vous n’avez pas suffisamment d’octrois pour effectuer l’opération demandée                                                                                   |
| `invalid_bearer_format`               | Le jeton Bearer fourni dans l’en-tête d’authentification n’était pas valide                                                                                  |
| `enrollment_conflict`                 | Il existe déjà une autre inscription pour le même utilisateur. Vous ne pouvez pas vous inscrire deux fois.                                                   |
| `tenant_not_found`                    | Le locataire associé est introuvable. Cela ne devrait normalement pas se produire, sauf si vous supprimez le locataire                                       |
| `login_transaction_not_found`         | La transaction d’authentification MFA n’est pas active ou a déjà expiré                                                                                      |
| `error_sending_push_notification`     | L’envoi de la notification poussée a échoué                                                                                                                  |
| `push_notification_wrong_credentials` | L’envoi de la notification poussée a échoué en raison d’identifiants incorrects                                                                              |
| `invalid_otp`                         | Le code OTP fourni n’était pas valide                                                                                                                        |
| `invalid_recovery_code`               | Le code de récupération fourni n’était pas valide                                                                                                            |
| `invalid_body`                        | La validation du corps de la requête a échoué. Requête incorrecte.                                                                                           |
| `invalid_query_string`                | La validation de la chaîne de requête a échoué. Requête incorrecte.                                                                                          |
| `enrollment_transaction_not_found`    | La transaction d’inscription MFA n’est pas active ou a expiré                                                                                                |
| `invalid_phone_number`                | Le numéro de téléphone fourni est invalide                                                                                                                   |
| `error_sending_sms`                   | Erreur d’envoi de SMS                                                                                                                                        |
| `feature_disabled`                    | La fonctionnalité demandée n’est actuellement pas disponible à l’échelle globale (contactez le fournisseur)                                                  |
| `feature_disabled_by_admin`           | La fonctionnalité demandée est actuellement désactivée par votre administrateur                                                                              |
| `pn_endpoint_disabled`                | Nous n’avons pas pu envoyer la notification poussée après plusieurs tentatives. Essayez de retirer votre compte de l’appareil, puis de l’ajouter de nouveau. |
| `too_many_sms`                        | Vous avez dépassé le nombre de SMS attribués à cet utilisateur                                                                                               |
| `too_many_pn`                         | Vous avez dépassé le nombre de notifications poussées attribuées à cet utilisateur                                                                           |
| `too_many_sms_per_tenant`             | Vous avez dépassé le nombre de SMS attribués à ce locataire                                                                                                  |
| `too_many_pn_per_tenant`              | Vous avez dépassé le nombre de notifications poussées attribuées à ce locataire                                                                              |
| `field_required`                      | Un champ est requis pour effectuer l’opération (cette erreur comporte un attribut field avec un code pour le champ : `otpCode`, `recoveryCode`)              |
| `method_not_found`                    | Vous avez demandé une méthode qui n’est actuellement pas prise en charge (cela ne devrait pas arriver)                                                       |
| `no_method_available`                 | Aucune méthode d’inscription n’est actuellement disponible (elles sont toutes désactivées)                                                                   |
| `enrollment_method_disabled`          | La méthode d’inscription spécifiée est désactivée; cette erreur contient également un champ .method                                                          |
| `auth_method_disabled`                | La méthode d’authentification spécifiée est désactivée; cette erreur contient également un champ .method                                                     |
| `invalid_otp_format`                  | Erreur de validation du format OTP                                                                                                                           |
| `invalid_recovery_code_format`        | Erreur de validation du format du code de récupération                                                                                                       |
| `transaction_expired`                 | La transaction a déjà expiré                                                                                                                                 |
| `already_enrolled`                    | Vous êtes déjà inscrit, impossible de vous inscrire de nouveau                                                                                               |
| `not_enrolled`                        | Vous n’êtes pas inscrit. Vous devez d’abord vous inscrire                                                                                                    |
| `invalid_enrollment`                  | L’inscription fournie à la méthode transaction#requestAuth n’est pas valide ou est null/undefined                                                            |
