---
description: Comment configurer les codes de récupération pour l’authentification multifacteur.
title: Configurer les codes de récupération pour l’authentification multifacteur (MFA)
---

Un code de récupération est un code unique généré par Auth0 qui permet à un utilisateur de récupérer l’accès à son compte. Ainsi, si un utilisateur n’a pas accès à l’appareil ou au compte utilisé lors de l’inscription à <Tooltip tip="Authentification multifacteur (MFA) : processus d’authentification qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, comme un code envoyé par SMS." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=multi-factor+authentication">l’authentification multifacteur</Tooltip> (MFA), il peut utiliser un code de récupération pour s’authentifier.

<div id="how-it-works">
  ## Fonctionnement
</div>

Lorsque vous utilisez <Tooltip tip="Universal Login : votre application redirige vers Universal Login, hébergé sur le serveur d’autorisation d’Auth0, pour vérifier l’identité d’un utilisateur." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Universal+Login">Universal Login</Tooltip> avec les codes de récupération activés :

1. Un utilisateur commence l’inscription MFA.
2. Auth0 génère un code de récupération.
3. Pendant l’inscription MFA, l’invite du code de récupération s’affiche pour l’utilisateur.
4. L’utilisateur enregistre le code de récupération et termine le processus d’inscription.
5. L’utilisateur peut ensuite terminer la MFA à l’aide du code de récupération qu’il a enregistré s’il perd l’accès à l’appareil ou au compte qu’il a inscrits à la MFA.

<Frame>![Exemple d’invite de code de récupération pendant l’inscription MFA](/docs/images/cdy7uua7fh8z/6LkXKthRal1r4vIPi3137l/87d45c4202087244b20259dd30642435/2025-02-26_14-31-30.png)</Frame>

Lorsque vous utilisez Universal Login avec les codes de récupération désactivés :

* Les utilisateurs ne verront pas l’invite du code de récupération pendant l’inscription MFA.
* Les utilisateurs ne peuvent pas s’authentifier avec un code de récupération.

<div id="enable-recovery-codes">
  ## Activer les codes de récupération
</div>

Les codes de récupération sont désactivés par défaut. Vous pouvez activer les codes de récupération dans [Auth0 Dashboard &gt; Sécurité &gt; Authentification multifacteur](https://manage.auth0.com/#/security/mfa).
