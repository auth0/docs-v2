---
title: Gérer les Rules dans Auth0 Dashboard
description: Découvrez comment gérer les Rules de l’ACL du locataire dans Auth0 Dashboard. 
---

Vous pouvez utiliser Auth0 Dashboard pour afficher la liste des Rules de l’ACL de votre locataire et les activer ou les désactiver individuellement.

<div id="view-list-of-rules">
  ## Voir la liste des Rules
</div>

Accédez à [Auth0 Dashboard &gt; Security &gt; Access Control](https://manage.auth0.com/#/security/access-control) pour voir la liste de vos Rules existantes.

| Nom de la colonne    | Description                                                                                  | Champ de réponse d’API associé |
| -------------------- | -------------------------------------------------------------------------------------------- | ------------------------------ |
| Priorité             | Nombre ordinal indiquant à quel moment la Rule est évaluée                                  | `priority`                     |
| Nom                  | Description de l’objectif ou de la fonction de la Rule                                      | `description`                  |
| Action               | L’[action](/fr-CA/docs/secure/tenant-access-control-list/reference#actions) exécutée par la Rule. | `action`                       |
| Statut               | Valeur indiquant si la Rule est active ou désactivée.                                       | `active`                       |
| Dernière mise à jour | Horodatage indiquant quand la Rule a été mise à jour pour la dernière fois.                 | `updated_at`                   |

![Liste des Rules de l’ACL du locataire dans l’Auth0 Dashboard](/docs/images/cdy7uua7fh8z/4093f9a39aba48bdf8d579c134dc4da5/Screenshot_2025-09-03_at_12.33.47.png)

<div id="enable-or-disable-a-rule">
  ## Activer ou désactiver une Rule
</div>

Vous pouvez activer ou désactiver une Rule précise à l’aide du bouton à côté de celle-ci.

Le champ **Statut** indique si la Rule est `Active` ou `Disabled`.

![Activer ou désactiver une Rule ACL du locataire dans l’Auth0 Dashboard](/docs/images/cdy7uua7fh8z/29b92e102142d3cb20e69a40e292f6b7/Screenshot_2025-09-03_at_12.34.05.png)
