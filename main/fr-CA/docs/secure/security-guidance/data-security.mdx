title: Sécurité des données

* [Adresses IP d’Auth0 pour les listes d’autorisation](/fr-CA/docs/secure/security-guidance/data-security/allowlist) - Si votre environnement est protégé par un pare-feu, les connexions à des bases de données personnalisées, les Actions, les Hooks et les Rules peuvent nécessiter l’ajout de certaines adresses IP d’Auth0 à la liste d’autorisation pour assurer leur bon fonctionnement.
* [Ajouter des attributs utilisateur à la liste de refus](/fr-CA/docs/secure/security-guidance/data-security/denylist) - Si certains champs utilisateur ne doivent pas être stockés dans les bases de données d’Auth0 pour des raisons de confidentialité, vous pouvez ajouter des attributs utilisateur à la liste de refus.
* [Stockage des données utilisateur](/fr-CA/docs/secure/security-guidance/data-security/user-data-storage) - Découvrez les différences entre le stockage des données dans Auth0 et dans une base de données externe pour les données utilisateur.
* [Stockage des jetons](/fr-CA/docs/secure/security-guidance/data-security/token-storage) - Découvrez comment et où stocker les jetons utilisés pour l’authentification par jeton.
