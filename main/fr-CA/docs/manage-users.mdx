title: Gérer les utilisateurs
---

La base de données infonuagique hébergée d’Auth0 stocke les données utilisateur, auxquelles vous pouvez accéder au moyen des profils utilisateur. Les renseignements sur les utilisateurs peuvent provenir de diverses sources, notamment des fournisseurs d’identité, de vos propres bases de données et des connexions d’entreprise (comme Active Directory ou SAML).
