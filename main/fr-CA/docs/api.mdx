<div class="api_card_wrapper">
  <Card title="Authentication API" href="/fr-CA/docs/api/authentication" arrow />
</div>

<Columns cols={2}>
  <Column>
    L’Authentication API expose les fonctionnalités d’identité d’Auth0 et les protocoles d’identité pris en charge (y compris OpenID Connect, OAuth et SAML).

    En règle générale, vous devriez utiliser cette API au moyen de l’un des SDK Auth0, comme [Auth0.js](/fr-CA/docs/libraries/auth0js), ou d’une bibliothèque comme [Lock](/fr-CA/docs/libraries/lock). Toutefois, si vous créez manuellement votre interface utilisateur d’authentification, vous devrez appeler l’Authentication API directement.
  </Column>

  <Column>
    Voici quelques exemples de tâches :

    * Obtenir des [jetons](/fr-CA/docs/secure/tokens) pendant l’authentification
    * Demander le profil d’un utilisateur à l’aide d’un [Jeton d’accès](/fr-CA/docs/secure/tokens/access-tokens)
    * Échanger des [Jetons d’actualisation](/fr-CA/docs/secure/tokens/refresh-tokens) contre de nouveaux jetons d’accès
    * Demander un défi pour l’[authentification multifacteur (MFA)](/fr-CA/docs/secure/multi-factor-authentication)
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="Management API" href="/fr-CA/docs/api/management/v2" arrow />
</div>

<Columns cols={2}>
  <Column>
    La Management API vous permet de gérer votre compte Auth0 par programmation afin d’automatiser la configuration de votre environnement. La plupart des tâches que vous pouvez effectuer dans l’Auth0 Dashboard peuvent également être effectuées par programmation à l’aide de cette API.
  </Column>

  <Column>
    Voici quelques exemples de tâches :

    * Enregistrer vos applications et vos API auprès d’Auth0
    * Configurer des [connexions](/fr-CA/docs/connections) pour permettre à vos utilisateurs de s’authentifier
    * [Gérer les utilisateurs](/fr-CA/docs/manage-users)
    * [Lier des comptes d’utilisateur](/fr-CA/docs/manage-users/user-accounts/user-account-linking/link-user-accounts)
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="API My Account" href="/fr-CA/docs/api/myaccount" arrow />
</div>

<Columns cols={2}>
  <Column>
    L’API Auth0 My Account fournit un ensemble dédié de points de terminaison permettant aux utilisateurs de gérer les renseignements de leur propre compte. Vous pouvez utiliser ces API pour créer des expériences libre-service dans vos applications ou ajouter progressivement des renseignements à un compte utilisateur.
  </Column>

  <Column>
    Voici quelques exemples de tâches :

    * Permettre à l’utilisateur final de gérer ses propres facteurs
    * Mettre à jour les détails d’une méthode d’authentification
    * Connaître les méthodes d’authentification offertes
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="API My Organization" href="/fr-CA/docs/api/myorganization" arrow />

<Columns cols={2}>
  <Column>
    L’API Auth0 My Organization fournit une interface sécurisée, définie au niveau de l’Organisation, qui permet à vos clients professionnels de gérer leurs propres Organisations dans votre locataire Auth0. Cette API constitue la base technique de l’administration déléguée intégrée et des intégrations axées sur les API. Pour en savoir plus, consultez [My Organization API and Embeddable UI Components](/fr-CA/docs/manage-users/my-organization-api).
  </Column>

  <Column>
    Voici quelques exemples de tâches :

    * Détails de l’Organisation Auth0 (nom, image de marque, nom d’affichage)
    * Configuration, propriété et relations propres à l’organisation
    * Fournisseurs d’identité (IdP) et provisionnement SCIM
    * Configuration des domaines et de Home Realm Discovery (HRD)
  </Column>
</Columns>

<div id="pagination">
  ## Pagination
</div>

Pour les locataires Public Cloud, les points de terminaison de la Management API renvoient au maximum 50 résultats par réponse. S’il y a plus de résultats qu’une réponse peut en renvoyer, incluez les paramètres `page` et `per_page` dans votre requête pour en récupérer davantage.

Consultez [Migrer vers les requêtes paginées des points de terminaison de la Management API v2](/fr-CA/docs/troubleshoot/product-lifecycle/past-migrations/migrate-to-paginated-queries) pour savoir comment mettre à jour vos applications.
