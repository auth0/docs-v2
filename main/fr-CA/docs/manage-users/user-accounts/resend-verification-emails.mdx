---
description: Décrit comment renvoyer des courriels de vérification aux utilisateurs au moyen d’Auth0 Management Dashboard et de l’Auth0 Management API.
title: Renvoyer les courriels de vérification
---

Si vous devez renvoyer un courriel de vérification à un utilisateur, vous pouvez le faire à l’aide d’Auth0 User Management Dashboard ou de l’Auth0 <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>.

<Tabs>
  <Tab title="Utiliser Auth0 Dashboard">
    Vous pouvez utiliser Auth0 Dashboard pour renvoyer un courriel de vérification à un utilisateur.

    1. Dans Auth0 Dashboard, accédez à [**User Management &gt; Users**](https://manage.auth0.com/#/users).
    2. Sélectionnez l’utilisateur.
    3. Dans le menu déroulant **Actions**, sélectionnez **Send Verification Email**.

    Si vous activez [Multiple Custom Domains](/fr-CA/docs/customize/custom-domains/multiple-custom-domains), l’option **Domaine** vous permet de sélectionner le domaine de notification de votre locataire.
  </Tab>

  <Tab title="Utiliser Auth0 Management API">
    Vous pouvez utiliser l’Auth0 Management API pour renvoyer un courriel de vérification à un utilisateur.

    1. Obtenez un [jeton d’accès à la Management API](/fr-CA/docs/secure/tokens/access-tokens/management-api-access-tokens) avec le scope `update:users`.
    2. Appelez le point de terminaison [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) de la Management API avec le `user_id` de l’utilisateur.

    #### Exemple de requête

    ```json lines
    {
       "user_id":"auth0|62463410c0242d00699461c8"
    }
    ```

    #### Exemple de réponse

    ```json lines
    {
       "type":"verification_email",
       "status":"pending",
       "created_at":"2022-03-31T23:08:05.612Z",
       "id":"job_PyfpRHlmpkSoDmCv"
    }
    ```

    Si vous activez [Multiple Custom Domains](/fr-CA/docs/customize/custom-domains/multiple-custom-domains), vous devez inclure l’en-tête HTTP `auth0-custom-domain`. Pour en savoir plus, consultez [Multiple Custom Domains](/fr-CA/docs/customize/custom-domains/multiple-custom-domains#customize-email-handling-using-the-management-api).

    #### Vérifier l’état de la tâche

    Vous pouvez vérifier l’état de la tâche créée avec le point de terminaison [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) de la Management API.

    1. Obtenez l’`id` de la tâche (renvoyé dans la réponse).
    2. Appelez le point de terminaison [Get a job](https://auth0.com/docs/api/management/v2#!/Jobs/get_jobs_by_id) de la Management API avec l’`id` de la tâche.

    ##### Exemple de réponse

    ```json lines
    {
       "type":"verification_email",
       "status":"completed",
       "created_at":"2022-03-31T23:08:05.612Z",
       "id":"job_PyfpRHlmpkSoDmCv"
    }
    ```
  </Tab>
</Tabs>
