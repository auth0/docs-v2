---
description: Décrit comment créer des utilisateurs dans l’Auth0 Dashboard.
title: Créer des utilisateurs
---

Vous pouvez créer un utilisateur à l’aide de l’Auth0 Dashboard ou de la [Auth0 Management API](https://auth0.com/docs/api/management/v2/users/post-users).

1. Accédez à [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users), puis sélectionnez **Create User**.
2. Sélectionnez la **Connexion**, puis saisissez le **Courriel** et le **Mot de passe** de votre utilisateur.

   | Attribut     | Description                                                                                                                                                                                                                                                                                                                       |
   | ------------ | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | Connexion    | La connexion de base de données à utiliser pour authentifier l’utilisateur. La liste déroulante affiche toutes les connexions de base de données configurées dans votre locataire. La connexion utilisée doit être associée à une application, sinon vous recevrez un message d’erreur indiquant que la connexion est désactivée. |
   | Courriel     | L’adresse courriel de l’utilisateur. Le format de l’adresse courriel doit être conforme aux spécifications fournies dans la [RFC 5321 de l’IETF](https://datatracker.ietf.org/doc/html/rfc5321#section-4.1.2)                                                                                                                     |
   | Mot de passe | Le mot de passe de l’utilisateur. Il n’existe aucune longueur maximale pour le mot de passe.                                                                                                                                                                                                                                      |
3. Sélectionnez **Create**.
   L’utilisateur est créé et vous serez redirigé vers le profil du nouvel utilisateur.

Si vous activez [Multiple Custom Domains](/fr-CA/docs/customize/custom-domains/multiple-custom-domains), vous pouvez sélectionner **Domaine** dans la section **Connexion** afin de choisir le domaine de notification pour votre locataire.

Si vous activez [Flexible Identifiers](/fr-CA/docs/authenticate/database-connections/flexible-identifiers-and-attributes), en plus de **email**, vous pouvez sélectionner **username**, **phone** ou une combinaison d’au moins deux de ces identifiants.
