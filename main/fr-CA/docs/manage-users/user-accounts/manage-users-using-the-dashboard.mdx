title: Gérer les utilisateurs à l’aide d’Auth0 Dashboard
---

Vous pouvez utiliser Auth0 Dashboard pour gérer vos utilisateurs. Pour commencer, accédez à [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users). C’est à cet endroit que vous pouvez créer, afficher, modifier ou supprimer des utilisateurs.

<Card title="La disponibilité varie selon le forfait Auth0">
  La disponibilité de cette fonctionnalité dépend à la fois de votre implémentation de connexion et de votre forfait Auth0 ou de votre entente personnalisée. Pour en savoir plus, consultez [Pricing](https://auth0.com/pricing).
</Card>

Par défaut, les attributs du profil utilisateur fournis par des <Tooltip tip="fournisseur d’identité (IdP) : service qui stocke et gère les identités numériques." cta="Consulter le glossaire" href="/fr-CA/docs/glossary?term=identity+providers">fournisseurs d’identité</Tooltip> autres qu’Auth0 (comme Google, Facebook ou X) ne peuvent pas être modifiés directement, car ils sont mis à jour à partir du fournisseur d’identité chaque fois que l’utilisateur se connecte. Pour en savoir plus sur les attributs provenant des fournisseurs d’identité, consultez [Configure Identity Provider Connection for User Profile Updates](/fr-CA/docs/manage-users/user-accounts/user-profiles/configure-connection-sync-with-auth0).

Pour pouvoir modifier les attributs racine `name`, `nickname`, `given_name`, `family_name` ou `picture` du profil utilisateur normalisé, vous devez [configurer la synchronisation de votre connexion avec Auth0](/fr-CA/docs/manage-users/user-accounts/user-profiles/configure-connection-sync-with-auth0) afin que les attributs de l’utilisateur soient mis à jour à partir du fournisseur d’identité uniquement lors de la création du profil utilisateur. Ces attributs racine pourront ensuite être modifiés individuellement ou par imports en masse.

Pour savoir comment gérer l’accès à <Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Consulter le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> pour les membres de votre équipe, consultez [Manage Dashboard Access](/fr-CA/docs/get-started/manage-dashboard-access).
