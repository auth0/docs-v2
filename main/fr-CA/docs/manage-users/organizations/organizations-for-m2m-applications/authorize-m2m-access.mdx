Pour autoriser l’accès Machine-to-Machine pour une application, vous devez l’autoriser à utiliser une organisation pour une API donnée. Pour ce faire, vous devez associer à l’organisation l’autorisation d’application de l’application pour l’API correspondante. Une fois l’association établie, l’application peut utiliser l’organisation lorsqu’elle demande des jetons pour l’API et les scopes définis dans l’autorisation d’application. Vous devrez définir cette association pour chaque API à laquelle votre application doit accéder pour l’organisation.

Vous pouvez autoriser l’accès M2M pour une application à l’aide de [Auth0 Dashboard](https://manage.auth0.com/) ou de la [Management API](https://auth0.com/docs/api/management/v2).

<Tabs>
  <Tab title="Auth0 Dashboard">
    Pour associer l’autorisation d’application d’une application à une organisation dans Auth0 Dashboard :

    1. Accédez à **Organizations** et choisissez l’organisation à associer.
    2. Sélectionnez l’onglet **Machine-to-Machine Access**.
    3. Cliquez sur **Add Access.**
    4. Cliquez sur l’application que vous voulez associer à l’organisation.
    5. Sélectionnez une API.
    6. Cliquez sur **Save.**
