---

Dans la section Organisations de votre [Auth0 Dashboard](https://manage.auth0.com/#/organizations), vous pouvez rechercher des [organisations](/fr-CA/docs/manage-users/organizations/organizations-overview) par nom ou par nom d’affichage.

<Frame>![Auth0 Dashboard - Organisations - Barre de recherche ](/docs/images/cdy7uua7fh8z/5CeeKLjPGxjTqy682lbnyj/3a41b87a369e65ca624b1822c99dbc57/2025-01-27_17-27-28.png)</Frame>

Le menu déroulant à côté de la barre de recherche vous permet d’effectuer une recherche par nom ou par nom d’affichage. Si vous saisissez trois caractères ou moins dans la barre de recherche, seules les correspondances exactes seront renvoyées. Si vous en saisissez quatre ou plus, des correspondances approximatives contenant la chaîne fournie seront renvoyées. Par exemple, si vous avez une organisation nommée « Big Holdings, Co. » et que vous saisissez les termes de recherche `hold` ou `ings`, l’organisation Big Holdings, Co. sera renvoyée.

Les recherches renvoient un maximum de 50 résultats. Si vous ne trouvez pas le résultat recherché, essayez d’utiliser un terme de recherche plus précis.

<Warning>
  La recherche d’organisations utilise un modèle de cohérence éventuelle. Les organisations créées ou modifiées récemment peuvent ne pas s’afficher immédiatement dans les résultats de recherche; toutefois, ceux-ci finiront par refléter fidèlement les entités de votre locataire.
</Warning>
