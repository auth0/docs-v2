---
description: Décrit comment voir les utilisateurs assignés à un rôle dans l’Auth0 Dashboard ou au moyen de la Management API.
title: Voir les utilisateurs assignés à des rôles
---

Vous pouvez voir les utilisateurs assignés à un [rôle](/fr-CA/docs/manage-users/access-control/rbac) dans l’<Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ou au moyen de la <Tooltip tip="Auth0 Dashboard : principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>. Les rôles sont utilisés dans le cadre de l’ensemble de fonctionnalités API Authorization Core.

<div id="prerequisites">
  ## Prérequis
</div>

Pour que le contrôle d’accès basé sur les rôles (RBAC) fonctionne correctement, vous devez l’activer pour votre API à l’aide de l’Auth0 Dashboard ou de la Management API. La fonctionnalité Authorization Core diffère de l’Authorization Extension. Pour comparer les deux, consultez [Authorization Core vs. Authorization Extension](/fr-CA/docs/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Auth0 Dashboard
</div>

1. Accédez à [Auth0 Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/users) et cliquez sur le nom du rôle à consulter.
2. Cliquez sur l’onglet **Users**.

Les renseignements suivants s’affichent pour chaque utilisateur :

| Colonne          | Description                                            |
| ---------------- | ------------------------------------------------------ |
| Image            | Image de l’utilisateur tirée de son profil.            |
| Nom              | Nom de l’utilisateur tiré de son profil.               |
| Adresse courriel | Adresse courriel de l’utilisateur tirée de son profil. |

<div id="management-api">
  ## Management API
</div>

Pour récupérer la liste des utilisateurs attribués à un rôle, consultez [Get a Role&#39;s Users](https://auth0.com/docs/api/management/v2#!/Roles/get_role_user).
