<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  L’ensemble de fonctionnalités [Authorization Core](/fr-CA/docs/manage-users/access-control/rbac) et [Authorization Extension](/fr-CA/docs/customize/extensions/authorization-extension) sont deux fonctionnalités entièrement distinctes. Pour gérer des groupes, des rôles ou des permissions, vous devez utiliser la fonctionnalité dans laquelle ils ont été créés à l’origine.
</Callout>

Pour que le contrôle d’accès basé sur les rôles (RBAC) fonctionne correctement, vous devez l’activer pour votre API à l’aide de l’Auth0 Dashboard ou de la <Tooltip tip="Management API : un produit qui permet aux clients d’effectuer des tâches administratives." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Management+API">Management API</Tooltip>. L’ensemble de fonctionnalités Authorization Core est distinct de l’Authorization Extension. Pour comparer les deux, consultez [Authorization Core vs. Authorization Extension](/fr-CA/docs/manage-users/access-control/authorization-core-vs-authorization-extension).

Nous offrons diverses fonctions pour vous aider à gérer vos rôles, accessibles soit dans l’<Tooltip tip="Auth0 Dashboard : le principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, soit dans l’Auth0 Management API :

* [Créer des rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/create-roles)
* [Modifier les définitions de rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/edit-role-definitions)
* [Ajouter des permissions aux rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/add-permissions-to-roles)
* [Retirer des permissions de rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/remove-permissions-from-roles)
* [Afficher les permissions d’un rôle](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/view-role-permissions)
* [Afficher les utilisateurs assignés à un rôle](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/view-users-assigned-to-roles)
* [Supprimer des rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/delete-roles)
