* [Ajouter des permissions aux rôles](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/add-permissions-to-roles)
* [Attribuer des permissions aux utilisateurs](/fr-CA/docs/manage-users/access-control/configure-core-rbac/rbac-users/assign-permissions-to-users)
* [Afficher les permissions d’un rôle](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/view-role-permissions)
* [Afficher les permissions d’un utilisateur](/fr-CA/docs/manage-users/access-control/configure-core-rbac/rbac-users/view-user-permissions)
* [Supprimer les permissions d’un rôle](/fr-CA/docs/manage-users/access-control/configure-core-rbac/roles/remove-permissions-from-roles)
