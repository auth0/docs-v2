title: Rechercher des utilisateurs
---

La recherche d’utilisateurs vous permet de récupérer les détails d’un profil utilisateur à l’aide de l’[Auth0 Management API](https://auth0.com/docs/api/management/v2). Les résultats de recherche peuvent être [affichés](/fr-CA/docs/manage-users/user-search/view-search-results-by-page), [triés](/fr-CA/docs/manage-users/user-search/sort-search-results) et [exportés](/fr-CA/docs/manage-users/user-migration/bulk-user-exports).

La plupart des champs du profil utilisateur ne sont pas renvoyés dans un [ID Token](/fr-CA/docs/secure/tokens/id-tokens) et ne sont pas non plus inclus dans la réponse du [point de terminaison `/userinfo`](https://auth0.com/docs/api/authentication#get-user-info) de l’Authentication API.

Lorsque vous recherchez des utilisateurs dans Auth0, vous pouvez utiliser plusieurs points de terminaison pour faire une recherche par ID, courriel ou selon d’autres critères :

<div id="search-result-terminology">
  ## Terminologie des résultats de recherche
</div>

Nous utilisons les termes suivants pour décrire les résultats de la recherche d’utilisateurs :

* **À cohérence éventuelle** : Les résultats de recherche peuvent ne pas refléter une opération d’écriture effectuée récemment. Toutefois, si vous répétez votre requête après un court délai, la réponse renverra des données à jour.
* **À cohérence immédiate** : Les résultats de recherche refléteront toutes les opérations d’écriture réussies, y compris celles effectuées peu avant votre requête.
