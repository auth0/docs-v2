title: Comptes utilisateur
---

La base de données infonuagique hébergée d’Auth0 stocke divers renseignements sur vos utilisateurs auxquels vous pouvez accéder. Ces renseignements sont disponibles dans un profil utilisateur, et vos utilisateurs sont regroupés par locataire. Les renseignements sur l’utilisateur peuvent provenir de diverses sources, notamment de <Tooltip tip="fournisseur d’identité (IdP) : Service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=identity+providers">fournisseurs d’identité</Tooltip>, de vos propres bases de données et de connexions d’entreprise (Active Directory, <Tooltip tip="fournisseur d’identité (IdP) : Service qui stocke et gère les identités numériques." cta="Voir le glossaire" href="/fr-CA/docs/glossary?term=SAML">SAML</Tooltip>).
