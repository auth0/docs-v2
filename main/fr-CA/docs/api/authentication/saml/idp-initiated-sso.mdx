`POST /login/callback`

Ce point de terminaison accepte une SAMLResponse d’authentification initiée par l’IdP provenant d’un fournisseur d’identité SAML. La connexion correspondant au fournisseur d’identité est spécifiée dans la chaîne de requête. L’utilisateur sera redirigé vers l’application spécifiée dans la section SAML Provider IdP-Initiated Sign On.

  ### Pour en savoir plus
</div>

* [SAML](https://auth0.com/docs/authenticate/protocols/saml/saml-configuration)

<div id="parameters">
  ## Paramètres
</div>

<ParamField body="connection" type="string" required>
  Le nom d’un fournisseur d’identité configuré pour votre application.
</ParamField>

<ParamField body="SAMLResponse" type="string" required>
  Une réponse SAML d’authentification initiée par l’IdP.
</ParamField>
