`POST /oauth/global-token-revocation/connection/{connection_name}`

Utilisez ce point de terminaison avec le [Okta Workforce Identity Cloud Universal Logout](https://developer.okta.com/docs/guides/oin-universal-logout-overview/) pour déconnecter les utilisateurs de vos applications. Une requête à ce point de terminaison révoque les cookies de session et les jetons d’actualisation, mais pas les jetons d’accès. Pour en savoir plus, consultez [Universal Logout](https://auth0.com/docs/authenticate/login/logout/universal-logout). La requête doit être authentifiée avant la révocation des sessions utilisateur. Consultez [Endpoint Authentication](https://developer.okta.com/docs/guides/oin-universal-logout-overview/#endpoint-authentication).

<div id="parameters">
  ## Paramètres
</div>

<ParamField path="connection_name" type="string" required>
  Le nom de la connexion pour laquelle vous souhaitez révoquer la session de l’utilisateur.
</ParamField>

<div id="request-body">
  ## Corps de la requête
</div>

<ParamField body="subject" type="object">
  <Expandable title="propriétés">
    <ParamField body="format" type="string" required />

    <ParamField body="iss" type="string" required>
      L’URL de l’émetteur (habituellement l’URL de votre serveur d’autorisation).
    </ParamField>

    <ParamField body="sub" type="string" required>
      L’identifiant du sujet (habituellement l’identifiant de l’utilisateur).
    </ParamField>
  </Expandable>
</ParamField>

<div id="response">
  ## Réponse
</div>

| Statut | Description                                                                                                    |
| ------ | -------------------------------------------------------------------------------------------------------------- |
| 200    | Les cookies de session et les jetons d’actualisation ont été révoqués avec succès.                             |
| 400    | Requête incorrecte, possiblement en raison d’un format `subject` invalide ou d’une authentification manquante. |
| 401    | Non autorisé, l’authentification a échoué.                                                                     |
| 500    | Erreur interne du serveur.                                                                                     |
