---
title: "Obtenir les métadonnées WS-Federation"
description: "Retourne les métadonnées WS-Federation pour un locataire Auth0."
---

<div id="endpoint">
  ## Point de terminaison
</div>

`GET /wsfed/FederationMetadata/2007-06/FederationMetadata.xml`

Ce point de terminaison retourne les métadonnées WS-Federation.

<div id="learn-more">
  ### En savoir plus
</div>

* [WS-Federation](https://auth0.com/docs/authenticate/protocols/ws-fed-protocol)

<div id="response-messages">
  ## Messages de réponse
</div>

| Statut | Description                                              |
| ------ | -------------------------------------------------------- |
| 200    | Réponse réussie contenant les métadonnées WS-Federation. |
