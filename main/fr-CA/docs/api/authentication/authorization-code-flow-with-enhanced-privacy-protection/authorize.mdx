`GET /authorize`

Après avoir appelé le point de terminaison `/oauth/par`, redirigez l’utilisateur final vers le point de terminaison `/authorize` à l’aide d’une requête `GET`.

<Note>
  Le point de terminaison `/authorize` répond en fonction des paramètres transmis au point de terminaison `/oauth/par`. Si vous demandez un `response_type`, vous devriez recevoir un code d’autorisation à utiliser avec le point de terminaison `/oauth/token`.
</Note>

<div id="parameters">
  ## Paramètres
</div>

<ParamField query="client_id" type="string" required>
  Le `client_id` de votre application. Obligatoire.
</ParamField>

<ParamField query="request_uri" type="string" required>
  La valeur `request_uri` reçue depuis le point de terminaison `/oauth/par`. Obligatoire.
</ParamField>

<div id="response">
  ## Réponse
</div>

| Statut | Description                                  |
| ------ | -------------------------------------------- |
| 302    | Redirige vers l’URI de redirection indiquée. |
