description: "Point de terminaison obsolète pour dissocier des comptes d’utilisateur; utilisez plutôt la Management API."
---

<div id="endpoint">
  ## Point de terminaison
</div>

`POST /login/unlink`

<Note>
  Ce point de terminaison est **obsolète**. Utilisez plutôt [DELETE /api/v2/users/{id}/identities/{provider}/{user_id}](/fr-CA/api/management/v2/users/delete-user-identity-by-user-id).
</Note>

Avec le `access_token` et le `user_id` d’un utilisateur connecté, ce point de terminaison dissocie le compte d’un utilisateur du fournisseur d’identité.

<div id="learn-more">
  ### En savoir plus
</div>

* [Dissocier des comptes d’utilisateur](https://auth0.com/docs/manage-users/user-accounts/user-account-linking/unlink-user-accounts)

  ## Messages de réponse
</div>

| Statut | Description                             |
| ------ | --------------------------------------- |
| 200    | Compte utilisateur dissocié avec succès |
