---
title: "Lister les authentificateurs"
description: "Renvoie une liste d’authentificateurs associés à votre application."
---

`GET /mfa/authenticators`

Renvoie une liste d’authentificateurs associés à votre application.

Pour accéder à ce point de terminaison, vous devez inclure un Jeton d’accès dans l’en-tête Authorization, avec les claims suivants :

* `scope`: `read:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="parameters">
  ## Paramètres
</div>

<ParamField header="Authorization" type="string" required>
  Bearer ACCESS&#95;TOKEN
</ParamField>

<div id="response">
  ## Réponse
</div>

| Statut | Description                  |
| ------ | ---------------------------- |
| 200    | Liste des authentificateurs. |
