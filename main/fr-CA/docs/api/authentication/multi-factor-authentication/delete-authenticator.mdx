---
title: "Supprimer un authentificateur"
description: "Supprimer l’authentificateur MFA associé à l’aide de son id."
---

<div id="endpoint">
  ## Point de terminaison
</div>

`DELETE /mfa/authenticators/{AUTHENTICATOR_ID}`

Supprime l’authentificateur associé à l’aide de son id. Vous pouvez obtenir les id des authentificateurs en répertoriant les authentificateurs.

Pour accéder à ce point de terminaison, vous devez inclure un Jeton d’accès dans l’en-tête Authorization, avec les claims suivantes :

* `scope`: `remove:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="path-parameters">
  ## Paramètres de chemin
</div>

<ParamField path="AUTHENTICATOR_ID" type="string" required>
  L’identifiant de l’authentificateur à supprimer.
</ParamField>

<div id="headers">
  ## En-têtes
</div>

<ParamField header="Authorization" type="string" required>
  Le Jeton d’accès obtenu lors de la connexion.
</ParamField>

<div id="response-messages">
  ## Messages de réponse
</div>

| Statut | Description                              |
| ------ | ---------------------------------------- |
| 204    | Authentificateur supprimé avec succès.   |
| 401    | Non autorisé - jeton d’accès non valide. |
| 404    | Authentificateur introuvable.            |
