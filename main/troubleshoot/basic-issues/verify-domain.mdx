* Ensure that the domain is the same as that used during authentication.
* If using an Auth0 [Custom Domain](/customize/custom-domains), it is important to use the same domain as used in the application to invoke authentication. See [Troubleshoot Custom Domains](/troubleshoot/integration-extensibility-issues/troubleshoot-custom-domains) for details.
