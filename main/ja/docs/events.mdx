<div id="use-cases">
  ## ユースケース
</div>

イベントは、Auth0 テナント内で発生した特定のアクションに関するリアルタイム通知です。対象は当初、ユーザーや組織に対する変更ですが、今後はさらに拡大される予定です。これらのイベントを監視する Event Stream を作成すると、Auth0 内で発生した非同期の変更 (つまり、すでに完了したエンティティまたはフローへの変更) をオーケストレーションし、外部で処理するためにさまざまな宛先のいずれかに転送できます。イベントは、たとえば次のようなさまざまなユースケースで利用できます。

* 新規顧客にウェルカムメールを送信したり、メールアドレスの確認を依頼したりする。
* ユーザーのライフサイクルの変更を監視し、CRM (顧客関係管理) システムや請求システムを更新する。
