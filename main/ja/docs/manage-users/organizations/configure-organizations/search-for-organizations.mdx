description: Auth0 Dashboard を使用して組織を検索する方法について説明します。
title: 組織を検索する
---

[Auth0 Dashboard](https://manage.auth0.com/#/organizations) の Organizations セクションでは、名前または表示名で[組織](/ja/docs/manage-users/organizations/organizations-overview)を検索できます。

<Frame>![Auth0 Dashboard - 組織 - 検索バー ](/docs/images/cdy7uua7fh8z/5CeeKLjPGxjTqy682lbnyj/3a41b87a369e65ca624b1822c99dbc57/2025-01-27_17-27-28.png)</Frame>

検索バーの横にあるドロップダウンメニューでは、名前または表示名で検索できます。検索バーに 3 文字以下を入力した場合は、完全一致のみが返されます。4 文字以上を入力すると、指定した文字列を含むあいまい一致が返されます。たとえば、`Big Holdings, Co.` という名前の組織があり、検索語として `hold` または `ings` を入力すると、Big Holdings, Co. 組織が返されます。

検索結果は最大 50 件です。目的の結果が見つからない場合は、より具体的な検索語を使用してください。

<Warning>
  組織の検索では結果整合性が採用されています。最近作成または変更された組織は、検索結果にすぐに表示されない場合がありますが、最終的にはテナント内のエンティティが正確に反映されます。
</Warning>
