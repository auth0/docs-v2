---
title: 組織ドメインを作成
description: 検証済みの組織ドメインを使用して、サインアップ時とログインフロー時に、ユーザーが正しい組織を選択できるようにします。
---

組織ドメインは、[Organization Domain Discovery](./define-organization-behavior) と連携して、サインアップ時および[ログインフロー](/ja/docs/manage-users/organizations/login-flows-for-organizations)時にユーザーが正しい組織を見つけられるよう支援します。

設定すると、検証済みの組織ドメインと Organization Domain Discovery により、ユーザーがメールアドレスを入力した際に、Auth0 はそのユーザーの組織を自動的に検出したり、組織の候補を絞り込んだりできます。組織ドメインと Organization Domain Discovery は、サインアップまたはログインできるユーザーを制限するものではありません。

<div id="create-an-organization-domain-with-the-auth0-dashboard">
  ## Auth0 Dashboard で組織ドメインを作成する
</div>

Auth0 Dashboard で組織ドメインを作成するには、次の手順に従います。

1. [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) に移動します。
2. **Domains** タブを選択します。
3. **Add Domain** を選択します。
   **Domain** フィールドに、組織の検出に使用するメールドメインを入力します。所有権が確認されるまで、ステータスは保留中です。
4. **Add Domain** を選択します。
5. 必要に応じて、[Organization Domain Discovery](./define-organization-behavior) を設定します。

<div id="create-an-organization-domain-with-the-management-api">
  ## Management API を使用して組織ドメインを作成する
</div>

Management API を使用して組織ドメインを作成するには、[Create an organization discovery domain endpoint](/ja/docs/api/management/v2/organizations/post-discovery-domains) (`POST /organizations/{id}/discovery-domains`) を呼び出します。必要に応じて、`use_for_organization_discovery` パラメーターを使用して、そのドメインで Organization Domain Discovery を有効にします。

Organization Domain Discovery を有効にするには、[Patch clients by ID endpoint](/ja/docs/api/management/v2/clients/patch-clients-by-id#body-organization-discovery-methods-one-of-0) (`PATCH /client/{id}`) を呼び出し、`organization_discovery_methods` パラメーターを設定します。
