ユーザー検索結果をページ分けするには、リクエストで `page`、`per_page`、`include_totals` パラメーターを使用します。

| Parameter        | Description                                                       |
| ---------------- | ----------------------------------------------------------------- |
| `page`           | ページ番号。0 から始まります。これが設定されていない場合は、存在するレコード数にかかわらず、最大 50 件のレコードを返します。 |
| `per_page`       | 1 ページあたりのユーザー数。                                                   |
| `include_totals` | クエリのサマリーを結果に含めるには、`true` に設定します。                                  |

<div id="limitation">
  ## 制限事項
</div>

Auth0 では、取得できるユーザーの総数は 1000 件に制限されています (たとえば、1 ページあたり 100 ユーザーを 10 ページ分) 。`page` パラメーターが設定されていない場合、存在するレコード数にかかわらず、返されるレコードは最大 50 件です。
