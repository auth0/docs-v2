title: ユーザーを検索
---

ユーザー検索では、Auth0 の [Management API](https://auth0.com/docs/api/management/v2) を使用して、ユーザープロファイルの詳細を取得できます。検索結果は、[表示](/ja/docs/manage-users/user-search/view-search-results-by-page)、[並べ替え](/ja/docs/manage-users/user-search/sort-search-results)、および [エクスポート](/ja/docs/manage-users/user-migration/bulk-user-exports) できます。

ほとんどのユーザープロファイルフィールドは [IDトークン](/ja/docs/secure/tokens/id-tokens) の一部として返されず、Authentication API の [`/userinfo` エンドポイント](https://auth0.com/docs/api/authentication#get-user-info) からのレスポンスにも含まれません。

Auth0 でユーザーを検索する場合は、複数のエンドポイントを使用して ID、メールアドレス、またはその他の条件で検索できます。

<div id="search-result-terminology">
  ## 検索結果の用語
</div>

ユーザーの検索結果を説明するために、次の用語を使用します。

* **結果整合性**: 検索結果に、直前に完了した書き込み操作がまだ反映されていない場合があります。ただし、少し時間をおいてリクエストを再実行すると、レスポンスには最新のデータが返されます。
* **即時整合性**: 検索結果には、リクエストの直前に行われたものも含め、成功したすべての書き込み操作の結果が反映されます。
