---
description: Auth0 Dashboard でユーザーを作成する方法について説明します。
title: ユーザーを作成する
---

Auth0 Dashboard または [Auth0 Management API](https://auth0.com/docs/api/management/v2/users/post-users) を使用してユーザーを作成できます。

1. [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) に移動し、**Create User** を選択します。
2. **Connection** を選択し、ユーザーの **Email**、**Password** を入力します。

   | 属性      | 説明                                                                                                                                       |
   | ------- | ---------------------------------------------------------------------------------------------------------------------------------------- |
   | 接続      | ユーザーの認証に使用するデータベース接続です。ドロップダウンには、テナントで設定されているすべてのデータベース接続が表示されます。使用する接続はアプリケーションに関連付けられている必要があります。そうでない場合は、接続が無効であることを示すエラーメッセージが表示されます。 |
   | メールアドレス | ユーザーのメールアドレスです。メールアドレスの形式は、[IETF の RFC 5321](https://datatracker.ietf.org/doc/html/rfc5321#section-4.1.2) に記載されている仕様に準拠している必要があります。      |
   | パスワード   | ユーザーのパスワードです。パスワードの長さに上限はありません。                                                                                                          |
3. **Create** を選択します。
   ユーザーが作成され、新しく作成したユーザーのユーザープロファイルに移動します。

[Multiple Custom Domains](/ja/docs/customize/custom-domains/multiple-custom-domains) を有効にすると、**Connection** セクションで **Domain** を選択して、テナントの通知ドメインを選択できます。

[Flexible Identifiers](/ja/docs/authenticate/database-connections/flexible-identifiers-and-attributes) を有効にすると、**email** に加えて、ユーザーの **username**、**phone**、またはそれらを 2 つ以上組み合わせて選択できます。
