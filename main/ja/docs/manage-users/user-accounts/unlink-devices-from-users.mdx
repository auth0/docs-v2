description: Auth0 Dashboard を使用して、ユーザーに割り当てられたデバイスの関連付けを解除する方法を説明します。
title: ユーザーからデバイスの関連付けを解除する
---

<Tooltip tip="Auth0 のサービスを設定するための主要な製品である Auth0 Dashboard。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> を使用して、ユーザーに割り当てられたデバイスの関連付けを解除できます。これにより、選択したデバイスの <Tooltip tip="Auth0 のサービスを設定するための主要な製品である Auth0 Dashboard。" cta="用語集を見る" href="/ja/docs/glossary?term=refresh+token">リフレッシュトークン</Tooltip> が失効します。

1. [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) に移動し、ユーザーの名前をクリックします。
2. **Devices** ビューをクリックし、関連付けを解除するデバイスの横にあるごみ箱アイコンをクリックして確認します。
