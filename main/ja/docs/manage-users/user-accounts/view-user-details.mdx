description: ユーザーとそのプロフィールの詳細を表示する方法。
title: ユーザー詳細を表示
---

[Users](https://manage.auth0.com/#/users) ページには、アプリに関連付けられているユーザーの一覧が表示されます。特定のユーザーを開くには、[Name] 列で該当するユーザーのプロフィール画像または名前をクリックします。ユーザーの詳細 ページが開き、そのユーザーの情報が表示されます。

ユーザーの詳細 ページには、次の 5 つのタブへのリンクがあります。

1. 詳細
2. デバイス
3. 履歴
4. 場所
5. Raw JSON

<div id="user-details-details-tab">
  ## ユーザーの詳細: 詳細タブ
</div>

詳細タブには、ユーザーのプロフィール情報の概要を確認できる 5 つのセクションがあります。

* **ユーザー識別情報** では、ユーザーが使用した識別子、関連する接続、アクセス権限などをひと目で確認できます。
* **メタデータ** には、`app_metadata` と `user_metadata` の情報が表示されます。これらの値は編集できます。
* **<Tooltip tip="IDプロバイダー（IdP）: デジタル ID を保存および管理するサービス。" cta="用語集を表示" href="/ja/docs/glossary?term=Identity+Provider">IDプロバイダー</Tooltip> の属性** には、認証プロバイダーから取得した情報が表示されます。IDプロバイダーの属性は読み取り専用である点に注意してください。
* **Passkeys** には、ユーザーが作成した [passkeys](/ja/docs/authenticate/database-connections/passkeys) が表示されます。
* **<Tooltip tip="多要素認証（MFA）: SMS で送信されるコードなど、ユーザー名とパスワードに加えて別の要素を使用するユーザー認証プロセス。" cta="用語集を表示" href="/ja/docs/glossary?term=Multi-Factor+Authentication">多要素認証</Tooltip>** には、ユーザーが登録した [MFA](/ja/docs/secure/multi-factor-authentication) 要素が表示されます。
  [Multiple Custom Domains](/ja/docs/customize/custom-domains/multiple-custom-domains) を有効にすると、テナントの通知ドメインを選択する **ドメイン** オプションを使用できます。

<div id="user-details-devices-tab">
  ## ユーザーの詳細: デバイス タブ
</div>

[デバイス] タブには、ユーザーが認証をリクエストしたデバイスの一覧が表示されます。デバイスで認可をリクエストすると、そのデバイスはユーザーのアカウントに関連付けられます。

ユーザーのログイン情報は、そのデバイスに割り当てられた<Tooltip tip="リフレッシュトークン: ユーザーに再ログインさせることなく、新しいアクセストークンを取得するために使用されるトークンです。" cta="用語集を表示" href="/ja/docs/glossary?term=Refresh+Token">リフレッシュトークン</Tooltip>に関連付けられます。リフレッシュトークンを無効にするには、デバイスの横にある**リンク解除**をクリックします。

<div id="user-details-history-tab">
  ## ユーザーの詳細: 履歴タブ
</div>

[履歴] タブには、過去 2 日間のユーザーのアカウント アクティビティのログが表示されます。

ログには、次の情報が含まれます。

* 発生したイベント
* イベントの発生日時
* イベントに関連付けられたアプリ
* 認証に使用された IDプロバイダー
* イベントの送信元 IP アドレス
* イベントの発生元

<div id="user-details-locations-tab">
  ## ユーザーの詳細: 場所 タブ
</div>

Locations タブには、ユーザーがアプリにログインした際の位置を示すピンが表示された地図が表示されます。

<div id="user-details-raw-json-tab">
  ## ユーザーの詳細: Raw JSON タブ
</div>

Raw JSON タブには、ユーザーのプロフィールに含まれるすべての情報が JSON 形式で表示されるため、ユーザーに関する利用可能な情報をすばやく確認できます。
