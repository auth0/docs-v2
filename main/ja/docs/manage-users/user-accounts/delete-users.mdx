---
description: ユーザーを削除する方法を説明します。
title: ユーザーの削除
---

1. [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) に移動します。
2. 削除するユーザーの名前をクリックします。
3. **Details** タブで最下部までスクロールし、**Delete user** の横にある **Delete** をクリックします。<br />

   この操作は元に戻せないという警告のポップアップが表示され、ユーザーを削除するかどうかの確認を求められます。確認するには、**Yes, Delete It** ボタンをクリックします。これにより、ユーザーは完全に削除されます。
