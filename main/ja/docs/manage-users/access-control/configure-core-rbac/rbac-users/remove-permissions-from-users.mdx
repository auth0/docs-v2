ユーザーに直接割り当てられた権限は、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要な製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> または <Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要な製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使用して削除できます。割り当てられた権限は、API の Authorization Core feature set で使用されます。詳細については、[ロールベースアクセス制御](/ja/docs/manage-users/access-control/rbac) を参照してください。

<div id="prerequisites">
  ## 前提条件
</div>

ロールベースアクセス制御 (RBAC) を正しく機能させるには、Dashboard または Management API を使用して、API で RBAC を有効にする必要があります。Authorization Core の機能は Authorization Extension とは異なります。比較については、[Authorization Core と Authorization Extension の比較](/ja/docs/manage-users/access-control/authorization-core-vs-authorization-extension)を参照してください。

<div id="use-the-dashboard">
  ## Dashboard を使用する
</div>

1. [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) に移動し、対象のユーザーの名前をクリックします。
2. **Permissions** ビューを開き、削除する権限の横にあるごみ箱アイコンをクリックして確認します。

   <Frame>![このユーザーに割り当てられている権限の一覧を示す Auth0 Dashboard のユーザー詳細画面](/docs/images/cdy7uua7fh8z/4vShfxoUcSpCDnM5FnAbsR/de85942623430e3cedc097addbaa2f9d/User_Details_-_Permissions_-_English.png)</Frame>

<div id="use-the-management-api">
  ## Management API を使用する
</div>

[Delete User Permissions endpoint](https://auth0.com/docs/api/management/v2#!/Users/delete_permissions) に対して `DELETE` リクエストを実行します。`USER_ID`、`MGMT_API_ACCESS_TOKEN`、`API_ID`、`PERMISSION_NAME` の各プレースホルダー値は、それぞれユーザー ID、Management API の <Tooltip tip="アクセストークン: API へのアクセスに使用される、不透明な文字列または JWT 形式の認可資格情報です。" cta="用語集を表示" href="/ja/docs/glossary?term=access+token">アクセストークン</Tooltip>、API ID、権限名に必ず置き換えてください。

  ```bash cURL
  curl --request DELETE \
    --url 'https://{yourDomain}/api/v2/users/USER_ID/permissions' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN' \
    --header 'cache-control: no-cache' \
    --header 'content-type: application/json' \
    --data '{ "permissions": [ { "resource_server_identifier": "API_ID", "permission_name": "PERMISSION_NAME" }, { "resource_server_identifier": "API_ID", "permission_name": "PERMISSION_NAME" } ] }'
  ```

</AuthCodeGroup>

| 値                       | 説明                                                                                                   |
| ----------------------- | ---------------------------------------------------------------------------------------------------- |
| `USER_ID`               | 更新対象のユーザーの ID。                                                                                       |
| `MGMT_API_ACCESS_TOKEN` | スコープ `update:users` を持つ [Management API のアクセストークン](https://auth0.com/docs/api/management/v2/tokens)。 |
| `API_ID`                | 指定したユーザーから削除する権限に関連付けられている API の ID。                                                                 |
| `PERMISSION_NAME`       | 指定したユーザーから削除する権限の名前。                                                                                 |
