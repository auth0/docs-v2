Auth0 の Dashboard を使用すると、ユーザーに割り当てられている [ロール](/ja/docs/manage-users/access-control/rbac) を確認できます。この操作は、<Tooltip tip="Management API: お客様が管理タスクを実行できるようにする製品。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使用して行うこともできます。割り当てられたロールは、API Authorization Core の機能セットで使用されます。

<div id="prerequisites">
  ## 前提条件
</div>

ロールベースアクセス制御 (RBAC) を正しく機能させるには、Dashboard または Management API を使用して、API に対してこれを有効にする必要があります。Authorization Core の機能は Authorization Extension とは異なります。比較については、[Authorization Core と Authorization Extension の比較](/ja/docs/manage-users/access-control/authorization-core-vs-authorization-extension)を参照してください。

<div id="dashboard">
  ## Dashboard
</div>

1. [Dashboard &gt; ユーザー管理 &gt; ユーザー](https://manage.auth0.com/#/users)に移動し、表示するユーザーの名前をクリックします。
2. **Roles**タブをクリックします。

各ロールについて、次の情報が表示されます。

| 列  | 説明             |
| -- | -------------- |
| 名前 | ロール定義で設定された名前。 |
| 説明 | ロール定義で設定された説明。 |

<div id="management-api">
  ## Management API
</div>

[Get User Roles endpoint](https://auth0.com/docs/api/management/v2#!/Users/get_user_roles) に `GET` リクエストを送信します。プレースホルダー値 `USER_ID` と `MGMT_API_ACCESS_TOKEN` は、それぞれ対象のユーザー ID と Management API の <Tooltip tip="アクセストークン: API へのアクセスに使用される、不透明な文字列または JWT 形式の認可資格情報です。" cta="用語集を表示" href="/ja/docs/glossary?term=access+token">アクセストークン</Tooltip> に必ず置き換えてください。

<AuthCodeGroup>
  ```bash cURL
  curl --request GET \
    --url 'https://{yourDomain}/api/v2/users/USER_ID/roles' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/users/USER_ID/roles");
  var request = new RestRequest(Method.GET);
  request.AddHeader("authorization", "Bearer MGMT_API_ACCESS_TOKEN");
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (
  	"fmt"
  	"net/http"
  	"io/ioutil"
  )

  func main() {

  	url := "https://{yourDomain}/api/v2/users/USER_ID/roles"

  	req, _ := http.NewRequest("GET", url, nil)

  	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```

| 値                       | 説明                                                                                                                  |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------- |
| `USER_ID`               | ロールを取得するユーザーの ID。                                                                                                   |
| `MGMT_API_ACCESS_TOKEN` | スコープ `read:users` および `read:roles` を持つ [Management API のアクセストークン](https://auth0.com/docs/api/management/v2/tokens)。 |
