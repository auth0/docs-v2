<Card title="利用可否は Auth0 プランによって異なります">
  この機能を利用できるかどうかは、お使いのログイン実装に加え、Auth0 のプランまたはカスタム契約によって異なります。詳しくは、[Pricing](https://auth0.com/pricing) を参照してください。
</Card>

組織機能は Auth0 プラットフォームに対する大きな拡張であり、B2B のお客様がパートナーや顧客をより適切に管理し、エンドユーザーによるアプリケーションや API へのアクセス方法をカスタマイズできるようにするものです。Auth0 のお客様は、組織を使用して次のことを行えます。

* Auth0 内で取引先やパートナーを表現し、そのメンバーシップを管理する。
* 取引先ごとに、ブランド化されたフェデレーションログインフローを設定する。
* 取引先やパートナーが B2B API を利用できるように、マシンツーマシンアクセスを提供する。
* 組織 API を使用して製品に管理機能を組み込み、それらの企業が自社の組織を管理できるようにする。
