---
description: Auth0 のパブリッククラウドおよびプライベートクラウドのデプロイオプションと、デプロイ用のチェックリストやツールについて説明します。
title: デプロイと監視
---

Auth0 は、パブリッククラウド、AWS 上のプライベートクラウド、または Azure 上のプライベートクラウドにデプロイできます。デプロイ用のツールとチェックリストを使うことで、導入をスムーズに進められます。監視ツールとイベントログを使用すると、システムの健全性やイベントを追跡できます。
