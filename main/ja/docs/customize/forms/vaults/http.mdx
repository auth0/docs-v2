<div id="configure-your-http-vault-connection-options">
  ## HTTP Vault 接続オプションを設定する
</div>

HTTP Vault 接続は、さまざまな認証オプションで設定でき、その後 HTTP Request Flow Actions で使用できます。

<Frame>![Dashboard > Actions > Forms > Vault > HTTP](/docs/images/customize/forms/vaults/http-vault-connection-options.png)</Frame>

<div id="configure-your-http-vault-connection-for-bearer-token">
  ## Bearer Token 用の HTTP Vault 接続 を設定する
</div>

**Bearer Token** 認可を使用して HTTP Vault 接続 を設定するには、必要なパラメーターを指定します。

<Frame>![Dashboard > Actions > Forms > Vault > HTTP](/docs/images/customize/forms/vaults/http-vault-connection-bearer-token.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメーター       | 説明                     |
| ------------ | ---------------------- |
| Bearer Token | 外部サービスの認可に使用する静的なトークン。 |

<div id="configure-your-http-vault-connection-for-oauth-client-credentials">
  ## OAuth Client Credentials 用に HTTP Vault 接続を設定する
</div>

**Client Credentials** 認可を使用して HTTP Vault 接続を設定するには、必要なパラメーターを指定する必要があります。

<Frame>![Dashboard > Actions > Forms > Vault > HTTP ](/docs/images/customize/forms/vaults/http-vault-connection-oauth-client-credentials.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| Parameter           | Description                                            |
| ------------------- | ------------------------------------------------------ |
| Tenant Endpoint     | アクセストークンを発行する認可サーバーのテナントエンドポイント。                       |
| Client ID           | Custom API 用に認可された M2M アプリケーションのクライアントID。              |
| Client Secret       | Custom API 用に認可された M2M アプリケーションのクライアントシークレット。          |
| Audience (Optional) | Custom API のオーディエンス。通常は Auth0 で定義した Custom API で使用します。 |
| Resource (Optional) | Custom API のリソース。Audience と同様の意味を持つ標準パラメーターです。         |
| Scope (Optional)    | スペース区切りの文字列形式のスコープリスト。                                 |

<div id="configure-your-http-vault-connection-for-api-key">
  ## API キー用の HTTP Vault 接続を設定する
</div>

**API キー**認可を使用して HTTP Vault 接続を設定するには、必要なパラメーターを指定する必要があります。

<Frame>![Dashboard > Actions > Forms > Vault > HTTP](/docs/images/customize/forms/vaults/http-vault-connection-api-key.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| Parameter | Description                                            |
| --------- | ------------------------------------------------------ |
| 名前        | ヘッダーまたはクエリパラメーターとして追加する API キーの名前。                     |
| 値         | ヘッダーまたはクエリパラメーターとして追加する API キーの値。                      |
| 追加先       | デフォルトではヘッダーです。API キーをヘッダーまたはクエリパラメーターのどちらに追加するかを指定します。 |

<div id="configure-your-http-vault-connection-for-basic-authentication">
  ## Basic Authentication 用に HTTP Vault 接続を設定する
</div>

**Basic Auth** を使用する HTTP Vault 接続を設定するには、必要なパラメーターを指定する必要があります。

<Frame>![Dashboard > Actions > Forms > Vault > HTTP ](/docs/images/customize/forms/vaults/http-vault-connection-basic-authentication.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメーター | 説明                                                             |
| ------ | -------------------------------------------------------------- |
| ユーザー名  | 連結して base64 エンコードし、Basic Authorization ヘッダーパラメーターとして追加するユーザー名。 |
| パスワード  | 連結して base64 エンコードし、Basic Authorization ヘッダーパラメーターとして追加するパスワード。 |

<Warning>
  Basic Auth 構成の HTTP Vault 接続は、Flows HTTP Request Action に現在組み込まれているオプションを置き換える予定です。
</Warning>
