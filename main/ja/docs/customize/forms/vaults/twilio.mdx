<div id="get-your-twilio-credentials">
  ## Twilio の認証情報を取得する
</div>

[Twilio のコンソールダッシュボード](https://console.twilio.com/)で取得できます。

<div id="configure-your-twilio-vault-connection">
  ## Twilio Vault 接続を設定する
</div>

Twilio Flow Actions 用の Twilio Vault 接続を構成するには、必要なパラメーターを設定します。

<Frame>![Dashboard > Actions > Forms > Vault > Twilio](/docs/images/customize/forms/vaults/twilio-vault-connection.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメーター      | 説明                               |
| ----------- | -------------------------------- |
| Account SID | Twilio API の認証に使用する Account SID。 |
| Auth token  | Twilio API の認証に使用する Auth Token。  |
