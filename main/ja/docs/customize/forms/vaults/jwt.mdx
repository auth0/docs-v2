<div id="configure-your-jwt-vault-connection">
  ## JWT Vault 接続を設定する
</div>

Flows JWT Actions で JSON Web Token の署名、デコード、検証を行うために JWT Vault 接続を設定するには、必要なパラメーターを設定する必要があります。

<Frame>![Dashboard > Actions > Forms > Vault > JWT](/docs/images/customize/forms/vaults/jwt-vault-connection.png)</Frame>

<div id="input-settings">
  ## 入力設定
</div>

| Parameter | Description                           |
| --------- | ------------------------------------- |
| Secret    | JWT のエンコードに使用する、HS256 アルゴリズム用のシークレット。 |
