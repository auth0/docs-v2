<div id="get-your-sendgrid-keys">
  ## SendGrid キーを取得する
</div>

[SendGrid ダッシュボード](https://app.sendgrid.com/settings/api_keys) の Settings ページにある API keys で確認できます。

<div id="configure-your-sendgrid-vault-connection">
  ## SendGrid Vault 接続を設定する
</div>

SendGrid Flow Actions 用の SendGrid Vault 接続を設定するには、必要なパラメーターを指定します。

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメーター  | 説明                            |
| ------- | ----------------------------- |
| API Key | SendGrid API の認可に使用する API キー。 |
