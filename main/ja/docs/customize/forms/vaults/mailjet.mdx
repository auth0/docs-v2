<div id="get-your-mailjet-keys">
  ## MailJet キーを取得する
</div>

キーは、[MailJet Dashboard](https://app.mailjet.com/account/api_keys) の Account ページにある API keys で確認できます。

<div id="configure-your-mailjet-vault-connection">
  ## MailJet Vault 接続を設定する
</div>

MailJet Flow Actions 用の MailJet Vault 接続を設定するには、必要なパラメーターを設定します。

<Frame>![ダッシュボード > Actions > Forms > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメーター     | 説明                            |
| ---------- | ----------------------------- |
| API Key    | MailJet API の認証に使用する API キー。  |
| Secret Key | MailJet API の認証に使用するシークレットキー。 |
