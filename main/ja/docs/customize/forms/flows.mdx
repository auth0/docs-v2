Flows を使用すると、Forms 内で実行するサーバー側ロジックを視覚的に構築し、収集した情報の処理、API を介した各種サービスとの統合、自動化のトリガーを行えます。

<Frame>![Dashboard > Forms > Flows](/docs/images/cdy7uua7fh8z/6u6Huh5CbH3GEQJv3AH7cx/e49ea613b2c46a43c83c89ffecddbafe/Flows.png)</Frame>

フォームで Flows のアクションを使用すると、次のことができます。

* フォームの各ステップに対して、カスタムのサーバー側検証を作成する。
* カスタム HTTP リクエストを実行して、独自の API と統合する。
* すぐに使える統合を使用して、サードパーティサービスと統合する。
* ユーザー入力やプロパティに応じて、異なるビジネスロジックを実行する。
* <Tooltip tip="JSON Web Token (JWT): 2 者間でクレームを安全に表現するために使用される標準的な IDトークン形式（また、多くの場合はアクセストークン形式）。" cta="用語集を見る" href="/ja/docs/glossary?term=JSON+Web+Tokens">JSON Web Tokens</Tooltip> を使用してデータを検証する。
