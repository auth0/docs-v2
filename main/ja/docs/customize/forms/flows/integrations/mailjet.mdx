  ## メールを送信
</div>

メールプロバイダーに Mailjet を使用してメールを送信します。

<div id="input-settings">
  ### 入力設定
</div>

| Parameter        | Description                                                                           |
| ---------------- | ------------------------------------------------------------------------------------- |
| 送信者名             | 送信者の名前                                                                                |
| 送信者メールアドレス (必須)  | 送信者のメールアドレス                                                                           |
| 受信者名             | 受信者の名前                                                                                |
| 受信者メールアドレス (必須)  | 受信者のメールアドレス                                                                           |
| CC名              | CCの名前                                                                                 |
| CCメールアドレス        | CCのメールアドレス                                                                            |
| 件名               | メールの件名                                                                                |
| メールタイプ           | HTML コンテンツのメールを送信するか、保存済みテンプレートを使用して送信します (任意)                                        |
| Message (必須)     | HTML 形式のメールメッセージ、またはテンプレート (任意) 。                                                     |
| テンプレート ID (任意)   | テンプレートを使用する場合、その ID は [ Mailjet のダッシュボード ](https://app.mailjet.com/template) で確認できます。 |
| カスタム変数 (任意)      | テンプレートに挿入するカスタム変数を追加します。                                                              |
| メールアクティビティを追跡    | メールアクティビティを追跡するには、このチェックボックスをオンにします。                                                  |

<div id="output-object">
  ### 出力オブジェクト
</div>

想定されるレスポンスとステータスコードについては、[Mailjet のドキュメント](https://dev.mailjet.com/email/reference/send-emails/)を参照してください。
