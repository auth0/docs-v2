<div id="send-telegram-message">
  ## Telegram メッセージを送信
</div>

Telegram チャットにメッセージを送信します

<Frame>![](/docs/images/cdy7uua7fh8z/kgSIPsWN5djsT1yQqg5Z8/fea0821d0e919c5101d01b84b968dd53/sendtelegrammessage.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| パラメーター        | 説明                                                                                |
| ------------- | --------------------------------------------------------------------------------- |
| Chat ID (必須)  | Telegram のチャット ID。[Telegram の BotFather](https://core.telegram.org/bots) から取得します。 |
| Messages (必須) | Telegram メッセージの本文です。                                                              |

<div id="output-object">
  ### 出力オブジェクト
</div>

想定されるレスポンスとステータスコードについては、[Telegram のドキュメント](https://core.telegram.org/method/messages.sendMessage)を参照してください。
