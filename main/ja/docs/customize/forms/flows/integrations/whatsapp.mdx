  ## メッセージを送信
</div>

メッセージを送信します。

<Frame>![](/docs/images/cdy7uua7fh8z/5zZPQw5KKYwBqxbR9jF1wZ/edba085456e3536f5167ff4c21bea77c/whatsapp.png)</Frame>

<div id="input-settings">
  ### 入力設定
</div>

| Parameter         | Description                                                                                                                                                                                |
| ----------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| 送信者電話 ID (必須)     | 送信者電話 ID は、[Facebook Business Manager WhatsAPP settings](https://developers.facebook.com/apps) で確認できます。これは電話番号ではない点に注意してください。                                                              |
| 受信者の電話番号 (必須)     | [E.164 形式](https://en.wikipedia.org/wiki/E.164)の受信者の電話番号です。                                                                                                                                |
| メッセージタイプ (必須)     | ドロップダウンメニューからメッセージタイプを選択します。選択できるタイプは次のとおりです。 <ul> <li>音声</li> <li>連絡先</li> <li>ドキュメント</li> <li>画像</li> <li>インタラクティブ</li> <li>位置情報</li> <li>ステッカー</li> <li>テンプレート</li> <li>テキスト</li> </ul> |
| メッセージオブジェクト (必須)  | メッセージオブジェクトの内容は、選択したメッセージタイプによって異なります。詳細については、[Messages](https://developers.facebook.com/docs/whatsapp/cloud-api/reference/messages) を参照してください。                                            |

<div id="output-object">
  ### 出力オブジェクト
</div>

想定されるすべてのステータスコードは、[Facebook のドキュメント](https://developers.facebook.com/docs/whatsapp/cloud-api/support/error-codes)で確認できます。
