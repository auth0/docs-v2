---
description: Auth0 とサードパーティのアプリケーションおよびサービスとの統合について説明します。
title: Auth0 連携
---

信頼できる Auth0 審査済みの連携機能を活用して、実装時間を短縮できます。Auth0 プラットフォームは高い拡張性を備えており、カスタムコードでアイデンティティフローを調整したり、サードパーティのアプリケーションやツールと連携したりすることで、固有のニーズに対応できます。

[Auth0 Marketplace](https://marketplace.auth0.com/) にアクセスして、当社の製品と連携できるサードパーティのアイデンティティソリューションを見つけて有効化してください。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 Manage Dashboard および Management API を使用してサードパーティの Marketplace 連携をインストールすると、その連携の利用規約およびプライバシーポリシー、ならびに Auth0 Marketplace の[利用規約](https://auth0.com/legal)に同意したものとみなされます。
</Callout>
