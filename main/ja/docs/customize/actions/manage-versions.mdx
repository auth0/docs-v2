Actions のバージョン履歴では、下書き版とデプロイ済み版の Actions を比較し、必要に応じて以前のバージョンに戻すことができます。

<div id="deploy-actions">
  ## Actions をデプロイする
</div>

Action をフローにバインドするには、事前にその Action をデプロイしておく必要があります。

1. [Auth0 Dashboard &gt; Actions &gt; Custom Actions](https://manage.auth0.com/#/actions/library) に移動します。
2. 対象の Action を選択します。
3. **Deploy** を選択します。

選択した Action がデプロイされ、フローで使用できる Action の一覧に追加されます。さらに、今後編集できるように、デプロイ済みの Action の新しいドラフトバージョンが作成されます。シークレットと依存関係はバージョンに紐付いているため、保存済みのシークレットや依存関係は新しいドラフトでも利用できます。

<div id="revert-actions">
  ## Actions を以前のバージョンに戻す
</div>

必要に応じて、Action を最後に正常に動作していたバージョンに戻すことができます。

1. [Auth0 Dashboard &gt; Actions &gt; Custom Actions](https://manage.auth0.com/#/actions/library) に移動します。
2. 対象の Action を選択します。
3. **Version History** を選択します。
4. 戻したいバージョンを選択し、**Revert** をクリックします。

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">
     バージョン履歴では、「Draft」は一度もデプロイされていないバージョンを示します。
   </Callout>

選択したバージョンの新しいコピーが作成され、デプロイされます。さらに、今後編集できるように、デプロイ済みの Action の新しいドラフト バージョンも作成されます。シークレットと依存関係はバージョンごとに関連付けられているため、保存済みのシークレットや依存関係は新しいドラフトでも利用できます。
