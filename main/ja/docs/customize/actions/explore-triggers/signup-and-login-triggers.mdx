サインアップとログインのトリガーの詳細については、以下のセクションを参照してください。

| Trigger                                                                                                          | Description                                              |
| ---------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------- |
| [ログイントリガー](/ja/docs/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger)                     | ログイントリガーは、Auth0テナント内のいずれかのアプリケーションにユーザーがログインしたときに実行されます。 |
| [ユーザー登録前トリガー](/ja/docs/customize/actions/explore-triggers/signup-and-login-triggers/pre-user-registration-trigger)  | ユーザー登録前トリガーは、ユーザーがデータベース接続またはパスワードレス接続に追加される前に実行されます。    |
| [ユーザー登録後トリガー](/ja/docs/customize/actions/explore-triggers/signup-and-login-triggers/post-user-registration-trigger) | ユーザー登録後トリガーは、ユーザーがデータベース接続またはパスワードレス接続に追加された後に実行されます。    |
