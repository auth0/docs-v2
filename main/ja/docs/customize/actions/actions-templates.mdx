Actions テンプレートギャラリーでは、Action を作成するためのさまざまなスターターテンプレートを提供しています。これらのテンプレートには、<Tooltip tip="多要素認証（MFA）: ユーザー名とパスワードに加えて、SMS で送信されるコードなどの要素を使用するユーザー認証プロセス。" cta="用語集を見る" href="/ja/docs/glossary?term=Multi-factor+Authentication">多要素認証</Tooltip>、アクセス制御、プロファイルの拡張など、さまざまなトリガーやユースケースに対応したサンプルコードが含まれています。

テンプレートには、変更してトリガーの一部として活用できるスターターコードが用意されています。ユースケースに合ったテンプレートを選択したら、アプリケーション固有の要件に合わせてコードをカスタマイズできます。

Actions テンプレートの使用方法を段階的に説明したガイドについては、[Write Your First Action](/ja/docs/customize/actions/write-your-first-action) を参照してください。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Actions テンプレートは、次の場所でも利用できます: [https://github.com/auth0/opensource-marketplace](https://github.com/auth0/opensource-marketplace)。
</Callout>

<div id="best-practices">
  ## ベストプラクティス
</div>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Actions テンプレート は、教育目的でのみ提供されています。

  これらのテンプレートは不完全なコードスニペットで構成されているため、いかなる状況でも本番環境で使用しないでください。開発者は、本番環境にデプロイする前に、Actions テンプレート を安全に実装し、完成させる責任があります。
</Callout>

* テンプレートがご利用の環境に合わせて適切にカスタマイズされていることを確認してください。コードは不完全なため、本番環境にインポートする前に、サンプルを環境に合わせて調整し、完成させる必要があります。
* テンプレートは、事前に編集して完成させるまでデプロイしないでください。
* Actions テンプレート には、コメント内に変更または削除してはならないタグが含まれていることがよくあります。
* `npm` パッケージは常に最新バージョンを使用してください。
* Actions テンプレート は、デフォルトで最新の Node.js バージョンを対象としています。テンプレートの作成後、必要に応じて追加の Node.js バージョンをサポートするように変更できます。

ここでも同様に適用されるため、[Actions Coding Guidelines](/ja/docs/customize/actions/action-coding-guidelines) を必ず確認してください。
