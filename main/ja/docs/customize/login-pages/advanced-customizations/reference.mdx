Auth0 では、ACUL で利用できる複数の SDK を提供しています。

<div id="acul-sdks">
  ## ACUL SDK
</div>

* [Auth0 ACUL React SDK](/ja/docs/libraries/acul/react-sdk)
  React アプリケーションへの統合を容易にする React Hooks を提供します。

  * [Auth0 ACUL JS SDK](/ja/docs/libraries/acul/js-sdk)
    TypeScript で記述された中核となる JavaScript ライブラリで、あらゆるフロントエンドフレームワークやプレーンな JavaScript で使用できます。

<div id="acul-sample-applications">
  ## ACUL サンプルアプリケーション
</div>

[ACUL サンプルアプリケーション](https://github.com/auth0-samples/auth0-acul-samples)には、完全なビルドパイプライン、デプロイワークフロー、一般的な Universal Login 画面の実装が含まれています。これらのサンプルアプリケーションは、画面の開発に役立ちます。サンプルは、ACUL React および JS SDK を使用する React 向けに提供されています。

`auth0 acul init` コマンドを使用すると、ACUL React または JS SDK を使用した React のスタータープロジェクトを初期化できます。詳しくは、GitHub の [Auth0 CLI](https://github.com/auth0/auth0-cli) を参照してください。

<div id="acul-resources">
  ## ACUL リソース
</div>

Auth0 では、Universal Login をカスタマイズするための各種リソースを提供しています。

<div id="auth0-management-api">
  ### Auth0 Management API
</div>

Management API を使用すると、個別の Universal Login 画面を更新したり、複数の画面を一括で設定したりできます。Auth0 Management API を使用するには、[アクセストークン](/ja/docs/secure/tokens/access-tokens/management-api-access-tokens)が必要です。

**個別の画面設定**

* [`GET /v2/prompts/{prompt}/screen/{screen}/rendering` エンドポイント](/ja/docs/api/management/v2/prompts/get-rendering)を使用すると、1 つの画面の設定を取得できます。

  * [`PATCH /v2/prompts/{prompt}/screen/{screen}/rendering` エンドポイント](/ja/docs/api/management/v2/prompts/patch-rendering)を使用すると、1 つの画面に対して ACUL を設定できます。

**画面の一括設定**

* [`GET /v2/prompts/rendering` エンドポイント](/ja/docs/api/management/v2/prompts/get-all-rendering)を使用すると、すべての画面の設定を取得できます。

  * [`PATCH /v2/prompts/rendering` エンドポイント](/ja/docs/api/management/v2/prompts/patch-bulk-rendering)を使用すると、画面を一括更新できます。

<div id="auth0-developer-tools">
  ### Auth0 開発者ツール
</div>

Auth0 は、ACUL の設定を含むテナント管理を簡素化するためのツールをいくつか提供しています。

* **Auth0 CLI**<p />
  `acul` コマンドを使用して、プロジェクトのひな形を作成し、ローカルで開発し、画面設定をデプロイできます。詳細については、GitHub の [Auth0 CLI](https://github.com/auth0/auth0-cli) を参照してください。

  * **Auth0 Deploy CLI**<p />
    JSON ファイルで定義された ACUL のレンダリング設定を含め、テナント全体の設定をコードとして管理します。詳細については、GitHub の [Auth0 Deploy CLI](https://github.com/auth0/auth0-deploy-cli) を参照してください。

  * **Auth0 Terraform provider**<p />
    ACUL の画面設定を定義します。詳細については、Terraform の [`promt_screen_renderer resource`](https://registry.terraform.io/providers/auth0/auth0/latest/docs/resources/prompt_screen_renderer) を参照してください。
