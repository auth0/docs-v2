description: SSO Dashboard Extension でアプリケーションを更新し、アプリケーションの SSO ログインを有効にする方法を説明します。
title: SSO Dashboard でアプリケーションを更新する
---

SSO Dashboard Extension を使用すると、複数のエンタープライズアプリケーションにわたるユーザーの <Tooltip tip="シングルサインオン（SSO）: ユーザーが 1 つのアプリケーションにログインすると、他のアプリケーションにも自動的にログインできるようにするサービスです。" cta="用語集を見る" href="/ja/docs/glossary?term=SSO">SSO</Tooltip> ログインを管理できます。

詳細については、[Single Sign-On Dashboard Extension](/ja/docs/customize/extensions/single-sign-on-dashboard-extension) を参照するか、[この拡張機能を GitHub で確認](https://github.com/auth0-extensions/auth0-sso-dashboard-extension)してください。

<div id="prerequisites">
  ## 前提条件
</div>

* [Auth0 で SSO Dashboardアプリケーションを作成する](/ja/docs/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)。
* [SSO Dashboard Extensionをインストールする](/ja/docs/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)。
* [SSO Dashboardにアプリケーションを追加する](/ja/docs/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)。

<div id="update-application">
  ## アプリケーションを更新
</div>

1. [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) に移動し、**Installed Extensions** ビューを選択します。
2. **SSO Dashboard** を選択します。
   管理者の場合は、`https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login` を使用して SSO Dashboard にログインすることもできます。
3. テナント名の下にあるドロップダウンから **Settings** を選択します。

* 有効になっているときにユーザーにアプリケーションを表示するかどうかを指定するには、**Publish** または **Unpublish** を選択します。
* アプリケーションの設定を更新するには、歯車アイコンを選択します。
* アプリケーションを削除するには、**X** を選択します。削除確認のポップアップが表示されます。
