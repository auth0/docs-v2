description: サードパーティサービスで使用できるログエクスポート拡張機能について説明します。
title: ログエクスポート拡張機能
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 では、すべてのログデータをサードパーティサービスに送信する方法として、[ログストリーミングサービス](/ja/docs/customize/log-streams)の利用を推奨しています。
</Callout>

Auth0 のログを外部サービスにエクスポートできます。

* Application Insights
* AWS Cloudwatch
* Azure Blob Storage
* Logentries
* Loggly
* Logstash
* Mixpanel
* Papertrail
* Segment
* Splunk
* Sumo Logic
