<div id="prerequisites">
  ## 前提条件
</div>

有効な SMS 配信および/または電話配信オプションを利用できる [Twilio](https://twilio.com/) アカウントが必要です。

<div id="configure-twilio">
  ## Twilio を設定する
</div>

1. [Twilio](https://www.twilio.com/login) アカウントにログインします。
2. [Twilio Console](https://console.twilio.com/) に移動します。Auth0 Dashboard で Twilio を設定する際に、Twilio SID と Twilio Auth Token が必要です。
3. Auth0 の[Dashboard &gt; ブランディング &gt; 電話番号プロバイダー](https://manage.auth0.com/#/branding/phone/provider) に移動します。**電話番号プロバイダー**として **Twilio** を選択します。
4. **配信方法**を選択します。顧客への配信方法として、**テキスト**、**音声**、または両方を選択できます。
5. **Twilio プロバイダー設定**で、**SMS 送信元**を選択します。

   1. **Use From** を選択した場合は、顧客が電話メッセージを受信する送信元のデフォルトの電話番号を入力します。
   2. [Messaging Service](https://www.twilio.com/docs/messaging/services) を選択した場合は、Twilio Messaging Service SID を入力します。この値は Twilio Console で確認できます。
   3. Twilio Console にある Twilio SID と Twilio Auth Token を入力し、**保存**をクリックします。

<div id="test-phone">
  ## テスト用電話番号
</div>

**Try** ボタンをクリックして、テスト電話メッセージを送信します。外部の電話番号プロバイダーとして Twilio が正常に設定されている場合、Auth0 は選択した配信方法を使用して確認電話メッセージを送信します。数分経ってもメッセージが届かない場合は、失敗が発生していないか [Auth0 logs](https://manage.auth0.com/#/logs) を確認してください。

Twilio の [Messaging Activity](https://console.twilio.com/us1/develop/sms/overview) ページには、ユーザーに送信されたすべての SMS の配信状況に関する情報が表示されます。
