Auth0 ワークフローの一環で送信される各メールには、送信元アドレス、件名、本文などを定義する対応するメールテンプレートがあります。

[外部の電話プロバイダーを設定](../phone-messages/configure-phone-messaging-providers)した後は、電話テンプレートを編集して電話メッセージをカスタマイズできます。

テキストメッセージテンプレートは、HTML の機能を拡張するオープンソースのテンプレート言語 [Liquid](https://liquidjs.com/index.html) をサポートしています。Liquid では、次のことができます。

* `{{ user.name }}` をユーザーの名前に置き換えるように、データに基づいて変数を動的に埋め込む
* `if` / `else` 文のようなロジックを実装する
* 文字列や配列の操作、数値演算などに使用できるフィルターでデータを変換する

[ダッシュボードの電話テンプレートページ](https://manage.auth0.com/#/phone)では、電話テンプレートの有効化、無効化、カスタマイズを行えます。

<div id="available-phone-templates">
  ## 利用可能な電話テンプレート
</div>

以下は、利用可能な電話テンプレートの概要です。

| 電話テンプレート名       | 説明                                          |
| --------------- | ------------------------------------------- |
| ブロックされたアカウント    | 不審なアクティビティにより、アカウントがブロックされたことを顧客に通知します。     |
| パスワードの変更        | パスワード変更用の code を顧客に送信します。                   |
| ワンタイムパスワード - 登録 | ワンタイムパスワード (OTP) の登録に使用する確認 code を顧客に送信します。 |
| ワンタイムパスワード - 検証 | ワンタイムパスワード (OTP) の検証に使用する確認 code を顧客に送信します。 |
