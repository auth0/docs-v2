  電話メッセージングプロバイダーは現在 Early Access 段階です。Auth0 のリリースについて詳しくは、[製品リリースステージ](/ja/docs/troubleshoot/product-lifecycle/product-release-stages)を参照してください。
</Callout>

Auth0 では、SMS や音声通話の管理、監視、トラブルシューティングに役立てるため、独自の電話メッセージングプロバイダーを設定できます。テナントごとに、SMS および音声通話全体で設定できる電話プロバイダーは 1 つだけです。

Auth0 では現在、次の電話メッセージングプロバイダーをサポートしています。

* [Twilio](/ja/docs/customize/phone-messages/configure-phone-messaging-providers/configure-twilio-as-a-phone-messaging-provider)
* [カスタム電話プロバイダー](/ja/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider)

Actions を使用してカスタム電話プロバイダーを設定することもできます。詳しくは、[カスタム電話プロバイダーを設定する](/ja/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider)および [Actions Triggers: custom-phone-provider Event Object](/ja/docs/customize/phone-messages/configure-phone-messaging-providers/configure-a-custom-phone-provider/actions-triggers-custom-phone-provider-event-object) を参照してください。
