<Warning>
  電話テンプレートをカスタマイズするには、[外部の電話プロバイダーを設定](../configure-phone-messaging-providers)する必要があります。
</Warning>

<div id="customize-phone-templates">
  ## 電話テンプレートをカスタマイズする
</div>

電話テンプレートをカスタマイズするには、次の手順に従います。

1. [**Dashboard &gt; Branding &gt; Phone Templates**](https://manage.auth0.com/#/phone) に移動します。

2. **Template** ドロップダウンから、更新する電話テンプレートを選択します。

3. まだ有効になっていない場合は、**Status** スライダーを切り替えてテンプレートを有効にします。

4. **From** の電話番号または **テキストメッセージテンプレート** を更新します。

5. **Save** を選択して変更を保存するか、**Try** を選択して変更内容をテストするか、**Reset** を選択して変更を元に戻します。

<div id="from-phone-number">
  ### 送信元電話番号
</div>

<Warning>
  SMS の送信元として [Twilio Messaging Services](https://www.twilio.com/docs/messaging/services) を使用している場合、カスタムの **From** 電話番号は設定できません。代わりに、Twilio の設定に応じて、Twilio が Messaging Service の送信者プールから番号を選択します。
</Warning>

**From** フィールドでは、テキストメッセージの送信元の電話番号を設定します。既定の **From** 電話番号は、電話プロバイダーの設定時に指定した番号です。

電話テンプレートにカスタムの **From** 電話番号を設定するには、このフィールドに電話番号を入力します。

<div id="text-message-template">
  ### テキストメッセージテンプレート
</div>

**テキストメッセージテンプレート**フィールドでは、テキストメッセージの内容を設定します。各テンプレートにはデフォルトのメッセージ本文が用意されており、これを編集することも、完全に削除して独自の内容を記述することもできます。

このフィールドでは、動的に生成されるコンテンツに Liquid 構文を使用できます。詳しくは、[電話番号テンプレートでサポートされる Liquid 構文](./supported-liquid-syntax)を参照してください。

<div id="test-updated-templates">
  ## 更新したテンプレートをテストする
</div>

電話テンプレートをテストするには、**Try** をクリックします。Auth0 は、テナント名を名前に持つ既定のアプリに電話メッセージを送信します。別のアプリケーション用のテンプレートをテストするには、電話メッセージフローを実行するサンプルユーザーを作成します。

特定のアプリケーションとユーザーについては、Management API の [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) エンドポイントを使用して、メールアドレス確認メールを手動で送信できます。
