<div id="prerequisites">
  ## 前提条件
</div>

* [Mailgun](https://mailgun.com) アカウントが必要です。

* 特定の [Auth0 IP アドレス](/ja/docs/secure/security-guidance/data-security/allowlist) からの接続を受け入れるように、プロバイダーを設定する必要があります。

<div id="configure-mailgun">
  ## Mailgun を設定する
</div>

1. [Mailgun](https://mailgun.com) にログインします。
2. Mailgun の **Sending &gt; Domains** に移動し、カスタムドメインを追加します。Mailgun では、検証済みのドメインからのみメールを送信できます。
3. Mailgun の **Settings &gt; API Keys** に移動します。API キーはアカウントの登録時に作成されているため、**Private API Key** からコピーします。
4. Auth0 の [Dashboard &gt; Branding &gt; メールプロバイダー](https://manage.auth0.com/#/branding/email_provider) に移動します。
5. **Use my own メールプロバイダー** トグルを有効にします。
6. **Mailgun** ロゴをクリックします。
7. **From** メールアドレスを指定し、先ほど Mailgun に追加した **Domain**、先ほどコピーした Mailgun の **API Key** を入力して、**Region** を選択します。
8. **Save** をクリックします。

<div id="send-a-test-email">
  ## テストメールを送信する
</div>

**Send Test Email** ボタンを使用して、テストメールを送信します。設定が正しければ、確認メールが届きます。数分待ってもメールアドレスに届かない場合は、[Auth0ログ](https://manage.auth0.com/#/logs)でエラーコードを確認してください。

設定が完了すると、[Mailgun Dashboard](https://app.mailgun.com/app/dashboard) に、ユーザーに送信されたすべてのメールの配信インサイトが表示されます。
