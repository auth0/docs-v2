Auth0 は、CMS のインストールを Auth0 アカウントと統合するための Content Management System Plugins/Extensions を提供しています。これらのプラグインにより、Enterprise 向けの <Tooltip tip="Single Sign-On (SSO): ユーザーが 1 つのアプリケーションにログインすると、他のアプリケーションにも自動的にログインできるサービスです。" cta="用語集を表示" href="/ja/docs/glossary?term=Single+Sign-on">シングルサインオン</Tooltip> (SSO) 、ソーシャルログイン、ユーザー名/パスワードによるログインを、すべてのインスタンスとプラットフォームで利用できます。

ログイン機能は、新しい Login Widget (Auth0 提供) によって実装されており、次のことが可能になります。

* Enterprise Directory (LDAP、AD、Google Workspace、Office365、および <Tooltip tip="Security Assertion Markup Language (SAML): パスワードなしで 2 者間の認証情報交換を可能にする標準化されたプロトコルです。" cta="用語集を表示" href="/ja/docs/glossary?term=SAML">SAML</Tooltip> プロバイダー) との SSO
* 複数の WordPress アカウント間でユーザー名/パスワードを共有する SSO
* 30 を超える [Social Providers](/ja/docs/authenticate/identity-providers) との SSO
* ユーザー管理ダッシュボード
* オプションの二要素認証
* WordPress と他のアプリケーション間の SSO
* レポートと分析
