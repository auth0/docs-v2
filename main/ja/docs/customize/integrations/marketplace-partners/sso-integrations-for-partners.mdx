<Tooltip tip="シングルサインオン（SSO）: ユーザーが1つのアプリケーションにログインすると、他のアプリケーションにも自動的にログインできるようにするサービス。" cta="用語集を見る" href="/ja/docs/glossary?term=SSO">SSO</Tooltip> (シングルサインオン) 統合では、Auth0 の ID を使用して、<Tooltip tip="シングルサインオン（SSO）: ユーザーが1つのアプリケーションにログインすると、他のアプリケーションにも自動的にログインできるようにするサービス。" cta="用語集を見る" href="/ja/docs/glossary?term=SAML">SAML</Tooltip> 経由で外部サービスにログインします。これにより、社内アプリケーションを持つ顧客は、Auth0 の ID を使って SSO でサードパーティベンダーのサービスにアクセスできます。

[Marketplace で利用可能な SSO 統合](https://marketplace.auth0.com/features/sso-integrations)では、Auth0 を外部ベンダーに接続するために必要な標準の SAML 設定値と、顧客固有の追加値に必要なフィールドの両方を定義します。これらの統合では、Auth0 はサービスプロバイダー (SP) 開始シナリオにおける <Tooltip tip="IDプロバイダー（IdP）: デジタル ID を保存および管理するサービス。" cta="用語集を見る" href="/ja/docs/glossary?term=identity+provider">IDプロバイダー</Tooltip> として機能します。これらの統合向けに Auth0 を設定する方法については、[Auth0 を SAML ID プロバイダーとして設定する](/ja/docs/authenticate/single-sign-on/outbound-single-sign-on/configure-auth0-saml-identity-provider)を参照してください。

パートナーの皆様にご提供いただくのは、この連携を機能させるためのデフォルトの SAML 設定です。これには、[SAML アサーションのカスタマイズ: SAML アサーション属性](/ja/docs/authenticate/protocols/saml/saml-configuration/customize-saml-assertions)に記載されている属性の一部またはすべてについての値、または顧客が要求するフィールドが含まれます。

統合を構築してテストするには、Auth0 で [テナントを作成](https://auth0.com/signup)し、上記のドキュメントに従って、ご利用のサービス向けに動作する SSO 統合を作成してください。

さらに、統合の構築を支援するために、[SSO 統合](https://github.com/auth0-marketplace/TEMPLATE-sso) を活用できます。

<div id="submit-your-sso-integration">
  ## SSO 統合を申請する
</div>

パートナーになる準備ができており、統合を申請する場合は、[partner portal](https://partners.marketplace.auth0.com/) でパートナーアカウントを作成し、次の手順を実行してください。

1. 事業の簡単な説明、法務およびマーケティング用のコンテンツ、会社ロゴ、利用規約、プライバシーポリシー、連絡先情報、サポート用メールアドレスを含むパートナープロフィールを入力します。
2. 統合コード、インストールガイド、名前や説明などの詳細を含む統合を申請します。必要に応じて、統合に 3 つのバリュープロポジションを追加できます。

**以下の設定とドキュメントが必要です。**

* 動作する SAML アドオンアプリケーションの設定に使用する JSON
* 統合を設定する顧客に表示するフィールドの一覧 (**Callback URL**、**<Tooltip tip="対象者: 発行されたトークンの対象者を一意に識別する値です。トークン内では aud という名前で表され、その値には IDトークン の場合はアプリケーション（クライアントID）、アクセストークン の場合は API（API Identifier）の ID が含まれます。" cta="用語集を見る" href="/ja/docs/glossary?term=Audience">対象者</Tooltip>**、**Signing Certificate** など)
* Markdown で記述したインストールガイド。
