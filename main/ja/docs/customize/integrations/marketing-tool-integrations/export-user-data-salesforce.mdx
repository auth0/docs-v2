---

この記事では、Auth0のユーザーデータをCSVファイルにエクスポートし、Data Import Wizardを使用してSalesforceにインポートする方法を説明します。システム要件については、Salesforceの記事 [Data Import Wizard](https://help.salesforce.com/articleView?id=data_import_wizard.htm) を参照してください。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。拡張機能のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Salesforce は CSV 形式のファイルインポートに対応しているため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性について **User Field** と **Column Name** を指定します。例:

| User Field    | Column Name |
| ------------- | ----------- |
| `email`       | メールアドレス     |
| `created_at`  | 作成日時        |
| `given_name`  | 名           |
| `family_name` | 姓           |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

アカウントの種類によっては、インポートを正常に完了するために追加の手順が必要になる場合があります。[Prepare Your Data for Import](https://help.salesforce.com/articleView?id=import_prepare.htm) に関する Salesforce の記事を確認してください。追加の手順については、[Salesforce: Import Data with the Data Import Wizard](https://help.salesforce.com/articleView?id=import_with_data_import_wizard.htm) を参照してください。

開始する前に、Salesforce アカウントにレコードのインポートに必要な[ユーザー権限](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm)があることを確認してください。

CSV ファイルを Salesforce にインポートするには、次の手順に従います。

1. [Salesforce](https://login.salesforce.com/) にログインします。
2. **Setup** に移動し、**Data Import Wizard** を開きます。
3. **Launch Wizard** をクリックします。
4. インポート設定を行い、表示されるフィールドを使用してアップロードする CSV ファイルを選択します。
5. **Next** をクリックします。
6. **Edit Field Mapping** ページで、CSV ファイルのデータフィールドを Salesforce のデータフィールドにマッピングします。続行するには **Next** をクリックします。
7. **Review** ページで情報を確認したら、**Start Import** をクリックします。
8. インポート状況の更新は、**Data Import Wizard** ホームページの **Recent Import Jobs** タブで確認できます。

以上です。Auth0 のユーザーを Salesforce に正常にインポートできました。
