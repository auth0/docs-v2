---

この記事では、Auth0 のユーザーデータを CSV ファイルとしてエクスポートし、Alterian Campaign Manager の Data Import ツールを使って Alterian にインポートする方法を説明します。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。拡張機能ページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Alterian では CSV 形式でファイルをインポートできるため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性について **User Field** と **Column Name** を指定します。例:

| User Field    | Column Name |
| ------------- | ----------- |
| `email`       | メールアドレス     |
| `created_at`  | 作成日時        |
| `given_name`  | 名           |
| `family_name` | 姓           |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

作業を始める前に、詳しくは [Alterian Campaign Manager: Data Import](https://cm.help.alterian.com/CM404/Default.htm#Customer_Analytics/Import_Export/Data_Import.htm) を参照してください。

CSV ファイルを Alterian にインポートするには、次の手順に従います。

1. Alterian Campaign Manager を開き、**Documents** をクリックします。
2. ファイルのインポート先ディレクトリを選択します。
3. **Upload Files** をクリックし、CSV ファイルを選択します。
4. **Upload** をクリックします。
5. アップロードが完了したら、**Data Import** ツールを開きます。
6. **New Imports data** ウィンドウで、表示された各フィールドを使用してインポート設定を構成します。
7. 設定を確認したら、**Run Processes** をクリックします。

以上で完了です。Auth0 のユーザーを Alterian に正常にインポートできました。
