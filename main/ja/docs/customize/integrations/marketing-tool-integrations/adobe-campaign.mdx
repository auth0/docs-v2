title: ユーザーデータを Adobe Campaign にエクスポートする
---

この記事では、Auth0 のユーザーデータを CSV ファイルにエクスポートし、[Adobe Campaign Import Wizard](https://docs.adobe.com/content/help/en/campaign-classic/using/getting-started/importing-and-exporting-data/importing-data.html) を使用して Adobe Campaign にインポートする方法を説明します。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、ダッシュボードの [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。拡張機能のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Adobe Campaign では CSV 形式のファイルをインポートできるため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性に対して **User Field** と **Column Name** を指定します。例:

| User Field    | Column Name |
| ------------- | ----------- |
| `email`       | メールアドレス     |
| `created_at`  | 作成日時        |
| `given_name`  | 名           |
| `family_name` | 姓           |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

開始する前に、詳細については [Adobe Campaign Documentation: Importing Data](https://docs.adobe.com/content/help/en/campaign-classic/using/getting-started/importing-and-exporting-data/importing-data.html) を参照してください。

Adobe Campaign のクライアントダッシュボードにログインし、**Profiles and Targets &gt; Jobs** に移動します。**Create** ボタンをクリックして **New Import** を選択し、新しいインポートジョブを作成します。

新しい **Import Wizard** ウィンドウが開きます。**Template Selection** ステップでは、インポートパラメーターを設定できます。

インポートパラメーターの設定が完了したら、**Next** ボタンをクリックして先に進みます。

**File to Import** ステップで、前のセクションで Auth0 からエクスポートしたユーザーデータの CSV ファイルをアップロードします。**Next** ボタンをクリックして **Field Mapping** に進みます。

次に、エクスポートファイルのスキーマを Adobe Campaign のデータベーススキーマにマッピングします。フィールド名とフィールドタイプが正しいことを確認したら、**Next** ボタンをクリックします。

残りの設定手順では、データの照合モードを定義し、インポートするユーザーのフォルダー、リスト、またはサービスを選択します。

最後に、**Data Import Execution** ウィンドウで **Start** ボタンをクリックしてインポートを開始します。

以上で完了です。Auth0 のユーザーが Adobe Campaign に正常にインポートされました。
