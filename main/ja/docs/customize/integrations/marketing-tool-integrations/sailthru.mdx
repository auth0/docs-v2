---

この記事では、Auth0 のユーザーデータを CSV ファイルにエクスポートし、Sailthru ダッシュボードにインポートする方法を説明します。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、ダッシュボードの [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。拡張機能のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Sailthru は CSV 形式でのファイルインポートに対応しているため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性について **User Field** と **Column Name** を指定します。たとえば、次のように設定します。

| User Field    | Column Name |
| ------------- | ----------- |
| `email`       | メールアドレス     |
| `created_at`  | 作成日時        |
| `given_name`  | 名           |
| `family_name` | 姓           |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ### ユーザーデータファイルをインポートする
</div>

開始前に、詳しくは [Sailthru Documentation: Adding Users to Sailthru and Sailthru Lists](https://getstarted.meetmarigold.com/engagebysailthru/Content/audience/managing-users/add-users-lists-st.html?TocPath=Audience%7CManaging%20Users%7C_____3#ListFileUpload) を参照してください。

CSV ファイルを Sailthru にインポートするには、次の手順に従います。

1. Sailthru にログインし、[Lists](https://my.sailthru.com/lists) に移動します。
2. リストを選択し、**Upload List** をクリックして CSV ファイルを選択します。
3. **Action** で **Add To List** を選択します。
4. **Replace Vars** オプションを設定し、必要に応じて通知の送信先メールアドレスを入力します。
5. 設定を確認し、**Submit** をクリックしてファイルをアップロードします。

これで、Auth0 のユーザーを Sailthru に正常にインポートできました。
