---

この記事では、Auth0のユーザーデータをCSVファイルにエクスポートし、その後Constant Contactのダッシュボードにインポートする方法を説明します。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、Dashboard の [Extensions](https://manage.auth0.com/#/extensions) セクションに移動して、**User Import / Export Extension** を開きます。拡張機能のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Constant Contact では CSV 形式のファイルをインポートできるため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性について **User Field** と **Column Name** を指定します。例:

| User Field    | Column Name |
| ------------- | ----------- |
| `email`       | メールアドレス     |
| `created_at`  | 作成日時        |
| `given_name`  | 名           |
| `family_name` | 姓           |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

CSV ファイルを Constant Contact にインポートするには、次の手順に従ってください。

1. [Constant Contact にログイン](https://login.constantcontact.com)し、**Contacts** に移動します。
2. **Contacts** ページで、**Add Contacts &gt; Upload from file** をクリックします。
3. **Upload from file** ページで CSV ファイルを指定し、**Continue** をクリックします。
4. 列を確認し、Constant Contact のフィールドに対応付けます。完了したら、**Continue** をクリックします。
5. **Select lists** ページで、連絡先を追加するリストを選択します。
6. **Upload** をクリックします。

以上です。Auth0 ユーザーの Constant Contact へのインポートは完了です。
