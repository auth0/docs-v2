title: ユーザーデータを Oracle Eloqua にエクスポートする
---

この記事では、Auth0 のユーザーデータを CSV ファイルにエクスポートし、Contact Upload Wizard を使用して Eloqua にインポートする方法を説明します。

<div id="create-a-user-data-file">
  ## ユーザーデータファイルを作成する
</div>

まず、ダッシュボードの [Extensions](https://manage.auth0.com/#/extensions) セクションに移動し、**User Import / Export Extension** を開きます。拡張機能のページで、メニューから **Export** を選択します。

次に、**Export Format** を必要なファイル形式に設定します。Eloqua は CSV 形式でのファイルのインポートに対応しているため、`Tab Separated Value file (*.csv)` オプションを選択します。

上部の **Fields** セクションで、エクスポートに含める各ユーザー属性に対して **User Field** と **Column Name** を指定します。たとえば、次のようになります。

| User Field    | Column Name   |
| ------------- | ------------- |
| `email`       | Email Address |
| `created_at`  | Created At    |
| `given_name`  | First Name    |
| `family_name` | Last Name     |

ユーザーフィールドを追加したら、**Export Users** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、次のセクションで使用する CSV ファイルをダウンロードします。

<div id="import-a-user-data-file">
  ## ユーザーデータファイルをインポートする
</div>

開始する前に、[Oracle Eloqua Help Center: Uploading Contacts](https://docs.oracle.com/en/cloud/saas/marketing/eloqua-user/Help/Contacts/Tasks/UploadingContacts.htm) にアクセスすると、詳細情報を確認できます。

CSV ファイルを Eloqua にインポートするには、次の手順に従ってください。

1. Eloqua にログインし、**Audience &gt; Contacts &gt; Upload** に移動して **Contact Upload Wizard** を開きます。
2. **Pick Data Source** タブで、表示されるフィールドを使用してインポートを設定します。
3. **Cloud** ボタンをクリックし、アップロードするファイルを選択します。
4. **Review** タブでファイルの内容を確認したら、**Next Step** をクリックします。
5. **Map Fields** タブで、CSV ファイルのデータを連絡先フィールドに対応付けるフィールドマッピング設定を入力します。続行するには、**Next Step** をクリックします。
6. ウィザードの最後の手順を完了し、**Finish** をクリックします。

これで完了です。Auth0 のユーザーは Eloqua に正常にインポートされました。
