  移行を支援するために、[Rules から Actions への移行](/ja/docs/customize/actions/migrate/migrate-from-rules-to-actions) と [Hooks から Actions への移行](/ja/docs/customize/actions/migrate/migrate-from-hooks-to-actions) のガイドを用意しています。さらに、機能比較、[Actions のデモ](https://www.youtube.com/watch?v=UesFSY1klrI)、および移行に役立つその他のリソースを紹介する専用の [Move to Actions](https://auth0.com/extensibility/movetoactions) ページもあります。

  Rules と Hooks の非推奨化の詳細については、ブログ記事 [Preparing for Rules and Hooks End of Life](https://auth0.com/blog/preparing-for-rules-and-hooks-end-of-life/) を参照してください。

  Rules を Actions に移行する方法については、[Migrate from Rules to Actions](/ja/docs/customize/actions/migrate/migrate-from-rules-to-actions) を参照してください。Hooks を Actions に移行する方法については、[Migrate from Hooks to Actions](/ja/docs/customize/actions/migrate/migrate-from-hooks-to-actions) を参照してください。

任意の [Hook](/ja/docs/customize/hooks) に対して、Dashboard または <Tooltip tip="Management API: お客様が管理タスクを実行できるようにする製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使用して複数の Hook Secret を作成できます。

Hook Secrets は、[Deploy Command-Line Interface (CLI) Tool](/ja/docs/deploy-monitor/deploy-cli-tool) を使用してインポートおよびエクスポートすることもできます。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  1 つの Hook につき、最大 20 個のシークレットを作成できます。
</Callout>

<div id="create-hook-secrets-using-the-dashboard">
  ## Dashboard を使用して Hook Secrets を作成する
</div>

1. [Auth0 Dashboard](https://manage.auth0.com/#/) の [Hooks](https://manage.auth0.com/#/hooks) ページに移動し、編集する Hook の横にある鉛筆アイコンをクリックします。
2. Hook エディターでレンチアイコンをクリックし、**Secrets** をクリックします。
3. **Add Secret** をクリックします。
4. シークレットの名前と値を入力し、**Save** をクリックします。

<div id="create-hook-secrets-using-the-management-api">
  ## Management API を使用して Hook Secrets を作成する
</div>

[Add Hook Secrets endpoint](https://auth0.com/docs/api/management/v2/#!/Hooks/post_secrets) に `POST` リクエストを送信します。`MGMT_API_ACCESS_TOKEN`、`HOOK_ID`、`HOOK_SECRET_KEY`、`HOOK_SECRET_VALUE` のプレースホルダー値は、それぞれ Management API の <Tooltip tip="アクセストークン: API にアクセスするために使用される、不透明な文字列または JWT の形式の認可資格情報。" cta="用語集を表示" href="/ja/docs/glossary?term=Access+Token">アクセストークン</Tooltip>、Hook ID、Hook のキーと値のペアに必ず置き換えてください。

| 値                       | 説明                                                                                                                                |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| `MGMT_API_ACCESS_TOKEN` | `create:hooks` [スコープ](/ja/docs/glossary?term=scope) を持つ [Management API のアクセストークン](https://auth0.com/docs/api/management/v2/tokens)。 |
| `HOOK_ID`               | シークレットを追加する対象の Hook の ID。                                                                                                         |
| `HOOK_SECRET_KEY`       | 指定した Hook に追加するシークレットの名前。このエンドポイントはキーと値のペアからなるオブジェクトを受け付けます。                                                                      |
| `HOOK_SECRET_VALUE`     | 指定した Hook に追加するシークレットの値。このエンドポイントはキーと値のペアからなるオブジェクトを受け付けます。                                                                       |
