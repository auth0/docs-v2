  Rules と Hooks の廃止予定の詳細については、ブログ記事 [Preparing for Rules and Hooks End of Life](https://auth0.com/blog/preparing-for-rules-and-hooks-end-of-life/) を参照してください。

Dashboard または <Tooltip tip="Management API: お客様が管理タスクを実行するための製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使用して、特定の Hook に追加された Hook Secrets を更新できます。

Hook Secrets は、Deploy コマンドラインインターフェイス (CLI) ツールを使用してインポートおよびエクスポートすることもできます。詳細については、[Deploy CLI ツール](/ja/docs/deploy-monitor/deploy-cli-tool) を参照してください。

<div id="update-hook-secrets-using-the-dashboard">
  ## Dashboard を使用して Hook Secrets を更新する
</div>

1. [Auth0 Dashboard &gt; Auth Pipeline &gt; Hooks](https://manage.auth0.com/#/) に移動し、編集する Hook の横にある鉛筆アイコンをクリックします。
2. Hook エディターでレンチアイコンをクリックし、**Secrets** をクリックします。
3. 編集する Hook Secret の値の横にある鉛筆と紙のアイコンをクリックします。
4. 選択した Hook Secret の名前や値を変更して、**Save** をクリックします。

<div id="update-hook-secrets-using-the-management-api">
  ## Management API を使用して Hook Secrets を更新する
</div>

[Update Hook Secrets endpoint](https://auth0.com/docs/api/management/v2/#!/Hooks/patch_secrets) に `PATCH` リクエストを送信します。`HOOK_ID` と `MGMT_API_ACCESS_TOKEN` のプレースホルダー値は、それぞれお使いの Hook ID と Management API の <Tooltip tip="アクセストークン: API へのアクセスに使用される認可資格情報で、不透明な文字列または JWT の形式を取ります。" cta="用語集を見る" href="/ja/docs/glossary?term=Access+Token">アクセストークン</Tooltip> に必ず置き換えてください。

| 値                       | 説明                                                                                                                                                                      |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `MGMT_API_ACCESS_TOKEN` | [スコープ](/ja/docs/glossary?term=scope) `update:hooks` を持つ Management API 用のアクセストークン。詳細は、[Management API アクセストークン](https://auth0.com/docs/api/management/v2/tokens)を参照してください。 |
| `HOOK_ID`               | シークレットを更新する対象の Hook の ID。                                                                                                                                               |
| `HOOK_SECRET_KEY`       | 指定した Hook で更新するシークレットの名前。このエンドポイントはキーと値のペアからなるオブジェクトを受け付けます。                                                                                                            |
| `HOOK_SECRET_VALUE`     | 指定した Hook で更新するシークレットの値。このエンドポイントはキーと値のペアからなるオブジェクトを受け付けます。                                                                                                             |

<Warning>
  指定した Hook に設定されている Hook Secrets を取得すると、値にはプレースホルダーテキスト `_VALUE_NOT_SHOWN` が含まれます。更新時にこのプレースホルダーテキストをそのまま送信しないよう注意してください。送信すると、シークレットの実際の値が上書きされます。
</Warning>
