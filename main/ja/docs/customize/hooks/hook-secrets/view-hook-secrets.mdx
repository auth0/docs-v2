Dashboard を使用して Hook に設定されている Hook Secret を確認するか、<Tooltip tip="Management API: 顧客が管理タスクを実行するための製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を使用して Hook Secret の一覧を取得できます。

<div id="view-hook-secrets-using-the-dashboard">
  ## Dashboard を使用して Hook シークレット を表示する
</div>

1. [Auth0 Dashboard &gt; Auth Pipeline &gt; Hooks](https://manage.auth0.com/#/) に移動し、シークレット を表示する Hook の横にある鉛筆アイコンをクリックします。
2. Hook Editor でレンチアイコンをクリックし、**シークレット** をクリックします。

選択した Hook に設定されているすべての シークレット が一覧表示されます。

<div id="get-hook-secrets-using-the-management-api">
  ## Management API を使用して Hook Secrets を取得する
</div>

[Get Hook Secrets エンドポイント](https://auth0.com/docs/api/management/v2/#!/Hooks/get_secrets)に対して `GET` リクエストを実行します。`HOOK_ID` と `MGMT_API_ACCESS_TOKEN` のプレースホルダー値は、それぞれ対象の Hook の ID と Management API の <Tooltip tip="アクセストークン: API へのアクセスに使用される認可資格情報で、不透明な文字列または JWT の形式を取ります。" cta="用語集を表示" href="/ja/docs/glossary?term=Access+Token">アクセストークン</Tooltip> に必ず置き換えてください。

<AuthCodeGroup>
  ```bash cURL
  curl --request GET \
    --url 'https://{yourDomain}/api/v2/hooks/HOOK_ID/secrets' \
    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN'
  ```

  ```csharp C#
  var client = new RestClient("https://{yourDomain}/api/v2/hooks/HOOK_ID/secrets");
  var request = new RestRequest(Method.GET);
  request.AddHeader("authorization", "Bearer MGMT_API_ACCESS_TOKEN");
  IRestResponse response = client.Execute(request);
  ```

  ```go Go
  package main

  import (
  	"fmt"
  	"net/http"
  	"io/ioutil"
  )

  func main() {

  	url := "https://{yourDomain}/api/v2/hooks/HOOK_ID/secrets"

  	req, _ := http.NewRequest("GET", url, nil)

  	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")

  	res, _ := http.DefaultClient.Do(req)

  	defer res.Body.Close()
  	body, _ := ioutil.ReadAll(res.Body)

  	fmt.Println(res)
  	fmt.Println(string(body))

  }
  ```

</AuthCodeGroup>

| 値                       | 説明                                                                                                                                                                                 |
| ----------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `HOOK_ID`               | シークレットを取得する Hook の ID。                                                                                                                                                             |
| `MGMT_API_ACCESS_TOKEN` | [scope](/ja/docs/glossary?term=scope) `update:hooks` を持つ Management API 用のアクセストークン。詳細については、[Management API Access Tokens](https://auth0.com/docs/api/management/v2/tokens) を参照してください。 |

<Warning>
  Hook Secrets は書き込み専用です。指定した Hook に設定されているシークレットを取得する場合、値にはプレースホルダーテキスト `_VALUE_NOT_SHOWN` が含まれます。
</Warning>
