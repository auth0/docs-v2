<div id="post-login-actions-and-social-connections">
  ## Post-login Actions とソーシャル接続
</div>

テナントで post-login Action とソーシャル接続 (Facebook など) を設定している場合、次の問題が発生することがあります。

ユーザーがソーシャル接続でサインアップし、その認証フローで post-login Action が失敗すると、Auth0 は &quot;Failed Signup&quot; (`fs`) イベントを記録します。ただし、Auth0 はこのユーザーのユーザープロファイルを作成します。

その後、ユーザーがそのソーシャル接続でログインすると、Auth0 は &quot;Success Login&quot; (`s`) イベントを記録しますが、&quot;Success Signup&quot; (`ss`) イベントは記録しません。
