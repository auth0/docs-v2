title: 拡張機能のトラブルシューティング
---

拡張機能で問題が発生している場合は、次の2つのいずれかの方法でトラブルシューティングを開始することをお勧めします。

1. アップグレードできる場合は、拡張機能をアップグレードします。
2. 拡張機能を再インストールします。

Auth0 が提供する各種拡張機能の詳細については、[Auth0 Extension](/ja/docs/customize/extensions) を参照してください。

<div id="upgrade-the-extension">
  ## 拡張機能 をアップグレードする
</div>

拡張機能 は常に利用可能な最新バージョンにアップグレードしておくことをお勧めします。

1. [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) に移動します。
2. **Installed Extensions** タブで、対象の 拡張機能 に利用可能なアップグレードがあるかどうかを確認します。
3. **Update to...** オプションをクリックします。

<div id="reinstall-the-extension">
  ## 拡張機能を再インストールする
</div>

拡張機能で問題が発生した場合は、再インストールを試してください。

1. [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) に移動します。
2. **Installed Extensions** タブで、対象の拡張機能を削除します。
3. Auth0 Dashboard からログアウトします。
4. Auth0 Dashboard に再度ログインします。
5. 拡張機能を再インストールし、再設定します。

<div id="contact-support">
  ## サポートへの問い合わせ
</div>

拡張機能で引き続き問題が発生しており、[Auth0 サポート](https://support.auth0.com)に問い合わせる必要がある場合は、**サポートチケットにエラーログや** [**Actions リアルタイムログ**](/ja/docs/customize/actions/actions-real-time-logs) **を必ず含めてください**。エラーやログがあると、Auth0 サポートは問題をより迅速に切り分けて対処できます。
