---

パスワードリセットが正常に完了した後にユーザーがリダイレクトされる URI は、次のすべての条件によって決まります。

1. テナントで Universal Login と Classic Login のどちらを使用しているか
2. Management API を介してパスワード変更チケットを直接生成するか
3. Authentication API、ログインページ、またはその他のソースからパスワード変更メールを送信するか

次のフローチャートは、テナントの設定に応じてリダイレクト URI を設定するためのガイドです。
