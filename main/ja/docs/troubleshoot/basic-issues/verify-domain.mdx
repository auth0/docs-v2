---
description: 問題をトラブルシューティングするためにドメインを確認する方法を説明します
title: ドメインを確認する
---

* ドメインが、認証時に使用したものと同一であることを確認してください。
* Auth0 の [カスタムドメイン](/ja/docs/customize/custom-domains) を使用している場合は、認証を開始する際に、アプリケーションで使用しているものと同じドメインを使用することが重要です。詳細については、[カスタムドメインのトラブルシューティング](/ja/docs/troubleshoot/integration-extensibility-issues/troubleshoot-custom-domains) を参照してください。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  問題のトラブルシューティングに役立つ各ログイベントの詳細については、[Monitoring ページ](/ja/docs/deploy-monitor/monitor) を参照してください。
</Callout>
