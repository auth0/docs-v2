<div id="user-issues">
  ## ユーザーに関する問題
</div>

<div id="if-you-do-not-have-your-mobile-device-or-your-mobile-device-is-turned-off">
  ### モバイル端末が手元にない場合、または電源がオフになっている場合
</div>

端末を紛失した場合は、初回登録時に提供されたリカバリーコードを使用して認証を完了できます。

1. ログインするには、メールアドレスとパスワードを入力し、**リカバリーコードを使用する**リンクをクリックします。
2. リカバリーコードを入力します。

リカバリーコードがない場合は、ログインできません。アカウントへのアクセスについては、システム管理者にお問い合わせください。

<div id="if-you-forget-your-password">
  ### パスワードを忘れた場合
</div>

パスワードを忘れた場合は、メールアドレス欄とパスワード欄の下にある **パスワードを覚えていませんか？** リンクをクリックします。続いて、メールアドレスを入力すると、パスワードのリセットに使用できるリンクを含むメールが送信されます。

<div id="if-your-transaction-expires">
  ### トランザクションの有効期限が切れた場合
</div>

<Tooltip tip="多要素認証 (MFA): SMS で送信される code など、username と password に加えて別の認証要素を使用するユーザー認証プロセス。" cta="用語集を見る" href="/ja/docs/glossary?term=MFA">MFA</Tooltip> を使用してログインする場合、最初の認証要素による認証から 2 つ目の認証要素による認証までに許可される時間は最大 5 分です。最初の認証要素でログインしてからどれだけ時間が経過したかは、表示されるメッセージのタイムスタンプで確認できます。

5 分を超えた場合は、もう一度ログインして、新しい code または通知を取得する必要があります。

SMS メッセージをリクエストしている場合は、レート制限を超えていないことを確認してください。

<div id="if-you-need-to-remove-or-delete-mfa-from-a-user-in-your-tenant">
  ### テナント内のユーザーのMFAを解除または削除する必要がある場合
</div>

ユーザーのMFAを解除、削除、またはリセットする必要がある場合は、[MFAをリセット](/ja/docs/secure/multi-factor-authentication/reset-user-mfa)してください。

<div id="sms-related-issues">
  ## SMS関連の問題
</div>

<div id="if-you-did-not-receive-an-sms-message">
  ### SMS メッセージを受信できない場合
</div>

SMS で 6 桁の code を受信できない場合は、入力した電話番号が正しいことを確認してください。正しい場合は、携帯電話の電波が届いていることも確認してください。

それでもメッセージを受信できない場合は、メッセージがブロックされていないか、通信事業者に確認してください。

<div id="sms-message-rate-limits">
  ### SMSメッセージのレート制限
</div>

1時間以内にお使いのデバイスに10件を超えるSMSメッセージを送信しようとすると、レート制限の超過を示すエラーメッセージが表示されます。

メッセージ送信の上限を超えた場合は、最初のメッセージをリクエストした時点から少なくとも1時間待ってから、次のリクエストを行う必要があります。以後は、さらに1時間経過するごとに1回ずつ追加で試行できます。

<div id="rejected-codes">
  ## 拒否されるcode
</div>

Guardian または Google Authenticator アプリの 6 桁のcodeがサインイン時に拒否される場合 (多くは `Incorrect Code` というメッセージが表示されます) 、まず、認証アプリ内の一覧で正しいアプリケーションを選択していることを確認してください。正しいアプリケーションを選択していることを確認済みであれば、モバイルデバイスの時計設定が正しいことを確認してください。ワンタイムパスワードは協定世界時 (UTC) に基づいて生成されるため、codeが機能するにはデバイスの時刻が正確である必要があります。

時計設定を確認するには、次の手順に従います。

* **Android デバイス** - **Settings** &gt; **Date &amp; Time** に移動します。**Automatic** の横にあるチェックボックスがオンになっていることを確認してください。これをオフにするには、**Settings &gt; Date &amp; Time** に移動します。**Automatic** の横にあるチェックボックスをタップして、チェックを外します。
* **iOS デバイス** - **Settings** &gt; **General** &gt; **Date &amp; Time** に移動します。**Set Automatically** を有効にします。この設定がすでに有効になっている場合は、いったん無効にしてから再度有効にできます。
