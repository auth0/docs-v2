以下では、セルフパスワード変更APIで発生する可能性のある各種エラーの code と、考えられる解決策を示します。

<div id="error-format">
  ## エラー形式
</div>

エラーメッセージは次の標準形式で返されます:

<div id="error-codes">
  ## エラーコード
</div>

<div id="invalid_request">
  ### invalid_request
</div>

このエラーは、無効なパラメーターを指定した場合に発生します。エラーメッセージには、必須パラメーターの不足や形式が無効であることなど、問題の内容が示されます。

<div id="invalid_user_password">
  ### invalid_user_password
</div>

このエラーは、誤った `username`/`email` と `old_password` の組み合わせが送信された場合に発生します。正しいユーザー名と `old_password` でリクエストを再試行してください。

<div id="change_password_error">
  ### change_password_error
</div>

このエラーは、基盤となる<Tooltip tip="IDプロバイダー（IdP）: デジタル ID を保存および管理するサービス。" cta="用語集を表示" href="/ja/docs/glossary?term=identity+provider">IDプロバイダー</Tooltip>側のさまざまな要因によって発生します。通常は、カスタムデータベースを使用していて、パスワード変更スクリプトを実装していない場合に発生します。
