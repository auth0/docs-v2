Auth0 Public Cloud サービスでは、次のエンドポイントが使用されます。

<div id="united-states-region">
  ## 米国リージョン
</div>

* [https://manage.auth0.com](https://manage.auth0.com)
* [https://auth0.com](https://auth0.com)
* [https://auth0.auth0.com](https://auth0.auth0.com)
* [https://login.us.auth0.com](https://login.us.auth0.com)
* [https://cdn.us.auth0.com](https://cdn.us.auth0.com)
* https://{yourAccount}.us.auth0.com
* https://{yourAccount}.guardian.us.auth0.com

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  CDN URLはリージョンによって異なります。2020年6月11日より前に作成されたテナントで、リージョンが米国の場合は `https://cdn.auth0.com` を使用し、ヨーロッパ、オーストラリア、または日本の場合は `eu`、`au`、`jp` を追加してください。テナントが2020年6月11日より後に作成された場合は、リージョンが米国であれば `https://cdn.us.auth0.com` を使用してください。
</Callout>

<div id="australia-canada-europe-japan-and-uk-regions">
  ## オーストラリア、カナダ、ヨーロッパ、日本、英国リージョン
</div>

* [https://manage.auth0.com](https://manage.auth0.com)
* [https://auth0.com](https://auth0.com)
* [https://auth0.auth0.com](https://auth0.auth0.com)
* `https://login.[au|ca|eu|jp|uk].auth0.com`
* `https://cdn.[au|ca|eu|jp|uk].auth0.com`
* `https://{yourAccount}.[au|ca|eu|jp|uk].auth0.com`
* `https://{yourAccount}.guardian.[au|ca|eu|jp|uk].auth0.com`
