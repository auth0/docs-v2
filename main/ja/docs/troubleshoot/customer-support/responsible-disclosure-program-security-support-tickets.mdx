title: 責任ある開示プログラム
description: 無料のサブスクリプションプランをご利用のお客様を含むすべてのお客様が、Auth0 Responsible Disclosure Program を通じてセキュリティ上の懸念を報告できます。
---

無料のサブスクリプションプランをご利用のお客様を含むすべてのお客様が、セキュリティ上の懸念を報告できます。セキュリティ上の懸念を送信する前に、詳細について [Auth0 の脆弱性開示ポリシー](https://www.okta.com/content/dam/okta---digital/en_us/legal/okta-vulnerability-disclosure-policy_v2.0.pdf) をご確認ください。

調査結果を公開したい場合は、[psirt@okta.com](mailto:psirt@okta.com) 宛てに、できるだけ当社の [GPG 暗号化キー](https://www.okta.com/vulnerability-reporting-policy/okta-pgp-key/) を使用して、直接レポートを送信する必要があります。

報奨を希望する場合は、Bugcrowd の [標準開示条件](https://www.bugcrowd.com/resources/essentials/standard-disclosure-terms/) が適用されます。また、報奨を受けるには、[Bugcrowd](https://bugcrowd.com/okta) 経由で送信する必要があります。詳細については、[Auth0 のバグ報奨金プログラム](https://bugcrowd.com/auth0-okta) をご覧ください。

法的情報については、[法的詳細](https://auth0.com/legal) をご確認ください。
