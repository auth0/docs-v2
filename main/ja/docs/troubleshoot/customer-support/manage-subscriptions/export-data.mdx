description: Auth0 からデータをエクスポートする方法について説明します。
title: データのエクスポート
---

Auth0 テナント内のすべてのデータは常にお客様の管理下にあり、<Tooltip tip="Management API: お客様が管理タスクを実行できるようにする製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> を介していつでも利用できます。

<div id="user-passwords">
  ## ユーザーのパスワード
</div>

API から取得できない情報は、セキュリティ上の理由により、Auth0 でホストされるデータベースユーザーのパスワードハッシュと秘密鍵のみです。それでも、この情報は [サポートチケット](https://support.auth0.com) を起票してリクエストできます。詳しくは、[パスワードハッシュと MFA シークレットのエクスポート](/ja/docs/manage-users/user-migration/export-password-hashes-and-mfa-secrets) を参照してください。この操作は Free サブスクリプションティアでは利用できません。また、特定の日時を指定したエクスポートのリクエストはお受けできず、保証もできません。

データベースにユーザーのパスワードを保存している場合は、ユーザーがログインするたびに Auth0 が問い合わせるカスタムデータベース接続を設定してください。この場合、ユーザーを Auth0 に段階的に移行することを選択しない限り、Auth0 がパスワードハッシュを保存することはありません。

<div id="auth0-components">
  ## Auth0 のコンポーネント
</div>

Auth0 は主に <Tooltip tip="OpenID: ログイン情報を収集および保存することなく、アプリケーションがユーザーのアイデンティティを検証できるようにする認証のオープン標準です。" cta="用語集を見る" href="/ja/docs/glossary?term=OpenID">OpenID</Tooltip> Connect (OIDC) を認証プロトコルとして使用しているため、標準ライブラリを使用してアプリケーションに統合できるはずです。[SAML](/ja/docs/authenticate/single-sign-on/outbound-single-sign-on/configure-auth0-saml-identity-provider) を介して Auth0 を統合する場合も同様です。Auth0 のすべての SDK、ライブラリ、サンプルは、[無償ソフトウェアとして GitHub で公開されています](https://github.com/auth0/)。

<div id="social-identity-providers">
  ## ソーシャルIDプロバイダー
</div>

Auth0 を使用しない場合でも、ソーシャル <Tooltip tip="Identity Provider (IdP): デジタルアイデンティティを保存および管理するサービス。" cta="用語集を表示" href="/ja/docs/glossary?term=identity+providers">IDプロバイダー</Tooltip> で同じ <Tooltip tip="OAuth 2.0: 認可プロトコルとワークフローを定義する認可フレームワーク。" cta="用語集を表示" href="/ja/docs/glossary?term=OAuth">OAuth</Tooltip> <Tooltip tip="OAuth 2.0: 認可プロトコルとワークフローを定義する認可フレームワーク。" cta="用語集を表示" href="/ja/docs/glossary?term=client+IDs">クライアントID</Tooltip> とシークレットを引き続き使用したい場合は、新たな同意ダイアログを表示しなくても、ユーザー情報へのアクセスを維持できます。

<div id="custom-code">
  ## カスタムコード
</div>

Auth0 のすべてのカスタムコード機能 (Rules、カスタムデータベーススクリプト、カスタム OAuth 接続など) は、Node.js のサンドボックスサービス上で実行されます。サンドボックスサービスで利用できるライブラリは、いずれも npm でも利用でき、標準的な Node.js コードで使用できます。
