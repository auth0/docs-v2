---
title: サービスレベル契約
description: Enterprise Public Cloud と Enterprise Private Cloud の両デプロイメントにおける、Auth0 のサービスレベル契約（SLA）の主要な要素の概要。
---

Auth0 プラットフォームの特定の中核サービスの月間平均可用性は、各月において少なくとも 99.99% とします。Auth0 がいずれかの暦月においてこの可用性サービスレベルを満たせなかった場合、事象の発生から 5 営業日以内にサービスレベルクレジットを請求できます。

Auth0 は、以下の表に記載のとおりクレジットを付与します。サブスクリプションの有効期限満了時点で未使用のクレジットがある場合、当社に対して負っているその他の料金または費用にそのクレジットを充当します。そのようなその他の料金または費用がない場合は、クレジット額を返金します。

| 可用性レベル          | サービスレベルクレジット        |
| --------------- | ------------------- |
| 98.50% - 99.98% | 月額サブスクリプション料金*の 5%  |
| 97.00% - 98.49% | 月額サブスクリプション料金*の 10% |
| &lt; 97.00%     | 月額サブスクリプション料金*の 20% |

*クレジットは、サービス障害が発生した暦月に適用されます。

上記の可用性レベルを下回る事象が発生したと思われる場合は、Auth0 のテクニカルアカウントマネージャーに連絡してください。Basic または Silver Success Plan のお客様は、Auth0 の営業担当者またはアカウントエグゼクティブに連絡してください。

Auth0 の[プライベートインスタンス](/ja/docs/deploy-monitor/deploy-private-cloud)が必要な場合は、詳細について [Auth0 Sales](https://auth0.com/get-started?place=documentation%20post\&type=link\&text=contact%20auth0%20sales) にお問い合わせください。

[Auth0 Status Page](https://status.auth0.com/) にアクセスして、Public Cloud 環境における現在および過去の稼働状況を確認してください。
