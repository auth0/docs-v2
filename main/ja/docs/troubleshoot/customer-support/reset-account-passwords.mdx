title: アカウントのパスワードをリセットする
---

パスワードを変更する必要がある場合や、Auth0 アカウントのパスワードを忘れた場合は、多くの場合、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 のメイン製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> から新しいパスワードを設定できます。パスワードをリセットすると、Auth0 セッションは失効します。

1. Auth0 アカウントのログイン画面で、アカウントに関連付けられているメールアドレスを入力し、**Continue,** をクリックします。次の画面で **Forgot password?** を選択します。
2. メールアドレスを確認し、**Continue** をクリックして送信します。パスワードのリセット手順が記載されたメールが送信されます。
3. リクエストが正常に受理されると、「パスワードをリセットする手順については、メールアドレス [your email] を確認してください。」と表示されます。

パスワードリセットには、いくつか特別なケースがあります。

* <Tooltip tip="多要素認証 (MFA): SMS 経由の code など、username とパスワードに加えて認証要素を使用するユーザー認証プロセスです。" cta="用語集を見る" href="/ja/docs/glossary?term=MFA">MFA</Tooltip> を有効にしていて、管理者アカウントの MFA リセットが必要な場合は、[Auth0 Support](https://support.auth0.com) にお問い合わせください。
* Auth0 へのログインにソーシャルアカウントまたはエンタープライズアカウントを使用している場合は、該当する <Tooltip tip="IDプロバイダー（IdP）: デジタルアイデンティティを保存および管理するサービスです。" cta="用語集を見る" href="/ja/docs/glossary?term=identity+provider">IDプロバイダー</Tooltip> でパスワードをリセットしてください。
* 管理者としてユーザーのパスワードをリセットする場合は、次の 2 つの方法があります。

  * Universal Login または Authentication API を使用して、インタラクティブなパスワードリセットフローを実行します。この方法では、ユーザーにリンク付きのメールが送信され、そのリンクを開くと Auth0 のパスワードリセットページで新しいパスワードを入力できます。
  * Management API または Dashboard を使用して、新しいパスワードを直接設定します。
