**Auth0.Android** には、ブラウザーを使用した認可フローを処理するための `WebAuthProvider` クラスが用意されています。では、独自実装や `AuthProvider` のネイティブ版を使用したい場合は、どうすればよいでしょうか？

<div id="the-authprovider-class">
  ## AuthProvider クラス
</div>

`AuthProvider` インターフェイスを実装するクラスを作成し、そのメソッドをオーバーライドします。

Auth0 がすでに提供している **Native** 実装を利用することもできます。現在利用可能な[ネイティブ ソーシャル認証](/ja/docs/libraries/lock-android/lock-android-native-social-authentication)プロバイダーは、Google と Facebook です。
