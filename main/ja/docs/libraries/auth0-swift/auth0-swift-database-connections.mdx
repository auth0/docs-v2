Authentication API には、データベースユーザーの認証とサインアップを行うためのメソッドが用意されています。

<div id="signing-up-with-a-database-connection">
  ## データベース接続を使用したサインアップ
</div>

サインアップするには、`createUser` メソッドを呼び出し、ユーザーの `email`、`password`、および `connection` 名を渡してサインアップ プロセスを開始する必要があります。保存する追加のユーザーメタデータを指定することもできます。

```swift lines
Auth0
    .authentication()
    .createUser(
        email: "support@auth0.com",
        password: "secret-password",
        connection: "Username-Password-Authentication",
        userMetadata: ["first_name": "First",
                       "last_name": "Last"])
    .start { result in
        switch result {
        case .success(let user):
            print("User: \(user)")
        case .failure(let error):
            print("Failed with \(error)")
        }
    }
```

<div id="login-with-a-database-connection">
  ## データベース接続でログインする
</div>

データベース接続でログインするには、ユーザーの `username`/`email`、パスワード、および認証に使用する接続の名前 (`Username-Password-Authentication` など) を指定して `login` を呼び出す必要があります。レスポンスとして Credentials オブジェクトが返されます。

```swift lines
Auth0
    .authentication()
    .login(
        usernameOrEmail: "support@auth0.com",
        password: "secret-password",
        realm: "Username-Password-Authentication",
        scope: "openid")
     .start { result in
         switch result {
         case .success(let credentials):
            print("Credentials: \(credentials)")
         case .failure(let error):
            print("Failed with \(error)")
         }
     }
```
