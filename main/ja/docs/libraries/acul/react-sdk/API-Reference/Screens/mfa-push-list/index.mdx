`mfa-push-list` 画面には、登録済みのプッシュ通知デバイスの一覧が表示され、ユーザーは認証に使用するデバイスを 1 つ選択できます。

<div id="import">
  ## インポート
</div>

各画面には、それぞれ専用のフックとメソッドが用意されています。SDK は、各画面で**部分インポート**と**ルートインポート**をサポートしています。

* 部分インポートを使用すると、特定のユースケースに必要なコードだけを取り込めます。
* ルートインポートを使用すると、単一のバンドルからすべての画面を読み込めるため、想定されるすべての画面を 1 つのビルドでまとめて処理したい場合に便利です。

```jsx Import Example
// ルートインポート
import { useMfaPushList } from '@auth0/auth0-acul-react';

// 部分インポート
import {
  useMfaPushList,
  // コンテキストフック
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // 共通フック
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // ユーティリティフック
  useChangeLanguage,
  // メソッド
  goBack,
  selectMfaPushDevice,
} from "@auth0/auth0-acul-react/mfa-push-list";

function MfaPushListScreen() {
  const { selectMfaPushDevice } = useMfaPushList();
  return (
    <button onClick={() => selectMfaPushDevice({ deviceIndex: 0 })}>
      Select Device
    </button>
  );
}
```

<div id="context-hooks">
  ## コンテキストフック
</div>

`mfa-push-list` 画面で Auth0 のコンテキストデータへの読み取り専用アクセスを提供する、画面スコープのフックです。`@auth0/auth0-acul-react/mfa-push-list` からインポートします。

<ParamField body="useBranding" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>
  このフックは、`mfa-push-list` 画面に表示されるロゴ、色、テーマ設定などのブランディング設定を提供します。

  ```jsx Example
  import { useBranding } from '@auth0/auth0-acul-react/mfa-push-list';
  function CustomTheme() {
    const branding = useBranding();
  }
  ```
</ParamField>

<ParamField body="useClient" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>
  このフックは、`mfa-push-list` 画面に関連する `id`、`name`、`logoUrl` などのクライアント設定を提供します。

  ```jsx Example
  import { useClient } from '@auth0/auth0-acul-react/mfa-push-list';
  function AppInfo() {
    const client = useClient();
  }
  ```
</ParamField>

<ParamField body="useOrganization" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>
  このフックは、MFA フローが組織スコープの場合に、ユーザーの組織に関する情報を提供します。組織コンテキストが存在しない場合は `null` を返します。

  ```jsx Example
  import { useOrganization } from '@auth0/auth0-acul-react/mfa-push-list';
  function OrgSelector() {
    const organization = useOrganization();
    if (!organization) {
      return <p>No organization context</p>;
    }
  }
  ```
</ParamField>

<ParamField body="usePrompt" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>
  このフックには、認証フロー内の現在のプロンプトに関するデータが含まれます。

  ```jsx Example
  import { usePrompt } from '@auth0/auth0-acul-react/mfa-push-list';
  function FlowInfo() {
    const prompt = usePrompt();
  }
  ```
</ParamField>

<ParamField body="useScreen" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>
  このフックには、設定やコンテキストを含む `mfa-push-list` 画面固有の詳細が含まれます。

  ```jsx Example
  import { useScreen } from '@auth0/auth0-acul-react/mfa-push-list';
  function ScreenDebug() {
    const screen = useScreen();
  }
  ```
</ParamField>

<ParamField body="useTenant" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>
  このフックには、`id` や関連メタデータなど、テナントに関するデータが含まれます。

  ```jsx Example
  import { useTenant } from '@auth0/auth0-acul-react/mfa-push-list';
  function TenantInfo() {
    const tenant = useTenant();
  }
  ```
</ParamField>

<ParamField body="useTransaction" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>
  このフックは、現在の MFA フローの state など、`mfa-push-list` 画面に関連するトランザクション固有のデータを提供します。

  ```jsx Example
  import { useTransaction } from '@auth0/auth0-acul-react/mfa-push-list';
  function TransactionInfo() {
    const transaction = useTransaction();
  }
  ```
</ParamField>

<ParamField body="useUntrustedData" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>
  このフックは、URL パラメーターから事前入力された値など、画面に渡される信頼されていないデータを扱います。

  ```jsx Example
  import { useUntrustedData } from '@auth0/auth0-acul-react/mfa-push-list';
  function PrefilledForm() {
    const untrustedData = useUntrustedData();
  }
  ```
</ParamField>

<ParamField body="useUser" type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>
  このフックは、`username`、`email`、利用可能な認証方法など、現在のユーザーの詳細を提供します。

  ```jsx Example
  import { useUser } from '@auth0/auth0-acul-react/mfa-push-list';
  function UserProfile() {
    const user = useUser();
  }
  ```
</ParamField>

<ParamField body="useMfaPushList" type={<a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/MfaPushListMembers">MfaPushListMembers</a>}>
  このフックは、`mfa-push-list` 画面で使用できるすべてのメソッドとコンテキストを返します。
</ParamField>

<div id="methods">
  ## メソッド
</div>

<ParamField body="goBack" type="Promise<void>">
  このメソッドは、MFA フローの前の画面に戻ります。

  ```jsx Example
  import { useMfaPushList } from '@auth0/auth0-acul-react/mfa-push-list';

  function GoBackButton() {
    const { goBack } = useMfaPushList();
    return (
      <button onClick={() => goBack()}>
        戻る
      </button>
    );
  }
  ```
</ParamField>

<ParamField body="selectMfaPushDevice" type="Promise<void>">
  このメソッドは、MFA チャレンジで使用するプッシュ通知デバイスを、登録済みデバイスの一覧から選択します。

  ```jsx Example
  import { useMfaPushList } from '@auth0/auth0-acul-react/mfa-push-list';

  function DeviceSelector() {
    const { selectMfaPushDevice } = useMfaPushList();
    return (
      <button onClick={() => selectMfaPushDevice({ deviceIndex: 0 })}>
        このデバイスを使用
      </button>
    );
  }
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [SelectMfaPushDeviceOptions](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/SelectMfaPushDeviceOptions)。
    </ParamField>

    <ParamField body="deviceIndex" type="number" required>
      登録済みデバイスの一覧から選択するデバイスのインデックスです。インデックスは 0 始まりです。
    </ParamField>
  </Expandable>
</ParamField>
