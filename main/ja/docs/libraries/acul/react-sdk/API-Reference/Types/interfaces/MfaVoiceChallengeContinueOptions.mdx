---
title: "MfaVoiceChallengeContinueOptions"
---

音声チャレンジ code を送信するためのオプション。

```ts Example
export interface MfaVoiceChallengeContinueOptions {
  /**
   * The verification code received via voice call.
   */
  code: string;

  /**
   * デバイスを30日間記憶して、以降のMFAチャレンジをスキップするオプションのフラグ。
   */
  rememberDevice?: boolean;

  /**
   * Additional custom options to pass with the request.
   */
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## インデックス指定可能
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

リクエストに追加で渡すカスタムオプション。

<div id="properties">
  ## プロパティ
</div>

<ParamField body="code" type="string">
  音声通話で受信した認証コードです。
</ParamField>

<ParamField body="rememberDevice?" type="boolean">
  デバイスを 30 日間記憶し、以降の MFA チャレンジを省略するための任意のフラグです。
</ParamField>
