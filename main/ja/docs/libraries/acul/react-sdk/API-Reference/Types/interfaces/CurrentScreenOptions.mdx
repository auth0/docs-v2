---
title: "CurrentScreenOptions"
---

Auth0 Universal Login フローにおける現在の画面コンテキストを表すオプションオブジェクトです。ランタイムで利用可能なクライアント、組織、テナント、トランザクション、画面のメタデータにアクセスできます。

```ts Example
export interface CurrentScreenOptions {
  client: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  organization: {
    id: string;
    metadata: Record<string, string> | null;
  } | null;
  prompt: {
    name: string;
  } | null;
  screen: {
    name: string;
  } | null;
  tenant: {
    enabledLocales: string[];
  } | null;
  transaction: {
    errors: TransactionError[] | null;
    state: string;
    locale: string;
  } | null;
  untrustedData: {
    authorizationParams: Record<string, string> | null;
  } | null;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="client" type="{ id: string; metadata: Record<string, string> | null } | null">
  現在の認可リクエストにおけるクライアントの詳細です。クライアント識別子と関連するメタデータが含まれます。
</ParamField>

<ParamField body="organization" type="{ id: string; metadata: Record<string, string> | null } | null">
  現在のリクエストにおける組織コンテキストです。組織識別子とメタデータが含まれます。
</ParamField>

<ParamField body="prompt" type="{ name: string } | null">
  現在のプロンプト設定です。どのプロンプトステップがレンダリングされているかを識別します。
</ParamField>

<ParamField body="screen" type="{ name: string } | null">
  現在の画面設定です。プロンプト内でどの画面がアクティブかを識別します。
</ParamField>

<ParamField body="tenant" type="{ enabledLocales: string[] } | null">
  テナントレベルの設定です。この Universal Login のデプロイで有効になっているロケールの一覧が含まれます。
</ParamField>

<ParamField body="transaction" type="{ errors: TransactionError[] | null; state: string; locale: string } | null">
  現在の認証フローにおけるトランザクションの状態です。検証エラー、state 識別子、アクティブなロケールが含まれます。
</ParamField>

<ParamField body="untrustedData" type="{ authorizationParams: Record<string, string> | null } | null">
  フローに渡される信頼できないデータです。たとえば、セキュリティ上の判断の根拠として依拠すべきではない認可パラメーターなどが含まれます。
</ParamField>
