---
title: "MfaPushPollingControl"
---

MFAプッシュ通知のポーリングセッションを管理するための制御インターフェースです。

このインターフェースは、MFAプッシュチャレンジが
承認されたかどうかを確認する長時間実行されるポーリングループを
開始、停止し、その状態を確認するための命令型メソッドを提供します。

````ts Example
export interface MfaPushPollingControl {
  /**
   * ポーリング処理を即座に停止します。
   *
   * - スケジュール済みのタイマーまたは保留中のリクエストをキャンセルします。
   * - 停止後、`isRunning()` は `false` を返します。
   * - 複数回呼び出しても安全です。2回目以降の呼び出しは効果がありません。
   *
   * @example
   * ```ts
   * const control = mfaPushChallengePush.pollingManager({ intervalMs: 5000 });
   * control.startPolling();
   *
   * // ユーザーがキャンセルした場合:
   * control.stopPolling();
   * ```
   */
  stopPolling: () => void;

  /**
   * ポーリング処理を開始または再開します。
   *
   * - ポーリングがすでにアクティブな場合、再度呼び出しても効果がありません。
   * - 以前に停止していた場合、呼び出すとポーリングループが再起動されます。
   *
   * @example
   * ```ts
   * control.startPolling(); // MFAプッシュチャレンジの確認を開始する
   * ```
   */
  startPolling: () => void;

  /**
   * ポーリング処理が現在実行中かどうかを示します。
   *
   * - ポーリングがアクティブでキャンセルされていない場合、`true` を返します。
   * - ポーリングが停止または完了している場合、`false` を返します。
   *
   * @example
   * ```ts
   * if (control.isRunning()) {
   *   console.log('Polling in progress...');
   * } else {
   *   console.log('Polling is stopped or completed.');
   * }
   * ```
   */
  isRunning: () => boolean;
}
````

<div id="properties">
  ## プロパティ
</div>

<ParamField body="isRunning" type="boolean">
  ポーリング処理が現在実行中かどうかを示します。

  ```ts Example
  if (control.isRunning()) {
    console.log('Polling in progress...');
  } else {
    console.log('Polling is stopped or completed.');
  }
  ```
</ParamField>

<ParamField body="startPolling" type="void">
  ポーリング処理を開始または再開します。

  * ポーリングがすでに実行中の場合、再度呼び出しても効果はありません。
  * 以前に停止していた場合は、これを呼び出すとポーリングループが再開されます。

  ```ts Example
  control.startPolling(); // MFAプッシュチャレンジの確認を開始
  ```
</ParamField>

<ParamField body="stopPolling" type="void">
  ポーリング処理を直ちに停止します。

  * スケジュールされたタイマーまたは保留中のリクエストをキャンセルします。
  * 停止後、`isRunning()` は `false` を返します。
  * 複数回呼び出しても安全です。2 回目以降の呼び出しでは何も起こりません。

  ```ts Example
  const control = mfaPushChallengePush.pollingManager({ intervalMs: 5000 });
  control.startPolling();

  // 後でユーザーがキャンセルした場合:
  control.stopPolling();
  ```
</ParamField>
