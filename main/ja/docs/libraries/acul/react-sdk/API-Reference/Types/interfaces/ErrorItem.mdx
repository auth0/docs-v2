---
title: "ErrorItem"
---

ACUL のエラーストア内の単一のエラー項目を表します。[`useErrors`](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors) から [`UseErrorsResult`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorsResult) オブジェクトの一部として返されます。基本の `Error` インターフェースを拡張したもので、`id`、`type`、および省略可能な `label` を持ちます。

```typescript
interface ErrorItem {
  id: string;
  code: string;
  message: string;
  type?: ErrorType;
  field?: string;
  label?: string;
  rules?: PasswordComplexityRule[];
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="id" type="string" required>
  このエラーインスタンスの一意の識別子です。`dismiss(id)` を使用して特定のエラーを指定するために使われます。
</ParamField>

<ParamField body="code" type="string" required>
  エラーの種類を識別する、機械可読なエラー code です。例: `"invalid_password"`、`"username_exists"`。
</ParamField>

<ParamField body="message" type="string" required>
  エラーの内容を人が理解しやすい形で説明したもので、UI への表示に適しています。
</ParamField>

<ParamField body="type" type={<a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a>}>
  エラーを `"auth0"`、`"validation"`、または `"configuration"` に分類します。`errors.byType()` でのフィルタリングに使用されます。
</ParamField>

<ParamField body="field" type="string">
  該当する場合は、このエラーに関連付けられたフォームフィールドです。例: `"username"`、`"email"`。`errors.byField()` でのフィルタリングに使用されます。
</ParamField>

<ParamField body="label" type="string">
  エラーの表示ラベル (任意) です。エラーメッセージに加えて、人にわかりやすいフィールド名が必要な場合に使用されます。
</ParamField>

<ParamField body="rules" type={<span><a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>}>
  エラーがパスワードの検証に関係する場合に、このエラーに関連付けられたパスワード複雑性ルールです。
</ParamField>
