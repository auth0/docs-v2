[`usePasskeyAutofill`](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasskeyAutofill) の返り値の型です。

```typescript
interface UsePasskeyAutofillResult {
  inputRef: RefObject<HTMLInputElement>;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="inputRef" type="RefObject<HTMLInputElement>" required>
  username 入力要素にアタッチする React ref です。これを関連付けると、SDK は入力の `autocomplete` 属性を自動的に `"username webauthn"` に設定し、パスキーの自動入力用のブラウザーの Conditional UI を有効にします。マークアップですでに `autocomplete="username webauthn"` を宣言している場合は、この ref の関連付けは省略可能です。
</ParamField>
