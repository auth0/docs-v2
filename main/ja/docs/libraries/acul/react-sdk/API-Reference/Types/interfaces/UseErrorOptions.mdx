---
title: "UseErrorOptions"
---

[`useError`](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/useError) フックのオプションオブジェクトです。Auth0 Universal Login 画面内でのエラー表示の挙動を制御します。

```ts Example
export interface UseErrorOptions {
  includeDevErrors?: boolean;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="includeDevErrors" type="boolean">
  `true` の場合、エラー出力に開発時向けの詳細なエラー情報が含まれます。開発中のデバッグには便利ですが、本番環境では無効にしてください。
</ParamField>
