---
title: "PasswordValidationResult"
---

[`usePasswordValidation`](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation) の戻り値の型です。パスワードの全体的な妥当性の結果と、パスワードの複雑さに関する各ルールの評価結果の一覧が含まれます。

```typescript
export interface PasswordValidationResult {
  isValid: boolean;
  results: PasswordComplexityRule[];
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="isValid" type="boolean" required>
  すべてのパスワードポリシーチェックの総合結果です。[`PasswordComplexityRule`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) のすべてのルールで `status` が `"valid"` の場合にのみ `true` になります。
</ParamField>

<ParamField body="results" type={<span><a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>} required>
  評価されたパスワードルールの配列です。各項目は [`PasswordComplexityRule`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) であり、その [`status`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule#status) フィールドに、そのルールの検証結果が反映されます。
</ParamField>
