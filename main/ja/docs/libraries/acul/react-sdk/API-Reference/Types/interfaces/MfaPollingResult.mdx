---
title: "MfaPollingResult"
---

[useMfaPolling](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/useMfaPolling) が返す結果オブジェクトです。

<div id="properties">
  ## プロパティ
</div>

<ParamField body="isRunning" type="boolean">
  MFA プッシュのポーリング プロセスが現在アクティブかどうかを示します。

  * `true` — ポーリングは実行中で、完了を待機しています。
  * `false` — ポーリングは停止しています。停止理由は、完了、
    手動キャンセル、またはコンポーネントのアンマウントです。
</ParamField>

<ParamField body="startPolling()" type="void">
  ポーリング プロセスを開始または再開します。

  * ポーリングがすでにアクティブな場合、この呼び出しによる影響はありません。
  * 以前に停止していた場合、これを呼び出すとポーリング ループを再開します。
</ParamField>

<ParamField body="stopPolling()" type="void">
  ポーリング プロセスをただちに停止します。

  * スケジュール済みのタイマーや進行中のリクエストをすべてキャンセルします。
  * 複数回呼び出しても安全です。以降の呼び出しによる影響はありません。
</ParamField>
