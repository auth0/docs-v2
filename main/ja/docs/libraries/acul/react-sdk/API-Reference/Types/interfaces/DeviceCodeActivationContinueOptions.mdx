---
title: "DeviceCodeActivationContinueOptions"
---

device code のアクティベーションを続行するためのペイロードです。

このインターフェースは `CustomOptions` を拡張しているため、すべてのカスタムオプションプロパティが
オブジェクトのトップレベルにマージされます。これによりネストを避け、このインターフェースを利用する側にとって
よりシンプルな構造になります。

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```

  ## インデックス可能
</div>
