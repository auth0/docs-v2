ResendCodeOptions
description Login Email Verification 画面の `resendCode` メソッドに指定するオプションです。
この操作は、ユーザーがメールアドレスに新しい確認 code を送信するよう要求した場合に使用されます。
たとえば、最初の code が届かなかった場合や、有効期限が切れている場合です。

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
