---
title: "ErrorType"
---

[`useErrors`](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors) で管理されるエラーを分類する文字列ユニオン型です。`errors.byType()` によるエラーの絞り込みに使用され、各 [`ErrorItem`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem) では `kind` プロパティとして表されます。

```typescript
type ErrorType = "auth0" | "validation" | "configuration";
```

<div id="values">
  ## 値
</div>

<ParamField body="&#x22;auth0&#x22;" type="ErrorType">
  認証または認可の処理中に Auth0 または独自のバックエンドから返されるエラー。
</ParamField>

<ParamField body="&#x22;validation&#x22;" type="ErrorType">
  リクエスト送信前の無効な入力など、クライアント側のフォーム検証で発生するエラー。
</ParamField>

<ParamField body="&#x22;configuration&#x22;" type="ErrorType">
  必要なコンテキスト外でフックを呼び出す場合など、SDK の統合不備や設定ミスによって発生するエラー。[`UseErrorOptions`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorOptions) で `includeDevErrors: true` を設定した場合にのみ表示されます。
</ParamField>
