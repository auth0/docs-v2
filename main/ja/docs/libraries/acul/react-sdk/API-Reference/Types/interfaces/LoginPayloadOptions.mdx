ログイン操作のオプション

```ts Example
export interface LoginOptions {
  /** ログインに使用するusername/メールアドレス */
  username: string;
  /** 認証用パスワード */
  password: string;
  /** 必要な場合のオプションのキャプチャ値 */
  captcha?: string;
  /** 追加のカスタムオプション */
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

追加のカスタムオプション
