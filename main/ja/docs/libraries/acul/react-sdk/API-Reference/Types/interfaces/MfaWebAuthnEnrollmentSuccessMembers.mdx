<ParamField body="continue" type="Promise<void>">
  WebAuthn への登録が正常に完了した後、ユーザーが認証フローを続行できるようにします。
  このメソッドは、`action: "default"` を指定して `/u/mfa-webauthn-enrollment-success` エンドポイントに POST リクエストを送信します。

  続行 アクションが正常に送信されると解決される Promise です。
  成功すると、通常はブラウザーがフローの次のステップにリダイレクトされます。

  #### スローされる例外

  フォームの送信に失敗した場合 (例: ネットワークの問題、無効な state) にエラーをスローします。

  ```typescript Example
  // 'sdk' が MfaWebAuthnEnrollmentSuccess のインスタンスであると仮定します
  try {
    await sdk.continue();
    // ユーザーは次のステップにリダイレクトされます。
  } catch (error) {
    console.error("Failed to continue after WebAuthn enrollment:", error);
    // エラーを処理します。ページがエラー付きで再レンダリングされた場合は、sdk.transaction.errors を確認してください。
  }
  ```
