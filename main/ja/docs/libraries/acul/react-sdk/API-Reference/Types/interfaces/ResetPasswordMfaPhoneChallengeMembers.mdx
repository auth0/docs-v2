<ParamField body="continue" type="Promise<void>">
  指定した方法 (SMS または音声通話) で、確認コードをユーザーの電話番号に送信します。
  このアクションは、ユーザーが「SMS でコードを送信」または「音声通話でコードを送信」ボタンをクリックしたときに対応します。
  これにより、Auth0 バックエンドでコード配信プロセスが開始されます。

  アクションがサーバーに正常に送信されると解決される Promise です。

  #### 送出

  送信に失敗した場合 (例: ネットワークの問題、無効な state) 、エラーを送出します。

  ```typescript Example
  import ResetPasswordMfaPhoneChallenge from '@auth0/auth0-acul-js/reset-password-mfa-phone-challenge';

  const screen = new ResetPasswordMfaPhoneChallenge();

  // SMS でコードを送信
  await screen.continue({ type: 'sms' });

  // 音声通話でコードを送信
  await screen.continue({ type: 'voice' });
  ```

  <Expandable title="パラメーター">
    <ParamField body="payload" type={<span><a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordMfaPhoneChallengeContinueOptions">ResetPasswordMfaPhoneChallengeContinueOptions</a></span>}>
      配信方法 (`sms` または `voice`) と任意のカスタム データを指定するオプションです。

      **プロパティ**

      <ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;">
        ユーザーが確認コードの受信に選択した配信方法です。

        * `sms`: テキストメッセージで確認コードを送信します。
        * `voice`: 音声通話で確認コードを送信します。
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="tryAnotherMethod" type="Promise<void>">
  ユーザーが別の MFA 方法を選択できる画面に移動します。
  このアクションは、ユーザーが「別の方法を試す」ボタンをクリックしたときに対応します。

  アクションがサーバーに正常に送信されると解決される Promise です。

  #### 送出

  送信に失敗した場合、エラーを送出します。

  ```typescript Example
  import ResetPasswordMfaPhoneChallenge from '@auth0/auth0-acul-js/reset-password-mfa-phone-challenge';

  const screen = new ResetPasswordMfaPhoneChallenge();

  // ユーザーに最初に SMS オプションが提示されていた場合:
  await screen.tryAnotherMethod({ type: 'sms' });

  // ユーザーに最初に音声通話オプションが提示されていた場合:
  await screen.tryAnotherMethod({ type: 'voice' });
  ```

  <Expandable title="パラメーター">

      現在のチャレンジ タイプ (`sms` または `voice`) と任意のカスタム データを指定するオプションです。`type` フィールドは、ユーザーがどのフローから遷移しようとしているかをバックエンドに知らせるために必要です。

      **プロパティ**

      <ParamField body="type" type="&#x22;sms&#x22; | &#x22;voice&#x22;">

        * `sms`: 現在の画面が SMS チャレンジ用であることを示します。
        * `voice`: 現在の画面が音声チャレンジ用であることを示します。
      </ParamField>
    </ParamField>
  </Expandable>
