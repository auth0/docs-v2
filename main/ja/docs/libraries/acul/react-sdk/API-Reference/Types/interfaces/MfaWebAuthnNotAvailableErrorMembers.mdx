MfaWebAuthnNotAvailableErrorMembers

````ts Example
export interface MfaWebAuthnNotAvailableErrorMembers extends BaseMembers {
  /**
   * WebAuthn が利用できないため、ユーザーが別の MFA 方式を選択できるようにします。
   * このアクションにより、ユーザーは利用可能または登録済みの他の MFA 要素から
   * 選択できる認証器選択画面に遷移します。
   *
   * @param {CustomOptions} [payload] - リクエストとともに送信するオプションのカスタムパラメーター。
   * @returns {Promise<void>} 'pick-authenticator' アクションの送信が成功したときに解決される Promise。
   * @throws {Error} フォームの送信に失敗した場合（例: ネットワークエラー、サーバー側のバリデーションエラー）にエラーをスローします。
   * @example
   * ```typescript
   * // 'sdk' は MfaWebAuthnNotAvailableError のインスタンスであると想定します
   * try {
   *   await sdk.tryAnotherMethod();
   *   // 成功時、Auth0 が認証器選択画面へのリダイレクトを処理します。
   * } catch (error) {
   *   console.error('Failed to switch to another MFA method:', error);
   *   // 必要に応じて、失敗をユーザーに通知するよう UI を更新します。
   * }
   * ```
   */
  tryAnotherMethod(payload?: CustomOptions): Promise<void>;
}
````

  ## メソッド

<ParamField body="tryAnotherMethod" type="Promise<void>">
  WebAuthn を利用できない場合に、ユーザーが別の MFA 方式を選択できるようにします。
  この操作により、ユーザーは認証器選択画面に移動し、
  利用可能または登録済みの他の MFA 要素を選択できます。

  &#39;pick-authenticator&#39; アクションの送信が成功すると解決される Promise です。

  #### 例外

  フォームの送信に失敗した場合 (例: ネットワークの問題、サーバー側の検証エラー) にエラーをスローします。

  ```typescript Example
  // 'sdk' は MfaWebAuthnNotAvailableError のインスタンスであるとします
  try {
    await sdk.tryAnotherMethod();
    // 成功すると、Auth0 が認証器選択画面へのリダイレクトを処理します。
  } catch (error) {
    console.error('Failed to switch to another MFA method:', error);
    // 必要に応じて、失敗をユーザーに知らせるために UI を更新します。
  }
  ```
