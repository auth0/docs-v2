---
title: "PasswordComplexityRule"
---

評価済みの単一のパスワード複雑性ルールを表します。[`usePasswordValidation`](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation) により、[`PasswordValidationResult`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordValidationResult) の一部として返されます。オプションの `items` 配列による階層的なルール構成をサポートします。

```typescript
export interface PasswordComplexityRule {
  code: string;
  label: string;
  status: "valid" | "error";
  isValid: boolean;
  args?: {
    count: number;
    total?: number;
    example?: string;
  };
  items?: PasswordComplexityRule[];
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="code" type="string" required>
  この複雑性ルールの機械可読な識別子です。例: `"lengthAtLeast"`、`"containsAtLeast"`。
</ParamField>

<ParamField body="label" type="string" required>
  ルールの人間が読める表示名です。パスワード強度インジケーターでの表示に適しています。
</ParamField>

<ParamField body="status" type="&#x22;valid&#x22; | &#x22;error&#x22;" required>
  このルールの検証結果です。ルールを満たした場合は `"valid"`、満たさない場合は `"error"` です。
</ParamField>

<ParamField body="isValid" type="boolean" required>
  ルールの検証結果を表すブール値です。`status` が `"valid"` の場合は `true` です。
</ParamField>

<ParamField body="args" type="object">
  ルールのしきい値を示したり、エラーメッセージに文脈を補足したりするための追加パラメーターです。

  <Expandable title="プロパティ">
    <ParamField body="count" type="number" required>
      このルールに関連付けられた数値しきい値または件数です。例: 最小文字数。
    </ParamField>

    <ParamField body="total" type="number">
      該当する場合に、このルールに関連する文字数または項目数の合計です。
    </ParamField>

    <ParamField body="example" type="string">
      該当する場合に、このルールを満たす値の例です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="items" type={<span><a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>}>
  複合ルールグループ用のネストされた複雑性ルールです。親ルールが複数の子ルールを集約する階層的なルール構成を可能にします。
</ParamField>
