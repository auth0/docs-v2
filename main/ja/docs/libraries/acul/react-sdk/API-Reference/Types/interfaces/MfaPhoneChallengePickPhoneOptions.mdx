PickPhoneOptions
&#39;pick-phone&#39; Action のオプションを定義します。
このインターフェイスには 1 つの電話番号しか表示されませんが、サーバーで電話番号選択画面へのリダイレクトが必要なシナリオでは、この Action が使用される場合があります。

```ts Example
export interface PickPhoneOptions extends CustomOptions {}
```

  ## インデックス可能
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`
