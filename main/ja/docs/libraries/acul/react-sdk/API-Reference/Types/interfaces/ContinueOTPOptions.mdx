---
title: "ContinueOTPOptions"
---

OTP チャレンジを続行するためのオプション

```ts Example
export interface ContinueOptions {
  /** ユーザーが入力したcode */
  code: string;
  /** ブラウザを記憶するかどうかを示す */
  rememberDevice?: boolean;
  /** 追加のカスタムオプション */
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

任意の追加のカスタムオプション

  ## プロパティ
</div>
