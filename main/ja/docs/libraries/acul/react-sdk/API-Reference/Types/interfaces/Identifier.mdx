---
title: "識別子"
---

サインアップフォームや認証フローで使用する、設定済みの識別子フィールド (例：メールアドレス、電話番号、ユーザー名) について説明します。

```ts Example
export interface Identifier {
  type: IdentifierType;
  required: boolean;
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="type" type={<span><a href="/docs/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ScreenIdentifierType">ScreenIdentifierType</a></span>} required>
  `'email'`、`'phone'`、`'username'` などの識別子のタイプです。
</ParamField>

<ParamField body="required" type="boolean" required>
  この識別子がサインアップ時に必須かどうかを示します。
</ParamField>
