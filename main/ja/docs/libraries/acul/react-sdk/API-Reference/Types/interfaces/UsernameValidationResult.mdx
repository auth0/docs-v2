---
title: "UsernameValidationResult"
---

1 つ以上のルールに照らしてusernameを検証した結果。

```ts Example
export interface UsernameValidationResult {
  isValid: boolean;
  errors: UsernameValidationError[];
}
```

<div id="properties">
  ## プロパティ
</div>

<ParamField body="isValid" type="boolean" required>
  username がすべての検証ルールを満たしているかどうかを示します。
</ParamField>

<ParamField body="errors" type={<span><a href="/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsernameValidationError">UsernameValidationError[]</a></span>} required>
  失敗した各ルールを表す [UsernameValidationError](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsernameValidationError) オブジェクトの配列です (該当する場合) 。

  [UsernameValidationResult.isValid](#isvalid) が `true` の場合、この配列は空になります。
</ParamField>
