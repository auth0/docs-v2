---
title: "ErrorsResult"
---

[`useErrors`](/ja/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors) フックの [`UseErrorsResult`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorsResult) の一部として返される、配列のようなオブジェクトです。`ReadonlyArray<ErrorItem>` を拡張しているため、直接イテレートしたり、スプレッド構文を使用したり、インデックスでアクセスしたりできます。型またはフィールドでエラーリストを照会するためのメソッドが 2 つ用意されています。

```typescript
interface ErrorsResult extends ReadonlyArray<ErrorItem> {
  byType(type: ErrorType, opts?: { field?: string }): readonly ErrorItem[];
  byField(field: string, opts?: { type?: ErrorType }): readonly ErrorItem[];
}
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="byType" type={<span>readonly <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem">ErrorItem</a>[]</span>}>
  指定した [`ErrorType`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType) に一致するすべてのエラーを返します。必要に応じて、特定のフィールドで絞り込むこともできます。

  <Expandable title="パラメーター">
    <ParamField body="type" type={<a href="/docs/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a>} required>
      絞り込みに使用するエラーの種類: `"auth0"`、`"validation"`、または `"configuration"`。
    </ParamField>

    <ParamField body="opts" type="{ field?: string }">
      オプションの絞り込み設定です。

      <Expandable title="プロパティ">
        <ParamField body="field" type="string">
          指定した場合、このフィールド名に関連するエラーだけに結果をさらに絞り込みます。
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="byField" type={<span>readonly <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem">ErrorItem</a>[]</span>}>
  指定したフィールド名に関連するすべてのエラーを返します。必要に応じて、特定の [`ErrorType`](/ja/docs/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType) で絞り込むこともできます。

  <Expandable title="パラメーター">
    <ParamField body="field" type="string" required>
      絞り込みに使用するフィールド名です。例: `"username"`、`"email"`。
    </ParamField>

    <ParamField body="opts" type={<span>&#123; type?: <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a> &#125;</span>}>
      オプションの絞り込み設定です。

      <Expandable title="プロパティ">
        <ParamField body="type" type={<a href="/docs/libraries/acul/react-sdk/API-Reference/Types/type-aliases/ErrorType">ErrorType</a>}>
          指定した場合、この種類のエラーだけに結果をさらに絞り込みます。
        </ParamField>
      </Expandable>
    </ParamField>
  </Expandable>
</ParamField>
