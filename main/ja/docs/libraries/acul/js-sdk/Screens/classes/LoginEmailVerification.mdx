LoginEmailVerification クラスは、`login-email-verification` 画面の機能を実装します。この画面では、ログイン時の本人確認のために、メールアドレスに送信されたワンタイムコードを入力するようユーザーに求めます。

import LoginEmailVerification from '@auth0/auth0-acul-js/login-email-verification';

<ParamField body="resendManager" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  このメソッドは、タイムアウトを管理する再送機能を提供します。`startResend()` メソッドを持つ [`ResendControl`](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/ResendControl) オブジェクトを返します。このメソッドは再送を開始し、ユーザーが再度再送できるようになるまでクールダウン期間を適用します。

  ```typescript Example
  import LoginEmailVerification from '@auth0/auth0-acul-js/login-email-verification';

  const loginEmailVerificationManager = new LoginEmailVerification();
  const { startResend } = loginEmailVerificationManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    }
  });

  // ユーザーが再送ボタンをクリックしたときに startResend を呼び出します
  startResend();
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [StartResendOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions)。
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      ユーザーが code を再送できるようになるまでのクールダウン時間 (秒) です。
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      再送ステータスが変化するたびに呼び出されるコールバックです。残り秒数と、現在再送機能が無効かどうかを返します。
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      クールダウンが終了し、再送が再び可能になったときに呼び出されるコールバックです。
    </ParamField>
  </Expandable>
</ParamField>
