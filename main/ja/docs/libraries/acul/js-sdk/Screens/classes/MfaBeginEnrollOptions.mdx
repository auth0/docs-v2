MfaBeginEnrollOptions クラスは、`mfa-begin-enroll-options` 画面の機能を実装します。この画面では、MFA 認証要素の選択と登録を行います。

</Frame>

<div id="constructors">
  ## コンストラクター
</div>

MFA Begin Enroll Options 画面マネージャーのインスタンスを作成します:

```typescript Example
import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';
const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
await mfaBeginEnrollOptions.enroll({
  action: 'push-notification'
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';
  const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
  mfaBeginEnrollOptions.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      変更先の言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中維持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="enroll" type="Promise<void>">
  このメソッドは、選択した認証要素で登録プロセスを続行します。

  ```typescript Example
  import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';

  const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
  await mfaBeginEnrollOptions.enroll({
    action: 'push-notification'
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [MfaEnrollOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/MfaEnrollOptions)。
    </ParamField>

    <ParamField body="action" type="&#x22;push-notification&#x22; | &#x22;otp&#x22; | &#x22;sms&#x22; | &#x22;phone&#x22; | &#x22;voice&#x22; | &#x22;webauthn-roaming&#x22;" required>
      登録する認証要素を示すアクションです。指定できる値は &quot;push-notification&quot;、&quot;otp&quot;、&quot;sms&quot;、&quot;phone&quot;、&quot;voice&quot;、&quot;webauthn-roaming&quot; のいずれかです。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーがない場合は空の配列を返します。
</ParamField>
