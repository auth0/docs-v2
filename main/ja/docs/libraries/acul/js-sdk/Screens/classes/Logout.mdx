Logout クラスは、`logout` 画面の機能を実装します。この画面では、ユーザーはログアウトするかどうかを確認またはキャンセルできます。

<div id="constructors">
  ## コンストラクター
</div>

Logout画面マネージャーのインスタンスを作成します。

```typescript Example
import Logout from '@auth0/auth0-acul-js/logout';
const logoutManager = new Logout();
logoutManager.confirmLogout({
  action: 'accept',
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import Logout from '@auth0/auth0-acul-js/logout';
  const logoutManager = new Logout();
  logoutManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      変更先の言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語がセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="confirmLogout" type="Promise<void>">
  このメソッドは、ログアウト操作を確定します (承認または拒否) 。

  ```typescript Example
  import Logout from '@auth0/auth0-acul-js/logout';
  const logoutManager = new Logout();
  logoutManager.confirmLogout({
    action: 'accept',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [ConfirmLogoutOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/ConfirmLogoutOptions)。
    </ParamField>

    <ParamField body="action" type="&#x22;accept&#x22; | &#x22;deny&#x22;" required>
      認証トランザクションの state です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーが存在しない場合は、空の配列を返します。
</ParamField>
