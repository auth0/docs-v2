PasskeyEnrollmentLocal クラスは、`passkey-enrollment-local` 画面の機能を実装します。この画面では、ユーザーはデバイスにローカルに保存されたパスキーを認証方法として登録できます。

</Frame>

<div id="constructors">
  ## コンストラクター
</div>

PasskeyEnrollmentLocal 画面マネージャーのインスタンスを生成します。

```typescript Example
import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
await passkeyEnrollmentLocalManager.continuePasskeyEnrollment();
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="abortPasskeyEnrollment" type="Promise<void>">
  このメソッドは、パスキー登録プロセスを中止します。

  ```typescript Example
  import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
  const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
  await passkeyEnrollmentLocalManager.abortPasskeyEnrollment({ doNotShowAgain: true });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [AbortEnrollmentOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/AbortEnrollmentOptions)。
    </ParamField>

    <ParamField body="doNotShowAgain?" type="boolean">
      `true` に設定すると、以降のセッションでパスキー登録のプロンプトを表示しません。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
  const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
  passkeyEnrollmentLocalManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      変更先の言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continuePasskeyEnrollment" type="Promise<void>">
  このメソッドは、パスキー登録プロセスを続行します。

  ```typescript Example
  import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
  const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
  await passkeyEnrollmentLocalManager.continuePasskeyEnrollment();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーがない場合は空の配列を返します。
</ParamField>
