LoginPasswordlessEmailCode クラスは、`login-passwordless-email-code` 画面の機能を実装します。この画面では、ユーザーのメールアドレスと code を収集します。

<div id="constructors">
  ## コンストラクター
</div>

LoginPasswordlessEmailCode 画面マネージャーのインスタンスを作成:

```typescript Example
import LoginPasswordlessEmailCode from '@auth0/auth0-acul-js/login-passwordless-email-code';
const loginPasswordlessEmailCodeManager = new LoginPasswordlessEmailCode();
loginPasswordlessEmailCodeManager.submitCode({
  code: '123456',
});
```

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import LoginPasswordlessEmailCode from '@auth0/auth0-acul-js/login-passwordless-email-code';
  const loginPasswordlessEmailCodeManager = new LoginPasswordlessEmailCode();
  loginPasswordlessEmailCodeManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">

    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">

  </Expandable>

<ParamField body="resendCode" type="Promise<void>">

  ```typescript Example
  import LoginPasswordlessEmailCode from '@auth0/auth0-acul-js/login-passwordless-email-code';
  const loginPasswordlessEmailCodeManager = new LoginPasswordlessEmailCode();

  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [CustomOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/CustomOptions)。

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` を接頭辞として持つ追加のカスタムフィールド (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="resendManager" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  このメソッドは、タイムアウト管理付きの再送機能を提供します。再送を実行する `startResend()` メソッドを持つ [`ResendControl`](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/ResendControl) オブジェクトを返し、ユーザーが再度再送できるようになるまでクールダウン期間を適用します。

  ```typescript Example
  import LoginPasswordlessEmailCode from '@auth0/auth0-acul-js/login-passwordless-email-code';

  const loginPasswordlessEmailCodeManager = new LoginPasswordlessEmailCode();
  const { startResend } = loginPasswordlessEmailCodeManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    }
  });

  // ユーザーが再送ボタンをクリックしたときに startResend を呼び出します
  startResend();
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [StartResendOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions)。
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      ユーザーがコードを再送できるようになるまでのクールダウン時間 (秒) 。
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      再送ステータスが変化するたびに呼び出されるコールバックです。残り秒数と、再送機能が現在無効かどうかを返します。
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      クールダウンが終了し、再送できるようになったときに呼び出されるコールバックです。
    </ParamField>
  </Expandable>
</ParamField>

  ```typescript Example
  import LoginPasswordlessEmailCode from '@auth0/auth0-acul-js/login-passwordless-email-code';
  const loginPasswordlessEmailCodeManager = new LoginPasswordlessEmailCode();
  loginPasswordlessEmailCodeManager.submitCode({
    code: '123456',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [SubmitCodeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/SubmitCodeOptions)。
    </ParamField>

    <ParamField body="code" type="string | number" required>
      ユーザーが入力した code です。
    </ParamField>

    <ParamField body="captcha?" type="string">
      キャプチャの code、またはキャプチャプロバイダーから返されたレスポンスです。Auth0 テナントで [Bot Detection](/ja/docs/secure/attack-protection/bot-detection) が有効になっている場合、このプロパティは必須です。

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">

    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchConnection" type="Promise<void>">
  このメソッドは、データベース接続 (パスワード) とパスワードレス接続 (メールアドレス/SMS OTP) を切り替えます。

  ```typescript Example
  import LoginPasswordlessEmailCode from '@auth0/auth0-acul-js/login-passwordless-email-code';
  const loginPasswordlessEmailCodeManager = new LoginPasswordlessEmailCode();
  loginPasswordlessEmailCodeManager.switchConnection({
    connection: 'email',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [SwitchConnectionOptionsEmailCode](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/SwitchConnectionOptionsEmailCode)。
    </ParamField>

    <ParamField body="connection" type="string" required>
      切り替え先の接続名です。パスワードレスには `'email'` または `'sms'` を使用し、パスワードベース認証には DB 接続名を使用します。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22;">
      `ulp-` で始まる追加のカスタムフィールド (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>
