---
title: "ログイン"
description: "Universal Login の `login` 画面をカスタマイズするために使用できるすべてのプロパティとメソッドについて説明します。"
---

Login クラスは、`login` 画面の機能を実装します。この画面では、ユーザーの識別子とパスワードを入力します。テナントによっては、この識別子はメールアドレス、電話番号、またはユーザー名になります。

<div id="constructors">
  ## コンストラクター
</div>

Login画面マネージャーのインスタンスを作成します。

```typescript Example
import Login from "@auth0/auth0-acul-js/login";
const loginManager = new Login();
loginManager.login({
  username: "testUser",
  password: "testPassword"
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` プレフィックスを持つ追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="federatedLogin" type="Promise<void>">
  このメソッドは、認証のためにユーザーをソーシャルまたはエンタープライズのIDプロバイダー (IdP) にリダイレクトします。

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.federatedLogin({
    connection: "google-oauth2"
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [FederatedLoginPayloadOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/FederatedLoginPayloadOptions).
    </ParamField>

    <ParamField body="connection" type="string" required>
      使用するソーシャル接続の名前。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションのエラー配列を取得します。エラーが存在しない場合は、空の配列を返します。トランザクションコンテキスト内のエラーオブジェクトの配列です。
</ParamField>

<ParamField body="getLoginIdentifiers" type={<span><a href="/docs/libraries/acul/js-sdk/Screens/type-aliases/ScreenIdentifierType">ScreenIdentifierType</a>[] | null</span>}>
  このメソッドは、`login` 画面で有効な識別子の種類を返します。

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.getLoginIdentifiers();
  ```
</ParamField>

<ParamField body="login" type="Promise<void>">
  このメソッドは、識別子と指定されたパスワードを使用してユーザーを認証します。

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.login({
    username: "testUser",
    password: "testPassword"
  });
  ```

  **メソッド パラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LoginPayloadOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LoginPayloadOptions)。
    </ParamField>

    <ParamField body="username" type="string" required>
      ユーザーの識別子です。
    </ParamField>

    <ParamField body="password" type="string" required>
      ユーザーのパスワードです。
    </ParamField>

    <ParamField body="captcha?" type="string">
      キャプチャの code、またはキャプチャ プロバイダーからのレスポンスです。Auth0 テナントで [Bot Detection](/ja/docs/secure/attack-protection/bot-detection) が有効になっている場合、このプロパティは必須です。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` を接頭辞に持つ追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="pickCountryCode" type="Promise<void>">
  このメソッドはユーザーを国コード選択リストにリダイレクトし、そこで電話番号の国番号プレフィックスを更新できます。

  ```typescript Example
  import Login from "@auth0/auth0-acul-js/login";
  const loginManager = new Login();
  loginManager.pickCountryCode();
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [CustomOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/CustomOptions)。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` 接頭辞が付いた追加のカスタムフィールド (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>
