MfaWebAuthnChangeKeyNickname クラスは、`mfa-webauthn-change-key-nickname` 画面の機能を実装します。この画面は、ユーザーが WebAuthn セキュリティキーのニックネームを変更する必要がある場合に表示されます。

<div id="constructors">
  ## コンストラクター
</div>

MFA WebAuthn Change Key Nickname 画面マネージャーのインスタンスを作成します。

```typescript Example
import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
await mfaWebAuthnChangeKeyNickname.continueWithNewNickname({ nickname: 'My YubiKey' });
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

  const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
  mfaWebAuthnChangeKeyNickname.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      使用する言語のロケールコード (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッションの間保持されます。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="continueWithNewNickname" type="Promise<void>">
  このメソッドは、ユーザーの WebAuthn セキュリティキーに新しいニックネームを送信します。

  ```typescript Example
  import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

  const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
  await mfaWebAuthnChangeKeyNickname.continueWithNewNickname({ nickname: 'My YubiKey' });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [MfaWebAuthnChangeKeyNicknameContinueOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/MfaWebAuthnChangeKeyNicknameContinueOptions)。
    </ParamField>

    <ParamField body="nickname" type="string" required>
      WebAuthn セキュリティキーの新しいニックネーム。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーがない場合は空の配列を返します。
