LoginPasswordlessSmsOtp クラスは、`login-passwordless-sms-otp` 画面の機能を実装します。この画面では、ユーザーの電話番号と code を入力します。

<div id="constructors">
  ## コンストラクター
</div>

LoginPasswordlessSmsOtp画面マネージャーのインスタンスを作成します:

```typescript Example

const loginPasswordlessSmsOtpManager = new LoginPasswordlessSmsOtp();

  code: '123456',
});
```

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import LoginPasswordlessSmsOtp from '@auth0/auth0-acul-js/login-passwordless-sms-otp';
  const loginPasswordlessSmsOtpManager = new LoginPasswordlessSmsOtp();
  loginPasswordlessSmsOtpManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      変更先の言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` プレフィックスが付いた追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーが存在しない場合は、空の配列を返します。
</ParamField>

<ParamField body="resendManager" type={<span><a href="/docs/libraries/acul/js-sdk/Screens/interfaces/ResendControl">ResendControl</a></span>}>
  このメソッドは、タイムアウト管理付きの再送機能を提供します。再送を開始し、ユーザーが再度再送できるようになるまでクールダウン期間を適用する `startResend()` メソッドを持つ [`ResendControl`](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/ResendControl) オブジェクトを返します。

  ```typescript Example
  import LoginPasswordlessSmsOtp from '@auth0/auth0-acul-js/login-passwordless-sms-otp';

  const loginPasswordlessSmsOtpManager = new LoginPasswordlessSmsOtp();
  const { startResend } = loginPasswordlessSmsOtpManager.resendManager({
    timeoutSeconds: 15,
    onStatusChange: (remainingSeconds, isDisabled) => {
      console.log(`Resend available in ${remainingSeconds}s, disabled: ${isDisabled}`);
    },
    onTimeout: () => {
      console.log('Resend is now available');
    }
  });

  // ユーザーが再送ボタンをクリックしたときに startResend を呼び出します
  startResend();
  ```

  **メソッドのパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [StartResendOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/StartResendOptions)。
    </ParamField>

    <ParamField body="timeoutSeconds?" type="number">
      ユーザーが code を再送できるようになるまでのクールダウン時間 (秒) です。
    </ParamField>

    <ParamField body="onStatusChange?" type={<span><a href="/docs/libraries/acul/js-sdk/Screens/type-aliases/OnStatusChangeCallback">OnStatusChangeCallback</a></span>}>
      再送ステータスが変化するたびに呼び出されるコールバックです。残り秒数と、現在再送機能が無効かどうかを返します。
    </ParamField>

    <ParamField body="onTimeout?" type="() => void">
      クールダウンが終了し、再送が再び可能になったときに呼び出されるコールバックです。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchConnection" type="Promise<void>">
  このメソッドは、データベース接続 (パスワード) とパスワードレス接続 (メール/SMS OTP) を切り替えます。

  ```typescript Example
  import LoginPasswordlessSmsOtp from '@auth0/auth0-acul-js/login-passwordless-sms-otp';
  const loginPasswordlessSmsOtpManager = new LoginPasswordlessSmsOtp();
  loginPasswordlessSmsOtpManager.switchConnection({
    connection: 'sms',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">

    <ParamField body="connection" type="string" required>
      切り替え先の接続名です。パスワードレスには `'email'` または `'sms'` を使用し、パスワードベースの認証には DB 接続名を使用します。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22;">
      `ulp-` がプレフィックスとして付いた追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
