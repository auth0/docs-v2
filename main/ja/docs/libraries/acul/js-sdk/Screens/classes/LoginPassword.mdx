LoginPassword クラスは、`login-password` 画面の機能を実装します。この画面では、ユーザーの識別子とパスワードを入力します。

  <img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/Em8VqaD4fPUx6FRevD8fh/a9133a19565d5d9ddc1454535a0a7a11/loginpassword.png" alt="" />
</Frame>

<div id="constructors">
  ## コンストラクター
</div>

LoginPassword 画面マネージャーのインスタンスを作成します。

```typescript Example
import LoginPassword from "@auth0/auth0-acul-js/login-password";
const loginPasswordManager = new LoginPassword();
loginPasswordManager.login({
  username: "testUser",
  password: "testPassword"
});
```

<div id="methods">
  ## メソッド
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  このメソッドは、Universal Login ページの表示言語を変更します。

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.changeLanguage({
    language: 'fr',
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LanguageChangeOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)。
    </ParamField>

    <ParamField body="language" type="string" required>
      変更先の言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      `'session'` に設定すると、選択した言語はセッション中保持されます。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` で始まる追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="federatedLogin" type="Promise<void>">
  このメソッドは、認証のためにユーザーをソーシャルまたはエンタープライズのIDプロバイダー (IdP) にリダイレクトします。

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.federatedLogin({
    connection: "google-oauth2"
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [FederatedLoginPayloadOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/FederatedLoginPayloadOptions)。
    </ParamField>

    <ParamField body="connection" type="string" required>
      使用するソーシャル接続の名前です。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  このメソッドは、コンテキストからトランザクションエラーの配列を取得します。エラーが存在しない場合は、空の配列を返します。
</ParamField>

<ParamField body="login" type="Promise<void>">
  このメソッドは、識別子と指定されたパスワードを使用してユーザーを認証します。

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.login({
    username: "testUser",
    password: "testPassword"
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [LoginPayloadOptions](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LoginPayloadOptions)。
    </ParamField>

    <ParamField body="username" type="string" required>
      ユーザーの識別子です。
    </ParamField>

    <ParamField body="password" type="string" required>
      ユーザーのパスワードです。
    </ParamField>

    <ParamField body="captcha?" type="string">
      キャプチャのcode、またはキャプチャプロバイダーからのレスポンスです。Auth0 テナントで [Bot Detection](/ja/docs/secure/attack-protection/bot-detection) が有効になっている場合、このプロパティは必須です。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22; | &#x22;undefined&#x22;">
      `ulp-` で始まる追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="switchConnection" type="Promise<void>">
  このメソッドは、データベース接続 (パスワード) とパスワードレス接続 (メールアドレス/SMS OTP) を切り替えます。

  ```typescript Example
  import LoginPassword from "@auth0/auth0-acul-js/login-password";
  const loginPasswordManager = new LoginPassword();
  loginPasswordManager.switchConnection({
    connection: "email"
  });
  ```

  **メソッドパラメーター**

  <Expandable title="パラメーター">
    <ParamField body="options">
      [SwitchConnectionOptionsLoginPassword](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/SwitchConnectionOptionsLoginPassword)。
    </ParamField>

    <ParamField body="connection" type="string" required>
      切り替え先の接続名です。パスワードレスには `'email'` または `'sms'` を使用し、パスワードベースの認証には DB 接続名を使用します。
    </ParamField>

    <ParamField body="[`key`: `string`]" type="&#x22;string&#x22; | &#x22;number&#x22; | &#x22;boolean&#x22;">
      `ulp-` で始まる追加のカスタムフィールドです (例: `'ulp-custom-field'`) 。
    </ParamField>
  </Expandable>
</ParamField>
