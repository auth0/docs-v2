---
title: "UserMembers"
---

```ts Example
export interface UserMembers {
  id: string | null;
  email: string | null;
  username: string | null;
  phoneNumber: string | null;
  picture: string | null;
  enrolledFactors: Array<string> | null;
  enrolledEmails: Array<EnrolledEmail> | null;
  enrolledPhoneNumbers: Array<EnrolledPhoneNumber> | null;
  enrolledDevices: Array<EnrolledDevice> | null;
  organizations: Organizations[] | null; // オプトイン 
  userMetadata: { [key: string]: string } | null; // オプトイン
  appMetadata: { [key: string]: string } | null; // オプトイン
}
```

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * オプトイン: これらのプロパティを画面で利用できるようにするには、明示的に設定する必要があります。詳しくは、[ACUL を設定する](/ja/docs/customize/login-pages/advanced-customizations/configure)を参照してください。

  * Auth0 サーバーが現在のユーザーを認識していない場合、次の画面では `user` プロパティを利用できません。

  | 画面          | 画面              | 画面           | 画面                     |
  | ----------- | --------------- | ------------ | ---------------------- |
  | ログイン画面      | MFA OTP 画面      | 招待受諾画面       | InterstitialCaptcha 画面 |
  | サインアップ画面    | MFA 電話番号画面      | デバイスコード画面    | パスワードリセット画面            |
  | パスキー画面      | MFA プッシュ画面      | メールアドレス確認画面  | MFA 国コード画面             |
  | チャレンジ画面     | MFA リカバリーコード画面  | ログアウト中断画面    | MFA ブラウザー機能検出画面        |
  | メールアドレス識別画面 | MFA WebAuthn 画面 | ログアウト完了画面    | MFA 登録画面               |
  | 電話番号識別画面    | 招待受諾画面          | ブルートフォース保護画面 |                        |
</Callout>
