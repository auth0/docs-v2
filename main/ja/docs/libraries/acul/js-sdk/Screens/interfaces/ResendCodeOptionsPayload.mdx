ResendCodeOptions
description Login メールアドレス確認画面の `resendCode` メソッドに指定するオプションです。
この操作は、ユーザーがメールアドレスに新しい確認用の code の送信を求めた場合に使用します。
たとえば、最初の code を受信していない場合や、有効期限が切れている場合です。

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```

  ## インデックス可能
