---
title: "DeviceCodeActivationMembers"
---

Device Code Activation 画面のメンバーを定義するインターフェース。

```ts Example
export interface DeviceCodeActivationMembers extends BaseMembers {
  /**
   * ユーザーが入力したデバイスcodeを送信します。
   * このActionは、ユーザーがデバイスに表示されたcodeを入力してフォームを送信したときにトリガーされます。
   *
   * @param {object} payload - ユーザーが入力したcodeおよびカスタムペイロードを含むオブジェクト。
   * @param {string} payload.code - ユーザーが入力したデバイスcode。
   *
   * @returns {Promise<void>} codeが正常に送信されたときに解決されるPromise。
   * 送信に失敗した場合はエラーで棄却されます。
   */
  continue(payload: ContinueOptions): Promise<void>;
}
```

  ## メソッド

<ParamField body="continue" type="Promise<void>">
  ユーザーが入力したデバイス code を送信します。
  このアクションは、ユーザーがデバイスに表示された code を入力してフォームを送信したときに実行されます。

  code が正常に送信されると resolve される Promise です。
  送信に失敗した場合は、エラーで reject されます。

  <Expandable title="パラメーター">
    <ParamField body="payload" type={<span><a href="/ja/docs/libraries/acul/js-sdk/Screens/interfaces/DeviceCodeActivationContinueOptions">DeviceCodeActivationContinueOptions</a></span>}>
      ユーザーが入力した code と、任意のカスタムペイロードを含むオブジェクトです。

      **プロパティ**

      <ParamField body="code" type="string" />
    </ParamField>
  </Expandable>
</ParamField>
