````ts Example
export interface MfaWebAuthnNotAvailableErrorMembers extends BaseMembers {
  /**
   * WebAuthn が利用できないため、ユーザーが別の MFA 方式を選択できるようにします。
   * このアクションは、ユーザーを認証器選択画面に遷移させ、利用可能な/登録済みの MFA 認証要素の中から選択できるようにします。
   *
   * @param {CustomOptions} [payload] - リクエストとともに送信するオプションのカスタムパラメーター。
   * @returns {Promise<void>} 'pick-authenticator' アクションの送信が成功したときに解決される Promise。
   * @throws {Error} フォームの送信が失敗した場合（例: ネットワークの問題、サーバー側のバリデーションエラー）にエラーをスローします。
   * @example
   * ```typescript
   * // 'sdk' は MfaWebAuthnNotAvailableError のインスタンスであると仮定します
   * try {
   *   await sdk.tryAnotherMethod();
   *   // 成功時、Auth0 が認証器選択画面へのリダイレクトを処理します。
   * } catch (error) {
   *   console.error('Failed to switch to another MFA method:', error);
   *   // 必要に応じて、失敗をユーザーに通知するよう UI を更新します。
   * }
   * ```
   */
  tryAnotherMethod(payload?: CustomOptions): Promise<void>;
}

<ParamField body="tryAnotherMethod" type="Promise<void>">
  WebAuthn を利用できない場合に、ユーザーが別の MFA方式を選択できるようにします。
  このアクションにより、ユーザーは認証器選択画面に移動し、そこで
  利用可能な他の登録済み MFA 認証要素を選択できます。

  「pick-authenticator」アクションが正常に送信されると解決される Promise です。

  #### 例外

  フォームの送信に失敗した場合は、エラーをスローします (例: ネットワークの問題、サーバー側の検証エラー) 。

  ```typescript Example
  // 'sdk' は MfaWebAuthnNotAvailableError のインスタンスであると想定します
  try {
    await sdk.tryAnotherMethod();
    // 成功すると、Auth0 が認証器選択画面へのリダイレクトを処理します。
  } catch (error) {
    console.error('Failed to switch to another MFA method:', error);
    // 必要に応じて、失敗したことをユーザーに知らせるために UI を更新します。
  }
  ```
