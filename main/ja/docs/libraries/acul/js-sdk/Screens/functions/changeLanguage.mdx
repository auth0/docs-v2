> **changeLanguage**(`options`: [`LanguageChangeOptions`](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)): `Promise`&lt;`void`&gt;

`change-language` アクションで新しいロケール設定をサーバーに送信し、選択した言語でインターフェースを再レンダリングします。

<div id="parameters">
  ## パラメーター
</div>

<ParamField body="options" type="LanguageChangeOptions" required>
  言語変更の構成です。[`LanguageChangeOptions`](/ja/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions) を参照してください。

  <Expandable title="options">
    <ParamField body="language" type="string" required>
      変更先の言語のロケールコードです (例: `'en'`、`'fr'`、`'es'`) 。Auth0 テナントで有効になっているロケールのいずれかである必要があります。
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      言語設定の永続化スコープです。現在サポートされているのは `"session"` のみです。
    </ParamField>
  </Expandable>
</ParamField>

<div id="returns">
  ## 戻り値
</div>

`Promise`&lt;`void`&gt;

ロケール設定が送信され、選択した言語で画面が再レンダリングされると解決されます。
