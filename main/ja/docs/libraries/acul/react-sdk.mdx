Auth0 ACUL React SDK を使用すると、最小限の手間でカスタムの Universal Login エクスペリエンスを構築できます。ログイン、サインアップ、パスワードレス、パスキー登録のいずれの場合でも、この SDK は最新の認証フローにスムーズに対応できるよう設計されています。

<div id="why-choose-the-auth0-acul-react-sdk">
  ### Auth0 ACUL React SDK を選ぶ理由
</div>

* **迅速な統合:** `login-id` や `signup-password` などの構築済みの画面をそのまま導入できます。
* **モダンな機能:** パスワードレス、パスキー、ソーシャルログインに対応しています。
* **柔軟なモジュール:** 必要なものだけをインポートできます。
* **統一された状態管理:** 単一のトランザクションオブジェクトで、あらゆる認証フローを処理できます。

<div id="architecture">
  ### アーキテクチャ
</div>

<div id="how-it-works">
  #### どのように機能しますか？
</div>

SDK は、モジュール性と拡張性を考慮して構築されています。全体の仕組みは次のとおりです。

* **画面:** `login-id` や `signup-password` などの各認証フローは、それぞれ独立したモジュールです。
* **送信ハンドラー:** フォーム送信と画面遷移を管理します。
* **動的フォーム:** 画面の設定に基づいて、フィールドやバリデーションを調整します。
* **ヘルパープロパティ:** ブランディングやトランザクションなどの共通データに、一貫した方法でアクセスできます。
