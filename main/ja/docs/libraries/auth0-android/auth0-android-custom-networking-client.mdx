このライブラリでは、一般的な構成に合わせてネットワーキングクライアントの動作をカスタマイズしたり、独自のネットワーキングクライアント実装を定義して使用したりできます。

`Auth0` クラスは、リクエストの送信時に使用する `NetworkingClient` で設定できます。デフォルトのクライアントでは、カスタム タイムアウト値、すべてのリクエストで送信するヘッダー、リクエスト/レスポンス情報をログに記録するかどうか (本番環境以外でのデバッグ用途のみ) を設定できます。さらに高度な構成が必要な場合は、`NetworkingClient` の独自実装を指定できます。

<div id="timeout-configuration">
  ### タイムアウトの設定
</div>

export const codeExample1 = `val netClient = DefaultClient(
    connectTimeout = 30,
    readTimeout = 30
)

val account = Auth0("{yourClientId}", "{yourDomain}")
account.networkingClient = netClient`;
