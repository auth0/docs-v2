title: Auth0 ライブラリの一般的な認証エラー
---

サインアップまたはログインの処理中に、ユーザーの操作や入力データが原因でエラーが発生することがあります。ここでは、認証に Auth0 ライブラリを使用する場合に発生する可能性がある、代表的なエラーの一覧を示します。

<div id="sign-up">
  ## サインアップ
</div>

サインアップに失敗した場合、よくあるエラーは次のとおりです。

| **エラー**                                         | **説明**                                                                       |
| ----------------------------------------------- | ---------------------------------------------------------------------------- |
| **invalid&#95;password**                        | 使用したパスワードが、その接続のパスワードポリシーに準拠していない場合                                          |
| **invalid&#95;signup**                          | サインアップしようとしているユーザーが無効です                                                      |
| **password&#95;dictionary&#95;error**           | 選択したパスワードが一般的すぎます                                                            |
| **password&#95;no&#95;user&#95;info&#95;error** | 選択したパスワードがユーザー情報に基づいています                                                     |
| **password&#95;strength&#95;error**             | 選択した[パスワードが弱すぎます](/ja/docs/authenticate/database-connections/password-strength) |
| **unauthorized**                                | このアプリケーションにはサインアップできない場合です。特定のRuleへの違反が原因である可能性があります                         |
| **user&#95;exists**                             | サインアップしようとしているユーザーは、すでに登録されています                                              |
| **username&#95;exists**                         | サインアップに使用しようとしているusernameは、すでに使用されています                                       |

<div id="log-in">
  ## ログイン
</div>

ログインに失敗した場合、最も一般的なエラーは次のとおりです。

| **エラー**                               | **説明**                                                                                                                                                                                                           |
| ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **access&#95;denied**                 | Web ベースの認証を使用している場合、OAuth2 の仕様に従ってリソースサーバーがアクセスを拒否します                                                                                                                                                            |
| **invalid&#95;user&#95;password**     | 認証に使用したusernameおよび/またはパスワードが無効です                                                                                                                                                                                    |
| **mfa&#95;invalid&#95;code**          | ユーザーが指定した <Tooltip tip="多要素認証（MFA）: SMS で送信されるコードなど、usernameとパスワードに加えて別の認証要素を使用するユーザー認証プロセス。" cta="用語集を見る" href="/ja/docs/glossary?term=multi-factor+authentication+%28MFA%29">多要素認証 (MFA) </Tooltip> コードが無効か、有効期限切れです |
| **mfa&#95;registration&#95;required** | 管理者が[多要素認証](/ja/docs/secure/multi-factor-authentication)を必須にしていますが、ユーザーは登録していません                                                                                                                                    |
| **mfa&#95;required**                  | 認証するには、ユーザーが[多要素認証](/ja/docs/secure/multi-factor-authentication)コードを入力する必要があります                                                                                                                                     |
| **password&#95;leaked**               | パスワードが漏えいしているため、別のパスワードを使用する必要がある場合                                                                                                                                                                              |
| **PasswordHistoryError**              | サインアップまたは更新時に指定したパスワードは、すでに使用されています ([password history](/ja/docs/authenticate/database-connections/password-options#password-history) 機能が有効な場合に報告されます)                                                              |
| **PasswordStrengthError**             | 指定したパスワードが接続の[強度要件](/ja/docs/authenticate/database-connections/password-strength)を満たしていません                                                                                                                          |
| **too&#95;many&#95;attempts**         | サインインの試行回数が多すぎるため、アカウントがブロックされています                                                                                                                                                                               |
| **unauthorized**                      | サインインしようとしているユーザーはブロックされています                                                                                                                                                                                     |
