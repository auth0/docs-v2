title: Management API アクセストークン
---

[Auth0 Management API](https://auth0.com/docs/api/management/v2) を呼び出すには、適切な <Tooltip tip="Access Token: API へのアクセスに使用される、不透明な文字列または JWT 形式の認可資格情報です。" cta="用語集を表示" href="/ja/docs/glossary?term=access+token">アクセストークン</Tooltip> を指定する必要があります。このトークンには、適切な <Tooltip tip="Access Token: API へのアクセスに使用される、不透明な文字列または JWT 形式の認可資格情報です。" cta="用語集を表示" href="/ja/docs/glossary?term=audience">オーディエンス</Tooltip> とスコープが含まれている必要があります。

<div id="audience">
  ## 対象者
</div>

<Tooltip tip="Management API: 顧客が管理タスクを実行するための製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> のオーディエンス値は `https://{yourDomain}/api/v2/` です。

<div id="scopes">
  ## スコープ
</div>

各 Management API エンドポイントには、必要なスコープのセットが定義されています。詳細については、[Management API Explorer](https://auth0.com/docs/api/management/v2/)を参照してください。

<div id="lifetime">
  ## 有効期間
</div>

Management API のデフォルトのトークン有効期間は、86,400 秒 (24 時間) です。

API ごとに[トークンの有効期間を設定](/ja/docs/secure/tokens/access-tokens/update-access-token-lifetime)できます。

<div id="security">
  ## セキュリティ
</div>

* **アクセストークン**: 一度発行されたアクセストークンは失効できません。Auth0 では、ユースケースで許容される範囲で、アクセストークンの有効期間を可能な限り短く設定することを推奨しています。
* **<Tooltip tip="リフレッシュトークン: ユーザーに再度ログインを求めることなく、新しいアクセストークンを取得するために使用されるトークン。" cta="用語集を見る" href="/ja/docs/glossary?term=Refresh+token">リフレッシュトークン</Tooltip>**: Dashboard、Authentication API、または Management API を使用して、[リフレッシュトークンを失効させる](/ja/docs/secure/tokens/refresh-tokens/revoke-refresh-tokens)ことができます。
* **<Tooltip tip="クライアントシークレット: クライアント（アプリケーション）が認可サーバーに対して認証を行うために使用するシークレットです。クライアントと認可サーバーのみが知るべきものであり、推測されないよう十分にランダムである必要があります。" cta="用語集を見る" href="/ja/docs/glossary?term=Client+secret">クライアントシークレット</Tooltip>**: Dashboard または Management API を使用して、[アプリケーションのクライアントシークレットをローテーションする](/ja/docs/get-started/applications/rotate-client-secret)ことができます。

<div id="quota">
  ## クォータ
</div>

トークンのクォータは、サブスクリプションプランによって決まります。現在のクォータは、Auth0 Support Center で[確認できます](https://support.auth0.com/reports/quota)。料金やクォータについてご不明な点がある場合は、[営業チームにお問い合わせください](https://auth0.com/contact-us)。

* [Auth0 の内部オーディエンス](/ja/docs/secure/tokens/access-tokens/get-access-tokens#control-access-token-audience)に対して発行されたトークンは、クォータにカウントされません。
* カスタムオーディエンスに対して発行されたトークンは、クォータにカウントされ、[Auth0 Management API のレート制限](/ja/docs/troubleshoot/customer-support/operational-policies/rate-limit-policy/management-api-endpoint-rate-limits)の対象になります。
