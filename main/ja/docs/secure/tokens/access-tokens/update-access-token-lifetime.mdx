<Tooltip tip="アクセストークン: API へのアクセスに使用される、不透明な文字列または JWT 形式の認可情報。" cta="用語集を見る" href="/ja/docs/glossary?term=access+token">アクセストークン</Tooltip>の有効期間は、<Tooltip tip="アクセストークン: API へのアクセスに使用される、不透明な文字列または JWT 形式の認可情報。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>で変更できます。

<div id="configure-access-token-lifetime">
  ## アクセストークンの有効期間を設定する
</div>

1. [Dashboard &gt; Applications &gt; APIs](https://manage.auth0.com/#/apis) に移動し、表示する API の名前を選択します。

   <Frame>![Dashboard Applications APIs List](/docs/images/cdy7uua7fh8z/3rhmhghYZDSi6YWHRA5yMQ/c71340259481b0b6787d5f3887cfda0f/dashboard-apis-list.png)</Frame>
2. **Access Token Expiration** の **Maximum Access Token Lifetime** フィールドを探します。

   <Frame>![Dashboard - API - Token Settings - Expiration](/docs/images/cdy7uua7fh8z/2FFfqqmntXL9CnGrSxea9M/810e5542553168ffcd6081a10bc2db2f/image2.png)</Frame>
3. この API 用に発行されるアクセストークンの有効期間を、希望する値で入力します (単位: 秒) 。

   * デフォルト値は `86,400` 秒 (24 時間) です。
   * 最大値は `2,592,000` 秒 (30 日) です。
4. **Save Changes** を選択します。

<div id="token-expiration-for-browser-flows">
  ## ブラウザフローのトークンの有効期限
</div>

**Implicit / Hybrid Flow Access Token Lifetime** フィールドは、implicit フローおよび hybrid フローを通じて API 向けに発行されるアクセストークンに関するものであり、ブラウザから開始されるすべてのフローを対象としているわけではありません。

たとえば、PKCE フロー (`auth0-js-spa` SDK で使用) はブラウザから開始できますが、参照されるのは **Implicit / Hybrid Flow Access Token Lifetime** の値ではなく、**Token Expiration** の値です。

<div id="restricted-lifetime-for-mfa-access-tokens">
  ## MFA アクセストークンの有効期間の制限
</div>

`{yourAuth0Domain}/mfa` の<Tooltip tip="対象者: 発行されたトークンの対象者を一意に識別する識別子です。トークンでは aud という名前で表され、その値には、IDトークンの場合はアプリケーション（クライアントID）の ID、アクセストークンの場合は API（API 識別子）の ID が含まれます。" cta="用語集を見る" href="/ja/docs/glossary?term=audience">オーディエンス</Tooltip>を持つアクセストークンの有効期間は、セキュリティ上の理由により `600` 秒 (10 分) に制限されており、変更できません。
