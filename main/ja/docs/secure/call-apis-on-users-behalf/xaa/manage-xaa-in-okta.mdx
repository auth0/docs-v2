エンドツーエンドのテスト環境の設定が完了したら、Okta Admin Console で、有効な XAA アプリケーション間の接続を管理できます。

<Frame>![](/docs/images/xaa/manage_xaa_in_okta.png)</Frame>

Okta Admin Console で、次の操作を行います。

1. **Applications &gt; Applications** に移動し、Resource App (例: Todo0) を選択します。
2. **Manage Connections** タブで、次を追加します。
   * Requesting Apps: SaaS アプリケーションに接続できるアプリケーション
   * Resource Apps: SaaS アプリケーションから接続できるアプリケーション

エンドツーエンドのテスト環境では、承認済みの Requesting App として Agent0 またはテストに使用するアプリケーションを追加します。
