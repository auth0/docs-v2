テナントログには、成功したログインイベントと失敗したログインイベントのエントリが含まれており、<Tooltip tip="Adaptive Multi-factor Authentication: ログイン試行の信頼度が低いと判断された場合にのみ、ユーザーに対してトリガーされる多要素認証 (MFA)。" cta="用語集を表示" href="/ja/docs/glossary?term=Adaptive+MFA">Adaptive MFA</Tooltip> のリスク評価スコアに関連する情報も記録されます。これらは、ルールのコンテキストオブジェクトで使用できるものと同じエントリです。Adaptive <Tooltip tip="Adaptive Multi-factor Authentication: ログイン試行の信頼度が低いと判断された場合にのみ、ユーザーに対してトリガーされる多要素認証 (MFA)。" cta="用語集を表示" href="/ja/docs/glossary?term=MFA">MFA</Tooltip> のエントリは、次の構造になっています。

```json lines expandable
{
  "date": "2020-06-26T15:12:43.654Z",
  "type": "s",
  "details": {
    "riskAssessment": {
      "confidence": "high",
      "assessments": {
        "UntrustedIP": {
          "confidence": "high",
          "code": "not_found_on_deny_list"
    	},
    	"NewDevice": {
      	  "confidence": "medium",
      	  "code": "partial_match",
          "details": {
            "device": "unknown",
            "useragent": "known",
          }
    	},
    	"ImpossibleTravel": {
      	  "confidence": "high",
      	  "code": "minimal_travel_from_last_login"
    	}
      }
    }
  },
  "description": "Successful login"
}
```

<div id="attack-protection-details">
  ## 攻撃対策の詳細
</div>

Auth0 は、Rules の実行に関する情報に加えて、Adaptive MFA の <Tooltip tip="Attack Protection: Auth0 が提供する、ブルートフォース対策、不審な IP のスロットリング、漏えいしたパスワードの検出、ボット検出、アダプティブ多要素認証など、攻撃を検出して軽減するための機能。" cta="用語集を見る" href="/ja/docs/glossary?term=attack+protection">攻撃対策</Tooltip> に関する情報もテナントログに含めます。Rules に表示されるのと同じデータ構造が、テナントログにも表示されます。

```json lines expandable
{
  "date": "2020-06-26T15:12:43.654Z",
  "type": "s",
  "details": {
	"anomalyDetection": {
  	  "confidence": "high",
  	  "assessments": {
    	    "UntrustedIP": {
      	    "confidence": "high",
      	    "reason": "ip not found",
      	    "code": "ip_not_found"
    	    },
    	    "NewDevice": {
      	    "confidence": "medium",
      	    "reason": "match useragent",
      	    "code": "match_useragent"
    	    },
    	    "ImpossibleTravel": {
      	    "confidence": "high",
      	    "reason": "minimal travel",
      	    "code": "minimal_travel_from_last_login"
    	    }
  	  }
	},
  },
  "description": "Successful login"
}
```
