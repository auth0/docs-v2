title: Auth0 MFA API
---

Auth0 には、[Universal Login](/ja/docs/authenticate/login/auth0-universal-login) を使用する、組み込みの <Tooltip tip="多要素認証（MFA）: SMS で送信される認証コードなど、ユーザー名とパスワードに加えて別の認証要素を使用するユーザー認証プロセス。" cta="用語集を表示" href="/ja/docs/glossary?term=multi-factor+authentication">多要素認証</Tooltip> (MFA) の登録フローと認証フローが用意されています。次のような場合は、[MFA API](https://auth0.com/docs/api/authentication#multi-factor-authentication) を使用します。

* [リソースオーナー パスワード グラントでユーザーを認証する](/ja/docs/secure/multi-factor-authentication/authenticate-using-ropg-flow-with-mfa)。
* [ユーザーが自分の認証要素を管理できる](/ja/docs/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api) インターフェイスを構築する。

MFA API を使用するには、アプリケーションで MFA グラントタイプを有効にする必要があります。<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 のメイン製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> で MFA グラントを有効にするには、[Dashboard &gt; Applications &gt; Advanced Settings &gt; Grant Types](https://manage.auth0.com/#/applications) に移動し、**MFA** を選択します。詳しくは、[多要素認証を有効にする](/ja/docs/secure/multi-factor-authentication/enable-mfa) を参照してください。

<div id="limitations">
  ## 制限事項
</div>

MFA API は、SMS、音声通話、Auth0 Guardian アプリ経由のプッシュ、メールアドレス、OTP の各認証要素に対応するよう設計されています。現時点では、Duo または WebAuthn での登録はサポートされていません。
