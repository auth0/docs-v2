title: MFA Widget のテーマオプション
---

Classic Login で <Tooltip tip="多要素認証（MFA）: SMS 経由の code など、username とパスワードに加えて別の要素を使用するユーザー認証プロセス。" cta="用語集を表示" href="/ja/docs/glossary?term=MFA">MFA</Tooltip> ページに独自の HTML を使用する場合、Auth0 MFA Widget を利用します。この Widget には、次の制限があります。

* メールアドレスを使用した MFA には対応していません。
* ユーザーが複数の要素を登録している場合、どれを使用するかは選択できません。MFA Widget は、最も安全性の高い要素でログインするよう求めます。
* <Tooltip tip="Universal Login: アプリケーションは、ユーザーの本人確認のために、Auth0 の認可サーバーでホストされる Universal Login にリダイレクトされます。" cta="用語集を表示" href="/ja/docs/glossary?term=Universal+Login">Universal Login</Tooltip> の[国際化](/ja/docs/customize/internationalization-and-localization/universal-login-internationalization)機能は利用しません。

MFA Widget のテーマオプションは、`theme` プロパティ配下の名前空間にあります。

| オプション          | 説明                                                                                     |
| -------------- | -------------------------------------------------------------------------------------- |
| `icon`         | ヘッダーに使用する画像の URL です。デフォルトは Auth0 ロゴです。推奨される高さは 58 ピクセルです。                              |
| `primaryColor` | MFA Widget のメインカラーを定義します。カスタム `icon` を使用する場合に、調和の取れた色合いにするのに役立ちます。デフォルトは `#ea5323` です。 |

<div id="examples">
  ## 例
</div>

```js lines
return new Auth0MFAWidget({

...

  theme: {
    icon: 'https://example.com/assets/logo.png',
    primaryColor: 'blue'
  },
  
...  
  
})
```

[auth0-guardian.js](https://github.com/auth0/auth0-guardian.js) を使用した [Guardian の基本的なウィジェットの例](https://github.com/auth0/auth0-guardian.js/blob/master/example/basic_widget.html) を参照してください。詳しくは、[readme](https://github.com/auth0/auth0-guardian.js/tree/master/example) を参照してください。
