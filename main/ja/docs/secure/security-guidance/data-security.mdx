title: データセキュリティ
---

* [Allow List 用の Auth0 IP アドレス](/ja/docs/secure/security-guidance/data-security/allowlist) - ご利用の環境がファイアウォールの内側にある場合、カスタムデータベース接続、Actions、Hooks、Rules を正しく機能させるには、特定の Auth0 IP アドレスを Allow List に追加する必要があることがあります。
* [ユーザー属性を Deny List に追加する](/ja/docs/secure/security-guidance/data-security/denylist) - プライバシー上の理由で Auth0 データベースに保存すべきでないユーザーフィールドがある場合は、ユーザー属性を Deny List に追加できます。
* [ユーザーデータの保存](/ja/docs/secure/security-guidance/data-security/user-data-storage) - ユーザーデータの保存に Auth0 のデータストアを使用する場合と、外部データベースを使用する場合の違いについて説明します。
* [トークンの保存](/ja/docs/secure/security-guidance/data-security/token-storage) - トークンベース認証で使用されるトークンを、どこにどのように保存するかについて説明します。
