title: Dashboard で Rules を管理
description: Auth0 Dashboard でテナント ACL Rule を管理する方法を説明します。
---

Auth0 Dashboard を使用すると、テナント ACL Rule の一覧を表示し、それぞれを個別に有効または無効にできます。

<div id="view-list-of-rules">
  ## Rule の一覧を表示する
</div>

既存のRuleの一覧を表示するには、[Dashboard &gt; セキュリティ &gt; アクセス制御](https://manage.auth0.com/#/security/access-control)に移動します。

| 列名    | 説明                                                                          | 関連する API レスポンス フィールド |
| ----- | --------------------------------------------------------------------------- | -------------------- |
| 優先度   | Ruleが評価される順序を示す序数                                                            | `priority`           |
| 名前    | Ruleの目的または機能の説明                                                              | `description`        |
| アクション | Ruleが実行する[アクション](/ja/docs/secure/tenant-access-control-list/reference#actions)。 | `action`             |
| ステータス | Ruleが有効か無効かを示す値。                                                             | `active`             |
| 最終更新  | Ruleが最後に更新された日時。                                                             | `updated_at`         |

![Auth0 Dashboard でテナント ACL Rule の一覧を表示する](/docs/images/cdy7uua7fh8z/4093f9a39aba48bdf8d579c134dc4da5/Screenshot_2025-09-03_at_12.33.47.png)

<div id="enable-or-disable-a-rule">
  ## Rule を有効または無効にする
</div>

特定の Rule は、その横にあるボタンを切り替えることで有効または無効にできます。

**Status** フィールドには、それに応じて Rule が `Active` または `Disabled` と表示されます。
