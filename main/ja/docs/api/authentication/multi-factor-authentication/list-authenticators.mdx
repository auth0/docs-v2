<div id="endpoint">
  ## エンドポイント
</div>

`GET /mfa/authenticators`

アプリケーションに関連付けられている認証器の一覧を返します。

このエンドポイントにアクセスするには、Authorization ヘッダーに次の claims を含むアクセストークンを設定する必要があります。

* `scope`: `read:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="headers">
  ## ヘッダー
</div>

<ParamField header="Authorization" type="string" required>
  Bearer ACCESS&#95;TOKEN
</ParamField>

<div id="response-schema">
  ## レスポンススキーマ
</div>

<ResponseSchema>
  レスポンスは、認証要素オブジェクトの配列です。

  <ResponseField name="id" type="string" required>
    認証要素の ID。
  </ResponseField>

  <ResponseField name="authenticator_type" type="string" required>
    認証要素の種類。
  </ResponseField>

  <ResponseField name="active" type="boolean" required>
    認証要素がアクティブかどうかを示します。
  </ResponseField>

  <ResponseField name="oob_channels" type="string">
    この認証要素がサポートする OOB チャネル。
  </ResponseField>

  <ResponseField name="name" type="string">
    認証要素の名前。
  </ResponseField>
</ResponseSchema>

<div id="response-messages">
  ## レスポンスメッセージ
</div>

| ステータス | 説明      |
| ----- | ------- |
| 200   | 認証器の一覧。 |
