title: "認証器 を削除"

---

`DELETE /mfa/authenticators/{AUTHENTICATOR_ID}`

ID を使用して、関連付けられている authenticator を削除します。authenticator ID は、authenticator の一覧を取得して確認できます。

このエンドポイントにアクセスするには、Authorization ヘッダーに次のクレームを含むアクセストークンを設定する必要があります。

* `scope`: `remove:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="parameters">
  ## パラメーター
</div>

<ParamField path="AUTHENTICATOR_ID" type="string" required>
  削除する認証器の ID。
</ParamField>

<ParamField header="Authorization" type="string" required>
  ログイン時に取得したアクセストークン。
</ParamField>

<div id="response">
  ## レスポンス
</div>

| Status | Description                |
| ------ | -------------------------- |
| 204    | 認証器 は正常に削除されました。 |
| 401    | 未認証 - 無効なアクセストークンです。       |
| 404    | 認証器 が見つかりません。    |
