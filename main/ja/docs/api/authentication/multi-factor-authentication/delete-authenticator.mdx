---
title: "認証器 を削除"
description: "ID を使用して、関連付けられている MFA 認証要素を削除します。"
---

<div id="endpoint">
  ## エンドポイント
</div>

`DELETE /mfa/authenticators/{AUTHENTICATOR_ID}`

ID を使用して、関連付けられている認証要素を削除します。認証要素 ID は、認証要素の一覧を取得して確認できます。

このエンドポイントにアクセスするには、Authorization ヘッダーに次のクレームを含むアクセストークンを設定する必要があります。

* `scope`: `remove:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="path-parameters">
  ## パスパラメーター
</div>

<ParamField path="AUTHENTICATOR_ID" type="string" required>
  削除する認証要素の ID。
</ParamField>

<div id="headers">
  ## ヘッダー
</div>

<ParamField header="Authorization" type="string" required>
  ログイン時に取得したアクセストークン。
</ParamField>

<div id="response-messages">
  ## レスポンスメッセージ
</div>

| Status | Description          |
| ------ | -------------------- |
| 204    | 認証器 は正常に削除されました。     |
| 401    | 未認証 - 無効なアクセストークンです。 |
| 404    | 認証器 が見つかりません。        |
