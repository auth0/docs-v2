---
title: "WS-Federation Metadata を取得"
description: "Auth0 テナントの WS-Federation Metadata を返します。"
---

<div id="endpoint">
  ## エンドポイント
</div>

`GET /wsfed/FederationMetadata/2007-06/FederationMetadata.xml`

このエンドポイントは WS-Federation Metadata を返します。

<div id="learn-more">
  ### 詳細
</div>

* [WS-Federation](https://auth0.com/docs/authenticate/protocols/ws-fed-protocol)

<div id="response-messages">
  ## レスポンスメッセージ
</div>

| ステータス | 説明                                 |
| ----- | ---------------------------------- |
| 200   | WS-Federation Metadata を含む成功レスポンス。 |
