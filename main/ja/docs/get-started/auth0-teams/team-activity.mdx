---
description: Team Activity では、チームオーナーがチームメンバーによって生成されたイベントログを表示し、監査できます。
title: Team Activity
---

Team Activity では、チームオーナーが、メンバーの Auth0 アカウントで実行されたアクティビティに関連する監査ログを表示できます。

Team Activity レポートは、Teams Dashboard の **Reports** セクションにあります。

Team Activity では、イベントを生成したメンバー、イベントタイプ、イベントの生成日時の概要を確認できます。

**詳細を表示** を選択すると、イベントの説明、イベントのステータス、イベントをトリガーしたユーザーの IP アドレス、ユーザーエージェントの詳細など、さらに詳しい情報を確認できます。

<div id="event-types">
  ## イベントタイプ
</div>

Team Activity のイベントタイプは以下のとおりです。

| イベントタイプ    | 説明                                   |
| ---------- | ------------------------------------ |
| チーム招待      | チームメンバー招待が作成された<br />チームメンバー招待が削除された |
| チームメンバー    | チームメンバーが削除された<br />チームメンバーのロールが更新された |
| セキュリティポリシー | セキュリティポリシーが更新された<br />ソーシャル接続が更新された  |
| チーム設定      | チーム設定が更新された                          |

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Dashboard ユーザーによる Auth0 および Teams Dashboard への認証や、そのほかのイベントのサポートは今後提供される予定です。
</Callout>

<div id="event-schema">
  ## イベントスキーマ
</div>

Team Activity のイベントスキーマを以下に示します。

```json lines
{  
    "id": "string", 
    "team_id": "string",
    "user_id": "string",
    "type": "string",
    "description": "string",
    "status": "string",
    "details": "object",
    "ip": "string",
    "hostname": "string",
    "user_agent": "string",
    "version": "string",
    "created_at": "date",
    "team_slug": "string",
    "user": "object"
}
```

| 名前            | 型      | 説明                                               |
| ------------- | ------ | ------------------------------------------------ |
| `id`          | 文字列    | イベントログの一意の識別子です。                                 |
| `team_id`     | 文字列    | チームの一意の識別子です。                                    |
| `user_id`     | 文字列    | イベントを生成したユーザーの一意の識別子です。                          |
| `type`        | 文字列    | イベントタイプです。                                       |
| `description` | 文字列    | イベントタイプの概要です。                                     |
| `status`      | 文字列    | イベントのステータスです。指定できる値は `Success` または `Failure` です。 |
| `details`     | オブジェクト | 記録されたイベントの詳細です。                                  |
| `ip`          | 文字列    | イベントログを生成したユーザーの IP アドレスです。                      |
| `hostname`    | 文字列    | イベントが実行された Auth0 App の FQDN です。                  |
| `user_agent`  | 文字列    | Web ブラウザーおよびデバイス OS の種類とバージョンです。                 |
| `version`     | 文字列    | アクティビティログの生成時に使用されたスキーマのバージョンです。                 |
| `created_at`  | 日時     | イベントログが生成された日時です。                                |
| `team_slug`   | 文字列    | チームを一意に識別する文字列です。                                |
| `user`        | オブジェクト | 名前、アバター URL、チームメンバーシップのステータス (Boolean) を含みます。    |

<div id="audit-log-retention">
  ## 監査ログの保持期間
</div>

Team Activity の監査ログの保持期間は、プランによって異なります。詳しくは、[料金ページ](https://auth0.com/pricing) を参照してください。
