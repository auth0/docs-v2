description: Auth0 テナントの Auth0 Dashboard ユーザーロールを編集する方法を説明します。
title: テナントメンバーの編集
---

テナント管理者であれば、ユーザーに割り当てられているロールを変更できます。変更は、ユーザーが Dashboard を再読み込みすると反映されます。

ユーザーには複数のロールを割り当てることができ、それらの権限は組み合わせて適用されます。あらかじめテナントメンバーを追加しておく必要があります。

1. [Dashboard &gt; Settings &gt; テナントメンバー](https://manage.auth0.com/#/tenant/admins) に移動します。
2. ユーザーの名前またはメールアドレスの右側にある **...** をクリックし、**Edit Roles** をクリックします。
   このオプションが表示されない場合、そのユーザーはまだ招待を承諾していません。
3. ロールを追加で選択するか、ロールを削除します。
4. **Save** をクリックします。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  ユーザーの権限は、Dashboard を再読み込みすると表示されます。ユーザーが再読み込みしない場合、特定の領域へのアクセス権がないことを示すエラーが発生することがあります。そのようなエラーが発生した場合は、ユーザーはいったんログアウトしてから再度ログインする必要があります。
</Callout>

**パスワードリセット**: ご自身または Dashboard ユーザーのいずれかがログインで問題に直面している場合は、詳細について [Reset Auth0 Account Password](/ja/docs/troubleshoot/customer-support/reset-account-passwords) を参照してください。

**メールアドレスが無効**: テナントメンバーのメールアドレスを確認し、必要に応じて更新する方法については、[Update Dashboard User Email Addresses](/ja/docs/get-started/manage-dashboard-access/update-dashboard-user-email) を参照してください。
