description: Auth0 テナントの Auth0 Dashboard へのアクセス権からテナントメンバーを削除する方法を説明します。
title: テナントメンバーを削除する
---

テナント管理者であれば、Auth0 Dashboard へのアクセスを取り消すことができます。

1. [Dashboard &gt; Settings &gt; テナントメンバー](https://manage.auth0.com/#/tenant/admins) に移動します。
2. ユーザーの名前またはメールアドレスの右側にある **...** をクリックし、**Remove** をクリックします。
3. **Save** をクリックします。
