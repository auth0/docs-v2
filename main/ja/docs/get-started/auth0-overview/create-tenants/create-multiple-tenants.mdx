---
description: Auth0 Dashboard を使用して追加のテナントを作成する方法を説明します。
title: 複数のテナントを作成する
---

複雑な構成に対応するため、<Tooltip tip="Auth0 Dashboard: サービスを構成するための Auth0 の主要製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> では、複数のテナントを設定して[異なる環境](/ja/docs/get-started/auth0-overview/create-tenants/set-up-multiple-environments)を作成できます。たとえば、2 つの異なるドメイン (1 つは内部向け、もう 1 つは公開向け) を用意したり、アプリケーションごとにユーザーのログイン方法を変えたりできます。これを実現するには、複数の Auth0 テナントを作成し、サポートが必要なアプリケーションやユーザーグループごとに、アプリケーション、接続、ユーザーのセットを分けられるようにします。

1. [Auth0 Dashboard](https://manage.auth0.com/#/) に移動し、テナント名を選択して、**テナントを作成** を選択します。

   <Frame>![Dashboard Tenant Drop-Down Menu Create Tenant](/docs/images/cdy7uua7fh8z/53KetqhNIGDs6N5cqJdQtz/5efad1137eb8b04031b30137c2fc5ac5/2025-02-26_14-51-39.png)</Frame>
2. 使用する **テナントドメイン** を入力し、**リージョン** を選択して、**作成** を選択します。
