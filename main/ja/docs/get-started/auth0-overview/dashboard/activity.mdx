アクティビティページには、アクティブユーザーやログイン失敗など、Auth0テナントに関する主要なデータの概要が表示されます。

<div id="things-to-know">
  ## 知っておくべきこと
</div>

* テナントの アクティビティページを表示するには、管理者ロール (Administrator や Owner など) を持つアカウントが必要です。
* **Support Access** ロールには、テナントの アクティビティページでメトリクスを表示するための読み取り専用アクセスがありますが、設定を変更することはできません。
* データの整合性を確保するため、日付ピッカーでは当日は選択できず、データが表示されるまでに最大 24 時間かかる場合があります。

<div id="metrics">
  ## メトリクス
</div>

テナントの アクティビティページには、以下が表示されます。

<div id="totals">
  ### 合計
</div>

ページ上部には、テナント内のユーザー、アプリケーション、API、接続の総数が表示されます。

<Frame>![Dashboard Activity ページのテナント合計の例](/docs/images/cdy7uua7fh8z/4wcwZDIhDQVD0vhLxg6k49/b5d36a70ad9a8bdbe8e83b984053ee99/dashboard-activity-totals.png)</Frame>

<div id="daily-active-users">
  ### 日次アクティブユーザー数
</div>

その日に認証に成功した (ログイン成功) か、認可アクティビティに成功した (トークン交換成功) ユニークユーザー数です。これは、各ユーザーの成功したログイベントに基づいて計算されます。グラフ上の点にマウスポインターを合わせると、各日のユニークユーザー数を確認できます。

<Frame>![Dashboard Activity ページのアクティブユーザー折れ線グラフの例](/docs/images/cdy7uua7fh8z/Ogdesg0DscG5NhPAQ4zS0/9e8d93b1e65c51b76490cda264db01c7/active_users.png)</Frame>

<div id="user-retention">
  ### ユーザー定着率
</div>

指定した期間中にアクティブだったユーザーの割合です。テナント内の全ユーザー数に占めるアクティブユーザー数から算出されます。

<Frame>![Dashboard アクティビティページのユーザー定着率の折れ線グラフの例](/docs/images/cdy7uua7fh8z/4Fm2nJoZl9yOpw6Siy5Vem/09cc8adb31ad0905456228551c0a9bc6/user_retention.png)</Frame>

<div id="signups">
  ### サインアップ
</div>

成功したユーザーのサインアップ数です。これには、設定済みの任意の接続 (データベース、ソーシャル、エンタープライズ) を介した新規ユーザー登録が含まれます。

<Frame>![Dashboard Activity ページのサインアップ折れ線グラフの例](/docs/images/cdy7uua7fh8z/y9guAxVBfFaaKIEGwCGRI/9b1a933e2da2d7a4cf168892b5fbdc35/sign_ups.png)</Frame>

<div id="failed-logins">
  ### ログイン失敗数
</div>

指定した期間内に発生したユーザーのログイン失敗 (`f` [ログイベントタイプ](/ja/docs/deploy-monitor/logs/log-event-type-codes)) の数です。

<Frame>![Dashboard Activity ページの「ログイン失敗数」折れ線グラフの例](/docs/images/cdy7uua7fh8z/35WccBqKhEd5z0uT4axrnZ/c964fdf97e1a33d298fa8906a77bef9c/failed_logins.png)</Frame>

<div id="compare-to-last-period">
  ## 前期間との比較
</div>

**前期間との比較**を有効にすると、次のようになります。

* グラフに差分の割合が表示されます。増加は緑色、減少は赤色で表示されます。
* 点線は前の期間、実線は現在の期間を示します。

<Frame>![Dashboard Activity ページの Active Users 折れ線グラフにマウスを重ねた例](/docs/images/cdy7uua7fh8z/3wbi2SCiWbI5JisOxMLD2Z/892abde733b368efd903d68d1d501c10/2025-02-26_14-01-31.png)</Frame>

<div id="view-data-by-time-frame">
  ## 期間を指定してデータを表示する
</div>

**前期間との比較** チェックボックスの横にある日付ピッカーを使用すると、指定した期間のデータを表示できます。直近 7/14/30/60 日のあらかじめ用意された期間を選択することも、カスタム範囲として **From** 日付と **To** 日付を指定することもできます。

<Frame>![Dashboard Activity ページの日付選択フォームの例](/docs/images/cdy7uua7fh8z/4ctRxO704Ufg3q5xn8riCi/e3b174f34a4b9e12f9321a975a10f6c5/date_picker.png)</Frame>
