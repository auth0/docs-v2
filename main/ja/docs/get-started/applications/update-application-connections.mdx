---
description: Auth0 Dashboard を使用して、アプリケーションで有効な接続を更新する方法を説明します。
title: アプリケーションの接続を更新
---

<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要プロダクト。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> を使用して、アプリケーションで有効な接続を変更できます。

1. [Dashboard &gt; Applications](https://manage.auth0.com/#/applications) に移動し、表示するアプリケーションの名前をクリックします。

   <Frame>![Dashboard のアプリケーション一覧](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. **接続** タブをクリックし、アプリケーションの該当する接続を有効または無効にします。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  [サードパーティアプリケーション](/ja/docs/get-started/applications/third-party-applications) では、ドメインレベルに昇格した接続、またはアプリケーションに明示的にマッピングされた接続を通じてユーザーを認証できます。詳しくは、[接続をドメインレベルに昇格](/ja/docs/authenticate/identity-providers/promote-connections-to-domain-level) をご覧ください。
</Callout>
