Auth0を利用するお客様には、共通する目標や要件があるのが一般的です。そこで、効率的にソリューションを提供できるよう、実際のお客様による実装経験に基づく知見を共有します。

<div id="application-architecture-scenarios">
  ## アプリケーションアーキテクチャのシナリオ
</div>

これらのシナリオでは、アプリケーションで採用できるさまざまな技術アーキテクチャの種類と、Auth0 がどのように役立つかを説明します。目的は、実装プロセスを最初から最後まで順を追って案内することです。

| 読む内容...                                                                                     | 学べる内容...                                                                             |
| ------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------ |
| [複数の組織](/ja/docs/get-started/architecture-scenarios/multiple-organization-architecture)        | 単一のIDプロバイダーを使用する組織、または複数のIDプロバイダーを使用する組織を利用するアプリケーションアーキテクチャを構成する方法。                 |
| [通常の Web アプリ向けシングルサインオン](/ja/docs/get-started/architecture-scenarios/sso-for-regular-web-apps) | Active Directory を使用する従業員と、別のユーザーストアを使用する契約業者向けに、SSO を使用する Web アプリケーションを構成および実装する方法。 |
| [サーバーアプリケーション + API](/ja/docs/get-started/architecture-scenarios/server-application-api)       | cron ジョブを使用してタイムシートのエントリを記録するタイムシート API を構成する方法。                                     |
| [シングルページアプリケーション + API](/ja/docs/get-started/architecture-scenarios/spa-api)                   | シングルページアプリケーションを使用してタイムシートのエントリを記録するタイムシート API を構成する方法。                              |
| [モバイルアプリケーション + API](/ja/docs/get-started/architecture-scenarios/mobile-api)                   | モバイルアプリケーションを使用して、一元化されたデータベースにタイムシートのエントリを記録するタイムシート API を構成する方法。                   |

<div id="implementation-checklists-and-resources">
  ## 実装チェックリストとリソース

| 読む項目                                                                        | 学べること                                                                                                        |
| --------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------ |
| [実装チェックリスト](/ja/docs/get-started/architecture-scenarios/checklists)            | SDLC プロジェクトの各フェーズに関連するタスクの計画方法。                                                                              |
| [実装リソース](/ja/docs/get-started/architecture-scenarios/implementation-resources) | 動画、用語集、ホワイトペーパー、ブログ記事、クイックスタート、API、SDK、Auth0 コミュニティフォーラム、Auth0 Support Center など、利用開始に役立つ Auth0 の各種リソースについて。 |
