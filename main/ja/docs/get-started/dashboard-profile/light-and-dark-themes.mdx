description: ダッシュボードでライトテーマとダークテーマを切り替える方法
title: ライトテーマとダークテーマ
---

Auth0 のお客様は、テナントのダッシュボードでライトモードまたはダークモードを使用できます。また、ローカルのシステム環境と同じ設定を使用するようにダッシュボードを設定することもできます。

ダッシュボードのテーマを変更する方法は 2 つあります。

1. ダッシュボード右上のプロフィールのドロップダウンメニューを選択します。
2. メニューからテーマを切り替えます。

同じドロップダウンで **Your Profile** をクリックし、下部までスクロールして、3 つのテーマから選択することもできます。
