description: Auth0 Dashboard 管理者がログインセッションを表示および取り消す方法について説明します。
title: Auth0 Dashboard ログインセッション管理
---

Login Sessions を使用すると、<Tooltip tip="Auth0 Dashboard: サービスを構成するための Auth0 の主要製品。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> 管理者は、複数のデバイスやブラウザーにまたがる各種 Dashboard アプリケーションのアクティブなセッションを追跡できます。Teams Dashboard、Support Center、Auth0 Dashboard にアクセスしたユーザーによって作成されたセッションを確認し、取り消すことができます。

Auth0 Dashboard の[プロフィールページ](https://manage.auth0.com/#/profile/sessions)では、現在アクティブなすべてのセッションの一覧を確認できます。

<Frame>![Auth0 Dashboard > プロフィール](/docs/images/cdy7uua7fh8z/3hBNRo2Bc4w6oKpF0mMt4o/3796b322593eae38eeba0a3af3698eca/English_user_profle_login_sessions.png)</Frame>

ログアウトする、またはセッションを取り消すには、対象のセッションを見つけて **Revoke** を選択します。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  セッションを取り消しても、拡張機能に属するセッションは自動的には削除されません。ただし、拡張機能のセッションは有効期間が短く、ログインセッションが取り消された場合や期限切れになった場合に更新されることはありません。
</Callout>
