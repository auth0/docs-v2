<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Guide は現在 EA で、US Public Cloud リージョンのユーザーに順次提供されています。Guide が不完全または不正確な回答を返す場合があるため、提供されるリンクを使用して必ず[確認と検証](#how-does-guide-avoid-ai-hallucinations%3F)を行ってください。Auth0 のリリースの詳細については、[Product Release Stages](/ja/docs/troubleshoot/product-lifecycle/product-release-stages)を参照してください。
</Callout>

Guide は、Auth0 に関する質問に回答する AI 搭載チャットボットです。情報ソースとして、[Documentation](https://auth0.com/docs)、[Blog](https://auth0.com/blog)、[Community](http://community.auth0.com) を利用します。Enterprise サブスクリプションのお客様向けには、[Security Center](/ja/docs/secure/security-center) の Auth0 テナントデータも活用し、テナントのセキュリティに関するベストプラクティスの監視、トラブルシューティング、実装を支援します。

Guide は、「パスキーを設定するにはどうすればよいですか？」のような質問に対して、要約した回答と、関連ドキュメントおよびテナントとユーザーのセキュリティに関する[インシデントプレイブック](/ja/docs/secure/attack-protection/playbooks) へのリンクを返します。

<div id="how-guide-works">
  ## Guide の仕組み
</div>

Guide はメッセージを処理して、何を求めているかを判断し、関連情報を Auth0 のソースから検索したうえで、参照元資料へのリンクとともに、見つかった内容を要約した回答を生成します。

追加の質問をしたり、提供されたリンクを使って [回答を検証](#how-does-guide-avoid-ai-hallucinations%3F) したり、回答を評価して Guide にフィードバックを送ったりできます。

Enterprise サブスクリプションをご利用のお客様向け: Security Center で [セキュリティ監視アラートを設定](/ja/docs/secure/security-center/security-alerts) している場合、Guide はインシデントのトラブルシューティングを支援し、関連する [インシデントプレイブック](/ja/docs/secure/attack-protection/playbooks) からセキュリティガイダンスを提案します。

たとえば、次のスクリーンショットに示すように、大規模なボットの急増によって <Tooltip tip="Bot Detection: Auth0 がログイン処理中にキャプチャを有効にすることで、ボットの疑いがあるトラフィックをブロックする攻撃対策の一形態です。" cta="用語集を表示" href="/ja/docs/glossary?term=Bot+Detection">Bot Detection</Tooltip> アラートが発生すると、Guide は影響を受ける認証フローでキャプチャを有効にするなど、[Bot Detection Attack プレイブック](/ja/docs/secure/attack-protection/playbooks/bot-detection-playbook) の対処手順を推奨し、テナントログ内の関連イベントを確認するよう促します。

<div id="use-guide">
  ## Guide を使用する
</div>

<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> の右上にある **Ask Guide** ボタンを選択すると、Guide を開けます。

**Ask Guide** を選択すると Guide がポップアップ表示され、Auth0 の機能について質問するよう求められます。Auth0 Dashboard で現在開いている場所に応じて、関連する質問が提案されることもあります。たとえば、**ブランディング &gt; <Tooltip tip="Universal Login: アプリケーションは、ユーザーの本人確認のために、Auth0 の認可サーバーでホストされる Universal Login にユーザーをリダイレクトします。" cta="用語集を表示" href="/ja/docs/glossary?term=Universal+Login">Universal Login</Tooltip>** を開いている場合、Guide は Universal Login に関する次のような質問を提示します。

* Universal Login とは何ですか？
* Universal Login の利点は何ですか？

<Frame>![Auth0 Dashboard の右上にある Ask Guide ボタンを選択して開きます。](/docs/images/cdy7uua7fh8z/e3lTjWgGC16Ob6zeUfU1V/ea8eef92439697bced3c147c9ca87d87/image3.png)</Frame>

<div id="questions-guide-can-answer">
  ### Guide で回答できる質問
</div>

Guide は、次のような Auth0 の製品やプラットフォームに関する質問に回答できます。

* Universal Login とは何ですか？
* <Tooltip tip="シングルサインオン（SSO）: ユーザーが 1 つのアプリケーションにログインすると、他のアプリケーションにも自動的にログインできるようにするサービスです。" cta="用語集を表示" href="/ja/docs/glossary?term=single+sign-on">シングルサインオン</Tooltip>統合を有効にするにはどうすればよいですか？
* パスキーを設定するにはどうすればよいですか？

<Frame>![Guide は Auth0 の製品やプラットフォームに関する質問に回答できます。](/docs/images/cdy7uua7fh8z/nGVAVWxnoqiXaFXTQQyHX/101b6fdfde5f8488d6d2401e5dee5a42/auth0-guide-sso-screenshot.png)</Frame>

Enterprise サブスクリプションをご利用のお客様の場合、Guide は Security Center のメトリクスやデータに関する Auth0 テナント関連の質問にも回答できます。たとえば、次のような質問です。

* この失敗した <Tooltip tip="多要素認証（MFA）: SMS で送信されるコードなど、ユーザー名とパスワードに加えて別の要素を使用するユーザー認証プロセスです。" cta="用語集を表示" href="/ja/docs/glossary?term=MFA">MFA</Tooltip> チャレンジをトラブルシューティングするにはどうすればよいですか？
* ボット保護アラートのしきい値がトリガーされた原因は何ですか？
* 「credential stuffing」アラートの修復手順は何ですか？

<div id="questions-guide-cannot-answer">
  ### Guide で回答できない質問
</div>

Guide は、Auth0 の製品やプラットフォームに関係のない質問や、テナントや設定に固有の質問には回答できません。たとえば、次のようなものです。

* 有効になっている接続は何ですか？
* 先月、私のアプリケーションにログインしたユーザーは何人ですか？
* Vim と Emacs では、どちらが優れていますか？

<Warning>
  個人情報やその他の機密データを Guide に送信しないでください。
</Warning>

<div id="disable-guide">
  ## Guide を無効にする
</div>

テナントで Guide を無効にするには、[Auth0 Support](https://support.auth0.com/) にお問い合わせください。

<div id="data-security-in-guide">
  ## Guide におけるデータセキュリティ
</div>

お客様のメッセージは、Okta のエンタープライズレベルのセキュリティおよびコンプライアンス基準によって保護されています。

Guide が行うこと:

* 提供されたメッセージを使用して、公開されている Auth0 のソースから関連情報を検索します。

Guide が行わないこと:

* お客様のアカウントやテナントのデータにアクセスすること。
* 提供されたメッセージを LLM のトレーニング目的に使用すること。
* お客様のメッセージを第三者と共有すること。

<div id="how-does-guide-avoid-ai-hallucinations">
  ## Guide はどのように AI のハルシネーションを防いでいますか？
</div>

Guide は Auth0 が作成したコンテンツに基づいて回答を生成し、すべての回答に参照元ドキュメントへのリンクを含めています。参照されたリンクをたどることで、元のソースデータを確認し、さらに詳しい情報を得ることができます。
