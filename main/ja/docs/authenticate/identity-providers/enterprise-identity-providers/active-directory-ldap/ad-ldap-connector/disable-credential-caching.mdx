---
description: Auth0 Management Dashboard を使用して、AD/LDAP エンタープライズ接続の接続レベルで認証情報のキャッシュを無効にする方法を説明します。
title: AD/LDAP 接続の認証情報キャッシュを無効にする
---

<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品です。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> を使用すると、[AD/LDAP 接続](/ja/docs/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector)の接続レベルで認証情報のキャッシュを無効にできます。

1. [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise) に移動し、**Active Directory/LDAP** 接続タイプを選択します。
2. 編集する接続名を選択します。
3. **Disable Cache** トグルをオンにします。
