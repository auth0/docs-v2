---
description: Connector Admin Console から AD/LDAP Connector の設定をインポートまたはエクスポートする方法を説明します。
title: AD/LDAP Connector 設定のインポートとエクスポート
---

AD/LDAP Connector の設定は、エクスポートすることも、以前にエクスポートした設定をインポートすることもできます。これは、高可用性のために AD/LDAP Connector を複数ノードでデプロイしている環境で役立ちます。1 つのノードで設定とテストを行った後、正常に動作する設定をエクスポートし、それ以降のすべてのノードにインポートできます。

1. ブラウザーで `http://localhost:8357` にアクセスして、AD/LDAP Connector サーバー上の **Connector Admin Console** を起動します。
2. **Import / Export** タブをクリックします。
3. `%Program Files(x86)%\Auth0\AD LDAP Connector\` フォルダー内の `config.json` ファイル、`certs` フォルダー、および `lib\\profileMapper.js` ファイルを含む .zip ファイルをエクスポートします。
4. インポートするには、**Import / Export** タブでバックアップ用の .zip ファイルをアップロードします。
