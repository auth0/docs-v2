---
description: Auth0 Dashboard を使用してアプリケーションのエンタープライズ接続を有効にする方法について説明します。
title: エンタープライズ接続を有効にする
---

<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 のメイン製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> を使用すると、アプリケーションのエンタープライズ接続を有効にできます。エンタープライズ接続を有効にするには、あらかじめ設定を完了しておく必要があります。

1. [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise) に移動し、表示する接続タイプを選択します。

   <Frame>![ダッシュボード - 接続 - エンタープライズ](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. 表示する接続の名前を選択します。

   <Frame>![ダッシュボード - 接続 - エンタープライズ - AD/LDAP - 一覧](/docs/images/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/4c0bd4a009df0b7f40a645613d3c723f/LDAP_page_-_EN.png)</Frame>
3. **Applications** ビューを選択し、該当するアプリケーションの接続を有効または無効にします。

   <Frame>![ダッシュボード - 接続 - エンタープライズ - アプリケーション](/docs/images/cdy7uua7fh8z/2Jwoyem76rmja1sveD88It/38fbae514304297bbbf72beeedb8892f/SSL_-_EN.png)</Frame>

エンタープライズ <Tooltip tip="Identity Provider (IdP): デジタル ID を保存および管理するサービスです。" cta="用語集を見る" href="/ja/docs/glossary?term=IdP">IdP</Tooltip> 接続を設定する際は、Auth0 の `/authorize,` `/whr,` または `/wsfed` エンドポイントに渡された `connection` パラメーターにより、データベースまたは Active Directory 経由で接続されている場合を**除き**、上流の IdP にリダイレクトされることに注意してください。
