description: 接続 ID の確認方法について説明します。
title: 接続 ID または名前を確認する
---

一部のユーザー管理タスクでは、接続 ID が必要です。接続 ID は、ユーザーのソースを一意に識別する識別子です。たとえば、ユーザーをインポートまたはエクスポートする場合や、<Tooltip tip="IDプロバイダー（IdP）: デジタルIDを保存して管理するサービス。" cta="用語集を表示" href="/ja/docs/glossary?term=identity+providers">IDプロバイダー</Tooltip>に接続する場合は、接続 ID を指定する必要があります。

1. [Auth0 Dashboard](https://manage.auth0.com/#/) に移動し、**Authentication** を選択してから、ID が必要な接続の種類を選択します。
2. ID が必要な接続の名前を選択します。
3. **Settings** ビューで **Name** フィールドを確認します。このフィールドに接続 ID が表示されます。
