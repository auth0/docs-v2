title: データベース接続に username を追加する
---

データベース接続では、ユーザーがメールアドレスではなく username を使用してサインインできるようにできます。

<div id="require-username">
  ## username を必須にする
</div>

1. [Auth0 Dashboard &gt; Authentication &gt; Database](https://manage.auth0.com/#/connections/database) に移動し、編集する接続の名前を選択します。
2. **Requires Username** を見つけて、トグルで username の必須設定を有効または無効にします。

   <Frame>![username 必須設定の有効化または無効化](/docs/images/cdy7uua7fh8z/1MlOrFmcSCOCRYKRxWz6xT/7a87dac16a31cbb6e608489c5843d6c7/2025-02-25_12-02-23.png)</Frame>

これがログイン画面にどのように反映されるかを確認するには、**Try Connection** ビューを選択します。**Requires Username** を有効にすると、新規ユーザーはサインアップ時に username とメールアドレスの両方を入力する必要があります。

<Frame>![username を含むログインフォーム](/docs/images/cdy7uua7fh8z/AiB37UZqyBPb8h55PZtzm/73e6fd0033f63f865c40f189db1620ef/2025-01-29_15-22-11.png)</Frame>

その後、ユーザーは username またはメールアドレスのいずれかでログインできます。**Requires Username** が無効のときに登録したユーザーには Username フィールドが保存されていないため、メールアドレスでログインする必要があります。

<div id="username-length">
  ### username の文字数
</div>

username に許可されるデフォルトの文字数は 1～15 文字です。ただし、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要な製品。" cta="用語集を表示" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> または <Tooltip tip="Management API: お客様が管理タスクを実行できるようにする製品。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip> v2 を使用すると、最小文字数と最大文字数を変更できます (最大 128 文字まで) 。

<div id="allowed-characters">
  ### 使用できる文字
</div>

username フィールドでは、次の文字を使用できます。

* 英数字 (アクセント記号は使用不可。自動的に小文字に変換されます) ;
* アットマーク (@) ;
* キャレット (^) ;
* ドル記号 ($) ;
* ピリオド (.) ;
* 感嘆符 (!) ;
* グレーブアクセント (`) ;
* ハイフン (-) ;
* 番号記号 (#) ;
* プラス記号 (+) ;
* シングルクォート (&#39;) ;
* チルダ (~) ;
* アンダースコア (&#95;) ;

ただし、メールアドレスは使用できません。これら以外の文字や記号は使用できず、Auth0 はカスタムデータベース入力の検証やサニタイズを行いません。
