title: Auth0 でのログイン認証
sidebarTitle: ログイン
description: Auth0 Universal Login または埋め込みログインを使用して、ユーザーのログイン認証を実装する方法を説明します。
---

Auth0 では、アプリケーションにログイン認証を実装する方法を 2 つ提供しています。

* **<Tooltip tip="Universal Login: ユーザーの本人確認のために、アプリケーションは Auth0 の認可サーバーでホストされる Universal Login にユーザーをリダイレクトします。" cta="用語集を表示" href="/ja/docs/glossary?term=Universal+Login">Universal Login</Tooltip>** は、Auth0 がホストするページを通じてユーザーがアプリケーションにログインする方法です。

* **埋め込みログイン** は、ユーザーがご自身でホストするページを通じてアプリケーションにログインする方法です。

大半のユースケースでは Universal Login を推奨しますが、分離型の認証フローでユーザーを認証するために、クライアント起点バックチャネル認証を使用することもできます。
