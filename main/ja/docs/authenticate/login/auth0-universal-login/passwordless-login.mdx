<Card title="利用可否は Auth0 プランによって異なります">この機能を利用できるかどうかは、Auth0 のプランまたはカスタム契約の内容によって異なります。詳しくは、[Pricing](https://auth0.com/pricing) を参照してください。</Card>

<Tooltip tip="Universal Login: アプリケーションは、ユーザーの本人確認のために、Auth0 の認可サーバーでホストされる Universal Login にリダイレクトされます。" cta="用語集を見る" href="/ja/docs/glossary?term=Universal+Login">
  Universal Login
</Tooltip>

は Auth0 による認証フローの実装であり、

<Tooltip tip="Authorization Server: ユーザーのアクセス範囲を定義する一元化されたサーバー。たとえば、認可サーバーでは、ユーザーが利用できるデータ、タスク、機能を制御できます。" cta="用語集を見る" href="/ja/docs/glossary?term=Authorization+Server">
  認可サーバー
</Tooltip>

の中核となる機能です。ユーザーが本人確認を行う必要があるたびに、アプリケーションは Universal Login にリダイレクトされ、Auth0 がそのユーザーの本人確認を行います。

<Tooltip tip="Passwordless: 第 1 要素としてパスワードに依存しない認証方式。" cta="用語集を見る" href="/ja/docs/glossary?term=Passwordless">
  パスワードレス
</Tooltip>

認証は、

<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要製品。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">
  Auth0 Dashboard
</Tooltip>

で Universal Login を設定する際に有効化できます。

まず、パスワードレス認証の方式として、SMS、メールアドレス、またはデバイスの生体認証に WebAuthn を使用するかを決定してください。現時点では、1 つのログインフローでこれら 3 つすべての認証方式をサポートすることはできません。

<div id="passwordless-with-email-or-sms">
  ## メールアドレスまたは SMS によるパスワードレス
</div>

Auth0 を設定すると、ユーザーはメールアドレスに送信されたマジックリンクまたはワンタイムパスワード (OTP) 、あるいは電話番号に送信された OTP を使用して認証できます。詳細については、[パスワードレス認証用のメールアドレスまたは SMS の設定](/ja/docs/authenticate/login/auth0-universal-login/passwordless-login/email-or-sms) を参照してください。

<div id="passwordless-with-webauthn-with-device-biometrics">
  ## デバイスの生体認証を使用する WebAuthn によるパスワードレス認証
</div>

デバイスの生体認証に対応した WebAuthn を使用すると、ユーザーはパスワード、または WebAuthn とデバイスの生体認証に対応した任意のデバイスで認証できます。

新しいデバイスからログインするたびに、Auth0 はパスワードの代わりに生体認証を使用するオプションを提示します。詳細については、[デバイスの生体認証を使用する WebAuthn によるパスワードレス認証](/ja/docs/authenticate/login/auth0-universal-login/passwordless-login/webauthn-device-biometrics) を参照してください。
