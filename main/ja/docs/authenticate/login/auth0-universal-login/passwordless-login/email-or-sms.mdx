title: パスワードレス認証用にメールまたはSMSを設定する
---

Auth0 では、<Tooltip tip="Universal Login: アプリケーションは Auth0 の認可サーバーでホストされる Universal Login にリダイレクトされ、ユーザーの本人確認が行われます。" cta="用語集を表示" href="/ja/docs/glossary?term=Universal+Login">Universal Login</Tooltip> を設定することで、ユーザーがメールでマジックリンクまたはワンタイムパスワード (OTP) 、あるいはSMSで OTP を使用して認証できるようになります。

SMS またはメールによる <Tooltip tip="Passwordless: 第 1 要素としてパスワードに依存しない認証形式です。" cta="用語集を表示" href="/ja/docs/glossary?term=Passwordless">パスワードレス</Tooltip> 認証を有効にするには、アプリケーション用の [パスワードレス接続](/ja/docs/authenticate/passwordless) を定義する必要があります。

接続を作成したら、それを使用するよう Universal Login を設定する必要があります。方法は 3 つあります。

<div id="universal-login">
  ## Universal Login
</div>

この方法では、パスワードレス接続をネイティブでサポートする Universal Login エクスペリエンスを使用します。詳しくは、[Universal Login を使用したパスワードレス](/ja/docs/authenticate/passwordless/passwordless-with-universal-login) を参照してください。

<div id="universal-login-lock-passwordless">
  ## Universal Login + Lock (パスワードレス)
</div>

この方法では、**Lock (パスワードレス) **テンプレートを使用したカスタムログインページを使い、[Lock for Web SDK](/ja/docs/libraries/lock) でユーザーを認証します。

1. [Dashboard &gt; Branding &gt; Universal Login](https://manage.auth0.com/#/login_settings) に移動し、**Login** タブをクリックします。
2. **Custom Login Page** トグルを有効にし、**Lock (パスワードレス) **テンプレートを選択します。HTML テンプレートが更新され、パスワードレスのカスタマイズオプションを含む Lock ウィジェットのコードが表示されます。
3. テンプレートをカスタマイズし、**Save Changes** をクリックします。

HTML と CSS を使用してログインフォームをカスタマイズし、変更内容は Dashboard 内でプレビューできます。**Lock (パスワードレス) **テンプレートのカスタマイズ方法について詳しくは、[Lock for Web SDK](/ja/docs/libraries/lock) の Passwordless セクションを参照してください。

<div id="universal-login-custom-ui-auth0js">
  ## Universal Login + カスタム UI + Auth0.js
</div>

この方法では、**Custom Login Form** テンプレートを使用するカスタムログインページを使って、[Auth0.js SDK](/ja/docs/libraries/auth0js) でユーザーを認証します。詳細については、[Universal Login によるパスワードレス認証](/ja/docs/authenticate/passwordless/implement-login/universal-login)を参照してください。
