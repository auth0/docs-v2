title: 埋め込みログイン
---

埋め込みログインを使用すると、ユーザーはアプリケーションに直接ログインし、認証のために認証情報を Auth0 サーバーに送信できます。Auth0 では、埋め込みログインの使用を推奨していません。詳しくは、[Centralized Universal Login vs. Embedded Login](/ja/docs/authenticate/login/universal-vs-embedded-login) を参照してください。

埋め込みログインを使用する場合は、[Cross-Origin Resource Sharing 用にアプリケーションを設定](/ja/docs/get-started/applications/set-up-cors)する必要があります。また、[カスタムドメイン](/ja/docs/customize/custom-domains)を設定することを推奨します。その後、アプリケーションに [Lock SDK](/ja/docs/libraries/lock) または [Auth0.js SDK](/ja/docs/libraries/auth0js) を実装するか、[Auth0 Authentication API](https://auth0.com/docs/api/authentication) を直接呼び出すことができます。

Auth0 は、複数のアプリケーションタイプで、<Tooltip tip="パスワードレス: 第1要素としてパスワードに依存しない認証形式です。" cta="用語集を表示" href="/ja/docs/glossary?term=Passwordless">パスワードレス</Tooltip>接続による埋め込みログインをサポートしています。
