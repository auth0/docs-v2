---
title: ID プロバイダー
description: ID プロバイダー、データベース、パスワードレス認証方式など、アプリケーションのユーザーソースについての概要。
---

接続とは、Auth0 とユーザーソースとの関係を指します。ユーザーソースには、外部 ID プロバイダー (Google や LinkedIn など) 、データベース、パスワードレス認証方式などがあります。Auth0 はアプリケーションとユーザーソースの間に位置し、抽象化レイヤーを提供することで、各ソースの実装変更や固有の違いの影響をアプリケーションが受けないようにします。

デフォルトでは、Auth0 はユーザーがログインするたびにユーザープロファイルデータを自動的に同期するため、接続元で行われた変更は Auth0 にも自動的に反映されます。同期を無効にすると、アプリケーション側からプロファイル属性を更新できるようになります。

アプリケーションには、任意の数の接続を設定できます。
