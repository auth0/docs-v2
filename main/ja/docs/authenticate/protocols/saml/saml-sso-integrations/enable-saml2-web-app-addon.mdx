description: Auth0 を SAML シングルサインオン（SSO）の IDプロバイダーとして使用するために、SAML2 Web App アドオンを有効にする方法を説明します。
title: SAML2 Web App アドオン を有効にする
---

アプリケーションで使用する SAML2 Web Application アドオンを設定できます。

1. [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) に移動します。
2. SAML2 Web Application を使用するアプリケーションを見つけ、アプリケーション名をクリックして設定画面を開きます。
3. **Addons** タブに移動します。
4. **SAML2 Web App** ボックスで、スライダーをクリックしてアドオンを有効にします。

   <Frame>![Dashboard Applications Applications Addons Tab SAML2 Web App Settings Tab](/docs/images/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/fe9dbbf306e6c587cb3326c00a3b4e1f/2025-02-27_13-59-00.png)</Frame>
5. **Settings** タブに移動し、SAML レスポンスを受信する **Application Callback URL** と **Settings** を指定します。**Settings** タブには、どの設定を有効にするかを判断する際に使用できるサンプルオブジェクトが表示されます。
6. 指定したパラメーターが有効かどうかをテストするには、**Debug** をクリックします。
7. 完了したら **Save** をクリックします。

<div id="provide-application-details">
  ## アプリケーションの詳細を入力する
</div>

SAML2 Web App にアプリケーションの詳細を入力する具体的な手順は環境によって異なりますが、必要な設定情報は、アドオンを設定した後に **Usage** タブで確認できます。

  ## 制限事項
</div>

SAML2 Web App アドオンは、パッシブ <Tooltip tip="Security Assertion Markup Language（SAML）: 2 者間でパスワードを使わずに認証情報を交換するための標準化されたプロトコル。" cta="用語集を表示" href="/ja/docs/glossary?term=SAML">SAML</Tooltip> リクエスト (例: `isPassive=true` の場合) をサポートしていません。
