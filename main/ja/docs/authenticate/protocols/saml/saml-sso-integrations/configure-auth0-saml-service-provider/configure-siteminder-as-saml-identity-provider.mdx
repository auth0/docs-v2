以下の手順に従って、SiteMinder を <Tooltip tip="Security Assertion Markup Language（SAML）: パスワードを使わずに 2 者間で認証情報を交換できるようにする標準化されたプロトコル。" cta="用語集を表示" href="/ja/docs/glossary?term=SAML">SAML</Tooltip> <Tooltip tip="Security Assertion Markup Language（SAML）: パスワードを使わずに 2 者間で認証情報を交換できるようにする標準化されたプロトコル。" cta="用語集を表示" href="/ja/docs/glossary?term=identity+provider">IDプロバイダー</Tooltip>として設定します。

<div id="prerequisite">
  ## 前提条件
</div>

[SiteMinder](https://www.siteminder.com/login/)のアカウントが必要です。

<div id="configure-siteminder-settings">
  ## SiteMinder の設定
</div>

SiteMinder を設定する際は、ほとんどのオプションで既定値を使用します。また、以下の設定手順では、次の Auth0 関連の値が必要です。

* **EntityID:** `urn:auth0:{yourTenant}`
* **Assertion Consumer Service URL:** `https://{yourDomain}/login/callback`
* **Logout URL:** `https://{yourDomain}/logout`
* SAML Request には **HTTP-Redirect** バインディング
* SAML Response には **HTTP-POST** バインディング

1. [SiteMinder](https://www.siteminder.com/login/) にログインします。
2. **SAML Service Provider Dialog** を開きます。
3. このサービスプロバイダーに適切な名前を付けます。
4. `NameIdentifier` を定義します。SiteMinder で認証されるユーザーに対してこれを生成する方法はいくつもあります。通常は、この値を User Directory 内のユーザープロパティのいずれかに `uid` としてマッピングします。
5. サービスプロバイダーの一般的な SAML プロパティを設定します。

| 設定               | 説明                       |
| ---------------- | ------------------------ |
| **SP ID**        | `urn:auth0:{yourTenant}` |
| **SAML Version** | 2.0                      |
| **Skew Time:**   | 30 秒                     |

6. **Assertion Consumer Service URL** を設定します。これは、SiteMinder が SAML トークンを POST で返送する先です。この Service Provider ({yourTenant}) は、SAML Response では `HTTP-POST` バインディングのみをサポートします。次を使用します。
   `https://{yourDomain}/login/callback`

7. **HTTP-Post** ボックスをオンにします。

8. トークンで送信する追加のユーザープロパティを設定します。認証済みユーザーについてこの Service Provider と共有したいその他のプロパティを追加します。一般的な値は、`name`、`lastname`、`email address` などです。この Service Provider は、先ほど定義した `NameIdentifier` をユーザーの一意の識別子として使用します。これらの属性は参照情報として扱われます。

9. **SLO Location URL:**`https://{yourDomain}/logout` を入力します。

10. (任意) このサービスプロバイダーはアサーションの暗号化をサポートしています。
    このオプションを使用するには、次の手順でサービスプロバイダーの公開鍵証明書をダウンロードします。

11. Auth0 Dashboard で **Connections** をクリックし、次に **Enterprise.** をクリックします。

12. SAMLP IDプロバイダー をクリックします。

13. セットアップアイコン (鉛筆) をクリックします。

14. 表示されたウィンドウで、7 番目 (最後) の箇条書きに .pem または .cer 形式の証明書をダウンロードするリンクがあります。

15. 必要な証明書をダウンロードし、SiteMinder の **Policy Server Keystore** に追加します。
