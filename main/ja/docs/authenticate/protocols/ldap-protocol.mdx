description: Auth0 で Lightweight Directory Access Protocol（LDAP）がどのようにサポートされるかを説明します。
title: Lightweight Directory Access Protocol
---

Lightweight Directory Access Protocol (LDAP) は、Internet Protocol (IP) ネットワーク上で分散ディレクトリ情報サービスにアクセスし、管理するためのアプリケーションプロトコルです。LDAP の役割は、Active Directory (AD) などの既存のディレクトリにアクセスできるようにすることです。Auth0 では、AD/LDAP と統合するためにネットワーク上にインストールする [Active Directory/LDAP Connector](/ja/docs/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector) を使用します。
