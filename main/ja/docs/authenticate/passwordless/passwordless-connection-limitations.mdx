使用する認証方式は、アプリケーションで設定した<Tooltip tip="Universal Login: アプリケーションは、ユーザーの本人確認を行うために、Auth0 の認可サーバーでホストされる Universal Login にリダイレクトされます。" cta="用語集を見る" href="/ja/docs/glossary?term=Universal+Login">Universal Login</Tooltip>のバージョンによって異なります。Universal Login を使用している場合は、[デバイスの生体認証を使用する WebAuthn によるパスワードレス認証](/ja/docs/authenticate/login/auth0-universal-login/passwordless-login/webauthn-device-biometrics)を設定するか、以下で説明する<Tooltip tip="Universal Login: アプリケーションは、ユーザーの本人確認を行うために、Auth0 の認可サーバーでホストされる Universal Login にリダイレクトされます。" cta="用語集を見る" href="/ja/docs/glossary?term=passwordless">パスワードレス</Tooltip>接続を使用できます。

次の方法で、メールアドレスまたは SMS を使用したパスワードレス認証を設定できます。

1. Universal Login + パスワードレス接続パラメータ
2. Classic Login + Lock
3. Classic Login + カスタム UI + Auth0.js

これらの実装について詳しくは、[パスワードレス認証用のメールアドレスまたは SMS を設定する](/ja/docs/authenticate/login/auth0-universal-login/passwordless-login/email-or-sms)を参照してください。

<div id="errors">
  ### エラー
</div>

埋め込みログインを使用している場合は、<Tooltip tip="カスタムドメイン: 専用の、または独自の名前を持つサードパーティのドメイン。" cta="用語集を表示" href="/ja/docs/glossary?term=custom+domains">カスタムドメイン</Tooltip>を使用することを推奨します。詳しくは、[カスタムドメイン](/ja/docs/customize/custom-domains)を参照してください。

<div id="considerations">
  ## 注意事項
</div>

* パスワードレスユーザーは、<Tooltip tip="Auth0 Dashboard: サービスを設定するための Auth0 の主要な製品です。" cta="用語集を見る" href="/ja/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> からは作成できません。直接 [Management API](https://auth0.com/docs/api/management/v2#!/Users/post_users) で作成してください。
* メールアドレスのパスワードレスマジックリンクを使用する場合、<Tooltip tip="Refresh Token: ユーザーに再度ログインを求めることなく、新しいアクセストークンを取得するために使用するトークンです。" cta="用語集を見る" href="/ja/docs/glossary?term=Refresh+tokens">リフレッシュトークン</Tooltip> は取得できず、OTP のみ取得できます。
