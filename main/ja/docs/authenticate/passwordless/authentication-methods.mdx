<Tooltip tip="パスワードレス: 第1要素としてパスワードに依存しない認証方式です。" cta="用語集を見る" href="/ja/docs/glossary?term=Passwordless">パスワードレス</Tooltip> 接続では、ユーザーはパスワードなしでログインできます。代わりに、さまざまな認証方式を利用できます。Auth0 のパスワードレス接続では、次の認証方式をサポートしています。

| 要素                                                                                 | 説明                                                                                                                                  |
| ---------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------- |
| [メール](/ja/docs/authenticate/passwordless/authentication-methods/email-otp)            | ユーザーはメールアドレスの入力を求められ、そのメールアドレス宛てに Auth0 がワンタイムコードを送信します。ユーザーはその code をアプリケーションに入力します。                                               |
| [マジックリンク](/ja/docs/authenticate/passwordless/authentication-methods/email-magic-link) | ユーザーはメールアドレスの入力を求められ、そのメールアドレス宛てに Auth0 がリンク付きのメールを送信します。ユーザーがそのリンクをクリックすると、アプリケーションに直接ログインします。                                     |
| [SMS](/ja/docs/authenticate/passwordless/authentication-methods/sms-otp)              | ユーザーは電話番号の入力を求められ、その電話番号宛てに Auth0 がワンタイムコードを送信します。Auth0 はデフォルトでコードの送信に Twilio を使用しますが、カスタム SMS ゲートウェイがある場合は、代わりにそれを使用するよう接続を変更できます。 |
