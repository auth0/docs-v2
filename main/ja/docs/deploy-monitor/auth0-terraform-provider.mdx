title: Auth0 Terraform Provider
---

Auth0 テナント構成の管理に利用できるツールは Deploy CLI だけではありません。[公式サポート対象の Terraform Provider](https://github.com/auth0/terraform-provider-auth0) もあります。[Terraform](https://terraform.io/) は、クラウドリソースの構成をコードとして表現するためのサードパーティ製ツールです。確立されたプラグインフレームワークを備えており、Auth0 を含む幅広いクラウドプロバイダーに対応しています。

Deploy CLI と Terraform Provider はどちらも Auth0 テナント構成の管理に役立ちますが、それぞれに長所と短所があります。

次のような場合は、Auth0 Terraform Provider の利用を検討してください。

* 開発ワークフローですでに Terraform を使用している
* テナント管理の要件が細かい、または少数の特定のリソースのみに関係している

次のような場合は、Auth0 Terraform Provider は適していない可能性があります。

* 開発ワークフローで Terraform を使用しておらず、追加の初期設定が必要になる
* 開発ワークフローの主な目的がテナントの一括管理である
* テナントに既存のリソースが多数あり、&quot;import&quot; にかなりの労力が必要になる可能性がある
