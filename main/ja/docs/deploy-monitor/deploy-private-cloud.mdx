Private Cloud パッケージは、次のような場合に利用できるマネージドサービスです。

* 組織でマルチテナントの Public Cloud サービスを利用できない
* Public Cloud オプションでは対応できない地理的なデプロイ要件がある
* 保証された 1 秒あたりのリクエスト数 (RPS) や、Public Cloud で利用可能な容量を上回る処理能力が必要である

Auth0 プライベートクラウドは、次の方法でデプロイできます。

| オプション                                                                                          | 説明                                                                      |
| ---------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------- |
| [**AWS 上の Private Cloud**](/ja/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | AWS 上で稼働する専用のマネージドクラウドサービスで、分離性、高パフォーマンス、開発用インスタンス、Geo-HA の追加機能などを提供します |
| [**Azure 上の Private Cloud**](/ja/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Azure 上で稼働する専用のクラウドサービスで、分離性、高パフォーマンス、開発用インスタンス、追加機能などを提供します            |

また、Auth0 Private Cloud のデプロイを補完する[追加機能](/ja/docs/deploy-monitor/deploy-private-cloud/private-cloud-add-on-features)も提供しています。

特定のサポート要件がある場合や、提供している Professional Services の詳細については、[Auth0 Sales にお問い合わせください](https://auth0.com/contact-us)。
