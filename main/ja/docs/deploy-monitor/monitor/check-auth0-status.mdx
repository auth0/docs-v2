[Auth0 Public Cloud Status](https://status.auth0.com) ページにアクセスして、Auth0 の Public Cloud 版のサービス可用性を確認します。

ドロップダウンメニューから **Region** を選択すると、Core Services (ユーザー認証やマシン間認証など) と Supporting Services (<Tooltip tip="多要素認証（MFA）: SMS 経由のコードなど、ユーザー名とパスワードに加えて別の認証要素を使用するユーザー認証プロセス。" cta="用語集を表示" href="/ja/docs/glossary?term=MFA">MFA</Tooltip>、<Tooltip tip="Management API: 顧客が管理タスクを実行するための製品。" cta="用語集を表示" href="/ja/docs/glossary?term=Management+API">Management API</Tooltip>、テナントログ、CDN など) のステータスを確認できます。

利用可能なリージョンの詳細については、[Create Tenants](/ja/docs/get-started/auth0-overview/create-tenants) の [Region, locality, and sub-locality](/ja/docs/get-started/auth0-overview/create-tenants#region-locality-and-sub-locality) 表を参照してください.‬

<div id="subscribe-to-public-cloud-status-updates">
  ## Public Cloud のステータス更新を購読する
</div>

更新を受け取るには、**Region** を選択してから **Subscribe** を選択します。ステータスを確認する方法は 2 つあります。

1. 最新のステータス更新を受け取るには、X で [@auth0status](https://x.com/auth0status) をフォローします。
2. テナントに影響するステータス更新を受け取るには、Auth0 の Atom フィードまたは RSS フィードを購読します。`{YOUR_TENANT_DOMAIN}` はテナントの **ドメイン** に置き換えてください。

テナントの **ドメイン** を確認するには、[Auth0 Dashboard &gt; Applications](https://manage.auth0.com/dashboard/#/applications) を開き、アプリケーションを選択してから **Settings** を開きます。テナントの **ドメイン** は **Basic Information** に表示されます。

<div id="check-auth0-status-for-private-cloud">
  ## Private Cloud 向けの Auth0 ステータスを確認する
</div>

[Auth0 Public Cloud Status](https://status.auth0.com) にアクセスし、**Private Cloud Login** を選択します。

認証が完了すると、Auth0 Converged Platform で利用可能な任意の非開発環境 (例: `travel0-prod-1`) のステータスを確認できます。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Private Cloud 環境のステータスにアクセスするには、関連する Public Cloud サポートテナントのメンバーである必要があります。詳細については、[Open and Manage Support Tickets](/ja/docs/troubleshoot/customer-support/open-and-manage-support-tickets) および [Manage Dashboard Access](/ja/docs/get-started/manage-dashboard-access) を参照してください。
</Callout>

Auth0 Public Cloud Status ページに戻るには、右上のナビゲーションで **Auth0 Public Cloud Status** を選択します。

<div id="subscribe-to-private-cloud-status-updates">
  ## Private Cloud のステータス更新を購読する
</div>

環境を選択し、**Subscribe** を選択します。

環境に影響するステータス更新を受け取るには、「Subscribe」ボックスに生成された URL をコピーして、Auth0 の Atom または RSS フィードを購読します。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  生成される ATOM / RSS フィード URL は Private Cloud 環境ごとに一意で、[Support に問い合わせる](https://support.auth0.com/)ことでローテーション可能なトークンが含まれています。
</Callout>
