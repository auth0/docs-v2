---
description: エラー、警告、成功イベント用のログフィルターを一覧表示します。
title: ログイベントのフィルター
---

[Dashboard](https://manage.auth0.com/#/logs) で **Filter** の下向き矢印をクリックすると、エラー、警告、成功イベントのログを絞り込めます。

<div id="error-event-filters">
  ## エラーイベントフィルター
</div>

| フィルター                                   | 説明                                                     |
| --------------------------------------- | ------------------------------------------------------ |
| Block Account                           | 単一のアカウントに対するログイン試行の失敗が10回を超えたため、IP がブロックされました          |
| Blocked IP Address                      | ログイン試行の失敗が100回を超えた、またはサインアップ試行が50回を超えたため、IP がブロックされました |
| Breached password                       | 漏えいしたパスワードでログインが試行されました                                |
| Device Confirmation Canceled by User    | ユーザーがデバイスを確認しませんでした                                    |
| Error sending MFA Push Notification     | MFA のプッシュ通知の送信に失敗しました                                  |
| Error sending MFA SMS                   | MFA の SMS の送信に失敗しました                                   |
| Failed API Operation                    | API 操作に失敗しました                                          |
| Failed Change Email                     | ユーザーのメールアドレスの変更に失敗しました                                 |
| Failed Change Password                  | ユーザーのパスワードの変更に失敗しました                                   |
| Failed Change Password Request          | パスワード変更リクエストに失敗しました                                    |
| Failed Change Phone Number              | ユーザーの電話番号の変更に失敗しました                                    |
| Failed Change Username                  | username の変更に失敗しました                                    |
| Failed Connector Provisioning           | AD/LDAP コネクターのプロビジョニングに失敗しました                          |
| Failed Delegation                       | 委任トークンの生成に失敗しました                                       |
| Failed Exchange                         | トークン交換                                                 |
| Failed Exchange                         | ネイティブソーシャルログイン                                         |
| Failed Exchange                         | 認可コードからアクセストークンへの交換                                    |
| Failed Exchange                         | クライアントクレデンシャルからアクセストークンへの交換                            |
| Failed Exchange                         | Password からアクセストークンへの交換                                |
| Failed Exchange                         | リフレッシュトークンからアクセストークンへの交換                               |
| Failed Exchange                         | Password と OOB チャレンジ からアクセストークンへの交換                   |
| Failed Exchange                         | Password と OTP Challenge からアクセストークンへの交換                |
| Failed Exchange                         | Password と MFA リカバリーコード からアクセストークンへの交換                |
| Failed Exchange                         | デバイスコードからアクセストークンへの交換                                  |
| Failed Login                            | ユーザーのログインに失敗しました                                       |
| Failed Login (invalid email/username)   | 無効な username が原因でユーザーのログインに失敗しました                      |
| Failed Login (wrong password)           | 無効なパスワードが原因でユーザーのログインに失敗しました                           |
| Failed Logout                           | ユーザーのログアウトに失敗しました                                      |
| Failed OIDC Back-Channel Logout Request | バックチャネルログアウト リクエストに失敗しました                              |
| Failed Post Change Password Hook        | パスワード変更後フックの実行に失敗しました                                  |
| Failed Post User Registration Hook      | ユーザー登録後フックの実行に失敗しました                                   |
| Failed Sending Notification             | メール通知の送信に失敗しました                                        |
| Failed Signup                           | サインアップに失敗しました                                          |
| Failed Silent Auth                      | サイレント認証に失敗しました                                         |
| Failed User Deletion                    | ユーザーの削除に失敗しました                                         |
| Failed Verification Email               | 確認メールの送信に失敗しました                                        |
| Failed Verification Email Request       | 確認メールのリクエスト処理に失敗しました                                   |
| Failed CORS                             | 指定されたアプリケーションの許可済みオリジン一覧に、オリジンが含まれていません                |
| Failed by Connector                     | AD/LDAP コネクターの障害                                       |
| Failed cross origin authentication      | クロスオリジン認証に失敗しました                                       |
| Failed device activation                | デバイスのアクティベーションに失敗しました                                  |
| Failed device authorization request     | デバイス認可リクエストに失敗しました                                     |
| MFA Enrollment start failed             | 多要素認証の登録開始に失敗しました                                     |
| OTP Auth failed                         | OTP 認証に失敗しました                                          |
| OTP Auth rejected                       | OTP 認証が拒否されました                                         |
| Rate Limit on API                       | 指定された時間内の Authentication API へのリクエスト数が上限に達しました         |
| Recovery failed                         | MFA リカバリーコードの処理に失敗しました                                 |
| Second factor email failed              | MFA 用メールの送信に失敗しました                                     |
| Too Many Calls to /delegation           | /delegation エンドポイントのレート制限を超過しました                       |
| Too Many Calls to /userinfo             | /userinfo エンドポイントのレート制限を超過しました                         |
| Too Many Invalid Device Codes           | 無効な デバイスコード に対するレート制限を超過しました                          |

<div id="warning-event-filters">
  ## 警告イベントフィルター
</div>

| フィルター      | 説明                        |
| ---------- | ------------------------- |
| 非推奨通知      | 機能は非推奨です                  |
| 失敗回数が多すぎます | 多要素認証のOTPの失敗回数が多すぎます      |
| 失敗回数が多すぎます | 多要素認証のリカバリーコードの失敗回数が多すぎます |
| ユーザーのインポート | ユーザーのインポートに失敗しました         |
| ログイン時の警告   | ログイン時の警告                  |
