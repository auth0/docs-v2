title: デプロイオプション
---

Auth0 では、パブリッククラウドとプライベートクラウドの両方のデプロイオプションに加え、エンタープライズ向けデプロイを支援するツールも提供しています。

Auth0 アイデンティティプラットフォームは、次の形態でデプロイできます。

| オプション                                                                                          | 説明                                                                         |
| ---------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------- |
| **Public Cloud**                                                                               | Auth0 のクラウド上で稼働するマルチテナントのクラウドサービスです。標準的な認証、パーソナライズ、ユーザー管理、セキュリティ機能を備えています。 |
| [**Private Cloud on AWS**](/ja/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-aws)     | AWS 上で稼働する専用のマネージドクラウドサービスです。分離性、高いパフォーマンス、開発用インスタンス、Geo-HA アドオンなどを提供します。  |
| [**Private Cloud on Azure**](/ja/docs/deploy-monitor/deploy-private-cloud/private-cloud-on-azure) | Azure 上で稼働する専用のクラウドサービスです。分離性、高いパフォーマンス、開発用インスタンス、Geo-HA アドオンなどを提供します。     |

**Private Cloud** パッケージは、次のような場合に利用できるマネージドサービスです。

* 組織でマルチテナントのパブリッククラウドサービスを利用できない
* 1 秒あたりのリクエスト数 (RPS) について保証された水準が必要である

特別なサポート要件がある場合、または Auth0 が提供する Professional Services の詳細については、[Auth0 Sales にお問い合わせください](https://auth0.com/contact-us)。
