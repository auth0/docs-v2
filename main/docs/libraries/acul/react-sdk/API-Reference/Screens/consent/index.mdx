</ParamField>

<ParamField body='useOrganization' type={<span>() =&gt; <a href="/docs-v2/main/docs/libraries/acul/react-sdkAPI-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>

Hook to access organization context and settings.

```jsx Example
import { useOrganization } from '@auth0/auth0-acul-react/login-id';
function OrgSelector() {
  const organization = useOrganization();
  if (!organization) {
    return <p>No organization context</p>;
  }
}
```
</ParamField>
