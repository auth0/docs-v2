The `organization-selection` screen is displayed when a user needs to enter an Organization name to continue the authentication flow. It allows the user to type their Organization name and proceed.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you to include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { useOrganizationSelection } from '@auth0/auth0-acul-react';

// partial import
import {
  useOrganizationSelection,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Methods
  continueWithOrganizationName,
} from '@auth0/auth0-acul-react/organization-selection';

function OrganizationSelectionScreen() {
  const { continueWithOrganizationName } = useOrganizationSelection();
  return (
    <button onClick={() => continueWithOrganizationName({ organizationName: 'my-org' })}>Continue</button>
  );
}
```

## Context Hooks

Screen-scoped hooks that provide read-only access to Auth0 context data on the `organization-selection` screen. Import them from `@auth0/auth0-acul-react/organization-selection`.

<ParamField body='useBranding' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>

This hook provides branding configurations, such as logo, colors, and theme settings displayed on the `organization-selection` screen.

```jsx Example
import { useBranding } from '@auth0/auth0-acul-react/organization-selection';
function CustomTheme() {
  const branding = useBranding();
}
```
</ParamField>

<ParamField body='useClient' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>

This hook provides client-related configurations, such as `id`, `name`, and `logoUrl`, for the `organization-selection` screen.

```jsx Example
import { useClient } from '@auth0/auth0-acul-react/organization-selection';
function AppInfo() {
  const client = useClient();
}
```
</ParamField>

<ParamField body='useOrganization' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>

This hook provides information about the user's Organization if the selection is Organization-scoped. Returns `null` when no Organization context is present.

```jsx Example
import { useOrganization } from '@auth0/auth0-acul-react/organization-selection';
function OrgSelector() {
  const organization = useOrganization();
  if (!organization) {
    return <p>No Organization context</p>;
  }
}
```
</ParamField>

<ParamField body='usePrompt' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>

## Methods

<ParamField body='continueWithOrganizationName' type={<span>void | Promise&lt;void&gt;</span>}>

This method submits the Organization name entered by the user and continues the authentication flow.

```jsx Example
import { useOrganizationSelection } from '@auth0/auth0-acul-react/organization-selection';

function ContinueButton() {
  const { continueWithOrganizationName } = useOrganizationSelection();
  return (
    <button onClick={() => continueWithOrganizationName({ organizationName: 'my-org' })}>
      Continue
    </button>
  );
}
```
