The `accept-invitation` screen is displayed when a user has been invited to join an organization. It allows the user to accept the invitation and proceed with the authentication flow.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you to include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { useAcceptInvitation } from '@auth0/auth0-acul-react';

// partial import
import {
  useAcceptInvitation,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Methods
  acceptInvitation,
} from '@auth0/auth0-acul-react/accept-invitation';

function AcceptInvitationScreen() {
  const { acceptInvitation } = useAcceptInvitation();
  return (
    <button onClick={() => acceptInvitation()}>Accept Invitation</button>
  );
}
```

## Context Hooks

Screen-scoped hooks that provide read-only access to Auth0 context data on the `accept-invitation` screen. Import them from `@auth0/auth0-acul-react/accept-invitation`.

<ParamField body='useBranding' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>


</ParamField>

<ParamField body='useOrganization' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>

This hook provides information about the organization the user has been invited to join. Returns `null` when no organization context is present.

```jsx Example
import { useOrganization } from '@auth0/auth0-acul-react/accept-invitation';
function OrgSelector() {
  const organization = useOrganization();
  if (!organization) {
    return <p>No organization context</p>;
  }
}
```
</ParamField>

## Methods

<ParamField body='acceptInvitation' type={<span>void | Promise&lt;void&gt;</span>}>

This method submits the user's acceptance of the organization invitation and proceeds with the authentication flow.

```jsx Example
import { useAcceptInvitation } from '@auth0/auth0-acul-react/accept-invitation';

function AcceptButton() {
  const { acceptInvitation } = useAcceptInvitation();
  return (
    <button onClick={() => acceptInvitation()}>
      Accept Invitation
    </button>
  );
}
```
