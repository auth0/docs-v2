The `reset-password-mfa-recovery-code-challenge` screen is displayed during the password reset flow when multi-factor authentication is required via a recovery code. It prompts the user to enter their recovery code to continue the reset process.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you to include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { useResetPasswordMfaRecoveryCodeChallenge } from '@auth0/auth0-acul-react';

// partial import
import {
  useResetPasswordMfaRecoveryCodeChallenge,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Methods
  continueMethod,
  tryAnotherMethod,
} from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';

function ResetPasswordMfaRecoveryCodeChallengeScreen() {
  const { continueMethod } = useResetPasswordMfaRecoveryCodeChallenge();
  return (
    <button onClick={() => continueMethod('ABC123')}>
      Verify Recovery Code
    </button>
  );
}
```

## Context Hooks

Screen-scoped hooks that provide read-only access to Auth0 context data on the `reset-password-mfa-recovery-code-challenge` screen. Import them from `@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge`.

<ParamField body='useBranding' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>

This hook provides branding configurations, such as logo, colors, and theme settings displayed on the `reset-password-mfa-recovery-code-challenge` screen.

```jsx Example
import { useBranding } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';
function CustomTheme() {
  const branding = useBranding();
}
```
</ParamField>

<ParamField body='useClient' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>

This hook provides client-related configurations, such as `id`, `name`, and `logoUrl`, for the `reset-password-mfa-recovery-code-challenge` screen.

```jsx Example
import { useClient } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';
function AppInfo() {
```

</ParamField>

<ParamField body='useOrganization' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>

This hook provides information about the user's Organization if the password reset is Organization-scoped. Returns `null` when no Organization context is present.

```jsx Example

function OrgSelector() {
  const organization = useOrganization();
  if (!organization) {
    return <p>No Organization context</p>;
  }
}
```
</ParamField>

## Methods

<ParamField body='continueMethod' type={<span>void | Promise&lt;void&gt;</span>}>

This method submits the recovery code to continue the MFA step in the password reset flow.

```jsx Example
import { useResetPasswordMfaRecoveryCodeChallenge } from '@auth0/auth0-acul-react/reset-password-mfa-recovery-code-challenge';

function VerifyRecoveryCodeForm() {
  const { continueMethod } = useResetPasswordMfaRecoveryCodeChallenge();
  return (
    <button onClick={() => continueMethod('ABC123')}>
      Verify Recovery Code
    </button>
  );
}
```

<ParamField body='tryAnotherMethod' type={<span>void | Promise&lt;void&gt;</span>}>

