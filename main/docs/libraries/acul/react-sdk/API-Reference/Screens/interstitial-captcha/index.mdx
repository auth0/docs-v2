The `interstitial-captcha` screen is displayed when a CAPTCHA challenge is required during the authentication flow. It prompts the user to complete the CAPTCHA before proceeding.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you to include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { useInterstitialCaptcha } from '@auth0/auth0-acul-react';

// partial import
import {
  useInterstitialCaptcha,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Methods
  submitCaptcha,
} from '@auth0/auth0-acul-react/interstitial-captcha';

function InterstitialCaptchaScreen() {
  const { submitCaptcha } = useInterstitialCaptcha();
  return (
    <button onClick={() => submitCaptcha({ captcha: 'token' })}>Submit</button>
  );
}
```

## Context Hooks

Screen-scoped hooks that provide read-only access to Auth0 context data on the `interstitial-captcha` screen. Import them from `@auth0/auth0-acul-react/interstitial-captcha`.

<ParamField body='useBranding' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>

This hook provides branding configurations, such as logo, colors, and theme settings displayed on the `interstitial-captcha` screen.

```jsx Example
import { useBranding } from '@auth0/auth0-acul-react/interstitial-captcha';
function CustomTheme() {
  const branding = useBranding();
}
```
</ParamField>

<ParamField body='useClient' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>

This hook provides client-related configurations, such as `id`, `name`, and `logoUrl`, for the `interstitial-captcha` screen.

```jsx Example
import { useClient } from '@auth0/auth0-acul-react/interstitial-captcha';
function AppInfo() {
  const client = useClient();
}
```
</ParamField>

<ParamField body='useOrganization' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>

This hook provides information about the user's organization if the CAPTCHA challenge is organization-scoped. Returns `null` when no organization context is present.

```jsx Example
import { useOrganization } from '@auth0/auth0-acul-react/interstitial-captcha';
function OrgSelector() {
  const organization = useOrganization();
  if (!organization) {
    return <p>No organization context</p>;
  }
}
```
</ParamField>

<ParamField body='usePrompt' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>

This hook contains data about the current prompt in the authentication flow.

```jsx Example
import { usePrompt } from '@auth0/auth0-acul-react/interstitial-captcha';
function FlowInfo() {
  const prompt = usePrompt();
}
```
</ParamField>

<ParamField body='useScreen' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>

This hook contains details specific to the `interstitial-captcha` screen, including its configuration and context.

```jsx Example

function ScreenDebug() {
  const screen = useScreen();
}
```
</ParamField>

<ParamField body='useTenant' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>

This hook contains data related to the tenant, such as `id` and associated metadata.

```jsx Example

function TenantInfo() {
```

</ParamField>

<ParamField body='useTransaction' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>

This hook provides transaction-specific data for the `interstitial-captcha` screen, such as active connections and current flow state.


<ParamField body='useUser' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>


<ParamField body='useInterstitialCaptcha' type={<a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/InterstitialCaptchaMembers">InterstitialCaptchaMembers</a>}>

</ParamField>

## Methods

<ParamField body='submitCaptcha' type={<span>void | Promise&lt;void&gt;</span>}>

This method submits the CAPTCHA token to complete the challenge and continue the authentication flow.

```jsx Example
import { useInterstitialCaptcha } from '@auth0/auth0-acul-react/interstitial-captcha';

function SubmitButton() {
  const { submitCaptcha } = useInterstitialCaptcha();
  return (
    <button onClick={() => submitCaptcha({ captcha: 'token' })}>
      Continue
    </button>
  );
}
```
