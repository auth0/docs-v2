The `logout-aborted` screen is displayed when a logout attempt is cancelled or denied by the user.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you to include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { useLogoutAborted } from '@auth0/auth0-acul-react';

// partial import
import {
  useLogoutAborted,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
} from '@auth0/auth0-acul-react/logout-aborted';

function LogoutAbortedScreen() {
  const screen = useLogoutAborted();
  // This is a display-only screen — no actions are available.
  return <p>Logout was cancelled.</p>;
}
```

## Context Hooks

Screen-scoped hooks that provide read-only access to Auth0 context data on the `logout-aborted` screen. Import them from `@auth0/auth0-acul-react/logout-aborted`.

<ParamField body='useBranding' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>

Provides branding configurations, such as logo, colors, and theme settings displayed on the logout-aborted screen.

```jsx Example
import { useBranding } from '@auth0/auth0-acul-react/logout-aborted';
function CustomTheme() {
  const branding = useBranding();
}
```
</ParamField>

<ParamField body='useClient' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>

Provides client-related configurations, such as `id`, `name`, and `logoUrl`, for the `logout-aborted` screen.

```jsx Example
import { useClient } from '@auth0/auth0-acul-react/logout-aborted';
function AppInfo() {
  const client = useClient();
}
```
</ParamField>

<ParamField body='useOrganization' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>

Provides information about the user's organization if the session is organization-scoped. Returns `null` when no organization context is present.

```jsx Example
import { useOrganization } from '@auth0/auth0-acul-react/logout-aborted';
function OrgSelector() {
  const organization = useOrganization();
  if (!organization) {
    return <p>No organization context</p>;
  }
}
```
</ParamField>

<ParamField body='usePrompt' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>

Contains data about the current prompt in the authentication flow.

```jsx Example
import { usePrompt } from '@auth0/auth0-acul-react/logout-aborted';
function FlowInfo() {
  const prompt = usePrompt();
}
```
</ParamField>

<ParamField body='useScreen' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembers">ScreenMembers</a></span>}>

Contains details specific to the `logout-aborted` screen, including its configuration and context.

```jsx Example
import { useScreen } from '@auth0/auth0-acul-react/logout-aborted';
function ScreenDebug() {
  const screen = useScreen();
}
```
</ParamField>

<ParamField body='useTenant' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>

Contains data related to the tenant, such as `id` and associated metadata.

```jsx Example
import { useTenant } from '@auth0/auth0-acul-react/logout-aborted';
function TenantInfo() {
  const tenant = useTenant();
}
```
</ParamField>

<ParamField body='useTransaction' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>

Provides transaction-specific data for the `logout-aborted` screen, such as active connections and current flow state.

```jsx Example
import { useTransaction } from '@auth0/auth0-acul-react/logout-aborted';
function TransactionInfo() {
  const transaction = useTransaction();
}
```
</ParamField>

<ParamField body='useUntrustedData' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>

Handles untrusted data passed to the screen, such as parameters from URL query strings.

```jsx Example
import { useUntrustedData } from '@auth0/auth0-acul-react/logout-aborted';
function PrefilledForm() {
  const untrustedData = useUntrustedData();
}
```
</ParamField>

<ParamField body='useUser' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>

Details of the active user, including `username`, `email`, and available authentication methods.

```jsx Example
import { useUser } from '@auth0/auth0-acul-react/logout-aborted';
function UserProfile() {
  const user = useUser();
}
```
</ParamField>
