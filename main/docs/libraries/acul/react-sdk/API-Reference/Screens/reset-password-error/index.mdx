The `reset-password-error` screen is displayed when an error occurs during the password reset flow, such as an expired or invalid reset link. It informs the user of the issue and provides guidance on next steps.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you to include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { useResetPasswordError } from '@auth0/auth0-acul-react';

// partial import
import {
  useResetPasswordError,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
} from '@auth0/auth0-acul-react/reset-password-error';

function ResetPasswordErrorScreen() {
  const screen = useScreen();
  return <p>{screen.name}</p>;
}
```

## Context Hooks

Screen-scoped hooks that provide read-only access to Auth0 context data on the `reset-password-error` screen. Import them from `@auth0/auth0-acul-react/reset-password-error`.

<ParamField body='useBranding' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>

This hook provides branding configurations, such as logo, colors, and theme settings displayed on the `reset-password-error` screen.

```jsx Example
import { useBranding } from '@auth0/auth0-acul-react/reset-password-error';
function CustomTheme() {
  const branding = useBranding();
}
```
</ParamField>

<ParamField body='useClient' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ClientMembers">ClientMembers</a></span>}>

This hook provides client-related configurations, such as `id`, `name`, and `logoUrl`, for the `reset-password-error` screen.

```jsx Example
import { useClient } from '@auth0/auth0-acul-react/reset-password-error';
function AppInfo() {
  const client = useClient();
}
```
</ParamField>

<ParamField body='useOrganization' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>

This hook provides information about the user's Organization if the password reset is Organization-scoped. Returns `null` when no Organization context is present.

```jsx Example
import { useOrganization } from '@auth0/auth0-acul-react/reset-password-error';
function OrgSelector() {
  const organization = useOrganization();
  if (!organization) {
    return <p>No Organization context</p>;
  }
}
```
</ParamField>

<ParamField body='usePrompt' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PromptMembers">PromptMembers</a></span>}>

This hook contains data about the current prompt in the authentication flow.

```jsx Example
import { usePrompt } from '@auth0/auth0-acul-react/reset-password-error';
function FlowInfo() {
  const prompt = usePrompt();
}
```
</ParamField>

<ParamField body='useScreen' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ScreenMembersOnResetPasswordError">ScreenMembersOnResetPasswordError</a></span>}>

This hook contains details specific to the `reset-password-error` screen, including its configuration and context.

```jsx Example
import { useScreen } from '@auth0/auth0-acul-react/reset-password-error';
function ScreenDebug() {
  const screen = useScreen();
}
```
</ParamField>

<ParamField body='useTenant' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TenantMembers">TenantMembers</a></span>}>

This hook contains data related to the tenant, such as `id` and associated metadata.

```jsx Example
import { useTenant } from '@auth0/auth0-acul-react/reset-password-error';
function TenantInfo() {
  const tenant = useTenant();
}
```
</ParamField>

<ParamField body='useTransaction' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/TransactionMembers">TransactionMembers</a></span>}>

This hook provides transaction-specific data for the `reset-password-error` screen, such as active connections and current flow state.

```jsx Example
import { useTransaction } from '@auth0/auth0-acul-react/reset-password-error';
function TransactionInfo() {
  const transaction = useTransaction();
}
```
</ParamField>

<ParamField body='useUntrustedData' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UntrustedDataMembers">UntrustedDataMembers</a></span>}>

This hook handles untrusted data passed to the screen, such as a prefilled email or username from URL parameters.

```jsx Example
import { useUntrustedData } from '@auth0/auth0-acul-react/reset-password-error';
function PrefilledForm() {
  const untrustedData = useUntrustedData();
}
```
</ParamField>

<ParamField body='useUser' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UserMembers">UserMembers</a></span>}>

This hook provides details of the active user, including `username`, `email`, and available authentication methods.

```jsx Example
import { useUser } from '@auth0/auth0-acul-react/reset-password-error';
function UserProfile() {
  const user = useUser();
}
```
</ParamField>

<ParamField body='useResetPasswordError' type={<a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ResetPasswordErrorMembers">ResetPasswordErrorMembers</a>}>
This hook returns all methods and context available on the `reset-password-error` screen.
</ParamField>
