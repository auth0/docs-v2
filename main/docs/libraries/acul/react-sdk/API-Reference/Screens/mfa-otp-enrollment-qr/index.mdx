The `mfa-otp-enrollment-qr` screen displays a QR code for the user to scan with their authenticator app to complete OTP enrollment.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you to include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle, which is useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { useMfaOtpEnrollmentQr } from '@auth0/auth0-acul-react';

// partial import
import {
  useMfaOtpEnrollmentQr,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Methods
  continueMethod,
  toggleView,
  tryAnotherMethod,
} from "@auth0/auth0-acul-react/mfa-otp-enrollment-qr";

function MfaOtpEnrollmentQrScreen() {
  const { continueMethod } = useMfaOtpEnrollmentQr();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Verify Code
    </button>
  );
}
```

## Context Hooks

Screen-scoped hooks that provide read-only access to Auth0 context data on the `mfa-otp-enrollment-qr` screen. Import them from `@auth0/auth0-acul-react/mfa-otp-enrollment-qr`.

<ParamField body='useOrganization' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>

## Methods

<ParamField body='continueMethod' type='Promise<void>'>

This method submits the OTP code to verify the authenticator app is configured correctly and complete enrollment.

```jsx Example
import { useMfaOtpEnrollmentQr } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';

function EnrollmentQrForm() {
  const { continueMethod } = useMfaOtpEnrollmentQr();
  return (
    <button onClick={() => continueMethod({ code: '123456' })}>
      Verify
    </button>
  );
}
```

<ParamField body='toggleView' type='Promise<void>'>

This method toggles the enrollment view between QR code and manual entry (secret key) modes.

```jsx Example
import { useMfaOtpEnrollmentQr } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';

function ToggleViewButton() {
  const { toggleView } = useMfaOtpEnrollmentQr();
  return (
    <button onClick={() => toggleView()}>
      Enter Code Manually
    </button>
  );
}
```

</ParamField>

<ParamField body='tryAnotherMethod' type='Promise<void>'>

This method navigates to the MFA method selection screen so the user can choose a different authentication factor.

```jsx Example
import { useMfaOtpEnrollmentQr } from '@auth0/auth0-acul-react/mfa-otp-enrollment-qr';

function TryAnotherMethodButton() {
  const { tryAnotherMethod } = useMfaOtpEnrollmentQr();
  return (
    <button onClick={() => tryAnotherMethod()}>
      Try Another Method
    </button>
  );
}
```

</ParamField>
