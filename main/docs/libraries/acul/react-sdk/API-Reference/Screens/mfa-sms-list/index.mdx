The `mfa-sms-list` screen displays a list of enrolled phone numbers and allows the user to select one to receive an SMS verification code.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you to include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle, which is useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { useMfaSmsList } from '@auth0/auth0-acul-react';

// partial import
import {
  useMfaSmsList,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Methods
  backAction,
  selectPhoneNumber,
} from "@auth0/auth0-acul-react/mfa-sms-list";

function MfaSmsListScreen() {
  const { selectPhoneNumber } = useMfaSmsList();
  return (
    <button onClick={() => selectPhoneNumber({ index: 0 })}>
      Use This Phone Number
    </button>
  );
}
```

## Context Hooks

Screen-scoped hooks that provide read-only access to Auth0 context data on the `mfa-sms-list` screen. Import them from `@auth0/auth0-acul-react/mfa-sms-list`.

<ParamField body='useBranding' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/BrandingMembers">BrandingMembers</a></span>}>


</ParamField>


</ParamField>

<ParamField body='useOrganization' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>

This hook provides information about the user's Organization if the MFA flow is Organization scoped. Returns `null` when no Organization context is present.

```jsx Example

  const organization = useOrganization();
  if (!organization) {
    return <p>No organization context</p>;
  }
}
```
</ParamField>

## Methods

<ParamField body='backAction' type='Promise<void>'>

This method navigates back to the previous screen in the MFA flow.

```jsx Example
import { useMfaSmsList } from '@auth0/auth0-acul-react/mfa-sms-list';

function BackButton() {
  const { backAction } = useMfaSmsList();
  return (
    <button onClick={() => backAction()}>
      Go Back
    </button>
  );
}
```

</ParamField>

<ParamField body='selectPhoneNumber' type='Promise<void>'>

This method selects a phone number from the enrolled list to use for the SMS challenge.

```jsx Example
import { useMfaSmsList } from '@auth0/auth0-acul-react/mfa-sms-list';

function PhoneNumberSelector() {
  const { selectPhoneNumber } = useMfaSmsList();
  return (
    <button onClick={() => selectPhoneNumber({ index: 0 })}>
      Use This Phone Number
    </button>
  );
}
```
