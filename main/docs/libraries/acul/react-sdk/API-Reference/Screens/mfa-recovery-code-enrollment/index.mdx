The `mfa-recovery-code-enrollment` screen displays a recovery code for the user to save, which can be used to access their account if they lose access to their primary MFA method.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { useMfaRecoveryCodeEnrollment } from '@auth0/auth0-acul-react';

// partial import
import {
  useMfaRecoveryCodeEnrollment,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Methods
  continueMethod,
} from "@auth0/auth0-acul-react/mfa-recovery-code-enrollment";

function MfaRecoveryCodeEnrollmentScreen() {
  const { continueMethod } = useMfaRecoveryCodeEnrollment();
  return (
    <button onClick={() => continueMethod({ isCodeCopied: true })}>
      I've Saved My Recovery Code
    </button>
  );
}
```

## Methods

<ParamField body='continueMethod' type={<span>void | Promise&lt;void&gt;</span>}>

This method confirms that the user has saved their recovery code and continues the enrollment flow.

```jsx Example
import { useMfaRecoveryCodeEnrollment } from '@auth0/auth0-acul-react/mfa-recovery-code-enrollment';

function ContinueButton() {
  const { continueMethod } = useMfaRecoveryCodeEnrollment();
  return (
    <button onClick={() => continueMethod({ isCodeCopied: true })}>
      I've Saved My Recovery Code
    </button>
  );
}
```
