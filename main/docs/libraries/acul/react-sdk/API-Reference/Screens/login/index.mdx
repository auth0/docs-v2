---
title: "Login"
---

The Login screen handles the primary authentication flow, allowing users to sign in with username/password credentials or via a federated identity provider. This screen is presented when a user attempts to log in to an application.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/5be9XfnulShqeXa4MXfXky/f3a5dded3539d4d683a525dfb6ad8042/Screenshot_2025-01-28_at_10.49.07.png" alt="ACUL Login"/></Frame>

```jsx Import Example
import React, { useState } from "react";
import {
  useLogin,
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useUntrustedData,
} from "@auth0/auth0-acul-react/login";

function LoginForm() {
  const { login, federatedLogin } = useLogin();
  return (
    <button onClick={() => login({ username: "user@example.com", password: "secret" })}>
      Sign In
    </button>
  );
}
```

<ParamField body='useOrganization' type={<span>() =&gt; <a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/OrganizationMembers">OrganizationMembers</a></span>}>

Provides information about the user's organization if the login is organization-scoped. Returns `null` when no organization context is present.

```jsx Example
import { useOrganization } from '@auth0/auth0-acul-react/login';

  const organization = useOrganization();
  if (!organization) {
    return <p>No organization context</p>;
  }
}
```
</ParamField>

## Methods

<ParamField body='federatedLogin' type={<span>void | Promise&lt;void&gt;</span>}>

Initiates authentication via a federated identity provider such as Google or GitHub.

```jsx Example
import { useLogin } from '@auth0/auth0-acul-react/login';

function SocialButton() {
  const { federatedLogin } = useLogin();
  return (
    <button onClick={() => federatedLogin({ connection: 'google-oauth2' })}>
      Sign in with Google
    </button>
  );
}
```

**Properties**

<ParamField body='connection' type='string'>
The social connection name to use (e.g. `google-oauth2`, `github`).

<ParamField body='login' type={<span>void | Promise&lt;void&gt;</span>}>

```jsx Example
import { useLogin } from '@auth0/auth0-acul-react/login';

function LoginForm() {

    <button onClick={() => login({ username: 'user@example.com', password: 'secret' })}>

}
```
