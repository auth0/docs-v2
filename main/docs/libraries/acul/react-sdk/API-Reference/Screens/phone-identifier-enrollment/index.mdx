title: "PhoneIdentifierEnrollment"
---

The `phone-identifier-enrollment` screen is displayed when a user needs to enroll a phone number as an identifier. It allows the user to select their preferred verification method (SMS or voice call) and continue the enrollment process or return to the previous step.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you to include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { usePhoneIdentifierEnrollment } from '@auth0/auth0-acul-react';

// partial import
import {
  usePhoneIdentifierEnrollment,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
  // Methods
  continuePhoneEnrollment,
  returnToPrevious,
} from '@auth0/auth0-acul-react/phone-identifier-enrollment';

function PhoneIdentifierEnrollmentScreen() {
  const { continuePhoneEnrollment } = usePhoneIdentifierEnrollment();
  return (
    <button onClick={() => continuePhoneEnrollment({ type: 'text' })}>Send Code via SMS</button>
  );
}
```

## Methods

<ParamField body='continuePhoneEnrollment' type={<span>void | Promise&lt;void&gt;</span>}>

This method submits the user's preferred phone verification method and continues the enrollment flow.

```jsx Example
import { usePhoneIdentifierEnrollment } from '@auth0/auth0-acul-react/phone-identifier-enrollment';

function EnrollButton() {
  const { continuePhoneEnrollment } = usePhoneIdentifierEnrollment();
  return (
    <button onClick={() => continuePhoneEnrollment({ type: 'text' })}>
      Send Code via SMS
    </button>
  );
}
```
