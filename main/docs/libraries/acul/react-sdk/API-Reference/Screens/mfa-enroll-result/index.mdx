The `mfa-enroll-result` screen displays the outcome of an MFA enrollment attempt and confirms whether the factor was successfully enrolled.

## Import

Each screen has its own set of hooks and methods. The SDK supports **partial import** and **root import** for each screen.
* Using partial import allows you include only the code you need for your specific use case.
* Using root import allows you to load all screens from a single bundle useful when you want a unified build to handle all possible screens.

```jsx Import Example
// root import
import { useMfaEnrollResult } from '@auth0/auth0-acul-react';

// partial import
import {
  useMfaEnrollResult,
  // Context hooks
  useUser,
  useTenant,
  useBranding,
  useClient,
  useOrganization,
  usePrompt,
  useScreen,
  useTransaction,
  useUntrustedData,
  // Common hooks
  useCurrentScreen,
  useAuth0Themes,
  useErrors,
  // Utility hooks
  useChangeLanguage,
} from "@auth0/auth0-acul-react/mfa-enroll-result";

function MfaEnrollResultScreen() {
  const { useScreen } = useMfaEnrollResult();
  return (
    <div>Enrollment complete</div>
  );
}
```
