---
title: "ErrorType"
---

A string union type that classifies errors managed by [`useErrors`](/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Used to filter errors with `errors.byType()` and present in each [`ErrorItem`](/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem) as the `kind` property.

```typescript
type ErrorType = "auth0" | "validation" | "configuration";
```

## Values

<ParamField body='"auth0"' type='ErrorType'>
  Errors returned by Auth0 or your own backend during authentication or authorization.
</ParamField>

<ParamField body='"validation"' type='ErrorType'>
  Errors from client-side form validation, such as invalid input before a request is made.
</ParamField>

<ParamField body='"configuration"' type='ErrorType'>
  Errors caused by incorrect SDK integration or misconfiguration, such as calling a hook outside of its required context. Only surfaced when `includeDevErrors: true` is set in [`UseErrorOptions`](/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorOptions).
</ParamField>
