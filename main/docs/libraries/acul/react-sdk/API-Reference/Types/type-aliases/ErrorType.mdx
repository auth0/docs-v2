---
title: "ErrorType"
---

A string union type that classifies errors managed by [`useErrors`](/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Used to filter errors with `errors.byKind()` and present in each [`ErrorItem`](/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorItem) as the `kind` property.

```typescript
type ErrorType = "server" | "client" | "developer";
```

## Values

<ParamField body='"server"' type='ErrorType'>
  Errors returned by Auth0 or your own backend during authentication or authorization.
</ParamField>

<ParamField body='"client"' type='ErrorType'>
  Errors from client-side validation, such as invalid form input before a request is made.
</ParamField>

<ParamField body='"developer"' type='ErrorType'>
  Errors caused by incorrect SDK integration or misuse, such as calling a hook outside of its required context. Only surfaced when `includeDevErrors: true` is set in [`UseErrorOptions`](/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UseErrorOptions).
</ParamField>
