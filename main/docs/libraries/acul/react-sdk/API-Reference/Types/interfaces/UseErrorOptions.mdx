---
title: "UseErrorOptions"
---

Options object for the [`useError`](/docs/libraries/acul/react-sdk/API-Reference/Hooks/useError) hook. Controls error display behavior within Auth0 Universal Login screens.

```ts Example
export interface UseErrorOptions {
  includeDevErrors?: boolean;
}
```

## Properties

<ParamField body='includeDevErrors' type='boolean'>
  When `true`, includes development-specific error details in the error output. Useful for debugging during development; should be disabled in production.
</ParamField>
