---
title: "MfaPushEnrollmentQrWithRememberOptions"
---

Options for continuing MFA push enrollment with an optional device trust setting.

Extends `CustomOptions` to allow passing custom options alongside the remember device preference.

```ts Example
export interface MfaPushEnrollmentQrWithRememberOptions extends CustomOptions {
  rememberDevice?: boolean;
  [key: string]: string | number | boolean | undefined;
}
```

## Indexable

\[`key`: `string`\]: `string` \| `number` \| `boolean` \| `undefined`

## Properties

<ParamField body='rememberDevice?' type='boolean'>
When set to `true`, registers this device as trusted so the user is not prompted for MFA push authentication on subsequent logins from the same device.
</ParamField>
