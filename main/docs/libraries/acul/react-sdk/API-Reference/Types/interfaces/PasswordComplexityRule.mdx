---
title: "PasswordComplexityRule"
---

Represents a single evaluated password complexity rule. Returned as part of [`PasswordValidationResult`](/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordValidationResult) by [`usePasswordValidation`](/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation). Supports hierarchical rule composition via the optional `items` array.

```typescript
export interface PasswordComplexityRule {
  code: string;
  label: string;
  status: "valid" | "error";
  isValid: boolean;
  args?: {
    count: number;
    total?: number;
    example?: string;
  };
  items?: PasswordComplexityRule[];
}
```

## Properties

<ParamField body='code' type='string' required>
  A machine-readable identifier for this complexity rule. Example: `"lengthAtLeast"`, `"containsAtLeast"`.
</ParamField>

<ParamField body='label' type='string' required>
  A human-readable display name for the rule, suitable for rendering in a password strength indicator.
</ParamField>

<ParamField body='status' type='"valid" | "error"' required>
  The validation outcome for this rule. `"valid"` if the rule passed, `"error"` if it failed.
</ParamField>

<ParamField body='isValid' type='boolean' required>
  Boolean representation of the rule outcome. `true` when `status` is `"valid"`.
</ParamField>

<ParamField body='args' type='object'>
  Additional parameters used to describe the rule's threshold or provide context for the error message.

  <Expandable title="properties">
    <ParamField body='count' type='number' required>
      The numeric threshold or count associated with this rule. Example: minimum character count.
    </ParamField>
    <ParamField body='total' type='number'>
      The total number of characters or items relevant to this rule, when applicable.
    </ParamField>
    <ParamField body='example' type='string'>
      An example value illustrating what satisfies this rule, when applicable.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body='items' type={<span><a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>}>
  Nested complexity rules for composite rule groups. Allows hierarchical rule composition where a parent rule aggregates multiple child rules.
</ParamField>
