Return type for [useResend](/docs/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

```ts Example
export interface UseResendReturn {
  remaining: number;
  disabled: boolean;
  startResend: () => void;
}
```

## Properties

<ParamField body='remaining' type='number' required>

Number of seconds remaining until the next resend attempt is allowed. Use this to drive a countdown timer in the UI.

</ParamField>

<ParamField body='disabled' type='boolean' required>

`true` when the resend action is currently unavailable, either because a resend is in progress or the cooldown period has not elapsed.

</ParamField>

<ParamField body='startResend' type='() => void' required>

Triggers a resend attempt immediately if the action is currently allowed.

</ParamField>
