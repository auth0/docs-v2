---
title: "UseErrorsResult"
---

Result object returned by [`useErrors`](/docs/libraries/acul/react-sdk/API-Reference/Hooks/useErrors). Provides access to the current error state and methods to dismiss individual or all errors on Auth0 Universal Login screens.

```ts Example
export interface UseErrorsResult {
  errors: ErrorsResult;
  hasError: boolean;
  dismiss: (id: string) => void;
  dismissAll: () => void;
}
```

## Properties

<ParamField body='errors' type={<a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/ErrorsResult">ErrorsResult</a>} required>
  The collection of current errors, including messages and identifiers.
</ParamField>

<ParamField body='hasError' type='boolean' required>
  `true` when one or more errors are present; useful for conditionally rendering error UI.
</ParamField>

<ParamField body='dismiss' type='(id: string) => void' required>
  Removes a specific error by its `id`.

<Expandable title="Parameters">
<ParamField body="id" type="string">
  The identifier of the error to remove.
</ParamField>
</Expandable>
</ParamField>

<ParamField body='dismissAll' type='() => void' required>
  Clears all active errors at once.
</ParamField>
