> **changeLanguage**(`options`: [`LanguageChangeOptions`](/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)): `Promise`\<`void`\>

Submits the new locale preference to the server with the `change-language` action and re-renders the interface in the selected language.

## Parameters

<ParamField body="options" type="LanguageChangeOptions" required>
The language change configuration. See [`LanguageChangeOptions`](/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).

<Expandable title="options">
<ParamField body="language" type="string" required>
A locale code for the target language (e.g. `'en'`, `'fr'`, `'es'`). Must be one of the locales enabled in your Auth0 tenant.
</ParamField>

<ParamField body="persist?" type='"session"'>
Persistence scope for the language preference. Currently only `"session"` is supported.
</ParamField>
</Expandable>
</ParamField>

## Returns

`Promise`\<`void`\>

Resolves when the locale preference has been submitted and the screen re-renders in the selected language.
