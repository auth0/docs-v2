title: "LogoutComplete"

The LogoutComplete screen class provides methods associated with the `logout-complete` screen. This screen is displayed when a user completes the logout process.

## Constructors

Create an instance of LogoutComplete screen manager

```ts Example
import LogoutComplete from '@auth0/auth0-acul-js/logout-complete';

const logoutCompleteManager = new LogoutComplete();
```
