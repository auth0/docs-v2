---
title: "ResetPasswordEmail"
---

The ResetPasswordEmail class implements the `reset-password-email` screen functionality. This screen notifies the user that a password reset email has been sent and allows them to resend it if needed.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/6uIv1ljdFyLLeToSfTcl5d/4fa2681220056964ed9179f45fd7c062/Screenshot_2025-01-27_at_18.01.26.png" alt="ACUL Reset_password_email"/></Frame>

## Constructors

Create an instance of ResetPasswordEmail screen manager

```ts Example
import ResetPasswordEmail from '@auth0/auth0-acul-js/reset-password-email';
const resetPasswordEmailManager = new ResetPasswordEmail();
resetPasswordEmailManager.resendEmail();
```

## Methods

<ParamField body='resendEmail' type='(options ?)'>

This method resends the password reset email to the user's registered email address.

```ts Example
import ResetPasswordEmail from '@auth0/auth0-acul-js/reset-password-email';

const resetPasswordEmailManager = new ResetPasswordEmail();
resetPasswordEmailManager.resendEmail();
```
