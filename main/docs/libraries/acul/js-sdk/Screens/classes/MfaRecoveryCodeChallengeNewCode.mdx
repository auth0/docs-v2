The MfaRecoveryCodeChallengeNewCode class implements the `mfa-recovery-code-challenge-new-code` screen functionality. This screen displays the MFA recovery code for safe keeping.

## Constructors

Create an instance of MFA Recovery Code Challenge New Code screen manager:

import MfaRecoveryCodeChallengeNewCode from '@auth0/auth0-acul-js/mfa-recovery-code-challenge-new-code';

const mfaRecoveryCodeChallengeNewCode = new MfaRecoveryCodeChallengeNewCode();

## Methods

<ParamField body="changeLanguage" type="Promise<void>">

This method changes the display language of the Universal Login page.

```typescript Example
import MfaRecoveryCodeChallengeNewCode from '@auth0/auth0-acul-js/mfa-recovery-code-challenge-new-code';

const mfaRecoveryCodeChallengeNewCode = new MfaRecoveryCodeChallengeNewCode();
mfaRecoveryCodeChallengeNewCode.changeLanguage({
  language: 'fr',
});
```
