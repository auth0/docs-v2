## Constructors

Create an instance of MFA Push Welcome screen manager:

```typescript Example
import MfaPushWelcome from '@auth0/auth0-acul-js/mfa-push-welcome';

const mfaPushWelcome = new MfaPushWelcome();
await mfaPushWelcome.enroll();
```

## Methods

<ParamField body='changeLanguage' type='Promise<void>'>

This method changes the display language of the Universal Login page.

```typescript Example
import MfaPushWelcome from '@auth0/auth0-acul-js/mfa-push-welcome';
const mfaPushWelcome = new MfaPushWelcome();
mfaPushWelcome.changeLanguage({
  language: 'fr',
});
```

**Method Parameters**

  The locale code for the desired language (for example, `'en'`, `'fr'`, `'es'`).
</ParamField>
