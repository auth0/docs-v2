---
title: "MfaVoiceEnrollment"
description: "Describes all the properties and methods available to customize the Universal Login `mfa-voice-enrollment` screen."
---

The MfaVoiceEnrollment class implements the `mfa-voice-enrollment` screen functionality. This screen is displayed when the user needs to enter a phone number to receive a voice call with a verification code to continue enrollment.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/10B062fqHG48WmJSPSHMmY/177fd3fb4fef03dccb75f8c6bab626bc/Screenshot_2025-04-23_at_18.39.35.png" alt="MfaVoiceEnrollment" /></Frame>

## Constructors

Create an instance of MFA Voice Enrollment screen manager:

```typescript Example
import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

const mfaVoiceEnrollment = new MfaVoiceEnrollment();
await mfaVoiceEnrollment.continue({ phone: '+1234567890' });
```

<ParamField body="changeLanguage" type="Promise<void>">

This method changes the display language of the Universal Login page.

```typescript Example
import MfaVoiceEnrollment from '@auth0/auth0-acul-js/mfa-voice-enrollment';

const mfaVoiceEnrollment = new MfaVoiceEnrollment();
mfaVoiceEnrollment.changeLanguage({
  language: 'fr',
});
```
