---

The MfaWebAuthnRoamingChallenge class implements the `mfa-webauthn-roaming-challenge` screen functionality. This screen is displayed when the user needs to verify their identity using a roaming authenticator, such as a security key.

## Constructors

Create an instance of MFA WebAuthn Roaming Challenge screen manager:

```typescript Example
import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
await mfaWebAuthnRoamingChallenge.verify();
```

## Methods

<ParamField body="changeLanguage" type="Promise<void>">

This method changes the display language of the Universal Login page.

```typescript Example
import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
mfaWebAuthnRoamingChallenge.changeLanguage({
  language: 'fr',
});
```

</ParamField>

<ParamField body="reportWebAuthnError" type="Promise<void>">

This method reports a WebAuthn API error encountered during the `navigator.credentials.get()` operation.

```typescript Example
import MfaWebAuthnRoamingChallenge from '@auth0/auth0-acul-js/mfa-webauthn-roaming-challenge';

const mfaWebAuthnRoamingChallenge = new MfaWebAuthnRoamingChallenge();
await mfaWebAuthnRoamingChallenge.reportWebAuthnError({
  error: { name: 'NotAllowedError', message: 'The operation was not allowed.' },
});
```
