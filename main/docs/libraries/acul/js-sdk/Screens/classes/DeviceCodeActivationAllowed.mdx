title: "DeviceCodeActivationAllowed"

The DeviceCodeActivationAllowed class implements the `device-code-activation-allowed` screen functionality. This screen is displayed when the device code activation has been successfully authorized.

## Constructors

Create an instance of DeviceCodeActivationAllowed screen manager

```ts Example
import DeviceCodeActivationAllowed from "@auth0/auth0-acul-js/device-code-activation-allowed";
const deviceCodeActivationAllowedManager = new DeviceCodeActivationAllowed();
```
