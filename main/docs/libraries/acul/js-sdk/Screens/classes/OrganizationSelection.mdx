title: "OrganizationSelection"

The OrganizationSelection class implements the `organization-selection` screen functionality. This screen allows users to select an organization to continue with.

## Constructors

Create an instance of OrganizationSelection screen manager

```ts Example
import OrganizationSelection from '@auth0/auth0-acul-js/organization-selection';

const organizationSelection = new OrganizationSelection();
await organizationSelection.continueWithOrganizationName({
  organizationName: 'testOrganizationName',
});
```

## Methods

<ParamField body='continueWithOrganizationName' type='(options ?)'>

This method continues with the selected organization name.

```typescript Example
import OrganizationSelection from '@auth0/auth0-acul-js/organization-selection';

const organizationSelection = new OrganizationSelection();
await organizationSelection.continueWithOrganizationName({
  organizationName: 'testOrganizationName',
});
```
