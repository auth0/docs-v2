title: "MfaWebAuthnError"
description: "Describes all the properties and methods available to customize the Universal Login `mfa-webauthn-error` screen."
---

The MfaWebAuthnError class implements the `mfa-webauthn-error` screen functionality. This screen is displayed when an error occurs during a WebAuthn operation.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/4YEVrKX7HVo2LYWyZGxajt/865744113a4dbf5beff6bbce6587d8dc/Screenshot_2025-05-27_at_20.15.31.png" alt="MfaWebAuthnError" /></Frame>

## Constructors

Create an instance of MFA WebAuthn Error screen manager:

```typescript Example
import MfaWebAuthnError from '@auth0/auth0-acul-js/mfa-webauthn-error';

const mfaWebAuthnError = new MfaWebAuthnError();
await mfaWebAuthnError.tryAgain();
```

## Methods

<ParamField body="changeLanguage" type="Promise<void>">

This method changes the display language of the Universal Login page.

```typescript Example
import MfaWebAuthnError from '@auth0/auth0-acul-js/mfa-webauthn-error';

const mfaWebAuthnError = new MfaWebAuthnError();
mfaWebAuthnError.changeLanguage({
  language: 'fr',
});
```
