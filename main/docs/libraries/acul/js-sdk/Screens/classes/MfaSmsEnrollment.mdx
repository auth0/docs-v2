---
title: "MfaSmsEnrollment"
description: "Describes all the properties and methods available to customize the Universal Login `mfa-sms-enrollment` screen."
---

The MfaSmsEnrollment class implements the `mfa-sms-enrollment` screen functionality. This screen is displayed when the user needs to enroll a phone number for SMS-based MFA.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/5DG03Sm0FFKKt9JOxOl5WN/5df6ce785b080baa951b749b4c472793/Screenshot_2025-02-21_at_11.55.42.png" alt="MfaSmsEnrollment" /></Frame>

## Constructors

Create an instance of MFA SMS Enrollment screen manager:

```typescript Example
import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

const mfaSmsEnrollment = new MfaSmsEnrollment();
await mfaSmsEnrollment.continueEnrollment({ phone: '1234567890' });
```

## Methods

<ParamField body="changeLanguage" type="Promise<void>">

This method changes the display language of the Universal Login page.

```typescript Example
import MfaSmsEnrollment from '@auth0/auth0-acul-js/mfa-sms-enrollment';

const mfaSmsEnrollment = new MfaSmsEnrollment();
mfaSmsEnrollment.changeLanguage({
  language: 'fr',
});
```
