The MfaWebAuthnChangeKeyNickname class implements the `mfa-webauthn-change-key-nickname` screen functionality. This screen is displayed when the user needs to change the nickname of their WebAuthn security key.

## Constructors

Create an instance of MFA WebAuthn Change Key Nickname screen manager:

```typescript Example
import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
await mfaWebAuthnChangeKeyNickname.continueWithNewNickname({ nickname: 'My YubiKey' });
```

## Methods

<ParamField body="changeLanguage" type="Promise<void>">

This method changes the display language of the Universal Login page.

```typescript Example
import MfaWebAuthnChangeKeyNickname from '@auth0/auth0-acul-js/mfa-webauthn-change-key-nickname';

const mfaWebAuthnChangeKeyNickname = new MfaWebAuthnChangeKeyNickname();
mfaWebAuthnChangeKeyNickname.changeLanguage({
  language: 'fr',
});
```
