The BruteForceProtectionUnblockFailure class implements the `brute-force-protection-unblock-failure` screen functionality. This screen is displayed after an unsuccessful attempt to unblock an account that was blocked due to brute force protection.

  <img style={{maxHeight:"400px"}} src="https://mintlify.s3.us-west-1.amazonaws.com/auth0/docs/images/cdy7uua7fh8z/ACUL/brute-force-protection-unblock-failure.png" alt="ACUL Brute Force Protection Unblock Failure" />
</Frame>

## Constructors

Create an instance of BruteForceProtectionUnblockFailure screen manager:

```typescript Example
import BruteForceProtectionUnblockFailure from '@auth0/auth0-acul-js/brute-force-protection-unblock-failure';
const bruteForceProtectionUnblockFailureManager = new BruteForceProtectionUnblockFailure();
bruteForceProtectionUnblockFailureManager.changeLanguage({ language: 'en' });
```

## Methods

<ParamField body='changeLanguage' type='Promise<void>'>

This method changes the display language of the Universal Login page.

```typescript Example
import BruteForceProtectionUnblockFailure from '@auth0/auth0-acul-js/brute-force-protection-unblock-failure';
const bruteForceProtectionUnblockFailureManager = new BruteForceProtectionUnblockFailure();
bruteForceProtectionUnblockFailureManager.changeLanguage({
  language: 'fr',
});
```
