title: "PasskeyEnrollmentLocal"
description: "Describes all the properties and methods available to customize the Universal Login `passkey-enrollment-local` screen."
---

The PasskeyEnrollmentLocal class implements the `passkey-enrollment-local` screen functionality. This screen allows users to enroll a passkey stored locally on their device as an authentication method.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/7FZiX1waboFh7cd7HmKSwX/df095b993e61cb453b3e8db8dec81ce3/Passkey_Enrollment_Local.png" alt="Passkey Enrollment Local"/></Frame>

## Constructors

Create an instance of PasskeyEnrollmentLocal screen manager:

```typescript Example
import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
await passkeyEnrollmentLocalManager.continuePasskeyEnrollment();
```

## Methods

<ParamField body='abortPasskeyEnrollment' type='Promise<void>'>

This method aborts the passkey enrollment process.

```typescript Example
import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
await passkeyEnrollmentLocalManager.abortPasskeyEnrollment({ doNotShowAgain: true });
```

**Method Parameters**

<Expandable title="Parameters">
 
<ParamField body="options">
  [AbortEnrollmentOptions](/docs/libraries/acul/js-sdk/Screens/interfaces/AbortEnrollmentOptions).
</ParamField>

<ParamField body='doNotShowAgain?' type='boolean'>
  When set to `true`, hides the passkey enrollment prompt for future sessions.
</ParamField>

</Expandable>
</ParamField>

<ParamField body='changeLanguage' type='Promise<void>'>

This method changes the display language of the Universal Login page.

```typescript Example
import PasskeyEnrollmentLocal from '@auth0/auth0-acul-js/passkey-enrollment-local';
const passkeyEnrollmentLocalManager = new PasskeyEnrollmentLocal();
passkeyEnrollmentLocalManager.changeLanguage({
  language: 'fr',
});
```

**Method Parameters**

<Expandable title="Parameters">
 
<ParamField body="options">
  [LanguageChangeOptions](/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
</ParamField>

<ParamField body="language" type="string" required>
  The locale code for the desired language (for example, `'en'`, `'fr'`, `'es'`).
</ParamField>

<ParamField body="persist?" type='"session"'>
  When set to `'session'`, the selected language persists for the duration of the session.
</ParamField>

</Expandable>
</ParamField>
