---
title: "Confirmation"
description: "Describes all the properties and methods available to customize the Universal Login `confirmation` screen."
---

The Confirmation class implements the `confirmation` screen functionality. This screen is displayed after OTP verification when Auth0 detects an auto-signup opportunity, allowing the user to confirm or decline account creation.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/ACUL/confirmation.png" alt="Confirmation screen"/></Frame>

## Constructors

Create an instance of Confirmation screen manager:

```typescript Example
import Confirmation from '@auth0/auth0-acul-js/confirmation';
const confirmationManager = new Confirmation();
await confirmationManager.proceedToSignup();
```

## Methods

<ParamField body='proceedToSignup' type='Promise<void>'>

This method confirms auto-signup, proceeding with account creation following OTP verification.

```typescript Example
import Confirmation from '@auth0/auth0-acul-js/confirmation';
const confirmationManager = new Confirmation();
await confirmationManager.proceedToSignup();
```
</ParamField>

<ParamField body='goBack' type='Promise<void>'>

This method declines auto-signup, returning the user to the login screen without creating an account.

```typescript Example
import Confirmation from '@auth0/auth0-acul-js/confirmation';
const confirmationManager = new Confirmation();
await confirmationManager.goBack();
```
</ParamField>

<ParamField body='changeLanguage' type='Promise<void>'>

This method changes the display language of the Universal Login page.

```typescript Example
import Confirmation from '@auth0/auth0-acul-js/confirmation';
const confirmationManager = new Confirmation();
confirmationManager.changeLanguage({
  language: 'fr',
});
```
