The Logout class implements the `logout` screen functionality. This screen allows users to confirm or deny the logout action.

## Constructors

Creates an instance of the Logout screen manager.

```ts Example
import Logout from '@auth0/auth0-acul-js/logout';

const logout = new Logout();

await logout.confirmLogout({ action: 'accept' });
```

## Methods

<ParamField body='confirmLogout' type='(options ?)'/>

This method confirms the logout action (accept or deny).

```typescript Example
import Logout from '@auth0/auth0-acul-js/logout';

const logout = new Logout();

await logout.confirmLogout({ action: 'accept' });
```
