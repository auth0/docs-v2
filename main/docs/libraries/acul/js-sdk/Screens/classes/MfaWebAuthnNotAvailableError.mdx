The MfaWebAuthnNotAvailableError class implements the `mfa-webauthn-not-available-error` screen functionality. This screen is displayed when WebAuthn is not available or supported on the user's current device or browser.

## Methods

<ParamField body="changeLanguage" type="Promise<void>">

This method changes the display language of the Universal Login page.

```typescript Example
import MfaWebAuthnNotAvailableError from '@auth0/auth0-acul-js/mfa-webauthn-not-available-error';

const mfaWebAuthnNotAvailableError = new MfaWebAuthnNotAvailableError();
mfaWebAuthnNotAvailableError.changeLanguage({
  language: 'fr',
});
```
