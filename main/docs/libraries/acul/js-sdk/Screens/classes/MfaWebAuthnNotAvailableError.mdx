try {
  await sdk.tryAnotherMethod();

<ParamField body='tryAnotherMethod' type='(options ?)'>

This method allows the user to select a different Multi-Factor Authentication method because WebAuthn is not available or supported on their current device/browser. This action navigates the user to a screen where they can choose from other available/enrolled MFA factors. Returns a promise that resolves upon successful submission of the 'pick-authenticator' action.

```typescript Example
// Assuming 'sdk' is an instance of MfaWebAuthnNotAvailableError
try {
  await sdk.tryAnotherMethod();
  // On success, Auth0 typically handles redirection to the MFA factor selection screen.
} catch (error) {
  console.error('Failed to initiate "try another method":', error);
  // Update UI to inform the user about the failure to switch methods.
}
```
