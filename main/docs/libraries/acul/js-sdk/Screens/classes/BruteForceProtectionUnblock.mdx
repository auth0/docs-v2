---
title: "BruteForceProtectionUnblock"
description: "Describes all the properties and methods available to customize the Universal Login `brute-force-protection-unblock` screen."
---

The BruteForceProtectionUnblock class implements the `brute-force-protection-unblock` screen functionality. This screen is displayed when a user's account has been blocked due to multiple failed login attempts and they need to submit an unblock request.

<Frame>
  <img style={{maxHeight:"400px"}} src="https://mintlify.s3.us-west-1.amazonaws.com/auth0/docs/images/cdy7uua7fh8z/ACUL/brute-force-protection-unblock.png" alt="ACUL Brute Force Protection Unblock" />
</Frame>

## Constructors

Create an instance of BruteForceProtectionUnblock screen manager:

```typescript Example
import BruteForceProtectionUnblock from '@auth0/auth0-acul-js/brute-force-protection-unblock';
const bruteForceProtectionUnblockManager = new BruteForceProtectionUnblock();
await bruteForceProtectionUnblockManager.unblockAccount();
```

<ParamField body='changeLanguage' type='Promise<void>'>

This method changes the display language of the Universal Login page.

```typescript Example

const bruteForceProtectionUnblockManager = new BruteForceProtectionUnblock();
bruteForceProtectionUnblockManager.changeLanguage({
  language: 'fr',
});
```
