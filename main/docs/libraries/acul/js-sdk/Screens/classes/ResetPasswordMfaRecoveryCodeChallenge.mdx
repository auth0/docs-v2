title: "ResetPasswordMfaRecoveryCodeChallenge"
---

The ResetPasswordMfaRecoveryCodeChallenge class implements the `reset-password-mfa-recovery-code-challenge` screen functionality. This screen prompts the user to enter a recovery code as part of the MFA step during a password reset flow.

## Constructors

Create an instance of ResetPasswordMfaRecoveryCodeChallenge screen manager

```ts Example
import ResetPasswordMfaRecoveryCodeChallenge from '@auth0/auth0-acul-js/reset-password-mfa-recovery-code-challenge';
const resetPasswordMfaManager = new ResetPasswordMfaRecoveryCodeChallenge();
const continueWithRecoveryCode = async () => {
 try {
   await resetPasswordMfaManager.continue('RECOVERY_CODE');
    console.log('Recovery code submitted successfully.');
 } catch (error) {
   console.error('Error submitting recovery code:', error);
 }
};
```

## Methods

<ParamField body='continue' type='(options ?)'>

This method submits the recovery code to continue the MFA challenge during the password reset flow.

```typescript Example
import ResetPasswordMfaRecoveryCodeChallenge from '@auth0/auth0-acul-js/reset-password-mfa-recovery-code-challenge';
const resetPasswordMfaManager = new ResetPasswordMfaRecoveryCodeChallenge();
const continueWithRecoveryCode = async () => {
 try {
   await resetPasswordMfaManager.continue('RECOVERY_CODE');
    console.log('Recovery code submitted successfully.');
 } catch (error) {
   console.error('Error submitting recovery code:', error);
 }
};
```
