The ResetPasswordMfaRecoveryCodeChallenge class implements the `reset-password-mfa-recovery-code-challenge` screen functionality. This screen prompts the user to enter a recovery code as part of the MFA step during a password reset flow.

## Constructors

Create an instance of ResetPasswordMfaRecoveryCodeChallenge screen manager:

```typescript Example
import ResetPasswordMfaRecoveryCodeChallenge from '@auth0/auth0-acul-js/reset-password-mfa-recovery-code-challenge';
const resetPasswordMfaRecoveryCodeChallengeManager = new ResetPasswordMfaRecoveryCodeChallenge();
await resetPasswordMfaRecoveryCodeChallengeManager.continue('RECOVERY_CODE');
```

<ParamField body='continue' type='Promise<void>'>
