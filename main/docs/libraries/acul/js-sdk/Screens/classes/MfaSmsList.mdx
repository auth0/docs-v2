The MfaSmsList class implements the `mfa-sms-list` screen functionality. This screen is displayed when the user needs to select an enrolled phone number for SMS-based MFA.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/5WKzkwCtJn5ZViHcl9xATn/6147dac2ff98da4ab190922ced35a1f2/Screenshot_2025-02-21_at_12.29.36.png" alt="MfaSmsList" /></Frame>

## Constructors

Create an instance of MFA SMS List screen manager:

```typescript Example
import MfaSmsList from '@auth0/auth0-acul-js/mfa-sms-list';

const mfaSmsList = new MfaSmsList();
await mfaSmsList.selectPhoneNumber({ index: 0 });
```
