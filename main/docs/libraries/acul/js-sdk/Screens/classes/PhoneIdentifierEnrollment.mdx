---
title: "PhoneIdentifierEnrollment"
description: "Describes all the properties and methods available to customize the Universal Login `phone-identifier-enrollment` screen."
---

The PhoneIdentifierEnrollment class implements the `phone-identifier-enrollment` screen functionality. This screen allows users to enroll a phone number as an identifier by selecting their preferred verification delivery method.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/7kuQwDVmt0NkHCbQObLj8d/b690fd899f7c8525532d6d640298f2cd/PhoneIdentifierEnrollment.png" alt="PhoneIdentifierEnrollment"/></Frame>

## Constructors

Create an instance of PhoneIdentifierEnrollment screen manager:

```typescript Example
import PhoneIdentifierEnrollment from '@auth0/auth0-acul-js/phone-identifier-enrollment';
const phoneIdentifierEnrollmentManager = new PhoneIdentifierEnrollment();
await phoneIdentifierEnrollmentManager.continuePhoneEnrollment({
  type: 'voice',
});
```

## Methods

<ParamField body='changeLanguage' type='Promise<void>'>

This method changes the display language of the Universal Login page.

```typescript Example
import PhoneIdentifierEnrollment from '@auth0/auth0-acul-js/phone-identifier-enrollment';
const phoneIdentifierEnrollmentManager = new PhoneIdentifierEnrollment();
phoneIdentifierEnrollmentManager.changeLanguage({
  language: 'fr',
});
```

**Method Parameters**

<Expandable title="Parameters">
 
<ParamField body="options">
  [LanguageChangeOptions](/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
</ParamField>

<ParamField body="language" type="string" required>
  The locale code for the desired language (for example, `'en'`, `'fr'`, `'es'`).
</ParamField>

<ParamField body="persist?" type='"session"'>
  When set to `'session'`, the selected language persists for the duration of the session.
</ParamField>

</Expandable>
</ParamField>

<ParamField body='continuePhoneEnrollment' type='Promise<void>'>

```typescript Example

const phoneIdentifierEnrollmentManager = new PhoneIdentifierEnrollment();
await phoneIdentifierEnrollmentManager.continuePhoneEnrollment({
  type: 'voice',
```
