---
title: "ResetPasswordMfaOtpChallenge"
---

The ResetPasswordMfaOtpChallenge class implements the `reset-password-mfa-otp-challenge` screen functionality. This screen prompts the user to enter the one-time password (OTP) from their authenticator app as part of the MFA step during a password reset flow.

## Constructors

Create an instance of ResetPasswordMfaOtpChallenge screen manager

```ts Example
import ResetPasswordMfaOtpChallenge from '@auth0/auth0-acul-js/reset-password-mfa-otp-challenge';

const resetPasswordMfaOtpChallengeManager = new ResetPasswordMfaOtpChallenge();
await resetPasswordMfaOtpChallengeManager.continue({
  code: '123456',
});
```

## Methods

<ParamField body='continue' type='(options ?)'>

This method submits the OTP code to continue the MFA challenge during the password reset flow.

```typescript Example
import ResetPasswordMfaOtpChallenge from '@auth0/auth0-acul-js/reset-password-mfa-otp-challenge';

const resetPasswordMfaOtpChallengeManager = new ResetPasswordMfaOtpChallenge();
await resetPasswordMfaOtpChallengeManager.continue({
  code: '123456',
});
```
