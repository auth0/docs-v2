The LogoutAborted class implements the `logout-aborted` screen functionality. This screen is displayed when a user aborts the logout process.

## Constructors

Create an instance of LogoutAborted screen manager:

```ts Example
import LogoutAborted from '@auth0/auth0-acul-js/logout-aborted';
const logoutAbortedManager = new LogoutAborted();
logoutAbortedManager.changeLanguage({
  language: 'fr',
});
```

## Methods

<ParamField body="changeLanguage" type="Promise<void>">

This method changes the display language of the Universal Login page.

```typescript Example
import LogoutAborted from '@auth0/auth0-acul-js/logout-aborted';
const logoutAbortedManager = new LogoutAborted();
logoutAbortedManager.changeLanguage({
  language: 'fr',
});
```
