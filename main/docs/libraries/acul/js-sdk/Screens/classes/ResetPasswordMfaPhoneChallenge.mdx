title: "ResetPasswordMfaPhoneChallenge"
---

The ResetPasswordMfaPhoneChallenge class implements the `reset-password-mfa-phone-challenge` screen functionality. This screen allows the user to request a verification code via SMS or voice call as part of the MFA step during a password reset flow.

## Methods

<ParamField body='continue' type='(options ?)'>

This method sends a verification code to the user's phone via the selected delivery method to continue the MFA challenge during the password reset flow.

```typescript Example
import ResetPasswordMfaPhoneChallenge from '@auth0/auth0-acul-js/reset-password-mfa-phone-challenge';

const resetPasswordMfaPhoneChallengeManager = new ResetPasswordMfaPhoneChallenge();
await resetPasswordMfaPhoneChallengeManager.continue({
  type: 'sms',
});
```

<ParamField body="options" type="OPTIONAL">
Optional payload.

<Expandable title="options properties">

<ParamField body='type' type='"sms" | "voice"' required>
The delivery method for the verification code. Use `sms` to send via text message or `voice` to send via a voice call.
</ParamField>

<ParamField body='[`key`: `string`]' type='"string" | "number" | "boolean" | "undefined"'>
Additional data collected from the user.
</ParamField>

</Expandable>
</ParamField>
</ParamField>

<ParamField body='tryAnotherMethod' type='(options ?)'>

This method allows the user to select a different MFA method to complete the challenge.

```typescript Example
import ResetPasswordMfaPhoneChallenge from '@auth0/auth0-acul-js/reset-password-mfa-phone-challenge';

const resetPasswordMfaPhoneChallengeManager = new ResetPasswordMfaPhoneChallenge();
await resetPasswordMfaPhoneChallengeManager.tryAnotherMethod({
  type: 'sms',
});
```

<ParamField body="options" type="OPTIONAL">
Optional payload.

<Expandable title="options properties">

<ParamField body='type' type='"sms" | "voice"'>
The delivery method that was presented on the current screen. Required by the API to correctly process the method switch.
</ParamField>

<ParamField body='[`key`: `string`]' type='"string" | "number" | "boolean" | "undefined"'>
Additional data collected from the user.
</ParamField>

</Expandable>
</ParamField>
</ParamField>
