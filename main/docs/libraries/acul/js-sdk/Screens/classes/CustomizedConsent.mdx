---
title: "CustomizedConsent"
description: "Describes all the properties and methods available to customize the Universal Login `customized-consent` screen."
---

The CustomizedConsent class implements the `customized-consent` screen functionality. This screen prompts the user to accept or deny the permissions and authorization details requested by a client application.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/6cK6DkpOJYTjubOGu2vpID/80c03b32275a2c332fe3216df0eca5c7/Consent.png" alt=""/></Frame>

## Constructors

Create an instance of CustomizedConsent screen manager:

```typescript Example
import CustomizedConsent from '@auth0/auth0-acul-js/customized-consent';
const customizedConsentManager = new CustomizedConsent();
await customizedConsentManager.accept();
```

## Methods

<ParamField body='accept' type='Promise<void>'>

This method submits the user's decision to accept the requested permissions and authorization details.

```typescript Example
import CustomizedConsent from '@auth0/auth0-acul-js/customized-consent';
const customizedConsentManager = new CustomizedConsent();
await customizedConsentManager.accept();
```
</ParamField>
***

<ParamField body='changeLanguage' type='Promise<void>'>

This method changes the display language of the Universal Login page.

```typescript Example
import CustomizedConsent from '@auth0/auth0-acul-js/customized-consent';
const customizedConsentManager = new CustomizedConsent();
customizedConsentManager.changeLanguage({
  language: 'fr',
});
```
