The MfaWebAuthnRoamingEnrollment class implements the `mfa-webauthn-roaming-enrollment` screen functionality. This screen is displayed when a user needs to enroll a WebAuthn roaming authenticator, for example: a security key.

## Constructors

Create an instance of MFA WebAuthn Roaming Enrollment screen manager:

```typescript Example
import MfaWebAuthnRoamingEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-roaming-enrollment';

const mfaWebAuthnRoamingEnrollment = new MfaWebAuthnRoamingEnrollment();
await mfaWebAuthnRoamingEnrollment.enroll();
```

## Methods

<ParamField body="changeLanguage" type="Promise<void>">

This method changes the display language of the Universal Login page.

```typescript Example
import MfaWebAuthnRoamingEnrollment from '@auth0/auth0-acul-js/mfa-webauthn-roaming-enrollment';

const mfaWebAuthnRoamingEnrollment = new MfaWebAuthnRoamingEnrollment();
mfaWebAuthnRoamingEnrollment.changeLanguage({
  language: 'fr',
});
```

**Method Parameters**

<ParamField body="showError" type="Promise<void>">

This method submits details about a WebAuthn browser error to the server.

```typescript Example

const mfaWebAuthnRoamingEnrollment = new MfaWebAuthnRoamingEnrollment();

  error: { name: 'NotAllowedError', message: 'The operation was not allowed.' },
});
```
