The MfaEnrollResult class implements the `mfa-enroll-result` screen functionality. This screen displays the result of an MFA factor enrollment.

## Constructors

Create an instance of MFA Enroll Result screen manager:

```typescript Example
import MfaEnrollResult from '@auth0/auth0-acul-js/mfa-enroll-result';

const mfaEnrollResult = new MfaEnrollResult();
```

## Methods

<ParamField body='changeLanguage' type='Promise<void>'>

This method changes the display language of the Universal Login page.

```typescript Example
import MfaEnrollResult from '@auth0/auth0-acul-js/mfa-enroll-result';
const mfaEnrollResult = new MfaEnrollResult();
mfaEnrollResult.changeLanguage({
  language: 'fr',
});
```
