The MfaEnrollResult class implements the `mfa-enroll-result` screen functionality.

<Frame><img style={{maxHeight:"400px"}} src="/docs/images/cdy7uua7fh8z/lfXmpkRzf0XoXV12vYMe9/a426e0cce6d405a0314a59e79cb4f918/Screenshot_2025-02-20_at_17.10.00.png" alt="MfaEnrollResult"/></Frame>

## Constructors

Create an instance of MfaEnrollResult screen. It initializes the `BaseContext` and sets up the `screen` property with an instance of `ScreenOverride` tailored for this screen. 

```typescript Example
import MfaEnrollResult from '@auth0/auth0-acul-js/mfa-enroll-result';

const mfaEnrollResultScreen = new MfaEnrollResult();

// Access screen data
const enrollmentStatus = mfaEnrollResultScreen.screen.data?.status;
const pageTitle = mfaEnrollResultScreen.screen.texts?.title;

console.log(`MFA Enrollment Status: ${enrollmentStatus}`);
console.log(`Page Title: ${pageTitle}`);
```
