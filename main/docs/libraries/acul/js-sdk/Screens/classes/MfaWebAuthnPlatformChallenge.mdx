<ParamField body='reportBrowserError' type='(options ?)'>

This method reports a browser-side error encountered during the WebAuthn `navigator.credentials.get()` operation. This method should be called when `verify()` fails due to a WebAuthn API error (e.g., user cancellation). It sends the error details to the server with a specific action format. Returns a promise that resolves when the error report is successfully submitted.

```typescript Example
import MfaWebAuthnPlatformChallenge from "@auth0/auth0-acul-js/mfa-webauthn-platform-challenge";
const sdk = new MfaWebAuthnPlatformChallenge();

await sdk.reportBrowserError( {
  error: 'message'
});
```

<ParamField body="options" type="OPTIONAL"> 
Contains the error object (with `name` and `message` properties from the browser's WebAuthn API DOMException) and any other custom options.

<ParamField body='tryAnotherMethod' type='(options ?)'>

This method allows the user to opt-out of the WebAuthn platform challenge and select a different MFA method. This action submits `action: "pick-authenticator"` to Auth0, which should navigate the user to an MFA factor selection screen. Returns a promise that resolves when the 'pick-authenticator' action is submitted.

```typescript Example
import MfaWebAuthnPlatformChallenge from "@auth0/auth0-acul-js/mfa-webauthn-platform-challenge";
const sdk = new MfaWebAuthnPlatformChallenge();

await sdk.tryAnotherMethod();
```

<ParamField body='verify' type='(options ?)'>

This method initiates the WebAuthn platform authenticator challenge. Returns a promise that resolves when the credential submission is initiated.

```typescript Example
import MfaWebAuthnPlatformChallenge from "@auth0/auth0-acul-js/mfa-webauthn-platform-challenge";
const sdk = new MfaWebAuthnPlatformChallenge();

await sdk.verify();
```

<ParamField body="options" type="OPTIONAL"> 
Optional parameters for the verification. This can include `rememberDevice` if `this.screen.showRememberDevice` is true, and any other custom key-value pairs to be sent in the form submission.

<Expandable title="options properties">

<ParamField body='[`key`: `string`]' type='"string" | "number" | "boolean" | "undefined"' >
Additional data collected from the user.
</ParamField>

<ParamField body='rememberDevice?' type='boolean' >
If true, attempts to remember the browser for future MFA challenges. Corresponds to the `rememberBrowser` form field. This is only applicable if `screen.showRememberDevice` is true.
</ParamField>

</Expandable>
</ParamField>
</ParamField>
