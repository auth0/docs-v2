If you are a tenant administrator, you can revoke access to the Dashboard.

1. Go to [Dashboard > Settings > Tenant Members](https://manage.auth0.com/#/tenant/admins).
2. Click **...** to the right of a user name or email and click **Remove**.
3. Click **Save**.
