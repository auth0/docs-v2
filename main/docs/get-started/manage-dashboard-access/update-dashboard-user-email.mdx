As a tenant administrator, you can update the email address associated with an existing <Tooltip tip="Auth0 Dashboard: Auth0's main product to configure your services." cta="View Glossary" href="/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> member by adding a new user with the new email address and removing the user associated with the old email address.

1. Go to [Dashboard > Settings > Tenant Members](https://manage.auth0.com/#/tenants/admins).
2. Click **Add Member**.
3. Enter the new email address and select the access permissions.
4. Click **Invite** to send the email invitation to the user.
5. Once the user accepts the invitation, remove the user associated with the old email address.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

It is not possible to update your own email address as an end user in Dashboard or through the API. If you need to update the email address associated with your account, please contact your administrator.

</Callout>
