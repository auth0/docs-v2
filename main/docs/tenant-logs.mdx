The Tenant Log Catalog is a searchable reference for all Auth0 tenant log event types. Each entry includes the full schema, field definitions, and descriptions to help you understand what data Auth0 generates and build reliable integrations.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
Log retention periods vary by subscription level. To learn more, read [Log Data Retention](/docs/deploy-monitor/logs/log-data-retention).
</Callout>

## Use cases

Tenant logs record actions performed within your Auth0 tenant — authentications, administrative changes, errors, and more. Use the catalog to look up event type schemas when you need to:

* **Debug authentication failures** — Identify the exact event type and schema for a failure in your logs so you can interpret fields like `description` and `details` accurately.
* **Build log stream integrations** — Understand the shape of each event before building a consumer in a SIEM, data warehouse, or analytics platform.
* **Monitor security events** — Correlate specific event types (such as failed logins, anomaly detections, or MFA challenges) with your security monitoring workflows.
* **Meet compliance requirements** — Know which fields are present in each event type to make informed decisions about PII obfuscation and data handling.
