The `event` object for the event-stream Actions provides both information about an Event Stream message and Action execution context.

<table>
  <thead>
    <tr>
      <th>Property</th>
      <th>Description</th>
    </tr>
  </thead>
  <tbody>

    <tr>
      <td>
        <p><code>event.message</code></p>
      </td>
      <td>
        <p>An object containing the event stream message information following one of the <a href="/docs/customize/events/event-types">Event Type Schemas</a></p>
      </td>
    </tr>

    <tr>
      <td>
        <p><code>event.secrets</code></p>
        <p><em>(Optional)</em></p>
      </td>
      <td>
        <p>Secret values associated with the Action.</p>
        <p>Each secret value can be accessed using <code>event.secrets.key</code></p>
      </td>
    </tr>

  </tbody>
</table>
