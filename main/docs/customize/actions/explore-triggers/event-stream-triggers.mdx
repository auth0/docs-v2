Actions in this flow are non-blocking (asynchronous), which means the Auth0 pipeline will continue to run without waiting for the Action to finish its execution. Thus, the Actionâ€™s outcome does not affect the Auth0 transaction.

## Common use cases

### Notification

A credentials-exchange Action can be used to deny an access token based on custom logic.

```javascript lines
/**
* Handler to be executed while processing events in an Event Stream.
* @param {Event} event - Details about the incoming event.
* @param {EventStreamAPI} api - Methods and utilities to define event stream processing.
*/
exports.onExecuteEventStream = async (event, api) => {
  return;
};
```
