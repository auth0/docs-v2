<Tooltip tip="Universal Login: Your application redirects to Universal Login, hosted on Auth0's Authorization Server, to verify a user's identity." cta="View Glossary" href="/docs/glossary?term=Universal+Login">Universal Login</Tooltip> consists of a set of pages that facilitate several account-related actions, such as logging in, enrolling <Tooltip tip="Universal Login: Your application redirects to Universal Login, hosted on Auth0's Authorization Server, to verify a user's identity." cta="View Glossary" href="/docs/glossary?term=multi-factor+authentication">multi-factor authentication</Tooltip> factors, or resetting passwords. Auth0 provides the text on these pages in several languages.

In some cases, you may want to modify the text on these pages to better match your application's tone or specific needs.

You can customize the text displayed on [Universal Login](/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) pages in all supported languages through either the <Tooltip tip="Auth0 Dashboard: Auth0's main product to configure your services." cta="View Glossary" href="/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> or the <Tooltip tip="Auth0 Dashboard: Auth0's main product to configure your services." cta="View Glossary" href="/docs/glossary?term=Management+API">Management API</Tooltip>. The following definitions, values, and variables apply to both methods.

## Configure prompts and screens

You can customize any of the available [prompts and screens](#prompt-values) through the Auth0 Dashboard or the [Management API](https://auth0.com/docs/api/management/v2).

<Tabs><Tab title="Use the Dashboard">

You can use the Dashboard to customize New Universal Login text prompts.

1. In the Dashboard, go to [**Branding > Universal Login > Advanced Options**](https://manage.auth0.com/#/custom_text), and then select **Custom Text**.
2. Select the **Prompt** and **Screen** that you want to edit, then make your desired changes. A live preview is available on the right.
3. When you're ready for your changes to appear on your production environment, select **Save Changes.**

</Tab><Tab title="Use the Management API">

You can use the Management API to customize New Universal Login text prompts.

1. Get a [Management API access token](/docs/secure/tokens/access-tokens/management-api-access-tokens/get-management-api-access-tokens-for-production) with the `read:prompts` and `update:prompts` scopes.
2. Call the Management API [Get custom text for a prompt](https://auth0.com/docs/api/management/v2#!/Prompts/get_custom_text_by_language) endpoint to get the existing configuration data for the specified prompt and language.
3. Modify the configuration data as desired, using the [supported parameters](#supported-parameters) and [prompt values](#prompt-values) for reference.
4. Call the Management API [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) endpoint with the updated configuration data.

   <Warning>

   The Management API [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) endpoint replaces all existing configuration data for the specified prompt with the configuration data provided in the `body` parameter.

   If you want to configure multiple `screen` values for a specified prompt, you must include all of them in a single call to the Management API.

   </Warning>

#### Example

You could set the `description` text component for the `login` prompt's `login` screen to `Log in to ACME's website` by calling the Management API [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) endpoint like so:

<CodeGroup>
```bash cURL lines
curl --request PUT \
  --url 'https://{yourDomain}/api/v2/prompts/login/custom-text/en' \
  --header 'authorization: Bearer {yourMgmtApiAccessToken}' \
  --header 'content-type: application/json' \
  --data '{ "login": { "description": "Login to ACME'\''s Website" } }'
```
```csharp C# lines
var client = new RestClient("https://{yourDomain}/api/v2/prompts/login/custom-text/en");
var request = new RestRequest(Method.PUT);
request.AddHeader("content-type", "application/json");
request.AddHeader("authorization", "Bearer {yourMgmtApiAccessToken}");
request.AddParameter("application/json", "{ \"login\": { \"description\": \"Login to ACME's Website\" } }", ParameterType.RequestBody);
IRestResponse response = client.Execute(request);
```

```go Go lines expandable
package main

import (
	"fmt"
	"strings"
	"net/http"
	"io/ioutil"
)

func main() {

	url := "https://{yourDomain}/api/v2/prompts/login/custom-text/en"

	payload := strings.NewReader("{ \"login\": { \"description\": \"Login to ACME's Website\" } }")

	req, _ := http.NewRequest("PUT", url, payload)

	req.Header.Add("content-type", "application/json")
	req.Header.Add("authorization", "Bearer {yourMgmtApiAccessToken}")

	res, _ := http.DefaultClient.Do(req)

	defer res.Body.Close()
	body, _ := ioutil.ReadAll(res.Body)

	fmt.Println(res)
	fmt.Println(string(body))

}
```

#### Delete custom text for a prompt

If you want to delete all custom text for a prompt, call the Management API [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) endpoint with an empty object (`{}`) as the `body` parameter:

```bash cURL lines
curl --request PUT \
```
