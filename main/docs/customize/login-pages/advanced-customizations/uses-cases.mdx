Auth0 offers several options to customize your login experience. Review the options to find the best solution for your needs:

<Table>
| <b>Feature</b>  | <b>Universal Login</b> | <b> ACUL</b> | <b>Forms for Actions</b> | <b>Classic Login</b> | <b> Embedded</b> |
| —---------------- | —---------------- | —---------------- | —---------------- | —---------------- | —---------------- |
| <b>Primary Use Case</b> | Fast, simple | Advanced UI customization and branding | Post-login user flows | Legacy UI customization | Native in-app experience |
| <b>Code / No Code solution</b> | Code and No Code | Code | Code | Code and No Code | Code |
| <b>Hosted</b> | Auth0 | Auth0 | Auth0 | Auth0 | Self-hosted |
| </b>Customize other features or services, such as MFA or Bot Detection</b>| Yes | No | Yes | Yes | Yes |
| <b>UI Control</b> | Low control | High control | Low | | |
| <b>Bring your own technology (BYOT) </b> | No | Yes  | No | No | Yes |
| <b>Effort </b> | Low | Medium | Medium | Low | High |
| <b>Build custom UI logic</b> | No |  Yes | No | No | Yes |
</Table>
