Auth0 offers several options to customize your login experience. Review the options to find the best solution for your needs:

| Feature      | [Universal Login](/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) | [ACUL](/docs/customize/login-pages/advanced-customizations) | [Forms for Actions](/docs/customize/forms) | [Classic Login](/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/classic-experience) | [Embedded](/docs/authenticate/login/embedded-login) | 
| ----------- | ----------- | ----------- | ----------- | ----------- | ----------- |
| **Primary Use Case**	      | Fast, simple       | Advanced UI customization and branding | Post-login user flows | Legacy UI customization | Native in-app experience |
| **Code / No Code solution**	      | Code and No Code       | Code | Code | Code and No Code | Code |
| **Hosted**	      | Fast, simple       | Advanced UI customization and branding | Post-login user flows | Legacy UI customization | Native in-app experience |
| **Customize other features or services, such as MFA or Bot Detection**	      | Yes       | No | Yes | Yes | Yes | 
| **UI Control**	      | Low control      | High control | Low control |  |  |
| **Bring your own technology (BYOT)**	      | No      | Yes | No | No | Yes |
| **Effort**	      | Low       | Medium | Medium | Low | High |
| **Build custom UI logic**	      | No       | Yes | No | No | Yes |
