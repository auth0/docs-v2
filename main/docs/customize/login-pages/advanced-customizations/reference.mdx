## Auth0 ACUL SDKs

	* [Auth0 ACUL React SDK](/docs/customize/login-pages/advanced-customizations/reference/react-sdk)
	Provides React Hooks and components to simplify state management and integration within React applications.

	* [Auth0 ACUL JS SDK](/docs/customize/login-pages/advanced-customizations/reference/js-sdk)
	The core JavaScript library, suitable for use with any frontend framework or plain JavaScript.


## ACUL sample applications
	The ACUL sample applications include a full build pipeline, deployment workflows, and implementations for common Universal Login screens. You can use these sample applications to assist you in the development of your screens. 

	* **ACUL React sample**
		The ACUL React sample application includes samples of over 30 screens covering the entire authentication flow, including MFA, password reset, among others, using the ACUL React SDK. To learn more, read the [ACUL React sample application](https://github.com/auth0-samples/auth0-acul-samples) on GitHub. 

	* **ACUL JS sample** 
		The ACUL JS sample application includes samples of the 3 core authentication screens, `login`, `login-id`, and `login-password`, using the ACUL JS SDK. To learn more, read the [ACUL JS sample application](https://github.com/auth0-samples/auth0-acul-samples) on GitHub.

## ACUL resources
You can use the following external resources to assist you with your ACUL implementation:

	* **Auth0 Management API** 
	The following Auth0 Management API endpoints are available

		* Individual screen configuration: 
			* Use `GET` [/v2/prompts/{prompt}/screen/{screen}/rendering](/docs/api/management/v2/prompts/get-rendering) to retrieve the configuration for a single screen.
			* Use `PATCH` [/v2/prompts/{prompt}/screen/{screen}/rendering](/docs/api/management/v2/prompts/patch-rendering) to update the screen.

## Auth0 developer tools
Auth0 provides several tools to simplify tenant management including ACUL configuration.


		* **Auth0 CLI** 
		Use the acul commands to scaffold projects, local development, and to deploy your screen configurations. To learn more, read [Auth0 CLI](https://github.com/auth0/auth0-cli) on GitHub.   

		* **Auth0 Deploy CLI**  
		Manage your entire tenant configuration as code, including ACUL rendering settings defined in YAML files. To learn more, read [Auth0 Deploy CLI](https://github.com/auth0/auth0-deploy-cli) on GitHub.

		* **Auth0 Terraform provider** 
		Define your ACUL screen configurations . To learn more, read [promt_screen_renderer resource](https://registry.terraform.io/providers/auth0/auth0/latest/docs/resources/prompt_screen_renderer) on Terraform. 
 

## ACUL changelog
Review the changelog for track updates, new features, and breaking changes for the ACUL platform and the ACUL SDKs.
