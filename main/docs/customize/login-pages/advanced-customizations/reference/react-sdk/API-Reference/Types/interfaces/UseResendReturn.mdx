Return type for [useResend](/docs/customize/login-pages/advanced-customizations/reference/react-sdk/API-Reference/Hooks/useResend).

## Properties

<ParamField body='remaining' type='number'>

Seconds remaining until the next resend attempt is allowed.

</ParamField>

<ParamField body='disabled' type='boolean'>

Whether the resend action is currently disabled.

</ParamField>

<ParamField body='startResend()' type='void'>

Start a resend attempt immediately, if allowed.

</ParamField>
