Auth0 provides a WordPress Plugin to integrate your sites with your Auth0 account. This plugin enables <Tooltip tip="Single Sign-On (SSO): Service that, after a user logs into one applicaton, automatically logs that user in to other applications." cta="View Glossary" href="/docs/glossary?term=Single+Sign-on">Single Sign-on</Tooltip> (SSO) for Enterprises, social login, user/password, and <Tooltip tip="Single Sign-On (SSO): Service that, after a user logs into one applicaton, automatically logs that user in to other applications." cta="View Glossary" href="/docs/glossary?term=passwordless">passwordless</Tooltip> login through all your instances.

Login features are implemented through a new form (powered by Auth0) that enables:

* SSO with Enterprise Directories (LDAP, AD, Google Workspace, Office365, and <Tooltip tip="Security Assertion Markup Language (SAML): Standardized protocol allowing two parties to exchange authentication information without a password." cta="View Glossary" href="/docs/glossary?term=SAML">SAML</Tooltip> Provider)
* SSO with over 30 [Social Providers](/docs/authenticate/identity-providers)
* SSO between WordPress installs and other applications
* User Management Dashboard
* Optional <Tooltip tip="Multi-factor authentication (MFA): User authentication process that uses a factor in addition to username and password such as a code via SMS." cta="View Glossary" href="/docs/glossary?term=Multi-factor+Authentication">Multi-factor Authentication</Tooltip>
* Optional Passwordless Authentication
* Reporting and Analytics
