The following is a summary of available email templates and when the emails are sent:

| Email template name | Sent when... |
|---------------------|--------------|
| [Blocked account email](/docs/email/email-templates/email-template-reference/blocked-account) | A user is blocked due to suspicious login attempts |
| [Change password code email](/docs/email/email-templates/email-template-reference/change-password-code) | A user requests a password change |
| [Change password link email](/docs/email/email-templates/email-template-reference/change-password-link) | A user requests a password change |
| [MFA enrollment email](/docs/email/email-templates/email-template-reference/mfa-enrollment) | An admin sends an Auth0 Guardian <Tooltip tip="Multi-factor authentication (MFA): User authentication process that uses a factor in addition to username and password such as a code via SMS." cta="View Glossary" href="/docs/glossary?term=MFA">MFA</Tooltip> enrollment invitation |
| [MFA verification code email](/docs/email/email-templates/email-template-reference/mfa-verification-code) | A user using a MFA email verifier needs a MFA verification code |
| [Password breach alert email](/docs/email/email-templates/email-template-reference/password-breach-alert) | Auth0 detects that the user is trying to access the application with a password that has been leaked by a third party |
| [Passwordless OTP email](/docs/email/email-templates/email-template-reference/passwordless-otp) | A user logs in with <Tooltip tip="Passwordless: Form of authentication that does not rely on a password as the first factor." cta="View Glossary" href="/docs/glossary?term=Passwordless">Passwordless</Tooltip> email authentication |
| [User invitation email](/docs/email/email-templates/email-template-reference/user-invitation) | A user is invited to an organization or application |
| [Verification code email](/docs/email/email-templates/email-template-reference/verification-code) | A user needs to prove they have access to the email on file for an account |
| [Verification link email](/docs/email/email-templates/email-template-reference/verification-link) | A user signs up or logs in for the first time |
| [Welcome email](/docs/email/email-templates/email-template-reference/welcome) | A user verifies their email address (if enabled), or a user signs up or logs in for the first time (if disabled) |
