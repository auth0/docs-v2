If you enable the **Welcome Email** template, this email is sent when the user verifies their email address.

If you disable the **Welcome Email** template, this email is sent when the user signs up or logs in for the first time.

## Template-Specific Variables

This email template supports the [common Liquid variables](/docs/customize/email/email-templates/supported-liquid-syntax) available to all email templates.
