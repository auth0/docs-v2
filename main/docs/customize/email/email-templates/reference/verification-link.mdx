If you enable the **Verification Email (Link)** template, this email is sent when a user sign up on a database connection. The message prompts them to confirm their email address by clicking on the included URL.

When the user follows the verification link, the `email_verified` flag is set in the user's profile.

## Template-Specific Variables

In addition to the [common Liquid variables](/docs/customize/email/email-templates/supported-liquid-syntax) available to all email templates, the verification link email template has the following variables:

| Variable | Description |
|----------|-------------|
| `url`    | The verification link |
