If you enable the **Change Password (Link)** template, this email is sent when a user requests a password change.

The email contains a link that brings the user to the [password reset page](/docs/customize/login-pages/classic-login/customize-password-reset-page). The password is not changed until the user follows the link and enters a new password.

## Template-Specific Variables

In addition to the [common Liquid variables](/docs/customize/email/email-templates/supported-liquid-syntax) available to all email templates, this email template has the following variables:

* `url`, the password reset link
