## Get your WhatsApp token

Learn how to generate a token at [WhatsApp](https://developers.facebook.com/blog/post/2022/12/05/auth-tokens/).

## Configure your WhatsApp Vault connection

To configure a WhatsApp Vault Connection for your WhatsApp Flow Actions, you need to set the required parameters.

<Frame>![Dashboard > Actions > Forms > Vault > WhatsApp](/docs/images/customize/forms/vaults/whatsapp-vault-connection.png)</Frame>

### Input settings

| Parameter               | Description                                     |
|-------------------------|-------------------------------------------------|
| Token                   | Generated Token for Whatsapp API authorization. |
