## Get your Twilio credentials

Find them at [Twilio's console dashboard](https://console.twilio.com/).

## Configure your Twilio Vault connection

To configure a Twilio Vault Connection for your Twilio Flow Actions, you need to set the required parameters.

<Frame>![Dashboard > Actions > Forms > Vault > Twilio](/docs/images/customize/forms/vaults/twilio-vault-connection.png)</Frame>

### Input settings

| Parameter               | Description                               |
|-------------------------|-------------------------------------------|
| Account SID             | Account SID for Twilio API authorization. |
| Auth token              | Auth Token for Twilio API authorization.  |
