## Get your Telegram token

Learn how to generate a token at [Telegram's Bot API](https://core.telegram.org/#bot-api).

## Configure your Telegram Vault Connection

To configure a Telegram Vault Connection for your Telegram Flow Actions, you need to set the required parameters.

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/telegram-vault-connection.png)</Frame>

### Input settings

| Parameter               | Description                                     |
|-------------------------|-------------------------------------------------|
| Token                   | Generated Token for Telegram API authorization. |
