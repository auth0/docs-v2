## Get your MailJet keys

Find them at [MailJet Dashboard's](https://app.mailjet.com/account/api_keys) Account page under the API keys.

## Configure your MailJet Vault Connection

To configure a MailJet Vault Connection for your MailJet Flow Actions, you need to set the required parameters.

<Frame>![Dashboard > Actions > Forms > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

### Input settings

| Parameter               | Description                                |
|-------------------------|--------------------------------------------|
| API Key                 | API Key for MailJet API authorization.     |
| Secret Key              | Secret Key for MailJet API authorization.  |
