## Get your SendGrid keys

Find them at [SendGrid dashboard's](https://app.sendgrid.com/settings/api_keys) Settings page under the API keys.

## Configure your SendGrid Vault Connection

To configure a SendGrid Vault Connection for your SendGrid Flow Actions, you need to set the required parameters.

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

### Input settings

| Parameter               | Description                             |
|-------------------------|-----------------------------------------|
| API Key                 | API Key for SendGrid API authorization. |
