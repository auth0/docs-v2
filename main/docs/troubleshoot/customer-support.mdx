## Other support resources

* The [Auth0 status page](https://status.auth0.com) contains information on current production status. Please check the status page before filing a ticket.

    If the status page contains a notification about an outage, our team is already working on restoring service and will keep the status page updated. We publish a root-cause analysis to the status page once we have investigated the issue. You can subscribe to notifications of any changes.

* The [official Auth0 support center](https://support.auth0.com/) has troubleshooting information and solutions to many common issues.

* Auth0's public [question and answer community](https://community.auth0.com/) offers support for everyone. You can search through existing questions and post new questions if yours hasn't been asked yet. There are no guaranteed response times for questions posted.
