All customers, even those with free subscription plans, can report security concerns. Please read [Auth0's Vulnerability Disclose Policy](https://www.okta.com/content/dam/okta---digital/en_us/legal/okta-vulnerability-disclosure-policy_v2.0.pdf) for more information before submitting a security concern.

If you would like to publish your findings, you must submit the report directly to us at [psirt@okta.com](mailto:psirt@okta.com), preferably using our [GPG encryption key](https://www.okta.com/vulnerability-reporting-policy/okta-pgp-key/).

If you would like to be rewarded, the Bugcrowdâ€™s [standard disclosure terms](https://www.bugcrowd.com/resources/essentials/standard-disclosure-terms/) apply, and you must submit via [Bugcrowd](https://bugcrowd.com/okta) to be rewarded for your submission. For more information, visit [Auth0's bug bounty program](https://bugcrowd.com/auth0-okta).

For legal information, read [Legal Details](https://auth0.com/legal).
