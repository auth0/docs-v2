Using Forms for Actions, you can customize Auth0's services to fit your specific use case.
You can create and configure the following form use cases:

* **Progressive profiling** to define a set of rules to progressively collect user information.
* **Custom signup steps** to extend and customize the signup or login flows with steps and fields.
* [Configure an update policy form](https://sus.auth0.com/docs/contentful/7ZaefQOyKukQKq8p31lTS) to require consent to allow a user to login to your application.
