## Get Started

<Steps>
  <Step title="Create a new project" stepNumber={1}>
    Create a new Android project for this Quickstart

    Open Android Studio and create a new project:
    - Choose **"Empty Views Activity"** template
    - Set **Application name**: `Auth0Android`
    - Set **Package name**: `com.auth0.samples.auth0android`
    - Choose **Kotlin** as the language
    - Set **Minimum SDK**: API 21 (Android 5.0)

    Or using Android CLI:

    ```shellscript
    # Create new project directory
    mkdir auth0-android && cd auth0-android
    ```
  </Step>
  <Step title="Install the Auth0 Android SDK" stepNumber={2}>
    Add the Auth0 Android SDK to your app's `build.gradle` file:

    ```gradle app/build.gradle {5} lines
    dependencies {
        implementation 'androidx.core:core-ktx:1.12.0'
        implementation 'androidx.appcompat:appcompat:1.6.1'
        implementation 'com.google.android.material:material:1.11.0'
        implementation 'com.auth0.android:auth0:3.9.1'
        // ... other dependencies
    }
    ```

    Ensure your project targets Java 8+ for Android and Kotlin compatibility:

    ```gradle app/build.gradle lines
    android {
        compileOptions {
            sourceCompatibility JavaVersion.VERSION_1_8
            targetCompatibility JavaVersion.VERSION_1_8
        }

        kotlinOptions {
            jvmTarget = '1.8'
        }
    }
    ```

    Add the Internet permission to your `AndroidManifest.xml`:

    ```xml app/src/main/AndroidManifest.xml {2} lines
    <uses-permission android:name="android.permission.INTERNET" />
    
    <application
        android:allowBackup="true"
        android:theme="@style/Theme.Auth0Android">
        <!-- ... -->
    </application>
    ```
  </Step>
  <Step title="Setup your Auth0 App" stepNumber={3}>
    Create a new Native Application on your Auth0 tenant and configure your project:

    1. Go to the [Auth0 Dashboard](https://manage.auth0.com/dashboard/) and create a new application
    2. Choose **"Native"** as the application type
    3. Set a name like "My Android App"
    4. Note down your **Domain** and **Client ID** from the application settings

    Create `app/src/main/res/values/auth0.xml` with your Auth0 credentials:

    ```xml app/src/main/res/values/auth0.xml lines
    <resources>
        <string name="com_auth0_domain">YOUR_AUTH0_DOMAIN</string>
        <string name="com_auth0_client_id">YOUR_CLIENT_ID</string>
    </resources>
    ```

    Replace `YOUR_AUTH0_DOMAIN` and `YOUR_CLIENT_ID` with your actual values from the Auth0 Dashboard.
  </Step>
  <Step title="Configure callback URLs and manifest" stepNumber={4}>
    Configure the manifest placeholders in your app's `build.gradle` file:

    ```gradle app/build.gradle {8,9,10} lines
    android {
        compileSdk 35
        
        defaultConfig {
            applicationId "com.auth0.samples.auth0android"
            minSdk 21
            targetSdk 35
            // Add the manifest placeholders for Auth0
            manifestPlaceholders = [auth0Domain: "@string/com_auth0_domain", auth0Scheme: "https"]
        }
        
        compileOptions {
            sourceCompatibility JavaVersion.VERSION_1_8
            targetCompatibility JavaVersion.VERSION_1_8
        }
        
        kotlinOptions {
            jvmTarget = '1.8'
        }
    }
    ```

    <Warning>
      Make sure your **Allowed Callback URLs** and **Allowed Logout URLs** in the Auth0 Dashboard both include:
      `https://YOUR_AUTH0_DOMAIN/android/com.auth0.samples.auth0android/callback`
      
      Replace `YOUR_AUTH0_DOMAIN` with your actual domain and update the package name if different.
      
      **Important**: Both callback and logout URLs must use the same format for the authentication flow to work properly.
    </Warning>
  </Step>
  <Step title="Create Authentication Components" stepNumber={5}>
    Create the main authentication logic in your `MainActivity.kt`:

    ```kotlin app/src/main/java/com/auth0/samples/auth0android/MainActivity.kt lines
    package com.auth0.samples.auth0android

    import android.os.Bundle
    import android.widget.Toast
    import androidx.appcompat.app.AppCompatActivity
    import com.auth0.android.Auth0
    import com.auth0.android.authentication.AuthenticationException
    import com.auth0.android.callback.Callback
    import com.auth0.android.provider.WebAuthProvider
    import com.auth0.android.result.Credentials
    import com.auth0.samples.auth0android.databinding.ActivityMainBinding

    class MainActivity : AppCompatActivity() {
        
        private lateinit var binding: ActivityMainBinding
        private lateinit var auth0: Auth0
        private var isAuthenticated = false

        override fun onCreate(savedInstanceState: Bundle?) {
            super.onCreate(savedInstanceState)
            binding = ActivityMainBinding.inflate(layoutInflater)
            setContentView(binding.root)

            // Initialize Auth0
            auth0 = Auth0.getInstance(this)

            setupUI()
        }

        private fun setupUI() {
            updateUI()
            
            binding.loginButton.setOnClickListener {
                login()
            }
            
            binding.logoutButton.setOnClickListener {
                logout()
            }
        }

        private fun login() {
            WebAuthProvider.login(auth0)
                .withScheme("https")
                .withScope("openid profile email")
                .start(this, object : Callback<Credentials, AuthenticationException> {
                    override fun onFailure(exception: AuthenticationException) {
                        showMessage("Login failed: ${exception.message}")
                    }

                    override fun onSuccess(credentials: Credentials) {
                        isAuthenticated = true
                        showUserProfile(credentials)
                        updateUI()
                        showMessage("Login successful!")
                    }
                })
        }

        private fun logout() {
            WebAuthProvider.logout(auth0)
                .withScheme("https")
                .start(this, object : Callback<Void?, AuthenticationException> {
                    override fun onFailure(exception: AuthenticationException) {
                        showMessage("Logout failed: ${exception.message}")
                    }

                    override fun onSuccess(result: Void?) {
                        isAuthenticated = false
                        clearUserProfile()
                        updateUI()
                        showMessage("Logged out successfully!")
                    }
                })
        }

        private fun showUserProfile(credentials: Credentials) {
            // You can access user information from the ID token
            val userName = credentials.user.name ?: "Unknown User"
            val userEmail = credentials.user.email ?: "No email"
            
            binding.userInfo.text = """
                âœ… Successfully authenticated!
                
                Name: $userName
                Email: $userEmail
            """.trimIndent()
        }

        private fun clearUserProfile() {
            binding.userInfo.text = ""
        }

        private fun updateUI() {
            if (isAuthenticated) {
                binding.loginButton.visibility = android.view.View.GONE
                binding.logoutButton.visibility = android.view.View.VISIBLE
                binding.userInfo.visibility = android.view.View.VISIBLE
                binding.welcomeText.text = "Welcome to Auth0 Android!"
            } else {
                binding.loginButton.visibility = android.view.View.VISIBLE
                binding.logoutButton.visibility = android.view.View.GONE
                binding.userInfo.visibility = android.view.View.GONE
                binding.welcomeText.text = "Get started by signing in to your account"
            }
        }

        private fun showMessage(message: String) {
            Toast.makeText(this, message, Toast.LENGTH_LONG).show()
        }
    }
    ```
  </Step>

## Troubleshooting

<Accordion title="ðŸ”§ Common Issues and Solutions">
  **Browser doesn't redirect back to app**
  - Verify callback URLs are correctly set in the Auth0 Dashboard
  - Check that manifest placeholders match your Auth0 domain
  - Ensure `auth0Scheme` is set to "https" for Android App Links

  **Build errors with ViewBinding**
  - Make sure `buildFeatures { viewBinding true }` is in your app's build.gradle
  - Clean and rebuild project: `./gradlew clean build`

  **Authentication cancelled by user**
  - Handle the `isAuthenticationCanceled` property in your error callback:
  ```kotlin
  override fun onFailure(exception: AuthenticationException) {
      when {
          exception.isAuthenticationCanceled -> showMessage("Login was cancelled")
          exception.isBrowserAppNotAvailable -> showMessage("No browser available")
          else -> showMessage("Login failed: ${exception.message}")
      }
  }
  ```

  **No compatible browser error**
  - Install Chrome or another modern browser on your device/emulator
  - Check if the device has a default browser set

  **Logo not displaying**
  - Download the Auth0 logo and place it in `app/src/main/res/drawable/`
  - Or remove the ImageView from the layout if you prefer no logo
</Accordion>

## Advanced Features

<Accordion title="Secure Credentials Storage">
  For production apps, use the `SecureCredentialsManager` to automatically handle token storage and refresh:

  ```kotlin MainActivity.kt expandable lines
  import com.auth0.android.authentication.AuthenticationAPIClient
  import com.auth0.android.authentication.storage.CredentialsManagerException
  import com.auth0.android.authentication.storage.SecureCredentialsManager
  import com.auth0.android.authentication.storage.SharedPreferencesStorage

  class MainActivity : AppCompatActivity() {
      private lateinit var credentialsManager: SecureCredentialsManager
      
      override fun onCreate(savedInstanceState: Bundle?) {
          super.onCreate(savedInstanceState)
          
          // Initialize credentials manager
          val authentication = AuthenticationAPIClient(auth0)
          val storage = SharedPreferencesStorage(this)
          credentialsManager = SecureCredentialsManager(this, authentication, storage)
          
          // Check if user is already logged in
          checkExistingCredentials()
      }
      
      private fun checkExistingCredentials() {
          if (credentialsManager.hasValidCredentials()) {
              credentialsManager.getCredentials(object : Callback<Credentials, CredentialsManagerException> {
                  override fun onSuccess(credentials: Credentials) {
                      isAuthenticated = true
                      showUserProfile(credentials)
                      updateUI()
                  }
                  
                  override fun onFailure(exception: CredentialsManagerException) {
                      // Handle error - credentials might have expired
                      showMessage("Please log in again")
                  }
              })
          }
      }
      
      // Update your login success callback:
      private fun onLoginSuccess(credentials: Credentials) {
          credentialsManager.saveCredentials(credentials)
          isAuthenticated = true
          showUserProfile(credentials)
          updateUI()
          showMessage("Login successful!")
      }
      
      // Update your logout method:
      private fun logout() {
          credentialsManager.clearCredentials()
          WebAuthProvider.logout(auth0)
              .withScheme("https")
              .start(this, object : Callback<Void?, AuthenticationException> {
                  override fun onSuccess(result: Void?) {
                      isAuthenticated = false
                      clearUserProfile()
                      updateUI()
                      showMessage("Logged out successfully!")
                  }
                  
                  override fun onFailure(exception: AuthenticationException) {
                      showMessage("Logout failed: ${exception.message}")
                  }
              })
      }
  }
  ```
</Accordion>
