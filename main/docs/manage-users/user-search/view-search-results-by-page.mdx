To page the user search results, use the `page`, `per_page`, and `include_totals` parameters at your request.

<table class="table"><thead>
<tr>
<th>Parameter</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><code>page</code></td>
<td>The page number, zero based. When this is not set, we return a maximum of 50 records, regardless of how many records exist.</td>
</tr>
<tr>
<td><code>per_page</code></td>
<td>The amount of users per page.</td>
</tr>
<tr>
<td><code>include_totals</code></td>
<td>Set to <code>true</code> to include a query summary as part of the result.</td>
</tr>
</tbody>
</table>

## Limitation

Auth0 limits the total number of users you can retrieve to 1000 (for example, 100 users per page for 10 pages). When the `page` parameter is not set, we return a maximum of 50 records, regardless of how many records exist.
