---
title: "Déclencheurs d’actions : post-connexion – Objet d’événement"
'description': "Découvrez l’objet d’événement du déclencheur d’action post-connexion, qui fournit des informations contextuelles sur un utilisateur unique se connectant via Auth0."
---

L’objet `event` pour le déclencheur d’actions post-connexion fournit des informations contextuelles sur un utilisateur unique se connectant via Auth0.

| Propriété | Description |
| --- | --- |

Valeurs possibles :<br/><ul> <li>`low`</li> <li>`medium`</li> <li>`high`</li> <li>`neutral`</li> </ul> </li> <li> `details` *Objet facultatif*. Score de risque global<br/>Valeurs possibles :<br/><ul> <li>`low`</li> <li>`medium`</li> <li>`high`</li> <li>`neutral`</li> </ul> </li> <li>`version` *Chaîne*.</li> </ul> </li> </ul> |
