---
title: "Déclencheurs d’action : credentials-exchange - Objet d’événement"
'description': "Découvrez l’objet d’événement du déclencheur d’action de l’échange d’identifiants, qui fournit des informations contextuelles sur la demande d’échange d’identifiants pour un client."
---

L’objet `event` pour le déclencheur d’Actions d’échange d’identifiants fournit des informations contextuelles sur la demande d’échange d’identifiants d’un client.

| Propriété | Description |
| --- | --- |
| `event.accessToken` | Informations relatives au jeton d’accès à émettre.<br/>Comprend les propriétés suivantes :<br/><ul> <li>`customClaims` *Dictionnaire*.</li> <li>`scope` *Tableau de chaînes*.</li> </ul> |
| `event.client` | Informations sur le client utilisé lors de cet échange de jetons.<br/>Comprend les propriétés suivantes :<br/><ul> <li> `client_id` *Chaîne*. L’ID client de l’application dans laquelle l’utilisateur se connecte. </li> <li> `metadata` *Dictionnaire*. Un objet qui contient les autres propriétés d’une application. </li> <li> `name` *Chaîne*. Le nom de l’application (tel que défini dans le Dashboard). </li> </ul> |
| `event.organization` | Détails sur l’Organization associée à la transaction en cours.<br/>Comprend les propriétés suivantes :<br/><ul> <li> `display_name` *Chaîne*. Le nom amical de l’Organization. </li> <li> `id` *Chaîne*. L’identifiant de l’Organization. </li> <li> `metadata` *Dictionnaire*. Métadonnées associées à l’Organization. </li> <li> `name` *Chaîne*. Le nom de l’Organization. </li> </ul> |
| `event.request` | Détails sur la requête qui a initié la transaction.<br/>Comprend les propriétés suivantes :<br/><ul> <li> `body` *Dictionnaire*. Le corps de la requête POST. Ces données ne seront disponibles que lors de l’actualisation du jeton, des flux d’échange de jetons de type identifiant client et de l’action de pré-enregistrement d’utilisateur. </li> <li> `geoip` *Objet*. Comprend les propriétés suivantes :<br/><ul> <li>`cityName` *Chaîne facultative*.</li> <li>`continentCode` *Chaîne facultative*.</li> <li>`countryCode` *Chaîne facultative*.</li> <li>`countryCode3` *Chaîne facultative*.</li> <li>`countryName` *Chaîne facultative*.</li> <li>`latitude` *Numéro facultatif*.</li> <li>`longitude` *Numéro facultatif*.</li> <li>`subdivisionCode` *Chaîne facultative*.</li> <li>`subdivisionName` *Chaîne facultative*.</li> <li>`timeZone` *Chaîne facultative*.</li> </ul> </li> <li> `hostname` *Chaîne facultative*. Le nom d’hôte utilisé pour le flux d’authentification. </li> <li> `ip` *Chaîne*. L’adresse IP d’origine de la requête. </li> <li> `language` *Chaîne facultative*. La langue demandée par le navigateur. </li> <li> `method` *Chaîne de caractères*. La méthode HTTP utilisée pour la requête </li> <li> `user_agent` *Chaîne facultative*. La valeur de l’en-tête `User-Agent` reçu lors de l’initiation de la transaction. </li> </ul> |
| `event.resource_server` | Informations sur le serveur de ressources qui délivre le jeton d’accès.<br/>Comprend les propriétés suivantes :<br/><ul> <li> `identifier` *Chaîne*. L’identifiant du serveur de ressources. Par exemple : `https://your-api.example.com`. </li> </ul> |
| `event.tenant` | Informations sur le locataire utilisées lors de cet échange de jetons.<br/>Comprend les propriétés suivantes :<br/><ul> <li> `id` *Chaîne*. Le nom du locataire. </li> </ul> |
| `event.transaction` | Informations sur la transaction d’échange d’informations.<br/>Comprend les propriétés suivantes :<br/><ul> <li> `requested_scopes` *Tableau de chaînes*. Les permissions spécifiées (le cas échéant) lors de la demande du jeton d’accès. </li> </ul> |
