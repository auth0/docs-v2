---
title: "Déclencheurs d’action : post-change-password - Objet événement"
'description': "Découvrez l’objet d’événement du déclencheur d’action post-change-password, qui fournit des informations contextuelles sur l’utilisateur nouvellement créé."
---

L’objet `event` pour le déclencheur d’actions post-change-password fournit des informations contextuelles sur l’utilisateur nouvellement créé.

| Propriété | Description |
| --- | --- |
| `event.connection` | Information sur la connexion utilisée pour la transaction en cours.<br/>Comprend les propriétés suivantes :<br/><ul> <li> `id` *Chaîne*. L’identifiant unique de la connexion. </li> <li> `metadata` *Dictionnaire optionnel*. Métadonnées associées à la connexion. </li> <li> `name` *Chaîne*. Le nom de la connexion utilisée pour authentifier l’utilisateur (tel que : `twitter` ou `some-g-suite-domain`). </li> <li> `strategy` *Chaîne*. Le type de connexion. Pour les connexions sociales, `event.connection.strategy === event.connection.name`. Pour les connexions d’entreprise, la stratégie est la suivante : `waad` (Windows Entra ID), `ad` (Active Directory/LDAP), `auth0` (connexions aux bases de données), etc. </li> </ul> |
| `event.request` | Détails sur la requête qui a initié la transaction.<br/>Comprend les propriétés suivantes :<br/><ul> <li> `geoip` *Objet*.Contient des informations géographiques sur la requête. Comprend les propriétés suivantes :<br/><ul> <li>`cityName` *Chaîne facultative*.</li> <li>`continentCode` *Chaîne facultative*.</li> <li>`countryCode` *Chaîne facultative*.</li> <li>`countryCode3` *Chaîne facultative*.</li> <li>`countryName` *Chaîne facultative*.</li> <li>`latitude` *Numéro facultatif*.</li> <li>`longitude` *Numéro facultatif*.</li> <li>`subdivisionCode` *Chaîne facultative*.</li> <li>`subdivisionName` *Chaîne facultative*.</li> <li>`timeZone` *Chaîne facultative*.</li> </ul> </li> <li> `hostname` *Chaîne facultative*. Le nom d’hôte utilisé pour le flux d’authentification. </li> <li> `ip` *Chaîne*. L’adresse IP d’origine de la requête. </li> <li> `language` *Chaîne facultative*. La langue demandée par le navigateur. </li> <li> `method` *Chaîne*. La méthode HTTP utilisée pour la requête </li> <li> `user_agent` *Chaîne facultative*. La valeur de l’en-tête `User-Agent` reçu lors de l’initiation de la transaction. </li> </ul> |
| `event.tenant` | Détails sur le locataire associé à la transaction en cours.<br/>Comprend les propriétés suivantes :<br/><ul> <li> `id` *Chaîne*. Le nom du locataire. </li> </ul> |
| `event.user` | Objet décrivant l’utilisateur au nom duquel la transaction en cours a été initiée.<br/>Comprend les propriétés suivantes :<br/><ul> <li> `email` *Chaîne facultative*. (unique) Adresse courriel de l’utilisateur. </li> <li> `email_verified` *Booléen facultatif*. Indique si l’utilisateur a vérifié son adresse courriel. </li> <li> `last_password_reset` *Chaîne facultative*. Horodatage indiquant la dernière fois que le mot de passe de l’utilisateur a été réinitialisé/modifié. Lors de la création de l’utilisateur, ce champ n’existe pas. Cette propriété n’est disponible que pour les connexions aux bases de données. </li> <li> `phone_number` *Chaîne facultative*. (unique) Numéro de téléphone de l’utilisateur. </li> <li> `phone_verified` *Booléen facultatif*. Indique si l’utilisateur a vérifié son numéro de téléphone. </li> <li> `user_id` *Chaîne facultative*. (unique) Identifiant unique de l’utilisateur. </li> <li> `username` *Chaîne facultative*. (unique) Nom d’utilisateur de l’utilisateur. </li> </ul> |
