'description': "Déclencheurs d’inscription et de connexion"
---

Pour plus d’informations sur les déclencheurs d’inscription et de connexion, voir les sections ci-dessous.

| Déclencheur | Description |
| --- | --- |
| [Déclencheur de connexion](https://auth0.com/docs/fr-ca/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger) | Les déclencheurs de connexion s’exécutent lorsqu’un utilisateur se connecte à une application dans un locataire Auth0. |
| [Déclencheur avant l’enregistrement de l’utilisateur](https://auth0.com/docs/fr-ca/customize/actions/explore-triggers/signup-and-login-triggers/pre-user-registration-trigger) | Le déclencheur avant l’enregistrement de l’utilisateur s’exécute avant qu’un utilisateur ne soit ajouté à une base de données ou à une connexion sans mot de passe. |
| [Déclencheur après l’enregistrement de l'utilisateur](https://auth0.com/docs/fr-ca/customize/actions/explore-triggers/signup-and-login-triggers/post-user-registration-trigger) | Le déclencheur après l’enregistrement de l'utilisateur s’exécute après qu’un utilisateur a été ajouté à une base de données ou à une connexion sans mot de passe. |
