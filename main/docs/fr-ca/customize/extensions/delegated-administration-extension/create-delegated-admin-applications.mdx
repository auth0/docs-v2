title: "Créer des applications administratives déléguées"

---

Utilisez la [Extension d’administration déléguée](/docs/fr-ca/customize/extensions/delegated-administration-extension) pour présenter la section **Utilisateurs** de <Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip> à un groupe d’utilisateurs sélectionnés sans leur permettre d’accéder au reste du tableau de bord. Avant d’ajouter l’extension d’administration déléguée, vous devez créer l’application d’administration déléguée dans Auth0.

* Naviguez vers [Auth0 Dashboard > Applications > Applications](https://manage.auth0.com/#/applications) et sélectionnez **Créer une application**.
* Entrez un nom descriptif pour votre application (par exemple, **Tableau de bord des utilisateurs**), sélectionnez un type **Application Web à page unique**, puis cliquez sur **Créer**.
* Dans la section **Settings (Paramètres)** définissez les **URL de rappel autorisées** et les **URL de déconnexion autorisée)** en fonction de votre localisation, puis cliquez sur **Enregistrer les modifications**.

**URL de rappel par région :**

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

Remplacez `{yourTenant}` par votre nom de locataire. Par exemple, si le nom de votre locataire est `tenant_name_example` , alors pour un locataire fonctionnant dans un environnement US-3, l’URL de rappel correspondante est : `https://tenant_name_example.us.webtask.run/auth0-delegated-admin/login`

</Callout>

| Emplacement | Région | Lien de connexion |
| --- | --- | --- |
| Australie | AU | `https://{yourTenant}.au.webtask.io/auth0-delegated-admin/login` |
| Europe | EU | `https://{yourTenant}.eu.webtask.io/auth0-delegated-admin/login` |
| Europe | EU-2 | `https://{yourTenant}.eu.webtask.run/auth0-delegated-admin/login` |
| Japon | JP-1 | `https://{yourTenant}.jp.webtask.run/auth0-delegated-admin/login` |
| UK | UK | `https://{yourTenant}.uk.webtask.run/auth0-delegated-admin/login` |
| États-Unis | US-1 | `https://{yourTenant}.us.webtask.io/auth0-delegated-admin/login` |
| États-Unis | US-3 | `https://{yourTenant}.us.webtask.run/auth0-delegated-admin/login` |

Ensuite, [installez Extension d’administration déléguées](/docs/fr-ca/customize/extensions/delegated-administration-extension/install-delegated-admin-extension).
