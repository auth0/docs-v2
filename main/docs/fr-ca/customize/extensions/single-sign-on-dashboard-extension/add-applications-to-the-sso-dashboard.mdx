Utilisez l’extension Dashboard à authentification unique pour gérer la connexion <Tooltip href="/docs/fr-ca/glossary?term=single-sign-on" tip="Authentification unique (SSO)
Service qui, après qu’un utilisateur se soit connecté à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire">SSO</Tooltip> de vos utilisateurs sur plusieurs applications d’entreprise.

## Prérequis

* [Créez l’application SSO Dashboard](/docs/fr-ca/dashboard/guides/extensions/sso-dashboard-create-app).
* [Installez l’extension SSO Dashboard](/docs/fr-ca/dashboard/guides/extensions/sso-dashboard-install-extension).

## Ajouter les applications

1. Allez à [Auth0 Dashboard > Extensions](https://manage.auth0.com/#/extensions), et sélectionnez votre nouvelle extension Dashboard SSO.
   Si vous êtes un administrateur, vous pouvez également vous connecter au Dashboard SSO en utilisant `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
2. Dans le coin droit supérieur, sélectionnez **Paramètres** à partir du menu déroulant situé sous votre nom de locataire.
3. Cliquez sur **Créer application** pour ajouter une nouvelle application.

   Le formulaire **Nouvelle application** s’affiche.
4. Remplissez les champs suivant pour la nouvelle application :

   | Champ | Description |
| --- | --- |
|   |   |
| **Type** | Une liste déroulante dans laquelle vous pouvez sélectionner SAML, OpenID-Connect ou WS-Federation selon le type d’application. |
| **Application** | Une liste déroulante dans laquelle vous pouvez sélectionner l’application que vous souhaitez ajouter. |
| **Nom** | Le nom est automatiquement renseigné en fonction de l’application que vous avez sélectionnée. Vous pouvez modifier le nom ou utiliser le nom par défaut. |
| **Logo** | L’URL du logo que vous souhaitez utiliser comme icône pour l’application. |
| **Callback** | L’une des **URL de rappel autorisées** dans les [Paramètres de l’application](https://manage.auth0.com/#/applications) pour l’application. |
| **Connexion** | (*Facultatif*) Le type de connexion. Vous pouvez ajouter ou modifier vos types de connexion disponibles dans la [section Connexions du tableau de bord de Auth0 Management](https://manage.auth0.com/#/connections/database). Si aucune connexion n’est définie et que l’utilisateur n’est pas connecté, l’utilisateur verra la page de connexion Auth0. |
| **URL personnalisée** | (*Facultatif*) Case à cocher pour utiliser une URL personnalisée plutôt que l’URL Auth0. Si vous cochez la case, un champ apparaît dans lequel vous pouvez saisir votre URL personnalisée. |
| **Activé** | Case à cocher pour que cette application soit visible (publiée) par vos utilisateurs. |
5. Sélectionnez **Créer**. Votre nouvelle application s’afficher ensuite sur la page **Applications** du SSO Dashboard avec les autres applications qui auront été créées.
6. Cliquez sur une application pour tester la connexion.
