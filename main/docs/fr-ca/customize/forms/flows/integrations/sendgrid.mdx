## Configurer la connexion à votre coffre-fort

Pour configurer une connexion au coffre-fort pour vos actions SendGrid, vous aurez besoin d’une **clé API** :

<Frame>![Dashboard > Actions > Forms > Flows > actions > SendGrid](/docs/images/fr-ca/cdy7uua7fh8z/7qruolnilHd2RUvA3dGKfV/a278873b5b0564364bfdd9e15ca1d5cf/sendgrid-vault-connection.png)</Frame>

Celle-ci peut être trouvée sur la page Settings (Réglages) du [tableau de bord SendGrid](https://app.sendgrid.com/settings/api_keys) sous les clés API.

## Send email (Envoyer un courriel)

Envoyer un courriel en utilisant SendGrid comme fournisseur de messagerie.

<Frame>![](/docs/images/fr-ca/cdy7uua7fh8z/3pT7MZCTVhcL9ueeke0HqQ/8583aedc27af021fe6a62430c6d59adf/sendGrid.png)</Frame>

### Paramètres de saisie

| Paramètre | Description |
| --- | --- |
| Nom de l’expéditeur | Le nom de l’expéditeur |
| Courriel de l’expéditeur (obligatoire) | L’adresse courriel de l’expéditeur |
| Nom du destinataire | Le nom du destinataire |
| Adresse courriel du destinataire (obligatoire) | L’adresse courriel du destinataire |
| Objet (obligatoire) | L’objet du courriel |
| Type de courriel (obligatoire) | Envoyer un courriel avec contenu HTML ou un modèle enregistré (optionnel) |
| Message (obligatoire) | Message en format HTML ou un modèle (optionnel). |
| ID du modèle (facultatif) | Si vous utilisez un modèle, son ID peut être trouvé dans le [ tableau de bord SendGrid ](https://mc.sendgrid.com/dynamic-templates). |
| Variables personnalisées (facultatif) | Ajoutez des variables personnalisées à insérer dans votre modèle. |

### Paramètres de sortie

Vous trouverez les codes de réponse et d’état attendus dans la [documentation de SendGrid](https://www.twilio.com/docs/sendgrid/api-reference/mail-send/mail-send).
