</Columns>## Pourquoi utiliser les outils de développement Auth0?

Les outils de développement d'Auth0 vous aident à:

- **Automatiser les flux de travail de déploiement**: Intégrez la configuration Auth0 dans vos pipelines CI/CD
- **Contrôler la version de votre configuration**: Suivez et gérez les modifications apportées à votre locataire Auth0
- **Maintenir la cohérence entre les environnements**: Assurez-vous que les environnements de développement, de staging et de production restent synchronisés
- **Favoriser la collaboration**: Partagez et examinez les modifications de configuration de locataire avec votre équipe
- **Réduire les erreurs manuelles**: Automatisez les tâches de configuration répétitives

## Choisir le bon outil

### Auth0 CLI
Utilisez l'interface de ligne de commande Auth0 lorsque vous avez besoin:
- D'un accès rapide aux API Auth0 depuis la ligne de commande
- De gestion de locataire interactive pendant le développement
- De créer et configurer des applications, des API et d'autres ressources
- De script et d'automatisation pour les flux de travail de développement

### Fournisseur Terraform
Utilisez le fournisseur Terraform lorsque vous:
- Utilisez déjà Terraform pour la gestion de l'infrastructure
- Avez besoin d'un contrôle précis sur des ressources spécifiques
- Souhaitez gérer Auth0 aux côtés d'autres fournisseurs d'infrastructure
- Nécessitez des pratiques d'infrastructure en tant que code déclaratives

### Outil Deploy CLI
Utilisez l'outil Deploy CLI lorsque vous:
- Devez gérer la configuration en masse du locataire
- Souhaitez exporter et importer des configurations de locataire entières
- Configurez des flux de travail multi-environnements
- Nécessitez des capacités d'intégration continue et de déploiement
