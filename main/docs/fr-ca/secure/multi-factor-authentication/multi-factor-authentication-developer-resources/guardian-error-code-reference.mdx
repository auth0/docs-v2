---
title: "Référence du code d’erreur Guardian"
'description': "Affiche la liste des codes d'erreur Guardian et leur description."
---

Les codes d'erreur permettent d'afficher des messages d'information et de distinguer les erreurs récupérables des erreurs irrécupérables.

| Code d’erreur | Description |
| --- | --- |
| `invalid_token` | Demande ou jeton de transaction non valide |
| `insufficient_scope` | Vous n’avez pas assez d’autorisations pour effectuer l’opération demandée |
| `invalid_bearer_format` | L’en-tête d’authentification du porteur n’était pas valide |
| `enrollment_conflict` | Il y a une autre inscription pour le même utilisateur. Vous ne pouvez pas vous inscrire deux fois. |
| `tenant_not_found` | Le locataire associé est introuvable. Ne devrait pas normalement se produire à moins que vous ne supprimiez le locataire |
| `login_transaction_not_found` | La transaction d’authentification mfa n’est pas active ou a déjà expiré |
| `error_sending_push_notification` | Échec de livraison de la notification poussée |
| `push_notification_wrong_credentials` | Échec de livraison des notifications poussées à cause d’un identifiant erroné |
| `invalid_otp` | Le code otp fourni n’était pas valide |
| `invalid_recovery_code` | Le code de récupération fourni n’était pas valide |
| `invalid_body` | Échec de validation de corps. Mauvaise requête. |
| `invalid_query_string` | Échec de validation de la chaîne de requête. Mauvaise requête. |
| `enrollment_transaction_not_found` | La transaction d’inscription mfa n’est pas active ou a expiré |
| `invalid_phone_number` | Le numéro de téléphone fourni n’est pas valide |
| `error_sending_sms` | Erreur de livraison par SMS |
| `feature_disabled` | La fonctionnalité demandée n’est actuellement pas disponible dans le monde entier (contacter le fournisseur) |
| `feature_disabled_by_admin` | La fonctionnalité demandée est actuellement désactivée par votre administrateur |
| `pn_endpoint_disabled` | Nous n’avons pas été en mesure de livrer la notification poussée après avoir essayé plusieurs fois. Essayez de supprimer votre compte sur l’appareil et de l’ajouter à nouveau. |
| `too_many_sms` | Vous avez dépassé le nombre de SMS affectés à votre utilisateur |
| `too_many_pn` | Vous avez dépassé le nombre de notifications poussées attribuées à votre utilisateur |
| `too_many_sms_per_tenant` | Vous avez dépassé le nombre de SMS affectés à votre locataire |
| `too_many_pn_per_tenant` | Vous avez dépassé le nombre de notifications poussées affectées à votre locataire |
| `field_required` | Un champ est requis pour effectuer l’opération (cette erreur a un attribut de champ avec un code pour le champ: `otpCode`, `recoveryCode`) |
| `method_not_found` | Vous avez demandé une méthode qui n’est pas actuellement prise en charge (cela ne devrait pas se produire) |
| `no_method_available` | Il n’existe actuellement aucune méthode d’inscription (toutes sont désactivées) |
| `enrollment_method_disabled` | La méthode d’inscription spécifiée est désactivée, cette erreur comporte également un champ .method |
| `auth_method_disabled` | La méthode d’authentification spécifiée est désactivée, cette erreur comporte également un champ .method |
| `invalid_otp_format` | Erreur de validation du format OTP |
| `invalid_recovery_code_format` | Erreur de validation du code de récupération |
| `transaction_expired` | La transaction a déjà expiré |
| `already_enrolled` | Vous êtes déjà inscrit, vous ne pouvez pas vous inscrire de nouveau. |
| `not_enrolled` | Vous n’êtes pas inscrit. Doit d’abord s’inscrire |
| `invalid_enrollment` | L’inscription fournie à la méthode transaction#requestAuth n’est pas valide ou est nulle/indéfinie |
