---
title: "Surveiller les événements liés aux clés d’identification dans les journaux des locataires"
'description': "Comment identifier et surveiller les événements liés aux clés dans les journaux de vos locataires."
---

Vous pouvez surveiller les flux de clés de chiffrement associés à vos connexions de base de données par l’intermédiaire des [journaux des locataires](/docs/fr-ca/deploy-monitor/logs).

Vous pouvez afficher les événements de journaux dans <Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip> ([Surveillance> Journaux](https://manage.auth0.com/#/logs)) ou récupérer des journaux en utilisant [Management API](https://auth0.com/docs/api/management/v2/logs/get-logs).

| __ Code d’événement __ | Scénario | Notes |
| --- | --- | --- |
| `ss` | Lors de l’inscription, un utilisateur a enregistré une clé d’identification comme méthode d’authentification principale. | Cet événement de journal contient le champ de tableau `details.authentication_methods` avec la valeur d’entrée `passkey`.<br/><br/>Si un utilisateur s’est enregistré avec un mot de passe, ce champ de tableau contient la valeur `pwd`. |
| `s` | Lors de la connexion, un utilisateur s’est authentifié avec une clé d’identification. | Cet événement de journal contient le tableau `details.prompts` qui répertorie l’ensemble des invites qu’un utilisateur a parcourues lors de la connexion.<br/><br/>Si un utilisateur s’est authentifié à l’aide d’une clé d’identification, le tableau contient un élément avec le champ suivant : `"performed_amr": ["phr"]`. |
| `gd_enrollment_complete` | Un flux d’enregistrement progressif ou local a été complété avec succès. | Cet événement de journal contient le champ `details.authenticator.type` avec la valeur `passkey` si un utilisateur a inscrit une clé d’identification. |
| `gd_webauthn_enrollment_failed` | Une erreur s’est produite lors d’un flux d’enregistrement progressif ou local. | Cet événement de journal contient le champ `details.authentication_methods array` avec la valeur `passkey` si un utilisateur a essayé d’enregistrer une clé d’identification. |
