Vous pouvez récupérer l’objet d’options pour une [connexion](/docs/docs/fr-ca/connections) à l’aide de <Tooltip href="/docs/docs/fr-ca/glossary?term=management-api" tip="Management API
Un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire">Management API</Tooltip> Auth0.

Effectuez un appel `GET` au point de terminaison [Obtenir une connexion](/docs/docs/fr-ca/api/management/v2#!/Connections/get_connections_by_id). Veillez à remplacer les valeurs d’espace réservé `CONNECTION_ID` et `MGMT_API_ACCESS_TOKEN` par votre ID de connexion et votre jeton d’accès à Management API, respectivement.

<AuthCodeGroup>
```bash cURL
curl --request GET \
  --url 'https://{yourDomain}/api/v2/connections/CONNECTION-ID?fields=options' \
  --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN'
```
```csharp C#
var client = new RestClient("https://{yourDomain}/api/v2/connections/CONNECTION-ID?fields=options");
var request = new RestRequest(Method.GET);
request.AddHeader("authorization", "Bearer MGMT_API_ACCESS_TOKEN");
IRestResponse response = client.Execute(request);
```
```go Go
package main

import (
	"fmt"
	"net/http"
	"io/ioutil"
)

func main() {

	url := "https://{yourDomain}/api/v2/connections/CONNECTION-ID?fields=options"

	req, _ := http.NewRequest("GET", url, nil)

	req.Header.Add("authorization", "Bearer MGMT_API_ACCESS_TOKEN")

	res, _ := http.DefaultClient.Do(req)

	defer res.Body.Close()
	body, _ := ioutil.ReadAll(res.Body)

	fmt.Println(res)
	fmt.Println(string(body))

}
```
