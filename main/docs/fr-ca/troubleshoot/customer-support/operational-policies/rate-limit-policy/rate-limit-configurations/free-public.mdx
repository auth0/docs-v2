Voir ci-dessous les politiques de limite anti-attaques pour le type d’abonnement Gratuit.



<Accordion title="Authentication API Limites anti-attaques pour Authentication API et les points de terminaison d’API dans le type d’abonnement gratuit.">

| [API](/docs/fr-ca/troubleshoot/customer-support/operational-policies/rate-limit-policy#api-rate-limits) | [Burst Request Limit (Limite de requêtes en rafale)](/docs/fr-ca/troubleshoot/customer-support/operational-policies/rate-limit-policy#rate-limit-algorithm) | [Sustained Request Limit (Limite de requêtes soutenues)](/docs/fr-ca/troubleshoot/customer-support/operational-policies/rate-limit-policy#rate-limit-algorithm) |
| --- | --- | --- |
| Authentication API | 300 | 300/minute |

| Point de terminaison | Méthode | Chemin d’accès | Limite de requêtes en rafale | Limite de requêtes soutenues | Type de limite |
| --- | --- | --- | --- | --- | --- |
| User Info | GET, POST | `/userinfo` | 10 | 5 par/min | Vers un identifiant utilisateur unique |
| Modifier un mot de passe | POST | `/dbconnections/change_password` &amp; `u/reset-password/request/:connection` | 10 | 1 par/min | Depuis une adresse IP vers une adresse courriel unique |
| Inscription* | POST | `/dbconnections/signup` | 50 | 50 par/min | Depuis une adresse IP |
| Obtenir un code ou un lien à usage unique | GET, POST | `/passwordless/start` | 50 | 50 par/h | Depuis une adresse IP |
| Obtenir un jeton | POST | `/oauth/token` | 30 | 30 par/sec | Toute requête |
| Authentification cross-origin | POST | `co/authenticate` | 5 | 5 par/sec | Toute requête |
| Authentification | POST | `/usernamepassword/login` | 5 | 5▌par/sec | Toute requête |
| Clés de jeton Web JSON | GET | `/.well-known/jwks.json` | 20 | 20 par/sec | Toute requête |
| Connexion native via réseau social | POST | `/oauth/token` | 50 | 50 par/min | Toute requête pour une connexion native via réseau social avec Apple ou Facebook |
| [Inscription pour une application dynamique (Client)](https://auth0.com/docs/api/authentication#dynamic-application-client-registration) | POST | `/oidc/register` | 5 | 5 par/sec | Toute requête |

\*Représente la limite par défaut. Vous pouvez configurer la limite du point de terminaison Inscription dans Auth0 Dashboard. Pour en apprendre davantage, consutltez [Limitation du débit pour les adresses IP suspectes](https://www.auth0.com/docs/secure/attack-protection/suspicious-ip-throttling).

</Accordion>
