<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/docs/ja-jp/glossary?term=auth0-dashboard" tip="Auth0 Dashboard: サービスを構成するためのAuth0の主製品。" cta="用語集の表示">Auth0 Dashboard</Tooltip>を用いた[デバイス認可フロー](/docs/ja-jp/flows/concepts/device-auth)中に、アプリが生成したユーザーコードを構成することができます。

1. [［Dashboard］>［Settings（設定）］](https://manage.auth0.com/#/tenant)に移動して、**［Advanced（詳細設定）］** タブをクリックします。
2. **Device Flow User Code Format（デバイスフローのユーザーコード形式）** セクションまでスクロールし、**ユーザーコード文字セット** と**ユーザーコードマスク** を見つけて、必要な設定を入力し、**［Save（保存）］** をクリックします。

   | 設定 | 説明 |
   | --- | --- |
   | **Device Flow User Code Format（デバイスフローのユーザーコード形式）** | ユーザーコードをランダム生成するときに使用される文字セット。 |
   | **User Code Mask（ユーザーコードマスク）** | ランダム生成されたユーザーコードの長さと形式の定義に使用されるマスク。この目的はユーザーコードをさらに読みやすく、入力しやすくすることにあります。 |
