## Auth0 Dashboard

Auth0 Dashboardを使って組織メンバーからロールを削除するには、以下を行います。

1. [［Auth0 Dashboard］>［Organization］](https://manage.auth0.com/#/organizations)に移動して、メンバーシップを構成したい組織を選択します。
2. **［Members（メンバー）］** ビューを選択し、ロールを削除したいメンバーの名前をクリックします。
3. 削除したいロールを見つけて、ゴミ箱アイコンをクリックし、確認します。

## Management API

Management APIを使って組織メンバーからロールを削除するには、次を行います：
`DELETE`呼び出しを`Delete Organization Member Roles`（組織メンバーロール削除）エンドポイントに対して行います。`ORG_ID`、`MGMT_API_ACCESS_TOKEN`、`USER_ID`、`ROLE_ID`のプレースホルダーをそれぞれ組織ID、管理APIアクセストークン、ユーザID、ロールIDで置き換えます。

  --header 'cache-control: no-cache' \

**Auth0ドメインを見つける**

テナント名がAuth0ドメインである場合は、地域のサブドメイン（テナントが米国地域にあり、2020年6月より前に作成された場合を除く）の後に「`.auth0.com`」が続きます。たとえば、テナント名が「`travel0`」の場合、Auth0のドメイン名は「`travel0.us.auth0.com`」になります。（テナントが米国内にあって、2020年6月よりも前に作成された場合、ドメイン名は「`https://travel0.auth0.com`」になります。）

カスタムドメインを使用している場合には、これがカスタムドメイン名になります。
