post-change-passwordアクショントリガーの`event`オブジェクトは、新規作成されたユーザーに関するコンテキスト情報を提供します。

| プロパティ | 説明 |
| --- | --- |
| `event.connection` | 現在のトランザクションに使用された 接続についての詳細。<br/>次のプロパティを含みます：<br/><ul> <li>`id` *文字列*。接続の一意の識別子。</li> <li> `metadata` *任意の辞書*。接続に関連したメタデータ。 </li> <li> `name` *文字列*。ユーザーの認証に使用される接続名（ `twitter`または`some-g-suite-domain`）。 </li> <li> `strategy` *文字列*。接続のタイプ。ソーシャル接続の場合、 `event.connection.strategy === event.connection.name`。エンタープライズ接続では、 `waad`（Microsoft Entra ID）、`ad`（Active Directory/LDAP）、`auth0`（データベース接続）など です。 </li> </ul> |
| `event.request` | トランザクションを開始した要求の詳細。<br/>次のプロパティを含みます：<br/><ul> <li> `geoip` *オブジェクト* 要求の地理情報を含みます。次のプロパティを含みます：<br/><ul> <li>`cityName` *任意の文字列*。</li> <li>`continentCode` *任意の文字列*。</li> <li>`countryCode` *任意の文字列*。</li> <li>`countryCode3` *任意の文字列*。</li> <li>`countryName` *任意の文字列*。</li> <li>`latitude` *任意の数字*。</li> <li>`longitude` *任意の数字*。</li> <li>`subdivisionCode` *任意の文字列*。</li> <li>`subdivisionName` *任意の文字列*。</li> <li>`timeZone` *任意の文字列*。</li> </ul> </li> <li>`hostname` *任意の文字列*。認証フローに使用されているホスト名。</li> <li> `ip` *文字列*。要求の発生元IPアドレス。 </li> <li> `language` *任意の文字列*。ブラウザーから要求された言語。 </li> <li> `method` *文字列*。要求に使用されたHTTPメソッド </li> <li> `user_agent` *任意の文字列*。トランザクションの開始時に受け取った`User-Agent`の ヘッダーの値。 </li> </ul> |
| `event.tenant` | 現在のトランザクションに関連するテナントの詳細。<br/>次のプロパティを含みます：<br/><ul> <li>`id` *文字列*。テナント名。</li> </ul> |
| `event.user` | 現在のトランザクションが開始されたユーザーを表すオブジェクト。<br/>次のプロパティを含みます：<br/><ul> <li> `email` *任意の文字列*。（一意）ユーザーのメールアドレス。 </li> <li>`email_verified` *任意のブール値*。ユーザーがメールアドレスを確認済みかどうかを示します。</li> <li> `last_password_reset` *任意の文字列*。ユーザーパスワードがリセット/変更された最終日時を示す タイムスタンプ。ユーザーの作成時に、このフィールドは存在しません。この プロパティはデータベース接続にのみ利用できます。 </li> <li> `phone_number` *任意の文字列*。 （一意）ユーザーの電話番号。 </li> <li> `phone_verified` *任意のブール値*。ユーザーが電話番号を検証済みかを示します。 </li> <li> `user_id` *任意の文字列*。 （一意）ユーザーの一意の識別子。 </li> <li> `username` *任意の文字列*。 （一意）ユーザーのユーザー名。 </li> </ul> |
