'description': "サインアップとログインのトリガー"

サインアップとログインのトリガーの詳細情報は、以下のセクションを参照してください。

| トリガー | 説明 |
| --- | --- |
| [ログイントリガー](https://auth0.com/docs/ja-jp/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger) | ログイントリガーは、ユーザーがAuth0テナント内の任意のアプリケーションにログインしたときに実行されます。 |
| [ユーザー登録前トリガー](https://auth0.com/docs/ja-jp/customize/actions/explore-triggers/signup-and-login-triggers/pre-user-registration-trigger) | ユーザー登録前トリガーは、ユーザーがデータベースまたはパスワードレス接続に追加される前に実行されます。 |
| [ユーザー登録後トリガー](https://auth0.com/docs/ja-jp/customize/actions/explore-triggers/signup-and-login-triggers/post-user-registration-trigger) | ユーザー登録後トリガーは、ユーザーがデータベースまたはパスワードレス接続に追加された後に実行されます。 |
