---

pre-user-registration アクショントリガーの`event`オブジェクトは、新規ユーザーの登録要求に関するコンテキスト情報を提供します。

| プロパティ | 説明 |
| --- | --- |
| `event.client`<br/>*（任意）* | このトランザクションが開始されたクライアントについての 情報。<br/>次のプロパティを含みます：<br/><ul> <li> `client_id` *文字列*。 ユーザーがログインしているアプリケーションのクライアントID。 </li> <li> `metadata` *辞書*。 別のアプリケーションプロパティを保持するためのオブジェクト。 </li> <li> `name` *文字列*。 アプリケーションの名前（Dashboardで定義された通り）。 </li> </ul> |
| `event.connection` | ユーザー登録に使用された接続の詳細。<br/>次のプロパティを含みます：<br/><ul> <li> `id` *文字列*。接続の一意の識別子。 </li> <li> `metadata` *任意の辞書*。 接続に関連したメタデータ。 </li> <li> `name` *文字列*。 ユーザーの認証に使用される接続名（ `twitter`または`some-g-suite-domain`など）。 </li> <li> `strategy` *文字列*。 接続のタイプ。ソーシャル接続の場合、 `event.connection.strategy === event.connection.name`。エンタープライズ接続では、 `waad`（Microsoft Entra ID）、`ad`（Active Directory/LDAP）、`auth0`（データベース接続）など です。 </li> </ul> |
| `event.request` | トランザクションを開始した要求の詳細。<br/>次のプロパティを含みます：<br/><ul> <li> `body` *辞書*。 POST要求のボディ。このデータは、 リフレッシュトークン、クライアント資格情報交換フロー、 PreUserRegistrationアクション中のみ利用できます。 </li> <li> `geoip` *オブジェクト*。次のプロパティを含みます：<br/><ul> <li> `cityName` *任意の文字列*。 </li> <li> `continentCode` *任意の文字列*。 </li> <li> `countryCode` *任意の文字列*。 </li> <li> `countryCode3` *任意の文字列*。 </li> <li> `countryName` *任意の文字列*。 </li> <li> `latitude` *任意の数字*。 </li> <li> `longitude` *任意の数字*。 </li> <li> `subdivisionCode` *任意の文字列*。 </li> <li> `subdivisionName` *任意の文字列*。 </li> <li> `timeZone` *任意の文字列*。 </li> </ul> </li> <li> `hostname` *任意の文字列*。 認証フローに使用されている ホスト名。 </li> <li> `ip` *文字列*。要求の発生元IPアドレス。 </li> <li> `language` *任意の文字列*。 ブラウザーから要求された言語。 </li> <li> `method` *文字列*。要求に使用されたHTTPメソッド </li> <li> `user_agent` *任意の文字列*。 トランザクションの開始時に受け取った`User-Agent`の ヘッダーの値。 </li> </ul> |
| `event.tenant` | 現在のトランザクションに関連するテナントの詳細。<br/>次のプロパティを含みます：<br/><ul> <li> `id` *文字列*。テナント名。 </li> </ul> |
| `event.transaction`<br/>*（任意）* | 現在のトランザクションについての詳細。<br/>次のプロパティを含みます：<br/><ul> <li> `acr_values` *文字列の配列*。 元の認証要求に提供されたすべてのacr_values。 </li> <li> `locale` *文字列*。 このトランザクションに使われるロケールで、 ブラウザーの要求言語とテナントの言語設定を 比較して決定される。 </li> <li> `login_hint` *任意の文字列*。 エンドユーザーが必要な場合にログインに使用する可能性のある、 ログイン識別子に関する認可サーバーへのヒント。 </li> <li> `prompt` *任意の文字列の配列*。 ユーザーに再認証と同意を求めるかどうかを示す 指示のリスト。 </li> <li> `protocol` *任意の文字列*。 含まれる可能性のある値は以下の通りです：<br/><ul> <li> `oidc-basic-profile` 最も使用される、Webベースのログイン。 </li> <li> `oidc-implicit-profile` モバイルデバイスおよびシングルページアプリで使用されます。 </li> <li> `samlp` SaaSアプリで使用されるSAMLプロトコール。 </li> <li> `wsfed` Office365といったMicrosoft製品で使用されるWS-Federation 。 </li> <li> `wstrust-usernamemixed` CRMおよびOffice365で使用されるWS-trustユーザー/パスワードログイン。 </li> <li> `oauth2-device-code` デバイスの認可フローで使用されるトランザクション。 </li> <li> `oauth2-resource-owner` 通常、データベース接続に使用されるユーザー/パスワード ログイン。 </li> <li> `oauth2-resource-owner-jwt-bearer` ユーザーの秘密鍵で署名されたBearer{" "} JWT を使用したログイン。 </li> <li> `oauth2-password` パスワード交換を使用したログイン。 </li> <li> `oauth2-access-token` リフレッシュトークン交換を使用したトークンのリフレッシュ。 </li> <li> `oauth2-refresh-token` リフレッシュトークン交換を使用したトークンのリフレッシュ。 </li> <li>`oauth2-token-exchange`</li> <li> `oidc-hybrid-profile` アクセストークンとリフレッシュトークンをアプリケーションが 安全に取得できるようにしながら、 IDトークンに即座にアクセスできるようにします。 </li> </ul> </li> <li> `redirect_uri` *任意の文字列*。 トランザクションが完了した後にAuth0がリダイレクトするブラウザーのURL。 </li> <li> `requested_scopes` *文字列の配列*。 この認証フローを開始するときに要求（必要ある場合）される スコープ。 </li> <li> `response_mode` *任意の文字列*。 認可エンドポイントから返されたパラメーターに使用される 認可サーバーのメカニズムに知らせます。<br/>含まれる可能性のある値は以下の通りです：<br/><ul> <li>`query`</li> <li>`fragment`</li> <li>`form_post`</li> <li>`web_message`</li> </ul> </li> <li> `response_type` *任意の文字列の配列*。 含まれる可能性のある値は以下の通りです：<br/><ul> <li>`code`</li> <li>`token`</li> <li>`id_token`</li> </ul> </li> <li> `state` *任意の文字列*。 Auth0がリダイレクトしてアプリケーションに戻る際に含まれ、 アプリが初期要求に追加する不透明な 任意の英数字の文字列。 </li> <li> `ui_locales` *文字列の配列*。 元の認証要求で提供されたui_locales。 </li> </ul> |
| `event.user` | 登録を試みているユーザーを説明するオブジェクト。<br/>次のプロパティを含みます：<br/><ul> <li> `app_metadata` *任意の辞書*。 サポートプラン、セキュリティロール、アクセス制御グループなどの ユーザーアクセスに影響するユーザーについての情報を保存する カスタムフィールドです。 </li> <li> `email` *任意の文字列*。 （一意）ユーザーのメールアドレス。 </li> <li> `family_name` *任意の文字列*。ユーザーの姓。 </li> <li> `given_name` *任意の文字列*。ユーザーの名。 </li> <li> `name` *任意の文字列*。ユーザーの氏名。 </li> <li> `nickname` *任意の文字列*。 ユーザーのニックネーム。 </li> <li> `phone_number` *任意の文字列*。 ユーザーの電話番号。 </li> <li> `picture` *任意の文字列*。 [ ユーザーのプロファイル写真 ](https://auth0.com/docs/ja-jp/users/change-user-picture) を指すURL。 </li> <li> `user_metadata` *任意の辞書*。 勤務先住所、自宅住所、ユーザーの設定など、 ユーザーのアクセスに影響を与えないユーザーの情報を 保存するカスタムフィールド。 </li> <li> `username` *任意の文字列*。 （一意）ユーザーのユーザー名。 </li> </ul> |
