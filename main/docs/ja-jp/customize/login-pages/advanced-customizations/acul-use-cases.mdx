ACULは、エンドユーザーのログインエクスペリエンスを全面的にカスタマイズできるよう、さまざまな技術スタックをサポートしており、高度なカスタマイズによって多種多様なビジネスケースに対応できます。 
以下のシナリオを使用すると、ユニバーサルログインの[テーマ](/docs/ja-jp/customize/login-pages/universal-login/customize-themes)、[テンプレート](/docs/ja-jp/customize/login-pages/universal-login/customize-templates)、[テキスト要素](/docs/ja-jp/customize/login-pages/universal-login/customize-text-elements)をカスタマイズできます。Auth0のホストするログインについて詳しくは、「[Auth0ユニバーサルログイン]」(/docs/ja-jp/authenticate/login/auth0-universal-login)をご覧ください。
