## Vault接続を構成する

Mailjetアクション用にVault接続を構成するには、 **API key（APIキー）** と **Secret key（秘密鍵）** が必要です。

<Frame>![Dashboard > Actions > Forms > Flows > actions > MailJet](/docs/images/ja-jp/cdy7uua7fh8z/1ydxHXrM2idHnUOAjWAW4R/55cdb295e87894dd9f89bed8784e3a5d/mailjet-vault-connection.png)</Frame>

いずれも[Mailjetダッシュボード](https://app.mailjet.com/account/api_keys)のアカウントページのAPIキーにあります。

## メールを送信する

Mailjetをメールプロバイダーとして使用してメールを送信します。

<Frame>![](/docs/images/ja-jp/cdy7uua7fh8z/5z1CqSZTrLs1oMdHKvHOiM/7e1fa3b5d7ea5a5b2ec51dd58fa1d2ed/mailjet.png)</Frame>

### 入力設定

| パラメーター | 説明 |
| --- | --- |
| Sender name（差出人名） | 差出人の名前です |
| Sender email (required)（差出人のメール（必須）） | 差出人のメールです |
| Recipient name（宛先名） | 宛先の名前です |
| Recipient email (required)（宛先のメール（必須）） | 宛先のメールです |
| CC name（CCの名前） | CCの名前です |
| CC email（CCのメール） | CCのメールです |
| Subject（件名） | メールの件名です |
| Email type（メールの種類） | HTMLコンテンツまたは保存済みのテンプレート（任意）を使ってメールを送信します |
| Message (required)（メッセージ（必須）） | HTML形式またはテンプレート（任意）のメッセージです。 |
| Template ID (optional)（テンプレートID（任意）） | テンプレートを使用する場合、IDは[ Mailjetのダッシュボード](https://app.mailjet.com/template)で確認できます。 |
| Custom variables (optional)（カスタム変数（任意）） | テンプレートに挿入するカスタム変数を追加します。 |
| Track email activity（メールアクティビティを追跡する） | メールアクティビティを追跡するには、このチェックボックスを選択します。 |

### 出力設定

予期される応答とステータスコードは、[Mailjetのドキュメント](https://dev.mailjet.com/email/reference/send-emails/)で確認できます。
