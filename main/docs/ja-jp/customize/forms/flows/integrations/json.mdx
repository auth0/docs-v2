以下に記載のJSONアクションは、JSONの作成、解析、文字列変換を行えるようにします。

## JSONオブジェクトを作成する

JSONオブジェクトを作成します。これは、フローにある他のアクションでコンテンツを再利用するのに便利です。

<Frame>![](/docs/images/ja-jp/cdy7uua7fh8z/1zJEENRPI2uONuCApPY98p/f1fce88a42fb13fb9856c95c562bd506/createjsonobject.png)</Frame>

### 入力設定

| パラメーター | 説明 |
| --- | --- |
| ボディー | JSONオブジェクト。 |

### 出力オブジェクト

| プロパティ | 種類 | 説明 |
| --- | --- | --- |
| `result` | オブジェクト | JSONオブジェクトです。 |

### 出力オブジェクトの例

```json lines
{
	"result": {
		"name": "John Doe"
	}
}
```

## JSONを解析する

JSON形式の文字列の中身を解析して、JSONオブジェクトに変換します。

<Frame>![](/docs/images/ja-jp/cdy7uua7fh8z/5HjIGbBiWik1XozAZczxg5/fe2b0c6f7c657c4209921f1a0978e54b/parsejson.png)</Frame>

### 入力設定

| パラメーター | 説明 |
| --- | --- |
| JSON文字列（必須） | JSON文字列です。 |

### 出力オブジェクト

| プロパティ | 種類 | 説明 |
| --- | --- | --- |
| `object` | オブジェクト | 解析されたJSONオブジェクトです。 |

### 出力オブジェクトの例

```json lines
{
  "object": {
    "name": "John Doe"
  }
}
```

## JSONを文字列に変換する

JSONオブジェクトを文字列に変換します。

<Frame>![](/docs/images/ja-jp/cdy7uua7fh8z/45acWYMlTGc5ZWMzK5Cqqr/6974ea640a76d2d9cc5814225e52734a/convertjsontostring.png)</Frame>

### 入力設定

| パラメーター | 説明 |
| --- | --- |
| JSONオブジェクト（必須） | 文字列に直列化されるJSONオブジェクトです。 |

### 出力オブジェクト

| プロパティ | 種類 | 説明 |
| --- | --- | --- |
| `json` | 文字列 | 直列化されたJSONオブジェクトです。 |

### 出力オブジェクトの例

```json lines
{
  "json": "{\"name\":\"John Doe\"}"
}
```
