このXMLアクションのリストを使用すると、JSONオブジェクトをXML文字列に変換したり、XML文字列をJSONオブジェクトに変換したりできます。

## JSONをXMLに変換する

JSONオブジェクトをXML文字列に変換します。

<Frame>![](/docs/images/ja-jp/cdy7uua7fh8z/3GW2EhBMIX3YDPaqc3rXrZ/7706a863700df51376d645a7d9292a27/convertjsontoxml.png)</Frame>

### 入力設定

| パラメーター | 説明 |
| --- | --- |
| JSON（必須） | XML文字列に変換されるJSONオブジェクトです。 |

### 出力オブジェクト

| プロパティ | 種類 | 説明 |
| --- | --- | --- |
| `xml` | 文字列 | 変換されたXML文字列です。 |

### 出力オブジェクトの例

```json lines
{
  "xml": "<?xml version=\"1.0\" encoding=\"UTF-8\" standalone=\"yes\"?>\n<name>John Doe</name>"
}
```

## XMLをJSONに変換する

XML文字列をJSONオブジェクトに変換します。

<Frame>![](/docs/images/ja-jp/cdy7uua7fh8z/42fO7BelEN4bnlUj70WhV8/e53ada552f7cc05c192466db4777471a/covertxmltojson.png)</Frame>

### 入力設定

| パラメーター | 説明 |
| --- | --- |
| XML（必須） | JSONオブジェクトに変換されるXML文字列です。 |

### 出力オブジェクト

| プロパティ | 種類 | 説明 |
| --- | --- | --- |
| `object` | オブジェクト | 変換されたJSONオブジェクトです。 |

### 出力オブジェクトの例

```json lines
{
  "object": {
    "name": "John Doe"
  }
}
```
