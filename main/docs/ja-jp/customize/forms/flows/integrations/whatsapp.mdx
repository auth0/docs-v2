## Vault接続を構成する

WhatsAppアクション用にVault接続を構成するには、 **［Token（トークン）］** が必要です。

<Frame>![Dashboard > Actions > Forms > Flows > actions > WhatsApp](/docs/images/ja-jp/cdy7uua7fh8z/1Qevi3V1UbkUjb6c51x6R4/44590932bdced6e50afa436835c7cb15/whatsapp-vault-connection.png)</Frame>

トークンの生成方法については、「[WhatsAppに認可トークンを使用する](https://developers.facebook.com/blog/post/2022/12/05/auth-tokens/)」をお読みください。

## メッセージを送信する

メッセージを送信します。

<Frame>![](/docs/images/ja-jp/cdy7uua7fh8z/5zZPQw5KKYwBqxbR9jF1wZ/edba085456e3536f5167ff4c21bea77c/whatsapp.png)</Frame>

### 入力設定

| パラメーター | 説明 |
| --- | --- |
| Sender phone ID（送信者の電話ID）（必須） | 送信者の電話IDは、[FacebookビジネスマネージャーのWhatsAPP設定](https://developers.facebook.com/apps)で確認できます。これは電話番号ではないことに注意してください。 |
| Recipient phone number（受信者の電話番号）（必須） | 受信者の電話番号（[E.164形式](https://en.wikipedia.org/wiki/E.164)）。 |
| Message type（メッセージタイプ）（必須） | ドロップダウンメニューから以下のメッセージタイプを選択します： <ul> <li>Audio（音声）</li> <li>Contacts（連絡先）</li> <li>Document（ドキュメント）</li> <li>Image（画像）</li> <li>Interactive（インタラクティブ）</li> <li>Location（ロケーション）</li> <li>Sticker（スティッカー）</li> <li>Template（テンプレート）</li> <li>Text（テキスト）</li> </ul> |
| Message object（メッセージオブジェクト）（必須） | メッセージオブジェクトのコンテンツは選択したメッセージタイプによって異なります。詳しくは、[Messages](https://developers.facebook.com/docs/whatsapp/cloud-api/reference/messages)でお読みください。 |

### 出力設定

予期されるすべてのステータスコードについては、[Facebookのドキュメント](https://developers.facebook.com/docs/whatsapp/cloud-api/support/error-codes)を参照してください。
