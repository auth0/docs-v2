以下に記載のTwilioアクションは、Twilioをプロバイダーに使ってSMSの送信や通話を行えるようにします。

## Vault接続を構成する

TwilioアクションにVault接続を構成するには、 **アカウントのSID** と **Authトークン** が必要です。

<Frame>![Dashboard > Actions > Forms > Flows > actions > Twilio](/docs/images/ja-jp/cdy7uua7fh8z/7MafHKBR1acJE0iGtab2J4/5f4d85a149b9dbd881b4c4558bb0e7bb/twilio-vault-connection.png)</Frame>

これらは[Twilioコンソールのダッシュボード](https://console.twilio.com/)で確認できます。

## SMSを送信する

SMSを送信します。

<Frame>![](/docs/images/ja-jp/cdy7uua7fh8z/4lPqmcYGCJ2oRHM3GSta6w/10b0141d163e4be601c2786895486440/sendsmstwilio.png)</Frame>

### 入力設定

| パラメーター | 説明 |
| --- | --- |
| 差出人（必須） | SMSの送信者の名前です。 |
| 宛先（必須） | [E.164形式](https://en.wikipedia.org/wiki/E.164)で表記した宛先の電話番号です。 |
| メッセージ（必須） | SMSの本文の内容です。 |

### 出力設定

予期されるすべてのステータスコードについては、[Twilioのドキュメント](https://www.twilio.com/docs/messaging/api/message-resource#message-status-values)を参照してください。

## 通話する

通話を行います。

<Frame>![](/docs/images/ja-jp/cdy7uua7fh8z/4JaAbJIVuVUJY40lhQuVUN/1a1bd6143c8033c12e18fa49252bd36b/makecalltwilio.png)</Frame>

### 入力設定

| パラメーター | 説明 |
| --- | --- |
| From（差出人）（必須） | Twilioの電話番号です。国番号を含める必要があります。たとえば、+18005550175では、1が米国の国番号です。 |
| To（宛先）（必須） | [E.164形式](https://en.wikipedia.org/wiki/E.164)で表記した宛先の電話番号です。 |
| TwiMLのペイロード（必須） | 音声メッセージのTwiML命令です。<br/>詳細については、[TwilioのTwiMLドキュメント](https://www.twilio.com/docs/voice/twiml)をお読みください。 |

### 出力設定

予期されるすべてのステータスコードについては、[Twilioのドキュメント](https://www.twilio.com/docs/voice/twiml)を参照してください。
