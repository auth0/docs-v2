---
title: "生産準備チェック：重大ではない修正"
'description': "生産導入前にAuth0アプリケーションを改善する方法に関する推奨事項"
---

以下のチェックでは、生産導入を成功させるための推奨事項（オプション）をすべて完了したかどうかを確認します。

チェックの出力の詳細については、[［Run Production Checks（生産チェックの実行）］](/docs/ja-jp/deploy-monitor/pre-deployment-checks/how-to-run-production-checks)を参照してください。

| 確認 | 説明 |
| --- | --- |
| ログストリーミングの構成 | テナントログエントリを外部ストレージ場所またはプロバイダーにストリームしていることを確認する。[ログストリーミングサービスを構成](https://auth0.com/docs/ja-jp/customize/log-streams)して、ログを30日間以上保持し、分析およびアラートを向上することをお勧めします。 |
| [カスタムドメイン](/docs/ja-jp/custom-domains)の構成 | [ユニバーサルログインでカスタムドメイン](https://manage.auth0.com/#/tenant/custom_domains)を使用して、エンドユーザーによりシームレスで安全なエクスペリエンスを提供します。 |
| [カスタムエラーページ](/docs/ja-jp/universal-login/custom-error-pages)の構成 | [アプリケーション固有の詳細とコーポレートブランディングでカスタムエラーページ](https://manage.auth0.com/#/account)を構成します。 |
| ユニバーサルログインの有効化 | 最新バージョンの[ユニバーサルログイン](https://auth0.com/docs/ja-jp/authenticate/login/auth0-universal-login/new-universal-login-vs-classic-universal-login)を使用していることを確認します。向上したアクセシビリティやパフォーマンス、ノーコードカスタマイゼーション、新機能の恩恵を受けるために、クラシックのログインページからユニバーサルログインに更新することをお勧めします。 |
| [メールテンプレート](/docs/ja-jp/email/custom)の構成 | アプリケーション固有の詳細とコーポレートブランディングで[カスタムメールテンプレートを構成](https://manage.auth0.com/#/emails)します。 |
| [多要素認証](/docs/ja-jp/mfa)の構成 | ユーザーの認証のセキュリティプラクティスとして、多要素認証を検討することをお勧めします。 |
| [テナント管理者のためのMFA](/docs/ja-jp/tutorials/manage-dashboard-admins)を有効にする | [テナント管理者のために多要素認証](https://manage.auth0.com/#/account/admins)を有効にします。 |
| [リダイレクトログアウト URL](/docs/ja-jp/logout#set-the-allowed-logout-urls-at-the-account-level)の定義 | アプリケーションの[許可されているリダイレクトログアウトURL](https://manage.auth0.com/#/account/advanced)を確認します。 |
| アプリケーションはJSON Web Token（JWT）署名アルゴリズムのためにHS256ではなくRS256を使う必要がある | 各アプリケーションに対してJSON Web Token（JWT）[署名アルゴリズム](/docs/ja-jp/apis#signing-algorithms)をHS256ではなくRS256に設定します。 |
| [アプリケーションログインURI](https://auth0.com/docs/ja-jp/get-started/applications/application-settings#application-uris)の設定 | すべての対話型アプリケーションに対して[アプリケーションログインURI](https://auth0.com/docs/ja-jp/get-started/applications/application-settings#application-uris)を指定します。 |
| [テナント許可ログアウトURL](https://auth0.com/docs/ja-jp/get-started/applications/application-settings#application-uris)の設定 | 1つ以上のパブリックアクセス可能なログアウトルートを作成し、そこを指すように[テナント設定を更新します](https://auth0.com/docs/ja-jp/get-started/applications/application-settings#application-uris)。 |
| [テナントログインURI](https://auth0.com/docs/ja-jp/get-started/tenant-settings#login-and-logout)の設定 | デフォルトのログインルートを公開し、そこを指すように[テナント設定を更新します](https://auth0.com/docs/ja-jp/get-started/tenant-settings#login-and-logout)。 |
