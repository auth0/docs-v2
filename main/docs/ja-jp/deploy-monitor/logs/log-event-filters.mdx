---
title: "ログイベントにフィルターをかける"
'description': "エラー、警告、成功イベントのログフィルターを一覧で示します。"
---

ログには、エラー、警告、および成功イベントのフィルターをかけることができます。[［Dashboard］](https://manage.auth0.com/#/logs)で **［Filter（フィルター）］** の下矢印をクリックしてください。

## エラーイベントフィルター

| フィルター | 説明 |
| --- | --- |
| Block Account（アカウントのブロック） | 単一アカウントへのログイン試行の失敗が10回を超えたためブロックされたIP |
| Blocked IP Address（ブロックされたIPアドレス） | ログイン試行の失敗が100回を超えた、またはサインアップ試行の失敗が50回を超えたためブロックされたIP |
| Breached password（パスワード漏洩） | 漏洩したパスワードを使用したログインの試行 |
| Device Confirmation Canceled by User（ユーザーによるデバイスの確認のキャンセル） | ユーザーがデバイスを確認しませんでした |
| Error sending MFA Push Notification（MFAプッシュ通知送信エラー） | MFAのプッシュ通知に失敗しました |
| Error sending MFA SMS（MFA SMSの送信エラー） | MFAのSMSに失敗しました |
| Failed API Operation（API操作の失敗） | APIの操作に失敗しました |
| Failed Change Email（メールの変更に失敗） | ユーザーメールの変更に失敗しました |
| Failed Change Password（パスワード変更に失敗） | ユーザーパスワードの変更に失敗しました |
| Failed Change Password Request（パスワード要求の変更に失敗） | パスワード要求の変更に失敗しました |
| Failed Change Phone Number（電話番号の変更に失敗） | ユーザーの電話番号の変更に失敗しました |
| Failed Change Username（ユーザー名の変更に失敗） | ユーザー名の変更に失敗しました |
| Failed Connector Provisioning（コネクターのプロビジョニングに失敗） | AD/LDAPコネクターのプロビジョニングに失敗しました |
| Failed Delegation（委任に失敗） | 委任トークンの生成に失敗しました |
| Failed Exchange（交換の失敗） | トークン交換 |
| Failed Exchange（交換の失敗） | ネイティブソーシャルログイン |
| Failed Exchange（交換の失敗） | アクセストークンの認可コード |

| Failed Exchange（交換の失敗） | アクセストークンのパスワードおよびOTPチャレンジ |
| Failed Exchange（交換の失敗） | アクセストークンのパスワードおよびMFA復旧コード |
| Failed Exchange（交換の失敗） | アクセストークンのデバイスコード |
| Failed Login（ログインに失敗） | ユーザーがログインに失敗しました |
| Failed Login (invalid email/username)（ログインに失敗（無効なメール/ユーザー名）） | 無効なユーザー名によりユーザーがログインに失敗しました |
| Failed Login (wrong password)（ログインに失敗（パスワードの間違い）） | 無効なパスワードによりユーザーがログインに失敗しました |
| Failed Logout（ログアウトに失敗） | ユーザーのログアウトに失敗しました |
| Failed OIDC Back-Channel Logout Request（OIDCバックチャネルログアウト要求に失敗） | バックチャネルログアウト要求に失敗しました |
| Failed Post Change Password Hook（パスワード変更後フックに失敗） | パスワード変更後のフックに失敗しました |
| Failed Post User Registration Hook（ユーザー登録後フックに失敗） | ユーザー登録後のフックに失敗しました |
| Failed Sending Notification（通知送信に失敗） | メール通知の送信に失敗しました |
| Failed Signup（サインアップに失敗） | サインアップに失敗しました |
| Failed Silent Auth（サイレント認証に失敗） | サイレント認証に失敗しました |
| Failed User Deletion（ユーザー削除に失敗） | ユーザーの削除に失敗しました |

| OTP Auth failed（OTP認証に失敗） | ワンタイムパスワード認証に失敗しました |
| OTP Auth rejected（OTP認証の拒否） | ワンタイムパスワード認証が拒否されました |
| Rate Limit on API（APIのレート制限） | 与えられた時間内でAuthentication APIへの最大要求数に達しました |
| Recovery failed（復旧に失敗） | 多要素の復旧コードに失敗しました |
| Second factor email failed（第2要素のメールに失敗） | MFAのメールに失敗しました |
| Too Many Calls to /delegation（/delegationの呼び出しが多すぎる） | /delegationエンドポイントのレート制限を超えました |
| Too Many Calls to /userinfo（/userinfoの呼び出しが多すぎる） | /userinfoエンドポイントのレート制限を超えました |
| Too Many Invalid Device Codes（無効なデバイスコードが多すぎる） | 無効なデバイスコードのレート制限を超えました |

## 警告イベントフィルター

| フィルター | 説明 |
| --- | --- |
| Deprecation Notice（廃止通知） | 機能が廃止された |
| Too many failures（失敗回数が多すぎる） | 多要素OTPの失敗回数が多すぎる |
| Too many failures（失敗回数が多すぎる） | 多要素復旧コードの失敗回数が多すぎる |
| Users import（ユーザーのインポート） | ユーザーのインポートに失敗 |
| Warning During Login（ログイン時の警告） | ログイン時の警告 |
