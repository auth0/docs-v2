Auth0 CLIは、Auth0 Management APIへの直接アクセスを提供するコマンドラインツールで、ターミナルからAuth0テナントを管理できます。開発中のインタラクティブな使用を想定して設計されており、自動化のためにスクリプトに統合できます。

## 主な機能

* **インタラクティブ管理**: コマンドラインからAuth0リソースを迅速に作成、更新、管理
* **認証サポート**: ユーザーログインとマシン間認証情報を含む複数の認証方法
* **リソース管理**: アプリケーション、API、接続、ユーザーなどの完全なCRUD操作
* **クロスプラットフォーム**: macOS、Linux、Windows対応
* **JSON出力**: スクリプトと自動化のための機械可読出力

## インストール

### macOS

Homebrewを使用:

```bash lines
brew tap auth0/auth0-cli
brew install auth0
```

### Linux

```bash lines
curl -sSfL https://raw.githubusercontent.com/auth0/auth0-cli/main/install.sh | sh -s -- -b /usr/local/bin
```

### Windows

Scoopを使用:

```powershell lines
scoop bucket add auth0 https://github.com/auth0/scoop-auth0-cli.git
scoop install auth0
```

または[GitHubリリースページ](https://github.com/auth0/auth0-cli/releases)から最新リリースをダウンロードしてください。
