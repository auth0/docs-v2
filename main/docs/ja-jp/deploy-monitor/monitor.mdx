---
title: "Auth0を監視する"
'description': "Auth0の利用状況を監視して追跡する方法と、テナントログを外部のツールに送信する方法について説明します。"
---

Auth0の実装、Auth0のステータスとサービス、そして、テナントログのイベントデータを監視することができます。

Xで[@auth0status](https://twitter.com/auth0status)をフォローして最新のステータス更新を取得し、Auth0実装を監視して外部サービスとアプリケーションのステータスと可用性を確認します。

| トピック | 説明 |
| --- | --- |
| [Auth0のステータスを確認する](/docs/ja-jp/monitor-auth0/check-auth0-status) | ロケールと地域のAuth0サービスの可用性を確認する方法、およびサービス更新にサブスクライブする方法。 |
| [外部サービスのステータスを確認する](/docs/ja-jp/monitor-auth0/check-external-services-status) | 外部IDプロバイダーのサービスを監視する方法。 |
| [アプリケーションを監視する](/docs/ja-jp/monitor-auth0/monitor-applications) | ユーザーに影響の可能性を報告するためにアプリケーションの正常性を監視する方法。 |
| [System Center Operations Managerを使ってAuth0を監視する](/docs/ja-jp/monitor-auth0/monitor-using-scom) | System Center Operations Manager（SCOM）または合成トランザクションをサポートするツールを使用して、Auth0を監視する方法。 |
| [ログ](/docs/ja-jp/monitor-auth0/logs) | ビジネスニーズのために分析可能なAuth0テナントイベントログについて。 |
| [ログストリーム](/docs/ja-jp/monitor-auth0/streams) | AWS EventBridge、Azure Event Grid、Datadogなどの外部サービスにテナントログのイベントストリームを作成する方法。 |
