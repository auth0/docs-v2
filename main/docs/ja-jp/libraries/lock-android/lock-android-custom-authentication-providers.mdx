**Auth0.Android** には、ブラウザーを使用して認証フローを処理する`WebAuthProvider`クラスがあります。しかし、独自の実装またはネイティブバージョンの`AuthProvider`を使用する場合にはどうすればよいでしょうか。

## AuthProviderクラス

`AuthProvider`インターフェイスを実装するクラスを作成し、そのメソッドをオーバーライドします。

Auth0で既に提供されている**ネイティブ** 実装を使用することもできます。現在利用可能な[ネイティブソーシャル認証](/docs/ja-jp/libraries/lock-android/lock-android-native-social-authentication)プロバイダーは、GoogleとFacebookです。
