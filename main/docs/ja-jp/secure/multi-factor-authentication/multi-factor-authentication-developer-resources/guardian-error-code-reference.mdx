---
title: "Guardianエラーコードのリファレンス"
'description': "Guardianのエラー コードと説明をリストします。"
---

エラーコードを使用して、情報メッセージを表示し、回復可能なエラーと回復不可能なエラーを区別します。

| エラーコード | 説明 |
| --- | --- |
| `invalid_token` | 無効な要求またはトランザクションのトークン |
| `insufficient_scope` | 要求された操作を実行する十分な権限がありません |
| `invalid_bearer_format` | 認証ヘッダーに配置されたベアラーは無効でした |
| `enrollment_conflict` | 同じユーザーで別の登録があります。登録は2回できません。 |
| `tenant_not_found` | 関連するテナントがありません。テナントを削除しない限り通常これは起こりません。 |
| `login_transaction_not_found` | MFA Authトランザクションがアクティブでないか、すでに期限が切れています。 |
| `error_sending_push_notification` | プッシュ通知の配信に失敗しました。 |
| `push_notification_wrong_credentials` | 誤った資格情報のためプッシュ通知の配信に失敗しました。 |
| `invalid_otp` | 提供されたOTPコードが無効でした。 |
| `invalid_recovery_code` | 提供された復旧コードが無効でした。 |
| `invalid_body` | 本文の検証に失敗しました。不正な要求です。 |
| `invalid_query_string` | クエリ文字列の検証に失敗しました。不正な要求です。 |
| `enrollment_transaction_not_found` | MFA登録トランザクションがアクティブでないか、期限が切れています。 |
| `invalid_phone_number` | 提供された電話番号が無効です。 |
| `error_sending_sms` | SMS配信エラー。 |
| `feature_disabled` | 要求された機能は現在全世界で利用可能ではありません（プロバイダーに連絡してください）。 |
| `feature_disabled_by_admin` | 要求された機能は現在管理者が無効にしています。 |
| `pn_endpoint_disabled` | 何度も再試行したためプッシュ通知を配信できませんでした。デバイスのアカウントを削除して、再度追加してください。 |
| `too_many_sms` | ユーザーに割り当てられるSMS数の上限を超過しました。 |
| `too_many_pn` | ユーザーに割り当てられるプッシュ通知数の上限を超過しました。 |
| `too_many_sms_per_tenant` | テナントに割り当てられるSMS数の上限を超過しました。 |
| `too_many_pn_per_tenant` | テナントに割り当てられるプッシュ通知数の上限を超過しました。 |
| `field_required` | 操作を実行するために必要なフィールドです（このエラーにはフィールドのコードとフィールド属性があります：`otpCode`、`recoveryCode`） |
| `method_not_found` | 現在サポートされていないメソッドを要求しました（起こってはならない）。 |
| `no_method_available` | 登録するメソッドが現在ありません（すべて無効になっている）。 |
| `enrollment_method_disabled` | 指定された登録メソッドは無効になっており、このエラーにはまた.methodフィールドがあります。 |
| `auth_method_disabled` | 指定された認証メソッドは無効になっており、このエラーにはまた.methodフィールドがあります。 |
| `invalid_otp_format` | OTP形式の検証エラー。 |
| `invalid_recovery_code_format` | 復旧コードの形式の検証エラー。 |
| `transaction_expired` | トランザクションはすでに期限が切れています。 |
| `already_enrolled` | すでに登録済みであり、再度登録できません。 |
| `not_enrolled` | 登録されていません。最初に登録する必要があります。 |
| `invalid_enrollment` | transaction#requestAuthメソッドに提供された登録は無効か、空/未定義です。 |
