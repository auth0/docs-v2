Auth0 permite importar usuarios desde aplicaciones externas mediante conexiones de base de datos personalizadas, la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip> de Auth0 o la extensión importación/exportación de usuarios.

<div id="automatic-migrations">
  ## Migraciones automáticas
</div>

Auth0 admite la migración automática de usuarios desde una [conexión de base de datos personalizada](/es/docs/authenticate/database-connections/custom-db) a Auth0. Esta práctica también se conoce como **migración progresiva** o **migración diferida**. Al activar esta función, tus usuarios:

* Se migran a Auth0 la primera vez que inician sesión después de configurar la integración.
* No tienen que restablecer su contraseña como consecuencia de la migración.

<Card title="La disponibilidad varía según el plan de Auth0">
  La disponibilidad de esta función depende tanto de tu implementación específica de inicio de sesión como de tu plan de Auth0 o acuerdo personalizado. Para obtener más información, consulta [Precios](https://auth0.com/pricing).
</Card>

Cuando un usuario se autentica mediante una conexión de base de datos personalizada marcada para importarse a Auth0, se lleva a cabo el siguiente proceso:

* Auth0 autentica a los usuarios migrados con la base de datos de Auth0.
* Si el usuario no ha sido migrado, Auth0 ejecuta tu script de inicio de sesión personalizado y, si el inicio de sesión se realiza correctamente, agrega al usuario a la base de datos de Auth0.
* En los inicios de sesión posteriores, las credenciales del usuario se recuperan desde Auth0, **NO** desde tu base de datos personalizada.
* Los usuarios nuevos se agregan automáticamente a la base de datos de Auth0.

<div id="bulk-user-imports-with-the-management-api">
  ## Importación masiva de usuarios con la Management API
</div>

Si ya tienes una base de datos de usuarios, puedes usar nuestro endpoint de la Management API [`/post_users_imports`](https://auth0.com/docs/api/management/v2#!/Jobs/post_users_imports) para poblar una conexión de base de datos con esta información.

<div id="migrate-users-with-the-user-importexport-feature">
  ## Migra usuarios con la función de importación/exportación de usuarios
</div>

La función de importación/exportación de usuarios te permite:

* Importar en bloque tus usuarios existentes de la base de datos a Auth0.
* Buscar y exportar algunos (o todos) tus usuarios de base de datos de Auth0.

Para usar la importación en bloque, debes ser Dashboard Editor - Users o Admin. Para usar la exportación en bloque, debes ser Dashboard Viewer - Users, Editor - Users o Admin.

Puedes importar y exportar datos de usuarios con la [función de importación/exportación de usuarios](/es/docs/manage-users/user-migration/bulk-user-import-export) disponible en el dashboard.
