---
description: Aprenda a otorgar membresía Just-in-Time (JIT) a organizaciones.
title: Otorgar membresía Just-in-Time a una conexión de organización
---

Otorgar membresía Just-in-Time permite que a los usuarios que inician sesión mediante una conexión habilitada se les otorgue automáticamente membresía en una [organización](/es/docs/manage-users/organizations/organizations-overview).

Para otorgar membresía automáticamente cuando los usuarios inician sesión en una organización, primero debe [configurar y habilitar una conexión](/es/docs/manage-users/organizations/configure-organizations/enable-connections) para la organización. Al habilitar la conexión, seleccione **Enable Auto-Membership** para permitir que todos los usuarios que inicien sesión con esa conexión se agreguen automáticamente como miembros de esta organización.
