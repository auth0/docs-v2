Al trabajar con [organizaciones](/es/docs/manage-users/organizations/organizations-overview) mediante programación, es posible que necesite recuperar una lista de miembros de una organización.

Puede ver los miembros asociados a una organización mediante el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>.

<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

1. Vaya a [Auth0 Dashboard &gt; Organizaciones](https://manage.auth0.com/#/organizations) y seleccione la organización cuya lista de miembros desea ver.
2. Seleccione la vista **Miembros**.

<div id="management-api">
  ## Management API
</div>

<Warning>
  Se pueden mostrar hasta 1000 miembros de la organización mediante Auth0 Dashboard o Management API, aunque podrían existir más.
</Warning>

Haga una llamada `GET` al endpoint `Get Organization Members`. Asegúrese de reemplazar los valores de marcador de posición `ORG_ID` y `MGMT_API_ACCESS_TOKEN` por el ID de su organización y el <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, usada para acceder a una API." cta="Ver glosario" href="/es/docs/glossary?term=Access+Token">Token de acceso</Tooltip> de Management API, respectivamente.

  ```bash cURL
  curl --request GET \

    --header 'authorization: Bearer MGMT_API_ACCESS_TOKEN'
  ```

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Encuentre su dominio de Auth0**

  Si su dominio de Auth0 se compone del nombre de su inquilino, su subdominio regional (a menos que su inquilino esté en la región de EE. UU. y se haya creado antes de junio de 2020) y `.auth0.com`. Por ejemplo, si el nombre de su inquilino fuera `travel0`, el nombre de su dominio de Auth0 sería `travel0.us.auth0.com`. (Si su inquilino estuviera en EE. UU. y se hubiera creado antes de junio de 2020, entonces el nombre de su dominio sería `https://travel0.auth0.com`).

  Si utiliza dominios personalizados, este debe ser el nombre de su dominio personalizado.
</Callout>

| Valor                   | Descripción                                                                                                                                      |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------ |
| `ORG_ID`                | ID de la organización cuyos miembros desea recuperar.                                                                                            |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso de la Management API](/es/docs/secure/tokens/access-tokens/management-api-access-tokens) con el scope `read:organization_members`. |

<div id="response-status-codes">
  ### Códigos de estado de respuesta
</div>

Los posibles códigos de estado de respuesta son los siguientes:

| Código de estado | Código de error      | Message                                                                                                      | Causa                                                                                             |
| ---------------- | -------------------- | ------------------------------------------------------------------------------------------------------------ | ------------------------------------------------------------------------------------------------- |
| `200`            |                      | Miembros recuperados correctamente.                                                                          |                                                                                                   |
| `400`            | `invalid_paging`     | La página solicitada supera el máximo permitido de 1000 registros.                                           | La API está limitada a devolver como máximo 1000 registros.                                       |
| `400`            | `invalid_body`       | Cuerpo de la solicitud no válido. El mensaje variará según la causa.                                         | La carga útil de la solicitud no es válida.                                                       |
| `401`            |                      | Token no válido.                                                                                             |                                                                                                   |
| `401`            |                      | Se recibió una firma no válida para la validación del JSON Web Token.                                        |                                                                                                   |
| `401`            |                      | El cliente no es global.                                                                                     |                                                                                                   |
| `403`            | `insufficient_scope` | scope insuficiente; se esperaba cualquiera de estos: `read:organization_members`.                            | Se intentó leer o escribir un campo no permitido con los alcances del Bearer Token proporcionado. |
| `429`            |                      | Demasiadas solicitudes. Revise los encabezados X-RateLimit-Limit, X-RateLimit-Remaining y X-RateLimit-Reset. |                                                                                                   |
