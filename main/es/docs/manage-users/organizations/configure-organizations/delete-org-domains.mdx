---
title: Eliminar dominios de una Organización
description: Elimine la asociación entre una Organización y un dominio mediante Auth0 Dashboard o la Management API.
---

Cuando elimina un dominio de una Organización, la asociación entre el dominio y la organización se elimina permanentemente. Esta operación no se puede deshacer.

<div id="delete-an-organization-domain-with-the-auth0-dashboard">
  ## Eliminar un dominio de una organización con Auth0 Dashboard
</div>

Para eliminar un dominio de una organización en Auth0 Dashboard:

1. Vaya a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations) y elija su organización.
2. Seleccione la pestaña **Domains**.
3. Junto al dominio que quiere eliminar, abra el menú de acciones (**...**) y seleccione **Delete**.
4. En la ventana que se abre, confirme la eliminación.

<div id="delete-an-organization-domain-with-the-management-api">
  ## Eliminar un dominio de una Organización con Management API
</div>

Para eliminar un dominio de una Organización con Management API, llama al [endpoint para eliminar un dominio de detección de una organización](/es/docs/api/management/v2/organizations/delete-discovery-domains-by-discovery-domain-id) (`DELETE /organizations/{id}/discovery-domains/{discovery_domain_id}`).
