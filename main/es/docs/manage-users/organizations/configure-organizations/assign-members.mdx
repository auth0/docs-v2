Para asignar un miembro a tu [organización](/es/docs/manage-users/organizations/organizations-overview), debes haber [creado el usuario](/es/docs/manage-users/user-accounts/create-users) en tu inquilino. Si no encuentras al usuario, puedes [invitarlo](/es/docs/manage-users/organizations/configure-organizations/invite-members) en su lugar.

Para gestionar miembros directamente, puedes usar el <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>.

<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

Para asignar miembros desde Auth0 Dashboard:

1. Vaya a [Auth0 Dashboard &gt; Organizaciones](https://manage.auth0.com/#/organizations) y seleccione la organización para la que desea configurar la pertenencia.
2. Seleccione la vista **Miembros**, luego **Agregar miembros** y, a continuación, **Agregar usuarios**.
3. Introduzca el nombre del usuario o de los usuarios que desea asignar como miembros de la organización y seleccione **Agregar usuario(s) a la organización**.

<div id="management-api">
  ## Management API
</div>

Para asignar miembros mediante la Management API:

Haz una solicitud `POST` al endpoint `Create Organization Members`. Asegúrate de reemplazar los valores de marcador de posición `ORG_ID`, `MGMT_API_ACCESS_TOKEN` y `USER_ID` por el id de tu organización, el <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o un JWT, que se utiliza para acceder a una API." cta="Ver glosario" href="/es/docs/glossary?term=Access+Token">Token de acceso</Tooltip> de la Management API y los ID de usuario, respectivamente.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Encuentre su dominio de Auth0**

  Su dominio de Auth0 se compone del nombre de su inquilino, su subdominio regional (a menos que su inquilino esté en la región de EE. UU. y se haya creado antes de junio de 2020) y `.auth0.com`. Por ejemplo, si el nombre de su inquilino fuera `travel0`, el nombre de su dominio de Auth0 sería `travel0.us.auth0.com`. (Si su inquilino estuviera en EE. UU. y se hubiera creado antes de junio de 2020, el nombre de su dominio sería `https://travel0.auth0.com`).

  Si usa dominios personalizados, este debe ser el nombre de su dominio personalizado.
</Callout>

| Valor                   | Descripción                                                                                                                                              |
| ----------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | ID de la organización a la que desea asignar miembros.                                                                                                   |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](/es/docs/secure/tokens/access-tokens/management-api-access-tokens) con el scope `create:organization_members`.     |
| `USER_ID`               | ID del usuario o de los usuarios que desea asignar a la organización especificada. Puede enviar un máximo de 10 miembros por organización por solicitud. |

<div id="response-status-codes">
  ### Códigos de estado de respuesta
</div>

Los posibles códigos de estado de respuesta son los siguientes:

| Código de estado | Código de error        | Mensaje                                                                                                      | Causa                                                                                                |
| ---------------- | ---------------------- | ------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------- |
| `204`            |                        | Miembros agregados correctamente a la organización.                                                          |                                                                                                      |
| `400`            | `invalid_body`         | Cuerpo de la solicitud no válido. El mensaje variará según la causa.                                         | La carga útil de la solicitud no es válida.                                                          |
| `400`            | `invalid_query_string` | Cadena de consulta de la solicitud no válida. El mensaje variará según la causa.                             | La cadena de consulta no es válida.                                                                  |
| `401`            |                        | Token no válido.                                                                                             |                                                                                                      |
| `401`            |                        | Se recibió una firma no válida para la validación de JSON Web Token.                                         |                                                                                                      |
| `401`            |                        | El cliente no es global.                                                                                     |                                                                                                      |
| `403`            | `insufficient_scope`   | scope insuficiente; se esperaba alguno de los siguientes: `create:organization_members`.                     | Se intentó leer o escribir en un campo no permitido con los alcances del Bearer Token proporcionado. |
| `429`            |                        | Demasiadas solicitudes. Revise los encabezados X-RateLimit-Limit, X-RateLimit-Remaining y X-RateLimit-Reset. |                                                                                                      |
