---
title: Crear dominios de Organización
description: Ayude a los usuarios a elegir la Organización correcta durante los flujos de registro e inicio de sesión mediante dominios de Organización verificados.
---

Los dominios de Organización funcionan con [Organization Domain Discovery](./define-organization-behavior) para ayudar a los usuarios a encontrar la Organización correcta durante el registro y en los [flujos de inicio de sesión](/es/docs/manage-users/organizations/login-flows-for-organizations).

Una vez configurados, los dominios de Organización verificados y Organization Domain Discovery ayudan a Auth0 a detectar automáticamente la organización de un usuario o a acotar las opciones de organización cuando los usuarios ingresan su dirección de correo electrónico. Los dominios de Organización y Organization Domain Discovery no restringen quién puede registrarse ni iniciar sesión.

<div id="create-an-organization-domain-with-the-auth0-dashboard">
  ## Crear un dominio de Organización en Auth0 Dashboard
</div>

Para crear un dominio de Organización en Auth0 Dashboard:

1. Vaya a [Auth0 Dashboard &gt; Organizations](https://manage.auth0.com/#/organizations).
2. Seleccione la pestaña **Domains**.
3. Elija **Add Domain**.
   En el campo **Domain**, introduzca el dominio de correo electrónico que desea asociar para la detección de organizaciones. El estado permanecerá como pendiente hasta que se confirme la propiedad.
4. Seleccione **Add Domain**.
5. Opcionalmente, configure [Organization Domain Discovery](./define-organization-behavior).

<div id="create-an-organization-domain-with-the-management-api">
  ## Crear un dominio de Organización con la Management API
</div>

Para crear un dominio de Organización con la Management API, llama al [endpoint Crear un dominio de detección de organizaciones](/es/docs/api/management/v2/organizations/post-discovery-domains) (`POST /organizations/{id}/discovery-domains`). De forma opcional, habilita el dominio para Organization Domain Discovery con el parámetro `use_for_organization_discovery`.

Para habilitar Organization Domain Discovery, llama al [endpoint Actualizar clientes por ID](/es/docs/api/management/v2/clients/patch-clients-by-id#body-organization-discovery-methods-one-of-0) (`PATCH /client/{id}`) y establece el parámetro `organization_discovery_methods`.
