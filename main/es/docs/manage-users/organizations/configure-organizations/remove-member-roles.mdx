Puede eliminar roles a los miembros de una [organización](/es/docs/manage-users/organizations/organizations-overview) mediante <Tooltip tip="Auth0 Dashboard: la consola principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: la consola principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>.

<div id="auth0-dashboard">
  ## Auth0 Dashboard
</div>

Para eliminar un rol de un miembro de una organización desde Auth0 Dashboard:

1. Vaya a [Auth0 Dashboard &gt; Organizaciones](https://manage.auth0.com/#/organizations) y seleccione la organización cuya pertenencia desea configurar.
2. Seleccione la vista **Miembros** y haga clic en el nombre del miembro al que desea quitarle un rol.
3. Busque el rol que desea eliminar, haga clic en el icono de la papelera y confirme.

<div id="management-api">
  ## Management API
</div>

Para eliminar roles de un miembro de una organización mediante la Management API:
Realiza una solicitud `DELETE` al endpoint `Delete Organization Member Roles`. Asegúrate de reemplazar los valores de marcador de posición `ORG_ID`, `MGMT_API_ACCESS_TOKEN`, `USER_ID` y `ROLE_ID` por el id de tu organización, el <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, utilizada para acceder a una API." cta="Ver glosario" href="/es/docs/glossary?term=Access+Token">Token de acceso</Tooltip> de la Management API, el id del usuario y el id del rol, respectivamente.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  **Encuentre su dominio de Auth0**

  Su dominio de Auth0 es el nombre de su inquilino, más su subdominio regional (a menos que su inquilino esté en la región de EE. UU. y se haya creado antes de junio de 2020), seguido de `.auth0.com`. Por ejemplo, si el nombre de su inquilino fuera `travel0`, el nombre de su dominio de Auth0 sería `travel0.us.auth0.com`. (Si su inquilino estuviera en EE. UU. y se hubiera creado antes de junio de 2020, entonces el nombre de su dominio sería `https://travel0.auth0.com`.)

  Si usa dominios personalizados, este debe ser el nombre de su dominio personalizado.
</Callout>

| Valor                   | Descripción                                                                                                                                               |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ORG_ID`                | ID de la organización de la que desea quitar roles a un miembro.                                                                                          |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](/es/docs/secure/tokens/access-tokens/management-api-access-tokens) con el scope `delete:organization_member_roles`. |
| `USER_ID`               | ID del usuario al que desea quitar los roles especificados.                                                                                               |
| `ROLE_ID`               | ID del rol que desea quitar del usuario especificado para la organización especificada.                                                                   |

<div id="response-status-codes">
  ### Códigos de estado de respuesta
</div>

Los posibles códigos de estado de respuesta son los siguientes:

| Código de estado | Código de error        | Message                                                                                                      | Causa                                                                                                      |
| ---------------- | ---------------------- | ------------------------------------------------------------------------------------------------------------ | ---------------------------------------------------------------------------------------------------------- |
| `204`            |                        | Roles eliminados correctamente del miembro de la organización.                                               |                                                                                                            |
| `400`            | `invalid_body`         | Cuerpo de la solicitud no válido. El mensaje variará según la causa.                                         | La carga útil de la solicitud no es válida.                                                                |
| `400`            | `invalid_query_string` | Cadena de consulta de la solicitud no válida. El mensaje variará según la causa.                             | La cadena de consulta no es válida.                                                                        |
| `401`            |                        | Token no válido.                                                                                             |                                                                                                            |
| `401`            |                        | Se recibió una firma no válida para la validación del JSON Web Token.                                        |                                                                                                            |
| `401`            |                        | El cliente no es global.                                                                                     |                                                                                                            |
| `403`            | `insufficient_scope`   | Alcance insuficiente; se esperaba alguno de los siguientes: `delete:organization_member_roles`.              | Se intentó leer o escribir un campo que no está permitido con los alcances del Bearer Token proporcionado. |
| `429`            |                        | Demasiadas solicitudes. Revise los encabezados X-RateLimit-Limit, X-RateLimit-Remaining y X-RateLimit-Reset. |                                                                                                            |
