Puedes configurar Organizaciones con <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>. Para obtener más información sobre los pasos necesarios para configurar tu organización, consulta [Crea tu primera organización](/es/docs/manage-users/organizations/create-first-organization).

<Card title="La disponibilidad varía según el plan de Auth0">
  La disponibilidad de esta función depende de tu plan de Auth0 o de tu acuerdo personalizado. Para obtener más información, consulta [Precios](https://auth0.com/pricing).
</Card>
