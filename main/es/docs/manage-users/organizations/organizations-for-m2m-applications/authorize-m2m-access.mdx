Para autorizar el acceso de Machine-to-Machine para una aplicación, debe habilitarla para usar una organización con una API específica. Para ello, debe asociar el client grant de la aplicación para la API correspondiente con la organización. Una vez asociada, la aplicación puede usar la organización al solicitar tokens para la API y los alcances definidos en el client grant. Deberá definir esta asociación para cada API a la que su aplicación necesite acceder dentro de la organización.

Puede autorizar el acceso M2M para una aplicación mediante el [Auth0 Dashboard](https://manage.auth0.com/) o la [Management API](https://auth0.com/docs/api/management/v2).

<Tabs>
  <Tab title="Auth0 Dashboard">
    Para asociar el client grant de una aplicación con una organización a través del Auth0 Dashboard:

    1. Vaya a **Organizaciones** y elija la organización con la que desea asociarla.
    2. Seleccione la pestaña **Machine-to-Machine Access**.
    3. Haga clic en **Add Access.**
    4. Haga clic en la aplicación que desea asociar con la organización.
    5. Seleccione una API.
    6. Haga clic en **Save.**
