---
description: Describe cómo ver los usuarios asignados a un rol mediante Auth0 Dashboard o Management API.
title: Ver usuarios asignados a roles
---

Puede ver los usuarios asignados a un [rol](/es/docs/manage-users/access-control/rbac) mediante <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>. Los roles se usan con el conjunto de características de API Authorization Core.

<div id="prerequisites">
  ## Requisitos previos
</div>

Para que el control de acceso basado en roles (RBAC) funcione correctamente, debe habilitarlo para su API mediante el Dashboard o la Management API. La funcionalidad de Authorization Core es distinta de la de Authorization Extension. Para consultar una comparación, lea [Authorization Core vs. Authorization Extension](/es/docs/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Dashboard
</div>

1. Vaya a [Dashboard &gt; User Management &gt; Roles](https://manage.auth0.com/#/users) y haga clic en el nombre del rol que quiere ver.
2. Haga clic en la pestaña **Users**.

Se muestra la siguiente información para cada usuario:

| Columna                         | Descripción                                                          |
| ------------------------------- | -------------------------------------------------------------------- |
| Imagen                          | Imagen del usuario en el perfil de usuario.                          |
| Nombre                          | Nombre del usuario en el perfil de usuario.                          |
| Dirección de correo electrónico | Dirección de correo electrónico del usuario en el perfil de usuario. |

<div id="management-api">
  ## Management API
</div>

Para obtener una lista de los usuarios asignados a un rol, consulta [Get a Role&#39;s Users](https://auth0.com/docs/api/management/v2#!/Roles/get_role_user).
