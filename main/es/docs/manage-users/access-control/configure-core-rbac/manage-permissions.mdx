* [Asignar permisos a usuarios](/es/docs/manage-users/access-control/configure-core-rbac/rbac-users/assign-permissions-to-users)
* [Ver permisos de roles](/es/docs/manage-users/access-control/configure-core-rbac/roles/view-role-permissions)
* [Ver permisos de usuarios](/es/docs/manage-users/access-control/configure-core-rbac/rbac-users/view-user-permissions)
