Puede ver los [roles](/es/docs/manage-users/access-control/rbac) asignados a un usuario en el Dashboard de Auth0. Esta tarea también puede realizarse con la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>. Los roles asignados se usan con el conjunto de características de API Authorization Core.

<div id="prerequisites">
  ## Requisitos previos
</div>

Para que el control de acceso basado en roles (RBAC) funcione correctamente, debe habilitarlo para su API mediante Dashboard o Management API. La funcionalidad de Authorization Core es diferente de la de Authorization Extension. Para comparar ambas, lea [Authorization Core vs. Authorization Extension](/es/docs/manage-users/access-control/authorization-core-vs-authorization-extension).

<div id="dashboard">
  ## Dashboard
</div>

1. Ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) y haz clic en el nombre del usuario que quieres ver.
2. Haz clic en la pestaña **Roles**.

Se muestra la siguiente información para cada rol:

| Columna     | Descripción                                      |
| ----------- | ------------------------------------------------ |
| Nombre      | Nombre del rol según la definición del rol.      |
| Descripción | Descripción del rol según la definición del rol. |

<div id="management-api">
  ## Management API
</div>

Haz una solicitud `GET` al [endpoint Get User Roles](https://auth0.com/docs/api/management/v2#!/Users/get_user_roles). Asegúrate de reemplazar los valores de marcador de posición `USER_ID` y `MGMT_API_ACCESS_TOKEN` por el ID de usuario y el <Tooltip tip="Token de acceso: credencial de autorización, en forma de cadena opaca o JWT, que se utiliza para acceder a una API." cta="Ver glosario" href="/es/docs/glossary?term=access+token">token de acceso</Tooltip> de la Management API, respectivamente.

| Valor                   | Descripción                                                                                                                             |
| ----------------------- | --------------------------------------------------------------------------------------------------------------------------------------- |
| `USER_ID`               | El ID del usuario del que quieres obtener los roles.                                                                                    |
| `MGMT_API_ACCESS_TOKEN` | [Token de acceso para la Management API](https://auth0.com/docs/api/management/v2/tokens) con los alcances `read:users` y `read:roles`. |
