---
title: Cómo bloquear y desbloquear usuarios como administrador del inquilino
sidebarTitle: Bloquear y desbloquear usuarios
description: Controle la capacidad de un usuario para iniciar sesión en su aplicación agregando o eliminando bloqueos del administrador del inquilino.
---

Los administradores del inquilino pueden bloquear usuarios para impedir que accedan a las aplicaciones (y desbloquearlos para volver a permitirles el acceso).

Si un usuario bloqueado intenta iniciar sesión en una aplicación, se le redirige fuera de la aplicación con el mensaje de error &quot;user is blocked&quot; en la URL.

<div id="add-tenant-administrator-blocks">
  ## Agregar bloqueos del administrador del inquilino
</div>

Para agregar un bloqueo del administrador del inquilino desde el Auth0 Dashboard:

1. Vaya a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users).
2. Seleccione el nombre del usuario que desea bloquear.
3. En la pestaña **Details**, junto a **Block user**, seleccione **Block**.

Esta opción también está disponible en el menú **...** a la derecha del nombre del usuario.

Para agregar un bloqueo del administrador del inquilino mediante la Management API, use el [endpoint Update a User](/es/docs/api/management/v2/users/patch-users-by-id) (`PATCH /users/{id}`) para establecer el [atributo `blocked` del usuario](/es/docs/api/management/v2/users/patch-users-by-id#body-blocked) en `true`.

<div id="remove-tenant-administrator-blocks">
  ## Eliminar bloqueos del administrador del inquilino
</div>

Cuando un administrador del inquilino bloquea a un usuario, el bloqueo no caduca automáticamente. Un administrador del inquilino debe desbloquear al usuario para restaurar su acceso a las aplicaciones.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Los bloqueos de [protección contra fuerza bruta](/es/docs/secure/attack-protection/brute-force-protection) se gestionan por separado de los bloqueos del administrador del inquilino. Obtenga más información sobre cómo [eliminar bloqueos de protección contra fuerza bruta](/es/docs/secure/attack-protection/brute-force-protection#removing-brute-force-protection-blocks).
</Callout>

Para eliminar un bloqueo del administrador del inquilino mediante el Auth0 Dashboard:

1. Vaya a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users).
2. Seleccione el nombre del usuario que tiene el bloqueo del administrador del inquilino.
3. En la pestaña **Details**, junto a **Block user**, seleccione **Unblock**.

Esta opción también está disponible en el menú **...** a la derecha del nombre del usuario.

Para eliminar un bloqueo del administrador del inquilino mediante la Management API, use el [endpoint Update a User](/es/docs/api/management/v2/users/patch-users-by-id) (`PATCH /users/{id}`) para establecer el [atributo `blocked`](/es/docs/api/management/v2/users/patch-users-by-id#body-blocked) del usuario en `false`.

Si usa la Management API para restablecer la contraseña de un usuario, también se eliminan los bloqueos del administrador del inquilino.
