---
description: Describe cómo crear usuarios en el Auth0 Dashboard.
title: Crear usuarios
---

Puede crear un usuario desde el Auth0 Dashboard o mediante la [Auth0 Management API](https://auth0.com/docs/api/management/v2/users/post-users).

1. Vaya a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users) y seleccione **Create User**.
2. Seleccione la **Conexión** y, a continuación, introduzca el **correo electrónico** y la **contraseña** del usuario.

   | Atributo           | Descripción                                                                                                                                                                                                                                                                                                                           |
   | ------------------ | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | Conexión           | La conexión de base de datos que se utilizará para autenticar al usuario. La lista desplegable muestra todas las conexiones de base de datos configuradas en su inquilino. La conexión que utilice debe estar asociada a una aplicación; de lo contrario, recibirá un mensaje de error que indica que la conexión está deshabilitada. |
   | Correo electrónico | La dirección de correo electrónico del usuario. El formato de la dirección de correo electrónico debe ajustarse a la especificación proporcionada en [RFC 5321 on IETF](https://datatracker.ietf.org/doc/html/rfc5321#section-4.1.2)                                                                                                  |
   | Contraseña         | La contraseña del usuario. No hay un límite máximo para la longitud de la contraseña.                                                                                                                                                                                                                                                 |
3. Seleccione **Create**.
   Se creará el usuario y se le redirigirá al perfil del usuario recién creado.

Si habilita [Múltiples dominios personalizados](/es/docs/customize/custom-domains/multiple-custom-domains), tendrá la opción de seleccionar **dominio** en la sección **Conexión** para elegir el dominio de notificación de su inquilino.

Si habilita [Identificadores flexibles](/es/docs/authenticate/database-connections/flexible-identifiers-and-attributes), además de **email** puede seleccionar el **username** o el **phone** del usuario, o una combinación de dos o más.
