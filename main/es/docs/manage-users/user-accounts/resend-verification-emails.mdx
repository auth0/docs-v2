---
description: Describe cómo reenviar correos electrónicos de verificación a los usuarios mediante el Dashboard de administración de usuarios de Auth0 y la Management API de Auth0.
title: Reenviar correos electrónicos de verificación
---

Si necesitas reenviar un correo electrónico de verificación a un usuario, puedes hacerlo con el Dashboard de administración de usuarios de Auth0 o la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip> de Auth0.

<Tabs>
  <Tab title="Usar Auth0 Dashboard">
    Puedes usar el Auth0 Dashboard para reenviar un correo electrónico de verificación a un usuario.

    1. En el Dashboard, ve a [**User Management &gt; Users**](https://manage.auth0.com/#/users).
    2. Selecciona el usuario.
    3. Selecciona el menú desplegable **Actions** y, a continuación, **Send Verification Email**.

    Si habilitas [Multiple Custom Domains](/es/docs/customize/custom-domains/multiple-custom-domains), tendrás disponible la opción **Dominio** para seleccionar el dominio de notificación de tu inquilino.
  </Tab>

  <Tab title="Usar Auth0 Management API">
    Puedes usar la Management API de Auth0 para reenviar un correo electrónico de verificación a un usuario.

    1. Obtén un [token de acceso de Management API](/es/docs/secure/tokens/access-tokens/management-api-access-tokens) con el scope `update:users`.
    2. Llama al endpoint de la Management API [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) con el `user_id` del usuario.

    #### Ejemplo de solicitud

    ```json lines
    {
       "user_id":"auth0|62463410c0242d00699461c8"
    }
    ```

    #### Ejemplo de respuesta

    ```json lines
    {
       "type":"verification_email",
       "status":"pending",
       "created_at":"2022-03-31T23:08:05.612Z",
       "id":"job_PyfpRHlmpkSoDmCv"
    }
    ```

    Si habilitas [Multiple Custom Domains](/es/docs/customize/custom-domains/multiple-custom-domains), debes incluir el encabezado HTTP `auth0-custom-domain`. Para obtener más información, consulta [Multiple Custom Domains](/es/docs/customize/custom-domains/multiple-custom-domains#customize-email-handling-using-the-management-api).

    #### Comprobar el estado del trabajo

    Puedes comprobar el estado del trabajo creado con el endpoint de la Management API [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email).

    1. Obtén el `id` del trabajo (devuelto en la respuesta).
    2. Llama al endpoint de la Management API [Get a job](https://auth0.com/docs/api/management/v2#!/Jobs/get_jobs_by_id) con el `id` del trabajo.

    ##### Ejemplo de respuesta

    ```json lines
    {
       "type":"verification_email",
       "status":"completed",
       "created_at":"2022-03-31T23:08:05.612Z",
       "id":"job_PyfpRHlmpkSoDmCv"
    }
    ```
  </Tab>
</Tabs>
