title: Administrar usuarios desde el Dashboard
---

Puedes usar el Dashboard para administrar a tus usuarios. Para empezar, ve a [Dashboard &gt; User Management &gt; Users](https://manage.auth0.com/#/users). Aquí puedes crear, ver, modificar o eliminar usuarios.

<Card title="La disponibilidad varía según el plan de Auth0">
  Tanto tu implementación de inicio de sesión como tu plan de Auth0 o acuerdo personalizado determinan si esta funcionalidad está disponible. Para obtener más información, consulta [Precios](https://auth0.com/pricing).
</Card>

De forma predeterminada, los atributos del perfil de usuario proporcionados por <Tooltip tip="Proveedor de identidad (IdP): Servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/es/docs/glossary?term=identity+providers">proveedores de identidad</Tooltip> distintos de Auth0 (como Google, Facebook o X) no se pueden editar directamente porque se actualizan desde el proveedor de identidad cada vez que el usuario inicia sesión. Para obtener más información sobre los atributos de los proveedores de identidad, consulta [Configurar la conexión del Proveedor de identidad para actualizar el perfil del usuario](/es/docs/manage-users/user-accounts/user-profiles/configure-connection-sync-with-auth0).

Para poder editar los atributos raíz `name`, `nickname`, `given_name`, `family_name` o `picture` del perfil de usuario normalizado, debes [configurar la sincronización de tu conexión con Auth0](/es/docs/manage-users/user-accounts/user-profiles/configure-connection-sync-with-auth0) para que los atributos del usuario se actualicen desde el proveedor de identidad solo cuando se cree el perfil del usuario. Estos atributos raíz estarán disponibles para su edición individualmente o mediante importaciones masivas

Para obtener información sobre cómo administrar el acceso de los miembros de tu equipo al <Tooltip tip="Auth0 Dashboard: producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, consulta [Administrar el acceso al Dashboard](/es/docs/get-started/manage-dashboard-access).
