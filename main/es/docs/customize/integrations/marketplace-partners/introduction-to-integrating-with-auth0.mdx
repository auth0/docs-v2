title: Introducción a la integración con Auth0
---

Auth0 ofrece un servicio de autenticación y autorización basado en estándares abiertos como OIDC, OAuth2, <Tooltip tip="Security Assertion Markup Language (SAML): Protocolo estandarizado que permite a dos partes intercambiar información de autenticación sin una contraseña." cta="Ver glosario" href="/es/docs/glossary?term=SAML">SAML</Tooltip>, entre otros. Pero, además, ofrecemos herramientas para que nuestros clientes puedan externalizar la centralización y gestión de usuarios, ciertos componentes de la seguridad de las aplicaciones y la conexión de redes de aplicaciones a una única fuente de identidad.

Una parte importante de esto es permitir que partners como usted creen un ecosistema de integraciones que pueda usarse para mejorar y ampliar lo que ya ofrecemos. Para ello, proporcionamos puntos de extensibilidad y métodos que permiten la personalización. Esto se traduce en más funcionalidad para los clientes con una menor carga de mantenimiento.

Casi el 90 % de nuestros clientes amplían la plataforma de Auth0 para adaptarla a sus requisitos específicos. Al colaborar con usted para resolver problemas y casos de uso comunes relacionados con la identidad, podemos acelerar el tiempo que tardan nuestros clientes en implementar sus flujos de trabajo de identidad.

<div id="what-can-be-extended">
  ## ¿Qué se puede ampliar?
</div>

A continuación, se muestra una lista de las distintas formas en que se puede personalizar Auth0. Algunos puntos de extensibilidad permiten instalar integraciones directamente, mientras que otros requieren una guía que el cliente pueda seguir, y algunos puntos de extensibilidad requieren una cuenta de pago de Auth0.

  * WS-Fed

  * Cargar JavaScript y CSS personalizados en el formulario de inicio de sesión
  * Plantillas de correo electrónico personalizadas
* **Fuentes de identidad**

  * [OpenID Connect](/es/docs/authenticate/identity-providers/enterprise-identity-providers/oidc)
  * [OAuth2](/es/docs/customize/integrations/marketplace-partners/social-connections-for-partners)

  * [Código personalizado que se conecta a una fuente de identidad remota](/es/docs/authenticate/database-connections/custom-db)
* [**Código personalizado tras una autenticación correcta**](/es/docs/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger)
