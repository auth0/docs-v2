La integración nativa de Auth0 con Vercel te permite conectar tu plataforma de identidad a tu proyecto de Vercel con una configuración mínima. Configura y personaliza tu pantalla de inicio de sesión sin tener que crearla desde cero. Luego, amplía y escala tus flujos de autenticación y autorización con funciones avanzadas, incluidas [Organizaciones B2B](/es/docs/manage-users/organizations), [Autorización detallada (FGA)](https://docs.fga.dev/), [Auth0 for AI Agents](https://auth0.com/ai/docs) y más.

La integración nativa de Auth0 con Vercel permite:

* El aprovisionamiento automático de una aplicación de Auth0 que se conecta a tu proyecto de Vercel.
* La precarga de tus credenciales de Auth0 en el dashboard de Vercel para una configuración rápida y sencilla.
* Compatibilidad lista para usar con tus aplicaciones de Next.js mediante los [SDK de Auth0](/es/docs/libraries).

La integración nativa de Auth0 con Vercel crea un nuevo entorno de inquilino de Auth0 para tu proyecto de Vercel. Si quieres usar una cuenta existente de Auth0, intégrala manualmente con Vercel al crear una aplicación en el Auth0 Dashboard.

<div id="prerequisites">
  ## Requisitos previos
</div>

Antes de comenzar, asegúrate de tener lo siguiente:

* Una cuenta de Vercel con un proyecto activo. Si es necesario, [crea una cuenta de Vercel](https://vercel.com/signup)
* Una aplicación de Next.js que use la versión más reciente del [Auth0 Next.js SDK](/es/docs/quickstart/webapp/nextjs)

Después de instalar la integración, [habilita la inserción en iframe](/es/docs/customize/login-pages/manage-core-configuration#iframe-embedding) para permitir que Universal Login o Classic Login se carguen en un iframe, tal como lo requiere Vercel.

<div id="install-integration">
  ## Instalar la integración
</div>

Instala la integración nativa de Auth0 para Vercel desde el Marketplace de Vercel:

1. Ve a **Integrations &gt; Browse Marketplace** y busca Auth0. Selecciona la **integración de Auth0** en **Native Integrations** y, después, selecciona **Install**.

2. Auth0 completa automáticamente la URL de localhost y la URL de devolución de llamada para tu aplicación. Selecciona tu **Installation Plan** y, después, selecciona **Continue**.

3. Ingresa un nombre para tu aplicación. Selecciona **Create**. Una vez que se hayan creado correctamente el nuevo inquilino de Auth0 y la aplicación, selecciona **Continue**.

4. Para conectarte a un proyecto de Vercel existente, selecciona **Connect Project** y, después, selecciona tu proyecto y tus entornos. Si es necesario, ingresa un prefijo personalizado para agregarlo a tus variables de entorno. Selecciona **Connect**. Una vez que hayas conectado correctamente un proyecto de Vercel, se te dirigirá a la sección [Primeros pasos](#getting-started) de la integración.

<div id="getting-started">
  ## Primeros pasos
</div>

La sección Primeros pasos del Dashboard de Vercel muestra un inicio rápido para poner tu aplicación en marcha de forma rápida y segura. También puedes administrar la configuración de la aplicación en la navegación lateral izquierda.

<div id="quickstart">
  ### Quickstart
</div>

Para comenzar con la integración nativa de Vercel de Auth0, sigue las instrucciones del Dashboard de Vercel para crear y desplegar tu aplicación.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Solo puedes configurar las variables de entorno de Auth0 en un entorno de producción.
</Callout>

<div id="manage-application-settings">
  ## Administrar la configuración de la aplicación
</div>

Puede administrar la configuración de su aplicación desde el [Vercel Dashboard](#vercel-dashboard) y el [Auth0 Dashboard](#auth0-dashboard).

<div id="vercel-dashboard">
  ### Dashboard de Vercel
</div>

Para administrar la configuración de la aplicación en el Dashboard de Vercel, ve a **Settings** en la navegación lateral izquierda.

Puedes actualizar la siguiente configuración de la aplicación:

* **Actualizar el nombre**: Actualiza el nombre de la aplicación.
* **Actualizar la configuración**:
  * **Parámetros**: Actualiza la URL de localhost y el endpoint de devolución de llamada de tu aplicación.
  * **Plan actual del nivel de instalación**: Ve a la configuración de la integración para actualizar el plan del nivel de instalación de tu integración. Para obtener más información, consulta [Settings](#settings).
* **Proteger este recurso**
  * **Rotar Secrets**: Rota los Secrets de tu aplicación. Para obtener más información, consulta [Rotating environment variables](https://vercel.com/docs/projects/environment-variables/managing-environment-variables#rotating-environment-variables).
  * **Entornos permitidos**: Selecciona si la integración está disponible para **Todos los entornos** o **Solo el entorno de producción**.
* **Eliminar la integración**: Cuando eliminas la integración, pasas automáticamente a un plan Free de Vercel. Para obtener más información, consulta [Delete integration](#delete-integration).

<div id="auth0-dashboard">
  ### Auth0 Dashboard
</div>

Administra la configuración adicional de la aplicación en el [Auth0 Dashboard](https://manage.auth0.com/).

Cuando instalas la integración nativa, Auth0 crea una nueva aplicación en un inquilino dedicado, preconfigurada con tus URI de Vercel, callbacks permitidos y URL de cierre de sesión.

Puedes modificar esta aplicación, por ejemplo, personalizando tu experiencia de Universal Login, igual que harías con cualquier otra aplicación en tu Auth0 Dashboard.

<div id="manage-integration">
  ## Administrar la integración
</div>

Para administrar la integración desde el Dashboard de Vercel, selecciona la pestaña **Integration** de tu proyecto, desplázate hasta la integración de Auth0 y selecciona **Manage**.

<div id="invoices">
  ### Facturas
</div>

En la página **Facturas**, puede ver una lista de facturas relacionadas con su integración de Auth0. Para obtener más información sobre las facturas, lea [Billing &amp; Invoices](https://vercel.com/docs/pricing/understanding-my-invoice).

<div id="settings">
  ### Configuración
</div>

En la página **Configuración**, agregue métodos de pago, cambie el plan asociado al nivel de instalación y elimine la integración.

<div id="payment-methods">
  #### Métodos de pago
</div>

Para añadir un método de pago, selecciona **Add**. Para obtener más información sobre los métodos de pago, consulta [Vercel pricing](https://vercel.com/docs/accounts/plans).

<div id="current-installation-level-plan">
  #### Plan actual del nivel de instalación
</div>

El plan del nivel de instalación se aplica a todos los productos de la instalación.

Para cambiar el plan del nivel de instalación:

1. En **Settings**, desplázate hasta **Current Installation Level Plan** y selecciona **Change Plan**.
2. Selecciona el plan del nivel de instalación que quieras.
3. Selecciona **Continuar**.
4. Revisa tu selección y luego selecciona **Update**.
5. Una vez que se haya actualizado la instalación, selecciona **Done**.

<div id="delete-integration">
  #### Eliminar la integración
</div>

Al eliminar la integración, pasará automáticamente al plan Free de Vercel.

Para eliminar la integración:

1. En **Settings**, desplácese hasta **Delete &lt;YOUR&#95;PRODUCT&gt;** y seleccione el botón **Remove &lt;YOUR&#95;PRODUCT&gt;**.
2. Introduzca el slug de la integración y su equipo, y luego seleccione **Reconozco que esto eliminará mi cuenta conectada a Auth0**.
3. Seleccione **Remove &lt;YOUR&#95;PRODUCT&gt;**.
