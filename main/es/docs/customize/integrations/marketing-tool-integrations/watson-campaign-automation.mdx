title: Exportar datos de usuario a Watson Campaign Automation
---

En este artículo, aprenderá a exportar los datos de usuario de Auth0 a un archivo CSV y luego a importarlos en el panel de Watson Campaign Automation.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para empezar, vaya a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abra la **User Import / Export Extension**. En la página de la extensión, seleccione **Export** en el menú.

A continuación, establezca **Export Format** en el formato de archivo requerido. Watson Campaign Automation acepta importaciones de archivos en formato CSV, así que elija la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, especifique un **User Field** y un **Column Name** para cada atributo de usuario que desee incluir en la exportación. Por ejemplo:

| User Field    | Column Name                     |
| ------------- | ------------------------------- |
| `email`       | Dirección de correo electrónico |
| `created_at`  | Fecha de creación               |
| `given_name`  | Nombre                          |
| `family_name` | Apellido                        |

Después de agregar los campos de usuario, haga clic en el botón **Export Users** para iniciar la exportación. Cuando esta finalice, descargue el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Para importar su archivo CSV a Watson Campaign Automation, siga estos pasos:

1. Inicie sesión en Watson Campaign Automation y vaya a **Data &gt; Import New**.
2. En el paso **Select File**, introduzca la configuración de la nueva base de datos en los campos correspondientes.
3. Seleccione la opción para subir un archivo desde un disco duro local y, a continuación, haga clic en **Browse** para seleccionarlo.
4. Configure el tipo de archivo como CSV (valores separados por comas).
5. Haga clic en **Next** para continuar al paso **Define Data Format**.
6. Asegúrese de que los datos se hayan procesado correctamente y, a continuación, haga clic en **Next**.
7. En el paso **Map Fields**, elija qué campos incluir en la importación y sus tipos.
8. Haga clic en **Next** para continuar al paso **Edit Field Settings**.
9. Edite la configuración de los campos predeterminados o nuevos según sea necesario.
10. Para iniciar la importación, haga clic en **Next**.

¡Eso es todo! Ha importado correctamente sus usuarios de Auth0 en Watson Campaign Automation.
