title: Exportar datos de usuarios a Oracle Eloqua
---

En este artículo, aprenderá a exportar los datos de usuario de Auth0 a un archivo CSV y, a continuación, importarlos en Eloqua con el asistente de carga de contactos.

<div id="create-a-user-data-file">
  ## Crear un archivo de datos de usuario
</div>

Para empezar, vaya a la sección [Extensions](https://manage.auth0.com/#/extensions) del Dashboard y abra la **User Import / Export Extension**. En la página de la extensión, seleccione **Export** en el menú.

A continuación, configure **Export Format** con el formato de archivo requerido. Eloqua acepta importaciones de archivos en formato CSV, así que elija la opción `Tab Separated Value file (*.csv)`.

En la parte superior, en la sección **Fields**, especifique un **User Field** y un **Column Name** para cada atributo de usuario que desee incluir en la exportación. Por ejemplo:

| User Field    | Column Name                     |
| ------------- | ------------------------------- |
| `email`       | Dirección de correo electrónico |
| `created_at`  | Fecha de creación               |
| `given_name`  | Nombre                          |
| `family_name` | Apellido                        |

Después de agregar los campos de usuario, haga clic en el botón **Export Users** para iniciar la exportación. Cuando la exportación se complete, descargue el archivo CSV para usarlo en la siguiente sección.

<div id="import-a-user-data-file">
  ## Importar un archivo de datos de usuario
</div>

Antes de comenzar, puede encontrar más información en [Oracle Eloqua Help Center: Uploading Contacts](https://docs.oracle.com/en/cloud/saas/marketing/eloqua-user/Help/Contacts/Tasks/UploadingContacts.htm).

Para importar su archivo CSV a Eloqua, siga estos pasos:

1. Inicie sesión en Eloqua y vaya a **Audience &gt; Contacts &gt; Upload** para abrir el **Contact Upload Wizard**.
2. Configure la importación mediante los campos proporcionados en la pestaña **Pick Data Source**.
3. Haga clic en el botón **Cloud** y seleccione el archivo que desea cargar.
4. Después de verificar el contenido del archivo en la pestaña **Review**, haga clic en **Next Step**.
5. En la pestaña **Map Fields**, especifique la configuración de asignación de campos para que los datos del archivo CSV coincidan con los campos de contacto. Haga clic en **Next Step** para continuar.
6. Complete el paso final del asistente y haga clic en **Finish**.

¡Listo! Importó correctamente sus usuarios de Auth0 a Eloqua.
