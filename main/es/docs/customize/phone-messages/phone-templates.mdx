Los correos electrónicos enviados como parte de los flujos de trabajo de Auth0 tienen su correspondiente plantilla de correo electrónico, que determina la dirección del remitente, el asunto, el cuerpo del mensaje y mucho más.

Después de [configurar un proveedor externo de telefonía](../phone-messages/configure-phone-messaging-providers), puedes modificar las plantillas de teléfono para personalizar tus mensajes telefónicos.

Las plantillas de mensajes de texto admiten [Liquid](https://liquidjs.com/index.html), un lenguaje de plantillas de código abierto que amplía la funcionalidad de HTML. Liquid te permite:

* Rellenar variables dinámicamente con datos, como reemplazar `{{ user.name }}` por el nombre del usuario
* Implementar lógica, como sentencias `if`/`else`
* Modificar datos con filtros para manipular cadenas y arreglos, usar operadores matemáticos y mucho más

Puedes habilitar, deshabilitar y personalizar las plantillas de teléfono en la [página de plantillas de teléfono del Dashboard](https://manage.auth0.com/#/phone).

  ## Plantillas de teléfono disponibles
</div>

A continuación, se muestra un resumen de las plantillas de teléfono disponibles:

| Nombre de la plantilla de teléfono      | Descripción                                                                                            |
| --------------------------------------- | ------------------------------------------------------------------------------------------------------ |
| Cuenta bloqueada                        | Alerta al cliente de que su cuenta se ha bloqueado como resultado de una actividad sospechosa.         |
| Cambiar contraseña                      | Envía al cliente su código para cambiar la contraseña.                                                 |
| Contraseña de un solo uso - Inscripción | Envía al cliente el código de verificación para inscribirse en la contraseña de un solo uso (OTP).     |
| Contraseña de un solo uso - Verificar   | Envía al cliente el código de verificación para la verificación de la contraseña de un solo uso (OTP). |
