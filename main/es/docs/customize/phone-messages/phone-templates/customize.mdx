<Warning>
  Para personalizar las plantilla de teléfono, debe [configurar un proveedor externo de mensajería telefónica](../configure-phone-messaging-providers).
</Warning>

<div id="customize-phone-templates">
  ## Personalizar plantillas de teléfono
</div>

Para personalizar una plantilla de teléfono:

1. Vaya a [**Dashboard &gt; Marca &gt; Plantillas de teléfono**](https://manage.auth0.com/#/phone).

2. En el menú desplegable **Plantilla**, seleccione la plantilla de teléfono que desea actualizar.

3. Active el interruptor **Estado** para habilitar la plantilla si aún no lo está.

4. Actualice el número de teléfono de **De** y/o la **Plantilla de mensaje de texto**.

5. Seleccione **Guardar** para guardar los cambios, **Probar** para probarlos o **Restablecer** para revertirlos.

<div id="from-phone-number">
  ### Número de teléfono del remitente
</div>

<Warning>
  Si usas [Twilio Messaging Services](https://www.twilio.com/docs/messaging/services) como origen de tus SMS, no puedes establecer un número de teléfono personalizado en **From**. En su lugar, Twilio elige un número del grupo de remitentes de tu Messaging Service según tu configuración de Twilio.
</Warning>

El campo **From** establece el número de teléfono desde el que se envían los mensajes de texto. El número de teléfono predeterminado de **From** es el que configuraste cuando configuraste tu proveedor de telefonía.

Para establecer un número de teléfono personalizado en **From** para una plantilla de teléfono, introduce el número de teléfono en este campo.

<div id="text-message-template">
  ### Plantilla de mensaje de texto
</div>

El campo **Plantilla de mensaje de texto** define el contenido del mensaje de texto. Cada plantilla incluye un cuerpo de mensaje predeterminado que puede modificar o descartar por completo para redactar el suyo propio.

Este campo admite la sintaxis Liquid para generar contenido dinámicamente. Para obtener más información, consulte [Sintaxis Liquid compatible para plantillas de teléfono](./supported-liquid-syntax).

<div id="test-updated-templates">
  ## Probar plantillas actualizadas
</div>

Para probar una plantilla de teléfono, haga clic en **Try**. Auth0 envía el mensaje telefónico a una aplicación predeterminada que lleva el nombre de su inquilino. Para probar plantillas para distintas aplicaciones, cree un usuario de ejemplo para completar los flujos de mensajes telefónicos.

Puede activar manualmente correos electrónicos de verificación para aplicaciones y usuarios específicos mediante el endpoint [Send an email address verification email](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email) de Management API.
