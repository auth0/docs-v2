---
title: Unified Phone Experience de Auth0
description: Obtén información sobre la Unified Phone Experience de Auth0 para la configuración de proveedores telefónicos.
---

Auth0 le permite configurar un proveedor telefónico para <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuarios que usa un factor además del username y la contraseña, como un code enviado por SMS." cta="Ver glosario" href="/es/docs/glossary?term=Multi-Factor+Authentication">Autenticación multifactor</Tooltip> (MFA) y <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuarios que usa un factor además del username y la contraseña, como un code enviado por SMS." cta="Ver glosario" href="/es/docs/glossary?term=Passwordless">sin contraseña</Tooltip> a nivel del inquilino con Unified Phone Experience. Esto elimina la necesidad de configurar un proveedor telefónico en varias ubicaciones, al tiempo que mantiene la flexibilidad al configurar los métodos de entrega.

Los inquilinos con proveedores telefónicos configurados pueden migrar a Unified Phone Experience y mantener las capacidades de mensajería telefónica en un solo lugar. Los nuevos inquilinos se inscriben automáticamente en Unified Phone Experience.
