title: Configurar un proveedor de telefonía personalizado con Twilio Verify
---

Puedes configurar un proveedor de telefonía personalizado con [Twilio Verify](https://www.twilio.com/docs/verify), que ayuda a combatir el fraude y a proteger las cuentas de usuario. Esto configura Twilio Verify como tu proveedor de telefonía personalizado.

<div id="prerequisites">
  ## Requisitos previos
</div>

Debes tener una cuenta de [Twilio](https://twilio.com/) con una opción válida de entrega por SMS y/o llamada telefónica.

<div id="set-up-twilio-verify-service">
  ## Configurar el servicio Twilio Verify
</div>

Puede crear un servicio de Twilio Verify desde la [Consola de Twilio](https://www.twilio.com/console/verify/services) o mediante la [API REST de Verify](https://www.twilio.com/docs/verify/api/service#create-a-verification-service). Necesitará lo siguiente para conectar su proveedor de telefonía personalizado de Auth0 con Twilio Verify:

* TWILIO&#95;ACCOUNT&#95;SID
* TWILIO&#95;AUTH&#95;TOKEN
* TWILIO&#95;VERIFY&#95;SID

<div id="connect-custom-phone-provider-with-twilio-verify">
  ## Conectar un proveedor de telefonía personalizado con Twilio Verify
</div>

1. En el Auth0 Dashboard, vaya a [Branding &gt; Phone Provider](https://manage.auth0.com/dashboard/#/phone/templates/phone/provider). Esto le llevará a la página **Phone Message Provider**.
2. Seleccione **Custom** como **Phone Provider**.
3. En **Provider Configuration**, agregue el siguiente ejemplo de código para permitir que la API de Twilio envíe mensajes telefónicos al número de teléfono del usuario:

```javascript lines expandable
/**
* Manejador que se ejecuta al enviar una notificación por teléfono
* @param {Event} event - Detalles sobre el usuario y el contexto en el que está iniciando sesión.
* @param {CustomPhoneProviderAPI} api - Métodos y utilidades para modificar el comportamiento del envío de notificaciones por teléfono.
*/
exports.onExecuteCustomPhoneProvider = async (event, api) => {
 const { TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN, TWILIO_VERIFY_SID } = event.secrets;

  const client = require('twilio')(TWILIO_ACCOUNT_SID, TWILIO_AUTH_TOKEN);

  // mapear el valor de voz de Auth0 a llamada
  const messageType = event.notification.delivery_method === 'voice' ? 'call' : 'sms';

  const { recipient, code } = event.notification;

  // agregar esto para corregir " 1333444999  "
  // "333 444 5555"
  const sanitizedNumber = recipient.replace(/\s/g, '').trim();

  await client.verify.v2.services(TWILIO_VERIFY_SID)
    .verifications.create({
      to: sanitizedNumber,
      channel: messageType,
      customCode: code
    })

};
```

3. Haz clic en el icono de **Key** para abrir el menú **Secrets**. Agrega los siguientes valores de la configuración de tu servicio de Twilio Verify:

* TWILIO&#95;ACCOUNT&#95;SID
* TWILIO&#95;AUTH&#95;TOKEN
* TWILIO&#95;VERIFY&#95;SID

4. Para cargar la biblioteca auxiliar de Twilio, haz clic en el icono de **Dependency**. Luego, haz clic en el botón **Add Dependency**. Cuando aparezca la ventana emergente **Add Dependency**, ingresa lo siguiente:

* **Name:** Twilio
* **Version:** latest (debería completarse automáticamente)

Cuando hagas clic en **Create**, Auth0 buscará la biblioteca auxiliar de Twilio y cargará la versión más reciente.

5. Haz clic en **Save**. Al hacer clic en **Save**, la Action se guarda y se implementa automáticamente. Para probar la configuración del proveedor de telefonía personalizado antes de usarla en un entorno de producción, haz clic en **Send Test Message**.
