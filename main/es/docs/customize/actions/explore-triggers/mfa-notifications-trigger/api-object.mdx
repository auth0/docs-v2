title: "Actions - Disparadores - send-phone-message - objeto de la API"

El objeto de la API del disparador send-phone-message de Actions incluye:

<div id="apicache">
  ## `api.cache`
</div>

Almacene y recupere datos que se conservan entre ejecuciones.

<div id="apicachedeletekey">
  ### `api.cache.delete(key)`
</div>

Elimina, si existe, un registro que describe un valor almacenado en caché en la clave proporcionada.

Devuelve un objeto `CacheWriteResult` con `type: "success"` si se eliminó un valor de la caché. Si la operación falla, devuelve `type: "error"`. En caso de error, el objeto devuelto tendrá una propiedad
`code` que indica la naturaleza del error.

| Parámetro | Descripción                                             |
| --------- | ------------------------------------------------------- |
| `key`     | *String*. La clave del registro almacenado en la caché. |

<div id="apicachegetkey">
  ### `api.cache.get(key)`
</div>

Obtiene un registro que describe un valor almacenado en caché para la
`key` proporcionada, si existe. Si se encuentra un registro, el valor almacenado en caché
se encuentra en la propiedad `value` del objeto devuelto.

Devuelve un registro de caché si se encuentra un elemento en la caché para la
`key` proporcionada. Los registros de caché son objetos con una
propiedad `value` que contiene el valor almacenado en caché, así como una
propiedad `expires_at` que indica el vencimiento máximo
del registro en milisegundos desde la época Unix.

**Importante:**
Esta caché está diseñada para datos efímeros de corta duración. Es posible que los elementos no estén
disponibles en transacciones posteriores, aunque aún se encuentren dentro de la
vida útil especificada.

| Parámetro | Descripción                                             |
| --------- | ------------------------------------------------------- |
| `key`     | *String*. La clave del registro almacenado en la caché. |

<div id="apicachesetkey-value-options">
  ### `api.cache.set(key, value, [options])`
</div>

Almacena o actualiza un valor de cadena en la caché con la clave especificada.

Los valores almacenados en esta caché se limitan al disparador en el que se establecen.
Están sujetos a los
[Límites de caché de Actions](/es/docs/customize/actions/limitations).

Los valores almacenados de esta forma tendrán una vigencia de hasta los valores
`ttl` o `expires_at` especificados. Si no se especifica ninguna
vigencia, se usará una vigencia predeterminada de 15 minutos. La vigencia no puede
superar la duración máxima indicada en
[Límites de caché de Actions](/es/docs/customize/actions/limitations).

| Parámetro            | Descripción                                                                                                                                                                                                                                                                                                                                                                                            |
| -------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------ |
| `key`                | *String*. La clave del registro almacenado en la caché.                                                                                                                                                                                                                                                                                                                                                |
| `value`              | *String*. El valor del registro que se va a almacenar.                                                                                                                                                                                                                                                                                                                                                 |
| `options`            | *Objeto opcional*. Opciones para ajustar el comportamiento de la caché.                                                                                                                                                                                                                                                                                                                                |
| `options.expires_at` | *Número opcional*. La hora absoluta de expiración en milisegundos desde la época de Unix. Aunque los registros en caché pueden expulsarse antes, nunca permanecerán más allá del valor de `expires_at` proporcionado.<br />*Nota:* Este valor no debe proporcionarse si también se proporcionó un valor para `ttl`. Si se proporcionan ambas opciones, se usará la expiración más temprana de las dos. |
| `options.ttl`        | *Número opcional*. El valor de tiempo de vida de esta entrada de caché en milisegundos. Aunque los valores en caché pueden expulsarse antes, nunca permanecerán más allá del valor de `ttl` proporcionado.<br />*Nota:* Este valor no debe proporcionarse si también se proporcionó un valor para `expires_at`. Si se proporcionan ambas opciones, se usará la expiración más temprana de las dos.     |
