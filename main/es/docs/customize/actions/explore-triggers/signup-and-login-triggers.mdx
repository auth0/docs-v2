Para obtener más información sobre los desencadenadores de registro e inicio de sesión, consulta las siguientes secciones.

| Desencadenador                                                                                                                                | Descripción                                                                                                                                               |
| --------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Desencadenador de inicio de sesión](/es/docs/customize/actions/explore-triggers/signup-and-login-triggers/login-trigger)                        | Los desencadenadores de inicio de sesión se ejecutan cuando un usuario inicia sesión en cualquier aplicación de un inquilino de Auth0.                    |
| [Desencadenador de prerregistro de usuario](/es/docs/customize/actions/explore-triggers/signup-and-login-triggers/pre-user-registration-trigger) | El desencadenador de prerregistro de usuario se ejecuta antes de que se agregue un usuario a una conexión de base de datos o una conexión sin contraseña. |
| [Desencadenador de posregistro de usuario](/es/docs/customize/actions/explore-triggers/signup-and-login-triggers/post-user-registration-trigger) | El desencadenador de posregistro de usuario se ejecuta después de que se agrega un usuario a una conexión de base de datos o una conexión sin contraseña. |
