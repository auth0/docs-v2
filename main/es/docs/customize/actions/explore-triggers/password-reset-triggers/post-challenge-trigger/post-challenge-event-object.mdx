            * `passkey` Se usó una passkey para autenticar como primer factor a un usuario de una conexión de base de datos.
            * `sms` Se usó una conexión SMS sin contraseña para autenticar al usuario como primer factor.
            * `email` Se usó una conexión de correo electrónico sin contraseña para autenticar al usuario como primer factor o para verificar el correo electrónico al restablecer la contraseña.
            * `phone_number` Se usó un número de teléfono para restablecer la contraseña.
