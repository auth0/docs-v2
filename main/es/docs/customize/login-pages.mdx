[Auth0 Universal Login](/es/docs/authenticate/login/auth0-universal-login) proporciona una función esencial de un <Tooltip tip="Servidor de autorización: Servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, tu servidor de autorización puede controlar los datos, las tareas y las funciones disponibles para un usuario." cta="Ver glosario" href="/es/docs/glossary?term=authorization+server">servidor de autorización</Tooltip>: el flujo de inicio de sesión. Cuando un usuario necesita demostrar su identidad para obtener acceso a tu aplicación, puedes redirigirlo a <Tooltip tip="Servidor de autorización: Servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, tu servidor de autorización puede controlar los datos, las tareas y las funciones disponibles para un usuario." cta="Ver glosario" href="/es/docs/glossary?term=Universal+Login">Universal Login</Tooltip> y dejar que Auth0 gestione el proceso de autenticación.

Universal Login es la principal solución de inicio de sesión alojada de Auth0. [Universal Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) ofrece herramientas de personalización fáciles de usar y una experiencia más simple y rápida para los usuarios finales. Como alternativa, Auth0 también admite [Classic Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/classic-experience), una experiencia de inicio de sesión alojada que utiliza controles de JavaScript para la personalización.

Para comparar las experiencias de inicio de sesión, consulta [Universal Login vs. Classic Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login).
