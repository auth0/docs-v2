---
title: Configuración personalizada del proveedor de correo electrónico con Actions
description: Escriba una Action para enviar mensajes a su proveedor de correo electrónico e implementar lógica personalizada para su caso de uso.
---

Para personalizar aún más la configuración de su proveedor de correo electrónico, puede configurar su inquilino con una <Tooltip tip="Action: Una función de Node.js que se ejecuta en puntos definidos durante el entorno de ejecución de Auth0." cta="Ver glosario" href="/es/docs/glossary?term=Action">Action</Tooltip>.

En lugar de usar nuestras integraciones de proveedor de correo electrónico o servidor SMTP, usted escribe el código de su Action para enviar mensajes al proveedor de correo electrónico o servidor SMTP que elija.

Esto le brinda más control sobre el proceso de entrega de correo electrónico y le permite implementar lógica personalizada para su caso de uso, como reintentar envíos fallidos, cambiar destinatarios o crear lógica específica de la organización.
