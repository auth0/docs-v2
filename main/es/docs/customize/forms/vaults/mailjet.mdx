<div id="get-your-mailjet-keys">
  ## Obtén tus claves de MailJet
</div>

Puedes encontrarlas en la página Cuenta del [panel de MailJet](https://app.mailjet.com/account/api_keys), en la sección de claves de API.

<div id="configure-your-mailjet-vault-connection">
  ## Configure la Conexión de MailJet Vault
</div>

Para configurar una conexión de MailJet Vault para sus Flow Actions de MailJet, debe establecer los parámetros requeridos.

<Frame>![Dashboard > Actions > Forms > Vault > MailJet](/docs/images/customize/forms/vaults/mailjet-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro     | Descripción                                              |
| ------------- | -------------------------------------------------------- |
| Clave de API  | Clave de API para la autorización de la API de MailJet.  |
| Clave secreta | Clave secreta para la autorización de la API de MailJet. |
