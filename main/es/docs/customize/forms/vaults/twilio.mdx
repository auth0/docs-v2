<div id="get-your-twilio-credentials">
  ## Obtén tus credenciales de Twilio
</div>

Encuéntralas en el [panel de control de la consola de Twilio](https://console.twilio.com/).

<div id="configure-your-twilio-vault-connection">
  ## Configura tu conexión de Twilio Vault
</div>

Para configurar una conexión de Twilio Vault para tus Twilio Flow Actions, debes establecer los parámetros requeridos.

<Frame>![Panel > Actions > Forms > Vault > Twilio](/docs/images/customize/forms/vaults/twilio-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro              | Descripción                                                       |
| ---------------------- | ----------------------------------------------------------------- |
| SID de la cuenta       | SID de la cuenta para la autenticación de la API de Twilio.       |
| Token de autenticación | Token de autenticación para la autenticación de la API de Twilio. |
