<div id="get-your-sendgrid-keys">
  ## Obtén tus claves de SendGrid
</div>

Encuéntralas en la página [Settings del panel de SendGrid](https://app.sendgrid.com/settings/api_keys), en la sección API keys.

<div id="configure-your-sendgrid-vault-connection">
  ## Configure la conexión de SendGrid Vault
</div>

Para configurar una conexión de SendGrid Vault para las Flow Actions de SendGrid, debe establecer los parámetros obligatorios.

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/sendgrid-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro    | Descripción                                     |
| ------------ | ----------------------------------------------- |
| Clave de API | Clave de API para autorizar la API de SendGrid. |
