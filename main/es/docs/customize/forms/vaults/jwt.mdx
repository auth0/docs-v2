  ## Configure la Conexión JWT Vault
</div>

Para configurar una Conexión JWT Vault para que sus Actions JWT de Flows firmen, decodifiquen y verifiquen tokens web JSON, debe establecer los parámetros obligatorios.

| Parámetro | Descripción                                                       |
| --------- | ----------------------------------------------------------------- |
| Secreto   | Secreto que se usará en el algoritmo HS256 para codificar el JWT. |
