  ## Obtén tu token de WhatsApp
</div>

Aprende a generar un token en [WhatsApp](https://developers.facebook.com/blog/post/2022/12/05/auth-tokens/).

<div id="configure-your-whatsapp-vault-connection">
  ## Configurar la conexión de WhatsApp Vault
</div>

Para configurar una conexión de WhatsApp Vault para las Actions de flujo de WhatsApp, debe establecer los parámetros obligatorios.

<Frame>![Dashboard > Actions > Forms > Vault > WhatsApp](/docs/images/customize/forms/vaults/whatsapp-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                       |
| --------- | ------------------------------------------------- |
| Token     | Token generado para autorizar la API de WhatsApp. |
