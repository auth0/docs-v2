<div id="get-your-telegram-token">
  ## Obtén tu token de Telegram
</div>

Aprende a generar un token en la [API de bots de Telegram](https://core.telegram.org/#bot-api).

<div id="configure-your-telegram-vault-connection">
  ## Configure la Conexión de Telegram Vault
</div>

Para configurar una Conexión de Telegram Vault para sus Actions de flujo de Telegram, debe establecer los parámetros obligatorios.

<Frame>![Dashboard > Actions > Forms > Vault > SendGrid](/docs/images/customize/forms/vaults/telegram-vault-connection.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro | Descripción                                       |
| --------- | ------------------------------------------------- |
| Token     | Token generado para autorizar la API de Telegram. |
