title: Routers

Con los nodos Router, puedes agregar Rules a Forms para crear ramificaciones de lógica condicional que redirijan a los usuarios entre nodos. Por ejemplo, puedes definir una condición de Rule para un atributo específico de `user_metadata` a fin de dirigir a un usuario a un nodo de paso o de flujo específico.

* El nodo Router agrega automáticamente una Rule de caso predeterminado. Si no se cumple ninguna condición de Rule, el recorrido continúa hacia el nodo vinculado.
* Las Rules se evalúan en el orden en que aparecen en el Router. Cuando se cumple una condición de Rule, el recorrido continúa hacia el nodo vinculado y no se evalúa ninguna otra condición.
* Puedes agregar hasta 10 condiciones de Rule al nodo Router.

<div id="add-a-router-rule">
  ## Agregar una Rule de Router
</div>

Para agregar una condición de Rule, sigue estos pasos:

1. En el editor de formularios, selecciona un nodo Router para abrir su configuración.
2. Selecciona **+ New Rule**.
3. Expande la Rule recién creada.
4. Para identificar la Rule, agrega un **Alias**.
5. Agrega una variable con el icono **+**.
6. Selecciona el operador de la Rule para la variable.
7. Agrega el valor o la variable que debe cumplir la Rule.
8. Vincula la nueva Rule al paso o nodo de flujo correspondiente.
9. Selecciona **Publish** para guardar.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  * Te recomendamos definir un alias para identificar fácilmente cada Rule en el nodo Router.
  * Puedes seleccionar entre varios operadores de Rule para evaluar operandos. Por ejemplo, puedes establecer first&#95;name `is equal to` Jane.
  * Los operandos pueden ser valores estáticos o variables.
  * Una Rule puede incluir operadores `AND` y `OR` para una evaluación más avanzada de la lógica condicional.
  * Después de agregar una Rule, asegúrate de vincularla al nodo deseado.
</Callout>

<div id="available-rule-operators">
  ## Operadores disponibles para Rule
</div>

Los operadores de Rule son:

* `is greater than`
* `is greater or equal than`
* `is less than`
* `is less than or equal`
* `is equal to`
* `is different to`
* `includes`
* `does not include`
* `starts with`
* `ends with`
* `exists`
* `does not exist`
* `is null`
* `is not null`
* `has not value`
* `is true`
* `is false`
