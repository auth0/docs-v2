<div id="send-email">
  ## Enviar correo electrónico
</div>

Envía un correo electrónico usando Mailjet como proveedor de correo electrónico.

<Frame>![](/docs/images/cdy7uua7fh8z/5z1CqSZTrLs1oMdHKvHOiM/7e1fa3b5d7ea5a5b2ec51dd58fa1d2ed/mailjet.png)</Frame>

<div id="input-settings">
  ### Configuración de entrada
</div>

| Parámetro                                         | Descripción                                                                                                   |
| ------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Nombre del remitente                              | Nombre del remitente                                                                                          |
| Correo electrónico del remitente (obligatorio)    | Correo electrónico del remitente                                                                              |
| Nombre del destinatario                           | Nombre del destinatario                                                                                       |
| Correo electrónico del destinatario (obligatorio) | Correo electrónico del destinatario                                                                           |
| Nombre en CC                                      | Nombre en CC                                                                                                  |
| Correo electrónico en CC                          | Correo electrónico en CC                                                                                      |
| Asunto                                            | Asunto del correo electrónico                                                                                 |
| Tipo de correo electrónico                        | Envía un correo electrónico con contenido HTML o mediante una plantilla guardada (opcional)                   |
| Mensaje (obligatorio)                             | Mensaje del correo electrónico en formato HTML o una plantilla (opcional).                                    |
| ID de plantilla (opcional)                        | Si usa una plantilla, puede encontrar su id en el [ Dashboard de Mailjet ](https://app.mailjet.com/template). |
| Variables personalizadas (opcional)               | Agregue variables personalizadas para insertarlas en la plantilla.                                            |
| Rastrear actividad del correo electrónico         | Casilla de verificación para rastrear la actividad del correo electrónico.                                    |

<div id="output-object">
  ### Objeto de salida
</div>

Puedes consultar la respuesta esperada y los códigos de estado en la [documentación de Mailjet](https://dev.mailjet.com/email/reference/send-emails/).
