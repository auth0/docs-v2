Esta lista de acciones JSON le permite crear, analizar y convertir JSON en cadenas de texto.

<div id="create-json-object">
  ## Crear un objeto JSON
</div>

Crea un objeto JSON. Esto resulta especialmente útil para reutilizar su contenido en otras acciones del flujo.

<Frame>![](/docs/images/cdy7uua7fh8z/1zJEENRPI2uONuCApPY98p/f1fce88a42fb13fb9856c95c562bd506/createjsonobject.png)</Frame>

### Configuración de entrada

| Parámetro | Descripción     |
| --------- | --------------- |
| Body      | El objeto JSON. |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo   | Descripción     |
| --------- | ------ | --------------- |
| `result`  | Objeto | El objeto JSON. |

### Ejemplo de objeto de salida

```json lines
{
	"result": {
		"name": "John Doe"
	}
}
```

<div id="parse-json">
  ## Analizar JSON
</div>

Convierte en un objeto JSON el contenido de una cadena de texto con formato JSON.

<Frame>![](/docs/images/cdy7uua7fh8z/5HjIGbBiWik1XozAZczxg5/fe2b0c6f7c657c4209921f1a0978e54b/parsejson.png)</Frame>

<div id="input-settings">
  ### Configuración de la entrada
</div>

| Parámetro                 | Descripción     |
| ------------------------- | --------------- |
| Cadena JSON (obligatoria) | La cadena JSON. |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo   | Descripción               |
| --------- | ------ | ------------------------- |
| `object`  | Objeto | El objeto JSON analizado. |

### Ejemplo de objeto de salida

```json lines
{
  "object": {
    "name": "John Doe"
  }
}
```

<div id="convert-json-to-string">
  ## Convertir JSON a cadena
</div>

Convierte un objeto JSON en una cadena.

<Frame>![](/docs/images/cdy7uua7fh8z/45acWYMlTGc5ZWMzK5Cqqr/6974ea640a76d2d9cc5814225e52734a/convertjsontostring.png)</Frame>

### Configuración de entrada

| Parámetro                 | Descripción                                      |
| ------------------------- | ------------------------------------------------ |
| Objeto JSON (obligatorio) | El objeto JSON que se serializará en una cadena. |

<div id="output-object">
  ### Objeto de salida
</div>

| Propiedad | Tipo   | Descripción                 |
| --------- | ------ | --------------------------- |
| `json`    | cadena | El objeto JSON serializado. |

<div id="output-object-example">
  ### Ejemplo del objeto de salida
</div>

```json lines
{
  "json": "{\"name\":\"John Doe\"}"
}
```
