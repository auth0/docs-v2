---

En la aplicación que expone la extensión Delegated Administration, hay dos vistas disponibles: **Users** y **Logs**. En la vista **Users**, puede ver y modificar los usuarios asociados a su cuenta de Auth0.

De forma predeterminada, se muestran todos los usuarios, pero puede filtrar la lista configurando un Filter Hook.

Para obtener más información sobre el Filter Hook, consulte [Delegation Administration: Filter Hook](/es/docs/customize/extensions/delegated-administration-extension/delegated-administration-hooks/delegated-administration-filter-hook).

<div id="user-actions-in-delegated-administration-dashboard">
  ## Acciones de usuario en el panel de Delegated Administration
</div>

Para limitar el número de opciones disponibles para alguien con acceso al Dashboard que expone la extensión Delegated Administration, configure un Access Hook.

Obtenga más información sobre Access Hooks en [Delegated Administration: Access Hook](/es/docs/customize/extensions/delegated-administration-extension/delegated-administration-hooks/delegated-administration-access-hook).

La opción **Change Profile** está disponible solo si ha configurado campos personalizados.

Tenga en cuenta la nueva opción **Reset Password** disponible a través de la extensión. Esta opción enviará un correo electrónico al usuario que le permitirá elegir una nueva contraseña. Para hacerlo, seleccione un usuario y seleccione **Actions** &gt; **Reset Password**.

Esto enviará un correo electrónico al usuario con un enlace para cambiar la contraseña.

Si su perfil indica que tiene el rol `Delegated Admin - Administrator`, la vista **Logs** le permite ver una lista de autenticaciones realizadas por sus usuarios (esta vista solo es visible para usuarios con el rol `Delegated Admin - Administrator`). El contenido de esta vista es un subconjunto de los datos que se muestran en [Auth0 Dashboard &gt; Monitoring &gt; Logs](https://manage.auth0.com/#/logs). El panel de Logs también muestra datos sobre las acciones administrativas realizadas en el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

<div id="create-users">
  ## Crear usuarios
</div>

Puede crear un nuevo usuario seleccionando el botón **+ Crear usuario** en la vista **Users**. Debe especificar el correo electrónico y la contraseña. Según su rol, es posible que no pueda establecer el **Departamento** al que pertenecerá el nuevo usuario.

Por ejemplo, los usuarios con el rol `Delegated Admin - Administrator` pueden ver el campo **Departamento** y seleccionar cualquiera de sus valores.

<Frame>![Extensions - Panel de administración - Crear administrador](/docs/images/cdy7uua7fh8z/4vrbU6nRuIkF30ZPM4yHnG/73fd2060fe9618c9f465025cb56deb02/create-user-admin.png)</Frame>

Por otro lado, Kelly, que tiene el rol `Delegated Admin - User` y pertenece al departamento de Finanzas, no puede ver el campo **Departamento**. El usuario que cree se asignará automáticamente al departamento de Finanzas.

<Frame>![Extensions - Panel de administración delegada - Crear usuario](/docs/images/cdy7uua7fh8z/2CSdiPbyiyPsURHH9X5VVZ/8416222627933bd35c044aa15c22dfae/create-user-kelly.png)</Frame>

<Card title="Tiempo de espera de la sesión de Delegated Administration">
  De forma predeterminada, el tiempo de expiración del token es de 10 horas. Sin embargo, por motivos de seguridad, al usar Delegated Administration, Auth0 no guarda ningún token en las cookies ni en `sessionStorage`. Debe iniciar una nueva sesión cada vez que se recargue la página.
</Card>
