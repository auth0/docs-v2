El Dashboard de <Tooltip tip="Inicio de sesión único (SSO): servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente su sesión en otras aplicaciones." cta="Ver glosario" href="/es/docs/glossary?term=Single+Sign-on">inicio de sesión único</Tooltip> (SSO) es una aplicación web diseñada para resolver un problema conocido por muchas personas. Las organizaciones de todos los tamaños mantienen diversas aplicaciones para gestionar distintas funciones empresariales, como contabilidad, RR. HH., desarrollo, soporte, etc. Recordar nombres de usuario, contraseñas y URL de inicio de sesión de todas sus aplicaciones puede resultar engorroso. Con esta extensión, puede simplificar la experiencia de autenticación al habilitar el inicio de sesión con SSO para sus usuarios en varias aplicaciones empresariales. Le permite crear una lista de todos los servicios en la nube en los que un usuario puede autenticarse mediante SSO.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Un usuario no debería poder acceder, desde el dashboard o de cualquier otra forma, al proveedor de servicios sin tener los permisos adecuados (grupos/roles) para hacerlo. Idealmente, estos usuarios no deberían ver en el dashboard ningún proveedor de servicios al que no se les haya concedido acceso.
</Callout>

El SSO Dashboard admite dos tipos de usuarios:

* **Usuarios** que pueden iniciar sesión en el dashboard para seleccionar una aplicación e iniciar sesión con SSO.
* **Administradores** que pueden configurar las aplicaciones visibles para los usuarios.

Para configurar esta extensión, haga lo siguiente:

1. [Crear una aplicación de SSO Dashboard](/es/docs/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)
2. [Instalar la extensión SSO Dashboard](/es/docs/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)
3. [Agregar aplicaciones al SSO Dashboard](/es/docs/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)
4. [Actualizar aplicaciones en el SSO Dashboard](/es/docs/customize/extensions/single-sign-on-dashboard-extension/update-applications-on-the-sso-dashboard)

También puede [ver esta extensión SSO Dashboard en GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).
