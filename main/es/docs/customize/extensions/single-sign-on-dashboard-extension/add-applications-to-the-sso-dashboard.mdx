description: Aprende a agregar aplicaciones a la extensión del panel de Single Sign-On (SSO) de Auth0 para habilitar el inicio de sesión único en tus aplicaciones.
title: Agregar aplicaciones al panel de Single Sign-On
---

Usa la extensión del panel de Single Sign-On (<Tooltip tip="Single Sign-On (SSO): servicio que, una vez que un usuario inicia sesión en una aplicación, inicia automáticamente su sesión en otras aplicaciones." cta="Ver glosario" href="/es/docs/glossary?term=SSO">SSO</Tooltip>) para gestionar el inicio de sesión único de tus usuarios en múltiples aplicaciones empresariales.

<div id="prerequisites">
  ## Requisitos previos
</div>

* [Cree la aplicación del panel de SSO](/es/docs/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application).
* [Instale la extensión del panel de SSO](/es/docs/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).

<div id="add-applications">
  ## Agregar aplicaciones
</div>

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y selecciona tu nueva extensión de panel de SSO.
   Si eres administrador, también puedes iniciar sesión en el panel de SSO mediante `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
2. En la esquina superior derecha, selecciona **Settings** en el menú desplegable que aparece debajo del nombre de tu inquilino.
3. Haz clic en **Create App** para agregar una aplicación nueva.

   Aparecerá el formulario **New Application**.
4. Completa los siguientes campos de la nueva aplicación:

   | Campo           | Descripción                                                                                                                                                                                                                                                                                                                     |
   | --------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
   | **Type**        | Un menú desplegable en el que seleccionas SAML, OpenID-Connect o WS-Federation, según el tipo de aplicación.                                                                                                                                                                                                                    |
   | **Application** | Un menú desplegable en el que seleccionas la aplicación que deseas agregar.                                                                                                                                                                                                                                                     |
   | **Name**        | El nombre se completa automáticamente en función de la aplicación que seleccionaste. Puedes cambiarlo o usar el valor predeterminado.                                                                                                                                                                                           |
   | **Logo**        | La URL del logotipo que deseas usar como icono de la aplicación.                                                                                                                                                                                                                                                                |
   | **Callback**    | Una de las **Allowed Callback URLs** de la [Application Settings](https://manage.auth0.com/#/applications) de la aplicación.                                                                                                                                                                                                    |
   | **Connection**  | (*Opcional*) El tipo de conexión. Puedes agregar o editar los tipos de conexión disponibles en la [sección Connections del panel de administración de Auth0](https://manage.auth0.com/#/connections/database). Si no se configura una conexión y el usuario no ha iniciado sesión, verá la página de inicio de sesión de Auth0. |
   | **Custom URL**  | (*Opcional*) Casilla para usar una URL personalizada en lugar de la URL de Auth0. Si la marcas, aparecerá un campo donde podrás introducir tu URL personalizada.                                                                                                                                                                |
   | **Enabled**     | Casilla para que esta aplicación sea visible (publicada) para tus usuarios.                                                                                                                                                                                                                                                     |
5. Selecciona **Create**. Tu nueva aplicación aparecerá en la página **Applications** del panel de SSO junto con cualquier otra aplicación que se haya creado.
6. Haz clic en una aplicación para probar la conexión.
