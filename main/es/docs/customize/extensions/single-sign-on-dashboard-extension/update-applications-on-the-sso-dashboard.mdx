title: Actualizar aplicaciones en SSO Dashboard
---

Usa la extensión SSO Dashboard para gestionar el inicio de sesión de <Tooltip tip="Single Sign-On (SSO): servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente su sesión en otras aplicaciones." cta="Ver glosario" href="/es/docs/glossary?term=SSO">SSO</Tooltip> para tus usuarios en varias aplicaciones empresariales.

Para obtener más información, consulta [Single Sign-On Dashboard Extension](/es/docs/customize/extensions/single-sign-on-dashboard-extension) o [consulta esta extensión en GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).

<div id="prerequisites">
  ## Requisitos previos
</div>

* [Cree la aplicación del SSO Dashboard](/es/docs/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application) en Auth0.
* [Instale la extensión SSO Dashboard](/es/docs/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension).
* [Agregue aplicaciones al SSO Dashboard](/es/docs/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard).

<div id="update-application">
  ## Actualizar una aplicación
</div>

1. Vaya a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y seleccione la vista **Installed Extensions**.
2. Seleccione **SSO Dashboard**.
   Si es administrador, también puede iniciar sesión en **SSO Dashboard** mediante `https://{yourTenant}.<REGION>8.webtask.io/auth0-sso-dashboard/admins/login`.
3. Seleccione **Settings** en el menú desplegable debajo del nombre de su inquilino.

* Para especificar si los usuarios pueden ver la aplicación cuando está habilitada, seleccione **Publish** o **Unpublish**.
* Para actualizar la configuración de una aplicación, seleccione el icono de engranaje.
* Para eliminar una aplicación, seleccione la **X**. Aparecerá una ventana emergente para confirmar la eliminación.
