title: Extensión Authentication API Debugger
---

La extensión Authentication API Debugger le permite probar varios endpoints de la API de autenticación de Auth0.

<div id="install-the-extension">
  ## Instalar la extensión
</div>

Para instalar esta extensión:

1. Vaya a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y seleccione **Auth0 Authentication API Debugger**. Se abrirá la ventana **Install Extension**.
2. Seleccione **Install**.

<div id="authorize-the-extension">
  ## Autorice la extensión
</div>

Dado que la extensión se comunicará con la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip> en su nombre para obtener información sobre las aplicaciones que ha configurado en su <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, deberá autorizar el acceso.

Para autorizar la extensión:

1. Si está siguiendo esta guía, ya debería estar en la vista [Installed Extensions](https://manage.auth0.com/#/extensions) de [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions). Seleccione **Auth0 Authentication API Debugger** para abrir la extensión. Aparecerá un cuadro de diálogo de consentimiento solicitando acceso a su cuenta.
2. Dé su consentimiento para que la extensión acceda a su cuenta. Después de hacerlo, la extensión se abrirá.

<div id="configure-the-extension">
  ## Configurar la extensión
</div>

Después de instalar la extensión, debe realizar la configuración básica. Para configurar la extensión:

1. Si está siguiendo esta guía, ya debería haber iniciado la extensión. Seleccione la vista **Configuration**.

   <Frame>![Dashboard - Extensions - Authentication API Debugger - Configuration](/docs/images/cdy7uua7fh8z/5riI24ArqUAKRXhZfNWIQ5/ab7d6825288752132d14e54db1e563d0/configuration.png)</Frame>
2. Configure los siguientes parámetros y seleccione **Save**:

* **Domain**: El dominio de su inquilino. Este campo es de solo lectura y se muestra únicamente con fines informativos.
* **Application**: La aplicación para la que desea iniciar los flujos de autenticación. Puede administrar sus aplicaciones en [Auth0 Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications).
* **Callback URL**: La URL de devolución de llamada de esta extensión. También debe agregar esta URL a **Allowed Callback URLs** en la configuración de la aplicación seleccionada.
* **State**: Información de estado opcional que puede enviarse con el flujo de autenticación.
* **Connection**: El nombre de la conexión con la que desea iniciar sesión. Puede usar este parámetro para omitir la pantalla principal de inicio de sesión e ir directamente a la pantalla de inicio de sesión del <Tooltip tip="Proveedor de identidad (IdP): Servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/es/docs/glossary?term=Identity+Provider">Proveedor de identidad</Tooltip> correspondiente.

<div id="execute-the-flows">
  ## Ejecutar los flujos
</div>

Después de configurar la extensión, puede ejecutar cualquiera de los flujos seleccionando la vista **OAuth2/OIDC**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Para algunos flujos, es posible que deba especificar más opciones de configuración además de las aplicadas en la configuración básica. Estas opciones dependerán del flujo que se esté ejecutando, por lo que le sugerimos consultar la [documentación de la API de autenticación de Auth0](/es/docs/api) para obtener más información sobre los parámetros aplicables a cada flujo.
</Callout>

<Frame>![Dashboard - Extensions - Authentication API Debugger - Configuración de OAuth OIDC](/docs/images/cdy7uua7fh8z/72iYPxZU9OiHkKkHtgPTjI/fc166f3ae74e595c8d916276e78edb99/oauth-oidc-settings.png)</Frame>

Una vez que haya ejecutado un flujo determinado, verá una pantalla que muestra la información devuelta por Auth0. La información exacta dependerá del flujo ejecutado. A continuación, se muestra un ejemplo de los resultados tras ejecutar un flujo normal de **inicio de sesión con OAuth2/OIDC**:

<Frame>![Dashboard - Extensions - Authentication API Debugger - Flujo ejecutado](/docs/images/cdy7uua7fh8z/13TRS8SQUd4OsfLYRWMoe6/3478778905ad253160022db1f1f6f31a/flow-executed.png)</Frame>

Para ejecutar otro flujo, seleccione la vista **Login** para volver a la pantalla **Configuration**.
