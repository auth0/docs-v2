Auth0 AD/LDAP Connector Health Monitor expone un endpoint de API configurable para que pueda supervisar sus conectores AD/LDAP.

<div id="configure-the-extension">
  ## Configurar la extensión
</div>

Para instalar y configurar esta extensión:

1. Vaya a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y seleccione **Auth0 AD/LDAP Connector Health Monitor**. Se abrirá la ventana **Install Extension**.
2. Configure los siguientes parámetros y seleccione **Install**:

* **Auth0&#95;Domain**: El dominio de su aplicación de Auth0, que puede encontrar en la [configuración de la aplicación de Auth0](https://manage.auth0.com/#/applications/\{yourClientId}/settings).
* **Auth0&#95;Global&#95;Client&#95;ID**: El <Tooltip tip="ID de cliente: valor de identificación asignado por Auth0 a su recurso registrado." cta="Ver glosario" href="/es/docs/glossary?term=Client+ID">ID de cliente</Tooltip> global de su aplicación de Auth0, que puede encontrar en la sección **Global Application Information** de la [configuración avanzada del Tenant](https://manage.auth0.com/#/tenant/advanced).
* **Auth0&#95;Global&#95;Client&#95;Secret**: El <Tooltip tip="Secreto del cliente: secreto que usa un cliente (aplicación) para autenticarse con el Servidor de autorización; solo deben conocerlo el cliente y el Servidor de autorización, y debe ser lo suficientemente aleatorio como para que no se pueda adivinar." cta="Ver glosario" href="/es/docs/glossary?term=Client+Secret">Secreto del cliente</Tooltip> global de su aplicación de Auth0, que puede encontrar en la sección **Global Application Information** de la [configuración avanzada del Tenant](https://manage.auth0.com/#/tenant/advanced).

<div id="use-the-extension">
  ## Usa la extensión
</div>

Para obtener la URL que usarás con tu servicio de monitorización:

1. Ve a [Auth0 Dashboard &gt; Extensions](https://manage.auth0.com/#/extensions) y haz clic en **Installed Extensions**.

2. Selecciona **Auth0 AD/LDAP Connector Health Monitor**. Se abrirá una ventana nueva con un fragmento JSON similar a este:

   ```json lines
   {
     "message": "Use this url: 'https://sandbox.it.auth0.com/api/run/auth0user/auth0-ldap-conector-health-monitor?connection={MY-LDAP-CONNECTOR}' for monitoring your AD/LDAP connector."
   }
   ```

3. Introduce el ID de tu conexión LDAP. Para obtenerlo, ve a  [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise), haz clic en la conexión y copia el valor del campo **Identifier**.

Una vez hecho esto, podrás usar la URL proporcionada con tu servicio de monitorización para supervisar tus conectores AD/LDAP.
