<div id="add-custom-claims-to-the-issued-token">
  ## Agregue claims personalizados al token emitido
</div>

Para agregar claims personalizados a sus tokens, puede hacerlo creando una regla adicional que permita a la Authorization Extension hacerlo. Los claims personalizados pueden tener espacio de nombres o no tenerlo.

Para obtener mГЎs informaciГіn, lea [Crear claims personalizados.](/es/docs/secure/tokens/json-web-tokens/create-custom-claims)

Debe limitar la cantidad de claims que agrega al token.

```javascript lines
  var namespace = 'http://yourdomain/claims/'; // Puede definir su propio espacio de nombres, pero no use un dominio de Auth0

  // Agregue los tokens con espacio de nombres. Elimine los que no sean necesarios para su escenario
  context.idToken[namespace + "permissions"] = user.permissions;
  context.idToken[namespace + "groups"] = user.groups;
  context.idToken[namespace + "roles"] = user.roles;
  
  callback(null, user, context);

```
