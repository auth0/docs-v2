title: Usar plantillas de dashboard de Auth0 con Datadog
---

Auth0 proporciona plantillas predefinidas que puedes usar para visualizar los datos de tu inquilino de Auth0 en Datadog. Puedes mostrar visualizaciones para supervisar el estado del tráfico de inicio de sesión de un inquilino. Las plantillas te permiten usar las agregaciones recomendadas por Auth0 o tomarlas como punto de partida para crear tus propias visualizaciones personalizadas.

| Plantilla               | Propósito                                                                                                                                         | Archivo descargable                                                                                           |
| ----------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------- |
| Dashboard de seguridad | Supervisa el tráfico de inicio de sesión del inquilino, incluidas las métricas de inicio de sesión, creación de cuentas y detección de anomalías. | [Auth0SecDashboard.json](https://cdn.auth0.com/website/docs/logs/streams/datadog/Auth0SecurityDashboard.json) |

También puedes organizar tu visualización con las siguientes funciones:

* Usa la selección de tiempo de Datadog para elegir el intervalo de tu visualización.
* Usa filtros para acotar el análisis de datos con parámetros configurables, como la dirección IP, el país, el usuario y la aplicación cliente.
* Usa **Edit Widgets** para reorganizar los mosaicos y las visualizaciones, así como para crear copias en las que puedas inspeccionar y añadir parámetros a las consultas según sea necesario.

<div id="prerequisites">
  ## Requisitos previos
</div>

* [Configura Log Streaming para Datadog](https://marketplace.auth0.com/integrations/datadog-log-streaming).
* Descarga de Auth0 el archivo JSON de la plantilla de dashboard deseada.

<div id="upload-dashboard-templates-in-datadog">
  ## Cargar plantillas de dashboard en Datadog
</div>

1. Inicie sesión en [Datadog](https://www.datadoghq.com/).
2. En la barra de navegación izquierda, vaya a **Dashboards**.
3. Seleccione **Screenboard** como tipo de dashboard.
4. En el lienzo en blanco, arriba a la derecha, seleccione el icono de engranaje y haga clic en **Import Dashboard JSON**.
5. Cargue el archivo JSON del dashboard que quiera usar (por ejemplo, `Auth0SecDashboard.json`).

<div id="security-dashboard-visualizations">
  ## Visualizaciones del Dashboard de seguridad
</div>

Una vez que haya importado el Dashboard de seguridad de Auth0, puede usar las siguientes visualizaciones para evaluar el estado de su inquilino o como punto de partida para crear sus propias visualizaciones y alertas en Datadog.

| Visualización                            | Propósito                                                                                                                                                                                                         |
| ---------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Login Success                            | Muestra la cantidad de eventos de inicio de sesión correctos del inquilino.                                                                                                                                       |
| Login Failures                           | Muestra la cantidad de eventos de inicio de sesión fallidos del inquilino. Los picos en los errores de inicio de sesión pueden ser una señal de tráfico anómalo.                                                  |
| Login Failures by Geolocation            | Muestra la cantidad de eventos de inicio de sesión fallidos del inquilino por geolocalización. Los picos de tráfico inesperados en determinadas zonas geográficas pueden ser una señal de tráfico anómalo.        |
| Number of Unique IPs sending traffic     | Muestra la cantidad de direcciones IP que envían tráfico al inquilino. Este número suele seguir patrones predecibles, y las variaciones importantes pueden indicar tráfico anómalo.                               |
| Top 5 IPs with failures                  | Muestra las 5 direcciones IP con la mayor cantidad de errores de inicio de sesión.                                                                                                                                |
| Account Creation Events in top countries | Muestra la cantidad de cuentas creadas en los principales países. Las variaciones en los patrones previsibles y en las zonas geográficas pueden indicar tráfico anómalo.                                          |
| Account Creation Events by Platform      | Muestra la cantidad de cuentas creadas por plataforma, como Windows, OSX, Android o iOS.                                                                                                                          |
| IPs with most accounts created           | Muestra las direcciones IP desde las que se ha creado la mayor cantidad de cuentas.                                                                                                                               |
| Anomaly Detection Events                 | Muestra la cantidad de eventos de detección de anomalías del inquilino, incluida la cantidad de direcciones IP bloqueadas debido a la protección contra fuerza bruta y la detección de contraseñas comprometidas. |
| Flujo de registros                       | Muestra eventos sin procesar del registro del inquilino.                                                                                                                                                          |

<div id="customize-dashboard">
  ## Personalizar dashboard
</div>

Puede personalizar su dashboard de seguridad de Auth0 en Datadog para añadir widgets de datos personalizados.

1. Vaya a su dashboard de seguridad de Auth0 y seleccione la opción **Dashboards** en la navegación lateral.
2. Seleccione el nombre del dashboard que creó en la lista.
3. Seleccione una visualización y arrástrela a la sección principal de la página.
4. Configure las preferencias del gráfico de datos.
