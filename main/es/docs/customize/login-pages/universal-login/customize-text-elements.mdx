<Tooltip tip="Universal Login: Su aplicación redirige a Universal Login, alojado en el Servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/es/docs/glossary?term=Universal+Login">Universal Login</Tooltip> consta de una serie de páginas que facilitan varias acciones relacionadas con la cuenta, como iniciar sesión, inscribirse en factores de <Tooltip tip="Universal Login: Su aplicación redirige a Universal Login, alojado en el Servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/es/docs/glossary?term=multi-factor+authentication">autenticación multifactor</Tooltip> o restablecer contraseñas. Auth0 proporciona el texto de estas páginas en varios idiomas.

En algunos casos, es posible que desee modificar el texto de estas páginas para adaptarlo mejor al tono o a las necesidades específicas de su aplicación.

Puede personalizar el texto que se muestra en las páginas de [Universal Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) en todos los idiomas admitidos mediante el <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> o la <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>. Las siguientes definiciones, valores y variables se aplican a ambos métodos.

<div id="configure-prompts-and-screens">
  ## Configurar prompts y pantallas
</div>

Puede personalizar cualquiera de los [prompts y pantallas](#prompt-values) disponibles a través del Auth0 Dashboard o de la [Management API](https://auth0.com/docs/api/management/v2).

<Tabs>
  <Tab title="Usa el Dashboard">
    Puedes usar el Dashboard para personalizar los textos de New Universal Login.

    1. En el Dashboard, ve a [**Marca &gt; Universal Login &gt; Opciones avanzadas**](https://manage.auth0.com/#/custom_text) y, a continuación, selecciona **Texto personalizado**.
    2. Selecciona la **pantalla** y la **pantalla** que quieras editar, y luego haz los cambios que desees. A la derecha, hay una vista previa en tiempo real.
    3. Cuando estés listo para que tus cambios aparezcan en tu entorno de producción, selecciona **Guardar cambios.**
  </Tab>

  <Tab title="Usa Management API">
    Puedes usar la Management API para personalizar los textos de las pantallas de New Universal Login.

    1. Obtén un [token de acceso para la Management API](/es/docs/secure/tokens/access-tokens/management-api-access-tokens/get-management-api-access-tokens-for-production) con los alcances `read:prompts` y `update:prompts`.
    2. Llame al endpoint de Management API [Obtener texto personalizado para una pantalla](https://auth0.com/docs/api/management/v2#!/Prompts/get_custom_text_by_language) para obtener los datos de configuración existentes para la pantalla y el idioma especificados.
    3. Modifique los datos de configuración como desee, tomando como referencia los [parámetros admitidos](#supported-parameters) y los [valores de pantalla](#prompt-values).
    4. Llama al endpoint [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) de la Management API con los datos de configuración actualizados.

       <Warning>
         El endpoint de Management API [Establecer texto personalizado para una pantalla específica](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) reemplaza todos los datos de configuración existentes de la pantalla especificada por los datos de configuración proporcionados en el parámetro `body`.

         Si desea configurar varios valores de `screen` para una pantalla determinada, debe incluirlos todos en una sola llamada a la Management API.
       </Warning>

    #### Ejemplo

    Puedes configurar el componente de texto `description` para la pantalla `login` del prompt `login` con el valor `Log in to ACME's website` llamando al endpoint [Set custom text for a specific prompt](https://auth0.com/docs/api/management/v2#!/Prompts/put_custom_text_by_language) de la Management API de la siguiente manera:

    <AuthCodeGroup>
      ```bash cURL
      curl --request PUT \
        --url 'https://{yourDomain}/api/v2/prompts/login/custom-text/en' \
        --header 'authorization: Bearer {yourMgmtApiAccessToken}' \
        --header 'content-type: application/json' \
        --data '{ "login": { "description": "Login to ACME'''s Website" } }'
      ```

      ```csharp C#
      var client = new RestClient("https://{yourDomain}/api/v2/prompts/login/custom-text/en");
      var request = new RestRequest(Method.PUT);
      request.AddHeader("content-type", "application/json");
      request.AddHeader("authorization", "Bearer {yourMgmtApiAccessToken}");
      request.AddParameter("application/json", "{ "login": { "description": "Login to ACME's Website" } }", ParameterType.RequestBody);
      IRestResponse response = client.Execute(request);
      ```

      ```go Go
      package main

      import (
      	"fmt"
      	"strings"
      	"net/http"
      	"io/ioutil"
      )

      func main() {

      	url := "https://{yourDomain}/api/v2/prompts/login/custom-text/en"

      	payload := strings.NewReader("{ "login": { "description": "Login to ACME's Website" } }")

      	req, _ := http.NewRequest("PUT", url, payload)

      	req.Header.Add("content-type", "application/json")
      	req.Header.Add("authorization", "Bearer {yourMgmtApiAccessToken}")

      	res, _ := http.DefaultClient.Do(req)

      	defer res.Body.Close()
      	body, _ := ioutil.ReadAll(res.Body)

      	fmt.Println(res)
      	fmt.Println(string(body))

      }
      ```

      ```java Java
      HttpResponse<String> response = Unirest.put("https://{yourDomain}/api/v2/prompts/login/custom-text/en")
        .header("content-type", "application/json")
        .header("authorization", "Bearer {yourMgmtApiAccessToken}")
        .body("{ "login": { "description": "Login to ACME's Website" } }")

      ```


<div id="supported-parameters">
  ### Parámetros admitidos
</div>

| Parámetro  | Definición                                                                                                                                                                                                                                    |
| ---------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `prompt`   | El paso del flujo de inicio de sesión. Para ver una lista de los valores disponibles, consulte [Valores de `prompt`](#prompt-values).                                                                                                         |
| `language` | El idioma al que se aplica la personalización. Para ver una lista de los idiomas disponibles, consulte [Internacionalización de Universal Login](/es/docs/customize/internationalization-and-localization/universal-login-internationalization). |
| `screen`   | La pantalla específica dentro de un paso del flujo.                                                                                                                                                                                           |
| `key`      | El identificador de un componente de texto.                                                                                                                                                                                                   |
| `text`     | El mensaje que se muestra al usuario.                                                                                                                                                                                                         |

<div id="prompt-values">

<div id="variables">
  ## Variables
</div>

Algunas pantallas tienen variables en el texto que se sustituyen en tiempo de ejecución según la información contextual. Dado que hay variables diferentes disponibles para cada pantalla, no se garantiza que una variable determinada funcione en ningún otro lugar que no sea su pantalla original.

| Variable                      | Descripción                                                                                                             |
| ----------------------------- | ----------------------------------------------------------------------------------------------------------------------- |
| $&#123;clientName&#125;       | Nombre de la aplicación de Auth0                                                                                        |
| $&#123;connectionName&#125;   | Nombre de la conexión (p. ej., Google)                                                                                  |
| $&#123;tenantName&#125;       | Nombre del inquilino de Auth0                                                                                           |
| $&#123;organizationName&#125; | Nombre de la organización del usuario si este se autentica en el contexto de una Organización                           |
| $&#123;companyName&#125;      | Nombre del inquilino de Auth0 o nombre de la Organización si el usuario se autentica en el contexto de una Organización |
| $&#123;userName&#125;         | Nombre del usuario que ha iniciado sesión                                                                               |
| $&#123;email&#125;            | Correo electrónico del usuario que ha iniciado sesión                                                                   |
| $&#123;appName&#125;          | Nombre de la aplicación personalizada de Guardian Push                                                                  |
