La página de configuración principal te permite administrar los siguientes ajustes de Universal Login y Classic Login:

* **Motor de Login**: Alterna entre Universal Login y Classic Login.
* **Incrustación en iframes**: Permite que los flujos de Login se carguen dentro de iframes en entornos de desarrollo.

<div id="settings">
  ## Configuración
</div>

<div id="universal-or-classic-login">
  ### Universal Login o Classic Login
</div>

Elige qué experiencia de login usa tu inquilino. Si cambias esta configuración, se verán afectadas todas las aplicaciones que redirigen a tu página de login de Auth0. Para obtener más información, consulta [Universal Login vs. Classic Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login).

<Steps>
  <Step title="Ir a la configuración principal">
    Ve a **Marca &gt; Universal Login &gt; Personalización Pro-code** y selecciona **Configuración principal**.
  </Step>

  <Step title="Seleccionar el tipo de login">
    En **Configuración**, selecciona **Universal Login** o **Classic Login**.
  </Step>
</Steps>

<div id="iframe-embedding">
  ### Incrustación en iframe
</div>

Habilite Universal Login o Classic Login para que se carguen cuando la aplicación y el flujo de inicio de sesión estén incrustados en un iframe, como ocurre con plataformas de interfaz de usuario generativa como Vercel. Cuando se habilita, la incrustación en iframe solo se aplica en entornos de desarrollo.

<Warning>
  Cuando se habilita la incrustación en iframe, se desactivan los encabezados de seguridad que protegen frente al clickjacking.
</Warning>

<Steps>
  <Step title="Vaya a la configuración principal">
    Vaya a **Marca &gt; Universal Login &gt; Personalizaciones Pro-code** y seleccione **Configuración principal**.
  </Step>

  <Step title="Habilite la compatibilidad con iframe">
    En **Configuración de iframe**, active **Habilitar compatibilidad con plataformas de interfaz de usuario generativa**.
  </Step>

  <Step title="Configure los orígenes permitidos">
    En **URL de iframe permitidas para Login**, introduzca los orígenes permitidos.

    Para v0, introduzca los siguientes orígenes:

    * `https://v0.app`
    * `https://*.v0.app`
    * `https://*.vusercontent.net`

    Cuando se crea una aplicación desde una plataforma de interfaz de usuario generativa como Vercel mediante la integración de Auth0 disponible, Auth0 rellena previamente los orígenes permitidos desde la plataforma. Verifique siempre los orígenes prerrellenados para asegurarse de que estén completos. Para obtener más información, consulte [Integrate with Vercel](/es/docs/customize/integrations/integrate-with-vercel).
  </Step>
</Steps>
