Auth0 ofrece distintos SDK para usar con ACUL.

<div id="acul-sdks">
  ## SDKs de ACUL
</div>

* [Auth0 ACUL React SDK](/es/docs/libraries/acul/react-sdk)
  Proporciona Hooks de React para simplificar la integración en aplicaciones de React.

  * [Auth0 ACUL JS SDK](/es/docs/libraries/acul/js-sdk)
    La biblioteca principal de JavaScript (escrita en TypeScript), apta para usarse con cualquier framework de frontend o con JavaScript puro.

<div id="acul-sample-applications">
  ## Aplicaciones de ejemplo de ACUL
</div>

Las [aplicaciones de ejemplo de ACUL](https://github.com/auth0-samples/auth0-acul-samples) incluyen un pipeline de compilación completo, flujos de trabajo de implementación e implementaciones para pantallas comunes de Universal Login. Estas aplicaciones de ejemplo le ayudan a desarrollar sus pantallas. Los ejemplos están disponibles en React con los SDK de ACUL para React y JS.

Puede usar el comando `auth0 acul init` para inicializar un proyecto base de React con los SDK de ACUL para React o JS. Para obtener más información, consulte [Auth0 CLI](https://github.com/auth0/auth0-cli) en GitHub.

<div id="acul-resources">
  ## Recursos de ACUL
</div>

Auth0 ofrece varios recursos para personalizar tu experiencia de Universal Login.

<div id="auth0-management-api">
  ### Auth0 Management API
</div>

Actualice pantallas individuales de Universal Login o configure varias pantallas a la vez con Management API. Para usar Auth0 Management API, debe tener un [token de acceso](/es/docs/secure/tokens/access-tokens/management-api-access-tokens).

**Configuración de pantallas individuales**

* El [endpoint `GET /v2/prompts/{prompt}/screen/{screen}/rendering`](/es/docs/api/management/v2/prompts/get-rendering) le permite recuperar la configuración de una sola pantalla.

  * El [endpoint `PATCH /v2/prompts/{prompt}/screen/{screen}/rendering`](/es/docs/api/management/v2/prompts/patch-rendering) le permite configurar ACUL en una sola pantalla.

**Configuración masiva de pantallas**

* El [endpoint `GET /v2/prompts/rendering`](/es/docs/api/management/v2/prompts/get-all-rendering) le permite recuperar la configuración de todas las pantallas.

  * El [endpoint `PATCH /v2/prompts/rendering`](/es/docs/api/management/v2/prompts/patch-bulk-rendering) le permite actualizar varias pantallas a la vez.

<div id="auth0-developer-tools">
  ### Herramientas para desarrolladores de Auth0
</div>

Auth0 proporciona varias herramientas para simplificar la administración del inquilino, incluida la configuración de ACUL.

* **Auth0 CLI**<p />
  Use los comandos `acul` para crear proyectos, desarrollar localmente y desplegar sus configuraciones de pantalla. Para obtener más información, consulte [Auth0 CLI](https://github.com/auth0/auth0-cli) en GitHub.

  * **Auth0 Deploy CLI**<p />
    Gestione toda la configuración de su inquilino como código, incluidas las opciones de renderizado de ACUL definidas en archivos JSON. Para obtener más información, consulte [Auth0 Deploy CLI](https://github.com/auth0/auth0-deploy-cli) en GitHub.

  * **Proveedor de Terraform de Auth0**<p />
    Defina las configuraciones de pantalla de ACUL. Para obtener más información, consulte [`promt_screen_renderer resource`](https://registry.terraform.io/providers/auth0/auth0/latest/docs/resources/prompt_screen_renderer) en Terraform.
