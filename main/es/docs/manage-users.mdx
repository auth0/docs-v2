title: Administrar usuarios
---

La base de datos en la nube alojada por Auth0 almacena datos de usuario a los que puedes acceder a través de los perfiles de usuario. La información del usuario puede provenir de diversas fuentes, incluidos Proveedores de identidad, tus propias bases de datos y conexiones empresariales (como Active Directory o SAML).
