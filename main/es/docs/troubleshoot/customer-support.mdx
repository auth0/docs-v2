<div id="other-support-resources">
  ## Otros recursos de soporte
</div>

* La [página de estado de Auth0](https://status.auth0.com) contiene información sobre el estado actual del entorno de producción. Revisa la página de estado antes de abrir un ticket.

  Si la página de estado incluye una notificación sobre una interrupción del servicio, nuestro equipo ya está trabajando para restablecerlo y mantendrá la página de estado actualizada. Publicamos un análisis de causa raíz en la página de estado una vez investigado el problema. Puedes suscribirte para recibir notificaciones sobre cualquier cambio.

* El [centro de soporte oficial de Auth0](https://support.auth0.com/) ofrece información para la resolución de problemas y soluciones para muchos problemas comunes.

* La [comunidad pública de preguntas y respuestas](https://community.auth0.com/) de Auth0 ofrece soporte para todos. Puedes buscar entre las preguntas existentes y publicar una nueva si la tuya aún no se ha planteado. No se garantizan tiempos de respuesta para las preguntas publicadas.
