title: Verificar el dominio
---

* Asegúrese de que el dominio sea el mismo que se usó durante la autenticación.
* Si usa un [dominio personalizado](/es/docs/customize/custom-domains) de Auth0, es importante usar el mismo dominio que usa la aplicación para iniciar la autenticación. Consulte [Solucionar problemas de dominios personalizados](/es/docs/troubleshoot/integration-extensibility-issues/troubleshoot-custom-domains) para obtener más información.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Consulte [la página de monitorización](/es/docs/deploy-monitor/monitor) para obtener más información sobre cada uno de los eventos de registro que pueden ayudarle a solucionar problemas.
</Callout>
