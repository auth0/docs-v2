---
description: Descripción de cómo configurar una URI de redirección para el cambio de contraseña
title: Redirecciones del cambio de contraseña
---

La URI a la que se redirige al usuario después de un restablecimiento de contraseña correcto depende de las siguientes condiciones:

1. si tu inquilino usa Universal o Classic Login, y
2. si generas un ticket para cambiar la contraseña directamente mediante la Management API, o
3. si envías un correo electrónico para cambiar la contraseña desde la Authentication API, la página de inicio de sesión o alguna otra fuente.

El siguiente diagrama de flujo sirve de guía para configurar la URI de redirección de tu inquilino según su configuración.
