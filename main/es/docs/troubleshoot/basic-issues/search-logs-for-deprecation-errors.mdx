description: Aprenda a ver errores y descripciones relacionados con las deprecaciones.
title: Comprobar errores de obsolescencia
---

Cuando las funcionalidades de Auth0 quedan obsoletas, pueden aparecer errores o avisos en los logs del inquilino que indican que sus aplicaciones están utilizando funcionalidades obsoletas. Esta guía le ayudará a buscar en sus logs mensajes relacionados con la deprecación, así como a comprender las posibles causas y resoluciones de elementos concretos.

Hay dos formas distintas de buscar mensajes de advertencia que muestren el uso de funcionalidades obsoletas: el Dashboard o la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>. Tenga en cuenta que, en cualquiera de los casos, el [período de retención de logs](/es/docs/deploy-monitor/logs/log-data-retention) está determinado por el nivel de suscripción de su cuenta.

<div id="use-the-dashboard">
  ## Use el Dashboard
</div>

Si su aplicación usa una función obsoleta, aparecerá un mensaje de **Aviso de deprecación** en la sección **Logs** del Dashboard. Los avisos de deprecación solo se muestran una vez por hora (la primera vez que se produce el error dentro de esa hora), en lugar de mostrarse para cada transacción de autenticación que involucre la función obsoleta.

1. Vaya a [Dashboard &gt; Monitoring &gt; Logs](https://manage.auth0.com/#/logs) y busque mensajes relacionados con la deprecación escribiendo `type:depnote` en el cuadro de búsqueda. Si existen, aparecerá una lista de mensajes de advertencia relacionados con la deprecación en sus registros. El campo **Description** proporciona información sobre la función obsoleta específica que se utilizó.
2. Haga clic en el enlace de la columna **Type** de cada elemento para mostrar información adicional, como el ID de cliente que identifica la aplicación cliente que usa la función obsoleta.
3. Haga clic en las entradas de registro y seleccione la pestaña **Context Data** para mostrar detalles sobre el elemento.

<div id="use-the-management-api">
  ## Usa la Management API
</div>

También puedes usar la Management API para buscar estos mensajes en los logs con `type:"depnote"`.

1. Ve a la [Management API](https://auth0.com/docs/api/management/v2). Si todavía no lo has hecho, [obtén un token de API](/es/docs/secure/tokens/access-tokens/management-api-access-tokens/get-management-api-access-tokens-for-production).
2. En el panel izquierdo, ve a [Logs &gt; Buscar eventos de log](https://auth0.com/docs/api/management/v2/#!/Logs/get_logs) y luego desplázate hacia abajo hasta **Parámetros**.
3. En el campo **q**, escribe: `type:"depnote"`.
4. Haz clic en el botón **TRY**. El campo **ID de cliente** de los resultados indicará qué aplicación de tu inquilino está usando la funcionalidad obsoleta.
