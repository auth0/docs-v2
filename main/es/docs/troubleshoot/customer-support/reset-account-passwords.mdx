---
description: Describe cómo restablecer las contraseñas de las cuentas de Auth0.
title: Restablecer contraseñas de cuentas
---

Si necesitas cambiar tu contraseña o has olvidado la contraseña de tu cuenta de Auth0, en la mayoría de los casos puedes establecer una nueva contraseña desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Los restablecimientos de contraseña hacen que las sesiones de Auth0 expiren.

1. En la pantalla de inicio de sesión de la cuenta de Auth0, introduce la dirección de correo electrónico asociada a tu cuenta y haz clic en **Continuar**; luego, en la siguiente pantalla, selecciona **¿Olvidaste tu contraseña?**
2. Confirma tu dirección de correo electrónico y haz clic en **Continuar** para enviarla. Recibirás un correo electrónico con más instrucciones para restablecer tu contraseña.
3. Si tu solicitud se recibió correctamente, verás &quot;Revisa la dirección de correo electrónico [tu correo electrónico] para obtener instrucciones para restablecer tu contraseña&quot;.

Hay algunas circunstancias especiales para el restablecimiento de contraseña:

* Si has habilitado <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación del usuario que usa un factor además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/es/docs/glossary?term=MFA">MFA</Tooltip> y necesitas restablecer MFA para tu cuenta de administrador, ponte en contacto con [Auth0 Support](https://support.auth0.com).
* Si usas una cuenta social o empresarial para iniciar sesión en Auth0, restablece tu contraseña con el <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/es/docs/glossary?term=identity+provider">proveedor de identidad</Tooltip> correspondiente.
* Si eres un administrador que intenta restablecer la contraseña de un usuario, hay dos métodos:

  * Usa el flujo interactivo de restablecimiento de contraseña con Universal Login o la Authentication API, en el que el usuario recibe un correo electrónico con un enlace que abre la página de restablecimiento de contraseña de Auth0 para introducir una nueva contraseña.
  * Establece directamente la nueva contraseña con la Management API o el Dashboard.
