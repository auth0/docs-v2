Auth0 proporciona soporte de conformidad con los [Términos del servicio](https://auth0.com/terms) y los [planes de precios y soporte](https://auth0.com/pricing/). Estos son los tipos de soporte que ofrece Auth0:

| Nivel                   | Descripción                                                                                                                                                             |
| ----------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Soportado               | Auth0 ha probado formalmente estos elementos, puede proporcionar nuevas funcionalidades y correcciones de errores (si corresponde), y proporcionará soporte al cliente. |
| Mantenimiento           | Auth0 puede proporcionar correcciones de errores (si corresponde) para estos elementos y proporcionará soporte al cliente.                                              |
| Correcciones de errores | Auth0 puede proporcionar correcciones de errores (si corresponde) para estos elementos, pero no proporcionará soporte al cliente.                                       |
| Comunidad               | Auth0 no proporcionará soporte al cliente para estos elementos, y toda la asistencia provendrá de la comunidad en general.                                              |
| Partner                 | Auth0 no proporcionará soporte al cliente para estos elementos, y toda la asistencia provendrá de los partners que los mantienen.                                       |

<div id="auth0-support-coverage">
  ## Cobertura de Auth0 Support
</div>

El soporte oficial que brinda Auth0 se limita a los idiomas, las plataformas, las versiones y las tecnologías que se enumeran específicamente. Todo lo que no figure en la lista, o figure como compatible únicamente con la comunidad, no es elegible para Auth0 Support. Si tiene preguntas sobre elementos no compatibles, puede publicarlas en el [foro público de Auth0](https://community.auth0.com/).

Para los elementos compatibles con la comunidad, solicite ayuda poniéndose en contacto con los desarrolladores responsables de ellos, por ejemplo, mediante el envío de incidencias en los repositorios de GitHub de las bibliotecas o los SDK correspondientes.

Auth0 Support se limita a la versión más reciente de las tecnologías enumeradas (salvo que se especifique lo contrario). Si utiliza una versión anterior, deberá actualizarla antes de solicitar soporte.
