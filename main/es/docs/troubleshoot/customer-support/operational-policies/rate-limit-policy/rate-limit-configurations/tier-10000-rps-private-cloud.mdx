title: Rendimiento de Private Cloud 10.000 RPS (100x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  La capacidad total sostenida para Authentication API y MyAccount API en un entorno suele admitirse hasta 500 RPS; sin embargo, determinadas cargas de trabajo específicas, un uso combinado intensivo de Authentication API y Management API, o ciertos flujos de autenticación como ROPG en Authentication API pueden provocar una degradación del rendimiento incluso con valores de RPS inferiores.
</Callout>

A continuación se muestran los límites de tasa del tipo de suscripción Private Cloud Performance 10.000 RPS (100x).

Por lo tanto, recomendamos implementar un inquilino por entorno de nube privada para mitigar riesgos.
