title: Rendimiento de Private Cloud de 500 RPS (5x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total combinado de Authentication API y MyAccount API en un entorno suele estar admitido hasta 500 RPS; sin embargo, ciertas cargas de trabajo específicas, el uso intensivo combinado de Authentication API y Management API, o determinados flujos de autenticación, como ROPG en Authentication API, pueden degradar el rendimiento con valores de RPS inferiores.
</Callout>

Consulte a continuación los límites de tasa correspondientes al tipo de suscripción Rendimiento de Private Cloud de 500 RPS (5x).

Estos límites se aplican a cada inquilino que cree en el entorno de nube privada. Por lo tanto, recomendamos desplegar un inquilino por entorno de nube privada para mitigar riesgos.
