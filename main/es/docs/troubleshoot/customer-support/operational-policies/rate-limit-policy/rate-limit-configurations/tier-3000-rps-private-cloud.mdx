title: Private Cloud Performance 3000 RPS (30x) y 3000 RPS (30x) Burst
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  En general, se admite un rendimiento sostenido total de hasta 500 RPS entre Authentication API y MyAccount API en un entorno; sin embargo, ciertas cargas de trabajo particulares, un uso combinado intensivo de Authentication API y Management API, o determinados flujos de autenticación como ROPG en Authentication API pueden provocar una degradación del rendimiento con valores de RPS más bajos.
</Callout>

Consulte a continuación los límites de tasa correspondientes a los tipos de suscripción Private Cloud Performance 3000 RPS (30x) y 3000 RPS (30x) Burst. Estos límites se aplican a cada inquilino que cree en el entorno de nube privada.

Por lo tanto, recomendamos implementar un inquilino por entorno de nube privada para mitigar el riesgo.
