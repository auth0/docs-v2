title: Rendimiento de Private Cloud 1500 RPS (15x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  En un entorno, generalmente se admite un rendimiento sostenido total de hasta 500 RPS entre Authentication API y MyAccount API; sin embargo, ciertas cargas de trabajo específicas, un uso combinado intensivo de Authentication API y Management API, o determinados flujos de autenticación como ROPG en Authentication API, pueden degradar el rendimiento incluso con valores de RPS más bajos.
</Callout>

Consulte a continuación los límites de tasa del tipo de suscripción Rendimiento de Private Cloud 1500 RPS (15x). Estos límites se aplican a cada inquilino que cree en el entorno de nube privada.

Por lo tanto, recomendamos implementar un inquilino por entorno de nube privada para mitigar riesgos.
