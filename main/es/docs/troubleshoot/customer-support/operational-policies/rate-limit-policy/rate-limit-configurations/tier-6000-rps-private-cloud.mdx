title: Rendimiento de Private Cloud 6000 RPS (60x) y Burst de 6000 RPS (60x)
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  El rendimiento sostenido total de Authentication API y MyAccount API en un entorno suele admitirse hasta 500 RPS; sin embargo, ciertas cargas de trabajo específicas, el uso combinado intensivo de Authentication API y Management API, o determinados flujos de autenticación como ROPG en Authentication API pueden degradar el rendimiento con valores de RPS más bajos.
</Callout>

Consulte a continuación los límites de tasa de los tipos de suscripción Private Cloud Performance 6000 RPS (60x) y 6000 RPS (60x) Burst.

Por lo tanto, recomendamos implementar un inquilino por entorno de nube privada para mitigar riesgos.
