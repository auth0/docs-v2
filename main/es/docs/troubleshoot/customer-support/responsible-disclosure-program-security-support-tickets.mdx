---
title: Programa de divulgación responsable
description: Todos los clientes, incluso aquellos con planes de suscripción gratuitos, pueden reportar problemas de seguridad en el Programa de divulgación responsable de Auth0.
---

Todos los clientes, incluso aquellos con planes de suscripción gratuitos, pueden reportar problemas de seguridad. Lea la [Política de divulgación de vulnerabilidades de Auth0](https://www.okta.com/content/dam/okta---digital/en_us/legal/okta-vulnerability-disclosure-policy_v2.0.pdf) para obtener más información antes de enviar un reporte de seguridad.

Si desea publicar sus hallazgos, debe enviarnos el informe directamente a [psirt@okta.com](mailto:psirt@okta.com), preferiblemente con nuestra [clave de cifrado GPG](https://www.okta.com/vulnerability-reporting-policy/okta-pgp-key/).

Si desea recibir una recompensa, se aplican las [condiciones estándar de divulgación](https://www.bugcrowd.com/resources/essentials/standard-disclosure-terms/) de Bugcrowd, y debe enviar el informe a través de [Bugcrowd](https://bugcrowd.com/okta) para poder recibir una recompensa por su envío. Para obtener más información, visite el [programa de recompensas por errores de Auth0](https://bugcrowd.com/auth0-okta).

Para obtener información legal, lea [Información legal](https://auth0.com/legal).
