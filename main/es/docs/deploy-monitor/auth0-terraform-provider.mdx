title: Auth0 Terraform Provider
---

Deploy CLI no es la única herramienta disponible para administrar la configuración de su inquilino de Auth0; también existe un [Terraform Provider con soporte oficial](https://github.com/auth0/terraform-provider-auth0). [Terraform](https://terraform.io/) es una herramienta de terceros para representar como código la configuración de sus recursos en la nube. Cuenta con un marco de complementos consolidado que admite una amplia variedad de proveedores de nube, incluido Auth0.

Tanto Deploy CLI como Terraform Provider le ayudan a administrar las configuraciones de su inquilino de Auth0, pero cada uno tiene su propio conjunto de ventajas y desventajas.

Puede que le convenga considerar Auth0 Terraform Provider si:

* Sus flujos de trabajo de desarrollo ya utilizan Terraform
* Sus necesidades de administración del inquilino son granulares o solo abarcan unos pocos recursos específicos

Puede que no le convenga considerar Auth0 Terraform Provider si:

* Su flujo de trabajo de desarrollo no usa Terraform, lo que requiere configuración adicional inicial
* Sus flujos de trabajo de desarrollo se centran principalmente en administrar sus inquilinos en bloque
* Su inquilino tiene muchos recursos existentes, lo que puede requerir un esfuerzo considerable para “importarlos&quot;
