description: Recomendaciones sobre cómo puede mejorar su aplicación de Auth0 antes de implementarla en producción
title: 'Comprobaciones de preparación para producción: correcciones no críticas'
---

Las siguientes comprobaciones verifican si ha completado todas las recomendaciones (opcionales) para implementar correctamente en producción.

Para obtener más información sobre el resultado de estas comprobaciones, consulte [Ejecutar comprobaciones de producción](/es/docs/deploy-monitor/pre-deployment-checks/how-to-run-production-checks).

| Comprobación                                                                                                                                 | Descripción                                                                                                                                                                                                                                                                                                                                        |
| -------------------------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| Configurar Log Streaming                                                                                                                     | Asegúrese de transmitir las entradas de log del inquilino a una ubicación de almacenamiento o a un proveedor externo. Recomendamos que [configure un servicio de Log Streaming](/es/docs/customize/log-streams) para conservar los logs durante más de 30 días y facilitar un mejor análisis y la generación de alertas.                              |
| Debe configurarse [dominio personalizado](/es/docs/customize/custom-domains)                                                                    | Use [dominios personalizados con Universal Login](https://manage.auth0.com/#/tenant/custom_domains) para ofrecer la experiencia más fluida y segura a sus usuarios finales.                                                                                                                                                                        |
| Debe configurarse la [página de error personalizada](/es/docs/customize/login-pages/custom-error-pages)                                         | [Configure una página de error personalizada](https://manage.auth0.com/#/account) con los datos específicos de su aplicación y la marca corporativa.                                                                                                                                                                                               |
| Habilitar Universal Login                                                                                                                    | Asegúrese de usar la versión más reciente de [Universal Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login). Recomendamos actualizar su página de Login de la experiencia Classic a Universal Login para beneficiarse de una mejor accesibilidad y rendimiento, personalización sin código y nuevas funciones. |
| Deben configurarse las [plantillas de correo electrónico](/es/docs/customize/email/manage-email-flow)                                           | [Configure plantillas de correo electrónico personalizadas](https://manage.auth0.com/#/emails) con los datos específicos de su aplicación y la marca corporativa.                                                                                                                                                                                  |
| Debe configurarse la [autenticación multifactor](/es/docs/secure/multi-factor-authentication)                                                   | Se recomienda considerar la autenticación multifactor como una práctica segura para autenticar a sus usuarios.                                                                                                                                                                                                                                     |
| Está habilitada la [MFA para administradores del inquilino](/es/docs/get-started/manage-dashboard-access)                                       | [Habilite la autenticación multifactor](https://manage.auth0.com/#/account/admins) para los administradores del inquilino.                                                                                                                                                                                                                         |
| Debe definirse la [URL de cierre de sesión de redirección](/es/docs/authenticate/login/logout#set-the-allowed-logout-urls-at-the-account-level) | Revise las [URL de cierre de sesión de redirección permitidas](https://manage.auth0.com/#/account/advanced) de su aplicación.                                                                                                                                                                                                                      |
| Las aplicaciones deben usar RS256 en lugar de HS256 para el algoritmo de firma de JSON Web Token (JWT)                                       | Configure el [algoritmo de firma](/es/docs/get-started/apis#signing-algorithms) de JSON Web Token (JWT) como RS256 en lugar de HS256 para cada aplicación.                                                                                                                                                                                            |
| Establecer [URI de Login de la aplicación](/es/docs/get-started/applications/application-settings#application-uris)                             | Indique una [URI de Login de la aplicación](/es/docs/get-started/applications/application-settings#application-uris) para todas las aplicaciones interactivas.                                                                                                                                                                                        |
| Establecer [URL de cierre de sesión permitida del inquilino](/es/docs/get-started/applications/application-settings#application-uris)           | Cree al menos una ruta de cierre de sesión accesible públicamente y [actualice la configuración del inquilino](/es/docs/get-started/applications/application-settings#application-uris) para que apunte a ella.                                                                                                                                       |
| Establecer [URI de Login del inquilino](/es/docs/get-started/tenant-settings#login-and-logout)                                                  | Publique una ruta de Login predeterminada y [actualice la configuración de su inquilino](/es/docs/get-started/tenant-settings#login-and-logout) para que apunte a ella.                                                                                                                                                                               |
