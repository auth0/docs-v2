title: Retención de datos de registro
---

El período de retención de registros de Auth0 depende de su nivel de suscripción.

| Plan             | Retención de registros |
| ---------------- | ---------------------- |
| Starter          | 1 día                  |
| B2C Essentials   | 5 días                 |
| B2C Professional | 10 días                |
| B2B Essentials   | 5 días                 |
| B2B Professional | 10 días                |
| Enterprise       | 30 días                |

[Auth0 Support](https://support.auth0.com) puede ayudarle a integrar servicios externos de registro si necesita conservar los registros más allá del tiempo de retención. Este enfoque también resulta útil para consolidar registros de toda su organización.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Auth0 no proporciona registros en tiempo real para su inquilino. Aunque hacemos todo lo posible por indexar los eventos a medida que llegan, es posible que observe algunos retrasos.
</Callout>

Además, puede usar Log Streaming de Auth0 para exportar datos de registro. Para obtener más información, vaya a [Log Streaming](https://marketplace.auth0.com/features/log-streaming) en Auth0 Marketplace.
