Vaya a la página [Auth0 Public Cloud Status](https://status.auth0.com) para comprobar la disponibilidad del servicio de la versión de nube pública de Auth0.

Seleccione una **Región** en el menú desplegable para ver el estado de los servicios principales (como la autenticación de usuarios y la autenticación Machine-to-Machine) y los servicios de soporte (como <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuarios que utiliza un factor además del nombre de usuario y la contraseña, como un code enviado por SMS." cta="Ver glosario" href="/es/docs/glossary?term=MFA">MFA</Tooltip>, <Tooltip tip="Management API: un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>, registros del inquilino y CDN).

Para obtener más información sobre las regiones disponibles, consulte la tabla [Región, localidad y sublocalidad](/es/docs/get-started/auth0-overview/create-tenants#region-locality-and-sub-locality) en [Crear inquilinos](/es/docs/get-started/auth0-overview/create-tenants).

<div id="subscribe-to-public-cloud-status-updates">
  ## Suscribirse a las actualizaciones de estado de la nube pública
</div>

Seleccione una **Region** y, a continuación, seleccione **Subscribe** para recibir actualizaciones. Puede elegir entre dos opciones para consultar el estado:

1. Siga a [@auth0status](https://x.com/auth0status) en X para obtener las actualizaciones de estado más recientes.
2. Suscríbase al feed Atom o RSS de Auth0 para recibir actualizaciones de estado que afecten a su inquilino. Reemplace `{YOUR_TENANT_DOMAIN}` por el **dominio** de su inquilino.

Para encontrar el **dominio** de su inquilino, vaya a [Auth0 Dashboard &gt; Applications](https://manage.auth0.com/dashboard/#/applications), seleccione una aplicación y, a continuación, vaya a **Settings**. El **dominio** de su inquilino aparece en **Basic Information**.

<div id="check-auth0-status-for-private-cloud">
  ## Consulta el estado de Auth0 para Private Cloud
</div>

Ve a [Auth0 nube pública Status](https://status.auth0.com) y luego selecciona **Private Cloud Login**.

Una vez que te hayas autenticado, podrás ver el estado de cualquier entorno que no sea de desarrollo (por ejemplo, `travel0-prod-1`) si está disponible en la Auth0 Converged Platform.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Para acceder al estado de tu entorno de Private Cloud, debes ser miembro del inquilino de soporte de nube pública asociado. Para obtener más información, consulta [Open and Manage Support Tickets](/es/docs/troubleshoot/customer-support/open-and-manage-support-tickets) y [Manage Dashboard Access](/es/docs/get-started/manage-dashboard-access).
</Callout>

Para volver a la página de Auth0 nube pública Status, selecciona **Auth0 nube pública Status** en la navegación de la esquina superior derecha.

<div id="subscribe-to-private-cloud-status-updates">
  ## Suscribirse a las actualizaciones de estado de Private Cloud
</div>

Seleccione un entorno y luego **Subscribe**.

Suscríbase al feed Atom o RSS de Auth0 copiando la URL generada en el cuadro &#39;Subscribe&#39; para recibir actualizaciones de estado que afecten a su entorno.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  La URL generada del feed ATOM / RSS es única para cada entorno de Private Cloud y contiene tokens que pueden rotarse al [contactar con Soporte](https://support.auth0.com/).
</Callout>
