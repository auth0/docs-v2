El SDK JS de ACUL de Auth0 te permite crear experiencias personalizadas de Universal Login con un esfuerzo mínimo. Tanto si trabajas con el inicio de sesión, el registro, la autenticación sin contraseña o la inscripción de claves de acceso, este SDK está diseñado para gestionar flujos de autenticación modernos con facilidad.

<div id="why-choose-the-auth0-acul-js-sdk">
  ### ¿Por qué elegir el SDK JS de ACUL de Auth0?
</div>

* **Integración rápida:** Incorpora pantallas predefinidas como `login-id` o `signup-password`.
* **Funciones modernas:** Compatibilidad con inicio de sesión sin contraseña, passkeys e inicios de sesión sociales.
* **Módulos flexibles:** Importa solo lo que necesites.
* **Gestión unificada del estado:** Usa un único objeto de transacción para gestionar todos los flujos de autenticación.

<div id="architecture">
  ### Arquitectura
</div>

<div id="how-it-works">
  #### ¿Cómo funciona?
</div>

El SDK está diseñado pensando en la modularidad y la escalabilidad. Así es como encaja todo:

* **Pantallas:** Cada flujo de autenticación, como `login-id` y `signup-password`, es un módulo independiente.
* **Manejadores de envío:** Gestionan los envíos de formularios y las transiciones.
* **Formularios dinámicos:** Adaptan los campos y las validaciones en función de la configuración de la pantalla.
* **Propiedades auxiliares:** Permiten acceder a datos comunes, como la marca o la transacción, de forma coherente.
