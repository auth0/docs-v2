El SDK de Auth0 ACUL React te permite crear experiencias personalizadas de Universal Login con un esfuerzo mínimo. Tanto si trabajas con inicio de sesión, registro, autenticación sin contraseña o inscripción de claves de acceso, este SDK está diseñado para gestionar fácilmente flujos de autenticación modernos.

<div id="why-choose-the-auth0-acul-react-sdk">
  ### ¿Por qué elegir el SDK de Auth0 ACUL React?
</div>

* **Integración rápida:** Incorpora pantallas prediseñadas como `login-id` o `signup-password`.
* **Funciones modernas:** Compatible con autenticación sin contraseña, passkeys e inicios de sesión sociales.
* **Módulos flexibles:** Importa solo lo que necesites.
* **Gestión unificada del estado:** Usa un único objeto de transacción para gestionar todos los flujos de autenticación.

<div id="architecture">
  ### Arquitectura
</div>

<div id="how-it-works">
  #### ¿Cómo funciona?
</div>

El SDK se diseñó pensando en la modularidad y la escalabilidad. Así es como se integra todo:

* **Pantallas:** Cada flujo de autenticación, como `login-id` y `signup-password`, es un módulo independiente.
* **Controladores de envío:** Gestionan los envíos de formularios y las transiciones.
* **Formularios dinámicos:** Adaptan los campos y las validaciones según la configuración de la pantalla.
* **Propiedades auxiliares:** Permiten acceder de forma coherente a datos comunes, como la marca o la transacción.
