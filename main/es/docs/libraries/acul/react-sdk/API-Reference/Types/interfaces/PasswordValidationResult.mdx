---
title: "PasswordValidationResult"
---

El tipo de retorno de [`usePasswordValidation`](/es/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasswordValidation). Contiene el resultado general de validación y una lista detallada de las reglas de complejidad de la contraseña evaluadas.

```typescript
export interface PasswordValidationResult {
  isValid: boolean;
  results: PasswordComplexityRule[];
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="isValid" type="boolean" required>
  Resultado global de todas las comprobaciones de la política de contraseñas. `true` solo si cada regla de [`PasswordComplexityRule`](/es/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) tiene `status` igual a `"valid"`.
</ParamField>

<ParamField body="results" type={<span><a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule">PasswordComplexityRule</a>[]</span>} required>
  Array de reglas de contraseña evaluadas. Cada elemento es un [`PasswordComplexityRule`](/es/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule) cuyo campo [`status`](/es/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/PasswordComplexityRule#status) refleja el resultado de validación calculado para esa regla.
</ParamField>
