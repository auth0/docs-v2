---
title: "LanguageChangeOptions"
---

Opciones que acepta la función devuelta por [`useChangeLanguage`](/es/docs/libraries/acul/react-sdk/API-Reference/Hooks/useChangeLanguage).

```typescript
interface LanguageChangeOptions {
  language: string;
  persist?: "session";
  [key: string]: string | number | boolean | undefined;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="language" type="string" required>
  El código de configuración regional al que cambiar. Debe coincidir con una de las configuraciones regionales habilitadas en tu inquilino de Auth0. Ejemplo: `"en"`, `"fr"`, `"es"`.
</ParamField>

<ParamField body="persist" type="&#x22;session&#x22;">
  Ámbito de persistencia de la preferencia de idioma. Actualmente, solo se admite `"session"`. El valor predeterminado es `"session"`.
</ParamField>

<div id="remarks">
  ## Observaciones
</div>

* Se pueden pasar campos personalizados adicionales junto con `language` y `persist`. Los campos personalizados deben tener el prefijo `"ulp-"` y admitir valores de tipo `string`, `number`, `boolean` o `undefined`.
