VerifyPlatformAuthenticatorOptions

```ts Example
export interface VerifyPlatformAuthenticatorOptions extends CustomOptions {
  /**
   * Opcional. Si es true y `screen.showRememberDevice` es true, indica que el usuario
   * ha elegido recordar este dispositivo, normalmente durante 30 días, para omitir MFA en inicios de sesión posteriores.
   * @type {boolean}
   */
  rememberDevice?: boolean;
}
```

<ParamField body="rememberDevice?" type="boolean">
  Opcional. Si es `true`, intenta recordar el navegador para futuros desafíos de MFA.
  Corresponde al campo de formulario `rememberBrowser`. Solo se aplica si `screen.showRememberDevice` es `true`.
</ParamField>
