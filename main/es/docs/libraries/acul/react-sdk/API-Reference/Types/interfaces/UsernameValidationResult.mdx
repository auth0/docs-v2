---
title: "UsernameValidationResult"
---

El resultado de validar un username según una o varias reglas.

```ts Example
export interface UsernameValidationResult {
  isValid: boolean;
  errors: UsernameValidationError[];
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="isValid" type="boolean" required>
  Indica si el username cumple todas las reglas de validación.
</ParamField>

<ParamField body="errors" type={<span><a href="/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsernameValidationError">UsernameValidationError[]</a></span>} required>
  Una matriz de objetos [UsernameValidationError](/es/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/UsernameValidationError) que representa
  las reglas individuales que no se cumplieron, si las hay.

  Esta matriz está vacía si [UsernameValidationResult.isValid](#isvalid) es `true`.
</ParamField>
