title: "UseResendOptions"
---

Opciones de configuración para [useResend](/es/docs/libraries/acul/react-sdk/API-Reference/Hooks/useResend).

```ts Example
export interface UseResendOptions {
  timeoutSeconds?: number;
  onTimeout?: () => void;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="timeoutSeconds" type="number" default="10">
  Duración de la cuenta regresiva (en segundos) antes de que se permita un nuevo reenvío.
  El valor predeterminado es `10`.
</ParamField>

<ParamField body="onTimeout" type="() => void">
  Callback que se invoca cuando finaliza la cuenta regresiva y la acción de reenvío vuelve a estar disponible.
</ParamField>
