---
title: "SelectMfaPushDeviceOptions"
---

Opciones para seleccionar un dispositivo de notificaciones push de MFA.

```ts Example
export interface SelectMfaPushDeviceOptions {
  /**
   * El índice del dispositivo que se seleccionará de la lista de dispositivos inscritos.
   * El índice es base 0.
   */
  deviceIndex: number;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propiedades
</div>

<ParamField body="deviceIndex" type="number">
  Índice del dispositivo que se seleccionará de la lista de dispositivos inscritos.
  El índice comienza en 0.
</ParamField>
