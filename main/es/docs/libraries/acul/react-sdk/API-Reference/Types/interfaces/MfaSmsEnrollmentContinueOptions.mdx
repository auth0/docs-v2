---
title: "MfaSmsEnrollmentContinueOptions"
---

Opciones para continuar con la inscripción en MFA mediante SMS.

Extiende `CustomOptions` para permitir pasar opciones personalizadas junto con el teléfono y el captcha.

```ts Example
export interface MfaSmsEnrollmentContinueOptions extends CustomOptions {
  phone: string;
  captcha?: string;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="indexable">
  ## Indexable
</div>

[`key`: `string`]: `string` | `number` | `boolean` | `undefined`

<div id="properties">
  ## Propiedades
</div>

<ParamField body="phone" type="string" required>
  El número de teléfono del usuario para la inscripción en SMS.
</ParamField>

<ParamField body="captcha?" type="string">
  Token de CAPTCHA opcional para la verificación.
</ParamField>
