---
title: "UseErrorOptions"
---

Objeto de opciones para el hook [`useError`](/es/docs/libraries/acul/react-sdk/API-Reference/Hooks/useError). Controla cómo se muestran los errores en las pantallas de Universal Login de Auth0.

```ts Example
export interface UseErrorOptions {
  includeDevErrors?: boolean;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="includeDevErrors" type="boolean">
  Cuando es `true`, incluye detalles de error específicos del entorno de desarrollo en la salida de errores. Es útil para depurar durante el desarrollo; debe deshabilitarse en producción.
</ParamField>
