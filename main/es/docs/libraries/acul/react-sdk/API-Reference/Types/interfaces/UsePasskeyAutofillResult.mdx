El tipo de valor devuelto de [`usePasskeyAutofill`](/es/docs/libraries/acul/react-sdk/API-Reference/Hooks/usePasskeyAutofill).

```typescript
interface UsePasskeyAutofillResult {
  inputRef: RefObject<HTMLInputElement>;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="inputRef" type="RefObject<HTMLInputElement>" required>
  Una referencia de React para adjuntarla al campo de entrada del username. Cuando se vincula, el SDK configura automáticamente el atributo `autocomplete` del campo como `"username webauthn"`, lo que habilita la IU condicional del navegador para el autocompletado de passkeys. Vincular esta referencia es opcional si `autocomplete="username webauthn"` ya está declarado en tu marcado.
</ParamField>
