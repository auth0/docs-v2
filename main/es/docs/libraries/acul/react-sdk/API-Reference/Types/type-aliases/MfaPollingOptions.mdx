```ts Example
type MfaPollingOptions = {
  intervalMs?: number;
  onCompleted?: () => void;
  onError?: (error: Error) => void;
}
```

<ParamField body="MfaPollingOptions" type="object">
  Opciones de configuración para gestionar un proceso de sondeo de notificaciones MFA push.

  Proporcione estas opciones al iniciar una sesión de sondeo para controlar con qué frecuencia
  se consulta el endpoint de desafío y cómo se gestionan los casos de éxito o error.

  <Expandable title="propiedades">
    <ParamField body="intervalMs" type="number">
      Intervalo opcional, en **milisegundos**, entre solicitudes de sondeo consecutivas.

      * Si se omite, se usa el intervalo predeterminado interno del SDK (normalmente, 5000 ms).
      * Elija un valor que equilibre la capacidad de respuesta con las consideraciones sobre límites de tasa.

      ```ts Valor predeterminado
      Específico de la implementación (normalmente, 5000 ms)
      ```
    </ParamField>

    <ParamField body="onCompleted" type="() => void">
      Función de callback opcional que se ejecuta cuando el desafío de MFA push se aprueba
      correctamente y el sondeo finaliza.

      * Se llama exactamente **una vez**, tras lo cual el sondeo se detiene automáticamente.
      * Resulta útil para avanzar en el flujo de inicio de sesión (por ejemplo, llamando a `continue()`).
    </ParamField>

    <ParamField body="onError" type="(error: Error) => void">
      Función de callback opcional que se invoca si se produce un error durante el sondeo.

      Recibe un objeto [ULError](/es/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/Error) que contiene:

      * `status` — el código de estado HTTP de la solicitud fallida

      * `responseText` — el cuerpo de la respuesta sin procesar, si está disponible

      * Se llama una vez por cada evento de error; el sondeo puede reintentarse según el tipo de error.

      * Úselo para mostrar mensajes de error o activar lógica personalizada de reintento o cancelación.

      **Propiedades**

      <ParamField body="error" type={<a href="/es/docs/libraries/acul/react-sdk/API-Reference/Types/interfaces/Error">Error</a>}>
        Un objeto de error que contiene el código de estado HTTP y el cuerpo de la respuesta sin procesar de la solicitud de sondeo fallida.
      </ParamField>
    </ParamField>
  </Expandable>
</ParamField>
