title: "MfaPushListMembers"
---

Interfaz que define los métodos y propiedades disponibles para la pantalla mfa-push-list

```ts Example
export interface MfaPushListMembers extends BaseMembers {
  /**
   * Selecciona un dispositivo registrado de la lista para iniciar el push de MFA.
   * @param payload Las opciones que contienen el índice del dispositivo.
   */
  selectMfaPushDevice(payload: SelectMfaPushDeviceOptions): Promise<void>;
  /**
   * Regresa a la pantalla anterior.
   * @param payload Opciones personalizadas opcionales para incluir con la solicitud.
   */
  goBack(payload?: CustomOptions): Promise<void>;
}
```
