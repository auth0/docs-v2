ResendCodeOptions
descripción Opciones del método `resendCode` en la pantalla de verificación de correo electrónico de inicio de sesión.
Esta operación se utiliza cuando el usuario solicita que se envíe un nuevo code de verificación a su correo electrónico,
por ejemplo, si no recibió el code inicial o si ya venció.

```ts Example
export interface ResendCodeOptions extends CustomOptions {}
```
