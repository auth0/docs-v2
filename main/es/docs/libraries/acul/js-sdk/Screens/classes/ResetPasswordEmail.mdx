---
title: "ResetPasswordEmail"
description: "Describe todas las propiedades y los métodos disponibles para personalizar la pantalla `reset-password-email` de Universal Login."
---

La clase ResetPasswordEmail implementa la funcionalidad de la pantalla `reset-password-email`. Esta pantalla notifica al usuario que se ha enviado un correo electrónico de restablecimiento de contraseña y le permite reenviarlo si es necesario.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla ResetPasswordEmail:

```typescript Example
import ResetPasswordEmail from '@auth0/auth0-acul-js/reset-password-email';
const resetPasswordEmailManager = new ResetPasswordEmail();
resetPasswordEmailManager.resendEmail();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import ResetPasswordEmail from '@auth0/auth0-acul-js/reset-password-email';
  const resetPasswordEmailManager = new ResetPasswordEmail();
  resetPasswordEmailManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera la lista de errores de la transacción del contexto, o una lista vacía si no hay ninguno.
</ParamField>

<ParamField body="resendEmail" type="Promise<void>">
  Este método reenvía el correo electrónico de restablecimiento de contraseña a la dirección de correo electrónico registrada del usuario.

  ```typescript Example
  import ResetPasswordEmail from '@auth0/auth0-acul-js/reset-password-email';
  const resetPasswordEmailManager = new ResetPasswordEmail();
  resetPasswordEmailManager.resendEmail();
  ```
</ParamField>
