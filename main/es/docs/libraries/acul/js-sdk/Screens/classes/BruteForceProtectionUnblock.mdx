title: "BruteForceProtectionUnblock"
description: "Describe todas las propiedades y métodos disponibles para personalizar la pantalla `brute-force-protection-unblock` de Universal Login."
---

La clase BruteForceProtectionUnblock implementa la funcionalidad de la pantalla `brute-force-protection-unblock`. Esta pantalla se muestra cuando la cuenta de un usuario se ha bloqueado debido a varios intentos fallidos de inicio de sesión y necesita enviar una solicitud de desbloqueo.

</Frame>

<div id="constructors">
  ## Constructores
</div>

Cree una instancia del administrador de la pantalla BruteForceProtectionUnblock:

```typescript Example
import BruteForceProtectionUnblock from '@auth0/auth0-acul-js/brute-force-protection-unblock';
const bruteForceProtectionUnblockManager = new BruteForceProtectionUnblock();
await bruteForceProtectionUnblockManager.unblockAccount();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de visualización de la página de Universal Login.

  ```typescript Example
  import BruteForceProtectionUnblock from '@auth0/auth0-acul-js/brute-force-protection-unblock';
  const bruteForceProtectionUnblockManager = new BruteForceProtectionUnblock();
  bruteForceProtectionUnblockManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma que quieres usar (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Si se establece en `'session'`, el idioma seleccionado se mantiene durante toda la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera el arreglo de errores de la transacción desde el contexto, o un arreglo vacío si no hay ninguno.
</ParamField>

<ParamField body="unblockAccount" type="Promise<void>">
  Este método envía la solicitud para desbloquear la cuenta.

  ```typescript Example
  import BruteForceProtectionUnblock from '@auth0/auth0-acul-js/brute-force-protection-unblock';
  const bruteForceProtectionUnblockManager = new BruteForceProtectionUnblock();
  await bruteForceProtectionUnblockManager.unblockAccount();
  ```
</ParamField>
