La clase MfaBeginEnrollOptions implementa la funcionalidad de la pantalla `mfa-begin-enroll-options`. Esta pantalla gestiona la selección y la inscripción de factores de MFA.

</Frame>

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla MFA Begin Enroll Options:

```typescript Example
import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';
const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
await mfaBeginEnrollOptions.enroll({
  action: 'push-notification'
});
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma de la página de Universal Login.

  ```typescript Example
  import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';
  const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
  mfaBeginEnrollOptions.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código de configuración regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="enroll" type="Promise<void>">
  Este método continúa el proceso de inscripción con el factor seleccionado.

  ```typescript Example
  import MfaBeginEnrollOptions from '@auth0/auth0-acul-js/mfa-begin-enroll-options';

  const mfaBeginEnrollOptions = new MfaBeginEnrollOptions();
  await mfaBeginEnrollOptions.enroll({
    action: 'push-notification'
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [MfaEnrollOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/MfaEnrollOptions).
    </ParamField>

    <ParamField body="action" type="&#x22;push-notification&#x22; | &#x22;otp&#x22; | &#x22;sms&#x22; | &#x22;phone&#x22; | &#x22;voice&#x22; | &#x22;webauthn-roaming&#x22;" required>
      La acción que indica qué factor se va a inscribir. Uno de &quot;push-notification&quot;, &quot;otp&quot;, &quot;sms&quot;, &quot;phone&quot;, &quot;voice&quot; o &quot;webauthn-roaming&quot;.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="getErrors" type={<span><a href="/es/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método obtiene el array de errores de la transacción desde el contexto, o un array vacío si no hay ninguno.
</ParamField>
