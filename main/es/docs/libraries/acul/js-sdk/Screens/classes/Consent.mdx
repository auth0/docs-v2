---
title: "Consent"
description: "Describe todas las propiedades y los métodos disponibles para personalizar la pantalla `consent` de Universal Login."
---

La clase Consent implementa la funcionalidad de la pantalla `consent`. Esta pantalla muestra al usuario los permisos solicitados y recoge su decisión de permitir o denegar el acceso.

<div id="constructors">
  ## Constructores
</div>

Crea una instancia del administrador de la pantalla de consent:

```typescript Example
import Consent from '@auth0/auth0-acul-js/consent';
const consentManager = new Consent();
await consentManager.accept();
```

<div id="methods">
  ## Métodos
</div>

<ParamField body="accept" type="Promise<void>">
  Este método envía la decisión del usuario de aceptar (conceder) los permisos solicitados.

  ```typescript Example
  import Consent from '@auth0/auth0-acul-js/consent';
  const consentManager = new Consent();
  await consentManager.accept();
  ```
</ParamField>

<ParamField body="changeLanguage" type="Promise<void>">
  Este método cambia el idioma que se muestra en la página de Universal Login.

  ```typescript Example
  import Consent from '@auth0/auth0-acul-js/consent';
  const consentManager = new Consent();
  consentManager.changeLanguage({
    language: 'fr',
  });
  ```

  **Parámetros del método**

  <Expandable title="Parámetros">
    <ParamField body="options">
      [LanguageChangeOptions](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).
    </ParamField>

    <ParamField body="language" type="string" required>
      El código regional del idioma deseado (por ejemplo, `'en'`, `'fr'`, `'es'`).
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Cuando se establece en `'session'`, el idioma seleccionado se conserva durante la sesión.
    </ParamField>
  </Expandable>
</ParamField>

<ParamField body="deny" type="Promise<void>">
  Este método envía la decisión del usuario de denegar (rechazar) los permisos solicitados.

  ```typescript Example
  import Consent from '@auth0/auth0-acul-js/consent';
  const consentManager = new Consent();
  await consentManager.deny();
  ```
</ParamField>

<ParamField body="getErrors" type={<span><a href="/docs/libraries/acul/js-sdk/Screens/interfaces/Error">Error</a>[]</span>}>
  Este método recupera del contexto el array de errores de la transacción, o un array vacío si no hay ninguno.
</ParamField>
