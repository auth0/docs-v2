> **changeLanguage**(`options`: [`LanguageChangeOptions`](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions)): `Promise`&lt;`void`&gt;

Envía al servidor la nueva preferencia de configuración regional mediante la acción `change-language` y vuelve a renderizar la interfaz en el idioma seleccionado.

<div id="parameters">
  ## Parámetros
</div>

<ParamField body="options" type="LanguageChangeOptions" required>
  La configuración para cambiar el idioma. Consulta [`LanguageChangeOptions`](/es/docs/libraries/acul/js-sdk/Screens/interfaces/LanguageChangeOptions).

  <Expandable title="options">
    <ParamField body="language" type="string" required>
      Un código de configuración regional del idioma de destino (p. ej., `'en'`, `'fr'`, `'es'`). Debe ser uno de los códigos regionales habilitados en tu tenant de Auth0.
    </ParamField>

    <ParamField body="persist?" type="&#x22;session&#x22;">
      Ámbito de persistencia de la preferencia de idioma. Actualmente, solo se admite `"session"`.
    </ParamField>
  </Expandable>
</ParamField>

<div id="returns">
  ## Devuelve
</div>

`Promise`&lt;`void`&gt;

Se resuelve cuando se envía la preferencia de configuración regional y la pantalla se vuelve a renderizar en el idioma seleccionado.
