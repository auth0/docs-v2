---
title: "UsernameValidationError"
---

Representa un único error de validación de username devuelto durante el proceso de validación.

Cada error tiene un `code` legible por máquina y un `message` legible para las personas.

```ts Example
export interface UsernameValidationError {
  /**
   * Un identificador único para la regla de validación que falló.
   * 
   * Example: `"username-too-short"`, `"username-invalid-characters"`
   */
  code: string;

  /**
   * Una descripción legible por humanos del error.
   * 
   * Example: `"Username must be at least 3 characters long."`
   */
  message: string;

  /**
   * El campo asociado con el error de validación, generalmente "username", "email" o "phone".
   */
  field: string;
}
```
