carga útil para continuar con la activación del código de dispositivo.

Esta interfaz amplía `CustomOptions`, lo que significa que todas las propiedades de las opciones personalizadas
se integran en el nivel superior del objeto. Esto evita el anidamiento y proporciona
una estructura más simple para quienes consumen esta interfaz.

```ts Example
export interface ContinueOptions extends CustomOptions {
  code: string;
}
```
