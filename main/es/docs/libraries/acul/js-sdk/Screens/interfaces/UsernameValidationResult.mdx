---
title: "UsernameValidationResult"
---

El resultado de validar un username con una o más reglas.

```ts Example
export interface UsernameValidationResult {
  /**
   * Indica si el username superó todas las reglas de validación.
   */
  isValid: boolean;

  /**
   * Un array de objetos {@link UsernameValidationError} que representan
   * las reglas individuales que fallaron, si las hubiera.
   *
   * Este array está vacío si {@link UsernameValidationResult.isValid} es `true`.
   */
  errors: UsernameValidationError[];
}
```
