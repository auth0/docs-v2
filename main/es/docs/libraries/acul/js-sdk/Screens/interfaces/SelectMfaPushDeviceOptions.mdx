---
title: "SelectMfaPushDeviceOptions"
---

Opciones para seleccionar un dispositivo de MFA para notificaciones push.

```ts Example
export interface SelectMfaPushDeviceOptions {
  /**
   * El índice del dispositivo que se seleccionará de la lista de dispositivos inscritos.
   * El índice es base 0.
   */
  deviceIndex: number;
  [key: string]: string | number | boolean | undefined;
}
```

<div id="properties">
  ## Propiedades
</div>

<ParamField body="deviceIndex" type="number">
  El índice del dispositivo que se va a seleccionar de la lista de dispositivos inscritos.
  El índice empieza en 0.
</ParamField>
