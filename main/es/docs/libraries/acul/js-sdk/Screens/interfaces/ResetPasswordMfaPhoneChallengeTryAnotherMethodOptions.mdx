TryAnotherMethodOptions

```ts Example
export interface TryAnotherMethodOptions extends CustomOptions {
  /**
   * El método de entrega que se presentó o seleccionó inicialmente en la pantalla actual.
   * El endpoint de la API lo requiere para procesar correctamente la acción 'pick-authenticator'.
   * - `sms`: Indica que la pantalla actual correspondía al desafío por SMS.
   * - `voice`: Indica que la pantalla actual correspondía al desafío por voz.
   * @type {'sms' | 'voice'}
   */
  type: 'sms' | 'voice';
}
```
