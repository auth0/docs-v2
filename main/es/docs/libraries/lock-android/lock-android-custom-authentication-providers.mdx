**Auth0.Android** incluye la clase `WebAuthProvider` para gestionar el flujo de autorización mediante el navegador. Pero ¿qué ocurre si quieres usar tu propia implementación o una versión nativa de `AuthProvider`?

<div id="the-authprovider-class">
  ## La clase AuthProvider
</div>

Cree una clase que implemente la interfaz `AuthProvider` y redefina sus métodos.

También puede usar cualquier implementación **Native** ya proporcionada por Auth0. Los proveedores de [autenticación social nativa](/es/docs/libraries/lock-android/lock-android-native-social-authentication) disponibles actualmente son Google y Facebook.
