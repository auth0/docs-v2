description: Describe los errores comunes de registro e inicio de sesión que puedes encontrar al autenticar a los usuarios con las librerías de Auth0.
title: Errores comunes de autenticación en las librerías de Auth0
---

Las acciones o los datos introducidos por tus usuarios durante los procesos de registro o de inicio de sesión pueden provocar errores. A continuación, se muestra una lista de los errores más comunes que puedes recibir si utilizas alguna de las librerías de Auth0 para la autenticación.

<div id="sign-up">
  ## Registro
</div>

En caso de que falle el registro, los errores más comunes son:

| **Error**                                       | **Descripción**                                                                                    |
| ----------------------------------------------- | -------------------------------------------------------------------------------------------------- |
| **invalid&#95;password**                        | Si la contraseña utilizada no cumple con la política de contraseñas de la conexión                 |
| **invalid&#95;signup**                          | El usuario que intenta registrar no es válido                                                      |
| **password&#95;dictionary&#95;error**           | La contraseña elegida es demasiado común                                                           |
| **password&#95;no&#95;user&#95;info&#95;error** | La contraseña elegida se basa en información del usuario                                           |
| **password&#95;strength&#95;error**             | La [contraseña es demasiado débil](/es/docs/authenticate/database-connections/password-strength)      |
| **unauthorized**                                | Si no puede registrarse en esta aplicación. Puede deberse al incumplimiento de una Rule específica |
| **user&#95;exists**                             | El usuario que intenta registrar ya se registró                                                    |
| **username&#95;exists**                         | El username con el que intenta registrarse ya está en uso                                          |

<div id="log-in">
  ## Iniciar sesión
</div>

En caso de error al iniciar sesión, los errores más comunes son:

| **Error**                             | **Descripción**                                                                                                                                                                                                                                                                                                                                                            |
| ------------------------------------- | -------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **access&#95;denied**                 | Cuando se utiliza autenticación basada en web, el servidor de recursos deniega el acceso según las especificaciones de OAuth2                                                                                                                                                                                                                                              |
| **invalid&#95;user&#95;password**     | El username y/o la contraseña usados para la autenticación no son válidos                                                                                                                                                                                                                                                                                                  |
| **mfa&#95;invalid&#95;code**          | El code de <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación del usuario que usa un factor además del username y la contraseña, como un code enviado por SMS." cta="Ver glosario" href="/es/docs/glossary?term=multi-factor+authentication+%28MFA%29">autenticación multifactor (MFA)</Tooltip> proporcionado por el usuario no es válido o ha expirado |
| **mfa&#95;registration&#95;required** | El administrador ha requerido [autenticación multifactor](/es/docs/secure/multi-factor-authentication), pero el usuario no se ha inscrito                                                                                                                                                                                                                                     |
| **mfa&#95;required**                  | El usuario debe proporcionar el code de [autenticación multifactor](/es/docs/secure/multi-factor-authentication) para autenticarse                                                                                                                                                                                                                                            |
| **password&#95;leaked**               | Si la contraseña se ha visto comprometida y es necesario usar otra distinta                                                                                                                                                                                                                                                                                                |
| **PasswordHistoryError**              | La contraseña proporcionada para registrarse o actualizarse ya se ha usado anteriormente (se informa cuando la función [historial de contraseñas](/es/docs/authenticate/database-connections/password-options#password-history) está habilitada)                                                                                                                              |
| **PasswordStrengthError**             | La contraseña proporcionada no cumple los [requisitos de seguridad](/es/docs/authenticate/database-connections/password-strength) de la conexión                                                                                                                                                                                                                              |
| **too&#95;many&#95;attempts**         | La cuenta está bloqueada debido a demasiados intentos de inicio de sesión                                                                                                                                                                                                                                                                                                  |
| **unauthorized**                      | El usuario con el que intentas iniciar sesión está bloqueado                                                                                                                                                                                                                                                                                                               |
