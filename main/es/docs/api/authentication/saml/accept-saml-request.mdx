`GET /samlp/{clientId}`

Use este endpoint para aceptar una solicitud SAML e iniciar sesión.

Opcionalmente, puede incluir un parámetro `connection` para iniciar sesión con un proveedor específico. Si no se especifica ninguna conexión, se mostrará la [página de inicio de sesión de Auth0](https://auth0.com/docs/authenticate/login/auth0-universal-login).

Opcionalmente, las solicitudes de inicio de sesión iniciadas por SP pueden incluir un parámetro `organization` para autenticar a los usuarios en el contexto de una organización. Para obtener más información, consulte [Organizaciones](https://auth0.com/docs/manage-users/organizations).

<div id="parameters">
  ## Parámetros
</div>

<ParamField path="clientId" type="string" required>
  ID de cliente de la aplicación.
</ParamField>

<div id="request-body">
  ## Cuerpo de la solicitud
</div>

<ParamField body="connection" type="string">
  Conexión que se utilizará durante el inicio de sesión.
</ParamField>

<ParamField body="organization" type="string">
  ID de la organización, si la autenticación se realiza en el contexto de una organización.
</ParamField>

<div id="response">
  ## Respuesta
</div>

| Estado | Descripción                            |
| ------ | -------------------------------------- |
| 200    | Solicitud SAML aceptada correctamente. |
