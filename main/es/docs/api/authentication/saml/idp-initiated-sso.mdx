`POST /login/callback`

Este endpoint acepta una SAMLResponse de inicio de sesión iniciado por el IdP de un Proveedor de identidad SAML. La conexión correspondiente al proveedor de identidad se especifica en la cadena de consulta. El usuario será redirigido a la aplicación especificada en la sección de inicio de sesión iniciado por el IdP del proveedor SAML.

<div id="parameters">
  ## Parámetros
</div>

<ParamField body="connection" type="string" required>
  El nombre de un Proveedor de identidad configurado para su aplicación.
</ParamField>

<ParamField body="SAMLResponse" type="string" required>
  Una respuesta SAML de inicio de sesión iniciado por el IdP.
</ParamField>
