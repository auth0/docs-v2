title: "Eliminar un autenticador"
description: "Elimina un autenticador asociado con su ID. Puede obtener los ID de los autenticadores al enumerarlos."
---

`DELETE /mfa/authenticators/{AUTHENTICATOR_ID}`

Elimina un autenticador asociado con su ID. Puede obtener los ID de los autenticadores al enumerarlos.

Para acceder a este endpoint, debe incluir un Token de acceso en el encabezado Authorization con las siguientes claims:

* `scope`: `remove:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="parameters">
  ## Parámetros
</div>

<ParamField path="AUTHENTICATOR_ID" type="string" required>
  El ID del autenticador que se va a eliminar.
</ParamField>

<ParamField header="Authorization" type="string" required>
  El Token de acceso obtenido durante el inicio de sesión.
</ParamField>

<div id="response">
  ## Respuesta
</div>

| Estado | Descripción                               |
| ------ | ----------------------------------------- |
| 204    | Autenticador eliminado correctamente.     |
| 401    | No autorizado: token de acceso no válido. |
| 404    | No se encontró el autenticador.           |
