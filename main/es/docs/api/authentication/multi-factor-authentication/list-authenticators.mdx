<div id="endpoint">
  ## Endpoint
</div>

`GET /mfa/authenticators`

Devuelve una lista de autenticadores asociados a su aplicación.

Para acceder a este endpoint, debe establecer un Token de acceso en el encabezado Authorization, con las siguientes claims:

* `scope`: `read:authenticators`
* `audience`: `https://{yourDomain}/mfa/`

<div id="headers">
  ## Encabezados
</div>

<ParamField header="Authorization" type="string" required>
  Bearer ACCESS&#95;TOKEN
</ParamField>

<div id="response-schema">
  ## Esquema de respuesta
</div>

<ResponseSchema>
  La respuesta es un array de objetos de autenticador.

  <ResponseField name="id" type="string" required>
    El ID del autenticador.
  </ResponseField>

  <ResponseField name="authenticator_type" type="string" required>
    El tipo de autenticador.
  </ResponseField>

  <ResponseField name="active" type="boolean" required>
    Indica si el autenticador está activo.
  </ResponseField>

  <ResponseField name="oob_channels" type="string">
    Los canales OOB que admite este autenticador.
  </ResponseField>

  <ResponseField name="name" type="string">
    El nombre del autenticador.
  </ResponseField>
</ResponseSchema>

<div id="response-messages">
  ## Mensajes de respuesta
</div>

| Estado | Descripción              |
| ------ | ------------------------ |
| 200    | Lista de autenticadores. |
