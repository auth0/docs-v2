`POST /dbconnections/change_password`

Envíe un correo electrónico para cambiar la contraseña a la dirección de correo electrónico y la `connection` que proporcionó el usuario.

Opcionalmente, puede proporcionar un ID de organización para admitir variables específicas de la organización en las [plantillas de correo electrónico personalizadas](https://auth0.com/docs/customize/email/email-templates#common-variables) e incluir los parámetros `organization_id` y `organization_name` en la URL de **Redirect To**.

**Nota:** Este endpoint solo funciona con conexiones de base de datos.

<div id="remarks">
  ### Observaciones
</div>

* Cuando el usuario haga clic en el enlace para cambiar la contraseña, se le redirigirá a una página en la que se le pedirá que introduzca una nueva contraseña.
* Este endpoint devolverá tres encabezados de respuesta HTTP que proporcionan información relevante sobre sus límites de frecuencia:
  * `X-RateLimit-Limit`: Número de solicitudes permitidas por minuto.
  * `X-RateLimit-Remaining`: Número de solicitudes disponibles. Cada nueva solicitud reduce este número en 1. Por cada minuto transcurrido, se vuelve a añadir una solicitud, por lo que este número aumenta en 1 cada vez.
  * `X-RateLimit-Reset`: Tiempo restante hasta que se reinicie el límite de tasa (`X-RateLimit-Limit`). El valor está en [segundos de época en UTC](https://en.wikipedia.org/wiki/Unix_time).

<div id="parameters">
  ## Parámetros
</div>

<ParamField body="client_id" type="string" required>
  El `client_id` de su cliente.
</ParamField>

<ParamField body="email" type="string" required>
  La dirección de correo electrónico del usuario.
</ParamField>

<ParamField body="connection" type="string" required>
  El nombre de la conexión de base de datos asociada a su cliente.
</ParamField>

<ParamField body="organization" type="string">
  El `organization_id` de la Organización asociada al usuario.
</ParamField>

<div id="response">
  ## Respuesta
</div>

| Estado | Descripción                                                     |
| ------ | --------------------------------------------------------------- |
| 200    | Se ha enviado un correo electrónico para cambiar la contraseña. |
| 400    | Solicitud incorrecta; faltan parámetros o no son válidos.       |
| 404    | No se encontró al usuario.                                      |
