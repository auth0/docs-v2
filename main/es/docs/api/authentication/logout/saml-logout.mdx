`POST /samlp/{client_id}/logout`

Use este endpoint para cerrar la sesión de un usuario en un inquilino de Auth0 configurado como proveedor de identidad (IdP) de SAML. El comportamiento del cierre de sesión depende de la configuración del complemento SAML2 Web App para la aplicación del inquilino de Auth0 que actúa como IdP de SAML. Para obtener más información, consulte [Log Users Out of SAML Identity Providers](https://auth0.com/docs/authenticate/login/logout/log-users-out-of-saml-idps#configure-slo-when-auth0-is-the-saml-idp).

<div id="remarks">
  ### Observaciones
</div>

* El cuerpo de la solicitud POST debe contener un mensaje SAML `<LogoutRequest>` válido. Para obtener más información, consulte [Assertions and Protocols for the OASIS Security Assertion Markup Language (SAML) V2.0 on Oasis](https://docs.oasis-open.org/security/saml/v2.0/saml-core-2.0-os.pdf).

<div id="parameters">
  ## Parámetros
</div>

<ParamField path="client_id" type="string" required>
  ID de cliente de la aplicación configurada con el [complemento SAML2 Web App](https://auth0.com/docs/authenticate/protocols/saml/saml-sso-integrations/enable-saml2-web-app-addon).
</ParamField>

<div id="response">
  ## Respuesta
</div>

| Estado | Descripción                                                           |
| ------ | --------------------------------------------------------------------- |
| 200    | Se cerró la sesión correctamente                                      |
| 400    | Solicitud incorrecta, posiblemente debido a un mensaje SAML no válido |
