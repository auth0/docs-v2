<div id="endpoint">
  ## Endpoint
</div>

`POST /login/unlink`

<Note>
  Este endpoint está **obsoleto**. Use [DELETE /api/v2/users/{id}/identities/{provider}/{user_id}](/es/api/management/v2/users/delete-user-identity-by-user-id) en su lugar.
</Note>

Con el `access_token` y el `user_id` de un usuario autenticado, este endpoint desvincula la cuenta del usuario del proveedor de identidad.

<div id="learn-more">
  ### Más información
</div>

* [Desvincular cuentas de usuario](https://auth0.com/docs/manage-users/user-accounts/user-account-linking/unlink-user-accounts)

  ## Mensajes de respuesta
</div>

| Estado | Descripción                                       |
| ------ | ------------------------------------------------- |
| 200    | Se desvinculó correctamente la cuenta del usuario |
