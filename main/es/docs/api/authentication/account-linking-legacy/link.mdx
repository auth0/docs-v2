`GET /authorize`

<Note>
  Este endpoint está **obsoleto** para vincular cuentas. En su lugar, se debe usar [POST /api/v2/users/{id}/identities](/es/api/management/v2#!/Users/post_identities). Para obtener más información, consulta el [Aviso de migración](/es/migrations/past-migrations#account-linking-removal).
</Note>

Llama a este endpoint cuando un usuario quiera vincular un segundo método de autenticación (por ejemplo, una conexión de base de datos con usuario y contraseña con Facebook).

Este endpoint activará el flujo de login para vincular una cuenta existente con una nueva. Esto devolverá una redirección 302 a la `connection` que el usuario actual quiere agregar. El usuario se identifica mediante el Token de acceso que se devolvió al completar correctamente el inicio de sesión.

<div id="remarks">
  ### Observaciones
</div>

* El valor de `redirect_uri` debe especificarse como una URL de devolución de llamada válida en la [configuración de la aplicación](https://manage.auth0.com/dashboard).

<div id="parameters">
  ## Parámetros
</div>

<ParamField query="response_type" type="string" required>
  Usa `code` para flujos del lado del servidor y `token` para flujos del lado del cliente.
</ParamField>

<ParamField query="client_id" type="string" required>
  El `client_id` de tu aplicación.
</ParamField>

<ParamField query="connection" type="string">
  El nombre de la conexión configurada para tu aplicación. Si es null, se redirigirá a la [página de Login de Auth0](https://auth0.com/#/login_page) y se mostrará el Login Widget con la primera conexión de base de datos.
</ParamField>

<ParamField query="redirect_uri" type="string" required>
  La URL a la que Auth0 redirigirá el navegador después de que el usuario haya concedido la autorización.
</ParamField>

<ParamField query="access_token" type="string" required>
  El Token de acceso del usuario que ha iniciado sesión.
</ParamField>

<div id="response">
  ## Respuesta
</div>

| Estado | Descripción                                     |
| ------ | ----------------------------------------------- |
| 302    | Redirección a la conexión para vincular cuentas |
