Puede desactivar la rotación del <Tooltip tip="Token de actualización: token utilizado para obtener un nuevo Token de acceso sin obligar a los usuarios a volver a iniciar sesión." cta="Ver glosario" href="/es/docs/glossary?term=refresh+token">token de actualización</Tooltip> en cada aplicación desde el Dashboard o la <Tooltip tip="Token de actualización: token utilizado para obtener un nuevo Token de acceso sin obligar a los usuarios a volver a iniciar sesión." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>.

<div id="disable-with-the-dashboard">
  ## Deshabilitar con el Dashboard
</div>

1. En el [Auth0 Dashboard](https://manage.auth0.com/#), vaya a **Applications &gt; Applications** y seleccione la aplicación que desea configurar.
2. En la pestaña **Settings**, ubique la sección **rotación del token de actualización** y desactive el interruptor **Allow Refresh Token Rotation**.
3. Seleccione **Save Changes** en la parte inferior de la pantalla.

<div id="disable-with-the-management-api">
  ## Deshabilitar con la Management API
</div>

1. Deshabilite la rotación del token de actualización para cada aplicación con la Management API:

   {/* codeblockOld.header.login.logInButton codeblockOld.header.login.configureSnippet */}

   <AuthCodeBlock children={codeExample} language="javascript" />

2. Configure la token de actualización sin rotación de la siguiente manera:

   ```http lines
   PATCH /api/v2/clients/{client_id}
   {
      "refresh_token": {
        "rotation_type": "non-rotating",
        "expiration_type": "non-expiring"
      }
   }
   ```
