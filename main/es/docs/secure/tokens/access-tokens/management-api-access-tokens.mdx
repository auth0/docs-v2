description: Aprende cómo funcionan los tokens de acceso de la Management API y cómo usarlos.
title: tokens de acceso de la Management API
---

Para invocar la [Auth0 Management API](https://auth0.com/docs/api/management/v2), debes proporcionar un <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, que se utiliza para acceder a una API." cta="Ver glosario" href="/es/docs/glossary?term=access+token">token de acceso</Tooltip> que contenga la <Tooltip tip="Token de acceso: credencial de autorización, en forma de una cadena opaca o JWT, que se utiliza para acceder a una API." cta="Ver glosario" href="/es/docs/glossary?term=audience">audiencia</Tooltip> y los alcances adecuados.

<div id="audience">
  ## Audiencia
</div>

El valor de audiencia para <Tooltip tip="Management API: un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip> es `https://{yourDomain}/api/v2/`.

<div id="scopes">
  ## Alcances
</div>

Cada endpoint de Management API requiere un conjunto específico de alcances. Consulte el [Management API Explorer](https://auth0.com/docs/api/management/v2/) para obtener más información.

<div id="lifetime">
  ## Duración
</div>

La duración predeterminada del token para la Management API es de 86,400 segundos (24 horas).

Puede [configurar la duración del token](/es/docs/secure/tokens/access-tokens/update-access-token-lifetime) para cada API por separado.

<div id="security">
  ## Seguridad
</div>

* **Token de acceso**: Una vez emitido, un token de acceso no puede revocarse. Auth0 recomienda restringir la duración de los tokens de acceso al mínimo posible según su caso de uso.
* **<Tooltip tip="Token de actualización: Token utilizado para obtener un Token de acceso renovado sin obligar a los usuarios a volver a iniciar sesión." cta="Ver glosario" href="/es/docs/glossary?term=Refresh+token">Token de actualización</Tooltip>**: Puede [revocar un token de actualización](/es/docs/secure/tokens/refresh-tokens/revoke-refresh-tokens) mediante el Dashboard, la Authentication API o la Management API.
* **<Tooltip tip="Secreto del cliente: Secreto utilizado por un cliente (aplicación) para autenticarse con el Servidor de autorización; solo deben conocerlo el cliente y el Servidor de autorización, y debe ser lo suficientemente aleatorio como para que no pueda adivinarse." cta="Ver glosario" href="/es/docs/glossary?term=Client+secret">Secreto del cliente</Tooltip>**: Puede [rotar el secreto del cliente de una aplicación](/es/docs/get-started/applications/rotate-client-secret) mediante el Dashboard o la Management API.

<div id="quota">
  ## Cuota
</div>

Las cuotas de tokens se determinan según el nivel de suscripción. Puede [consultar su cuota actual](https://support.auth0.com/reports/quota) en el Auth0 Support Center. Si tiene preguntas sobre los precios o las cuotas, [póngase en contacto con nuestro equipo de ventas](https://auth0.com/contact-us).

* Los tokens emitidos para [audiencias internas de Auth0](/es/docs/secure/tokens/access-tokens/get-access-tokens#control-access-token-audience) no cuentan para su cuota.
* Los tokens emitidos para audiencias personalizadas cuentan para su cuota y están sujetos a los [límites de frecuencia de Auth0 Management API](/es/docs/troubleshoot/customer-support/operational-policies/rate-limit-policy/management-api-endpoint-rate-limits).
