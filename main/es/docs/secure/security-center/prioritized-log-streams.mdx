description: Más información sobre flujo de registros priorizado, que pueden proteger su automatización de seguridad.
title: Flujos de registros priorizados
---

**Flujos de registros priorizados** es una función del Dashboard que resalta eventos de seguridad. En caso de ataque o de un aumento de la actividad de los usuarios, flujos de registros priorizados y los eventos predefinidos seguirán activos para que los procesos y la automatización de seguridad posteriores no se vean afectados.

Puede crear un nuevo flujo de registros priorizado con un servicio compatible o configurar un webhook personalizado para transmitirlos a un servicio que ya utilice, y todos los registros pueden exportarse a un servicio de análisis de eventos. Los flujos de registros priorizados no cuentan para el límite de flujos de registros indicado en el plan de precios.

<div id="configure-a-prioritized-log-stream">
  #### Configurar un flujo de registros priorizado
</div>

1. Sigue las instrucciones de la [página de Log Streams](/es/docs/customize/log-streams) para configurar un nuevo flujo de registros.
2. Mientras configuras el nuevo flujo, ve a la pestaña **Settings** de tu Auth0 Dashboard y marca la casilla **Prioritized Log** para activar los eventos de seguridad predefinidos que se detallan a continuación. Selecciona **Save**.
3. Ve a la página de **Stream** y busca la etiqueta **Prioritized**.

<div id="prioritized-logs-event-types">
  #### Tipos de eventos de registros priorizados
</div>

Auth0 clasifica los siguientes códigos de evento como eventos relacionados con la seguridad:

| Código de evento  | Evento                                                       |
| ----------------- | ------------------------------------------------------------ |
| `limit_mu`        | Dirección IP bloqueada                                       |
| `limit_sul`       | Cuenta bloqueada (limitación por IP)                         |
| `limit_wc`        | Cuenta bloqueada (protección contra ataques de fuerza bruta) |
| `pwd_leak`        | Contraseña comprometida al iniciar sesión                    |
| `ublkdu`          | Se levantó el bloqueo de inicio de sesión del usuario        |
| `signup_pwd_leak` | Contraseña comprometida al registrarse                       |
| `reset_pwd_leak`  | Contraseña comprometida al restablecer                       |
| `gd_send_sms`     | SMS de MFA enviado                                           |
