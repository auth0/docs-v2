Auth0 proporciona dos API diferentes para ayudarle a usted y a sus usuarios a administrar los factores de autenticación <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación que utiliza un factor adicional además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/es/docs/glossary?term=MFA">MFA</Tooltip>.

<div id="auth0-authentication-api">
  ## API de autenticación de Auth0
</div>

Use los [endpoints de autenticación multifactor de la Authentication API](https://auth0.com/docs/api/authentication#multi-factor-authentication) para permitir que sus usuarios administren sus propios factores de autenticación de MFA.

Este método se basa en autenticarse mediante el flujo de autenticación <Tooltip tip="Propietario del recurso: entidad (como un usuario o una aplicación) capaz de conceder acceso a un recurso protegido." cta="Ver glosario" href="/es/docs/glossary?term=Resource+Owner">Propietario del recurso</Tooltip> Password Grant (ROPG). Para obtener más información, lea [Gestionar factor de autenticación MFA con Authentication API](/es/docs/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authenticator-factors-mfa-api).

<div id="auth0-management-api">
  ## Auth0 Management API
</div>

Use los [endpoints de usuarios de Management API](https://auth0.com/docs/api/management/v2) para gestionar los factores de autenticación MFA de sus usuarios.

Este método requiere autenticarse mediante una aplicación confidencial. Para obtener más información, consulte [Gestione los métodos de autenticación con Management API](/es/docs/secure/multi-factor-authentication/manage-mfa-auth0-apis/manage-authentication-methods-with-management-api).
