---
description: Enumera los códigos de error de Guardian y sus descripciones.
title: Referencia de códigos de error de Guardian
---

Usa los códigos de error para mostrar mensajes informativos y distinguir entre errores recuperables e irrecuperables.

| Error Code                            | Description                                                                                                                                    |
| ------------------------------------- | ---------------------------------------------------------------------------------------------------------------------------------------------- |
| `invalid_token`                       | Solicitud o token de transacción no válido                                                                                                     |
| `insufficient_scope`                  | No tienes permisos suficientes para realizar la operación solicitada                                                                           |
| `invalid_bearer_format`               | El token bearer incluido en el encabezado de autenticación no era válido                                                                       |
| `enrollment_conflict`                 | Ya existe otra inscripción para el mismo usuario. No puedes inscribirte dos veces.                                                             |
| `tenant_not_found`                    | No se encuentra el tenant asociado. Normalmente esto no debería ocurrir, salvo que hayas eliminado el tenant                                   |
| `login_transaction_not_found`         | La transacción de autenticación MFA no está activa o ya ha expirado                                                                            |
| `error_sending_push_notification`     | Error al enviar la notificación push                                                                                                           |
| `push_notification_wrong_credentials` | Error al enviar la notificación push debido a credenciales incorrectas                                                                         |
| `invalid_otp`                         | El código OTP proporcionado no era válido                                                                                                      |
| `invalid_recovery_code`               | El código de recuperación proporcionado no era válido                                                                                          |
| `invalid_body`                        | Falló la validación del cuerpo de la solicitud. Solicitud incorrecta.                                                                          |
| `invalid_query_string`                | Falló la validación de la cadena de consulta. Solicitud incorrecta.                                                                            |
| `enrollment_transaction_not_found`    | La transacción de inscripción MFA no está activa o ha expirado                                                                                 |
| `invalid_phone_number`                | El número de teléfono proporcionado no es válido                                                                                               |
| `error_sending_sms`                   | Error en la entrega del SMS                                                                                                                    |
| `feature_disabled`                    | La funcionalidad solicitada no está disponible globalmente en este momento (contacta con el proveedor)                                         |
| `feature_disabled_by_admin`           | La funcionalidad solicitada está deshabilitada actualmente por tu administrador                                                                |
| `pn_endpoint_disabled`                | No pudimos entregar la notificación push después de reintentarlo muchas veces. Intenta eliminar tu cuenta del dispositivo y volver a añadirla. |
| `too_many_sms`                        | Has superado la cantidad de SMS asignados a tu usuario                                                                                         |
| `too_many_pn`                         | Has superado la cantidad de notificaciones push asignadas a tu usuario                                                                         |
| `too_many_sms_per_tenant`             | Has superado la cantidad de SMS asignados a tu tenant                                                                                          |
| `too_many_pn_per_tenant`              | Has superado la cantidad de notificaciones push asignadas a tu tenant                                                                          |
| `field_required`                      | Se requiere un campo para realizar la operación (este error incluye un atributo field con un código para el campo: `otpCode`, `recoveryCode`)  |
| `method_not_found`                    | Has solicitado un método que actualmente no es compatible (no debería ocurrir)                                                                 |
| `no_method_available`                 | Actualmente no hay ningún método disponible para la inscripción (todos están deshabilitados)                                                   |
| `enrollment_method_disabled`          | El método de inscripción especificado está deshabilitado; este error también incluye un campo .method                                          |
| `auth_method_disabled`                | El método de autenticación especificado está deshabilitado; este error también incluye un campo .method                                        |
| `invalid_otp_format`                  | Error de validación del formato de OTP                                                                                                         |
| `invalid_recovery_code_format`        | Error de validación del formato del código de recuperación                                                                                     |
| `transaction_expired`                 | La transacción ya ha expirado                                                                                                                  |
| `already_enrolled`                    | Ya estás inscrito; no puedes inscribirte de nuevo                                                                                              |
| `not_enrolled`                        | No estás inscrito. Primero debes inscribirte                                                                                                   |
| `invalid_enrollment`                  | La inscripción proporcionada al método transaction#requestAuth no es válida o es null/undefined                                                |
