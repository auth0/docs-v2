title: Consejos generales de seguridad
---

Por lo general, los usuarios no pueden evitar que ciertos sitios sufran vulneraciones de seguridad, pero sí pueden hacer algunas cosas para ayudar a mantener sus cuentas seguras.

<div id="check-emails-carefully">
  ### Revise cuidadosamente los correos electrónicos
</div>

Verifique de dónde proviene un correo electrónico y los links que incluye. A menudo, los correos electrónicos de phishing no incluyen el nombre del usuario, sino algo genérico como &quot;Estimado cliente&quot;.

<div id="reset-passwords-directly-from-sites">
  ### Restablece las contraseñas directamente desde los sitios
</div>

Haz siempre el restablecimiento de contraseña desde el sitio real y no a través de enlaces potencialmente falsos en correos electrónicos. Además, ten en cuenta que la URL de un sitio web seguro siempre comienza con `https`.

Aquí tienes algunos enlaces para restablecer contraseñas en sitios de uso frecuente:

* [Google](https://www.google.com/accounts/recovery/)
* [Facebook](https://www.facebook.com/settings)
* [X](https://x.com/settings/password)

<div id="never-enter-personal-or-financial-information-in-emails">
  ### Nunca introduzca información personal o financiera en correos electrónicos
</div>

Los correos electrónicos, en general, no son muy seguros y no son una buena forma de comunicar información confidencial. Una empresa o aplicación de confianza no le pediría información de esta manera. Asegúrese de no introducir información confidencial a través de enlaces falsos en los correos electrónicos.

<div id="never-download-files-from-unreliable-sources">
  ### Nunca descargues archivos de fuentes no fiables
</div>

La mayoría de los navegadores web detectan sitios sospechosos. Debería aparecer una alerta cuando intentes acceder a un sitio malicioso. Nunca descargues archivos de correos electrónicos o sitios web sospechosos.

<div id="do-not-reuse-passwords">
  ### No reutilice contraseñas
</div>

Cuando un sitio sufre una filtración de datos de usuario, si un usuario utiliza las mismas credenciales en otro, también se puede acceder a la información de esos otros sitios. La única forma de evitarlo es no reutilizar contraseñas en varios sitios. El problema es que recordar innumerables contraseñas resulta frustrante y, a menudo, imposible. Una solución a este problema es usar un gestor de contraseñas. Hay muchos gestores de contraseñas disponibles que pueden ayudar a los usuarios a utilizar contraseñas distintas y seguras para cada cuenta sin tener que recordar todas ellas.

<div id="use-strong-passwords">
  ### Use contraseñas seguras
</div>

Cuanto más larga sea una contraseña, más difícil será adivinarla mediante métodos de fuerza bruta. Muchos sitios permiten usar frases de contraseña (una frase u oración en lugar de una sola palabra compleja). Procura que las contraseñas sean largas y usa una combinación de caracteres especiales, números y letras mayúsculas y minúsculas.

<div id="keep-software-current">
  ### Mantenga el software actualizado
</div>

Las aplicaciones publican parches y actualizaciones cuando detectan vulnerabilidades de seguridad en sus sistemas. Mantener actualizadas las aplicaciones, los navegadores web y los sistemas operativos puede ayudar a prevenir incidentes de seguridad.

<div id="check-the-security-of-your-email-inbox">
  ### Comprueba la seguridad de tu bandeja de entrada
</div>

Si usas Gmail, Google ofrece la herramienta [Security Checkup](https://myaccount.google.com/security-checkup) para informarte de si hay algún problema de seguridad relacionado con tu bandeja de entrada.

También puedes usar herramientas de terceros, como sitios web como [HaveIBeenPwned](https://haveibeenpwned.com/PwnedWebsites), para ver si podría haber problemas de seguridad asociados a tu dirección de correo electrónico.
