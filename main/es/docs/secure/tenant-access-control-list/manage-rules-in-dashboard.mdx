---
title: Administrar reglas en Auth0 Dashboard
description: Aprenda a administrar las reglas de Tenant ACL en Auth0 Dashboard. 
---

Puede usar Auth0 Dashboard para ver una lista de sus reglas de Tenant ACL y activarlas o desactivarlas individualmente.

<div id="view-list-of-rules">
  ## Ver la lista de reglas
</div>

Ve a [Dashboard &gt; Security &gt; Access Control](https://manage.auth0.com/#/security/access-control) para ver una lista de las reglas existentes.

| Nombre de la columna | Descripción                                                                                  | Campo de respuesta de la API relacionado |
| -------------------- | -------------------------------------------------------------------------------------------- | ---------------------------------------- |
| Prioridad            | Número ordinal que indica cuándo se evalúa la regla                                          | `priority`                               |
| Nombre               | Descripción del propósito o la funcionalidad de la regla                                     | `description`                            |
| Acción               | La [acción](/es/docs/secure/tenant-access-control-list/reference#actions) que realiza la regla. | `action`                                 |
| Estado               | Valor que indica si la regla está activa o deshabilitada.                                    | `active`                                 |
| Última actualización | Marca de fecha y hora de la última actualización de la regla.                                | `updated_at`                             |

![Ver la lista de reglas de Tenant ACL en el Auth0 Dashboard](/docs/images/cdy7uua7fh8z/4093f9a39aba48bdf8d579c134dc4da5/Screenshot_2025-09-03_at_12.33.47.png)

<div id="enable-or-disable-a-rule">
  ## Habilitar o deshabilitar una regla
</div>

Puede habilitar o deshabilitar una regla específica con el interruptor situado junto a ella.

El campo **Status** muestra la regla como `Active` o `Disabled`, según corresponda.
