title: Monitorear eventos de claves de acceso en los registros del inquilino
---

Puedes monitorear los flujos de claves de acceso asociados con tus conexiones de base de datos a través de los [registros del inquilino](/es/docs/deploy-monitor/logs).

Puedes ver los eventos de registro en el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> ([Monitoring &gt; Logs](https://manage.auth0.com/#/logs)) o recuperar los registros mediante la [Management API](https://auth0.com/docs/api/management/v2/logs/get-logs).

| Código de evento                | Escenario                                                                                             | Notas                                                                                                                                                                                                                                                                                                       |
| ------------------------------- | ----------------------------------------------------------------------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `ss`                            | Durante el registro, un usuario configuró una clave de acceso como método principal de autenticación. | Este evento de registro contiene el campo de array `details.authentication_methods` con la entrada `passkey`.<br /><br />Si un usuario se registró con una contraseña, este campo de array contiene la entrada `pwd`.                                                                                       |
| `s`                             | Durante el inicio de sesión, un usuario se autenticó con una clave de acceso.                         | Este evento de registro contiene el array `details.prompts`, que enumera el conjunto de pantallas por las que pasó un usuario durante el inicio de sesión.<br /><br />Si un usuario se autenticó con una clave de acceso, el array contiene un elemento con el siguiente campo: `"performed_amr": ["phr"]`. |
| `gd_enrollment_complete`        | Se completó correctamente un flujo de inscripción progresiva o local.                                 | Este evento de registro contiene el campo `details.authenticator.type` con el valor `passkey` si un usuario inscribió una clave de acceso.                                                                                                                                                                  |
| `gd_webauthn_enrollment_failed` | Se produjo un error durante un flujo de inscripción progresiva o local.                               | Este evento de registro contiene el campo de array `details.authentication_methods` con la entrada `passkey` si un usuario intentó inscribir una clave de acceso.                                                                                                                                           |
