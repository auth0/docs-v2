---
title: Universal Login vs. Classic Login
description: Compara las funciones disponibles en las experiencias de Universal Login y Classic Login
---

[Auth0 Universal Login](/es/docs/authenticate/login/auth0-universal-login) proporciona la funcionalidad esencial de un <Tooltip tip="Servidor de autorización: servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, su servidor de autorización puede controlar los datos, las tareas y las funciones disponibles para un usuario." cta="Ver glosario" href="/es/docs/glossary?term=authorization+server">servidor de autorización</Tooltip>: el flujo de inicio de sesión. Cuando un usuario necesita demostrar su identidad para acceder a su aplicación, puede redirigirlo a <Tooltip tip="Servidor de autorización: servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, su servidor de autorización puede controlar los datos, las tareas y las funciones disponibles para un usuario." cta="Ver glosario" href="/es/docs/glossary?term=Universal+Login">Universal Login</Tooltip> y dejar que Auth0 gestione el proceso de autenticación.

Universal Login es la principal solución de inicio de sesión alojada de Auth0. [Universal Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/universal-experience) incluye herramientas de personalización fáciles de usar y ofrece una experiencia más simple y rápida para los usuarios finales. Como alternativa, Auth0 también admite [Classic Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login/classic-experience), una experiencia de inicio de sesión alojada que utiliza controles de JavaScript para su personalización.

<Warning>
  En este momento, los esfuerzos de desarrollo activo de Auth0 se centran en Universal Login, y Classic Login ya no recibe actualizaciones. A menos que su caso de uso específico requiera la experiencia Classic, se recomienda implementar Universal Login.
</Warning>

Para ayudarle a elegir la solución que mejor se adapte a sus necesidades, la siguiente tabla compara las funciones de ambas experiencias de inicio de sesión.
