description: Describe la información que se muestra en la página de error predeterminada de Auth0 y los parámetros que se usan en los eventos de error de autorización.
title: Página de error predeterminada de Auth0
---

Si su URL de devolución de llamada es válida, cuando se produce un error de autorización, el <Tooltip tip="Servidor de autorización: servidor centralizado que contribuye a definir los límites del acceso de un usuario. Por ejemplo, su servidor de autorización puede controlar los datos, las tareas y las funcionalidades disponibles para un usuario." cta="Ver glosario" href="/es/docs/glossary?term=Authorization+Server">Servidor de autorización</Tooltip> devuelve los parámetros error y state correspondientes a su URL de devolución de llamada. Si su URL de devolución de llamada no es válida, la aplicación mostrará la página de error genérica predeterminada de Auth0. También puede personalizar las páginas de error. Para obtener más información, consulte [Páginas de error personalizadas](/es/docs/customize/login-pages/custom-error-pages).

Para encontrar el nombre de página predeterminado de la página de error genérica, consulte [Usar el control de versiones para administrar sus páginas de Universal Login](/es/docs/customize/login-pages/classic-login/version-control).

<div id="display">
  ## Visualización
</div>

La página de error genérica muestra la siguiente información si ha configurado los campos correspondientes en la [Configuración del Tenant](https://manage.auth0.com/#/tenant/).

| Campo                         | Descripción                                                         |
| ----------------------------- | ------------------------------------------------------------------- |
| Nombre amigable               | Nombre de su empresa                                                |
| URL del logotipo              | URL del logotipo de su empresa                                      |
| Correo electrónico de soporte | Dirección de correo electrónico del equipo de soporte de su empresa |
| URL de soporte                | URL de la página de soporte de su empresa                           |

<div id="parameters">
  ## Parámetros
</div>

El Servidor de autorización devuelve parámetros añadidos a la URL como una cadena de consulta.

| Parámetro           | Descripción                                                                   |
| ------------------- | ----------------------------------------------------------------------------- |
| `client_id`         | Identificador de la aplicación de Auth0.                                      |
| `connection`        | Conexión utilizada en el momento del error.                                   |
| `lang`              | Idioma configurado en el momento del error.                                   |
| `error`             | Código del error.                                                             |
| `error_description` | Descripción del error.                                                        |
| `tracking`          | Identificador que usa Auth0 para encontrar errores en los registros internos. |

Los parámetros que se muestran varían según el tipo de error y son específicos de la solicitud. Por ejemplo, si la solicitud no contenía un `client_id` y produjo un error, la página de error no devolverá ningún <Tooltip tip="ID de cliente: valor de identificación asignado por Auth0 a su recurso registrado." cta="Ver glosario" href="/es/docs/glossary?term=client+ID">ID de cliente</Tooltip>.
