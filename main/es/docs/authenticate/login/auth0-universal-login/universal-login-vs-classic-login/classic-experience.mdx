description: Descripción general de la experiencia de Classic Login
title: Experiencia de Classic Login
---

Classic Login es una experiencia de inicio de sesión alojada en Auth0 que se basa en JavaScript para su personalización. Implementar Classic Login es menos complejo que integrar directamente el proceso de autenticación en su aplicación, y puede ayudar a evitar los riesgos de la autenticación de origen cruzado.

Classic Login se basa en las bibliotecas de JavaScript de Auth0 ([Lock.js](/es/docs/libraries/lock), [auth0.js](/es/docs/libraries/auth0js), <Tooltip tip="Autenticación multifactor (MFA): proceso de autenticación de usuarios que utiliza un factor adicional además del nombre de usuario y la contraseña, como un código por SMS." cta="Ver glosario" href="/es/docs/glossary?term=MFA">MFA</Tooltip> Widget y Password Reset). Al personalizar las páginas de Classic Login en el <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>, las plantillas predeterminadas usan las mismas bibliotecas de JavaScript para crear una transición más natural entre la interfaz de usuario predeterminada y una personalizada.

Después de elegir una plantilla predeterminada para sus páginas de Classic Login, puede modificarla para adaptarla a sus necesidades. También puede personalizar diversos elementos de comportamiento y apariencia del widget Lock. Las plantillas de Auth0.js ofrecen flexibilidad adicional, ya que le permiten crear una interfaz de usuario personalizada y modificarla para que coincida con el estilo de su aplicación.

<Warning>
  Actualmente, Auth0 centra sus esfuerzos de desarrollo activo en Universal Login, y Classic Login ya no recibe actualizaciones. A menos que su caso de uso específico requiera la experiencia Classic, se recomienda implementar Universal Login.

  Para ver una comparación de características entre las experiencias de inicio de sesión, consulte [Universal Login vs. Classic Login](/es/docs/authenticate/login/auth0-universal-login/universal-login-vs-classic-login)
</Warning>
