title: Autenticación de Login en Auth0
sidebarTitle: Login
description: Conoce las formas de implementar la autenticación de inicio de sesión para tus usuarios con Universal Login de Auth0 o Login integrado.
---

Auth0 ofrece dos formas de implementar la autenticación de inicio de sesión en tus aplicaciones:

* **<Tooltip tip="Universal Login: Tu aplicación redirige a Universal Login, alojado en el Servidor de autorización de Auth0, para verificar la identidad de un usuario." cta="Ver glosario" href="/es/docs/glossary?term=Universal+Login">Universal Login</Tooltip>** donde los usuarios inician sesión en tu aplicación mediante una página alojada por Auth0.

* **Login integrado** donde los usuarios inician sesión en tu aplicación mediante una página que alojas tú.

Para la gran mayoría de los casos de uso, recomendamos Universal Login, pero también puedes usar Client-Initiated Backchannel Authentication para autenticar a los usuarios con un flujo de autenticación desacoplado.
