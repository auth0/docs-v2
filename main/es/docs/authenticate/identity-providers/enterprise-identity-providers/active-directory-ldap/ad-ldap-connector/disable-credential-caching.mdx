title: Deshabilitar el almacenamiento en caché de credenciales de una conexión AD/LDAP
---

Puede deshabilitar el almacenamiento en caché de credenciales a nivel de conexión para una [conexión AD/LDAP](/es/docs/authenticate/identity-providers/enterprise-identity-providers/active-directory-ldap/ad-ldap-connector) mediante <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Vaya a [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise) y seleccione el tipo de conexión **Active Directory/LDAP**.
2. Seleccione el nombre de la conexión que desea editar.
3. Active el interruptor **Disable Cache**.
