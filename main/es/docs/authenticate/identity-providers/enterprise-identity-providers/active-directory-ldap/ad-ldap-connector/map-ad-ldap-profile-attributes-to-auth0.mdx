---
description: Describe cómo asignar atributos de perfil de AD/LDAP a atributos del perfil de usuario de Auth0 mediante el Profile Mapper en la Connector Admin Console.
title: Asignar atributos de perfil de AD/LDAP al perfil de usuario de Auth0
---

Puede modificar la asignación de atributos de perfil de AD/LDAP a atributos del perfil de usuario de Auth0.

1. En un navegador, vaya a `http://localhost:8357` para iniciar la **Connector Admin Console** en el servidor del conector AD/LDAP.
2. Haga clic en **Profile Mapper** para abrir un editor de código que asigna atributos de un servicio de directorio de origen (representado por la variable `raw_data`) a una variable que se devuelve para rellenar el perfil de usuario de Auth0.
   La primera parte de la función crea una variable llamada `profile` e incluye una asignación para la parte principal del perfil de usuario de Auth0. Puede definir atributos adicionales debajo con una sintaxis de la forma:
   `profile['department'] = raw_data['companydept'];`
   En este ejemplo, `department` es el nombre del atributo en el perfil de usuario de Auth0 y `companydept` es el nombre del atributo en el servicio de directorio de origen (como AD).
3. Haga clic en **Save** cuando termine de modificar la configuración de LDAP.
4. Después de que el conector ejecute las pruebas, asegúrese de que todas se muestren en verde.

| Prueba   | Descripción                                                                                                                   | Solución de problemas                                                                                    |
| -------- | ----------------------------------------------------------------------------------------------------------------------------- | -------------------------------------------------------------------------------------------------------- |
| Prueba 1 | Intenta establecer una conexión TCP con el servidor LDAP y el puerto especificados.                                           | Verifique la conectividad básica de red y la configuración del firewall que podría impedir esa conexión. |
| Prueba 2 | Intenta realizar un enlace LDAP en el servidor LDAP y el puerto especificados con el username y la contraseña proporcionados. | Verifique la cadena de conexión LDAP, la ruta de búsqueda, el username y la contraseña.                  |
| Prueba 3 | Intenta realizar una búsqueda LDAP en el directorio para comprobar los privilegios del username especificado.                 | Verifique los privilegios del username en el directorio de destino.                                      |
| Prueba 4 | Intenta establecer una conexión con el servidor de Auth0.                                                                     | Verifique la conectividad de red y la configuración del firewall que podría impedir esa conexión.        |
