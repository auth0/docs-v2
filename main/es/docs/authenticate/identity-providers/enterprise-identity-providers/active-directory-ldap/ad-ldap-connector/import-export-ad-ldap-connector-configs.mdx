---
description: Describe cómo importar o exportar la configuración de un AD/LDAP Connector desde la Connector Admin Console.
title: Importar y exportar configuraciones de AD/LDAP Connector
---

Puede exportar la configuración de AD/LDAP Connector o importar una configuración exportada previamente. Esto resulta útil en implementaciones con más de un nodo de AD/LDAP Connector para alta disponibilidad. Puede configurar y probar la configuración en un nodo, exportar la configuración que funciona e importarla en todos los nodos posteriores.

1. En un navegador, vaya a `http://localhost:8357` para abrir la **Connector Admin Console** en el servidor de AD/LDAP Connector.
2. Haga clic en la pestaña **Import / Export**.
3. Exporte un archivo .zip que contenga el archivo `config.json`, la carpeta `certs` y el archivo `lib\\profileMapper.js` de la carpeta `%Program Files(x86)%\Auth0\AD LDAP Connector\`.
4. Para importar, cargue un archivo .zip de copia de seguridad en la pestaña **Import / Export**.
