description: Aprenda a actualizar los conectores AD/LDAP.
title: Actualizar los conectores AD/LDAP
---

Si hay varias instancias del conector AD/LDAP en su implementación, Auth0 le recomienda realizar los pasos que se indican a continuación para cada instancia, de una en una, de modo que solo una instancia esté inactiva en un momento dado.

<div id="admin-console-windows">
  ## Consola de administración (Windows)
</div>

1. Abra la Consola de administración en `http://localhost:8357/`. Si la Consola de administración tiene acceso saliente a Internet, verificará si hay una nueva versión disponible y la mostrará en la parte superior de la página:

   <Frame>![Pantalla de la Consola de administración para actualizar los conectores AD/LDAP](/docs/images/cdy7uua7fh8z/6Yp2uRnEU3RMeCqgHa6Ztl/b9c04ab83b9fd725910291b01ac9d05d/connector-update-available.png)</Frame>
2. Vaya a la pestaña **Update** para actualizar la instalación de la máquina actual a la versión más reciente. La actualización tardará unos 2 minutos y los registros del actualizador se mostrarán en la Consola de administración.

<div id="updater-script-windows">
  ## Script de actualización (Windows)
</div>

El script de actualización actualizará el Conector AD/LDAP desde la línea de comandos mediante la ejecución de los siguientes pasos:

1. Verifique si hay una actualización disponible.
2. Haga una copia de seguridad de la configuración existente, los certificados y `profileMapper.js`.
3. Desinstale el Conector AD/LDAP.
4. Descargue la actualización.
5. Instale el Conector AD/LDAP.
6. Restaure la configuración existente, los certificados y `profileMapper.js`.
7. Inicie el servicio de Windows.
8. Para ejecutar el script de actualización, ejecute la siguiente instrucción en la línea de comandos:
   `@powershell -NoProfile -ExecutionPolicy Bypass -Command "iex ((new-object net.webclient).DownloadString('https://cdn.auth0.com/connector/windows/update-connector.ps1'))"`

El script de actualización usa comandos específicos de PowerShell que solo están disponibles en Windows PowerShell 3.0 y versiones posteriores. Si ejecuta Windows 2008 o Windows 2008 R2, es posible que primero deba actualizar la versión de PowerShell.

Para obtener más información, consulte [Download Window Management Framework](https://www.microsoft.com/en-us/download/details.aspx?id=34595).

<div id="manual-update-windowslinux">
  ## Actualización manual (Windows/Linux)
</div>

1. Verifique la versión que tiene instalada. Pase el cursor sobre el indicador de estado del Conector en la consola. El texto emergente indicará el estado actual y la versión instalada.
2. Descargue la versión más reciente del [instalador](https://cdn.auth0.com/connector/windows/adldap-5.0.10.msi?_ga=2.58973869.1675886805.1589821069-1123834949.1586212750).
   Use la [versión](https://github.com/auth0/ad-ldap-connector/releases/tag/v5.0.10) del repositorio de GitHub para otras plataformas.
3. Haga una copia de seguridad de la configuración actual desde la Consola de administración o manualmente.

   1. En la Consola de administración, haga clic en el botón **Download** para generar un archivo .zip que contiene el archivo `config.json`, la carpeta `certs` y el archivo `lib\\profileMapper.js`.
   2. Para hacerlo manualmente, haga una copia de seguridad de los mismos archivos indicados anteriormente desde la carpeta `%Program Files(x86)%\Auth0\AD LDAP Connector\`. Esta ruta corresponde a Windows. En otras plataformas, los archivos pueden estar en otra ubicación, pero contendrán los mismos elementos.
4. Ejecute el instalador y siga las instrucciones.
5. Cierre el cuadro de diálogo Configuration sin cambiar nada.
6. Restaure los archivos desde la Consola de administración o manualmente.

   1. En la Consola de administración, cargue el archivo .zip en la pestaña **Import / Export**.
   2. Para hacerlo manualmente, copie los archivos de los que hizo una copia de seguridad anteriormente.
7. Reinicie el servicio **Auth0 AD LDAP** desde la consola de servicios.
