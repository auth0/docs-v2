---
description: Describe cómo habilitar conexiones empresariales para aplicaciones desde el Auth0 Dashboard.
title: Habilitar conexiones empresariales
---

Puede habilitar conexiones empresariales para aplicaciones desde el <Tooltip tip="Auth0 Dashboard: producto principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Para habilitar su conexión empresarial, primero debe haberla configurado.

1. Vaya a [Auth0 Dashboard &gt; Authentication &gt; Enterprise](https://manage.auth0.com/#/connections/enterprise) y seleccione el tipo de conexión que desea ver.

   <Frame>![Dashboard - Conexiones - Enterprise](/docs/images/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/b3454e60a4463e99353603fd11a71983/Enterprise_Connections_-_EN.png)</Frame>
2. Seleccione el nombre de la conexión que desea ver.

   <Frame>![Dashboard - Conexiones - Enterprise - AD/LDAP - Lista](/docs/images/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/4c0bd4a009df0b7f40a645613d3c723f/LDAP_page_-_EN.png)</Frame>
3. Seleccione la vista **Aplicaciones** y habilite o deshabilite la conexión para las aplicaciones correspondientes.

   <Frame>![Dashboard - Conexiones - Enterprise - Aplicaciones](/docs/images/cdy7uua7fh8z/2Jwoyem76rmja1sveD88It/38fbae514304297bbbf72beeedb8892f/SSL_-_EN.png)</Frame>

Cuando configure su conexión empresarial de <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/es/docs/glossary?term=IdP">IdP</Tooltip>, recuerde que el parámetro `connection` que se pasa al endpoint `/authorize`, `/whr` o `/wsfed` de Auth0 redirigirá al IdP de origen, **a menos que** esté conectada mediante una base de datos o Active Directory.
