---
description: Aprenda a encontrar el ID de una conexión.
title: Localizar el ID o el nombre de la conexión
---

Algunas tareas de administración de usuarios requieren un ID de conexión, que es un identificador único de una fuente de usuarios. Por ejemplo, si importa o exporta usuarios, o se conecta a <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/es/docs/glossary?term=identity+providers">proveedores de identidad</Tooltip>, debe proporcionar un ID de conexión.

1. Vaya a [Auth0 Dashboard](https://manage.auth0.com/#/), seleccione **Authentication** y, a continuación, el tipo de conexión cuyo ID necesita.
2. Seleccione el nombre de la conexión cuyo ID necesita.
3. En la vista **Settings**, busque el campo **Name**; contiene el ID de la conexión.
