---
title: Autenticación sin contraseña con inicio de sesión integrado
description: Describe cómo implementar la autenticación sin contraseña con Universal Login.
---

Para integrar la interfaz de usuario de inicio de sesión en su aplicación, puede usar nuestra [Embedded Passwordless API](/es/docs/authenticate/passwordless/implement-login/embedded-login/relevant-api-endpoints) o nuestros SDKs.

Según el tipo de aplicación que quiera crear, deberá implementarla de forma diferente:

* Para aplicaciones de página única (SPA) (por ejemplo, Angular o React), lea [Autenticación sin contraseña integrada para aplicaciones de página única (SPA)](/es/docs/authenticate/passwordless/implement-login/embedded-login/spa).
* Para aplicaciones nativas (por ejemplo, iOS, Android o aplicaciones de escritorio), lea [Inicio de sesión sin contraseña integrado en aplicaciones nativas](/es/docs/authenticate/passwordless/implement-login/embedded-login/native).
* Para aplicaciones web tradicionales (por ejemplo, NodeJS, Java, Rails o .NET), lea [Inicio de sesión sin contraseña integrado en aplicaciones web tradicionales](/es/docs/authenticate/passwordless/implement-login/embedded-login/webapps).

El inicio de sesión integrado en aplicaciones web tradicionales y SPA requiere configurar la autenticación entre orígenes. Para obtener más información, lea [Autenticación entre orígenes](/es/docs/authenticate/login/cross-origin-authentication).
