---
description: Aprenda a desaprovisionar usuarios de su integración SAML.
title: Desaprovisionar usuarios en integraciones SAML
---

Si necesita quitarles a los usuarios el acceso a la aplicación, como mínimo deberá desaprovisionarlos en el <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y gestiona identidades digitales." cta="Ver glosario" href="/es/docs/glossary?term=identity+provider">proveedor de identidad</Tooltip>. Según el proveedor de identidad, los pasos necesarios para desaprovisionar una cuenta de usuario varían. Consulte con su proveedor para obtener más instrucciones.

Una vez que la cuenta de un usuario se elimine o deshabilite en el proveedor de identidad, el usuario no podrá iniciar sesión.

También puede que quiera eliminar las cuentas de usuario de Auth0 de los usuarios que ya hayan sido desaprovisionados si Auth0 es el proveedor de servicios o si su aplicación se integra con Auth0. Independientemente de si Auth0 es el proveedor de identidad o de servicios, puede eliminar usuarios mediante el [Dashboard](https://manage.auth0.com/#/users) o mediante la <Tooltip tip="Management API: un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip>.
