Auth0 es una plataforma de identidad para gestionar el acceso a sus aplicaciones.

Si no tiene experiencia en la gestión de identidades y accesos (IAM), familiarícese con algunos conceptos básicos y planifique la solución que mejor se adapte a su tecnología y necesidades. Si ya está familiarizado con IAM, puede empezar de inmediato a crear.

***

</div>

Para empezar rápidamente, elija el tipo de aplicación para seguir un tutorial de inicio rápido paso a paso.

  <Card title="Aplicación nativa/móvil" href="/es/docs/quickstart/native">
    Aplicación móvil o de escritorio que se ejecuta de forma nativa en un dispositivo, como iOS o Android.
