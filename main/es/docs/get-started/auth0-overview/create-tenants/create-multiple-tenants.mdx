---
description: Describe cómo crear un inquilino adicional en el Auth0 Dashboard.
title: Crear varios inquilinos
---

Puede configurar varios inquilinos para crear [distintos entornos](/es/docs/get-started/auth0-overview/create-tenants/set-up-multiple-environments) en el <Tooltip tip="Auth0 Dashboard: el principal producto de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip> y así permitir configuraciones complejas. Por ejemplo, podría tener dos dominios independientes (uno interno y otro orientado al público), o quizá quiera que los usuarios inicien sesión de forma diferente en distintas aplicaciones. Para lograrlo, cree más de un inquilino de Auth0, de modo que pueda tener conjuntos independientes de aplicaciones, conexiones y usuarios para las aplicaciones y los grupos de usuarios que necesita admitir.

1. Vaya al [Auth0 Dashboard](https://manage.auth0.com/#/), seleccione el nombre de su inquilino y luego **Crear inquilino**.

   <Frame>![Menú desplegable del inquilino en el Dashboard para crear un inquilino](/docs/images/cdy7uua7fh8z/53KetqhNIGDs6N5cqJdQtz/5efad1137eb8b04031b30137c2fc5ac5/2025-02-26_14-51-39.png)</Frame>
2. Introduzca el **dominio del inquilino** que desee, seleccione una **región** y luego **Crear**.
