El [Auth0 Dashboard](https://manage.auth0.com/#) es donde administra todos los aspectos de su suscripción y la configuración de Auth0.

<Frame>![Página de actividad de Auth0 Dashboard](/docs/images/cdy7uua7fh8z/6k0cRLNEPko6vY6Kw7nIQD/4590d1b9d3e79f8d9c9970d733b08d02/2025-01-28_14-34-41.png)</Frame>

Consta de varias secciones por las que puede navegar mediante el menú de la barra lateral de la izquierda.

<div id="configure-implementation">
  ## Configurar la implementación
</div>

La siguiente tabla ofrece una breve descripción general de las diferentes secciones del Dashboard y de lo que puede hacer en cada una.

| -------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Aplicaciones**           | Administre sus aplicaciones, API e integraciones de inicio de sesión único (SSO).<br />**Aplicaciones**: Para cada una de las aplicaciones con las que quiera autenticar usuarios mediante Auth0, registre una aplicación.<br />**API**: Para cada una de las API que quiera proteger con Auth0, registre una API. Cree nuevas API y administre las existentes.<br />**Integraciones SSO**: Vea y habilite servicios externos para SSO. Cree nuevas integraciones SSO y configure, revise y administre su configuración.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                        |

Conéctese a bases de datos existentes mediante scripts de JavaScript basados en plantillas que se ejecutan en el servidor de Auth0 durante cada autenticación. Migre gradualmente a Auth0 una base de datos existente de credenciales heredadas a medida que los usuarios se autentican (no se requiere restablecimiento de contraseña).<br />**Social**: Configure proveedores de identidad sociales (como Facebook, X y Github) mediante los cuales sus usuarios puedan iniciar sesión.<br />**Empresarial**: Configure proveedores de identidad empresariales (como Active Directory, SAML y Office 365) mediante los cuales sus usuarios puedan iniciar sesión con sus credenciales empresariales.<br />**Sin contraseña**: Permita que sus usuarios se registren e inicien sesión mediante códigos OTP (enviados por correo electrónico o SMS) o enlaces de un solo clic, en lugar de contraseñas. |

<div id="manage-account-settings">
  ## Administrar la configuración de la cuenta
</div>

En la parte superior izquierda, puedes ver el nombre y el icono de tu Tenant, junto con una pequeña flecha. Esta flecha muestra un menú desplegable que puedes usar para configurar distintos aspectos de tu cuenta:

* [**Configuración**](/es/docs/get-started/tenant-settings): Configura varios aspectos de tu Tenant.
* [**Invitar a un miembro**](/es/docs/get-started/manage-dashboard-access): Agrega un usuario adicional como administrador o con otro rol a la configuración de tu Tenant.
* [**Uso de cuota**](https://support.auth0.com/reports/quota): Consulta información sobre el uso de cuota de tu suscripción y tus Tenants.
* [**Crear tenant**](/es/docs/get-started/auth0-overview/create-tenants): Úsalo para crear un tenant nuevo.
* **Cambiar de tenant**: Si tienes [varios Tenants](/es/docs/get-started/auth0-overview/create-tenants/create-multiple-tenants), usa esta opción para alternar entre ellos. Si creas una aplicación para un Tenant, no la verás en la lista de otro Tenant.

En la parte superior derecha, verás varias opciones en las que puedes hacer clic:

* **Buscar**: Busca aplicaciones, elementos del Marketplace, usuarios y otro contenido relacionado con tu Tenant.
* **Cuéntanos tus necesidades:** Ponte en contacto con un experto de Auth0 para que te ayude a alcanzar tus objetivos con Auth0.
* **Documentación:** Un enlace al sitio de documentación que estás leyendo ahora mismo.
* **Campana de notificaciones:** Te informa sobre nuevas comunicaciones de Auth0 o sobre la configuración de tu Tenant.
* **Perfil:** Un enlace a tu perfil del Tenant local, así como opciones para [cambiar tu tema](/es/docs/get-started/dashboard-profile/light-and-dark-themes) y cerrar sesión.
