Consulta esta guía de referencia para ver ejemplos en lenguaje natural que te permiten interactuar con tu Tenant de Auth0 mediante Auth0 MCP Server y cualquier [MCP Client](https://modelcontextprotocol.io/clients).

<div id="application-management">
  ## Gestión de aplicaciones
</div>

| Herramienta                 | Descripción                                                               | Ejemplos en lenguaje natural                                                                                                                                                                                                                                                                                                            |
| --------------------------- | ------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `auth0_list_applications`   | Enumera todas las aplicaciones del tenant de Auth0 o búscalas por nombre. | &quot;Muéstrame todas mis aplicaciones de Auth0.&quot;<br /><br />&quot;Busca aplicaciones con &#39;api&#39; en el nombre.&quot;<br /><br />&quot;¿Qué aplicaciones tengo en mi tenant de Auth0?&quot;                                                                                                                                  |
| `auth0_get_application `    | Obtiene los detalles de una aplicación de Auth0 específica.               | &quot;Muéstrame los detalles de la aplicación llamada &#39;Customer Portal&#39;.&quot;<br /><br />&quot;Obtén información sobre mi aplicación con ID de cliente {clientID}.&quot;<br /><br />&quot;¿Cuáles son las URL de devolución de llamada de mi &#39;Mobile App&#39;?&quot;                                                       |
| `auth0_create_application ` | Crea una nueva aplicación de Auth0.                                       | &quot;Crea una nueva aplicación de página única llamada &#39;Analytics Dashboard&#39;.&quot;<br /><br />&quot;Configura una nueva aplicación móvil nativa llamada &#39;iOS Client&#39;.&quot;<br /><br />&quot;Crea una aplicación Machine-to-Machine para nuestro servicio en segundo plano.&quot;                                     |
| `auth0_update_application`  | Actualiza una aplicación de Auth0 existente.                              | &quot;Actualiza las URL de devolución de llamada de mi &#39;Web App&#39; para incluir `https://staging.example.com/callback`.&quot;<br /><br />&quot;Cambia la URL de cierre de sesión de &#39;Customer Portal&#39;.&quot;<br /><br />&quot;Agrega metadatos del entorno de desarrollo a mi aplicación &#39;Admin Dashboard&#39;.&quot; |

</div>

<div id="set-up-a-new-api-with-client-access">

</div>

```bash wrap lines
Crea una nueva API llamada "Inventory API" con alcances de lectura y escritura, luego actualiza mi aplicación "Admin Dashboard" para que tenga acceso a ella.
```

<div id="create-and-deploy-a-security-action">
  #### Crear y desplegar una Action de seguridad
</div>

```bash wrap lines
Crea un Action que requiera MFA para los usuarios que inicien sesión desde fuera de nuestra red corporativa y, a continuación, impleméntalo en el flujo de login.
```

</div>

```bash wrap lines
Muéstrame todos los intentos de Login fallidos de la semana pasada e identifica cualquier patrón o actividad sospechosa.
```

<div id="best-practices">
  ## Prácticas recomendadas
</div>

1. Sé específico en tus solicitudes: Incluye nombres, ID u otra información identificativa al referirte a recursos concretos.
2. Usa lenguaje natural: No necesitas memorizar los nombres de las herramientas ni los parámetros; simplemente describe lo que quieres hacer.
3. Revisa antes de confirmar: Cuando Claude sugiera cambios, revísalos detenidamente antes de confirmarlos.
4. Empieza por lo básico: Comienza con consultas sencillas para familiarizarte con cómo Claude interpreta tus solicitudes.

<div id="troubleshooting">
  ## Solución de problemas
</div>

Si Claude no puede realizar una operación, ten en cuenta lo siguiente:

1. Autenticación: Asegúrate de haber iniciado sesión con Auth0 CLI (`auth0 login`).
2. Permisos: Verifica que tu cuenta tenga los permisos necesarios en Auth0.
3. Precisión: Intenta ser más específico en tu solicitud.
