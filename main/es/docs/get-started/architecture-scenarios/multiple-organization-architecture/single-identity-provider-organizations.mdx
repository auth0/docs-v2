---

En el escenario de Organizaciones con un único <Tooltip tip="Proveedor de identidad (IdP): servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/es/docs/glossary?term=Identity+Provider">Proveedor de identidad</Tooltip> (IdP), cada organización que usa la función [Auth0 Organizations](/es/docs/manage-users/organizations/organizations-overview) utiliza exactamente una conexión de Auth0. Esta función permite que cada aplicación seleccione la organización adecuada definida en el Tenant de Auth0, que autenticará directamente las credenciales del usuario o reenviará la solicitud de autenticación al IdP correspondiente para que la procese.

En este escenario, seguiremos usando el ejemplo de [Travel0 Corporate Booking](/es/docs/get-started/architecture-scenarios/multiple-organization-architecture). En este ejemplo, la aplicación de reservas corporativas equivale a su aplicación y Travel0 corresponde a su empresa.

<Frame>![Escenarios de arquitectura - Multitenencia - Diagrama - Usuarios aislados, aplicaciones compartidas, organizaciones A y B](/docs/images/cdy7uua7fh8z/1YBaMaMozamiTv8hXmNB2D/21e9270a59021a12209bdcb4345147cb/isolated-users_shared-apps_org-ab.png)</Frame>

<Warning>
  La función Auth0 Organizations actualmente admite aplicaciones que usan los protocolos [SAML](/es/docs/authenticate/protocols/saml/saml-configuration), [Open ID Connect](/es/docs/authenticate/protocols/openid-connect-protocol) y/o el marco [OAuth 2](/es/docs/authenticate/protocols/oauth). No se admiten aplicaciones que requieran [WS-Fed](/es/docs/authenticate/protocols/ws-fed-protocol).
</Warning>

Independientemente de cómo se procesen las credenciales, ya sea que las credenciales del usuario se almacenen en una conexión de base de datos (como en Hoekstra &amp; Associates), se validen mediante una conexión empresarial (como en MetaHexa Bank) o mediante una conexión social, su aplicación se comporta de la misma manera. La función [Organizations](/es/docs/manage-users/organizations/organizations-overview) le permite configurar y controlar esta experiencia.
