---
description: Describe cómo crear una integración de inicio de sesión único (SSO) para una aplicación. Solo para usar con tenants heredados.
title: Habilitar integraciones de inicio de sesión único para aplicaciones
---

De forma predeterminada, el <Tooltip tip="Single Sign-On (SSO): Servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión de ese usuario en otras aplicaciones." cta="Ver glosario" href="/es/docs/glossary?term=single+sign-on">inicio de sesión único</Tooltip> (SSO) sin interrupciones está habilitado para todos los nuevos tenants de Auth0; sin embargo, los **tenants heredados** pueden [elegir si habilitar esta función a nivel de tenant](/es/docs/get-started/tenant-settings/enable-sso-for-legacy-tenants). Si no has habilitado el SSO a nivel de tenant, puedes habilitarlo por aplicación.

Puedes habilitar el SSO para tu aplicación desde el <Tooltip tip="Auth0 Dashboard: El producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

<div id="prerequisite">
  ## Requisito previo
</div>

Antes de habilitar el SSO para una aplicación, primero debes crear y configurar una conexión para cada <Tooltip tip="Proveedor de identidad (IdP): Servicio que almacena y administra identidades digitales." cta="Ver glosario" href="/es/docs/glossary?term=identity+provider">proveedor de identidad</Tooltip> que quieras usar. En el caso de los proveedores de identidad sociales, asegúrate de que la conexión no utilice claves de desarrollador si usas [Universal Login](/es/docs/authenticate/login/auth0-universal-login).

<div id="enable-sso">
  ## Habilitar SSO
</div>

1. Ve a [Auth0 Dashboard &gt; Applications &gt; SSO Integrations](https://manage.auth0.com/#/externalapps) y selecciona el nombre de la integración para verla.
2. Desplázate hasta la parte inferior de la página **Configuración**, localiza **Usar Auth0 en lugar del IdP para el inicio de sesión único**, habilita el interruptor y selecciona **Guardar cambios**.
