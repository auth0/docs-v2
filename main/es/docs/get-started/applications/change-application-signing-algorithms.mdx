---
description: Describe cómo cambiar el algoritmo de firma de una aplicación mediante Auth0 Dashboard.
title: Cambiar el algoritmo de firma de una aplicación
---

Puede cambiar el <Tooltip tip="Algoritmo de firma: algoritmo utilizado para firmar digitalmente los tokens y garantizar que no hayan sido manipulados." cta="Ver glosario" href="/es/docs/glossary?term=signing+algorithm">algoritmo de firma</Tooltip> de su aplicación desde el Dashboard.

1. Vaya a [Dashboard &gt; Applications &gt; Applications](https://manage.auth0.com/#/applications) y seleccione el nombre de la aplicación para verla.

   <Frame>![Lista de aplicaciones del Dashboard](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Desplácese hasta la parte inferior de la pestaña **Settings** y seleccione **Show Advanced Settings**.
3. Vaya a la pestaña **OAuth**.

   <Frame>![Dashboard Applications Application Settings Tab Advanced Settings OAuth Tab](/docs/images/cdy7uua7fh8z/5bbq6VrI9tUMht0PwMimre/08574482efef944b16ecfbb445be50ab/Oauth_Tab_-_English.png)</Frame>
4. Busque **JsonWebToken Signature Algorithm** y seleccione el algoritmo de firma adecuado para la aplicación.
5. Haga clic en **Save Changes**.
