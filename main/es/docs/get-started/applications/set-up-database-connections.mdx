title: Configurar conexiones de base de datos
---

Puede configurar conexiones de base de datos para aplicaciones desde el <Tooltip tip="Auth0 Dashboard: la consola principal de Auth0 para configurar sus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Las conexiones de base de datos configuradas pueden usarse para iniciar sesión en su aplicación. Para configurar una conexión, siga estos pasos.

<AccordionGroup>
  <Accordion icon="circle-1" title="1 Vaya a Auth0 Dashboard">
    Vaya a [Auth0 Dashboard &gt; Authentication &gt; Database](http://manage.auth0.com/#/connections/database) y seleccione **Create DB Connection**.

    <Frame>![Dashboard - Authentication - Database - Lista de conexiones de base de datos](/docs/images/cdy7uua7fh8z/1Hf6ctA9xhsLOQO9IyG0VU/9061b8436368e003b165efb890d86473/DB_Connections_-_EN.png)</Frame>
  </Accordion>

  <Accordion icon="circle-2" title="Asigne un nombre a la conexión">
    Escriba un nombre para la conexión y seleccione **Create**.

    <Frame>![Dashboard - Authentication - Database - Nueva conexión de BD](/docs/images/cdy7uua7fh8z/224NpYTYh56bICfblpAJ6O/711d7fff6565d5dff7bde62ac5bf485a/Create_Database_Connections_-_English.png)</Frame>
  </Accordion>

  <Accordion icon="circle-3" title="Habilite la conexión">
    Seleccione la vista **Applications** y habilite el interruptor para cada aplicación de Auth0 que deba poder usar esta conexión.

    <Frame>![Dashboard - Connections - Database - Edit - Ver aplicaciones](/docs/images/cdy7uua7fh8z/Kjd9GRzWgHJis31lQVJlW/a9fcea0cc19b6602a94971b85b155bb0/2025-02-26_14-44-12.png)</Frame>
  </Accordion>
</AccordionGroup>

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Para permitir que las [aplicaciones de terceros](/es/docs/get-started/applications/third-party-applications) autentiquen a los usuarios mediante una conexión de base de datos, la conexión debe promoverse al nivel de dominio. Para obtener más información, consulte [Promover conexiones a nivel de dominio](/es/docs/authenticate/identity-providers/promote-connections-to-domain-level).
</Callout>
