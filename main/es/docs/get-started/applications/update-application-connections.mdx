---
description: Describe cómo actualizar las conexiones habilitadas de una aplicación mediante el Dashboard de Auth0.
title: Actualizar las conexiones de una aplicación
---

Puedes cambiar las conexiones habilitadas de tu aplicación desde el <Tooltip tip="Auth0 Dashboard: el producto principal de Auth0 para configurar tus servicios." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>.

1. Ve a [Dashboard &gt; Applications](https://manage.auth0.com/#/applications) y haz clic en el nombre de la aplicación para verla.

   <Frame>![Dashboard Applications List](/docs/images/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/cd608fcfae22e195b604e2707e5a848d/App_List_-_EN.png)</Frame>
2. Haz clic en la pestaña **Connections** y habilita o deshabilita las conexiones correspondientes para la aplicación.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Las [aplicaciones de terceros](/es/docs/get-started/applications/third-party-applications) pueden autenticar usuarios mediante conexiones promovidas al nivel de dominio o conexiones asignadas explícitamente a la aplicación. Para obtener más información, consulta [Promover las conexiones al nivel de dominio](/es/docs/authenticate/identity-providers/promote-connections-to-domain-level).
</Callout>
