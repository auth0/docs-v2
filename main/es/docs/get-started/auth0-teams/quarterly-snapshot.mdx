title: Información sobre Quarterly Snapshot
---

Quarterly Snapshot ofrece un resumen de los aspectos más destacados del uso y del valor a nivel de cuenta, incluida información agregada sobre cómo te sitúas con respecto a la cuota, tu entorno de identidad y la detección de amenazas.

La función se habilita automáticamente cuando se aprovisiona Auth0 Teams. Está disponible exclusivamente para determinados clientes Enterprise gestionados por su equipo de cuenta.

<div id="things-to-know">
  ## Aspectos que debe tener en cuenta
</div>

* Quarterly Snapshot solo está disponible a través de [Auth0 Teams](/es/docs/get-started/auth0-teams). Auth0 Teams debe estar aprovisionado para la cuenta antes del inicio del trimestre natural.
* Quarterly Snapshot es una funcionalidad disponible para un grupo selecto de clientes Enterprise gestionados por el DTAM.
* Los propietarios del equipo y los visualizadores de informes pueden ver y descargar Quarterly Snapshot en la sección **Reports** del Dashboard de Auth0 Teams.
* Los propietarios del equipo pueden administrar la lista de visualizadores de informes a través del Dashboard de Auth0 Teams.
* Los objetos de métricas de Quarterly Snapshot pueden variar en función de los derechos asignados a la cuenta, las funcionalidades habilitadas y el uso.
* Todas las métricas se agregan a nivel de cuenta. Para obtener más información, consulte la tabla **Metrics** que aparece a continuación.

<div id="metrics">
  ## Métricas
</div>

Estas son las métricas que puede ver en su Quarterly Snapshot.

| Métrica                                   | Qué mide                                                                                                                                                                                                                                                                                                                      | Comentarios                                                                                                                                   |
| ----------------------------------------- | ----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------- |
| Usuarios activos mensuales                | Mide la cantidad de usuarios activos mensuales únicos de la cuenta en comparación con su cuota.                                                                                                                                                                                                                               | Para obtener información detallada a nivel de inquilino, consulte el informe [Support Center Quota](https://support.auth0.com/reports/quota). |
| Tokens de Machine to Machine              | Muestra la cantidad de tokens de acceso emitidos por Auth0 para la [concesión de credenciales de cliente](/es/docs/get-started/authentication-and-authorization-flow/client-credentials-flow). No se cuentan los tokens emitidos para Auth0 Management API ni para otras API integradas de Auth0.                                | Para obtener información detallada a nivel de inquilino, consulte el informe [Support Center Quota](https://support.auth0.com/reports/quota). |
| Conexiones empresariales activas          | Esta métrica muestra la cantidad de conexiones empresariales que usa una cuenta.                                                                                                                                                                                                                                              | Para obtener más detalles sobre su cuota, póngase en contacto con su ejecutivo de cuenta.                                                     |
| Registros de usuarios nuevos              | Muestra cuántos usuarios nuevos (registros de usuario) se agregaron a sus inquilinos durante los meses de este trimestre.                                                                                                                                                                                                     | Consulte la [página Activity](/es/docs/get-started/auth0-overview/dashboard/activity) para obtener más información sobre los usuarios nuevos.    |
| Usuarios inactivos                        | Esta métrica representa el promedio mensual de usuarios que no se han autenticado en el último mes.                                                                                                                                                                                                                           |                                                                                                                                               |
| Contraseña comprometida                   | Cantidad de intentos bloqueados con credenciales comprometidas por la función [detección de contraseñas comprometidas](/es/docs/secure/attack-protection/breached-password-detection).                                                                                                                                           |                                                                                                                                               |
| Protección contra ataques de fuerza bruta | Solicitudes bloqueadas por la función [protección contra ataques de fuerza bruta](/es/docs/secure/attack-protection/brute-force-protection), si está habilitada. Esta métrica resume todas las solicitudes bloqueadas específicamente en los inquilinos de producción.                                                           |                                                                                                                                               |
| Suspicious IP Throttling                  | Solicitudes bloqueadas por la función [Suspicious IP Throttling](/es/docs/secure/attack-protection/suspicious-ip-throttling), si está habilitada. Esta métrica suma todas las solicitudes bloqueadas específicamente en los inquilinos de producción.                                                                            |                                                                                                                                               |
| Detección de bots                         | Solicitudes que no lograron resolver el desafío CAPTCHA. Esta métrica suma todas las solicitudes de bots bloqueadas por la función [detección de bots](/es/docs/secure/attack-protection/bot-detection) después de no resolver un desafío CAPTCHA.                                                                               |                                                                                                                                               |
| MFA adaptativo                            | Solicitudes en las que se emitió un desafío de [MFA](/es/docs/secure/multi-factor-authentication/adaptive-mfa) debido a una puntuación de confianza baja. Esta métrica suma todos los intentos de inicio de sesión en los que se emitió un desafío de autenticación multifactor y la puntuación de confianza calculada fue baja. |                                                                                                                                               |
