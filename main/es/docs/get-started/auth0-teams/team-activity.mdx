title: Actividad del equipo
---

La Actividad del equipo permite a los propietarios del equipo ver los registros de auditoría asociados a las actividades de los miembros en su cuenta de Auth0.

El informe de Actividad del equipo se encuentra en la sección **Reports** del Dashboard de Teams.

La Actividad del equipo proporciona una vista resumida del miembro que generó el evento, el tipo de evento y la fecha en que se generó.

Puede seleccionar **View Details** para encontrar más información, como la descripción del evento, su estado, la dirección IP del usuario que lo desencadenó, los detalles del agente de usuario y más.

<div id="event-types">
  ## Tipos de eventos
</div>

A continuación se describen los tipos de eventos de actividad del equipo.

| Tipo de evento           | Descripción                                                                             |
| ------------------------ | --------------------------------------------------------------------------------------- |
| Invitación al equipo     | Invitación de miembro del equipo creada<br />Invitación de miembro del equipo eliminada |
| Miembro del equipo       | Miembro del equipo eliminado<br />Rol del miembro del equipo actualizado                |
| Política de seguridad    | Política de seguridad actualizada<br />Conexiones sociales actualizadas                 |
| Configuración del equipo | Configuración del equipo actualizada                                                    |

<div id="event-schema">
  ## Esquema de eventos
</div>

A continuación, se describe el esquema de eventos de Actividad del equipo.

```json lines
{  
    "id": "string", 
    "team_id": "string",
    "user_id": "string",
    "type": "string",
    "description": "string",
    "status": "string",
    "details": "object",
    "ip": "string",
    "hostname": "string",
    "user_agent": "string",
    "version": "string",
    "created_at": "date",
    "team_slug": "string",
    "user": "object"
}
```

| Nombre        | Tipo         | Descripción                                                                            |
| ------------- | ------------ | -------------------------------------------------------------------------------------- |
| `id`          | cadena       | Identificador único del registro de eventos.                                           |
| `team_id`     | cadena       | Identificador único del equipo.                                                        |
| `user_id`     | cadena       | Identificador único del usuario que generó el evento.                                  |
| `type`        | cadena       | Tipo de evento.                                                                        |
| `description` | cadena       | Resumen del tipo de evento.                                                            |
| `status`      | cadena       | Estado del evento. Los valores posibles son `Success` o `Failure`.                     |
| `details`     | objeto       | Detalles del evento registrado.                                                        |
| `ip`          | cadena       | Dirección IP del usuario que generó el registro de eventos.                            |
| `hostname`    | cadena       | FQDN de la aplicación de Auth0 en la que se realizó el evento.                         |
| `user_agent`  | cadena       | Navegador web, y tipo y versión del sistema operativo del dispositivo.                 |
| `version`     | cadena       | Versión del esquema con la que se generó el registro de actividad.                     |
| `created_at`  | fecha y hora | Fecha y hora en que se generó el registro de eventos.                                  |
| `team_slug`   | cadena       | Identificador de texto único del equipo.                                               |
| `user`        | objeto       | Contiene el nombre, la URL del avatar y el estado de pertenencia al equipo (booleano). |
