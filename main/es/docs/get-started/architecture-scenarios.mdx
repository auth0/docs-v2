<div id="application-architecture-scenarios">
  ## Escenarios de arquitectura de aplicaciones
</div>

Estos escenarios describen los distintos tipos de arquitecturas tecnológicas que puede usar su aplicación y cómo Auth0 puede ayudar. El objetivo de estos escenarios es guiarle a través del proceso de implementación de principio a fin.

| Lea...                                                                                                                          | Para aprender...                                                                                                                                                    |
| ------------------------------------------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| [Múltiples organizaciones](/es/docs/get-started/architecture-scenarios/multiple-organization-architecture)                         | Cómo configurar una arquitectura de aplicación que use organizaciones con un único Proveedor de identidad o con varios Proveedores de identidad.                    |
| [Inicio de sesión único para aplicaciones web tradicionales](/es/docs/get-started/architecture-scenarios/sso-for-regular-web-apps) | Cómo configurar e implementar una aplicación web que use SSO para empleados que usan Active Directory y contratistas que usan un almacén de usuarios independiente. |
| [Aplicación de servidor + API](/es/docs/get-started/architecture-scenarios/server-application-api)                                 | Cómo configurar una API de registro de horas con una tarea cron para registrar entradas de horas.                                                                   |
| [Aplicación de página única + API](/es/docs/get-started/architecture-scenarios/spa-api)                                            | Cómo configurar una API de registro de horas con una aplicación de página única para registrar entradas de horas.                                                   |
| [Aplicación móvil + API](/es/docs/get-started/architecture-scenarios/mobile-api)                                                   | Cómo configurar una API de registro de horas con una aplicación móvil para registrar entradas de horas en una base de datos centralizada.                           |
