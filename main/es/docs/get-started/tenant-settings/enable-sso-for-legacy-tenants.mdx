description: Describe cómo habilitar el inicio de sesión único (SSO) para un inquilino mediante el Auth0 Dashboard. Solo para inquilinos heredados.
title: Habilitar el inicio de sesión único para inquilinos
---

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">
  Solo para inquilinos heredados. De forma predeterminada, el inicio de sesión único sin interrupciones (SSO) está habilitado para todos los inquilinos nuevos de Auth0; sin embargo, los inquilinos heredados pueden decidir si habilitan esta función.
</Callout>

Puede habilitar <Tooltip tip="Single Sign-On (SSO): Servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión de ese usuario en otras aplicaciones." cta="Ver glosario" href="/es/docs/glossary?term=SSO">SSO</Tooltip> para su inquilino mediante el <Tooltip tip="Single Sign-On (SSO): Servicio que, después de que un usuario inicia sesión en una aplicación, inicia automáticamente la sesión de ese usuario en otras aplicaciones." cta="Ver glosario" href="/es/docs/glossary?term=Auth0+Dashboard">Auth0 Dashboard</Tooltip>. Si decide no habilitar el SSO a nivel de inquilino, puede [habilitarlo por aplicación](/es/docs/get-started/applications/enable-sso-for-applications). Si no ve esta configuración disponible en el Dashboard, ya tiene habilitado Seamless SSO; este selector es exclusivamente para inquilinos heredados.

1. Vaya a [Dashboard &gt; Tenant Settings](https://manage.auth0.com/#/tenant).

   <Frame>![Pestaña General Settings de Dashboard Tenant Settings](/docs/images/cdy7uua7fh8z/4okToiwlkNQBxwr8QGE3Rs/7e96db0efab4082ebb9f78cb43ee302c/Basic_Settings_-_EN.png)</Frame>
2. Vaya a la pestaña **Advanced**.
3. Desplácese hasta la sección **Log In Session Management**. Busque **Enable Seamless SSO**.
4. Introduzca los valores de los campos **Inactivity timeout** (4320 minutos de forma predeterminada) y **Require log in after** (10080 minutos de forma predeterminada).

Cuando termine, también debe [configurar los ajustes de duración de la sesión](/es/docs/manage-users/sessions/configure-session-lifetime-settings).
