---
description: Cómo encontrar el nombre de tu inquilino de Auth0
title: Encuentra el nombre de tu Tenant o su ID
---

Cada inquilino de Auth0 tiene un nombre. Usarás este nombre con frecuencia en tu código para identificar el inquilino. Cuando veas la variable `{yourTenantId}`, `{yourTenantName}`, `{yourTenant}` o una similar en la documentación de Auth0, sustituye la variable por el nombre del inquilino.

1. Ve al [Auth0 Dashboard](https://manage.auth0.com/#).
2. En la esquina superior izquierda, busca el nombre del inquilino junto al logotipo de Auth0.
