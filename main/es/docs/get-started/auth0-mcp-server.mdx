El Auth0 Model Context Protocol (MCP) Server conecta agentes de IA con tu tenant de Auth0, lo que les permite realizar operaciones complejas de varios pasos en Auth0, como crear aplicaciones, gestionar usuarios o implementar Actions. Aprovecha [Model Context Protocol (MCP)](https://modelcontextprotocol.io/introduction), un protocolo estandarizado que permite a los agentes de IA comunicarse con herramientas y API externas para ejecutar tareas en tu nombre.

Diseñado para desarrolladores que usan herramientas basadas en IA como Cursor, Windsurf o Claude Desktop, el Auth0 MCP Server transforma la forma en que administran su infraestructura de identidad, haciendo que trabajar con los servicios de Auth0 sea más rápido y sencillo.

<div id="authentication-and-credential-security">
  ## Autenticación y seguridad de las credenciales
</div>

Auth0 MCP Server da prioridad a una seguridad sólida mediante la gestión de la autenticación, la validación de solicitudes y la garantía de una comunicación segura con la <Tooltip tip="Management API: Un producto que permite a los clientes realizar tareas administrativas." cta="Ver glosario" href="/es/docs/glossary?term=Management+API">Management API</Tooltip> de Auth0. Esto incluye:

* [Flujo de autorización de dispositivos OAuth 2.0](/es/docs/get-started/authentication-and-authorization-flow/device-authorization-flow): Autentica de forma segura las solicitudes con Auth0.
* [Almacenamiento seguro de credenciales](/es/docs/get-started/applications/rotate-credentials): Las credenciales se almacenan de forma segura en el llavero del sistema y nunca se exponen en texto sin formato.
* [Permisos mínimos de API](/es/docs/secure/tokens/access-tokens/management-api-access-tokens): Solo se solicitan los permisos esenciales, en estricto cumplimiento del principio de privilegio mínimo.

<div id="supported-tools-and-integrations">
  ## Herramientas e integraciones compatibles
</div>

Auth0 MCP Server actualmente se integra con los siguientes [clientes MCP](https://modelcontextprotocol.io/clients), incluidos:

* [Claude Desktop](https://claude.ai/download)
* [Cursor](https://www.cursor.com/downloads)
* [Windsurf](https://www.windsurf.com/editor)

Se añadirán integraciones adicionales a medida que vayan surgiendo nuevos clientes destacados. Para ver la lista completa de herramientas compatibles y sus funciones detalladas, consulta nuestra [referencia de herramientas de Auth0 Model Context Protocol (MCP) Server](/es/docs/get-started/auth0-mcp-server/auth0-mcp-tools-reference).
