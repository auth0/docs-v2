<div class="api_card_wrapper">
  <Card title="Authentication API" href="/es/docs/api/authentication" arrow />
</div>

<Columns cols={2}>
  <Column>
    La Authentication API expone funcionalidades de identidad para Auth0 y protocolos de identidad compatibles (incluidos OpenID Connect, OAuth y SAML).

    Normalmente, debería usar esta API a través de uno de los SDK de Auth0, como [Auth0.js](/es/docs/libraries/auth0js), o una biblioteca como [Lock](/es/docs/libraries/lock). Sin embargo, si crea manualmente su interfaz de autenticación, deberá llamar directamente a la Authentication API.
  </Column>

  <Column>
    Algunas tareas de ejemplo incluyen:

    * Obtener [tokens](/es/docs/secure/tokens) durante la autenticación
    * Solicitar el perfil de un usuario mediante un [Token de acceso](/es/docs/secure/tokens/access-tokens)
    * Intercambiar [Tokens de actualización](/es/docs/secure/tokens/refresh-tokens) por nuevos Tokens de acceso
    * Solicitar un desafío para la [autenticación multifactor (MFA)](/es/docs/secure/multi-factor-authentication)
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="Management API" href="/es/docs/api/management/v2" arrow />
</div>

<Columns cols={2}>
  <Column>
    La Management API le permite administrar su cuenta de Auth0 mediante programación para que pueda automatizar la configuración de su entorno. La mayoría de las tareas que puede realizar en el Auth0 Management Dashboard también pueden realizarse mediante programación con esta API.
  </Column>

  <Column>
    Algunas tareas de ejemplo incluyen:

    * Registrar sus aplicaciones y APIs en Auth0
    * Configurar [conexiones](/es/docs/connections) con las que sus usuarios pueden autenticarse
    * [Administrar usuarios](/es/docs/manage-users)
    * [Vincular cuentas de usuario](/es/docs/manage-users/user-accounts/user-account-linking/link-user-accounts)
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="My Account API" href="/es/docs/api/myaccount" arrow />
</div>

<Columns cols={2}>
  <Column>
    La Auth0 My Account API proporciona un conjunto específico de endpoints para que los usuarios administren la información de su propia cuenta. Puede usar estas APIs para crear experiencias de autoservicio en sus aplicaciones o añadir progresivamente detalles a una cuenta de usuario.
  </Column>

  <Column>
    Algunas tareas de ejemplo incluyen:

    * Permitir que el usuario final administre sus propios factores
    * Actualizar los detalles de un método de autenticación
    * Saber qué métodos de autenticación están disponibles
  </Column>
</Columns>

<div class="api_card_wrapper">
  <Card title="My Organization API" href="/es/docs/api/myorganization" arrow />
</div>

<Columns cols={2}>
  <Column>
    La Auth0 My Organization API proporciona una interfaz segura con alcance de Organización que permite a sus clientes empresariales administrar sus propias Organizaciones dentro de su inquilino de Auth0. Esta API sirve como base técnica para la administración delegada integrada y las integraciones orientadas a la API. Para obtener más información, consulte [My Organzation API and Embeddable UI Components](/es/docs/manage-users/my-organization-api).
  </Column>

  <Column>
    Algunas tareas de ejemplo incluyen:

    * Detalles de la Organización de Auth0 (nombre, marca y nombre para mostrar)
    * Configuración, propiedad y relaciones específicas de la organización
    * Proveedores de identidad (IdP) y aprovisionamiento de SCIM
    * Configuración de dominios y Home Realm Discovery (HRD)
  </Column>
</Columns>

<div id="pagination">
  ## Paginación
</div>

Para los inquilinos de Public Cloud, los endpoints de Management API devuelven un máximo de 50 resultados por respuesta. Cuando haya más resultados de los que se pueden devolver en una sola respuesta, incluye los parámetros `page` y `per_page` en tu solicitud para recuperar más resultados.

Consulta [Migrate to Management API v2 Endpoint Paginated Queries](/es/docs/troubleshoot/product-lifecycle/past-migrations/migrate-to-paginated-queries) para obtener información sobre cómo actualizar tus aplicaciones.
