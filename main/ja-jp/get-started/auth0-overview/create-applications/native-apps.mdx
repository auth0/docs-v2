Auth0をネイティブアプリケーションと統合するには、まず、<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/ja-jp/glossary?term=auth0-dashboard" tip="Auth0 Dashboard: サービスを構成するためのAuth0の主製品。" cta="用語集の表示">Auth0 Dashboard</Tooltip>を使ってアプリをAuth0に登録する必要があります。デバイス（i0S、Androidなど）でネイティブ実行されているモバイル、デスクトップ、ハイブリッドのアプリが含まれます。

1. [［Dashboard］>［Applications（アプリケーション）］>［Applications（アプリケーション）］](https://manage.auth0.com/#/applications)に移動して、**［Create Application（アプリケーションの作成）］** をクリックします。
2. アプリケーションにわかりやすい名前を付けて、**［Native（ネイティブ）］** を選択してから **［Create（作成）］** をクリックします。

   新しいアプリケーションが作成されたら、**［Application Settings（アプリケーション設定）］** に移動して、以下のタブを表示します。

3. モバイルアプリを開発している場合には、iOSまたはAndroidに必要なパラメーターを **［Advanced Settings（詳細設定）］** に入力してから、 **［Save Changes（変更の保存）］** をクリックします。

   * iOSアプリには、[**［Team ID（チームID）］** ](/ja-jp/get-started/applications/enable-universal-links-support-in-apple-xcode)[と](/ja-jp/get-started/applications/enable-universal-links-support-in-apple-xcode)[**［App Bundle Identifier（アプリのバンドル識別子）］** ](/ja-jp/get-started/applications/enable-universal-links-support-in-apple-xcode)を入力します。
   * Androidアプリには、[**［App Package Name（アプリのパッケージ名）］** ](/ja-jp/get-started/applications/enable-android-app-links-support)[と](/ja-jp/get-started/applications/enable-android-app-links-support)[**［Key Hashes（キーハッシュ）］** ](/ja-jp/get-started/applications/enable-android-app-links-support)を入力します。

## 次の手順

アプリケーションを登録して構成したら、通常は次のような手順が必要になります：

* 接続を構成し、アプリケーションに対して有効化する。
* Auth0に登録したアプリケーションを使用するように、アプリのコードを変更する。当社の[「Auth0クイックスタート」](/ja-jp/quickstarts)を参照してください。詳しい手順と各種テクノロジー用のサンプルをご紹介しています。また、ログインとログアウトの実装、ユーザーセッションの処理、ユーザープロファイル情報の取得・表示などの方法も説明しています。
* [Auth0 API](/ja-jp/api)を使用する。

  + [認証API](/ja-jp/api/authentication)によって、すべての主要なID関連機能（ログイン、ログアウト、ユーザープロファイルの取得など）が処理されます。ほとんどのユーザーは、クイックスタート、[Auth0.js ライブラリー](/ja-jp/libraries/auth0js)、または[Lockウィジェット](/ja-jp/libraries/lock)を通じてこのAPIを使用します。ただし、認証UIをすべて手動でビルドする場合は、このAPIと直接やり取りする必要があります。
  + [Management API](/ja-jp/api/management/v2)を使用すると、Auth0 Dashboardからもアクセスできるさまざまなタスク（ユーザーの作成、アプリケーションの付与タイプの設定など）を自動化できます。
