チームアクティビティを使用すると、チーム所有者は、Auth0アカウントで実行されたメンバーアクティビティに関連付けられた監査ログを表示できます。

チームアクティビティレポートは、Teams Dashboardの **［Reports（レポート）］** セクションにあります。

チームアクティビティには、イベントを生成したメンバー、イベントの種類、およびイベントが生成された日付の概要ビューが表示されます。

**［View Details（詳細を表示）］** を選択すると、イベントの説明、イベントのステータス、イベントをトリガーしたユーザーのIPアドレス、ユーザーエージェントの詳細などの詳細情報を確認できます。

## イベントスキーマ

チームアクティビティイベントスキーマについては以下で説明します。

```json lines
{  
    "id": "string", 
    "team_id": "string",
    "user_id": "string",
    "type": "string",
    "description": "string",
    "status": "string",
    "details": "object",
    "ip": "string",
    "hostname": "string",
    "user_agent": "string",
    "version": "string",
    "created_at": "date",
    "team_slug": "string",
    "user": "object"
}
```

## 監査ログの保存

チームアクティビティ監査ログの保持期間はプランによって異なります。詳細については、[価格](https://auth0.com/pricing)ページをご覧ください。
