Auth0ダッシュボードを使用してアプリケーションのデータベース接続をテストできます。構成したデータベース接続は、アプリケーションへのログインに使用できます。

適切にテストするためにはあらかじめ[データベース接続を設定](/ja-jp/get-started/applications/set-up-database-connections)しておき、データベース接続用の [ユーザーを作成](/ja-jp/manage-users/user-accounts/create-users)しておく必要があります。

1. [［Auth0 Dashboard］>［Authentication（認証）］>［Databaseデータ​ベース）］](https://manage.auth0.com/#/connections/database)に移動し、テストしたい接続の横にある **［Try（試す）］** の矢印を選択します。

   <Frame>![Dashboard - 認証 - データベース - データベース接続リスト](/images/ja-jp/cdy7uua7fh8z/1Hf6ctA9xhsLOQO9IyG0VU/4852ec12c491cc4e5ff8cf52e1ec9672/DB_Connections_-_JP.png)</Frame>

2. テストユーザーのユーザー名とパスワードを入力します。すべてが正しく設定されていれば、**It Works!（成功しました）** というページが表示されます。

   <Frame>![Dashboard - Connections - Database - Try - Success](/images/ja-jp/cdy7uua7fh8z/5DWJWLf8zvZJH0rSlhK0cr/f33377edf6f82f17e11e08ec30d8d75a/connection-social-try-success.png)</Frame>
