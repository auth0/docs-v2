<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/ja-jp/glossary?term=auth0-dashboard" tip="Auth0 Dashboard: サービスを構成するためのAuth0の主製品。" cta="用語集の表示">Auth0 Dashboard</Tooltip>を使用して、アプリケーションの有効化された接続を変更できます。

1. [［Dashboard］>［Applications（アプリケーション）］](https://manage.auth0.com/#/applications)に移動し、アプリケーションの名前をクリックして表示します。

   <Frame>![Dashboardのアプリケーションリスト](/images/ja-jp/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/6a547af05f8729ea00700a851e13c62b/App_List_-_JP.png)</Frame>

2. **［Connections（接続）］** タブをクリックし、アプリケーションの該当する接続を有効化または無効化します。
