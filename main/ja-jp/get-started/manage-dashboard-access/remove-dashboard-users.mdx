テナント管理者は、Dashboardへのアクセスを取り消すことができます。

1. [［Dashboard］>［Settings（設定）］>［Tenant Members（テナントメンバー）］](https://manage.auth0.com/#/tenant/admins)に移動します。
2. ユーザー名またはメールアドレスの隣にある**［...］** をクリックしてから、**［Remove（削除）］** をクリックします。
3. **［Save（保存）］** をクリックします。
