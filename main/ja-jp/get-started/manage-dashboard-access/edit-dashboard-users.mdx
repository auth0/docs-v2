テナント管理者である場合、割り当てられたユーザーロールを変更することができます。変更は、ユーザーがDashboardをリフレッシュした後に有効になります。

ユーザーには複数のロールを割り当てることができますが、その場合、アクセス権は統合されます。テナントメンバーは、事前に追加されている必要があります。

1. [［Dashboard］>［Settings（設定）］>［Tenant Members（テナントメンバー）］](https://manage.auth0.com/#/tenant/admins)に移動します。
2. ユーザー名またはメールアドレスの隣にある **［...］** をクリックしてから、**［Edit Roles（ロールの編集）］** をクリックします。このオプションが表示されない場合、そのユーザーが招待をまだ受け入れていないことを意味します。
3. 追加するロールを選択するか、ロールを削除します。
4. **［Save（保存）］** をクリックします。

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

ユーザーがDashboardを更新すると、ユーザーの権限が表示されます。更新しなかった場合は、特定の領域にアクセスする権限がないというエラーが表示されることがあります。このようなエラーが発生した場合、ユーザーはいったんログアウトして、ログインし直す必要があります。

</Callout>

**パスワードのリセット** ：ご自身またはDashboardユーザーの誰かが、ログイン時に問題が発生している場合、詳細については、「[Auth0アカウントパスワードをリセットする](/ja-jp/troubleshoot/customer-support/reset-account-passwords)」をお読みください。

**無効なメールアドレス** ：テナントメンバーのメールアドレスを表示して、必要に応じて更新するための詳細については、「[Dashboardユーザーのメールアドレスを更新する](/ja-jp/get-started/manage-dashboard-access/update-dashboard-user-email)」をお読みください。
