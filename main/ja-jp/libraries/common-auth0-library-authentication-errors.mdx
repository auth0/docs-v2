サインアップまたはログインのプロセス中にユーザーが実行するアクションやデータ入力によって、エラーが発生することがあります。認証にいずれかのAuth0ライブラリーを使用した場合に発生する可能性のある、最も一般的なエラーをリストにまとめました。

## サインアップ

サインアップに失敗した場合の最も一般的なエラーは以下のとおりです。

<table class="table"><thead>
<tr>
<th><strong>エラー</strong></th>
<th><strong>説明</strong></th>
</tr>
</thead>
<tbody>
<tr>
<td><strong>invalid_password</strong></td>
<td>パスワードが接続のパスワードポリシーに適合していません</td>
</tr>
<tr>
<td><strong>invalid_signup</strong></td>
<td>サインアップさせようとしているユーザーが無効です</td>
</tr>
<tr>
<td><strong>password_dictionary_error</strong></td>
<td>指定されたパスワードが一般的すぎます</td>
</tr>
<tr>
<td><strong>password_no_user_info_error</strong></td>
<td>指定されたパスワードがユーザー情報に基づいています</td>
</tr>
<tr>
<td><strong>password_strength_error</strong></td>
<td>指定された<a href="/ja-jp/connections/database/password-strength">パスワードが弱すぎ</a>ます</td>
</tr>
<tr>
<td><strong>unauthorized</strong></td>
<td>このアプリケーションにはサインアップできません。特定のルールに違反している可能性があります</td>
</tr>
<tr>
<td><strong>user_exists</strong></td>
<td>サインアップさせようとしているユーザーはすでにサインアップ済みです</td>
</tr>
<tr>
<td><strong>username_exists</strong></td>
<td>サインアップさせようとしているユーザー名はすでに使用されています</td>
</tr>
</tbody>
</table>

## ログイン

ログインに失敗した場合の最も一般的なエラーは以下のとおりです。

<table class="table"><thead>
<tr>
<th><strong>エラー</strong></th>
<th><strong>説明</strong></th>
</tr>
</thead>
<tbody>
<tr>
<td><strong>access_denied</strong></td>
<td>Webベースの認証でリソースサーバーがOAuth2の仕様に従ってアクセスを拒否</td>
</tr>
<tr>
<td><strong>invalid_user_password</strong></td>
<td>認証に使用されたユーザー名および/またはパスワードが無効</td>
</tr>
<tr>
<td><strong>mfa_invalid_code</strong></td>
<td>ユーザーが提供した<Tooltip id="react-containers-DefinitionTooltip-0">多要素認証（MFA）</Tooltip>のコードが無効または期限切れ</td>
</tr>
<tr>
<td><strong>mfa_registration_required</strong></td>
<td>管理者は<a href="/ja-jp/mfa">多要素認証</a>を必須にしているが、ユーザーがまだ登録していない</td>
</tr>
<tr>
<td><strong>mfa_required</strong></td>
<td>ユーザーは認証のために<a href="/ja-jp/mfa">多要素認証</a>のコードを提供しなければならない</td>
</tr>
<tr>
<td><strong>password_leaked</strong></td>
<td>漏洩されたパスワードのため、別のパスワードが必要</td>
</tr>
<tr>
<td><strong>PasswordHistoryError</strong></td>
<td>サインアップまたは更新で提供されたパスワードがすでに使用済み（<a href="/ja-jp/connections/database/password-options#password-history">パスワード履歴</a>機能が有効な場合に報告される）</td>
</tr>
<tr>
<td><strong>PasswordStrengthError</strong></td>
<td>提供されたパスワードが接続の<a href="/ja-jp/connections/database/password-strength">強度の要件</a>を満たしていない</td>
</tr>
<tr>
<td><strong>too_many_attempts</strong></td>
<td>サインイン試行が多すぎるためアカウントがブロックされた</td>
</tr>
<tr>
<td><strong>unauthorized</strong></td>
<td>サインインしようとしているユーザーがブロックされている</td>
</tr>
</tbody>
</table>
