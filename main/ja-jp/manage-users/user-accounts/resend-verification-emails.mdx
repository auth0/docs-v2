ユーザーに確認メールを再送信する必要がある場合は、Auth0 Management DashboardまたはAuth0 <Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/ja-jp/glossary?term=management-api" tip="Management API: 顧客が管理タスクを実行できるようにするための製品。" cta="用語集の表示">Management API</Tooltip>を使用して再送信できます。

<Tabs><Tab title="Auth0 Dashboardの使用">

Auth0 Dashboardを使用して、ユーザーに検証メールを再送できます。

1. Dashboardから、[**［User Management（ユーザー管理）］ > ［Users（ユーザー）］** ](https://manage.auth0.com/#/users)に移動します。
2. ユーザーを選択します。
3. **［Actions（アクション）］** ドロップダウンメニューを選択し、** ［Send Verification Email（検証メールを送る）］** を選択します。

</Tab><Tab title="Auth0 Management APIの使用">

Auth0 Management APIを使用して、ユーザーに検証メールを再送できます。

1. `update:users`スコープを持つ[Management APIのアクセストークン](/ja-jp/secure/tokens/access-tokens/management-api-access-tokens)を取得します。
2. ユーザーの`user_id`を使用して、Management APIの [メールアドレス検証メールの送信](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email)エンドポイントを呼び出します。

#### 要求例

```json lines
{
   "user_id":"auth0|62463410c0242d00699461c8"
}
```

#### 応答例

```json lines
{
   "type":"verification_email",
   "status":"pending",
   "created_at":"2022-03-31T23:08:05.612Z",
   "id":"job_PyfpRHlmpkSoDmCv"
}
```

#### ジョブステータスを確認

Management APIの[メールアドレス検証メールの送信](https://auth0.com/docs/api/management/v2#!/Jobs/post_verification_email)エンドポイントで作成されたジョブステータスを確認できます。

1. ジョブの（応答で返された）`id`を取得します。
2. ジョブの`id`で、Management APIの[ジョブの取得](https://auth0.com/docs/api/management/v2#!/Jobs/get_jobs_by_id)エンドポイントを呼び出します。

##### 応答例

```json lines
{
   "type":"verification_email",
   "status":"completed",
   "created_at":"2022-03-31T23:08:05.612Z",
   "id":"job_PyfpRHlmpkSoDmCv"
}
```

</Tab></Tabs>
