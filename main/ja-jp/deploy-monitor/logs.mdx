Auth0は、ニーズに合わせて分析できるイベントログを提供しています。以下を行うことができます。

* テナント管理者が実行したアクションを表示する
* <Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/ja-jp/glossary?term=management-api" tip="Management API: 顧客が管理タスクを実行できるようにするための製品。" cta="用語集の表示">Management API</Tooltip>を介して実行された操作を表示する
* ユーザーが行った認証を表示する
* 品質管理テストでは検出しにくい断続的なエラーを表示する
* セキュリティのためにフォレンジックデータをキャプチャし、異常検出のトリガーを特定する
* 使用傾向のパターンを特定する
* 認証エラーやManagement APIの呼び出しエラーを確認する
* ログの自動スキャンをセットアップしてレート制限エラーを確認する。ユーザーに影響が及ぶ前に、レート制限を超えそうなアクティビティに予見的に対処できる
* `X-Correlation-ID`ヘッダー値を一致させることにより、Management API イベントを開始要求と相関させます。

## ログストリーミングとエクスポート

ログストリーミングを使用すると、任意の処理・分析ツールにログをエクスポートできます。また、いくつかの分析サービスとの統合や、Webhookを使用してカスタムのログストリーミングを作成する方法も用意されています。詳細については、[［Log Streaming（ログストリーミング）］](/ja-jp/customize/log-streams)をお読みください。
