アプリケーションでSAML2 Webアプリケーションアドオンを使用するように構成することができます。

1. [［Dashboard］＞［アプリケーション］＞［アプリケーション］](https://manage.auth0.com/#/applications)に移動します。
2. SAML2 Webアプリケーションを使用するアプリケーションを特定し、アプリケーションの名前をクリックしてその構成設定に移動します。
3. **［Addons（アドオン）］** タブに移動します。
4. **［SAML2 Web App（SAML2 Webアプリ）］** ボックスでスライダーをクリックしてアドオンを有効にします。

   <Frame>![Dashboard アプリケーション アプリケーションアドオンタブ SAML2 Webアプリ 設定タブ](/images/ja-jp/cdy7uua7fh8z/6dJgYkcOgMZ73HVTkAWt1x/0126f55506d82cfd73d0bcb7ebd6fa9f/2025-02-27_14-00-16.png)</Frame>

5. **［Settings（設定）］** タブに進み、SAML応答を受ける **［Application Callback URL（アプリケーションコールバックURL）］** と **［Settings（設定）］** に入力します。 **［Settings（設定）］** タブに、どの設定を有効にするかを判断するために使えるサンプルオブジェクトがあります。
6. **［Debug（デバッグ）］** をクリックして、入力したパラメーターが有効かどうかをテストします。
7. 完了したら、 **［Save（保存）］** をクリックします。

## アプリケーションの詳細を入力する

SAML2 Webアプリにアプリケーションの詳細を入力するための具体的な手順は場合によって異なりますが、必要な構成情報はアドオンをセットアップした後、 **［Usage（使用状況）］** タブで取得できます。
