<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-2" href="/ja-jp/glossary?term=security-assertion-markup-language" tip="Security Assertion Markup Language（SAML）: パスワードなしに二者間で認証情報を交換できる標準化プロトコル。" cta="用語集の表示">Security Assertion Markup Language</Tooltip>（SAML）プロトコルは、2つのエンティティ間でパスワードを使用せずに認証および認可を行うための、オープンスタンダードでXMLベースのフレームワークです。

* **サービスプロバイダー** （SP）は、IDプロバイダーを信頼することに同意し、ユーザーを認証します。
* **IDプロバイダー** （<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-2" href="/ja-jp/glossary?term=idp" tip="IDプロバイダー（IdP）: デジタルIDを保存および管理するサービス。" cta="用語集の表示">IdP</Tooltip>）はユーザーを認証し、サービスプロバイダーに対して、ユーザーが認証されたことを示す認証アサーションを提供します。

Auth0はSAMLプロトコルをサポートしており、IdP、SP、またはその両方として機能できます。これには以下が含まれます。

* SAML2 Webアプリケーション
* SAML <Tooltip data-tooltip-id="react-containers-DefinitionTooltip-0" href="/ja-jp/glossary?term=single-sign-on" tip="シングルサインオン（SSO）: ユーザーが1つのアプリケーションにログインした後、そのユーザーを他のアプリケーションに自動的にログインさせるサービス。" cta="用語集の表示">SSO</Tooltip>統合
* IdPを起点とするSSO
* Microsoft Active Directoryフェデレーションサービス（ADFS）
* SAML要求の署名と暗号化

## サポートされているSAMLバインディングおよびオプション

Auth0は次のSAMLバインディングをサポートします。

* HTTPリダイレクト
* HTTP POST

Auth0は次のSAMLオプションをサポートします。

* WebブラウザーSSOプロファイル
* シングルログアウトプロファイル
* 名前ID管理プロファイル
* 名前IDマッピングプロファイル
