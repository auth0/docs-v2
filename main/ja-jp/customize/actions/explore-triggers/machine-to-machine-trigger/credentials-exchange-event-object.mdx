credentials-exchangeアクショントリガーの`event`オブジェクトは、クライアント資格情報交換の要求に関するコンテキスト情報を提供します。

<table class="table">
<thead>
<tr>
<th>プロパティ</th>
<th>説明</th>
</tr>
</thead>
<tbody>
<tr>
<td>
<p>`event.accessToken`</p>
</td>
<td>
<p>発行されるアクセストークンについての情報。</p>
<p>次のプロパティを含みます：</p>
<ul>
<li>`customClaims` <em>辞書</em>。</li>
<li>`scope` <em>文字列の配列</em>。</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.client`</p>
</td>
<td>
<p>このトークンの交換中に使用されたクライアントについての情報。</p>
<p>次のプロパティを含みます：</p>
<ul>
<li>`client_id` <em>文字列</em>。<span>ユーザーがログインしているアプリケーションのクライアントID。</span></li>
<li>
`metadata` <em>辞書</em>。<span>別のアプリケーションプロパティを保持するためのオブジェクト。</span>
</li>
<li>
`name` <em>文字列</em>。<span>アプリケーションの（Dashboardで定義された）名前。</span>
</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.request`</p>
</td>
<td>
<p>トランザクションを開始した要求の詳細。</p>
<p>次のプロパティを含みます：</p>
<ul>
<li>`body` <em>辞書</em>。<span>POST要求のボディ。このデータは、リフレッシュトークン、クライアント資格情報交換フロー、PreUserRegistrationアクション中のみ利用できます。</span></li>
<li>
`geoip` <em>オブジェクト</em>。<p>次のプロパティを含みます：</p>
<ul>
<li>`cityName` <em>任意の文字列</em>。</li>
<li>`continentCode` <em>任意の文字列</em>。</li>
<li>`countryCode` <em>任意の文字列</em>。</li>
<li>`countryCode3` <em>任意の文字列</em>。</li>
<li>`countryName` <em>任意の文字列</em>。</li>
<li>`latitude` <em>任意の数字</em>。</li>
<li>`longitude` <em>任意の数字</em>。</li>
<li>`subdivisionCode` <em>任意の文字列</em>。</li>
<li>`subdivisionName` <em>任意の文字列</em>。</li>
<li>`timeZone` <em>任意の文字列</em>。</li>
</ul>
</li>
<li>`hostname` <em>任意の文字列</em>。<span>認証フローに使用されているホスト名。</span></li>
<li>
`ip` <em>文字列</em>。<span>要求の発生元IPアドレス。</span>
</li>
<li>
`language` <em>任意の文字列</em>。<span>ブラウザーから要求された言語。</span>
</li>
<li>
`method` <em>文字列</em>。<span>要求に使用されたHTTPメソッド</span>
</li>
<li>
`user_agent` <em>任意の文字列</em>。<span>トランザクションを開始した時に受け取った `User-Agent`ヘッダーの
              値。</span>
</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.resource_server`</p>
</td>
<td>
<p>アクセスを発行したリソースサーバーの情報アクセストークンを発行したtokeResource Server。</p>
<p>次のプロパティを含みます：</p>
<ul>
<li>`identifier` <em>文字列</em>。<span>リソースサーバーの識別子。例：`https://your-api.example.com`。</span></li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.tenant`</p>
</td>
<td>
<p>このトークンの交換中に使用されたテナントの情報。</p>
<p>次のプロパティを持ちます：</p>
<ul>
<li>`id` <em>文字列</em>。<span>テナント名。</span></li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.transaction`</p>
</td>
<td>
<p>資格情報交換トランザクションについての情報。</p>
<p>次のプロパティを含みます：</p>
<ul>
<li>`requested_scopes` <em>文字列の配列</em>。<span>アクセストークンを要求するときに（必要ある場合）指定されたスコープ。</span></li>
</ul>
</td>
</tr>
</tbody>
</table>
