post-change-passwordアクショントリガーの`event`オブジェクトは、新規作成されたユーザーに関するコンテキスト情報を提供します。

<table class="table">
<thead>
<tr>
<th>プロパティ</th>
<th>説明</th>
</tr>
</thead>
<tbody>
<tr>
<td>
<p>`event.connection`</p>
</td>
<td>
<p>現在のトランザクションに使用された
          接続についての詳細。</p>
<p>次のプロパティを含みます：</p>
<ul>
<li>`id` <em>文字列</em>。<span>接続の一意の識別子。</span></li>
<li>
`metadata` <em>任意の辞書</em>。<span>接続に関連したメタデータ。</span>
</li>
<li>
`name` <em>文字列</em>。<span>ユーザーの認証に使用される接続名（
              `twitter`または`some-g-suite-domain`）。</span>
</li>
<li>
`strategy` <em>文字列</em>。<span>接続のタイプ。ソーシャル接続の場合、
              `event.connection.strategy === event.connection.name`。エンタープライズ接続では、
              `waad`（Microsoft Entra ID）、`ad`（Active
              Directory/LDAP）、`auth0`（データベース接続）など
              です。</span>
</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.request`</p>
</td>
<td>
<p>トランザクションを開始した要求の詳細。</p>
<p>次のプロパティを含みます：</p>
<ul>
<li>
`geoip`
<em>オブジェクト</em>
<span>
要求の地理情報を含みます。<p>次のプロパティを含みます：</p>
<ul>
<li>`cityName` <em>任意の文字列</em>。</li>
<li>`continentCode` <em>任意の文字列</em>。</li>
<li>`countryCode` <em>任意の文字列</em>。</li>
<li>`countryCode3` <em>任意の文字列</em>。</li>
<li>`countryName` <em>任意の文字列</em>。</li>
<li>`latitude` <em>任意の数字</em>。</li>
<li>`longitude` <em>任意の数字</em>。</li>
<li>`subdivisionCode` <em>任意の文字列</em>。</li>
<li>`subdivisionName` <em>任意の文字列</em>。</li>
<li>`timeZone` <em>任意の文字列</em>。</li>
</ul>
</span>
</li>
<li>`hostname` <em>任意の文字列</em>。<span>認証フローに使用されているホスト名。</span></li>
<li>
`ip` <em>文字列</em>。<span>要求の発生元IPアドレス。</span>
</li>
<li>
`language` <em>任意の文字列</em>。<span>ブラウザーから要求された言語。</span>
</li>
<li>
`method` <em>文字列</em>。<span>要求に使用されたHTTPメソッド</span>
</li>
<li>
`user_agent` <em>任意の文字列</em>。<span>トランザクションの開始時に受け取った`User-Agent`の
              ヘッダーの値。</span>
</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.tenant`</p>
</td>
<td>
<p>現在のトランザクションに関連するテナントの詳細。</p>
<p>次のプロパティを含みます：</p>
<ul>
<li>`id` <em>文字列</em>。<span>テナント名。</span></li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.user`</p>
</td>
<td>
<p>現在のトランザクションが開始されたユーザーを表すオブジェクト。</p>
<p>次のプロパティを含みます：</p>
<ul>
<li>
`email` <em>任意の文字列</em>。<span>（一意）ユーザーのメールアドレス。</span>
</li>
<li>`email_verified` <em>任意のブール値</em>。<span>ユーザーがメールアドレスを確認済みかどうかを示します。</span></li>
<li>
`last_password_reset` <em>任意の文字列</em>。<span>ユーザーパスワードがリセット/変更された最終日時を示す
              タイムスタンプ。ユーザーの作成時に、このフィールドは存在しません。この
              プロパティはデータベース接続にのみ利用できます。</span>
</li>
<li>
`phone_number` <em>任意の文字列</em>。
            <span>（一意）ユーザーの電話番号。</span>
</li>
<li>
`phone_verified` <em>任意のブール値</em>。<span>ユーザーが電話番号を検証済みかを示します。</span>
</li>
<li>
`user_id` <em>任意の文字列</em>。
            <span>（一意）ユーザーの一意の識別子。</span>
</li>
<li>
`username` <em>任意の文字列</em>。
            <span>（一意）ユーザーのユーザー名。</span>
</li>
</ul>
</td>
</tr>
</tbody>
</table>
