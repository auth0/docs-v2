この記事では、Auth0からユーザーデータをCSVファイルとしてエクスポートし、Constant Contactダッシュボードにインポートする方法を説明します。

## ユーザーデータファイルを作成する

Dashboardの[［Extensions（拡張機能）］](https://manage.auth0.com/#/extensions)セクションに移動して、 **［User Import / Export Extension（ユーザーのインポート/エクスポート拡張機能）］** を開きます。拡張機能ページで、メニューから **［Export（エクスポート）］** を選択します。

次に、 **［Export Format（エクスポート形式）］** を必要なファイル形式に設定します。Constant Contactは、CSV形式のファイルインポートを許可しているため、`［Tab Separated Value file (*.csv)（タブ区切り値ファイル）］`を選択します。

**［Fields（フィールド）］** セクションの一番上で、各ユーザー属性の **［User Field（ユーザーフィールド）］** と **［Column Name（列名）］** を入力し、エクスポートに含めます。例：

ユーザーフィールドを追加したら、 **［Export Users（ユーザーのエクスポート）］** ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、CSVファイルをダウンロードして、以下の処理に使用します。

## ユーザーデータファイルのインポート

開始する前に、「[Constant Contactナレッジベース：連絡先メールアドレスのファイルのインポートまたはアップロード](https://knowledgebase.constantcontact.com/articles/KnowledgeBase/5296-import-or-upload-a-file-of-contact-email-addresses)

Constant ContactにCSVファイルをインポートするには、次の手順に従ってください。

1. [Constant Contactにログイン](https://login.constantcontact.com)して、 **［Contacts（連絡先）］** に移動します。
2. **［Contacts（連絡先）］** ページから、 **［Add Contacts（連絡先の追加）］>［Upload from file（ファイルからアップロード）］** をクリックします。
3. **［Upload from file（ファイルからアップロード）］** ページでCSVファイルを指定し、 **［Continue（続行）］** をクリックします。
4. 列を確認し、Constant Contactのフィールドと一致させます。終了したら、 **［Continue（続行）］** をクリックします。
5. **［Select lists（リストの選択）］** ページで、連絡先を追加するリストを選択します。
6. **[Upload （アップロード）]** をクリックします。

これで作業完了です！Auth0ユーザーがConstant Contactにインポートされました。
