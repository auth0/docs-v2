この記事では、ユーザーデータをCSVファイルでAuth0にエクスポートし、データインポートウィザードを使用してSalesforceにインポートする方法について説明します。システム要件を確認するには、Salesforceの記事「[データインポートウィザード](https://help.salesforce.com/articleView?id=data_import_wizard.htm)」にアクセスしてください。

## ユーザーデータファイルを作成する

［Dashboard］の[［Extensions（拡張機能）］](https://manage.auth0.com/#/extensions)に移動し、 **［User Import / Export Extension（ユーザーのインポート/エクスポート拡張機能）］** を開きます。拡張機能ページのメニューから **［Export（エクスポート）］** を選択します。

次に、 **［Export Format（エクスポート形式）］** を必要なファイル形式に設定します。SalesforceはCSV形式のファイルインポートを受け付けるため、`［Tab Separated Value file（タブ区切り値ファイル)（*.csv）］`オプションを選択してください。

**［Fields（フィールド）］** セクションの一番上で、それぞれのユーザー属性に対して **［User Field（ユーザーフィールド）］** と **［Column Name（コラム名）］** を提供し、エクスポートに含めてください。例：

ユーザーフィールドの追加後、 **［Export Users（ユーザーのエクスポート）］** ボタンをクリックし、エクスポートを開始します。エクスポートが完了したら、CSVファイルをダウンロードし、以下のセクションで使用します。

## ユーザーデータファイルをインポートする

異なるアカウントタイプでは、インポートが成功するために追加のステップが必要な場合があります。Salesforceの記事「[データのインポートの準備](https://help.salesforce.com/articleView?id=import_prepare.htm)」を参照してください。追加の手順については、「[Salesforce：データインポートウィザードを使用したデータのインポート](https://help.salesforce.com/articleView?id=import_with_data_import_wizard.htm)」にアクセスしてください。

開始する前に、Salesforceアカウントが記録のインポートに必要な[ユーザー権限](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm)を所有していることを確認してください。

SalesforceにCSVファイルをインポートするには、次の手順に従ってください。

1. [Salesforce](https://login.salesforce.com/)にログインします。
2. **［設定］** に移動し、 **［データインポートウィザード］** を開きます。
3. **［ウィザードを起動する］** をクリックします。
4. インポートを構成し、CSVファイルを選択することで、提供されたフィールドを使用してアップロードできます。
5. **［次へ］** をクリックします。
6. **フィールドマッピングの編集** ページから、CSVファイルのデータフィールドをSalesforceデータフィールドにマッピングします。続行するには **［次へ］** をクリックしてください。
7. **確認** ページから情報を検証したあと、 **［インポートを開始］** をクリックします。
8. **データインポートウィザード** のホームページの **［最近のインポートジョブ］** タブから、インポートの更新状況を確認してください。

これで作業完了です。SalesforceへのAuth0ユーザーのインポートに成功しました。
