この記事では、ユーザーデータをCSVファイルでAuth0にエクスポートし、コンタクトアップロードウィザードでEloquaにインポートする方法について説明します。

## ユーザーデータファイルを作成する

Dashboardの[［Extensions（拡張機能）］](https://manage.auth0.com/#/extensions)セクションに移動して、**［User Import / Export Extension（ユーザーのインポート/エクスポート拡張機能）］**を開きます。拡張機能ページで、メニューから**［Export（エクスポート）］**を選択します。

次に、**［Export Format（エクスポート形式）］**を必要なファイル形式に設定します。EloquaはCSV形式のファイルインポートを受け付けるため、`［Tab Separated Value file（タブ区切り値ファイル）（*.csv）］`オプションを選択してください。

**［Fields（フィールド）］**セクションの一番上で、それぞれのユーザー属性に対して**［User Field（ユーザーフィールド）］**と**［Column Name（列名）］**を提供し、エクスポートに含めてください。例：

ユーザーフィールドを追加したら、**［Export Users（ユーザーのエクスポート）］**ボタンをクリックしてエクスポートを開始します。エクスポートが完了したら、CSVファイルをダウンロードして、以下の処理に使用します。

## ユーザーデータファイルのインポート

開始する前に、「[Oracle Eloquaヘルプセンター：コンタクトのアップロード](https://docs.oracle.com/cloud/latest/marketingcs_gs/OMCAA/index.html#Help/Contacts/Tasks/UploadingContacts.htm)」で詳細を確認することができます。

EloquaにCSVファイルをインポートするには、次の手順に従ってください。

1. Eloquaにログインし、**［Audience（オーディエンス）］>［Contacts（コンタクト）］>［Upload（アップロード）］**の順に移動し、**［Contact Upload Wizard（コンタクトアップロードウィザード）］**を開きます。
2. **［Pick Data Source（データソースの選択）］**タブの指定されたフィールドを使ってインポートを構成します。
3. **［Cloud（クラウド）］**ボタンをクリックし、アップロードするファイルを選択します。
4. **［Review（確認）］**タブからファイルの内容を検証した後、**［Next Step（次の手順）］**をクリックします。
5. **［Map Fields（マップフィールド）］**タブで、CSVファイルデータがコンタクトフィールドに一致するように、フィールドマッピング設定値を入力します。続行するには、**［Next Step（次の手順）］**をクリックします。
6. ウィザードの最後の手順を完了し、**［Finish（終了）］**をクリックします。

これで作業完了です！EloquaへのAuth0ユーザーのインポートに成功しました。
