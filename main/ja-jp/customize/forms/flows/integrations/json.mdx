以下に記載のJSONアクションは、JSONの作成、解析、文字列変換を行えるようにします。

## JSONオブジェクトを作成する

JSONオブジェクトを作成します。これは、フローにある他のアクションでコンテンツを再利用するのに便利です。

<Frame>![](/images/ja-jp/cdy7uua7fh8z/1zJEENRPI2uONuCApPY98p/f1fce88a42fb13fb9856c95c562bd506/createjsonobject.png)</Frame>

### 入力設定

<table class="table">
<thead>
<tr>
<th><b>パラメーター</b></th>
<th><b>説明</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>ボディー</td>
<td>JSONオブジェクト。</td>
</tr>
</tbody>
</table>

### 出力オブジェクト

<table class="table">
<thead>
<tr>
<th><b>プロパティ</b></th>
<th><b>種類</b></th>
<th><b>説明</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>`result`</td>
<td>オブジェクト</td>
<td>JSONオブジェクトです。</td>
</tr>
</tbody>
</table>

### 出力オブジェクトの例

```json lines
{
	"result": {
		"name": "John Doe"
	}
}
```
