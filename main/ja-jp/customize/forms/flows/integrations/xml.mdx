このXMLアクションのリストを使用すると、JSONオブジェクトをXML文字列に変換したり、XML文字列をJSONオブジェクトに変換したりできます。

## JSONをXMLに変換する

JSONオブジェクトをXML文字列に変換します。

<Frame>![](/images/ja-jp/cdy7uua7fh8z/3GW2EhBMIX3YDPaqc3rXrZ/7706a863700df51376d645a7d9292a27/convertjsontoxml.png)</Frame>

### 入力設定

<table class="table">
<thead>
<tr>
<th><b>パラメーター</b></th>
<th><b>説明</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>JSON（必須）</td>
<td>XML文字列に変換されるJSONオブジェクトです。</td>
</tr>
</tbody>
</table>

### 出力オブジェクト

<table class="table">
<thead>
<tr>
<th><b>プロパティ</b></th>
<th><b>種類</b></th>
<th><b>説明</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>`xml`</td>
<td>文字列</td>
<td>変換されたXML文字列です。</td>
</tr>
</tbody>
</table>

### 出力オブジェクトの例

```json lines
{
  "xml": "<?xml version=\"1.0\" encoding=\"UTF-8\" standalone=\"yes\"?>\n<name>John Doe</name>"
}
```
