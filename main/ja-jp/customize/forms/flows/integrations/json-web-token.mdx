この<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-2" href="/ja-jp/glossary?term=json-web-token" tip="JSON Web Token（JWT）: 二者間のクレームを安全に表現するために使用される標準IDトークン形式（および多くの場合、アクセストークン形式）。" cta="用語集の表示">JSON Web Token</Tooltip>アクションのリストを使用して、フロー内でJWTの生成、検証、デコードを行うことができます。

## Vault接続の構成

**HS256アルゴリズム** を使用してJSON Web TokenアクションのためにVault接続を構成するには、 **シークレット** が必要です。

<Frame>![Dashboard > Actions > Forms > Flows > actions > JSON web token](/images/ja-jp/cdy7uua7fh8z/1WnUlIe2QZf39G45PHcQGp/40d768da226a0cefa09c62c5cab78577/jwt-vault-connection.png)</Frame>

## JSON Web Tokenに署名する

JSON Web Tokenを生成します。

<Frame>![](/images/ja-jp/cdy7uua7fh8z/3XQqUMKrmrc8fUkCsDhI2T/43f7f8d38673d8fa4f9dba7242e044d9/sign-json-web-token.png)</Frame>

### 入力設定

<table class="table">
<thead>
<tr>
<th><b>パラメーター</b></th>
<th><b>説明</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>Payload（ペイロード）</td>
<td>
              エンコードするデータです。OpenID標準に従ってフォーマットすることをお勧めします。	
          </td>
</tr>
<tr>
<td>Subject（件名）</td>
<td>
             JWTの件名を識別します。
          </td>
</tr>
<tr>
<td>Issuer（発行者）</td>
<td>
              JWTの発行者を識別します。	
          </td>
</tr>
<tr>
<td>Audience（利用者）</td>
<td>
             JWTの対象となる受信者を識別します。例：  admin.your_domain.com
          </td>
</tr>
<tr>
<td>Expires in（有効期限）</td>
<td>
             有効期限を識別します。これ以降JWTの処理は受け入れられなくなります。 
          </td>
</tr>
</tbody>
</table>

### 出力オブジェクト

<table class="table">
<thead>
<tr>
<th><b>プロパティ</b></th>
<th><b>種類</b></th>
<th><b>説明</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>`token`</td>
<td>文字列</td>
<td>JSON Web Tokenの文字列です。</td>
</tr>
</tbody>
</table>

### 出力オブジェクトの例

```json lines
{
  "token": "eyJhbGciOiJIUzI1N..."
}
```

## JSON Web Tokenをデコードする

提供されたJSON Web Tokenをデコードします。

<Frame>![](/images/ja-jp/cdy7uua7fh8z/7I25WVtppllW6qhC5sALvH/70143bbe0d96920e1d9dfabbb1d6aeff/decode-json-web-token.png)</Frame>

### 入力設定

<table class="table">
<thead>
<tr>
<th><b>パラメーター</b></th>
<th><b>説明</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>トークン（必須）</td>
<td>復号化されるJSON Web Tokenの文字列です。	</td>
</tr>
</tbody>
</table>

### 出力オブジェクト

<table class="table">
<thead>
<tr>
<th><b>プロパティ</b></th>
<th><b>種類</b></th>
<th><b>説明</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>`payload`</td>
<td>オブジェクト</td>
<td>復号化された有効なJSON Web Tokenの内容です。</td>
</tr>
</tbody>
</table>

### 出力オブジェクトの例

```json lines
{
  "header": {
    "alg": "HS256",
    "typ": "JWT"
  },
  "payload": {
    "sub": "1234567890",
    "name": "John Doe",
    "iat": 1516239022
  },
  "signature": "SflKxwRJSMeKKF2QT4fwpMeJf36POk6yJV_adQssw5c"
}
```

## JSON Web Tokenを検証する

JSON Web Tokenのデータを検証し、データが変更されていないか、または改ざんされていないかを確認して、その真正性を保証します。

<Frame>![](/images/ja-jp/cdy7uua7fh8z/15MDJcfZqtC46h6mRPOGcz/55e0785db066a54f836d30199fa5f295/verify-json-web-token.png)</Frame>

### 入力設定

<table class="table">
<thead>
<tr>
<th><b>パラメーター</b></th>
<th><b>説明</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>トークン（必須）</td>
<td>
              検証されるJSON Web Tokenの文字列です。
          </td>
</tr>
<tr>
<td>発行者</td>
<td>
              検証されるJWTの発行者です。
          </td>
</tr>
<tr>
<td>オーディエンス</td>
<td>
             検証されるJWTの受信対象オーディエンスです。
          </td>
</tr>
</tbody>
</table>

### 出力オブジェクト

<table class="table">
<thead>
<tr>
<th><b>プロパティ</b></th>
<th><b>種類</b></th>
<th><b>説明</b></th>
</tr>
</thead>
<tbody>
<tr>
<td>`valid`</td>
<td>ブール値</td>
<td>JWTに有効な署名があるかに応じて、`true`または`false`を返します。</td>
</tr>
<tr>
<td>`cause`</td>
<td>文字列</td>
<td>`valid`プロパティが`false`の場合にメッセージを表示します。</td>
</tr>
<tr>
<td>`payload`</td>
<td>オブジェクト</td>
<td>復号化された有効なJSON Web Tokenの内容です。</td>
</tr>
</tbody>
</table>

### 出力オブジェクトの例

```json lines
{
  "valid": true,
  "header": {
    "alg": "HS256",
    "typ": "JWT"
  },
  "payload": {
    "sub": "1234567890",
    "name": "Jane Doe",
    "iat": 1516239022
  },
  "signature": "SflKxwRJSMe..."
}
```

```json lines
{
  "valid": false,
  "cause": "INVALID_SIGNATURE"
}
```
