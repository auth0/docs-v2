当社の[ログストリーミングサービス](/ja-jp/customize/log-streams)を使って、すべてのログデータをサードパーティサービスに送信してみることをお勧めします。

</Callout>

Auth0 ログを外部サービスにエクスポートします。

* Application Insights
* AWS Cloudwatch
* Azure Blob Storage
* Logentries
* Loggly
* Logstash
* Mixpanel
* Papertrail
* Segment
* Splunk
* Sumo Logic
