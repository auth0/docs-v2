Si vous avez du code personnalisé qui s’exécute dans Auth0 en appelant un service au sein de votre réseau, ou si vous avez configuré un fournisseur SMTP sur site dans Auth0, vous aurez peut-être besoin de configurer votre pare-feu pour qu’il autorise le trafic entrant depuis Auth0.

Voici les fonctionnalités dont vous pourriez avoir besoin pour autoriser le trafic entrant depuis Auth0 :

* [Hooks](/docs/fr-ca/customize/hooks)

## Appels sortants

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

Les adresses IP sont susceptibles d’être modifiées. En cas de changement d’adresse IP, Auth0 envoie des notifications plusieurs mois avant les changements. Les listes sont à jour au moment de la rédaction.

</Callout>

Quand Auth0 passe des appels sortants, les adresses IP sont statiques. Auth0 convertit les adresses IP internes en l’une des options affichées pour les communications utilisant NAT.

### Nuage privé

Pour les locataires Private Cloud (Nuage privé), les adresses IP que vous devez autoriser à traverser votre pare-feu sont propres à l’environnement du locataire. Auth0 pourrait recevoir les adresses IP privées du locataire si vous activez des fonctionnalités comme les Journaux des locataires, la limitation d’IPs suspects, les bases de données personnalisées, et les actions qui leur sont reliées.

Ces adresses IP sont connues comme **IP de sortie principale** et sont listées sous les données de configuration de l’environnement disponible dans le [Centre d’assistance Auth0](https://support.auth0.com/tenants/private).

## Appels entrants

Les adresses IP associées aux appels entrants à Auth0 peuvent être variables en raison d’un manque d’adresses IP fixes sur les équilibreurs de charge. Dans ce cas, les règles de pare-feu doivent utiliser le nom du service (par exemple : `<YOUR_TENANT>.<YOUR_REGION>.auth0.com`).

Si votre abonnement Auth0 vous permet de configurer un domaine personnalisé autogéré, vous pouvez configurer ce domaine personnalisé avec une adresse IP statique. Les domaines personnalisés autogérés vous permettent de contrôler le point d’entrée au réseau et vous garantissent une adresse IP fixe. Pour en savoir plus sur les plans d’abonnement, consultez [la liste de prix Auth0](https://auth0.com/pricing).
