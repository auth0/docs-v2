Vous pouvez changer la durée de vie du jeton d’accès en utilisant <Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>.

## Configurer la durée de vie du jeton d’accès

1. Allez à [Dashboard > Applications > API](https://manage.auth0.com/#/apis) et sélectionnez le nom de l’API pour le voir.

   <Frame>![Liste des API d’applications Dashboard](/images/fr-ca/cdy7uua7fh8z/3rhmhghYZDSi6YWHRA5yMQ/29b5fc5ca1dcedd82e9e7fff1fc8ecc8/2025-02-27_10-28-06.png)</Frame>

2. Trouvez le champ **Expiration du jeton** sous **Paramètres du jeton**.

   <Frame>![Dashboard (Tableau de bord) - API - Token Settings (Paramètres du jeton) - Expiration](/images/fr-ca/cdy7uua7fh8z/2FFfqqmntXL9CnGrSxea9M/48597b2a2ba990964562cf712844016a/Access_Token_Settings_-_French.png)</Frame>

3. Saisissez la durée de vie désirée (en secondes) pour les jetons d’accès émis pour cet API.

   * La valeur par défaut est de `86,400` secondes (24 heures).
   * La valeur maximale est de `2,592,000 seconds` (30 heures).
4. Sélectionnez **Enregistrer les modifications**.

## Expiration du jeton pour les flux du navigateur

Le champ **Expiration du jeton pour les flux du navigateur** fait référence aux jetons mis pour l’API par le biais de flux implicites et hybrides et ne couvre pas tous les flux initiés depuis les navigateurs.

Par exemple, le flux PKCE (utilisé dans la trousse SDK `auth0-js-spa`) peut être initié à partir du navigateur, mais il fait référence à la valeur d’**Expiration du jeton**, et non **à la valeur d’expiration du jeton pour les flux du navigateur**.

## Durée de vie restreinte pour les jetons d’accès MFA

La durée de vie des jetons d’accès avec l'<Tooltip data-tooltip-id="react-containers-DefinitionTooltip-1" tip="">audience</Tooltip> `{yourAuth0Domain}/mfa` sont restreint à `600` secondes (10 minutes) pour des raisons de sécurité et cela ne peut pas être changé.
