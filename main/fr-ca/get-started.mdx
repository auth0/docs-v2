Bienvenue sur Auth0, plateforme d’identité qui vous aide à gérer l’accès à vos applications. Si vous débutez dans la gestion des identités et des accès (IAM), familiarisez-vous avec certains concepts de base et planifiez la solution qui répondra à vos besoins et à la technologie que vous utilisez. Si vous connaissez déjà IAM, vous pouvez démarrer sans tarder.

---

## Développement

##### Pour être rapidement opérationnel, choisissez votre type d’application pour obtenir un tutoriel de démarrage rapide étape par étape.

<Card title="Native/Mobile App" href="/docs/fr-ca/quickstart/native">

Mobile or Desktop app that runs natively on a device

e.g., iOS, Android

</Card>

<Card title="Single Page App" href="/docs/fr-ca/quickstart/spa">

JavaScript web app that runs in the browser

e.g., AngularJS + Node.js, React

</Card>

<Card title="Regular Web App" href="/docs/fr-ca/quickstart/webapp">

Traditional web app that runs on the server

e.g., Express.js, ASP.NET

</Card>

<Card title="Backend/API" href="/docs/fr-ca/quickstart/backend">

An API or service protected by Auth0

e.g., Express.js API, ASP.NET API

</Card>
