Vérifiez les résultats de la vérification de production pour votre locataire avant de déployer votre environnement de production.

1. Accédez au Auth0 Management Dashboard et connectez-vous à votre compte.
2. À partir de la liste déroulante, sélectionnez le locataire pour lequel vous souhaitez exécuter les vérifications de production
3. À partir de la même liste déroulante, cliquez sur **Exécuter la vérification de préparation** pour lancer l’interface

Facultatif :

1. Sélectionnez une ou plusieurs [Applications](/docs/fr-ca/get-started/applications) associées à ce locataire pour lequel vous souhaitez que les vérifications soient exécutées en cliquant sur **Filtres**.
2. Cliquez sur **Appliquer**.

Une fois le test terminé, votre écran s’actualisera automatiquement pour afficher les résultats de votre vérification.

## Résultats de la vérification de production

Il existe trois catégories de corrections des résultats de vérification :

* [Corrections critiques](/docs/fr-ca/deploy-monitor/pre-deployment-checks/production-check-required-fixes)
* [Corrections non critiques](/docs/fr-ca/deploy-monitor/pre-deployment-checks/production-check-recommended-fixes)
* [Meilleures pratiques](/docs/fr-ca/deploy-monitor/pre-deployment-checks/production-checks-best-practices)

Voici les résultats possibles de votre vérification :

* Réussie
* Échouée
* Meilleures pratiques

Sous chaque ensemble de résultats, Auth0 vous indique le nombre de vérifications réussies par votre application, ainsi que le nombre de vérifications nécessitant un examen supplémentaire.

Si votre application **a échoué** à une ou plusieurs vérifications, vous verrez :

* Le nom de la vérification
* Informations sur ce que recherche la vérification
* Un lien de **Configuration** vers la zone appropriée où vous pouvez apporter les corrections nécessaires pour que votre application passe la vérification
* Lien hypertexte vers la documentation correspondante

Toutes les vérifications que votre application **a réussi** sont regroupées dans l’objet de réussite de l’ensemble de résultats. Vous pouvez consulter le nom et les détails de la vérification, accéder à la documentation associée et utiliser le lien hypertexte pour vous rendre dans la section de configuration correspondante, où vous pourrez apporter des modifications si nécessaire.
