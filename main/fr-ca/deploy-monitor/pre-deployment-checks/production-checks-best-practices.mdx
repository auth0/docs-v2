Les vérifications suivantes ne peuvent pas être automatisées, nous recommandons donc de vérifier manuellement ces points avant le déploiement en production.

<table class="table"><thead>
<tr>
<th>Vérifier</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td>Externaliser <a href="/docs/fr-ca/connections/database/mysql#4-add-configuration-parameters">Paramètres de configuration</a></td>
<td>[Externaliser, au lieu de coder en dur, tous les paramètres de configuration] (https://manage.auth0.com/#/connections/database), tels que les informations d’identification, les chaînes de connexion, les clés API, etc., lors du développement de règles, de crochets ou de connexions de base de données personnalisées.</td>
</tr>
<tr>
<td>Examinez les valeurs de délai d’expiration <dfn id="react-containers-DefinitionTooltip-0"><span class="tooltip-portal-underlined-word" data-react-universal-portal="">SSO</span></dfn></td>
<td>Vérifiez les valeurs par défaut des <a href="https://manage.auth0.com/#/account/advanced">vvaleurs du délai d’expiration des témoins d’authentification unique (SSO)</a> et assurez-vous qu’elles correspondent à vos exigences.</td>
</tr>
<tr>
<td>Locataires et administrateurs</td>
<td>Vérifiez tous les locataires et administrateurs locataires pour vous assurer qu’ils sont corrects. Désaffectez les locataires qui ne sont plus utilisés. Assurez-vous que les administrateurs locataires sont limités aux utilisateurs nécessaires.</td>
</tr>
<tr>
<td>Vérifiez les identifiants client dans le code de l’application</td>
<td>Assurez-vous que les identifiants client dans le code de votre application correspondent à leurs configurations d’application Auth0.</td>
</tr>
<tr>
<td>Ajouter les adresses IP publiques d’Auth0 à la liste d’autorisation</td>
<td>Autorisez les adresses IP d’Auth0 si vous vous connectez à des services internes ou à des services derrière un pare-feu lorsque vous utilisez des règles, des crochets ou des bases de données personnalisées. Vous pouvez obtenir une liste d’adresses IP dans l’info-bulle lors de la configuration de l’un de ces éléments.</td>
</tr>
<tr>
<td>Examinez la protection contre les attaques</td>
<td>Il est recommandé que vous <a href="https://auth0.com/docs/secure/attack-protection">examiniez la capacité et la configuration de la protection Auth0</a>.</td>
</tr>
</tbody>
</table>
