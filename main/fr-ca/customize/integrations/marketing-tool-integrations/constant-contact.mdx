Dans cet article, vous apprendrez à exporter les données utilisateur depuis Auth0 vers un fichier CSV et à les importer dans le tableau de bord Constant Contact.

## Créer un fichier de données utilisateur

Commencez par vous rendre dans la section [Extensions](https://manage.auth0.com/#/extensions) du tableau de bord et ouvrez**l’extension User Import/Export (Importation/Exportation des utilisateurs).**. Sur la page d’extension, sélectionnez **Export (Exporter)** depuis le menu.

Ensuite, définissez le **Format d’exportation** au format de fichier requis. Constant Contact accepte les importations de fichiers au format CSV. Choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut de la section **Champs**, indiquez un **Champ utilisateur** et **Nom de colonne** pour chaque attribut d’utilisateur à inclure dans l’exportation. Par exemple :

<table class="table"><thead>
<tr>
<th>Champ utilisateur</th>
<th>Nom de la colonne</th>
</tr>
</thead>
<tbody>
<tr>
<td>`email`</td>
<td>Adresse courriel</td>
</tr>
<tr>
<td>`created_at`</td>
<td>Créé à</td>
</tr>
<tr>
<td>`given_name`</td>
<td>Prenom</td>
</tr>
<tr>
<td>`family_name`</td>
<td>Nom de famile</td>
</tr>
</tbody>
</table>

Après avoir ajouté les champs utilisateur, cliquez sur la touche **Exporter les utilisateurs** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV que vous utiliserez dans la section suivante.

## Importer un fichier de données utilisateur

Avant de commencer, vous pouvez obtenir de plus amples informations en consultant le [Centre de ressources Constant Contact : Importer ou téléverser un fichier d’adresses courriel de contact](https://knowledgebase.constantcontact.com/articles/KnowledgeBase/5296-import-or-upload-a-file-of-contact-email-addresses)

Pour importer votre fichier CSV dans Constant Contact, procédez comme suit :

1. [Connectez-vous à Constant Contact](https://login.constantcontact.com) et accédez à **Contacts**.
2. Sur la page **Contacts**, cliquez sur **Ajouter des contacts > Téléverser à partir d’un fichier**.
3. Fournissez votre fichier CSV sur la page **Téléverser à partir d’un fichier** et cliquez sur **Continuer**.
4. Consultez les colonnes et comparez-les avec les champs de Constant Contact. Une fois terminé, cliquez sur **Continuer**.
5. Sur la page **Sélectionner des listes**, choisissez la ou les listes auxquelles vous souhaitez ajouter vos contacts.
6. Cliquez sur **Téléverser**.

Et voilà! Vous avez importé avec succès vos utilisateurs Auth0 dans Constant Contact.
