Dans cet article, vous apprendrez à exporter les données d’utilisateur depuis Auth0 vers un fichier CSV et à les importer dans le [tableau de bord MailChimp](https://login.mailchimp.com/).

## Créer un fichier de données d’utilisateur

Commencez par vous rendre dans la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard (Tableau de bord) et ouvrez **l’extension User Import/Export (Importation/Exportation des utilisateurs).**. Dans la page de l’extension, sélectionnez **Export (Exporter)** dans le menu.

Ensuite, choisissez un **Format d’exportation** approprié. MailChimp accepte les importations de fichiers au format CSV, choisissez donc l’option `Tab Separated Value file (*.csv)`.

Pour en savoir plus, consultez le [Centre de ressources MailChimp : Instructions de mise en forme pour votre fichier d’importation](https://kb.mailchimp.com/lists/growth/format-guidelines-for-your-import-file).

En haut de la section **Champs**, indiquez un **Champ d’utilisateur** et un **Nom de colonne** pour chaque attribut d’utilisateur à inclure dans l’exportation. Pour MailChimp, un champ de courriel avec le nom de colonne `Adresse courriel` est requis, assurez-vous donc de l’inclure. Par exemple :

<tr>
<th>Champ utilisateur</th>

</tr>
</thead>
<tbody>
<tr>
<td>`email`</td>
<td>Adresse courriel</td>
</tr>
<tr>
<td>`created_at`</td>
<td>Créé à</td>
</tr>
<tr>

<td>Prenom</td>
</tr>

<td>Nom de famile</td>
</tr>
</tbody>
</table>

Après avoir ajouté les champs utilisateur, cliquez sur la touche **Exporter les utilisateurs** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV que vous utiliserez dans la section suivante.

## Importer un fichier de données d’utilisateur

Avant de commencer, vous pouvez trouver plus d’informations en consultant le [Centre de ressources MailChimp : Importer des abonnés dans une liste](https://kb.mailchimp.com/lists/growth/import-subscribers-to-a-list).

Connectez-vous à votre compte MailChimp et accédez à la page **Listes**. Sélectionnez une liste dans laquelle vous souhaitez importer vos utilisateurs Auth0 ([ou créez une nouvelle liste](https://kb.mailchimp.com/lists/growth/create-a-new-list)).

1. Dans votre page MailChimp List, cliquez sur **Importer les contacts** dans le menu **Ajouter les contacts**.
2. Sélectionnez l’option `CSV or tab-delimited text file` comme source d’importation.
3. Téléversez le fichier CSV de données utilisateur que vous avez exporté depuis Auth0 dans la section précédente. MailChimp interprétera vos données d’utilisateur sur la page suivante.
4. Vérifiez que les noms de colonnes et les types de champs sont corrects.
5. Cliquez sur la touche **Suivant**.
6. Vérifiez vos sélections et définissez la catégorie d’importation.
7. Cliquez sur la touche **Importer** pour lancer l’importation d’utilisateurs.

C’est tout! Vous avez importé avec succès vos utilisateurs Auth0 dans MailChimp.
