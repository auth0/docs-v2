Dans cet article, vous apprendrez à exporter les données utilisateur depuis Auth0 vers un fichier CSV et à les importer dans le tableau de bord Sailthru.

## Créer un fichier de données utilisateur

Commencez par vous rendre à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez **l’extension Importation/Exportation des utilisateurs**. Sur la page d’extension, sélectionnez **Exporter** depuis le menu.

Ensuite, définissez le **format d’exportation** selon le format de fichier requis. Sailthru accepte les importations de fichiers au format CSV, alors sélectionnez l’option `Tab Separated Value file (*.csv)`.

En haut de la section **Champs**, indiquez un **Champ utilisateur** et un **Nom de colonne** pour chaque attribut d’utilisateur à inclure dans l’exportation. Par exemple :

<table class="table"><thead>
<tr>
<th>Champ utilisateur</th>
<th>Nom de la colonne</th>
</tr>
</thead>
<tbody>
<tr>
<td>`email`</td>
<td>Adresse courriel</td>
</tr>
<tr>
<td>`created_at`</td>
<td>Créé à</td>
</tr>
<tr>
<td>`given_name`</td>
<td>Prenom</td>
</tr>
<tr>
<td>`family_name`</td>
<td>Nom de famile</td>
</tr>
</tbody>
</table>

Après avoir ajouté les champs utilisateur, cliquez sur la touche **Exporter les utilisateurs** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV que vous utiliserez dans la section suivante.

### Importer un fichier de données utilisateur

Avant de commencer, vous pouvez obtenir de plus amples informations en consultant la documentation de [Sailthru : Ajouter des utilisateurs dans Sailthru et aux listes de Sailthru](https://getstarted.sailthru.com/audience/managing-users/add-users-to-sailthru-and-lists/#List_File_Upload).

Pour importer votre fichier CSV dans Sailthru, suivez les étapes suivantes :

1. Connectez-vous à Sailthru et accédez aux [Listes](https://my.sailthru.com/lists).
2. Choisissez une liste, puis cliquez sur **Téléverser la liste** et sélectionnez votre fichier CSV.
3. Sélectionnez **Ajouter à la liste** en guise **d’action**.
4. Définissez votre option **Remplacer les variables** et saisissez une adresse de courriel pour recevoir des notifications si vous le souhaitez.
5. Vérifiez vos paramètres et cliquez sur **Soumettre** pour téléverser le fichier.

C’est tout! Vous avez bien importé vos utilisateurs Auth0 dans Sailthru.
