Dans cet article, vous découvrirez comment exporter des données utilisateur depuis Auth0 vers un fichier CSV, puis les importer dans Salesforce en utilisant l’assistant d’importation de données. Pour consulter la configuration système requise, veuillez consulter l’article Salesforce [Assistant d’importation de données](https://help.salesforce.com/articleView?id=data_import_wizard.htm).

## Créer un fichier de données utilisateur

Commencez par vous rendre à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez **l’extension Importation/Exportation des utilisateurs**. Sur la page d’extension, sélectionnez **Exporter** depuis le menu.

Ensuite, définissez le **Format d’exportation** au format de fichier requis. Salesforce accepte les importations de fichiers au format CSV, alors sélectionnez l’option `Tab Separated Value file (*.csv)`.

En haut dans la section **Champs**, saisissez un **User Champ utilisateur** et **Nom de la colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

<tr>
<th>Champ utilisateur</th>
<th>Nom de la colonne</th>
</tr>
</thead>
<tbody>
<tr>
<td>`email`</td>
<td>Adresse courriel</td>
</tr>
<tr>
<td>`created_at`</td>
<td>Créé à</td>
</tr>
<tr>
<td>`given_name`</td>
<td>Prenom</td>
</tr>

<td>`family_name`</td>
<td>Nom de famile</td>
</tr>
</tbody>
</table>

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Exporter les utilisateurs** pour démarrer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV à utiliser dans la section suivante.

## Importer un fichier de données utilisateur

Différents types de comptes peuvent nécessiter des étapes supplémentaires pour une importation réussie. Prenez un moment pour consulter l’article de Salesforce [Préparez vos données pour l’importation](https://help.salesforce.com/articleView?id=import_prepare.htm). Pour des instructions supplémentaires, veuillez consulter dans [Importer des données avec l’assistant d’importation de données](https://help.salesforce.com/articleView?id=import_with_data_import_wizard.htm).

Avant de commencer, assurez-vous que votre compte Salesforce dispose des [autorisations utilisateur](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm) requises pour importer des données.

Pour importer votre fichier CSV dans Salesforce, procédez comme suit :

1. Connectez-vous à [Salesforce](https://login.salesforce.com/).
2. Accéder à **Configuration** et ouvrez l'**Assistant d’importation de données**.
3. Cliquez sur **Lancer l’assistant**.
4. Configurez l’importation et sélectionnez votre fichier CSV à télécharger en utilisant les champs fournis.
5. Cliquez sur **Suivant**.
6. Sur la page **Modifier le mappage des champs**, mappez les champs de données de votre fichier CSV aux champs de données Salesforce. Cliquez sur **Suivant** pour continuer.
7. Après avoir vérifié les informations sur la page **Examiner**, cliquez sur **Démarrer l’importation**.
8. Consultez l’onglet **Taches d’importation récentes** sur la page d’accueil **Assistant d’importation de données** pour les mises à jour sur l’état de votre importation.

C’est fini! Vous avez importé avec succès vos utilisateurs Auth0 dans Salesforce.
