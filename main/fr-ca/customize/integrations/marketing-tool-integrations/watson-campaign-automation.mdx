Dans cet article, vous apprendrez à exporter les données d’utilisateur depuis Auth0 vers un fichier CSV et à les importer dans le tableau de bord de Watson Campaign Automation.

## Créer un fichier de données utilisateur

Commencez par vous rendre à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez **l’extension Importation/Exportation des utilisateurs**. Sur la page d’extension, sélectionnez **Exporter** depuis le menu.

Ensuite, définissez le **format d’exportation** sur le format de fichier requis. Watson Campaign Automatisation accepte les importations de fichiers au format CSV. Sélectionnez donc l’option `Tab Separated Value file (*.csv)`.

En haut de la section **Champs**, indiquez un **Champ utilisateur** et **Nom de colonne** pour chaque attribut d’utilisateur à inclure dans l’exportation. Par exemple :

<table class="table"><thead>
<tr>
<th>Champ utilisateur</th>
<th>Nom de la colonne</th>
</tr>
</thead>
<tbody>
<tr>
<td>`email`</td>
<td>Adresse courriel</td>
</tr>
<tr>
<td>`created_at`</td>
<td>Créé à</td>
</tr>
<tr>
<td>`given_name`</td>
<td>Prenom</td>
</tr>
<tr>
<td>`family_name`</td>
<td>Nom de famile</td>
</tr>
</tbody>
</table>

Après avoir ajouté les champs utilisateur, cliquez sur la touche **Exporter les utilisateurs** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV que vous utiliserez dans la section suivante.

## Importer un fichier de données utilisateur

Pour importer votre fichier CSV dans Watson Campaign Automation, procédez comme suit :

1. Connectez-vous à Watson Campaign Automation et accédez à **Données > Importer des nouvelles**.
2. À l’étape **Sélectionner le fichier**, saisissez les paramètres de la nouvelle base de données dans les champs fournis.
3. Sélectionnez l’option permettant de téléverser un fichier à partir d’un disque dur local, puis cliquez sur **Parcourir** pour sélectionner le fichier.
4. Définissez le type de fichier sur CSV (valeurs séparées par des virgules).
5. Cliquez sur **Suivant** pour procéder à l’étape **Définir le format des données**.
6. Assurez-vous que vos données ont été traitées correctement, puis cliquez sur **Suivant**.
7. À l’étape **Mapper les champs**, sélectionnez les champs à inclure dans l’importation ainsi que leurs types.
8. Cliquez sur **Suivant** pour procéder à l’étape **Modifier les paramètres du champ**.
9. Modifiez les paramètres par défaut ou les nouveaux champs le cas échéant.
10. Pour lancer l’importation, cliquez sur **Suivant**.

C’est tout! Vous avez importé avec succès vos utilisateurs Auth0 dans Watson Campaign Automatisation.
