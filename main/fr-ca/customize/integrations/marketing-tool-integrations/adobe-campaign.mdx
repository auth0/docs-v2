Dans cet article, vous apprendrez à exporter les données des utilisateurs dans Auth0 vers un fichier CSV puis à les importer dans Adobe Campaign à l’aide de l’[assistant d’importation Adobe Campaign](https://docs.adobe.com/content/help/en/campaign-classic/using/getting-started/importing-and-exporting-data/importing-data.html).

## Créer un fichier de données utilisateur

Commencez par vous rendre dans la section [Extensions](https://manage.auth0.com/#/extensions) du tableau de bord, et ouvrez l’extension **l’extension User Import/Export (Importation/Exportation des utilisateurs)**. Sur la page Extension, sélectionnez **Export (Exporter)** dans le menu.

Ensuite, définissez le **format d’exportation** sur le format de fichier requis. Adobe Campaign accepte les importations de fichiers au format CSV. Choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut de la section **Champs**, indiquez un **Champ utilisateur** et **Nom de colonne** pour chaque attribut d’utilisateur à inclure dans l’exportation. Par exemple :

<table class="table"><thead>
<tr>
<th>Champ utilisateur</th>
<th>Nom de la colonne</th>
</tr>
</thead>
<tbody>
<tr>
<td>`email`</td>
<td>Adresse courriel</td>
</tr>
<tr>
<td>`created_at`</td>
<td>Créé à</td>
</tr>
<tr>
<td>`given_name`</td>
<td>Prenom</td>
</tr>
<tr>
<td>`family_name`</td>
<td>Nom de famile</td>
</tr>
</tbody>
</table>

Après avoir ajouté les champs utilisateur, cliquez sur la touche **Exporter les utilisateurs** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV que vous utiliserez dans la section suivante.

## Importer un fichier de données utilisateur

Avant de commencer, vous pouvez obtenir de plus amples informations en consultant la documentation [Adobe Campaign : Importation de données](https://docs.adobe.com/content/help/en/campaign-classic/using/getting-started/importing-and-exporting-data/importing-data.html).

Connectez-vous au tableau de bord de votre client Adobe Campaign et accédez à **Profils et cibles > Tâches**. Créez une nouvelle tâche d’importation en cliquant sur le bouton **Créer** et en sélectionnant **Nouvelle importation**.

Une nouvelle fenêtre de l’**assistant d’importation** s’ouvre. À l’étape **Sélection du modèle**, vous pouvez définir vos paramètres d’importation.

Une fois que vous avez configuré vos paramètres d’importation, cliquez sur le bouton **Suivant** pour continuer.

À l’étape **Fichier à importer**, téléchargez le fichier CSV des données utilisateur que vous avez exporté depuis Auth0 dans la section précédente. Cliquez sur le bouton **Suivant** pour passer au **Mappage des champs**.

Ensuite, faites correspondre le schéma du fichier d’exportation au schéma de votre base de données Adobe Campaign. Vérifiez que les noms et les types de champs sont corrects, puis cliquez sur le bouton **Suivant**.

Effectuez les étapes de configuration restantes en définissant votre mode de rapprochement des données et en sélectionnant un dossier, une liste ou un service pour les utilisateurs importés.

Enfin, commencez l’importation en cliquant sur le bouton **Commencer**dans la fenêtre **Exécution de l’importation de données**.

Et voilà! Vous avez importé avec succès vos utilisateurs Auth0 dans Adobe Campaign.
