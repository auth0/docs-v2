Dans cet article, vous découvrirez comment exporter des données utilisateur depuis Auth0 vers un fichier CSV, puis les importer dans Salesforce Marketing Cloud en utilisant Email Studio.

Pour en savoir plus, veuillez consulter l’article de Salesforce [Liste de contrôle de démarrage pour Email Studio](https://help.salesforce.com/articleView?id=mc_es_get_started_with_email_studio.htm&type=5).

## Créer un fichier de données utilisateur

Commencez par vous rendre à la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez **l’extension Importation/Exportation des utilisateurs**. Sur la page d’extension, sélectionnez **Exporter** depuis le menu.

Ensuite, définissez le **Format d’exportation** au format de fichier requis. Salesforce Marketing Cloud accepte les importations de fichiers au format CSV, alors sélectionnez l’option `Tab Separated Value file (*.csv)`.

En haut dans la section **Champs**, saisissez un **User Champ utilisateur** et **Nom de la colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

<tr>

</tr>
</thead>
<tbody>
<tr>
<td>`email`</td>
<td>Adresse courriel</td>
</tr>
<tr>
<td>`created_at`</td>
<td>Créé à</td>
</tr>
<tr>
<td>`given_name`</td>
<td>Prenom</td>

<td>`family_name`</td>
<td>Nom de famile</td>
</tr>
</tbody>
</table>

Après avoir ajouté les champs utilisateur, cliquez sur le bouton **Exporter les utilisateurs** pour démarrer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV à utiliser dans la section suivante.

## Importer un fichier de données utilisateur

Avant de commencer, assurez-vous que votre compte Salesforce dispose des autorisations utilisateur requises pour importer des données. Vous devriez également prendre un moment pour revoir les étapes décrites dans l’article de Salesforce. [Email Studio : Importer des abonnés](https://help.marketingcloud.com/en/documentation/exacttarget/subscribers/subscribers_for_interactive_marketing_hub/imports/importing_subscribers/) et consultez aussi [De quelles autorisations ai-je besoin pour importer des enregistrements?](https://help.salesforce.com/articleView?id=faq_import_general_permissions.htm).

Pour importer votre fichier CSV dans Salesforce Marketing Cloud, procédez comme suit :

1. Connectez-vous à Salesforce Marketing Cloud et ouvrez **Email Studio**.
2. Accéder à **Abonnés > Listes**.
3. Choisissez la liste à importer et sélectionnez **Importer**.
4. Après l’introduction de l’assistant d’importation, sélectionnez votre fichier CSV comme **Source de téléchargement** et sélectionnez CSV comme **Format des données**. Saisissez les paramètres restants et cliquez sur **Suivant**.
5. Dans la boîte de dialogue **Mapper les attributs**, associez les champs de données de votre fichier CSV aux champs de données corrects. Lorsque vous avez terminé vos associations, cliquez sur **Suivant**.
6. Après avoir vérifié vos associations dans la boîte de dialogue **Associations confirmées**, cliquez sur **Commencer** pour démarrer l’importation.

C’est fini! Vous avez importé avec succès vos utilisateurs Auth0 dans Salesforce Marketing Cloud.
