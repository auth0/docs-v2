Dans cet article, vous apprendrez à exporter les données des utilisateurs dans Auth0 vers un fichier CSV, puis à les importer dans Eloqua à l’aide de l’assistant de téléversement des contacts.

## Créer un fichier de données utilisateur

Commencez par vous rendre dans la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard et ouvrez **l’extension Importation/Exportation des utilisateurs**. Sur la page de l’extension, sélectionnez **Exporter** dans le menu.

Ensuite, définissez le **Format d’exportation** sur le format de fichier requis. Eloqua accepte les importations de fichiers au format CSV; choisissez donc l’option `Tab Separated Value file (*.csv)`.

En haut de la section **Champs**, indiquez un **Champ utilisateur** et un **Nom de colonne** pour chaque attribut utilisateur à inclure dans l’exportation. Par exemple :

<table class="table"><thead>
<tr>
<th>Champ utilisateur</th>
<th>Nom de la colonne</th>
</tr>
</thead>
<tbody>
<tr>
<td>`email`</td>
<td>Adresse courriel</td>
</tr>
<tr>
<td>`created_at`</td>
<td>Créé à</td>
</tr>
<tr>
<td>`given_name`</td>
<td>Prenom</td>
</tr>
<tr>
<td>`family_name`</td>
<td>Nom de famile</td>
</tr>
</tbody>
</table>

Après avoir ajouté les champs utilisateur, cliquez sur la touche **Exporter les utilisateurs** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV à utiliser dans la section suivante.

## Importer un fichier de données utilisateur

Avant de commencer, vous pouvez trouver plus d’informations en visitant le [Centre d’aide Oracle Eloqua : Téléversement des contacts](https://docs.oracle.com/cloud/latest/marketingcs_gs/OMCAA/index.html#Help/Contacts/Tasks/UploadingContacts.htm).

Pour importer votre fichier CSV dans Eloqua, procédez comme suit :

1. Connectez-vous à Eloqua et naviguez vers **Audience > Contacts > Téléverser** pour ouvrir le **Assistant de téléversement de contacts**.
2. Configurez l’importation à l’aide des champs fournis dans l’onglet **Choisir la source de données**.
3. Cliquez sur la touche **Nuage** et sélectionnez le fichier à téléverser.
4. Après avoir vérifié le contenu du fichier sur l’onglet **Examen**, cliquez sur **Étape suivante**.
5. Sous l’onglet **Mapper les champs**, saisissez les paramètres de mappage de champ pour faire correspondre les données de votre fichier CSV aux champs de contact. Cliquez sur **Étape suivante** pour continuer.
6. Complétez la dernière étape de l’assistant et cliquez sur **Terminer**.

C’est terminé! Vous avez importé avec succès vos utilisateurs Auth0 dans Eloqua.
