Dans cet article, vous découvrirez comment exporter des données utilisateur depuis Auth0 vers un fichier CSV, puis comment les importer dans Alterian en utilisant l’outil d’importation de données.

## Créer un fichier de données d’utilisateur

Commencez par vous rendre dans la section [Extensions](https://manage.auth0.com/#/extensions) du Dashboard (Tableau de bord) et ouvrez **l’extension User Import/Export (Importation/Exportation des utilisateurs).**. Dans la page de l’extension, sélectionnez **Export (Exporter)** à partir du menu.

Ensuite, définissez le **Format d’exportation** au format de fichier requis. Alterian accepte les importations de fichiers au format CSV. Sélectionnez alors l’option `Tab Separated Value file (*.csv)`.

En haut de la section **Champs**, indiquez un **Champ utilisateur** et **Nom de colonne** pour chaque attribut d’utilisateur à inclure dans l’exportation. Par exemple :

<table class="table"><thead>
<tr>
<th>Champ utilisateur</th>
<th>Nom de la colonne</th>
</tr>
</thead>
<tbody>
<tr>
<td>`email`</td>
<td>Adresse courriel</td>
</tr>
<tr>
<td>`created_at`</td>
<td>Créé à</td>
</tr>
<tr>
<td>`given_name`</td>
<td>Prenom</td>
</tr>
<tr>
<td>`family_name`</td>
<td>Nom de famile</td>
</tr>
</tbody>
</table>

Après avoir ajouté les champs utilisateur, cliquez sur la touche **Exporter les utilisateurs** pour lancer l’exportation. Une fois l’exportation terminée, téléchargez le fichier CSV que vous utiliserez dans la section suivante.

## Importer un fichier de données d’utilisateur

Avant de commencer, vous pouvez obtenir de plus amples informations en consultant [Alterian Campaign Manager : Importation de données](http://cm.help.alterian.com/CM404/Default.htm#Customer_Analytics/Import_Export/Data_Import.htm).

Pour importer votre fichier CSV dans Alterian, procédez comme suit :

1. Ouvrez Alterian Campaign Manager et cliquez sur **Documents**.
2. Sélectionnez le répertoire dans lequel vous souhaitez importer le fichier.
3. Cliquez **Téléverser les fichiers** et sélectionnez votre fichier CSV.
4. Cliquez sur **Téléverser**.
5. Une fois le téléversement terminé, ouvrez l’outil **Importation de données**.
6. Configurez vos paramètres d’importation à l’aide des champs fournis dans la fenêtre **Nouvelles données d’importation**.
7. Après avoir vérifié vos paramètres, cliquez sur **Exécuter les processus**.

C’est tout! Vous avez bien importé vos utilisateurs Auth0 dans Alterian.
