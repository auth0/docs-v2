Le Dashboard à authentification unique (<Tooltip href="/docs/fr-ca/glossary?term=single-sign-on" tip="Authentification unique (SSO)
Service qui, après qu’un utilisateur se soit connecté à une application, le connecte automatiquement à d’autres applications." cta="Voir le glossaire">SSO</Tooltip>) est une application Web conçue pour résoudre un problème familier à de nombreuses personnes. Les organisations de toutes tailles maintiennent une variété d’applications différentes pour gérer diverses fonctions commerciales telles que la comptabilité, les ressources humaines, le développement, le soutien, etc. Se souvenir des noms d’utilisateur, des mots de passe et des URL de connexion pour toutes vos applications peut être fastidieux. Avec cette extension, vous pouvez simplifier l’expérience d’authentification en activant la connexion à SSO pour vos utilisateurs sur plusieurs applications d’entreprise. Elle vous permet de créer une liste de tous les services infonuagiques pour lesquels un utilisateur peut s’authentifier avec des connexions à SSO.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

Les utilisateurs ne doivent pas pouvoir accéder, à partir du tableau de bord ou d’une autre manière, au fournisseur de services sans disposer des autorisations appropriées (groupes/rôles) pour ce faire. Idéalement, ces utilisateurs ne devraient pas voir sur le tableau de bord les fournisseurs de services auxquels ils n’ont pas accès.

</Callout>

Le Dashboard à authentification unique prend en charge deux types d’utilisateurs :

* Les **utilisateurs** qui peuvent se connecter au tableau de bord pour sélectionner une application à laquelle se connecter avec la SSO.
* Les **administrateurs** qui peuvent configurer des applications visibles par les utilisateurs.

Pour configurer cette extension, procédez comme suit :

1. [Créez une application Dashboard à authentification unique](/docs/fr-ca/customize/extensions/single-sign-on-dashboard-extension/create-sso-dashboard-application)
2. [Installez l’extension Dashboard à authentification unique](/docs/fr-ca/customize/extensions/single-sign-on-dashboard-extension/install-sso-dashboard-extension)
3. [Ajoutez les applications au Dashboard à authentification unique](/docs/fr-ca/customize/extensions/single-sign-on-dashboard-extension/add-applications-to-the-sso-dashboard)
4. [Mettez à jour les applications dans le Dashboard à authentification unique](/docs/fr-ca/customize/extensions/single-sign-on-dashboard-extension/update-applications-on-the-sso-dashboard)

Vous pouvez également [afficher cette extension de Dashboard à authentification unique sur GitHub](https://github.com/auth0-extensions/auth0-sso-dashboard-extension).
