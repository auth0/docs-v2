L’extension des journaux Webtask en temps réel affiche en temps réel tous les journaux de code personnalisé dans votre compte. Cela inclut les sorties et exceptions `console.log`. Les fichiers`Console.logs` sont utiles si vous avez des problèmes avec les scripts de base de données. Vous pouvez afficher les résultats de vos scripts de base de données tels que les messages de réussite, les messages d’erreur et les déclarations `console.log()`.

Pour en savoir plus, consultez [Journaux Webtask Auth0 en temps réel](https://github.com/auth0/auth0-extension-realtime-logs) dans le référentiel Github d’Auth0.

<Warning>

L’extension Journaux Webtask en temps réel utilise la requête de l’utilisateur : vous devez donc faire attention à ne pas consigner d’informations sensibles. Par exemple, vos scripts de base de données personnalisés interagissent largement avec l’objet `user` qui peut contenir des informations sensibles. La consignation de l’objet complet peut conduire à la divulgation d’informations sensibles dans l’extension Journaux Webtask.

Auth0 conseille fortement de ne pas adopter de telles pratiques. **Nous vous conseillons d’être attentif aux objets que vous enregistrez et de veiller à ce qu’aucune information sensible ne soit enregistrée.**

</Warning>

## Configuration de l’extension

Installation et configuration de l’extension :

1. Accédez à la page [Extensions](https://manage.auth0.com/#/extensions) de [Auth0 Dashboard](https://manage.auth0.com/#), et cliquez sur la case **Journaux Webtask en temps réel**. La fenêtre **Installer l’extension** s’ouvre.
2. Cliquez sur la touche **Installer**.

## Utilisation de l’extension

Pour afficher vos journaux :

1. Accédez à [Auth0 Dashboard > Extensions](https://manage.auth0.com/#/extensions), et sélectionnez **Extensions installées**.
2. Sélectionnez **Journaux Webtask en temps réel**.

Pour afficher les journaux en mode plein écran, cliquez sur **Mode plein écran**. Pour quitter le mode plein écran, appuyez sur la touche **Échappement**.

Pour effacer tous les journaux, cliquez sur **Effacer la console**.

## Règles de débogage

Vous pouvez utiliser l’extension Journaux Webtask en temps réel pour déboguer toutes les règles de votre implémentation, y compris toutes les sorties et exceptions `console.log`. Pour en savoir plus, veuillez consulter [Règles d'Auth0](/docs/fr-ca/customize/rules).

Dans l’exemple suivant, vous allez créer une règle générique « Hello World », l’exécuter, puis utiliser l’extension Journaux Webtask en temps réel pour consulter les résultats en temps réel.

1. Accédez à [Auth0 Dashboard > Règles du pipeline](https://manage.auth0.com/#/rules) et sélectionnez **Créer une règle**.
2. Sélectionnez le modèle **Règle vide**, collez le code ci-dessous (ou modifiez-le comme vous le souhaitez), et **Enregistrer les modifications**.

   ```javascript lines
   function (user, context, callback) {
     user.hello = 'world';
     console.log('===> set "hello" for ' + user.name);
     callback(null, user, context);
   }
   ```

   

   
3. Ouvrez un nouvel onglet, accédez à [Auth0 Dashboard > Extensions](https://manage.auth0.com/#/extensions), et sélectionnez **Extensions installées**.
4. Sélectionnez **Journaux Webtask en temps réel**. Vous visualisez maintenant les journaux en temps réel et vous êtes prêt à essayer votre règle.
5. Passez à votre onglet Règles et cliquez sur **Essayez cette règle**.
6. Passez à votre onglet Journaux Webtask en temps réel pour afficher les résultats.
