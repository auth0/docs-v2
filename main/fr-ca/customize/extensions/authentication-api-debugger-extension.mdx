L’extension Débogueur de l’API d’authentification vous permet de tester différents points de terminaison de l’API d’authentification Auth0.

## Installer l’extension

Pour installer cette extension :

1. Accédez à [Auth0 Dashboard > Extensions](https://manage.auth0.com/#/extensions), puis sélectionnez **Débogueur de l’API d’authentification Auth0**. La fenêtre **Installer l’extension** s’ouvre.
2. Sélectionnez **Installer**.

## Autoriser l’extension

Étant donné que l’extension communiquera avec <Tooltip href="/docs/fr-ca/glossary?term=management-api" tip="Management API
Un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire">Management API</Tooltip> en votre nom pour récupérer des détails sur les applications que vous avez configurées dans votre Tableau de bord Auth0, vous devrez autoriser son accès.

Pour autoriser l’extension :

1. Si vous suivez ce guide, vous devriez déjà être sur la vue [Extensions installées](https://manage.auth0.com/#/extensions) à partir de [Auth0 Dashboard > Extensions](https://manage.auth0.com/#/extensions). Sélectionnez **Débogueur de l’API d’authentification Auth0** pour lancer l’extension. Une boîte de dialogue de consentement s’ouvrira et vous demandera l’accès à votre compte.
2. Consentez à autoriser l’extension à accéder à votre compte. Une fois votre consentement donné, l’extension se lance.

## Configurer l’extension

Une fois l’extension installée, vous devez appliquer la configuration de base. Pour configurer l’extension :

1. Si vous suivez ce guide, vous devriez déjà avoir lancé l’extension. Sélectionnez la vue **Configuration**.

   <Frame>![Dashboard - Extensions - Authentication API Debugger - Configuration](/images/fr-ca/cdy7uua7fh8z/5riI24ArqUAKRXhZfNWIQ5/ab7d6825288752132d14e54db1e563d0/configuration.png)</Frame>

2. Définissez les paramètres de configuration suivants et sélectionnez **Enregistrer** :

* **Domaine** : Le domaine de votre locataire. Ce champ est en lecture seule et s’affiche à titre informatif.
* **Application** : Application pour laquelle vous souhaitez lancer des flux d’authentification. Vous pouvez gérer vos applications à partir de [Auth0 Dashboard > Applications > Applications](https://manage.auth0.com/#/applications).
* **Callback URL** : L’URL de rappel URL pour cette extension. Vous devez également ajouter cette URL URL dans le champ **URL de rappel autorisées** dans les paramètres de l’application sélectionnée.
* **State** : Informations facultatives relatives à l’état pouvant être envoyées avec le flux d’authentification.
* **Connection** : Nom de la connexion avec laquelle vous souhaitez vous connecter. Vous pouvez utiliser ce paramètre pour contourner l’écran de connexion principal et accéder directement à l’écran de connexion du fournisseur d’identité concerné.

## Exécuter les flux

Après avoir configuré l’extension, vous pouvez exécuter n’importe lequel des flux en sélectionnant la vue **OAuth2/OIDC**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

Pour certains flux, il se peut que vous deviez spécifier davantage de paramètres que ceux appliqués pour la configuration de base. Ces paramètres dépendront du flux exécuté. Nous vous suggérons donc de consulter la documentation de l'[Auth0 Authentication API](https://auth0.com/docs/api) pour en savoir plus sur les paramètres applicables à chaque flux.

</Callout>

<Frame>![Dashboard - Extensions - Authentication API Debugger - OAuth OIDC Settings](/images/fr-ca/cdy7uua7fh8z/72iYPxZU9OiHkKkHtgPTjI/fc166f3ae74e595c8d916276e78edb99/oauth-oidc-settings.png)</Frame>

Une fois que vous avez exécuté un flux particulier, vous verrez un écran qui affiche les informations renvoyées par Auth0. Les informations exactes retournées dépendront du flux qui a été exécuté. Voici un exemple de résultats après l’exécution d’un flux de **connexion OAuth2/OIDC** normal :

<Frame>![Dashboard - Extensions - Authentication API Debugger - Flow Executed](/images/fr-ca/cdy7uua7fh8z/13TRS8SQUd4OsfLYRWMoe6/3478778905ad253160022db1f1f6f31a/flow-executed.png)</Frame>

Pour exécuter un autre flux, sélectionnez la vue **Connexion** pour revenir à l’écran **Configuration**.
