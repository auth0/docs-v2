L’objet `event` pour le déclencheur d’Actions d’échange d’identifiants fournit des informations contextuelles sur la demande d’échange d’identifiants d’un client.

<table class="table">
<thead>
<tr>
<th>Propriété</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td>
<p>`event.accessToken`</p>
</td>
<td>
<p>Informations relatives au jeton d’accès à émettre.</p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>`customClaims` <em>Dictionnaire</em>.</li>
<li>`scope` <em>Tableau de chaînes</em>.</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.client`</p>
</td>
<td>
<p>Informations sur le client utilisé lors de cet échange de jetons.</p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`client_id` <em>Chaîne</em>.
            <span>L’ID client de l’application dans laquelle l’utilisateur se connecte.</span>
</li>
<li>
`metadata` <em>Dictionnaire</em>.
            <span>Un objet qui contient les autres propriétés d’une application.</span>
</li>
<li>
`name` <em>Chaîne</em>.
            <span>Le nom de l’application (tel que défini dans le Dashboard).</span>
</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.organization`</p>
</td>
<td>
<p>Détails sur l’Organization associée à la transaction en cours.</p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`display_name` <em>Chaîne</em>.
            <span>Le nom amical de l’Organization.</span>
</li>
<li>
`id` <em>Chaîne</em>.
            <span>L’identifiant de l’Organization.</span>
</li>
<li>
`metadata` <em>Dictionnaire</em>.
            <span>Métadonnées associées à l’Organization.</span>
</li>
<li>
`name` <em>Chaîne</em>.
            <span>Le nom de l’Organization.</span>
</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.request`</p>
</td>
<td>
<p>Détails sur la requête qui a initié la transaction.</p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`body` <em>Dictionnaire</em>.
            <span>Le corps de la requête POST. Ces données ne seront disponibles
              que lors de l’actualisation du jeton, des flux d’échange de jetons de type identifiant client et
              de l’action de pré-enregistrement d’utilisateur.</span>
</li>
<li>
`geoip` <em>Objet</em>.
            <p>Comprend les propriétés suivantes :</p>
<ul>
<li>`cityName` <em>Chaîne facultative</em>.</li>
<li>`continentCode` <em>Chaîne facultative</em>.</li>
<li>`countryCode` <em>Chaîne facultative</em>.</li>
<li>`countryCode3` <em>Chaîne facultative</em>.</li>
<li>`countryName` <em>Chaîne facultative</em>.</li>
<li>`latitude` <em>Numéro facultatif</em>.</li>
<li>`longitude` <em>Numéro facultatif</em>.</li>
<li>`subdivisionCode` <em>Chaîne facultative</em>.</li>
<li>`subdivisionName` <em>Chaîne facultative</em>.</li>
<li>`timeZone` <em>Chaîne facultative</em>.</li>
</ul>
</li>
<li>
`hostname` <em>Chaîne facultative</em>.
            <span>Le nom d’hôte utilisé pour le flux
              d’authentification.</span>
</li>
<li>
`ip` <em>Chaîne</em>.
            <span>L’adresse IP d’origine de la requête.</span>
</li>
<li>
`language` <em>Chaîne facultative</em>.
            <span>La langue demandée par le navigateur.</span>
</li>
<li>
`method` <em>Chaîne de caractères</em>.
            <span>La méthode HTTP utilisée pour la requête</span>
</li>
<li>
`user_agent` <em>Chaîne facultative</em>.
            <span>La valeur de l’en-tête `User-Agent` reçu lors de l’initiation
              de la transaction.</span>
</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.resource_server`</p>
</td>
<td>
<p>
          Informations sur le serveur de ressources
          qui délivre le jeton d’accès.
        </p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`identifier` <em>Chaîne</em>.
            <span>L’identifiant du serveur de ressources. Par exemple :
              `https://your-api.example.com`.</span>
</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.tenant`</p>
</td>
<td>
<p>Informations sur le locataire utilisées lors de cet échange de jetons.</p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`id` <em>Chaîne</em>.
            <span>Le nom du locataire.</span>
</li>
</ul>
</td>
</tr>
<tr>
<td>
<p>`event.transaction`</p>
</td>
<td>
<p>Informations sur la transaction d’échange d’informations.</p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`requested_scopes` <em>Tableau de chaînes</em>.
            <span>Les  permissions spécifiées (le cas échéant) lors de la demande du
               jeton d’accès.</span>
</li>
</ul>
</td>
</tr>
</tbody>
</table>
