<td>
<p>`event.authentication`</p>
<p><em>(Facultatif)</em></p>
</td>
<td>
<p>
          Détails concernant les signaux d’authentification obtenus pendant le flux de connexion.
        </p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`methods` <em>Tableau d’objets</em>.
            <p>
              Contient les méthodes d’authentification utilisées par un utilisateur
              au cours de sa session.
            </p>
<p>Éléments du tableau :</p>
<ul>
<li>
<p>L’un des schémas d’objets suivants :</p>
<ul>
<li>
<p>Un objet avec les propriétés suivantes :</p>
<ul>
<li>
`name` <em>Chaîne</em>.
                        <p>
                          Le nom du premier facteur qui a été complété.
                          Les valeurs comprennent :
                        </p>
<ul>
<li>
`"federated"` Une connexion sociale ou
                            d’entreprise a été utilisée pour authentifier
                            l’utilisateur en tant que premier facteur.
                          </li>
<li>
`"pwd"` Un mot de passe a été utilisé pour
                            authentifier un utilisateur de connexion de base de données comme premier
                            facteur.
                          </li>
<li>
`"passkey"` Une clé d’identification a été utilisée pour
                            authentifier un utilisateur de connexion de base de
                            données comme premier facteur.
                          </li>
<li>
`"sms"` Une connexion SMS sans mot de passe a été utilisée pour
                            authentifier l’utilisateur en tant que premier facteur.
                          </li>
<li>
`"email"` Une connexion courriel sans mot de passe
                            a été utilisée pour authentifier l’utilisateur en tant
                            que premier facteur ou vérifier le courriel pour la réinitialisation du mot de passe.
                          </li>
<li>
`"phone_number"` Un numéro de téléphone a été utilisé
                            pour la réinitialisation du mot de passe.
                          </li>
<li>
`"mock"` Utilisé pour les tests internes.
                          </li>
<li>
`string` Méthode d’authentification personnalisée
                            indiquée par une URL (comme deuxième facteur ou facteur ultérieur).
                          </li>
</ul>
</li>
<li>`timestamp` <em>Chaîne</em>.</li>
</ul>
</li>
<li>
<p>Un objet avec les propriétés suivantes :</p>
<ul>
<li>
`name` <em>La valeur « mfa »</em>.
                        <span>L’utilisateur a procédé à une authentification multifacteur
                            (deuxième facteur ou facteurs ultérieurs).</span>
</li>
<li>`timestamp` <em>Chaîne</em>.</li>
</ul>
</li>
</ul>
</li>
</ul>
</li>
<li>
`riskAssessment` <em>Objet facultatif</em>.
            <p>
              Détails sur les évaluations de risques obtenues lors du processus
              de connexion ou de réinitialisation du mot de passe.
            </p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`assessments` <em>Objet</em>.
                <p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`ImpossibleTravel` <em>Objet facultatif</em>.
                    <p>
                      Détermine si l’utilisateur se connecte à partir d’un lieu
                      signalant un trajet impossible.
                    </p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`code` <em>Chaîne</em>.
                        <p>Valeurs possibles :</p>
<ul>
<li>`minimal_travel_from_last_login`</li>
<li>`travel_from_last_login`</li>
<li>
`substantial_travel_from_last_login`
</li>
<li>
`impossible_travel_from_last_login`
</li>
<li>`invalid_travel`</li>
<li>`missing_geoip`</li>
<li>`anonymous_proxy`</li>
<li>`unknown_location`</li>
<li>`initial_login`</li>
<li>`location_history_not_found`</li>
<li>`assessment_not_available`</li>
</ul>
</li>
<li>
`confidence` <em>Chaîne</em>.
                        <p>Valeurs possibles :</p>
<ul>
<li>`low`</li>
<li>`medium`</li>
<li>`high`</li>
<li>`neutral`</li>
</ul>
</li>
</ul>
</li>
<li>
`NewDevice` <em>Objet facultatif</em>.
                    <p>
                      Détermine si l’utilisateur se connecte à partir d’un appareil connu.
                    </p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`code` <em>Chaîne</em>.
                        <p>Valeurs possibles :</p>
<ul>
<li>`match`</li>
<li>`partial_match`</li>
<li>`no_match`</li>
<li>`initial_login`</li>
<li>`unknown_device`</li>
<li>`no_device_history`</li>
<li>`assessment_not_available`</li>
</ul>
</li>
<li>
`confidence` <em>Chaîne</em>.
                        <p>Valeurs possibles :</p>
<ul>
<li>`low`</li>
<li>`medium`</li>
<li>`high`</li>
<li>`neutral`</li>
</ul>
</li>
<li>
`details` <em>Objet facultatif</em>.
                        <p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`device` <em>Chaîne facultative</em>.
                            <p>Valeurs possibles :</p>
<ul>
<li>`known`</li>
<li>`unknown`</li>
</ul>
</li>
<li>
`useragent` <em>Chaîne facultative</em>.
                            <p>Valeurs possibles :</p>
<ul>
<li>`known`</li>
<li>`unknown`</li>
</ul>
</li>
</ul>
</li>
</ul>
</li>
<li>
`UntrustedIP` <em>Objet facultatif</em>.
                    <p>
                      Indique si l’IP a été trouvée dans le répertoire Auth0 des
                      IP à faible réputation.
                    </p>
<p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`code` <em>Chaîne</em>.
                        <p>Valeurs possibles :</p>
<ul>
<li>`not_found_on_deny_list`</li>
<li>`found_on_deny_list`</li>
<li>`invalid_ip_address`</li>
<li>`assessment_not_available`</li>
</ul>
</li>
<li>
`confidence` <em>Chaîne</em>.
                        <p>Valeurs possibles :</p>
<ul>
<li>`low`</li>
<li>`medium`</li>
<li>`high`</li>
<li>`neutral`</li>
</ul>
</li>
<li>
`details` <em>Objet facultatif</em>.
                        <p>Comprend les propriétés suivantes :</p>
<ul>
<li>
`category` <em>Chaîne facultative</em>.
                          </li>
<li>
`ip` <em>Chaîne facultative</em>.
                            <span>L’adresse IP à l’origine de la demande.</span>
</li>
<li>
`matches` <em>Chaîne facultative</em>.
                          </li>
<li>`source` <em>Chaîne facultative</em>.</li>
</ul>
</li>
</ul>
</li>
</ul>
</li>
<li>
`confidence` <em>Chaîne</em>.
                <p>Score de risque global</p>
<p>Valeurs possibles :</p>
<ul>
<li>`low`</li>
<li>`medium`</li>
<li>`high`</li>
<li>`neutral`</li>
</ul>
</li>
<li>`version` <em>Chaîne</em>.</li>
</ul>
</li>
</ul>
</td>

            <span>Un objet qui contient les autres propriétés d’une application.</span>
