Auth0 allows you to configure your own SMTP email provider so you can more completely manage, monitor, and troubleshoot your email communications. L’infrastructure de messagerie intégrée d’Auth0 ne doit être utilisée que pour les courriels de test.

## Prérequis

* Pour que les courriels puissent être envoyés de Auth0 à votre SMTP, des ports doivent être ouverts et il faut autoriser les connexions entrantes provenant d’adresses IP spécifiques. Pour mettre à jour la liste des adresses IP, accédez à [Dashboard > Image de marque > Fournisseur de messagerie électronique](https://manage.auth0.com/#/branding/email_provider). Voir [Ajouter des adresses IP à AllowList](/docs/fr-ca/secure/security-guidance/data-security/allowlist) pour plus de détails.
* Vous devez disposer d’un compte [Mailgun](https://mailgun.com).

## Configure Mailgun

1. Connectez-vous à votre compte [Mailgun](https://mailgun.com).
2. Dans Mailgun, allez à **Envoi > Domaines** et ajoutez votre domaine personnalisé. Mailgun n’autorise l’envoi de courriels qu’à partir de domaines vérifiés.
3. Dans Mailgun, allez à **Paramètres > Clés API**. Votre clé API a été créée lorsque vous avez créé votre compte; copiez-la depuis **Clé API privée**.
4. Accédez à [Auth0 Dashboard > Image de marque> Fournisseur de messagerie électronique](https://manage.auth0.com/#/branding/email_provider).
5. Activez la bascule **Utiliser mon propre fournisseur de messagerie**.
6. Cliquez sur le logo **Mailgun**.
7. Fournissez une adresse courriel **De**, saisissez le **Domaine** que vous avez précédemment ajouté à Mailgun, saisissez la **Clé API** Mailgun que vous avez précédemment copiée, puis sélectionnez votre **Région**.
8. Cliquez sur **Enregistrer**.

## Courriel de test

Vous pouvez maintenant envoyer un courriel de test en utilisant le bouton **Envoyer un courriel de test**. Si vous avez tout configuré correctement, vous recevrez un courriel de confirmation. Si vous ne recevez pas de courriel après quelques minutes, veuillez vérifier vos [Journaux Auth0](https://manage.auth0.com/#/logs) pour toute erreur.

Le [tableau de bord Mailgun](https://app.mailgun.com/app/dashboard) affiche maintenant les informations de livraison pour tous les courriels envoyés à vos utilisateurs.
