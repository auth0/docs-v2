Lorsque Auth0 envoie un courriel pour informer un utilisateur que son compte a été bloqué en raison d’une activité suspecte, le message contient un lien permettant de réactiver l’origine de la demande.

Le courriel envoyé à l’utilisateur se lit comme suit :

`Tentative d’accès bloqué`

`Nous avons détecté une activité suspecte à partir de l’adresse IP 123.23.123.23 de Buenos Aires, en Argentine. Si ce n’est pas votre adresse IP, cliquez ici pour la débloquer. Sinon, ignorez ce courriel. Clarification : votre compte n’est pas bloqué, seul l’accès à celui-ci depuis l’IP susmentionnée.`

`Merci,`

`Appli par défaut`

Pour personnaliser le modèle utilisé pour ce message, suivez ces étapes :

1. Rendez-vous à [Dashboard > Branding (Image de marque) > Email Templates (Modèles de courriel)](https://manage.auth0.com/#/templates).
2. Dans la liste déroulante **Template (Modèle)** sélectionnez **Blocked Account Email (Courriel de comptes bloqués)**.
3. Apportez les modifications souhaitées. Pour en savoir plus, consultez [Personnaliser les modèles de courriel](/docs/fr-ca/customize/email/email-templates).
4. Pour tester, cliquez sur **Try (Essayer)**, saisissez une adresse courriel valide que vous avez le droit d’afficher et choisissez le type de connexion adéquat.
5. Cliquez sur **Cancel (Annuler)** pour revenir à la page du modèle de courriel. Vous pouvez apporter des modifications et réessayer, puis sélectionner **Save (Enregistrer)** lorsque vous êtes satisfait du résultat.
