Explique les codes d’erreur et les solutions possibles pouvant survenir avec l’API de modification automatique du mot de passe.

## Format d’erreur

Les messages d’erreur sont renvoyés au format standard :

## Codes d’erreur

### invalid_request

Cette erreur se produit lorsque vous saisissez des paramètres non valides. Les messages d’erreur décrivent le problème, par exemple un paramètre obligatoire ou un format non valide.

### invalid_user_password

Cette erreur résulte de l’envoi d’une mauvaise combinaison des paramètres `username`/`email` et `old_password`. Lancez une nouvelle demande avec le bon nom d’utilisateur et `old_password`.

### change_password_error

Cette erreur résulte de diverses conditions liées au fournisseur d’identité sous-jacent. En général, cela se produit lorsque vous utilisez une base de données personnalisée et que vous n’avez pas implémenté le script de changement de mot de passe.
