Vous pouvez supprimer votre locataire Auth0 ou le réinitialiser à la configuration par défaut dans <Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>. L’équipe d’assistance Auth0 peut vous aider à supprimer les données de votre compte Auth0.

## Supprimer un locataire

<Warning>

La suppression d’un locataire est une action irréversible et destructive qui ne peut être annulée. Les locataires supprimés ne peuvent pas être restaurés, le nom du locataire ne peut pas être réutilisé lors de la création d’un nouveau locataire et les métriques (indicateurs) liées au locataire sont perdues.

</Warning>

Avant de supprimer un locataire, envisagez les options suivantes :

* [Changer la propriété du locataire](/docs/fr-ca/get-started/manage-dashboard-access).
* [Réinitialiser la configuration du locataire](#reset-tenant-configuration).

### Supprimer le domaine personnalisé

Si vous avez configuré un domaine personnalisé pour un locataire que vous souhaitez supprimer, vous devez d’abord supprimer le domaine personnalisé. Si vous ne le faites pas, il ne sera pas possible d’utiliser ce domaine personnalisé avec un autre locataire.

### Déterminer le processus de suppression

Le processus de suppression d’un locataire dépend du [plan d’abonnement](https://www.auth0.com/pricing) auquel il est associé et au [type de locataire](/docs/fr-ca/get-started/auth0-overview/create-tenants/child-tenants) (principal ou enfant). Vous pourrez peut-être supprimer le locataire dans [Auth0 Dashboard](#delete-tenant-in-auth0-dashboard), ou vous devrez peut-être [contacter l’assistance ou consulter la communauté Auth0](#contact-auth0-support-or-community).

### Rétrograder le locataire

Avant de pouvoir supprimer un locataire dans le tableau de bord, vous devez le faire passer un plan gratuit. Pour en savoir plus, consultez [Passer à un plan gratuit ou annuler des abonnements Auth0](/docs/fr-ca/troubleshoot/customer-support/manage-subscriptions/downgrade-or-cancel-subscriptions).

### Supprimer le locataire dans Dashboard

Vous pouvez supprimer un locataire dans le tableau de bord si vous avez le [niveau d’accès approprié](/docs/fr-ca/get-started/manage-dashboard-access/feature-access-by-role).

1. Accédez à [Dashboard > Settings (Paramètres)](https://manage.auth0.com/#/tenant/advanced), puis passez à la vue **Advanced (Avancé)**.
2. Recherchez **Danger Zone (Zone de danger)** en bas de la page, puis sélectionnez **Delete (Supprimer)**.
3. Consultez l’invite de confirmation et confirmez que vous souhaitez supprimer définitivement votre locataire.
4. Sélectionnez **Yes, Delete Tenant (Oui, supprimer le locataire)**.

### Contacter l’assistance ou consulter la communauté Auth0

Si votre locataire dispose d’un plan Essai, Essentiel, Professionnel ou Entreprise et que vous ne pouvez pas supprimer votre locataire dans Auth0 Dashboard, [ouvrez un ticket auprès de l’assistance Auth0](/docs/fr-ca/troubleshoot/customer-support/open-and-manage-support-tickets) pour obtenir de l’aide.

Si votre locataire dispose d’un plan Gratuit et que vous ne pouvez pas le supprimer dans Auth0 Dashboard, [publiez le problème dans la communauté Auth0](https://community.auth0.com/) et demandez de l’aide.

## Réinitialiser la configuration du locataire

Vous pouvez réinitialiser votre locataire à sa configuration par défaut avec l’outil [Auth0 Deploy CLI](https://github.com/auth0/auth0-deploy-cli).

L’outil Deploy CLI est un outil qui vous aide à gérer la configuration de votre locataire Auth0. Il s’intègre dans vos flux de développement en tant qu’interface de ligne de commande ou en tant que module de nœud. Pour en savoir plus sur l’outil Deploy CLI et commencer à l’utiliser, veuillez consultez [Outil Deploy CLI](/docs/fr-ca/deploy-monitor/deploy-cli-tool).

## Supprimer les données du compte Auth0

Si vous souhaitez supprimer les données de votre compte Auth0 (y compris l’adresse courriel, les informations de facturation, etc.), suivez les étapes pour supprimer votre locataire, puis [publiez dans la communauté Auth0](https://community.auth0.com/) et demandez de l’aide.

Pour en savoir plus sur les responsabilités d’Auth0 concernant le traitement des données, veuillez consulter [Politique de confidentialité d’Okta](https://www.okta.com/privacy-policy/).
