Vous pouvez utiliser <Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip> pour mettre à jour vos informations de facturation. Cette fonction n’est disponible que pour les administrateurs de locataires sur tous les abonnements basés sur les locataires. Pour les abonnements basés sur un compte, vous devez être propriétaire d’équipe pour accéder à cette fonctionnalité.

## Paiement et facturation

Pour mettre à jour vos informations de facturation :

1. Allez à [Auth0 Dashboard > Settings(Paramètres)](https://manage.auth0.com/#/dashboard/settings).

   <Callout icon="file-lines" color="#0EA5E9" iconType="regular">

   Si vous utilisez une équipe, vous trouverez ce paramètre sous **Teams (Équipes) > Settings (Paramètres) > Subscriptions (Abonnements)**.

   </Callout>

2. Dans **Tenant Settings (Paramètres du locataire)**,sélectionnez **Payment & Billing (Paiement et facturation)**.
3. Cliquez sur **Add Payment Details (Ajouter des détails de paiement )** pour saisir vos informations de paiement.

4. Cliquez sur **Update (Mettre à jour)** pour enregistrer vos modifications.
