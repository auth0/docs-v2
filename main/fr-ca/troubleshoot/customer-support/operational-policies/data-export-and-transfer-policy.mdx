## Exportation des données

Si vous souhaitez exporter vos données depuis Auth0, vous pouvez le faire de plusieurs manières. Veuillez noter que ces outils n’exportent pas les hachages de mot de passe des utilisateurs de votre base de données hébergée par Auth0. Vous pouvez toujours demander ces informations en ouvrant un [ticket de soutien](https://support.auth0.com/). Si vous choisissez de mettre fin à votre utilisation de notre service, veuillez consulter [Exportation des données](/docs/fr-ca/troubleshoot/customer-support/manage-subscriptions/export-data) pour plus de détails.

### Utilisez l’extension Importation/Exportation

Vous pouvez utiliser notre [Extension d’importation/exportation](/docs/fr-ca/manage-users/user-migration/user-import-export-extension) pour exporter presque toutes les données de votre locataire Auth0.

### Utiliser Management API

Pour exporter certains ensembles de données de manière programmatique, utilisez le point de terminaison d’exportation en masse des utilisateurs, proposé par <Tooltip href="/docs/fr-ca/glossary?term=management-api" tip="Management API
Un produit permettant aux clients d’effectuer des tâches administratives." cta="Voir le glossaire">Management API</Tooltip>.

## Transférer des données

Auth0 ne transférera pas de données d’un locataire Auth0 à un autre. Ceci s’applique aussi bien aux clients du Nuage public qu’aux clients du Nuage privé.

Toutes les données de votre locataire Auth0 sont toujours sous votre contrôle et sont disponibles via Management API à tout moment. The only information which is not available through the API are the password hashes of your Auth0-hosted database users and private keys, for security reasons.
