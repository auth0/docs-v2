Si vous devez changer votre mot de passe ou si vous avez oublié le mot de passe de votre compte Auth0, dans la plupart des cas, vous pouvez définir un nouveau mot de passe à partir du <Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip> (Tableau de bord Auth0). La réinitialisation du mot de passe entraîne l’expiration des sessions Auth0.

1. Sur l’écran de connexion au compte Auth0, saisissez l’adresse courriel associée à votre compte et cliquez sur **Continuer**, puis choisissez **Mot de passe oublié?** sur l’écran suivant.
2. Confirmez votre adresse courriel et cliquez sur **Continuer** pour valider. Vous recevrez un courriel contenant des instructions supplémentaires sur la réinitialisation de votre mot de passe.
3. Si votre demande a bien été reçue, vous verrez apparaître le message suivant : « Veuillez consulter le courriel [votre courriel] pour obtenir des instructions sur la réinitialisation de votre mot de passe ».

Il existe des circonstances particulières de réinitialisation du mot de passe :

* Si vous avez activé la <Tooltip href="/docs/fr-ca/glossary?term=multifactor-authentication" tip="Authentification multifacteur (MFA)
Processus d’authentification de l’utilisateur qui utilise un facteur en plus du nom d’utilisateur et du mot de passe, tel qu’un code par SMS." cta="Voir le glossaire">MFA</Tooltip> et que vous avez besoin d’une réinitialisation de la MFA pour votre compte administrateur, contactez le [soutien Auth0](https://support.auth0.com).
* Si vous utilisez un compte social ou d’entreprise pour vous connecter à Auth0, réinitialisez votre mot de passe auprès du fournisseur d’identité approprié.
* Si vous êtes un administrateur et que vous essayez de réinitialiser le mot de passe d’un utilisateur, il existe deux méthodes :

  + Utilisez le flux de réinitialisation de mot de passe interactif avec Universal Login (Connexion universelle) ou le Authentication API (API d’authentification) où l’utilisateur reçoit un courriel avec un lien qui ouvre la page de réinitialisation de mot de passe Auth0 pour saisir un nouveau mot de passe.
  + Définissez directement le nouveau mot de passe en utilisant Management API ou le Dashboard.
