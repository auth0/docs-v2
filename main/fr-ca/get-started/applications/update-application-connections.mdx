You can change your application’s enabled connections using the <Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>.

1. Rendez-vous à [Dashboard > Applications](https://manage.auth0.com/#/applications) et cliquez sur le nom de l’application à afficher.

   <Frame>![Tableau de bord - Liste des applications](/images/fr-ca/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/f29e9dd02a5fb61311387cc3c93ac006/App_List_-_FR.png)</Frame>

2. Cliquez sur l’onglet **Connexions**, puis activez ou désactivez les connexions appropriées pour l’application.
