Les [liens d’applications Android](https://developer.android.com/training/app-links) permettent à une application de se désigner comme gestionnaire par défaut d’un type de lien donné. Par exemple, cliquer sur une URL dans un courriel ouvrirait le lien dans l’application désignée. Ce guide vous montrera comment activer la prise en charge des liens d’applications Android pour votre application enregistrée par Auth0 à l’aide de l’<Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>.

1. Accédez à [Auth0 Dashboard > Applications > Applications](https://manage.auth0.com/#/applications) et sélectionnez le nom de l’application à afficher.

   <Frame>![Tableau de bord - Liste des applications](/images/fr-ca/cdy7uua7fh8z/1ecNwGgFQZxdP57p0tp3jT/f29e9dd02a5fb61311387cc3c93ac006/App_List_-_FR.png)</Frame>

2. Faites défiler vers le bas de la page **Paramètres** et sélectionnez **Afficher les paramètres avancés**.
3. Sélectionnez **Paramètres de l’appareil**, indiquez le [Nom du paquet d’applications](https://developer.android.com/studio/build/application-id) et les empreintes SHA256 du certificat de signature de votre application Android, puis sélectionnez **Enregistrer les modifications**.

   <Frame>![Dashboard (Tableau de bord) Applications Onglet Application Settings (Paramètres d’application) Advanced Settings (Paramètres avancés) Onglet Device Settings (Paramètres de périphérique)](/images/fr-ca/cdy7uua7fh8z/YKKPtWtswHn2FZGFSVWNw/e2d9450d2c288e0bddc97b5148ae6c4a/Device_Settings_-_FR.png)</Frame>

Vous pouvez utiliser la commande suivante pour générer l’empreinte digitale à l’aide de l’outil keytool Java de votre terminal :
`keytool -list -v -keystore my-release-key.keystore`

Pour en savoir plus sur la signature de certificats, consultez la documentation du développeur [Signer votre application](https://developer.android.com/studio/publish/app-signing.html) d’Android.

## Tester la liaison

1. Accédez à l’URL suivante dans votre navigateur : `https://{yourDomain}/.well-known/assetlinks.json`
2. Si la liaison est réussie, vous retournerez le JSON suivant (formaté pour une meilleure lisibilité) :

   ```json lines
   [{
     "relation": ["delegate_permission/common.handle_all_urls"],
     "target": {
       "namespace": "android_app",
       "package_name": "com.mycompany.app1",
       "sha256_cert_fingerprints":
       ["14:6D:E9:83:C5:73:06:50:D8:EE:B9:95:2F:34:FC:64:16:A0:83:42:E6:1D:BE:A8:8A:04:96:B2:3F:CF:44:E5"]
     }
   }]
   ```

   

   

Pour en savoir plus sur le test de votre lien vers l’application, veuillez consulter l’article [Vérifier les liens vers l’application Android](https://developer.android.com/training/app-links/verify-site-associations) au [https://developer.android.com](https://developer.android.com).
