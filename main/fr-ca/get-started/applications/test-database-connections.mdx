Vous pouvez tester des connexions de base de données pour les applications utilisant <Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>. Ces connexions configurées peuvent être utilisées pour se connecter à votre application.

Pour tester correctement, vous devez déjà avoir [configuré votre connexion de base de données](/docs/fr-ca/get-started/applications/set-up-database-connections) et [créé un utilisateur](/docs/fr-ca/manage-users/user-accounts/create-users) pour votre connexion de base de données.

1. Accédez à [Auth0 Dashboard > Authentification > Base de données](https://manage.auth0.com/#/connections/database), puis sélectionnez la flèche **Essayer** à côté de la connexion que vous souhaitez tester.

   <Frame>![Tableau de bord - Authentification - Base de données - Liste des connexions de la base de données](/images/fr-ca/cdy7uua7fh8z/1Hf6ctA9xhsLOQO9IyG0VU/da4a28587afe9f08cad475093d81f40d/DB_Connections_-_FR.png)</Frame>

2. Saisissez votre nom d'utilisateur et votre mot de passe de test. Si tout est correctement configuré, la page **Ça marche !** s'affiche :

   <Frame>![Dashboard - Connections - Database - Try - Success](/images/fr-ca/cdy7uua7fh8z/5DWJWLf8zvZJH0rSlhK0cr/f33377edf6f82f17e11e08ec30d8d75a/connection-social-try-success.png)</Frame>
