If you are a tenant administrator, you can change a user’s assigned role(s). The changes take effect after users refresh their Dashboard.

Vous pouvez attribuer à un utilisateur plusieurs rôles; les autorisations seront alors combinées. Vous devez avoir déjà ajouté le membre au locataire.

1. Accédez à [Dashboard > Paramètres > Membres locataires](https://manage.auth0.com/#/tenant/admins).
2. Click **...** to the right of a user name or email and click **Edit Roles**.
   If you don’t see that option, it means that the user has not yet accepted their invitation.
3. Sélectionnez des rôles supplémentaires ou supprimez-en.
4. Cliquez sur **Enregistrer**.

<Callout icon="file-lines" color="#0EA5E9" iconType="regular">

Les autorisations de l’utilisateur seront visibles lorsqu’il actualisera son tableau de bord. Si l’utilisateur n’actualise pas son tableau de bord, il pourra rencontrer des erreurs indiquant qu’il n’a pas accès à une zone donnée. Si l’une de ces erreurs se produit, l’utilisateur devra se déconnecter et se reconnecter.

</Callout>

**Réinitialisation des mots de passe**: Si vous ou les utilisateurs de votre Dashboard rencontrez des problèmes de connexion, consultez [Réinitialiser le mot de passe du compte Auth0](/docs/fr-ca/troubleshoot/customer-support/reset-account-passwords).

**Adresse courriel non valide**: Consultez [Mettre à jour les adresses courriel des utilisateurs du Dashboard](/docs/fr-ca/get-started/manage-dashboard-access/update-dashboard-user-email) pour savoir comment afficher les adresses courriel des membres de votre locataire et les mettre à jour si nécessaire.
