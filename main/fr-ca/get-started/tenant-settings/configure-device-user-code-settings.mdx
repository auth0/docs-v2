Vous pouvez configurer les paramètres du code utilisateur généré par votre application au cours du [Flux d’autorisation d’appareils](/docs/fr-ca/flows/concepts/device-auth) à l’aide d’<Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>.

1. Allez dans [Dashboard (Tableau de Bord) > Settings (Paramètres)](https://manage.auth0.com/#/tenant) et cliquez sur l’onglet **Advanced (Avancé)**.
2. Faites défiler jusqu’à la section **Format du code utilisateur du flux d’appareils** , localisez **Jeu de caractères du code utilisateur** et **Masque du code utilisateur**, saisissez les paramètres souhaités et cliquez sur **Enregistrer**.

<table class="table"><thead>
<tr>
<th>Paramètre</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><strong>Format du code utilisateur du flux de l'appareil</strong></td>
<td>Jeu de caractères utilisé lors de la génération aléatoire d’un code utilisateur.</td>
</tr>
<tr>
<td><strong>Masque du code utilisateur</strong></td>
<td>Masque utilisé pour définir la longueur et le format d’un code utilisateur généré de manière aléatoire. Son but est d’augmenter la lisibilité du code utilisateur et sa facilité de saisie.</td>
</tr>
</tbody>
</table>
