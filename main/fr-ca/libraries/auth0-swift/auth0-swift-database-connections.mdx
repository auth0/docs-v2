<Card title="Authentification des bases de données sur les plateformes natives">

L’authentification par nom d’utilisateur, courriel et mot de passe à partir des applications natives est désactivée par défaut pour les nouveaux locataires depuis le 8 juin 2017. Les utilisateurs sont priés d’utiliser la Connexion universelle et d’effectuer l’authentification Web à la place. Si vous souhaitez continuer, vous devez d’abord activer Password Grant Type (Type d’autorisation de mot de passe) dans votre tableau de bord. Voir [(Application Grant Types) Types d’autorisation d’application](/docs/fr-ca/get-started/applications/application-grant-types) pour en savoir plus.

</Card>

Authentication API fournit des méthodes pour authentifier et inscrire des utilisateurs de bases de données.

## Inscription avec une connexion par base de données

L’inscription nécessite d’appeler la méthode `createUser`, en passant l’`email`, le `password` et le nom de la `connection` de l’utilisateur pour initier le processus d’inscription. Vous pouvez également indiquer des métadonnées utilisateur supplémentaires à stocker.

```swift lines
Auth0
    .authentication()
    .createUser(
        email: "support@auth0.com",
        password: "secret-password",
        connection: "Username-Password-Authentication",
        userMetadata: ["first_name": "First",
                       "last_name": "Last"])
    .start { result in
        switch result {
        case .success(let user):
            print("User: \(user)")
        case .failure(let error):
            print("Failed with \(error)")
        }
    }
```

## Ouverture de session avec une connexion par base de données

L’ouverture de session avec une connexion par base de données nécessite d’appeler `login` avec le nom d’utilisateur/l’adresse courriel, le mot de passe et le nom de la connexion (comme `Username-Password-Authentication`) avec lesquels vous voulez vous authentifier. La réponse sera un objet Identifiants.

```swift lines
Auth0
    .authentication()
    .login(
        usernameOrEmail: "support@auth0.com",
        password: "secret-password",
        realm: "Username-Password-Authentication",
        scope: "openid")
     .start { result in
         switch result {
         case .success(let credentials):
            print("Credentials: \(credentials)")
         case .failure(let error):
            print("Failed with \(error)")
         }
     }
```
