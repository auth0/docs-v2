Avec les connexions sans mot de passe, les utilisateurs peuvent se connecter sans mot de passe. Ils peuvent à la place utiliser une variété d’autres méthodes d’authentification. Les connexions Auth0 sans mot de passe prennent en charge les méthodes d’authentification suivantes :

<table class="table"><thead>
<tr>
<th>Facteur</th>
<th>Description</th>
</tr>
</thead>
<tbody>
<tr>
<td><a href="/docs/fr-ca/connections/passwordless/authentication-methods/email-otp">Courriel</a></td>
<td>L’utilisateur est invité à saisir son adresse courriel, à laquelle Auth0 envoie un code à usage unique. L’utilisateur saisit le code dans votre application.</td>
</tr>
<tr>
<td><a href="/docs/fr-ca/connections/passwordless/authentication-methods/email-magic-link">Magic Link</a></td>
<td>L’utilisateur est invité à saisir son adresse courriel, à laquelle Auth0 envoie un courriel avec un lien. L’utilisateur clique sur le lien et est directement connecté à votre application.</td>
</tr>
<tr>
<td><a href="/docs/fr-ca/connections/passwordless/authentication-methods/sms-otp">SMS</a></td>
<td>L’utilisateur est invité à saisir son numéro de téléphone, auquel Auth0 envoie un code à usage unique. Par défaut, Auth0 utilise Twilio pour envoyer le code, mais si vous disposez d’une passerelle SMS personnalisée, vous pouvez modifier votre connexion pour l’utiliser à la place.</td>
</tr>
</tbody>
</table>
