La [connexion universelle](/docs/fr-ca/authenticate/login/auth0-universal-login) est une mise en œuvre du flux de connexion par Auth0. Chaque fois qu’un utilisateur doit prouver son identité, vos applications redirigent vers la connexion universelle et Auth0 fait le nécessaire pour garantir l’identité de l’utilisateur. C’est le moyen privilégié pour mettre en œuvre l’authentification sans mot de passe.

Pour mettre en œuvre l’authentification sans mot de passe dans la connexion universelle, vous devez personnaliser le code HTML de la page de connexion.

1. Accédez à [Dashboard > Image de marque > Connexion universelle](https://manage.auth0.com/#/login_settings) et cliquez sur **Options avancées**.

2. Cliquez sur l’onglet **Connexion**, puis activez le bouton **Personnaliser la page de connexion**.

<Frame>![Custom Login](/images/fr-ca/cdy7uua7fh8z/5iwLcz2O100hLuxlKN4rSs/e4f35f8c44516b98b3584d56bcaac12d/manage.auth0.com_dashboard_us_pavo_login_page__2_.png)</Frame>

3. À partir du menu déroulant Defaut Template (Modèle par défaut), sélectionnez le modèle **Custom Login Form (Formulaire de connexion personnalisé)**. Le modèle HTML sera mis à jour avec du code en utilisant CSS et la trousse SDK Auth0.js.

4. Utilisez HTML et CSS pour personnaliser le modèle selon vos spécifications, puis cliquez sur **Save Changes (Enregistrer les modifications)**. Pour en savoir plus sur l’utilisation de la trousse SDK Auth0.js avec le modèle **Custom Login Form (Formulaire de connexion personnalisé)**, voir [Auth0.js SDK](/docs/fr-ca/libraries/auth0js).

5. Sélectionnez l’onglet **Aperçu** pour consulter vos modifications. Assurez-vous de sélectionner la bonne application pour laquelle vous souhaitez prévisualiser la page de connexion.

Pour intégrer la connexion universelle dans votre application, veuillez vous référer à nos [Démarrages rapides](/docs/fr-ca/quickstarts), où vous trouverez des exemples complets pour tous les types d’applications et plateformes populaires.
