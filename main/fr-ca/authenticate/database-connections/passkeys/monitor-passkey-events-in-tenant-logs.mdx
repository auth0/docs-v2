Vous pouvez surveiller les flux de clés de chiffrement associés à vos connexions de base de données par l’intermédiaire des [journaux des locataires](/docs/fr-ca/deploy-monitor/logs).

Vous pouvez afficher les événements de journaux dans <Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip> ([Surveillance> Journaux](https://manage.auth0.com/#/logs)) ou récupérer des journaux en utilisant [Management API](https://auth0.com/docs/api/management/v2/logs/get-logs).

<table class="table"><thead>
<tr>
<th>__ Code d’événement __</th>
<th><strong>Scénario</strong></th>
<th><strong>Notes</strong></th>
</tr>
</thead>
<tbody>
<tr>
<td>`ss`</td>
<td>Lors de l’inscription, un utilisateur a enregistré une clé d’identification comme méthode d’authentification principale.</td>
<td>Cet événement de journal contient le champ de tableau `details.authentication_methods` avec la valeur d’entrée `passkey`.<br/><br/>Si un utilisateur s’est enregistré avec un mot de passe, ce champ de tableau contient la valeur `pwd`.</td>
</tr>
<tr>
<td>`s`</td>
<td>Lors de la connexion, un utilisateur s’est authentifié avec une clé d’identification.</td>
<td>Cet événement de journal contient le tableau `details.prompts` qui répertorie l’ensemble des invites qu’un utilisateur a parcourues lors de la connexion.<br/><br/>Si un utilisateur s’est authentifié à l’aide d’une clé d’identification, le tableau contient un élément avec le champ suivant : `"performed_amr": ["phr"]`.</td>
</tr>
<tr>
<td>`gd_enrollment_complete`</td>
<td>Un flux d’enregistrement progressif ou local a été complété avec succès.</td>
<td>Cet événement de journal contient le champ `details.authenticator.type` avec la valeur `passkey` si un utilisateur a inscrit une clé d’identification.</td>
</tr>
<tr>
<td>`gd_webauthn_enrollment_failed`</td>
<td>Une erreur s’est produite lors d’un flux d’enregistrement progressif ou local.</td>
<td>Cet événement de journal contient le champ `details.authentication_methods array` avec la valeur `passkey` si un utilisateur a essayé d’enregistrer une clé d’identification.</td>
</tr>
</tbody>
</table>
