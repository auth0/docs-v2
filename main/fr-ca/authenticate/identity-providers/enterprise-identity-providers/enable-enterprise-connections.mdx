Vous pouvez activer les connexions d’entreprise pour les applications à l’aide d'<Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>. Pour activer votre connexion d’entreprise, vous devez déjà l’avoir configurée.

1. Allez dans [Auth0 Dashboard > Authentification > Entreprise](https://manage.auth0.com/#/connections/enterprise) et sélectionnez le type de connexion à afficher.

   <Frame>![Dashboard (Tableau de bord) - Connections (Connexions) - Enterprise (Entreprise)](/images/fr-ca/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/4effc408fd5dda9818d307c7b798243c/Enterprise_Connections_-_FR.png)</Frame>

2. Sélectionnez le nom de la connexion à afficher.

   <Frame>![Dashboard – Connexions – Entreprise – Identifiant Entra – Liste](/images/fr-ca/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/518fbad91eae21aedcc881ffc6a19f5b/LDAP_page_-_FR.png)</Frame>

3. Sélectionnez l’affichage **Applications**, et activez ou désactivez la connexion pour la ou les applications appropriées.

   <Frame>![Dashboard – Connexions – Entreprise – Applications](/images/fr-ca/cdy7uua7fh8z/2Jwoyem76rmja1sveD88It/92474e59204a7b992ea2680b751d95ba/SSL_-_FR.png)</Frame>

Lorsque vous configurez votre connexion <Tooltip href="/docs/fr-ca/glossary?term=idp" tip="Fournisseur d’identité (IdP)
Service de stockage et de gestion des identités numériques." cta="Voir le glossaire">IdP</Tooltip> d’entreprise, rappelez-vous que le paramètre de `connection` passé au point de terminaison `/authorize,` `/whr,` ou `/wsfed` Auth0 se redirigera vers l’IdP amont, **sauf** si la connexion est effectuée via une base de données ou Active Directory.
