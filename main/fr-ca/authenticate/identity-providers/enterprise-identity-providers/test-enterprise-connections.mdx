Vous pouvez tester les connexions d’entreprise des applications à l’aide de <Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>.

Pour tester correctement, vous devez avoir [configuré votre connexion d’entreprise](/docs/fr-ca/authenticate/identity-providers/enterprise-identity-providers/enable-enterprise-connections).

1. Naviguez vers [Auth0 Dashboard > Authentification > Entreprise](https://manage.auth0.com/#/connections/enterprise), et sélectionnez le type de connexion à afficher.

   <Frame>![Dashboard (Tableau de bord) - Connections (Connexions) - Enterprise (Entreprise)](/images/fr-ca/cdy7uua7fh8z/1fSTcrZpkgkPR64NnI1lr8/4effc408fd5dda9818d307c7b798243c/Enterprise_Connections_-_FR.png)</Frame>

2. Sélectionnez la flèche **Essayer** en regard de la connexion que vous souhaitez tester.

   <Frame>![Dashboard – Connexions – Entreprise – Identifiant Entra – Liste](/images/fr-ca/cdy7uua7fh8z/1sxfMEARFGmA6cPMlWkx3c/518fbad91eae21aedcc881ffc6a19f5b/LDAP_page_-_FR.png)</Frame>

3. Connectez-vous et donnez votre accord pour autoriser l’accès à votre application. Si la configuration est correcte, la page **Ça marche!** s’affiche.

   <Frame>![Dashboard - Connections - Database - Try - Success](/images/fr-ca/cdy7uua7fh8z/5DWJWLf8zvZJH0rSlhK0cr/f33377edf6f82f17e11e08ec30d8d75a/connection-social-try-success.png)</Frame>
