La Connexion universelle est l’implémentation par Auth0 du flux d’authentification, qui est la caractéristique clé d’un Serveur d’autorisation. Chaque fois qu’un utilisateur a besoin de prouver son identité, vos applications redirigent vers Connexion universelle et Auth0 garantit l’identité de l’utilisateur. Vous pouvez choisir d’activer l’authentification Sans mot de passe lorsque vous configurez la Connexion universelle dans l’<Tooltip href="/docs/fr-ca/glossary?term=auth0-dashboard" tip="Auth0 Dashboard
Principal produit d’Auth0 pour configurer vos services." cta="Voir le glossaire">Auth0 Dashboard</Tooltip>.

Tout d’abord, déterminez si vous souhaitez que les utilisateurs utilisent SMS, courriel ou WebAuthn pour les données biométriques d’un appareil comme méthode d’authentification sans mot de passe. Il n’est pas possible actuellement de prendre en charge les trois méthodes d’authentification dans un seul flux de connexion.

## Sans mot de passe avec courriel ou SMS

Vous pouvez configurer Auth0 de sorte que les utilisateurs puissent s’authentifier avec un lien magique ou un mot de passe à usage unique (OTP) envoyé à leur adresse de courriel, ou un OTP envoyé à leur numéro de téléphone. Pour en savoir plus, consultez [Configurer courriel ou SMS pour l’authentification sans mot de passe](/docs/fr-ca/authenticate/login/auth0-universal-login/passwordless-login/email-or-sms).

## Sans mot de passe avec WebAuthn en utilisant les données biométriques d’un appareil.

Lorsque vous utilisez WebAuthn avec les données biométriques d’un appareil, les utilisateurs pourront s’authentifier avec leur mot de passe ou avec l’un de leurs appareils qui prennent en charge WebAuthn avec les données biométriques d’un appareil .

Chaque fois qu’ils se connectent à partir d’un nouvel appareil, Auth0 leur offre la possibilité d’utiliser des données biométriques au lieu d’un mot de passe. Pour en savoir plus, consultez [Sans mot de passe avec WebAuthn utilisant les données biométriques d’un appareil](/docs/fr-ca/authenticate/login/auth0-universal-login/passwordless-login/webauthn-device-biometrics).
