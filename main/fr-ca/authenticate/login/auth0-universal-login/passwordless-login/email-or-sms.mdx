Auth0 vous permet de configurer la connexion universelle afin que les utilisateurs puissent s’authentifier avec un lien magique ou un mot de passe à usage unique (OTP) par courriel ou par SMS.

Pour activer l’authentification sans mot de passe par SMS ou par courriel, vous devez définir une [connexion sans mot de passe pour votre application](/docs/fr-ca/authenticate/passwordless).

Une fois la connexion créée, vous devez configurer la connexion universelle pour pouvoir l’utiliser. Trois options s’offrent à vous :

## Universal Login (Connexion universelle) d’Auth0

Cette méthode utilise l’expérience de connexion universelle, qui prend en charge les connexions sans mot de passe. Pour en savoir plus, consultez [Connexion universelle sans mot de passe](/docs/fr-ca/authenticate/passwordless/passwordless-with-universal-login).

## Connexion universelle + Lock (sans mot de passe)

Cette méthode utilise une page de connexion personnalisée avec le modèle **Lock (sans mot de passe)** et authentifie l’utilisateur avec la [trousse SDK Lock pour Web](/docs/fr-ca/libraries/lock) :

1. Allez à [Dashboard > Branding (Image de marque) > Universal Login (Connexion universelle)](https://manage.auth0.com/#/login_settings) et cliquez sur l’onglet **Login (Connexion)**.
2. Activez la bascule **Custom Login Page (Page de connexion personnalisée)** et sélectionnez le modèle **Lock (passwordless (sans mot de passe))**. Le modèle HTML sera mis à jour avec un code utilisant le gadget logiciel Lock avec des options de personnalisation sans mot de passe.
3. Personnalisez le modèle et cliquez sur **Save Changes (Enregistrer les modifications)**.

Vous pouvez utiliser le code HTML et les feuilles de style CSS pour personnaliser le formulaire de connexion et prévisualiser les modifications dans Dashboard. Pour en savoir plus sur la manière de personnaliser le modèle **Lock (sans mot de passe)**, lisez la section Sans mot de passe dans [trousse SDK Lock pour Web](/docs/fr-ca/libraries/lock).

## Connexion universelle + IU personnalisée + Auth0.js

Cette méthode utilise une page de connexion personnalisée avec le modèle **Custom Login Form (Formulaire de connexion personnalisée)**, et authentifie les utilisateurs avec la [trousse SDK Auth0.js](/docs/fr-ca/libraries/auth0js). Pour en savoir plus, lisez [Authentification sans mot de passe avec la connexion universelle](/docs/fr-ca/authenticate/passwordless/implement-login/universal-login).
