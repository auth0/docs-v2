La base de données hébergée d’Auth0 stocke les données d’utilisateur qui sont disponibles dans les profils utilisateurs. Les informations relatives aux utilisateurs peuvent provenir de différentes sources, dont les fournisseurs d’identité, vos propres bases de données et les connexions d’entreprise (comme Active Directory ou SAML).
